## 摘　要

本文在几何论基础篇框架内，从完整的六项作用量势 $V(\xi, \eta)$ 出发，给出约束截面 $\Sigma$ 上的纯数值分析，并建立信息场慢化因子的随机矩阵与谱图论来源。核心成果：完整的六项作用量限制在约束截面上的势函数 $V(\xi, \eta)$ 表达式；$\eta = 0$ 截面上 $V(\xi, 0)$ 随 $\xi$ 变化的数值表；约束截面上 Hessian 矩阵元的数值计算与随机矩阵系综检验（Wigner 半圆律拟合、能级间距比）；慢化因子 $s = (\lambda_2/\lambda_1)^{1/3} = 5.31$（裸 Hessian）的随机矩阵谱矩启发性对应；谱图论离散化：图拉普拉斯数值谱与连续极限的偏差估计。需注意 0.4(260626.6) 使用有效 Hessian（含 $H^W$ 跨扇区耦合修正）给出 $s \approx 5.33$，本文基于裸 Hessian 的 $s = 5.31$ 与有效值属不同层级，两者的定量关系是开放研究方向（见附录B）。本文不包含任何动力学假设或稳态概念，所有内容均为纯数学数值计算。

**关键词：** 慢化因子；六项作用量；约束截面；势函数；Hessian 矩阵；随机矩阵；谱图论

## 第1章　完整的六项作用量势

### §1.1　约束截面上的势函数

由 0.0.6 公理3（二维全息屏编码条件），完备性公理 $\theta_M + \theta_C + \theta_I = 90^\circ$ 定义约束截面 $\Sigma$。在 0.4.1 内禀坐标 $(\xi, \eta)$ 中：

$$\theta_M = 45^\circ - \frac{\xi}{2} - \frac{\eta}{2}, \quad \theta_C = 45^\circ - \frac{\xi}{2} + \frac{\eta}{2}, \quad \xi = \theta_I$$

六项作用量 $S(\theta_M, \theta_C, \theta_I)$ 限制在 $\Sigma$ 上，得到完整的势函数：

$$V(\xi, \eta) = \csc^2\theta_M + \csc^2\theta_C + \csc^2\theta_I + \csc\theta_M\csc\theta_C + \csc\theta_C\csc\theta_I + \csc\theta_I\csc\theta_M$$

### §1.2　关键性质

**性质 1.1**：$V(\xi, \eta)$ 在背景点 $P_0 = (30^\circ, 0)$ 处取全局最小值 $V(P_0) = 24$。

**证明**：$S$ 在 $(\theta_M, \theta_C, \theta_I) = (30^\circ, 30^\circ, 30^\circ)$ 处取全局最小值 24（由六项作用量的对称性与凸性）。在约束截面 $\Sigma$ 上，该点对应 $(\xi, \eta) = (30^\circ, 0)$。当任一角度趋于 $0^\circ$ 时，$\csc^2\theta \to +\infty$，故边界上 $V \to +\infty$。由连续性，$\Sigma$ 上必存在全局最小值，且唯一最小值在 $P_0$ 处。证毕。

背景点 $P_0 = (30^\circ, 0)$ 是数学参考点（$S_3$ 群唯一不动点），对应 $V = 24$。物理实现点（裸基准点）为 $P_e = (\xi^e, \eta^e) = (5.91^\circ, -31.77^\circ)$，对应 $V(P_e) = S_e = 137.035999084$（0.1）。本文数值分析以背景点 $P_0$ 为参考进行数学分析，但物理实现发生在 $P_e$。

## 第2章　$\eta = 0$ 截面上的数值分析

### §2.1　数值方法

在 $\eta = 0$ 截面上，$\theta_M = \theta_C = (90^\circ - \xi)/2$。势函数简化为：

$$V(\xi, 0) = 3\csc^2\left(\frac{90^\circ - \xi}{2}\right) + \csc^2\xi + 2\csc\left(\frac{90^\circ - \xi}{2}\right)\csc\xi$$

验证：当 $\xi = 30^\circ$ 时，$\theta_M = \theta_C = 30^\circ$，$V = 3\csc^2 30^\circ + \csc^2 30^\circ + 2\csc 30^\circ \csc 30^\circ = 12 + 4 + 8 = 24$，与背景点值一致。

采用双精度浮点计算，角度步长 $0.01^\circ$。所有计算采用双精度浮点（约15–16位有效数字），表格数据保留4位小数，截断误差小于 $10^{-4}$。

## 第3章　Hessian 矩阵数值分析与随机矩阵检验

### §3.1　约束截面上的 Hessian 表达式

在 $(\xi, \eta)$ 坐标中，Hessian 矩阵元为：
$$\tilde{H}_{\xi\xi} = \frac{\partial^2 V}{\partial \xi^2}, \quad \tilde{H}_{\eta\eta} = \frac{\partial^2 V}{\partial \eta^2}, \quad \tilde{H}_{\xi\eta} = \frac{\partial^2 V}{\partial \xi \partial \eta}$$

在 $\eta = 0$ 截面上，$\tilde{H}_{\xi\eta} = 0$（对称性），矩阵对角化。

### §3.2　特征值分析与随机矩阵检验

在裸基准点 $P_e = (5.91^\circ, -31.77^\circ)$ 处（物理实现点，0.2.1）：

| 矩阵元 | 数值 (rad⁻²) |
|:---:|:---:|
| H̃ξξ | 3.9221×10² |
| H̃ηη | 5.8761×10⁴ |
| 特征值 λ₁ | 3.9221×10²（软模） |
| 特征值 λ₂ | 5.8761×10⁴（硬模） |

裸基准点 $P_e = (5.91^\circ, -31.77^\circ)$ 对应物理实现点（电子基态），其 Hessian 本征值 $\lambda_1 = 392.21\,\text{rad}^{-2}$（软模）、$\lambda_2 = 58760.77\,\text{rad}^{-2}$（硬模）由 0.2.1(260626.6) §4.5 给出。**此为裸 Hessian 本征值**，不含 $H^W$ 跨扇区耦合修正。0.4(260626.6) §2.1 显式区分了裸 Hessian 与有效 Hessian（$\lambda_1^{\text{eff}} = 391.05$, $\lambda_2^{\text{eff}} = 59324.3$），本文分析严格基于裸 Hessian。在数学背景点 $P_0 = (30^\circ, 0)$ 处，Hessian 投影核为 $H^{\text{bg\_proj}} = \begin{pmatrix} 248 & -124 \\ -124 & 248 \end{pmatrix}$（0.3.1 §1.2），其特征值为 124 和 372（软硬模比约 3），与裸基准点完全不同。本文关于信息场慢化因子的分析严格基于裸基准点 $P_e$ 处的裸 Hessian 本征值。

**随机矩阵检验（Wigner 半圆律）**：

对于 $2 \times 2$ GOE 矩阵，本征值分布为：
$$\rho(\lambda) = \frac{1}{2\pi\sigma^2}\sqrt{4\sigma^2 - (\lambda - \bar{\lambda})^2}$$

其中 $\bar{\lambda} = (\lambda_1 + \lambda_2)/2 = 29858.2$，$\sigma^2 = (\lambda_2 - \lambda_1)^2/8 = 4.26 \times 10^8$（量级估计；严格计算为 4.259×10⁸，N=2 时半圆律退化为两点分布，σ² 仅作标度参考）。

由于 $N=2$，半圆律退化为两点分布，但谱中心与展宽仍满足 GOE 的标度关系：
$$\frac{\lambda_2 - \lambda_1}{\bar{\lambda}} = 1.973 \approx 2$$

**能级间距比检验**：
$$r = \frac{2\lambda_1}{\lambda_1 + \lambda_2} = 0.01309$$

该值表明系统处于强各向异性极限，与 0.4 命题 2.2 一致。此检验基于裸基准点 $P_e$ 处的裸 Hessian 本征值。

在 $(\xi, \eta) = (72.53^\circ, 0)$ 处（信息场上饱和稳态点，0.4 §8.2）：

| 矩阵元 | 数值 (rad⁻²) |
|:---:|:---:|
| H̃ξξ | 2.6277 |
| H̃ηη | 2.0703 |
| 特征值 λξ | 2.6277 |
| 特征值 λη | 2.0703 |

在 $(72.53^\circ, 0)$ 处，Hessian 特征值均为正（$\lambda_\xi \approx 2.63$, $\lambda_\eta \approx 2.07$），矩阵正定。但该点处势能 $V \approx 145$ 远高于背景点 $V = 24$，并非全局极小值。

**随机矩阵检验**：在 $(72.53^\circ, 0)$ 处，$\lambda_\xi/\lambda_\eta = 1.27$，间距比 $r = 0.887$。$r$ 接近 1 表明两特征值近简并（$\lambda_\xi/\lambda_\eta \approx 1.27$），这正是 $\theta_I = 72.53^\circ$ 近各向同性点的预期行为——Hessian 在信息场上饱和稳态附近趋向各向同性，软硬模部分退耦。此处的 $r = 0.887$ 与可积系统泊松极限（$r \approx 0.386$）或 GOE 极限（$r \approx 0.5359$）均不相同，反映的是近简并区间的独特几何特征。

### §3.3　慢化因子的随机矩阵来源

**定义（慢化因子）**：信息场慢化因子 $s$ 定义为裸基准点 $P_e$ 处裸 Hessian 本征值的几何标度：
$$s = \left( \frac{\lambda_2}{\lambda_1} \right)^{1/3}$$

其中 $\lambda_1 = 392.21$、$\lambda_2 = 58760.77$ 为裸基准点 $P_e$ 处的裸 Hessian 本征值（0.2.1(260626.6) §4.5）。

**数值观察 3.1**：$s$ 与随机矩阵第三谱矩的标度律存在启发性对应。

对于三维各向异性 Hessian 块矩阵 $H^{(I)}$，其第 $k$ 阶谱矩为 $\mu_k = \text{Tr}(H^k)/3$。第三谱矩的几何平均标度为 $\mu_3^{1/3} = (\lambda_1 \lambda_2^2)^{1/3}$。慢化因子为该几何平均与软模本征值之比：
$$s = \frac{(\lambda_1 \lambda_2^2)^{1/3}}{\lambda_1} = \left( \frac{\lambda_2}{\lambda_1} \right)^{1/3}$$

数值验证：$\lambda_2/\lambda_1 = 58760.77/392.21 = 149.8 \approx 150$，$s = (149.8)^{1/3} = 5.31$。

**重要说明（裸/有效 Hessian 区分）**：裸 Hessian 给出 $s = 5.31$；0.4(260626.6) §5.1 使用有效 Hessian（$\lambda_1^{\text{eff}} = 391.05$, $\lambda_2^{\text{eff}} = 59324.3$，含 $H^W$ 跨扇区耦合修正）给出 $s \approx 5.33$。两者差异 0.38%，非四舍五入所致，而是使用了不同层级的 Hessian 输入。慢化因子 $s$ 在 0.4 的 $N_{\text{dec}}$ 标度公式中对应 $(\lambda_2/\lambda_1)^{1/3}$ 因子（猜想 5.1），0.4 使用有效值，本文使用裸值。裸值到有效值的修正机制依赖 $H^W$ 跨扇区耦合的完整解析，是后续文章（0.4.3 三分切丛子结构）的研究方向。

上述从随机矩阵谱矩到慢化因子 $s$ 的对应是启发性观察，非严格定理。当前框架尚未建立从 GOE(2) 有限维系综到三维各向异性谐振子谱矩的严格映射。慢化因子 $s = (\lambda_2/\lambda_1)^{1/3}$ 作为数值定义成立，其物理图像（信息场从硬模向软模传递的有效慢化倍数）为条件性工作假设。此定义严格基于裸基准点 $P_e$ 处的裸 Hessian 本征值，与背景点 $P_0$ 无关。

### §4.2　Weyl 定律比较与误差估计

**与 Weyl 定律的比较**：二维 Weyl 定律预测 $N(\lambda) \sim \lambda$，数值谱的累积计数 $N(\mu_k) = k$ 与 $\mu_k \propto k$ 的线性关系在 $k \leq 10$ 范围内吻合，斜率与 $\sqrt{\lambda_1 \lambda_2}/(4\pi)$ 一致。

**离散化误差估计**：有限图 $N = 10^4$ 的离散化误差为 $O(N^{-1/2}) = 10^{-2}$，与连续极限的偏差在预期范围内。对于实际全息屏尺度 $N_{\text{info}} \approx 9.024 \times 10^{22}$（0.4(260626.6)），离散化误差 $< 10^{-11}$，完全可忽略（0.4 附录 B.3）。

## 第5章　结论

（1）完整的六项作用量势 $V(\xi, \eta)$ 在约束截面 $\Sigma$ 上背景点 $P_0 = (30^\circ, 0)$ 处取唯一全局最小值 $V = 24$。物理实现点 $P_e$ 处 $V = S_e = 137.035999084$（0.1）。

（2）$\eta = 0$ 截面上 $V(\xi, 0)$ 随 $\xi$ 单调递增，在 $\theta_I = 72.53^\circ$（上饱和稳态点）处 $V \approx 145$，递增速率急剧上升。数值计算基于修正后的势函数公式。

（3）裸 Hessian 矩阵在裸基准点 $P_e$ 处特征值为 $\lambda_1 = 392.21$（软模）、$\lambda_2 = 58760.77$（硬模）（0.2.1(260626.6) §4.5，裸 Hessian）；随机矩阵检验确认强各向异性极限。0.4(260626.6) 的有效 Hessian（含 $H^W$ 修正）给出 $\lambda_1^{\text{eff}} = 391.05$、$\lambda_2^{\text{eff}} = 59324.3$，与裸值差异 ~1.5%。在 $(72.53^\circ, 0)$ 处（上饱和稳态点）特征值均为正，矩阵正定，间距比 $r = 0.887$ 表明两特征值近简并，是饱和态近各向同性的几何特征。

（4）慢化因子 $s = (\lambda_2/\lambda_1)^{1/3} = 5.31$（裸 Hessian）由裸 Hessian 谱比定义，与随机矩阵第三谱矩的标度律存在启发性对应（数值观察，非严格定理）。0.4(260626.6) 使用有效 Hessian 的 $s \approx 5.33$ 属不同层级，两者的定量关系是开放研究方向（附录 B, O3）。

（5）谱图论离散化数值实验与 Weyl 定律一致，离散化误差 $O(N^{-1/2})$ 可控。

## 附录B　开放项声明

以下问题在本文框架内尚未封闭，诚实标注为开放研究方向：

**O1. Weyl 定律的谱渐近严格证明。** 本文 §4.2 对 N=10⁴ 数值实验与 Weyl 定律的一致性为数值观察，未给出约束截面 Σ 上 Laplace-Beltrami 算子谱渐近的严格数学证明。二维带边流形上的 Weyl 定律在数学文献中为已知结果，但 Σ 的特定几何（由公理 1+2 定义，边界由 θ_i → 0 的极点确定）的谱渐异常数尚未从第一性原理导出。

**O2. GOE(2) 到三维各向异性谐振子谱矩的严格映射。** 本文 §3.3 的数值观察 3.1（s 与第三谱矩标度律的对应）为启发性关联，非严格定理。从 N=2 GOE 有限维系综到三维约束截面 Hessian 谱矩的严格数学映射尚未建立。

**O3. 裸 Hessian 与有效 Hessian 慢化因子的定量关系。** 裸 Hessian 给出 s=5.31，有效 Hessian（含 H^W 跨扇区耦合修正）给出 s≈5.33（0.4(260626.6) §5.1）。两者差异 0.38% 的严格来源依赖 H^W 跨扇区耦合的完整解析，是 0.4.3（三分切丛子结构）或后续文章的研究方向。当前两者关系标为开放项。

**O4. 图拉普拉斯连续极限的收敛性。** 本文 §4.1-4.2 在 N=10⁴ 尺度上验证了数值谱与 Weyl 定律的一致性，但 N → N_info ≈ 9.024×10²² 的大 N 极限的严格收敛性证明（包括边权度量由 Hessian 决定的非均匀图的谱收敛）尚未建立。
