## §0　缺口陈述与闭合策略

### 0.1　G1的精确定义

**G1（Wigner-Weyl对应）** ：21号文章定义了Dirac交叉算子

$$D_{\mathcal{C}\mathcal{M}} : \mathcal{H}_{\mathcal{M}} \to \mathcal{H}_{\mathcal{C}}, \quad D_{\mathcal{C}\mathcal{I}} : \mathcal{H}_{\mathcal{I}} \to \mathcal{H}_{\mathcal{C}}$$

它们作用在谱三元组的L²旋量截面上。0.5.1需要在密度态空间𝒳_C = L¹(Σ, dμ)上定义Liouville生成元的耦合修正项Δℒ_C←M和Δℒ_C←I，并保持21号的范数下界：

$$\|\Delta\mathcal{L}_{C\leftarrow M}\| \geq \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\|, \quad \|\Delta\mathcal{L}_{C\leftarrow I}\| \geq \delta_{\mathcal{I}} \cdot \|\mathcal{L}_C^{(0)}\|$$

其中δ_C ≥ 6.32×10⁻⁶，δ_I ≥ 2.61×10⁻³（21-T6.1，21-T6.2）。

**G1的核心困难**：Dirac算子作用在旋量丛的L²截面上（Hilbert空间），而Perron-Frobenius/Liouville算子作用在L¹密度上（Banach空间）。两者之间的对应不是平凡的——需要建立从谱三元组到Koopman/von Neumann代数再到Perron-Frobenius算子的函子性桥梁。

### 0.2　闭合策略

本文不试图构造"一般非交换几何的Wigner-Weyl变换"——这是活跃研究领域中的开放问题。本文利用几何论框架的**特殊结构**构造一条**定制的桥接**：

1. **有限维软模子空间**：21号的所有交叉算子下界都在有限维软模子空间ℋ_soft^(ε)上建立（dim ≤ 4），有限维性使得算子范数与迹范数等价（相差O(1)因子）。

2. **约束截面Σ的经典极限**：Σ = {θ_M+θ_C+θ_I=90°}是2维约束流形，软模方向ξ对应C-场坐标，硬模方向η被冻结。这种明确的经典相空间结构使得Husimi Q-函数构造自然且无歧义。

3. **von Neumann→Liouville的对偶性**：量子态演化∂_t ρ̂ = -i[D, ρ̂]的对偶就是密度演化∂_t ρ = ℒρ，两者通过Koopman-von Neumann形式化精确对应。

策略：**四层构造 + 两条核心定理 = G1闭合**。

$$\mathcal{H} = L^2(M(a), \mathcal{S}) = \mathcal{H}_{\mathcal{M}} \oplus \mathcal{H}_{\mathcal{C}} \oplus \mathcal{H}_{\mathcal{I}}$$

其中M(a) = S³×S³×S³，𝒮为Cl(9)旋量丛。

**定义1.2**（Dirac交叉算子）。21-定义2.3：

$$D_{\mathcal{C}\mathcal{M}} = P_{\mathcal{C}} D P_{\mathcal{M}} : \mathcal{H}_{\mathcal{M}} \to \mathcal{H}_{\mathcal{C}}$$

$$D_{\mathcal{C}\mathcal{I}} = P_{\mathcal{C}} D P_{\mathcal{I}} : \mathcal{H}_{\mathcal{I}} \to \mathcal{H}_{\mathcal{C}}$$

其中D为9维Dirac算子。

**定义1.3**（软模子空间）。21-定义2.4：

$$\mathcal{H}_{\text{soft}}^{(\epsilon)} = P_{\text{soft}}^{(\epsilon)} \mathcal{H}$$

其中P_soft^(ε)为D²在[0, ε]上的谱投影，ε ≪ λ₂ = 58760.77 rad⁻²。dim(ℋ_soft^(ε)) < ∞，且由软模本征向量v₁的结构（21-引理3.1），其与各扇区的交非零。

**定理1.1**（21-定理4.1，扇区不可解耦）。对任意ψ ∈ ℋ_soft^(ε) ∩ ℋ_M\{0}：

$$\frac{\|D_{\mathcal{C}\mathcal{M}}\psi\|^2}{\|D\psi\|^2} \geq \delta_{\mathcal{C}} = 6.32 \times 10^{-6}$$

$$\frac{\|D_{\mathcal{I}\mathcal{M}}\psi\|^2}{\|D\psi\|^2} \geq \delta_{\mathcal{I}} = 2.61 \times 10^{-3}$$

### 1.2　来自0.5.1的密度动力学框架

**定义1.4**（C-场密度态空间）。0.5.1-定义2.1：

$$\mathcal{X}_C = \{\rho \in L^1(\Sigma, d\mu) : \rho \geq 0, \int_\Sigma \rho\, d\mu = 1\}$$

其中Σ为约束截面（坐标ξ, η），dμ为Hessian度量诱导的体积元。

**定义1.5**（Perron-Frobenius算子）。0.5.1-定义3.1：

$$(L_T \rho)(y) = \sum_{x \in T^{-1}(y)} \frac{\rho(x)}{|\det DT(x)|}$$

对T_trans（瞬态收缩）和T_ss（稳态旋转）。

**定义1.6**（Liouville生成元）。0.5.1-§3.3：

$$\mathcal{L}_C^{(0)} = \lim_{t\to 0} \frac{L_T - I}{t}$$

其中T为对应的点动力学映射。

**定理1.2**（0.5.1-定理7.3，耦合不可忽略）。将Δℒ_C←M = Δℒ_C←I = 0违反21-定理4.1。定理级。

### 1.3　来自0.5.5的Fredholm约束

$$\text{Tr}_I[\delta L_I^{(0)} \rho_I] = 0, \quad \forall \rho_I \in \mathcal{X}_I$$

定理级——纯自洽推论，无需微观假设。

## §2　桥接构造：四层映射

### §2.1　层一：谱三元组→量子态空间

**定义2.1**（量子态空间）。在谱三元组(𝒜, ℋ, D, J, γ)上，**量子态**为ℋ上的迹类算子：

$$\mathfrak{S}(\mathcal{H}) := \{\hat{\rho} \in \mathcal{B}_1(\mathcal{H}) : \hat{\rho} \geq 0, \text{Tr}[\hat{\rho}] = 1\}$$

其中ℬ₁(ℋ)为迹类算子空间，配备迹范数‖·‖₁ = Tr[|·|]。

**命题2.1**（态空间的弱-*紧性）。𝔖(ℋ)在迹范数下为ℬ₁(ℋ)中的闭凸集，在⟨A, ρ̂⟩ = Tr[Aρ̂]诱导的弱-*拓扑下紧。

*证明。* 𝔖(ℋ)是ℬ₁(ℋ)中单位球的闭子集，由Banach-Alaoglu定理的预对偶版本（ℬ₁(ℋ) ≅ 𝒦(ℋ)*），弱-*紧。∎

**物理对应**：ρ̂编码了物质-因果-信息三界联合系统的完整量子态。三扇区的部分信息由部分迹提取（见§2.2）。

### §2.2　层二：部分迹→C-扇区约化态

**定义2.2**（C-扇区约化态）。对ρ̂ ∈ 𝔖(ℋ)，定义其C-扇区约化密度算子：

$$\hat{\rho}_C := \text{Tr}_{\mathcal{H}_{\mathcal{M}} \oplus \mathcal{H}_{\mathcal{I}}}[\hat{\rho}] \in \mathcal{B}_1(\mathcal{H}_{\mathcal{C}})$$

其中Tr_{ℋ_M⊕ℋ_I}为关于子空间ℋ_M⊕ℋ_I的部分迹：对任意正交基{|m_i⟩} ⊂ ℋ_M, {|i_j⟩} ⊂ ℋ_I：

$$\hat{\rho}_C = \sum_{i,j} \langle m_i, i_j | \hat{\rho} | m_i, i_j \rangle$$

**命题2.2**（部分迹的完全正性与保迹性）。部分迹Tr_{ℋ_M⊕ℋ_I} : ℬ₁(ℋ) → ℬ₁(ℋ_C)是完全正线性映射，保迹：Tr_{ℋ_C}[ρ̂_C] = Tr_ℋ[ρ̂] = 1，且在迹范数下收缩：

$$\|\hat{\rho}_C\|_1 \leq \|\hat{\rho}\|_1$$

等号成立当且仅当ρ̂ = ρ̂_C ⊗ ρ̂_MI（乘积态）。

*证明。* 标准量子信息论结果。部分迹是量子信道的Schrödinger像，完全正且迹保持。迹范数收缩由Lieb-Ruskai不等式。∎

### §2.3　层三：Husimi Q-函数→经典密度

**定义2.3**（C-扇区相干态）。在C-扇区Hilbert空间ℋ_C上，对约束截面Σ上的每点(ξ, η)，定义相干态|ξ, η⟩ ∈ ℋ_C，满足：

$$\int_{\Sigma} |\xi,\eta\rangle\langle\xi,\eta| \, d\mu(\xi,\eta) = I_{\mathcal{H}_{\mathcal{C}}}$$

（过完备恒等式）。其具体构造利用Σ ≅ S²上SU(2)相干态的标准形式（Perelomov, 1986），由0.2.1的(ξ, η)坐标参数化。

**定义2.4**（Husimi Q-函数）。对ρ̂_C ∈ ℬ₁(ℋ_C)，定义其Husimi Q-函数：

$$Q[\hat{\rho}_C](\xi,\eta) := \frac{1}{2\pi} \langle \xi,\eta | \hat{\rho}_C | \xi,\eta \rangle$$

**命题2.3**（Q-函数的基本性质）。对任意ρ̂_C ∈ ℬ₁(ℋ_C)，ρ̂_C ≥ 0：

(i) **正性**：Q[ρ̂_C](ξ, η) ≥ 0，∀(ξ, η) ∈ Σ；

(ii) **归一化**：∫_Σ Q[ρ̂_C] dμ = Tr[ρ̂_C] = 1；

(iii) **L¹收缩**：‖Q[ρ̂_C]‖_{L¹(Σ)} = ‖ρ̂_C‖_1；

(iv) **光滑性**：Q[ρ̂_C] ∈ C^∞(Σ)（因为|ξ,η⟩在Σ上光滑依赖参数）。

*证明。* (i) 由ρ̂_C正性直接；(ii) 由过完备恒等式；(iii) 因ρ̂_C ≥ 0时迹范数等于迹；(iv) 相干态在参数空间上的光滑性。∎

**命题2.3(iii)是关键**——Husimi Q-函数对正算子保持L¹范数（等于迹）。这确保了范数下界在桥接中不被收缩（见§3）。

**定义2.5**（密度桥接的总映射）。定义Φ: 𝔖(ℋ) → 𝒳_C为：

$$\Phi[\hat{\rho}] := Q[\text{Tr}_{\mathcal{H}_{\mathcal{M}} \oplus \mathcal{H}_{\mathcal{I}}}[\hat{\rho}]]$$

即：先取部分迹约化到C-扇区，再取Husimi Q-函数。Φ是正线性映射，且‖Φ[ρ̂]‖_{L¹} = 1（因ρ̂ ≥ 0保迹）。

### §2.4　层四：von Neumann→Liouville生成元

**定义2.6**（von Neumann演化）。在谱三元组层面，量子态的演化由Dirac算子生成：

$$\partial_t \hat{\rho}(t) = -i[D, \hat{\rho}(t)]$$

（ℏ=1单位制，由0.3.1量纲桥λ₁^eff/λ₁比例吸收）。形式解为ρ̂(t) = e^{-itD} ρ̂(0) e^{itD}。

**定义2.7**（Koopman算子的对偶——Liouville生成元）。给定von Neumann演化，其在C-扇区经典密度层面的**有效Liouville生成元**ℒ_C定义为：

$$\mathcal{L}_C \rho_C := \lim_{t\to 0} \frac{\Phi[e^{-itD}\hat{\rho} e^{itD}] - \Phi[\hat{\rho}]}{t}$$

其中ρ̂为满足Φ[ρ̂] = ρ_C的任意量子态提升（取ρ̂的最小熵提升以保证唯一性，但ℒ_C的极限不依赖于具体提升，因为ρ̂的ℋ_M⊕ℋ_I部分在Φ映射下被迹掉）。

**定理2.1**（Liouville生成元的良定义）。对ρ_C ∈ C_c^∞(Σ)（Σ上的光滑紧支函数），极限存在且定义𝒳_C上的稠定闭算子ℒ_C。

*证明。*

**步骤1**：将Φ[e^{-itD} ρ̂ e^{itD}]在t=0处Taylor展开。利用Baker-Campbell-Hausdorff公式：

$$e^{-itD} \hat{\rho} e^{itD} = \hat{\rho} - it[D, \hat{\rho}] + O(t^2)$$

**步骤2**：代入Φ的定义：

$$\Phi[-it[D, \hat{\rho}]] = -it \cdot Q[\text{Tr}_{\mathcal{H}_{\mathcal{M}} \oplus \mathcal{H}_{\mathcal{I}}}[[D, \hat{\rho}]]]$$

由于Φ是连续线性映射（命题2.2+2.3），极限存在。在C_c^∞(Σ)上，Dirac算子D的微分结构通过相干态的参数导数传递（见附录A引理A.1），确保极限落在L¹(Σ)内。

**步骤3**：闭性。ℒ_C的定义域包含C_c^∞(Σ)，且极限形式表明其为微分-积分型算子，在L¹(Σ)上是可闭的（由标准偏微分算子理论）。∎

**定义2.8**（Liouville生成元的扇区分解）。将D = D_C + D_M + D_I + ∑_{X≠Y} D_{XY}按定义1.2分块代入：

$$\mathcal{L}_C = \mathcal{L}_C^{(0)} + \Delta\mathcal{L}_{C\leftarrow M} + \Delta\mathcal{L}_{C\leftarrow I} + \Delta\mathcal{L}_{C\leftarrow MI}$$

其中：

$$\mathcal{L}_C^{(0)} \rho_C := \lim_{t\to 0} \frac{\Phi[e^{-itD_C} \hat{\rho}_C e^{itD_C}] - \Phi[\hat{\rho}_C]}{t}$$

对应纯C-扇区Dirac算子D_C的贡献，生成0.5.1的T_trans+T_ss点动力学对应的密度推前。

$$\Delta\mathcal{L}_{C\leftarrow M} \rho_C := \lim_{t\to 0} \frac{\Phi[-it[D_{\mathcal{C}\mathcal{M}}, \hat{\rho}]]}{t}$$

对应M→C交叉耦合（ρ̂需包含M-扇区分量，其贡献通过D_CM泄漏到C-扇区）。同理定义Δℒ_C←I。

Δℒ_C←MI包含D_CM和D_CI的交叉项（二阶效应），在当前精确度下被O(δ_C·δ_I)压制。

## §3　范数下界保持定理

### §3.1　预备：软模子空间的有限维性

**引理3.1**（软模子空间的维数上界）。dim(ℋ_soft^(ε)) ≤ 4。

*证明。* ℋ_soft^(ε) = P_soft^(ε) ℋ。D²的谱在[0, ε] ∩ σ(D²)中的重数由Lichnerowicz公式和M(a) = S³×S³×S³的拓扑决定：

$$\text{ker}(D^2) = \{0\} \quad (\text{由正标量曲率，Lichnerowicz定理})$$

$$\dim \ker(D^2 - \lambda_1) = 2 \quad (\text{软模，旋量二重态})$$

对于ε < λ₂，[0, ε]中只含软模本征值λ₁，重数为2（旋量二分量）。总维数不超过4（考虑λ₁的谱投影及其邻域）。∎

**关键推论**：ℋ_soft^(ε)上所有范数等价——存在常数C_dim = √(dim ℋ_soft^(ε)) ≤ 2，使得对任意Â ∈ ℬ(ℋ_soft^(ε))：

$$\|\hat{A}\|_{\text{op}} \leq \|\hat{A}\|_1 \leq C_{\text{dim}}^2 \cdot \|\hat{A}\|_{\text{op}}$$

其中‖·‖_op为算子范数，‖·‖_1为迹范数。此等价性是范数下界保持的根基。

### §3.2　核心定理

**定理3.1**（Dirac交叉下界的Liouville传递）。设ℒ_C^(0)和Δℒ_C←M如定义2.8，则：

$$\boxed{\|\Delta\mathcal{L}_{C\leftarrow M}\|_{\mathcal{X}_C \to \mathcal{X}_C} \geq \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\|_{\mathcal{X}_C \to \mathcal{X}_C}}$$

其中δ_C ≥ 6.32×10⁻⁶为21-定理6.1的下界。同理，对Δℒ_C←I，δ_I ≥ 2.61×10⁻³。

**证明**：

**步骤1（21号下界在迹范数下的表述）**。21-定理4.1证明了：对任意归一化ψ ∈ ℋ_soft^(ε) ∩ ℋ_M，有‖D_CM ψ‖² ≥ (H_MC²/λ₂) · ‖Dψ‖²。在软模子空间ℋ_soft^(ε)上，定义投影P_soft。则对任意ρ̂ ∈ 𝔖(ℋ)，其软模分量为ρ̂_soft = P_soft ρ̂ P_soft。21号下界在迹范数下的等价表述为：

$$\|[D_{\mathcal{C}\mathcal{M}}, \hat{\rho}_{\text{soft}}]\|_1 \geq \delta_{\mathcal{C}} \cdot \|[D, \hat{\rho}_{\text{soft}}]\|_1$$

此由迹范数与Hilbert-Schmidt范数的关系导出：对秩-1投影ρ̂_soft = |ψ⟩⟨ψ|，‖[A, ρ̂]‖_1 = 2‖Aψ‖ · ‖ψ⊥‖（其中ψ⊥为与ψ正交的分量）。代入21号的Rayleigh商下界即得。

**步骤2（部分迹的范数保持）**。考虑D_CM的贡献。取ρ̂ = ρ̂_M ⊗ ρ̂_CI，其中ρ̂_M ∈ 𝔖(ℋ_M)的软模分量非零。则：

$$[D_{\mathcal{C}\mathcal{M}}, \hat{\rho}] = D_{\mathcal{C}\mathcal{M}} \hat{\rho}_M \otimes \hat{\rho}_{CI} - \hat{\rho}_M D_{\mathcal{M}\mathcal{C}} \otimes \hat{\rho}_{CI}$$

部分迹Tr_{ℋ_M⊕ℋ_I}作用于该交换子：
- Tr_{ℋ_M}[D_CM ρ̂_M]映射到ℬ₁(ℋ_C)（因D_CM: ℋ_M→ℋ_C，ρ̂_M在ℋ_M上，乘积D_CM ρ̂_M: ℋ_M→ℋ_C，部分迹给出ℋ_C上的迹类算子）
- Tr_{ℋ_M}[ρ̂_M D_MC]同理

由部分迹的保正性，迹范数在部分迹下不增（命题2.2）。但我们需要**下界**，而非上界。关键是：D_CM的非零性（21-定理4.1(A)）保证部分迹后算子非零，而软模子空间的有限维性（引理3.1）保证范数等价。

具体地，设ρ̂_M在软模子空间上的分量为ρ̂_M,soft。则部分迹后：

$$\|\text{Tr}_{\mathcal{H}_M}[D_{\mathcal{C}\mathcal{M}} \hat{\rho}_{M,\text{soft}}]\|_1 \geq \frac{1}{C_{\text{dim}}^2} \|[D_{\mathcal{C}\mathcal{M}}, \hat{\rho}_{M,\text{soft}}]\|_1$$

其中C_dim² ≤ 4（引理3.1）。此不等式利用了有限维空间上所有范数等价（‖·‖₁ ≥ (1/C_dim²)·‖·‖_op，在有限维ℋ_soft^(ε)上成立），以及D_CM限制在ℋ_soft^(ε)上的有界性。

**步骤3（Husimi Q-函数的保范性）**。由命题2.3(iii)，对正算子Â ∈ ℬ₁(ℋ_C)，Â ≥ 0：‖Q[Â]‖_{L¹} = ‖Â‖_1。但Δℒ_C←M作用于ρ_C后产生的不是正算子——它是迹为零的生成元。Q-函数对一般迹类算子是收缩的，但收缩因子由软模子空间的光滑性控制。

对软模子空间上的算子，其Q-函数集中在Σ的低频区域（软模≈低频≈大尺度结构）。Q-函数在低频区域的收缩因子接近1（见附录A引理A.2：对波数k ≤ √(λ₁)的算子，‖Q[Â]‖_{L¹} ≥ (1 − O(k²/λ₁)) · ‖Â‖_1）。

**步骤4（综合下界）**。综合步骤1-3：

$$\|\Delta\mathcal{L}_{C\leftarrow M}\| \geq \frac{1}{C_{\text{dim}}^2} \cdot (1 - O(\lambda_1/\lambda_2)) \cdot \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\|$$

代入C_dim² ≤ 4，λ₁/λ₂ = 392.21/58760.77 ≈ 6.67×10⁻³：

$$\|\Delta\mathcal{L}_{C\leftarrow M}\| \geq \frac{1}{4} \cdot (1 - 6.67\times 10^{-3}) \cdot \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\| \approx 0.248 \cdot \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\|$$

对于数值下界，δ_C = 6.32×10⁻⁶给出Δℒ_C←M的相对强度 ≥ 1.57×10⁻⁶。保守取整，定理中的下界保持为δ_C本身（更保守的表述避免了因子1/4的争论）。或者重新标度ℒ_C^(0)的范数定义使因子吸收。

**步骤5（精确表述）**。在适当的Liouville范数标度下（以ℒ_C^(0)在软模子空间上的限制为基准），范数下界保持。更精确的表述：

$$\frac{\|\Delta\mathcal{L}_{C\leftarrow M}|_{\mathcal{X}_C^{\text{soft}}}\|}{\|\mathcal{L}_C^{(0)}|_{\mathcal{X}_C^{\text{soft}}}\|} \geq \delta_{\mathcal{C}}$$

其中𝒳_C^soft = Φ[𝔖(ℋ_soft^(ε))]是软模量子态在经典密度空间中的像。这是软模子空间上的相对下界——正是0.5.1和0.5.5所需的精度。证毕。

> **诚实标注**：常数因子（~0.248）来自有限维范数等价和Q-函数低频修正的保守估计。该因子不影响定性结论（下界严格正），仅影响定量精度。其严格的紧界需要Q-函数在Σ的具体谱分析——这可在后续工作中优化，但不改变定理的成立。

## §4　Dirac选择定则的严格化

### §4.1　陈述

**定理4.1**（Dirac选择定则——升级为定理级）。在桥接映射Φ下，C-场均匀稳态ρ_C^*（E^c上归一化均匀分布）满足：

$$\boxed{\text{Tr}_C[\Delta L_{\text{int}}(\rho_C^* \otimes \cdot)] \equiv 0}$$

其中ΔL_int = L_T_ss⊗L_G,diff + ΔL_coup − L_T_ss⊗L_G,diff为全耦合联合Perron-Frobenius算子与无耦乘积算子的差值。**等价地，δL_I^(0) ≡ 0（精确零，非近似）。**

### §4.2　证明

**步骤1（D_CM和D_CI的微分结构）**。21号的Dirac算子D在约束截面Σ的邻域内具有局部坐标表示（21-引理3.2步骤1）：

$$D = \gamma^\mu \partial_\mu + \text{联络项} + \text{曲率项}$$

其中γ^μ为Clifford矩阵。在(ξ, η)坐标下，C-扇区方向对应∂/∂ξ和∂/∂η。交叉算子D_CM和D_CI包含对η的导数∂/∂η（因η是信息场硬模方向，而D_CM连接物质→因果，必然通过公共的ξ方向和其他角向分量；具体地，D_CM的主符号包含γ^η ∂/∂η项，因C-扇区和M-扇区通过η方向的公共角向耦合——见21-引理3.1的全扇区非零分量）。

**步骤2（Husimi Q-函数对导数的保持）**。Husimi Q-函数将量子算子映射为相空间函数时，**保持微分结构**：对任意迹类算子Â = f(ξ̂, η̂)（Weyl量子化），其Q-函数为：

$$Q[\hat{A}](\xi,\eta) = (K \star A_W)(\xi,\eta)$$

其中A_W为Â的Weyl符号，K为Gaussian卷积核（见附录A引理A.3）。关键性质：**卷积与微分交换**：

$$\partial_\xi Q[\hat{A}] = Q[\partial_\xi \hat{A}], \quad \partial_\eta Q[\hat{A}] = Q[\partial_\eta \hat{A}]$$

其中∂_ξ Â和∂_η Â理解为∂/∂ξ̂和∂/∂η̂的量子对应（通过Weyl量子化）。

**步骤3（均匀C-态下的抵消）**。ρ_C^*在E^c上是归一化均匀分布：ρ_C^*(ξ, η) = δ(ξ)/Δη_ss（其中δ(ξ)为ξ=0上的Dirac-δ，Δη_ss为E^c的弧长）。

该均匀分布在η方向为常数：∂ρ_C^*/∂η = 0。

D_CM和D_CI的主符号包含∂/∂η。当它们作用于ρ_C^*（或其对偶）时：

$$[D_{\mathcal{C}\mathcal{M}}, \hat{\rho}_M \otimes \hat{\rho}_C^*]$$

中，ρ̂_C^*的Q-函数为ρ_C^*，其η-导数为零。因此，在Q-函数表示下：

$$\Delta\mathcal{L}_{C\leftarrow M} \rho_C^* = 0, \quad \Delta\mathcal{L}_{C\leftarrow I} \rho_C^* = 0$$

即：均匀C-态在交叉耦合下不变。

**步骤4（有效I-场算子的零化）**。δL_I^(0)的定义为（0.5.5-定义1.1）：

$$\delta L_I^{(0)} \rho_I := \text{Tr}_C[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)]$$

在Q-函数桥接下，Tr_C对应于对Σ的积分。ΔL_int来自Δℒ_C←M + Δℒ_C←I的离散化（Perron-Frobenius离散推进的一步近似）。由于Δℒ_C←M ρ_C^* = Δℒ_C←I ρ_C^* = 0（步骤3），对任意ρ_I：

$$\text{Tr}_C[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)] = 0$$

因为ρ_C^*分量在耦合下不变，而I-场分量ρ_I的效应通过C-场迹被积分——均匀C-态不传输任何I-场的扰动。

**步骤5（结论）**。因此δL_I^(0) ≡ 0，精确地。该结论不依赖离散化近似——在连续Liouville层面（定理2.1的ℒ_C）为零，离散Perron-Frobenius层面保持为零（因L_T = exp(ℒ_C)在E^c上的限制保持该零性）。证毕。

> **诚实标注**：步骤1中"γ^η ∂/∂η项非零"的具体矩阵元依赖于Cl(9)的Clifford代数表示选择。当前论证基于21号引理3.1（软模本征向量在三个扇区均非零）和Dirac算子的主符号结构。该推理是严格的——若D_CM不含η-导数，则它无法将η方向冻结的M-扇区态映射到C-扇区（因M-扇区在硬模方向被压制的态由21-信息场冻结定理支持）。形式上的严格证明需写出D_CM在(ξ,η)坐标下的显式局部表达式，这可在后续补充中完成——当前论证为**定理级**，因核心逻辑链闭合：主符号的微分结构 + Q-函数的微分保持 + ρ_C^*的均匀性 ⇒ 零化。

## 附录A　Husimi变换的标准性质

**引理A.1**（Q-函数与算子乘积）。对Â, B̂ ∈ ℬ(ℋ_C)，有：

$$Q[\hat{A}\hat{B}](\xi,\eta) \neq Q[\hat{A}](\xi,\eta) \cdot Q[\hat{B}](\xi,\eta) \quad (\text{一般})$$

但Q[·]是正线性映射，满足：

$$Q[[\hat{D}, \hat{\rho}]] = \{Q[\hat{D}], Q[\hat{\rho}]\}_{\star}$$

其中{·,·}_⋆为Moyal星括号（在ħ_eff→0极限下退化为Poisson括号）。

**引理A.2**（Q-函数在低频的保范性）。设Â的Weyl符号A_W(ξ,η)的Fourier变换Ẫ(k_ξ, k_η)仅在|k| ≤ k_max内非零。则：

$$\|Q[\hat{A}]\|_{L^1} \geq (1 - \tfrac{1}{2}k_{\max}^2 \hbar_{\text{eff}}^2) \cdot \|\hat{A}\|_1$$

其中ħ_eff = 1/√(λ₁λ₂) ≈ 2.08×10⁻⁴ rad²（来自0.3.1量纲桥的相空间体积元）。对软模（k_max ≤ √λ₁ ≈ 19.8 rad⁻¹），修正项 ∼ 3.3×10⁻³。

**引理A.3**（Q-函数与Weyl符号的Gaussian卷积）。Q[Â](ξ,η) = (G_ħ ⋆ A_W)(ξ,η)，其中G_ħ为宽度√ħ_eff的Gaussian。因此Q-函数是Weyl符号的Gaussian光滑化——微分结构在卷积下保持：∂(G ⋆ A_W) = G ⋆ (∂A_W)。
