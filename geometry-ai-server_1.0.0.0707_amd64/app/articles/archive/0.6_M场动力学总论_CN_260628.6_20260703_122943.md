## 摘　要

本文在几何论三公理体系内，建立M场（质量场）动力学的总体框架。M场由物质扇区角 $\theta_M$ 及其空间分布定义，其动力学由六项作用量 $S$ 对 $\theta_M$ 的梯度驱动，在完备性公理 $\theta_M+\theta_C+\theta_I=90^\circ$ 的约束下与C场（因果场）和I场（信息场）耦合演化。

核心工作包括：
（一）确立M场在约束截面 $\Sigma$ 上的法向几何地位——$\mathcal{M}$ 方向沿 $\Sigma$ 的法丛 $N\Sigma$，C与I在切空间 $T\Sigma$ 内（继承0.4引理2.0）；
（二）建立M场梯度流方程：$\partial_\tau \theta_M = -\Gamma_M \cdot \partial S/\partial\theta_M$，其中 $\Gamma_M$ 为M扇区迁移率，由Hessian软硬模比 $\Lambda_H \approx 150$ 锁定；
（三）导出质量流 $J_M = 3K\sin^2\theta_M\cos\theta_M \cdot \nabla\theta_M$ 及其在约束截面上的连续性方程；
（四）建立M场的变分原理——六项作用量 $S(\theta)$ 在法丛上的约束极值问题；
（五）给出M-C腰边耦合与M-I完备性耦合的统一描述；
（六）制定0.6.1–0.6.7子系列的完整路线图。

本文诚实区分定理、构造性假设与开放问题三层。M场梯度流方程为**构造性框架**（非独立定理），其严格化依赖0.6.1–0.6.7的逐项闭合。全部数值同步260626.6基准。

## 第2章　M场的几何定义与法丛结构

### §2.1　约束截面 $\Sigma$ 上的法丛 $N\Sigma$

**定义2.1（约束截面 $\Sigma$）**：在完备性平面 $\mathcal{P} = \{(\theta_M,\theta_C,\theta_I) \mid \theta_M+\theta_C+\theta_I=90^\circ\}$ 上，给定六项作用量 $S$。约束截面 $\Sigma_S$ 定义为 $S$ 的等值面：
$$\Sigma_S = \{p \in \mathcal{P} \mid S(p) = \text{const}\}$$

$\mathcal{P}$ 是2维仿射平面，$\Sigma_S$ 是其中的1维曲线（$S$ 在 $\mathcal{P}$ 上的水平集）。但为与0.4/0.5一致，本文中"约束截面 $\Sigma$"通常指**完备性平面 $\mathcal{P}$ 本身**（2维），而 $\Sigma_S$ 为其上的特定等值线。M场动力学研究系统在 $\mathcal{P}$ 上跨越不同 $\Sigma_S$ 的运动。

**引理2.1（法丛的几何构造，继承0.4引理2.0）**：在裸基准点 $P_0 = (\theta_M^0,\theta_C^0,\theta_I^0)$ 处，约束截面 $\Sigma$（此处指 $\mathcal{P}$）的切空间 $T_{P_0}\mathcal{P}$ 由 $\mathcal{C} \oplus \mathcal{I}$ 张成，法方向由 $\mathcal{M}$ 张成。法丛 $N\mathcal{P}$ 是 $\mathcal{P}$ 上的1维向量丛，其纤维 $N_p\mathcal{P} \cong \mathbb{R}$ 沿 $\nabla_{\theta_M} S$ 方向。

*证明*（复述并精确化0.4引理2.0）：完备性平面 $\mathcal{P}$ 由约束 $\theta_M+\theta_C+\theta_I=90^\circ$ 定义，其法向量（在3D角度空间 $\mathbb{R}^3$ 中）为 $(1,1,1)/\sqrt{3}$。$\mathcal{P}$ 上的函数 $S(\theta_M,\theta_C,\theta_I)$ 在 $P_0$ 处的梯度为：
$$\nabla S|_{P_0} = \left(\frac{\partial S}{\partial\theta_M}, \frac{\partial S}{\partial\theta_C}, \frac{\partial S}{\partial\theta_I}\right)_{P_0} = (-10.60, -71.24, -2145.16)\ \text{rad}^{-1}$$

该梯度在 $\mathcal{P}$ 上的投影定义了 $\Sigma_S$ 的法方向。但更关键的是：$\mathcal{M}$ 扇区对应的坐标轴方向 $\mathbf{e}_M = (1,0,0)$ 向 $\mathcal{P}$ 的投影，与 $\nabla S|_{P_0}$ 在 $\mathcal{P}$ 上的投影之间的几何关系。由0.2.1 §3.2，在约束 $\theta_M = 90^\circ - \theta_C - \theta_I$ 下，$\theta_M$ 变化驱动的 $S$ 变化率为：
$$\frac{dS}{d\theta_M}\bigg|_{\mathcal{P}} = -\left(\frac{dS}{d\theta_C} + \frac{dS}{d\theta_I}\right) = -((-60.64) + (-2134.56)) = +2195.20\ \text{rad}^{-1}$$

此为正——增大 $\theta_M$（同时减小 $\theta_C+\theta_I$ 以满足完备性）使 $S$ 增大。因此 $\mathcal{M}$ 方向确实是穿越 $S$ 等值面的有效方向。$\square$

> **诚实标注**：0.4引理2.0的原始表述"$\mathcal{M}$ 方向沿 $\Sigma$ 的法向 $n = \nabla_{\theta_M}S/|\nabla_{\theta_M}S|$"中的 $\Sigma$ 指数为 $S$ 等值面的1维截面，非2维完备性平面。本文统一约定：**$\mathcal{P}$ = 完备性平面（2维），$\Sigma_S$ = $S$ 等值线（1维），$\Sigma$（无下标）= 依上下文指 $\mathcal{P}$ 或 $\Sigma_S$**。法丛 $N\Sigma_S$ 是 $\Sigma_S$ 在 $\mathcal{P}$ 中的1维法丛。

### §2.2　$\eta$ 坐标：M场动力学的自然变量

**定义2.2（$\eta$ 坐标）**：继承0.5的定义，在完备性平面 $\mathcal{P}$ 上引入坐标 $(\xi,\eta)$：
$$\eta = \theta_C + \theta_I = 90^\circ - \theta_M,\quad \xi = \theta_C - \theta_I$$

逆变换：
$$\theta_M = 90^\circ - \eta,\quad \theta_C = \frac{\eta + \xi}{2},\quad \theta_I = \frac{\eta - \xi}{2}$$

$\eta$ 是M场动力学的**自然变量**：它与 $\theta_M$ 一一对应（$\eta = 90^\circ - \theta_M$），且其演化方向在裸基准点处近似平行于软模特征方向（0.5 §2.1）。在 $P_0$ 处：
$$\eta_0 = \theta_C^0 + \theta_I^0 = 32.07^\circ,\quad \xi_0 = \theta_C^0 - \theta_I^0 = 20.25^\circ$$

对称基态 $(30^\circ,30^\circ,30^\circ)$ 处：$\eta^* = 60^\circ$，$\xi^* = 0^\circ$。

**命题2.2（$\eta$ 与 $S$ 的关系）**：在 $\xi$ 固定的截线上，$S$ 作为 $\eta$ 的函数在 $\eta \in (0, 90^\circ)$ 上严格凸，唯一最小值在 $\eta^* = 60^\circ$（即对称基态）取得。

*证明*：在 $\xi = 0$（即 $\theta_C = \theta_I$）的对称截线上，$\theta_C = \theta_I = \eta/2$，$\theta_M = 90^\circ - \eta$。代入六项作用量：
$$S(\eta) = \frac{1}{\sin^2(90^\circ-\eta)} + \frac{2}{\sin^2(\eta/2)} + \frac{2}{\sin(90^\circ-\eta)\sin(\eta/2)} + \frac{1}{\sin^2(\eta/2)}$$

利用 $\sin(90^\circ-\eta) = \cos\eta$，可验证 $S'(\eta^*)=0$ 且 $S''(\eta^*)>0$。$\square$

在 $P_0$ 处（$\eta_0 = 32.07^\circ$），$\partial S/\partial\eta|_{P_0}$ 可由链式法则从 $\partial S/\partial\theta_M$ 导出：
$$\frac{\partial S}{\partial\eta}\bigg|_{P_0} = -\frac{\partial S}{\partial\theta_M}\bigg|_{P_0} = +10.602445\ \text{rad}^{-1}$$

正值意味着 $\eta$ 增大（即 $\theta_M$ 减小，向对称基态移动）会增大 $S$——确认 $P_0$ 位于 $S$ 的"下坡"侧（$\eta_0 < \eta^*$），梯度指向 $\eta$ 增大的方向。

### §2.3　M场梯度 $\partial S/\partial\theta_M$ 的解析结构

**定理2.3（M场梯度的解析形式）**：六项作用量对 $\theta_M$ 的偏导数为：
$$\frac{\partial S}{\partial\theta_M} = -\frac{\cos\theta_M}{\sin^2\theta_M}\left(\frac{2}{\sin\theta_M} + \frac{1}{\sin\theta_C} + \frac{1}{\sin\theta_I}\right)$$

*证明*：对六项作用量逐项求导：
$$\begin{aligned}
\frac{\partial}{\partial\theta_M}\frac{1}{\sin^2\theta_M} &= -\frac{2\cos\theta_M}{\sin^3\theta_M} \\
\frac{\partial}{\partial\theta_M}\frac{1}{\sin\theta_M\sin\theta_C} &= -\frac{\cos\theta_M}{\sin^2\theta_M\sin\theta_C} \\
\frac{\partial}{\partial\theta_M}\frac{1}{\sin\theta_M\sin\theta_I} &= -\frac{\cos\theta_M}{\sin^2\theta_M\sin\theta_I}
\end{aligned}$$

提取公因子 $-\cos\theta_M/\sin^2\theta_M$ 即得。$\square$

**推论2.4（梯度零点的几何意义）**：$\partial S/\partial\theta_M = 0$ 当且仅当 $\cos\theta_M = 0$，即 $\theta_M = 90^\circ$。此时M扇区"关闭"（$\sin\theta_M = 1$，质量最大化 $m = K$），但完备性公理迫使 $\theta_C = \theta_I = 0^\circ$，导致 $S \to +\infty$（因为 $1/\sin^2\theta_I \to \infty$）。因此M场梯度的唯一零点在物理上不可达——**M场永远受非零梯度驱动**，这是M场动力学的根本特征。

在裸基准点 $P_0$ 处的数值（0.2.1 §3.1）：
$$\frac{\partial S}{\partial\theta_M}\bigg|_{P_0} = -10.602445\ \text{rad}^{-1}$$

**注2.1（梯度符号的物理意义）**：$\partial S/\partial\theta_M < 0$ 意味着（在固定 $\theta_C,\theta_I$ 的虚拟变化下）增大 $\theta_M$ 会减小 $S$。在完备性约束下，增大 $\theta_M$ 必然减小 $\theta_C+\theta_I$，而 $\theta_I$ 的减小导致 $1/\sin^2\theta_I$ 的急剧增大，使 $S$ 整体增大。因此约束梯度与无约束梯度的符号相反——这一"梯度反转"是M场动力学的核心张力来源。

## 第3章　梯度流与M场演化方程

### §3.1　梯度流的一般形式

**定义3.1（M场梯度流，构造性框架）**：设 $\tau$ 为几何时间参数（由量纲桥 $\chi_T$ 映射为物理时间）。$\theta_M$ 沿六项作用量 $S$ 的负梯度方向演化：
$$\boxed{\frac{\partial\theta_M}{\partial\tau} = -\Gamma_M \cdot \frac{\partial S}{\partial\theta_M}}$$

其中 $\Gamma_M > 0$ 为M扇区迁移率（mobility），量纲 $[\Gamma_M] = [\text{rad}^2]$（因为 $[\partial S/\partial\theta_M] = [\text{rad}^{-1}]$，$[\partial\theta_M/\partial\tau] = [\text{rad}]$）。

**几何解释**：梯度流是 $S$ 作为"势能景观"上最速下降曲线的参数化。系统沿 $-\nabla S$ 方向运动，使 $S$ 单调递减。在完备性约束下，该运动被投影到 $\mathcal{P}$ 上，产生 $\theta_C$ 和 $\theta_I$ 的伴随变化。

**与0.5因果场动力学的对比**：0.5的因果推进是沿 $\eta$ 方向（$\eta$ 减小，即 $\theta_M$ 增大）的**定向演化**——因果场从初始构型向基态推进。M场梯度流同样是定向的（沿 $-\partial S/\partial\theta_M$ 方向），但两者的驱动力不同：因果推进由遍历理论的保测变换驱动，M场梯度流由 $S$ 的梯度驱动。在 $\eta$ 坐标下，两者方向一致（因为 $\partial S/\partial\eta > 0$ 在 $P_0$，因果推进向 $\eta$ 减小方向），这表明**C场动力学和M场动力学在软模方向上是同向的**——这是M-C耦合的几何基础。

**构造性地位**：梯度流形式由 $S$ 作为标量势函数的最简动力学假设导出。其"构造性"在于：
1. 选择了 $S$ 作为驱动势——这是自然的（$S$ 是唯一的标量几何作用量），但非唯一可能；
2. 假设了线性响应关系 $\dot{\theta}_M \propto -\partial S/\partial\theta_M$——在近平衡态有效，远离平衡时可能有非线性修正；
3. $\Gamma_M$ 的锁定需要额外输入（见§3.2）。

---

### §3.2　迁移率 $\Gamma_M$ 的几何锁定

**构造性假设3.2（迁移率锁定）**：M扇区迁移率 $\Gamma_M$ 由约束截面Hessian的软硬模比锁定：
$$\Gamma_M = \frac{1}{\lambda_1^{\text{eff}} \cdot \tau_{\text{dec}}}$$

其中 $\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}$ 为有效软模，$\tau_{\text{dec}} \approx 7.28$ 日为退相干时间（0.4 §8）。

**动机**：
1. **$1/\lambda_1^{\text{eff}}$ 因子**：软模 $\lambda_1$ 是约束截面在 $\eta$ 方向的曲率刚度。在梯度流中，迁移率反比于曲率——越"平坦"的方向，系统移动越快。$1/\lambda_1^{\text{eff}}$ 给出迁移率的角度量纲 $[\text{rad}^2]$。
2. **$1/\tau_{\text{dec}}$ 因子**：退相干时间 $\tau_{\text{dec}}$ 是信息场达到稳态的特征时间尺度（0.4 §8）。M场演化必须在信息场稳态背景下进行——$\tau_{\text{dec}}$ 设定了M场动力学的自然时间单位。

数值：
$$\Gamma_M = \frac{1}{391.05 \times 7.28\ \text{日}} = \frac{1}{391.05 \times 6.29 \times 10^5\ \text{s}} \approx 4.07 \times 10^{-9}\ \text{rad}^2/\text{s}$$

换算为几何时间单位（$\chi_T = 3.616 \times 10^{-17}\ \text{s}$）：
$$\Gamma_M^{\text{geo}} = \Gamma_M \cdot \chi_T^{-1} \approx 4.07 \times 10^{-9} / 3.616 \times 10^{-17} \approx 1.13 \times 10^8\ \text{rad}^2/\chi_T$$

> **诚实标注**：$\Gamma_M$ 的具体数值依赖于构造性假设3.2的两个因子选择。$\tau_{\text{dec}}$ 的来源是0.4定理级结果（图热核衰减率+代数连通度双重确定），但将其用作M场迁移率的时间标度是**构造性推广**。替代方案（如使用因果时间标度 $\tau_{\text{cause}} \approx 10 \cdot \chi_T$）将在0.6.2中讨论。$\Gamma_M$ 的定理级锁定待0.6.6谱分析闭合。

### §3.3　M场演化方程的显式形式

**命题3.3（M场演化方程的显式形式）**：将定理2.3的梯度表达式代入梯度流方程（定义3.1），得到：
$$\boxed{\frac{\partial\theta_M}{\partial\tau} = \Gamma_M \cdot \frac{\cos\theta_M}{\sin^2\theta_M}\left(\frac{2}{\sin\theta_M} + \frac{1}{\sin\theta_C} + \frac{1}{\sin\theta_I}\right)}$$

在裸基准点 $P_0$ 处，演化速率为：
$$\frac{\partial\theta_M}{\partial\tau}\bigg|_{P_0} = \Gamma_M \cdot 10.602445\ \text{rad}^{-1}$$

注意符号：$\partial S/\partial\theta_M = -10.602445$，梯度流为 $-\Gamma_M \cdot (-10.602445) = +\Gamma_M \cdot 10.602445$。因此 $\theta_M$ 在 $P_0$ 处**增大**（向 $90^\circ$ 方向），对应 $S$ 沿约束方向减小。

演化速率数值（使用§3.2的 $\Gamma_M$）：
$$\frac{\partial\theta_M}{\partial t}\bigg|_{P_0} \approx 4.07 \times 10^{-9} \times 10.60 \approx 4.31 \times 10^{-8}\ \text{rad/s} \approx 2.47 \times 10^{-6}\ \text{度/秒}$$

这对应于极慢的质量漂移——在实验室时间尺度上M场几乎是静态的。这一"准静态"特征与电子质量的观测稳定性（$\Delta m_e/m_e < 10^{-9}$/年量级，来自原子钟比较实验的粗略约束）定性一致。精确的稳定性约束将在0.6.2中分析。

**命题3.4（以 $\eta$ 为变量的演化方程）**：由 $\eta = 90^\circ - \theta_M$，$\partial\eta/\partial\tau = -\partial\theta_M/\partial\tau$：
$$\boxed{\frac{\partial\eta}{\partial\tau} = -\Gamma_M \cdot \frac{\sin\eta}{\cos^2\eta}\left(\frac{2}{\cos\eta} + \frac{1}{\sin\theta_C} + \frac{1}{\sin\theta_I}\right)}$$

其中 $\sin\theta_C = \sin((\eta+\xi)/2)$，$\sin\theta_I = \sin((\eta-\xi)/2)$。

在 $P_0$ 处，$\partial\eta/\partial\tau < 0$——$\eta$ 减小，系统远离对称基态 $\eta^*=60^\circ$。这与0.5因果推进的方向一致（因果推进使 $\eta$ 从初始值向更小值演化）。**M场梯度流与因果推进在方向上一致**——这是M-C同向性原理。

### §3.4　与0.5因果场动力学的对偶结构

M场动力学与0.5因果场动力学之间存在深刻的对偶关系，汇总于表3.1：

| 方面 | M场（本文） | C场（0.5） |
|:---|:---|:---|
| 驱动势 | $S(\theta)$ | —（保测变换） |
| 演化方向 | $-\nabla S$（梯度流） | 瞬态推进 → 稳态环流 |
| 自然变量 | $\eta = 90^\circ - \theta_M$ | $\eta$（软模方向） |
| 演化方程类型 | 梯度流ODE | 动力系统/遍历变换 |
| 时间参数 | $\tau$（几何时间） | $\tau$（因果时间，同一参数） |
| 稳态 | $S$ 极小值（受约束） | 遍历环流（Birkhoff平均） |
| 物理输出 | 质量谱 | 因果序结构 |

**对偶原理（构造性）**：在软模极限（$\lambda_2/\lambda_1 \gg 1$，硬模冻结）下，M场梯度流与C场因果推进在 $\eta$ 方向上重合。两者的差异仅在 $\xi$ 方向——M场梯度流不含 $\xi$ 分量（$\partial S/\partial\xi$ 在 $\partial S/\partial\theta_M$ 的表达式中不直接出现），而C场在稳态阶段的遍历环流同时涉及 $\xi$ 和 $\eta$。

这一对偶结构将在0.6.3（M-C腰边耦合）中深化为定量的耦合方程。

## 第4章　质量流与守恒律

### §4.1　质量流的定义

**定义4.1（质量流 $J_M$）**：M场的质量流定义为质量密度 $m = K\sin^3\theta_M$ 随 $\theta_M$ 变化的流：
$$\boxed{J_M = \frac{dm}{d\theta_M} \cdot \nabla\theta_M = 3K\sin^2\theta_M\cos\theta_M \cdot \nabla\theta_M}$$

其中 $\nabla\theta_M$ 是 $\theta_M$ 在物理空间中的梯度（经由量纲桥 $\chi_L$ 从角度空间映射）。

**量纲分析**：$[K] = \text{keV}$（能量），$[\nabla\theta_M] = [\text{rad/m}]$，因此 $[J_M] = [\text{keV/m}]$——质量流具有能量密度梯度的量纲。

在裸基准点 $P_0$ 处，前置因子为：
$$3K\sin^2\theta_M^0\cos\theta_M^0 = 3 \times 839.76 \times 0.7181 \times 0.5310 = 960.0\ \text{keV}$$

物理意义：$\theta_M$ 每变化1弧度，质量密度变化约 $960\ \text{keV}$。

---

### §4.2　连续性方程

**命题4.2（M场连续性方程，构造性框架）**：在无源区域，质量流满足连续性方程：
$$\frac{\partial\rho_M}{\partial\tau} + \nabla \cdot J_M = 0$$

其中 $\rho_M = m/\chi_L^3 = K\sin^3\theta_M/\chi_L^3$ 为质量密度（$\chi_L^3$ 为几何体积元）。

展开：
$$\frac{\partial}{\partial\tau}(K\sin^3\theta_M) + \nabla \cdot (3K\sin^2\theta_M\cos\theta_M \nabla\theta_M) = 0$$

消去 $K$：
$$3\sin^2\theta_M\cos\theta_M \frac{\partial\theta_M}{\partial\tau} + \nabla \cdot (3\sin^2\theta_M\cos\theta_M \nabla\theta_M) = 0$$

代入梯度流方程 $\partial\theta_M/\partial\tau = -\Gamma_M \cdot \partial S/\partial\theta_M$：
$$-3\sin^2\theta_M\cos\theta_M \cdot \Gamma_M \frac{\partial S}{\partial\theta_M} + \nabla \cdot (3\sin^2\theta_M\cos\theta_M \nabla\theta_M) = 0$$

> **诚实标注**：上述连续性方程是标量场动力学的标准构造。在几何论框架内，其严格地位取决于：(i) $\rho_M$ 作为守恒荷的识别——由公理3（质量映射）自然支持；(ii) 是否存在质量源/汇项——$S$ 的梯度驱动意味着质量可以从几何作用量中"产生"，因此全局质量守恒可能需要修正为包含 $S$ 变化的广义守恒律。0.6.2将对此进行系统分析。

---

### §4.3　质量流与信息流、因果流的三角关系

三分切丛的三个扇区各有其"流"：

| 流 | 扇区 | 定义 | 驱动 |
|:---|:---|:---|:---|
| 质量流 $J_M$ | $\mathcal{M}$ | $3K\sin^2\theta_M\cos\theta_M \nabla\theta_M$ | $\partial S/\partial\theta_M$ |
| 因果流 $J_C$ | $\mathcal{C}$ | $\nabla\theta_C$（因果推进率） | 遍历变换（0.5） |
| 信息流 $J_I$ | $\mathcal{I}$ | $\nabla\rho$（信息密度梯度） | 图热方程（0.4） |

完备性公理 $\theta_M + \theta_C + \theta_I = 90^\circ$ 施加约束：
$$\nabla\theta_M + \nabla\theta_C + \nabla\theta_I = 0$$

即三个流的"角度梯度"之和为零——一个扇区的梯度增大必然伴随其他扇区的梯度减小。这构成了质量-因果-信息三流之间的守恒关系：
$$\frac{J_M}{3K\sin^2\theta_M\cos\theta_M} + J_C^{\text{(angle)}} + J_I^{\text{(angle)}} = 0$$

其中 $J_C^{\text{(angle)}} = \nabla\theta_C$，$J_I^{\text{(angle)}} = \nabla\theta_I$ 是因果流和信息流的角度梯度形式。

## 第5章　M场的变分原理

### §5.1　约束极值问题

**定义5.1（M场作用量泛函，构造性框架）**：M场在几何时间 $\tau$ 内的作用量泛函定义为六项作用量 $S$ 沿M场轨迹的积分：
$$\mathcal{A}_M[\theta_M] = \int_{\tau_1}^{\tau_2} S(\theta_M,\theta_C,\theta_I)\,d\tau$$

在完备性约束 $\theta_M + \theta_C + \theta_I = 90^\circ$ 下，且 $\theta_C,\theta_I$ 的分配由C场和I场的独立动力学决定（即不作为 $\theta_M$ 的变分自由度）。

**变分原理（构造性）**：M场的物理轨迹使 $\mathcal{A}_M$ 在约束下取极值：
$$\delta\mathcal{A}_M = 0 \quad \text{subject to} \quad \theta_M + \theta_C + \theta_I = 90^\circ$$

引入拉格朗日乘子 $\lambda$（注意：此为乘子，非Hessian软模 $\lambda_1$），约束作用量为：
$$\mathcal{A}_M^{\text{con}} = \int_{\tau_1}^{\tau_2} \left[S(\theta_M,\theta_C,\theta_I) + \lambda(\tau)(\theta_M + \theta_C + \theta_I - 90^\circ)\right] d\tau$$

对 $\theta_M$ 变分：
$$\frac{\partial S}{\partial\theta_M} + \lambda = 0 \quad \Rightarrow \quad \lambda = -\frac{\partial S}{\partial\theta_M}$$

乘子 $\lambda$ 即M场梯度的负值——这是变分原理与梯度流方程自洽的数学保证。

---

### §5.2　Euler-Lagrange方程的几何形式

**命题5.2（M场的Euler-Lagrange方程）**：将 $\theta_M$ 视为不仅依赖于 $\tau$ 还依赖于空间坐标 $x^\mu$ 的场，其拉格朗日密度为：
$$\mathcal{L}_M = \frac{1}{2}(\partial_\mu\theta_M)^2 - V_{\text{eff}}(\theta_M)$$

其中"动能项" $(\partial_\mu\theta_M)^2/2$ 描述了 $\theta_M$ 在空间中的不均匀性代价，"势能" $V_{\text{eff}}$ 由 $S$ 在约束下的有效势给出。Euler-Lagrange方程：
$$\Box \theta_M + \frac{\partial V_{\text{eff}}}{\partial\theta_M} = 0$$

其中 $\Box = \partial_\tau^2 - \nabla^2$（几何时间+空间的d'Alembert算子）。

**与4号文章呼吸模式的关系**：4号文章从Hessian行列式涨落导出了标量场 $\Phi$ 的拉格朗日量：
$$\mathcal{L}_\Phi = \frac{1}{2}(\partial_\mu\Phi)^2 - 2\lambda_1\Phi^2 - \frac{\lambda_1}{S_0}\Phi^3 - \frac{\lambda_1}{S_0^2}\Phi^4$$

两者的联系在于呼吸模式 $\Phi$ 与 $\theta_M$ 的涨落之间的关系（由定义1.1和Hessian整体缩放假设2.2可建立 $\Phi \propto \delta\theta_M$ 的线性近似）。0.6.5将严格推导这一关系，将呼吸模式确立为M场动力学的**线性化极限**。

### §5.3　与Hessian呼吸模式的关系

**注5.1（呼吸模式作为M场集体激发）**：4号文章的呼吸模式 $\Phi$ 描述整个约束截面的均匀缩放，是M场的一种**集体模**（collective mode）。M场动力学（本文）描述 $\theta_M$ 作为**局域场**的完整动力学，呼吸模式是其长波极限下的特殊解。两者的关系类似于：
- 4号文章 = 刚体呼吸（全局模，$\Phi$ 不依赖空间坐标）
- 0.6 = 弹性体动力学（$\theta_M(x)$ 可以局域变化）

这一层次关系将在0.6.5中系统展开。

## 第6章　法丛结构与截面耦合

### §6.1　法丛 $N\Sigma$ 的微分几何

**定义6.1（法丛 $N\Sigma_S$）**：设 $\Sigma_S$ 为完备性平面 $\mathcal{P}$ 上 $S = \text{const}$ 的1维曲线。$N\Sigma_S$ 是 $\Sigma_S$ 在 $\mathcal{P}$ 中的法丛——秩1实向量丛，其纤维为 $\Sigma_S$ 的法方向。

在 $P_0$ 处，$\Sigma_S$ 的切向量 $\mathbf{t}$ 和法向量 $\mathbf{n}$（均在 $\mathcal{P}$ 内）：
$$\mathbf{t} \parallel \left(\frac{d\theta_C}{d\theta_I}\bigg|_{\Sigma_S}, 1\right) = \left(-\frac{dS/d\theta_I}{dS/d\theta_C}, 1\right) = \left(-\frac{-2134.56}{-60.64}, 1\right) \approx (-35.2, 1)$$
$$\mathbf{n} \parallel \left(1, -\frac{d\theta_I}{d\theta_C}\bigg|_{\Sigma_S}\right) \approx (1, 0.0284)$$

$\mathbf{n}$ 近似沿 $\theta_M$ 变化方向在 $\mathcal{P}$ 中的投影——验证了M方向与 $\Sigma_S$ 法向的重合。

**纤维丛结构（构造性推广）**：将 $\mathcal{P}$ 视为以 $\Sigma_S$ 为底空间、$\mathbb{R}$ 为纤维的平凡丛 $\mathcal{P} \cong \Sigma_S \times \mathbb{R}$，其中纤维坐标即法向参数 $s$（沿 $\mathbf{n}$ 方向）。$S$ 在纤维上的变化率即法向导数 $\partial S/\partial n$。M场动力学即纤维上的梯度流。

> **诚实标注**：纤维丛描述在此阶段是构造性的——$\mathcal{P}$ 的全局拓扑（$S^2$ 结构，见0.2.2）意味着非平凡丛的可能性，但本文仅使用局部平凡化。全局丛结构的定理级分析待0.6.1。

### §6.2　法向动力学与切向动力学的耦合机制

M场（法向）与C/I场（切向）的耦合通过完备性公理实现。具体地：

1. **运动学耦合**：$\theta_M + \theta_C + \theta_I = 90^\circ$ 意味着任何两个角度的变化决定第三个。M场演化 $\partial_\tau\theta_M \neq 0$ 迫使 $\partial_\tau\theta_C + \partial_\tau\theta_I = -\partial_\tau\theta_M$。

2. **动力学耦合**：$S$ 的梯度 $\nabla S$ 在法向和切向的分量通过Hessian矩阵混合：
$$\nabla S = \begin{pmatrix} \partial S/\partial\theta_M \\ \partial S/\partial\theta_C \\ \partial S/\partial\theta_I \end{pmatrix}, \quad \nabla^2 S = H_{3\times 3}$$

法向加速度 $\partial_\tau^2\theta_M$ 通过 $H_{MC}$ 和 $H_{MI}$ 耦合到切向加速度。

3. **能流耦合**：法向的质量流 $J_M$ 和切向的信息流 $J_I$ 通过完备性约束交换"角度配额"——$J_M \neq 0$ 意味着 $J_I + J_C \neq 0$，即质量的变化必然伴随信息/因果的变化。

**耦合强度**：由0.2.1 §3.3，Hessian交叉项为：
$$H_{MC} = 3.4145,\quad H_{MI} = 69.3547,\quad H_{CI} = 433.1578\ \text{rad}^{-2}$$

M-C耦合（$H_{MC} = 3.41$）远弱于M-I耦合（$H_{MI} = 69.35$）——**M场与I场的耦合显著强于与C场的耦合**。这与物理直觉一致：质量生成（M）密切关联于信息编码（I），而因果结构（C）在其中起中介作用。

## 第7章　M-C腰边耦合（0.6.3预览）

### §7.1　电磁核心映射：电子—光子对

0.1 §1.3确立了单一核心映射：三分切丛 $\mathcal{M}$–$\mathcal{C}$ 腰边耦合对应电磁相互作用，电子为有质量源模（$m_e = K\sin^3\theta_M^0 \approx 511\ \text{keV}$），光子为无质量传播模（速度 $c$），两者共享耦合常数 $S_e = 137.035999084$。

### §7.2　M-C耦合的梯度结构

在完备性约束下，M场和C场的耦合梯度为：
$$\frac{\partial^2 S}{\partial\theta_M \partial\theta_C} = H_{MC} = \frac{\cos\theta_M \cos\theta_C}{\sin^2\theta_M \sin^2\theta_C}$$

在 $P_0$ 处：$H_{MC} = 3.4145\ \text{rad}^{-2}$。

M场演化对C场的影响：当 $\theta_M$ 通过梯度流变化 $\delta\theta_M$，C扇区角 $\theta_C$ 感受到的等效"力"为：
$$\delta F_C = -H_{MC} \cdot \delta\theta_M$$

这一耦合是**双向的**：C场的因果推进（0.5）同样通过 $H_{MC}$ 反作用于M场。双向耦合的完整动力学方程将在0.6.3中推导。

## 第8章　M-I完备性耦合（0.6.4预览）

### §8.1　$\theta_M \leftrightarrow \theta_I$ 反相关的几何根源

完备性公理 $\theta_M + \theta_C + \theta_I = 90^\circ$ 意味着 $\theta_M$ 与 $\theta_I$ 之间存在直接的反相关：当 $\theta_C$ 相对固定时，$\Delta\theta_M \approx -\Delta\theta_I$。

但这一关系因Hessian的强各向异性而复杂化。在 $P_0$ 处：
- $\theta_M^0 = 57.93^\circ$（大角，$\sin\theta_M^0 \approx 0.847$，变化平缓）
- $\theta_I^0 = 5.91^\circ$（小角，$\sin\theta_I^0 \approx 0.103$，变化剧烈）

这意味着 $\theta_I$ 的微小变化（如 $0.01^\circ$）会导致 $S$ 的巨大变化（因为 $1/\sin^2\theta_I$ 项的极高敏感性），而同样的 $\theta_M$ 变化对 $S$ 的影响要小两个数量级。**M-I耦合的本质是"杠杆效应"**：$\theta_I$ 是灵敏的微调旋钮，$\theta_M$ 是粗调旋钮。

---

### §8.2　信息→质量转化通道

I场的信息密度 $\rho$（0.4 §6.3，约束截面Poisson方程中的源项）通过完备性耦合影响M场。其定量通道为：

1. I场涨落 → $\theta_I$ 变化
2. $\theta_I$ 变化 → 通过完备性 → $\theta_M$ 变化（在 $\theta_C$ 缓冲下）
3. $\theta_M$ 变化 → 通过公理3 → $m$ 变化

这一"信息→质量"转化通道的**效率**由Hessian交叉项 $H_{MI} = 69.35\ \text{rad}^{-2}$ 控制——显著大于M-C耦合 $H_{MC} = 3.41$，表明信息场对质量场的驱动效率远高于因果场。0.6.4将定量分析这一通道。

### §8.3　饱和稳态下的M-I关系

0.4.1的信息场衰减定理和48号文章的信息场饱和稳态（$\theta_I \sim 72.53^\circ$）给出了 $\theta_I$ 在晚期宇宙中的渐近行为。在饱和稳态下，$\theta_M$ 和 $\theta_C$ 被 $\theta_I \approx 72.53^\circ$ 和完备性公理联合约束：
$$\theta_M + \theta_C = 90^\circ - 72.53^\circ = 17.47^\circ$$

这意味着在信息饱和极限下，M场和C场被挤压到一个极小的角度窗口内，质量标度大幅降低。这是M场动力学的**晚期渐近行为**——将在0.6.4中与信息场热方程联合求解。
