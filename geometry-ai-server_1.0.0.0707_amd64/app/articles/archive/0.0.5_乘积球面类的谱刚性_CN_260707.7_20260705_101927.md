# 乘积球面类的谱刚性

**——九维约束乘积流形上的 Laplace–Beltrami 谱唯一性定理**

**编号：** 0.0.5  
**版本：** 260705.2

## 摘　要

本文研究一类由三个互锁常数 $k_0>1$（二分紧致性常数，$S_3$ 的极大正规子群指数）、$\Lambda>8/3$（三等分比例参数，$S_3$ 的共轭类数）与 $\ell_0>0$（标度常数）锁定的九维约束乘积球面 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$。证明在该类流形上，前三个非零 Laplace–Beltrami 特征值即构成模空间的完备坐标；若两个此类流形具有相同的前三个非零 Laplace 特征值，则它们必然等距。此外，建立 Dirac 指标定理（$\hat{A}(M(a))=1$）、Laplace 低能本征模式分类、谱作用量中 Yang–Mills 项与 Hessian 软硬模的代数关系、法丛曲率-Hessian 锁定定理，以及热核系数与谱三元组重建的代数锁定。最后给出谱刚性的普适维数必要条件，解释一般情形下谱刚性失败而约束子类中成立的方法论根源。

**关键词：** 谱刚性；Laplace–Beltrami 算子；Dirac 指标；Atiyah-Singer 指标定理；乘积球面；模空间；维数必要条件；等谱非等距；法丛曲率；谱作用量；热核展开

## 1　引言：从两个特征值到三个特征值

1966 年，Kac 提出著名的"听音辨形"问题：能否通过 Laplace 算子的谱来唯一确定流形的等距类？1992 年，Gordon、Webb 与 Wolpert 构造了等谱非等距的平面区域反例，证明在一般流形类中谱刚性不成立。

此前若仅使用"前两个非零 Laplace 特征值"，在乘积球面模型 $S^3(a)\times S^3(b)\times S^3(c)$ 中，前两个特征值 $3/a^2$ 与 $3/b^2$ 只能确定两个半径之比，无法唯一确定第三个半径 $c$；即便假设 $c$ 由附加比例锁定，前两个特征值仍不足以提供内部自洽性检验。本文的修正：将"前两个特征值"提升为"前三个特征值"。本文所有构造均基于 0.0.0《零与谱》（版本 260705.2）确立的 $S_3$ 谱权重 $6:2:1$ 与三分切丛结构；半径比的具体实现采用本文构造假设 3.0（尺度比实现规则），而角度约束 $\theta_1+\theta_2+\theta_3=90^\circ$ 则来自 0.0.3–0.0.7 公理体系中的公理 3。

在乘积球面模型中，前三个非零特征值对应模式 $(1,0,0)$、$(2,0,0)$、$(0,1,0)$，其显式表达式为

$$\lambda_1^\Delta = \frac{3}{a^2}, \quad \lambda_2^\Delta = \frac{8}{a^2}, \quad \lambda_3^\Delta = \frac{3}{b^2} = \frac{3\Lambda}{a^2}$$

其中 $b^2/a^2 = 1/\Lambda$。由 $\lambda_1^\Delta$ 与 $\lambda_2^\Delta$ 可独立解出 $a$（且比值 $\lambda_2^\Delta/\lambda_1^\Delta = 8/3$ 作为内部自洽性检验），再由 $\lambda_3^\Delta$ 解出 $b$；而 $c$ 由比例锁定与固定常数 $k_0$ 确定。因此，前三个特征值完全确定所有尺度参数。

本文采取的策略是：在 0.0.0《零与谱》确立的 $S_3$ 谱权重 $6:2:1$ 基础上，引入几何实现规则将权重转化为尺度比，构造一个高度约束的流形子类，使其模空间由单一尺度因子参数化，从而谱刚性自动成立。这一策略的成功关键在于：通过 $S_3$ 三扇区结构、比例锁定与切丛平凡，将无穷维的"一般流形类"压缩为一维的约束子类，从而满足了谱刚性的维数必要条件。

## 2　互锁函数的群论来源

**引理 2.0（三分切丛置换群的刚性）** 设 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 为三分切丛，$G$ 为其扇区置换对称群。若 $G$ 在三个扇区上的作用忠实且可迁，并包含任意两扇区对换，则 $G \cong S_3$，且 $S_3$ 是满足此条件的唯一有限群。

*证明.* 三个对象的忠实置换群必为 $S_3$ 的子群。可迁性要求轨道大小为 3，故 $|G| \geq 3$。$S_3$ 的可迁子群仅有 $A_3 \cong \mathbb{Z}_3$（循环）与 $S_3$ 本身。若要求包含对换（奇置换），则 $A_3$ 不足，唯一可能为 $S_3$。 $\square$

**注 2.0'** 此引理与 0.0.0《零与谱》**命题 1.1（最简非交换对称群）** 等价，后者从"零维源点"的本体论角度论证了 $S_3$ 的最小性。0.0.0 进一步将这一对称性作为推导的**出发点**：零维源点 $\mathcal Z$ 携带对称群 $S_3$，并由此严格导出谱权重 $6:2:1$（0.0.0 命题 4.1）。

**定理 2.0'（三分切丛扇区置换的刚性）** 设 $M = S^3 \times S^3 \times S^3$ 配备乘积度量，且存在切丛分解 $TM = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 使得三个扇区在度规层面完全等价（即任意置换诱导等距自同构）。则：
1. 该分解的扇区置换群 $G$ 必包含所有两扇区对换；
2. 满足 (1) 的有限群唯一同构于 $S_3$。

进而 $\Lambda(G)=|\text{Conj}(S_3)|=3$，$k_0(G)=[S_3:A_3]=2$ 是群论函数的必然输出。

*证明.* (1) 若三个扇区度规等价，则任意对换（如 $\mathcal{M}\leftrightarrow\mathcal{C}$）保持乘积度量不变，故诱导等距自同构。因此所有两扇区对换均属于 $G$。

(2) 由引理 2.0，含对换的可迁置换群只能是 $S_3$。因此 $G\cong S_3$ 是三分等价结构下非平凡置换的必然推论，与 0.0.0 出发点自洽。$\square$

**定义 2.0（互锁函数与紧致函数）** 设 $G$ 为有限群，定义：
- **互锁函数** $\Lambda(G) := |\text{Conj}(G)|$（共轭类数，由表示论基本定理等于 $G$ 的不可约复表示数）；
- **紧致函数** $k_0(G) := [G : N_{\max}]$（极大正规子群的指数）。

**命题 2.0** 对于三分切丛 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 的置换对称群 $G = S_3$：
$$\Lambda(S_3) = 3, \quad k_0(S_3) = 2.$$

*证明。* $S_3$ 的共轭类分解为：恒等元 $\{e\}$（1 个）、对换 $\{(12),(13),(23)\}$（3 个）、3-循环 $\{(123),(132)\}$（2 个），共 3 个共轭类，故 $\Lambda(S_3)=|\text{Conj}(S_3)|=3$。

$S_3$ 的正规子群只有 $\{e\}$ 与 $A_3 \cong \mathbb{Z}_3$（交错子群）。极大正规子群为 $A_3$，其指数 $[S_3 : A_3] = 6/3 = 2$，故 $k_0(S_3)=2$。$\square$

**注 2.0** 由引理 2.0，$S_3$ 是三分切丛置换对称群的唯一选择。互锁常数 $\Lambda=3$ 与 $k_0=2$ 不是人为设定的自由参数，而是群论函数在 $S_3$ 上的必然取值。若三分切丛具有 $S_4$ 置换对称性（四扇区结构），则 $\Lambda(S_4)=5$，$k_0(S_4)=2$，那将是一个与十方几何论完全不同的理论。十方几何论之所以锁定 $\Lambda=3$、$k_0=2$，根源在于其群结构为 $S_3$（三扇区），而引理 2.0 已证明该群选择的唯一性。


## 3　定义与假设

### 3.1　约束乘积球面的定义

设 $a > 0$ 为整体尺度因子。由命题 2.0，三分切丛置换群 $S_3$ 的群论函数输出为：
$$\Lambda = \Lambda(S_3) = 3 \quad \text{（三等分比例参数）}, \quad k_0 = k_0(S_3) = 2 \quad \text{（二分紧致性常数）}.$$
这些是 $S_3$ 群论结构的必然取值。

**构造假设 3.0（尺度比实现规则）**　0.0.0《零与谱》从 $S_3$ 的正规子群链严格导出谱权重 $6:2:1$（0.0.0 命题 4.1）。本文采用如下**几何实现规则**：若某一层几何由剩余对称性 $H$ 控制，则其特征长度平方与 $|H|^{-1}$ 成正比（等价于与谱权重 $w_H=[S_3:H]$ 成正比，整体归一化后）。以 $S_3$ 正规子群链 $\{e\}\lhd A_3\lhd S_3$ 的三层阶数倒数比为
$$\frac{1}{|\{e\}|}:\frac{1}{|A_3|}:\frac{1}{|S_3|}=1:\frac13:\frac16=6:2:1.$$
取倒数平方根并归一化，得三个球面因子的半径比
$$R_1 : R_2 : R_3 = 1 : \frac{1}{\sqrt{3}} : \frac{1}{\sqrt{6}}.$$
后文所有涉及半径比的结论，均理解为“在构造假设 3.0 下成立”。

记 $\ell_0 > 0$ 为标度常数（由谱单位选择定理与基态条件 $S_{\min}=24$ 锁定，详见 0.0.6）。

**定义 3.1（约束乘积球面）**　称九维闭流形（紧致、无边界）

$$M(a) = S^3(a) \times S^3(a/\sqrt{3}) \times S^3(a/\sqrt{6})$$

在配备乘积度量 $g = g_a \oplus g_{a/\sqrt{3}} \oplus g_{a/\sqrt{6}}$ 时，构成一个**约束乘积球面**（Constrained Product Sphere, CPS）。记

$$b = \frac{a}{\sqrt{3}}, \quad c = \frac{a}{\sqrt{6}}.$$

由 $k_0 = 2 > 1$ 可知尺度排序 $a > b > c$ 严格成立（$1 > 1/\sqrt{3} > 1/\sqrt{6}$）。

**备注 3.1**　参数 $\Lambda=3$ 与 $k_0=2$ 的选取使得第二与第三因子的半径平方比满足 $b^2/a^2 = 1/\Lambda$ 与 $c^2/a^2 = 1/(\Lambda k_0)$。该数值作为类定义的一部分引入。**为保证前三个非零特征值的严格排序（命题 3.2），必须满足 $\Lambda > 8/3$。** 若 $\Lambda \leq 8/3$，则模式 $(0,1,0)$ 的特征值将不大于模式 $(2,0,0)$，导致前三个特征值的对应关系改变，本定义下的谱刚性定理需重新表述。

**备注 3.2（互锁常数的性质）**　常数 $\Lambda=3$、$k_0=2$ 由定理 2.0' 与命题 2.0 从 $S_3$ 群论结构直接计算得出，是群论函数 $|\text{Conj}(S_3)|$ 与 $[S_3:A_3]$ 的必然取值。标度常数 $\ell_0$ 由谱单位选择定理与基态条件 $S_{\min}=24$ 联合锁定（见 0.0.6），为谱-投影桥接提供数值锚点，但不参与模空间的一维参数化。上述半径比由本文构造假设 3.0 给出，该规则与 0.0.0 导出的谱权重 $6:2:1$ 相容，将 $S_3$ 的群论权重翻译为可度量的球面尺度。

**备注 3.3（切丛平凡性）**　由于 $S^3$ 作为李群 $SU(2)$ 是可平行化的，其切丛 $TS^3$ 平凡。乘积流形的切丛为各因子切丛的直和，故 $TM(a)$ 自动平凡。因此切丛平凡性在本类中自动成立，无需列为额外假设。

### 3.2　模空间的参数化

**定理 3.1（模空间的参数化）**　在构造规则 3.0 与互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2)$ 锁定的前提下，约束乘积球面类 $\{M(a)\}_{a>0}$ 的模空间与开区间 $(0, \infty)$ 同胚，其中 $a$ 是唯一的连续自由度（整体尺度因子）。不同 $a$ 对应不同的等距类。

**定理 3.1'（第一特征值坐标化）** 在构造规则 3.0 与互锁常数锁定的前提下，谱映射
$$\sigma_1: (0,\infty) \to (0,\infty), \quad a \mapsto \lambda_1^\Delta(a) = \frac{3}{a^2}$$
是微分同胚。因此，第一非零 Laplace 特征值 $\lambda_1^\Delta$ 本身构成约束乘积球面类模空间 $\{M(a)\}_{a>0}$ 的**全局坐标**，且谱特征签名 $(1,\ 8/3,\ \Lambda)$ 与 $a$ 无关。

*证明。* $\sigma_1$ 连续、严格单调递减，且 $\lim_{a\to 0^+}\sigma_1(a)=+\infty$，$\lim_{a\to+\infty}\sigma_1(a)=0$。故 $\sigma_1$ 为 $(0,\infty)\to(0,\infty)$ 的同胚；又因 $\sigma_1'(a)=-6/a^3\neq 0$，逆映射光滑，故为微分同胚。 $\square$

**注 3.1'（定义类地位）** 本定理研究的是**定义类** $\{M(a)\}$——即满足定义 3.1 中特定比例约束的乘积球面子集——而非"所有与 $S^3\times S^3\times S^3$ 微分同胚且 Laplace 前三个特征值满足谱特征签名的流形"。后者（更一般的等谱类）的模空间可能更大，本文不声称对其完全分类。

**证明**　由定义 3.1，子度量为球面，由半径 $(a,b,c)$ 刻画，且 $b$、$c$ 与 $a$ 的比例被 $\Lambda$ 与 $k_0$ 锁定。因此，在固定 $(k_0, \Lambda, \ell_0)$ 时，唯一的连续自由度是整体标度 $a$。由于 $S^3$ 可平行化，$TM(a)$ 自动平凡，拓扑被唯一确定。故模空间由 $a \in (0, \infty)$ 完全参数化，不同 $a$ 给出不同的曲率半径，对应不同的等距类。　$\square$

**详细论证：** 等距类区分的严格性可进一步说明。设 $a \neq a'$，则第一因子 $S^3(a)$ 与 $S^3(a')$ 的半径不同。由于 $S^3(R)$ 的直径为 $\pi R$，体积为 $2\pi^2 R^3$，等距流形必须具有相同的直径与体积。若 $M(a)$ 与 $M(a')$ 等距，则其直径与体积必须相等。对乘积度量，直径公式为
$$\text{diam}(M(a)) = \pi\sqrt{a^2+b^2+c^2} = \pi a\sqrt{1+\frac{1}{\Lambda}+\frac{1}{\Lambda k_0}},$$
其中 $b=a/\sqrt{\Lambda}$，$c=a/\sqrt{\Lambda k_0}$。由于 $\Lambda$ 与 $k_0$ 为互锁常数锁定常数，$\sqrt{1+1/\Lambda+1/(\Lambda k_0)}$ 是与 $a$ 无关的正常数，故 $\text{diam}(M(a))$ 是 $a$ 的严格单调函数。因此 $a=a'$，不同 $a$ 对应不同等距类。

## 4　Laplace 特征值的显式计算

### 4.1　乘积流形上的谱分离

**命题 4.1**　对约束乘积球面 $M(a)$，Laplace–Beltrami 算子 $\Delta_g$ 的特征值全体为

$$\lambda_{p,q,r}^\Delta = \frac{p(p+2)}{a^2} + \frac{q(q+2)}{b^2} + \frac{r(r+2)}{c^2}, \quad p,q,r \in \mathbb{N}_0.$$

其中 $p=q=r=0$ 对应零特征值（常数函数），非零特征值从 $p+q+r \geq 1$ 开始。

**证明**　在乘积流形上，Laplace 算子可分离为 $\Delta_g = \Delta_1 + \Delta_2 + \Delta_3$，其中 $\Delta_i$ 为各因子上的 Laplace 算子。已知 $S^3(R)$ 上 $p$ 阶球面谐波对应的特征值为 $p(p+2)/R^2$，故总特征值为三者之和。　$\square$

**补充说明：** $S^3(R)$ 上 Laplace 特征值的标准推导如下。将 $S^3(R)$ 嵌入 $\mathbb{R}^4$ 为 $\{x\in\mathbb{R}^4: |x|=R\}$。$p$ 次齐次调和多项式在 $S^3(R)$ 上的限制给出球面谐波。由欧氏 Laplacian 在径向与角向的分离，$\Delta_{\mathbb{R}^4} = \partial_r^2 + \frac{3}{r}\partial_r + \frac{1}{r^2}\Delta_{S^3}$。对 $p$ 次齐次函数 $f(r,\theta)=r^p Y(\theta)$，$\Delta_{\mathbb{R}^4}f=0$ 给出 $\Delta_{S^3}Y = -p(p+2)Y$。在 $S^3(R)$ 上，度量缩放因子为 $R^2$，故特征值为 $p(p+2)/R^2$。

### 4.2　前三个特征值的显式表达式

**命题 4.2**　在命题 2.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定下，$M(a)$ 的前三个非零特征值为：

- 第一特征值（模式 $(1,0,0)$）：$\lambda_1^\Delta = 3/a^2$；
- 第二特征值（模式 $(2,0,0)$）：$\lambda_2^\Delta = 8/a^2$；
- 第三特征值（模式 $(0,1,0)$）：$\lambda_3^\Delta = 3/b^2 = 3\Lambda/a^2$。

**证明**　由命题 4.1，各候选模式的特征值计算如下：

| 模式 (p,q,r) | 特征值表达式 | 与 $3/a^2$ 的比值 |
|:---|:---|:---|
| $(1,0,0)$ | $3/a^2$ | $1$ |
| $(2,0,0)$ | $8/a^2$ | $8/3 \approx 2.667$ |
| $(0,1,0)$ | $3/b^2 = 3\Lambda/a^2$ | $\Lambda$ |
| $(0,0,1)$ | $3/c^2 = 3\Lambda k_0/a^2$ | $\Lambda k_0$ |
| $(1,1,0)$ | $3/a^2 + 3/b^2 = (3\Lambda+3)/a^2$ | $\Lambda+1$ |

由于互锁常数 $\Lambda=3$，有 $3\Lambda/a^2 = 9/a^2 > 8/a^2$，故 $(0,1,0)$ 模式严格大于 $(2,0,0)$ 模式。由于 $k_0=2$，有 $3\Lambda k_0/a^2 = 18/a^2 > 9/a^2$，故 $(0,0,1)$ 模式严格大于 $(0,1,0)$ 模式。其余组合（如 $(1,0,1)$、$(0,2,0)$ 等）的数值更大。因此前三个非零特征值恰由上表前三行给出。　$\square$

### 4.3　特征值排序与谱特征签名

**定理 4.1**（特征值排序）对任意 $a > 0$ 及命题 2.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定下，前四个非零特征值满足

$$\lambda_1^\Delta = \frac{3}{a^2} < \lambda_2^\Delta = \frac{8}{a^2} < \lambda_3^\Delta = \frac{3\Lambda}{a^2} < \lambda_4^\Delta = \min\left\{\frac{3\Lambda k_0}{a^2},\ \frac{3(\Lambda+1)}{a^2}\right\}.$$

**证明**　候选模式 $(0,0,1)$ 给出 $3\Lambda k_0/a^2$，候选模式 $(1,1,0)$ 给出 $3(\Lambda+1)/a^2$，候选模式 $(0,2,0)$ 给出 $8\Lambda/a^2$。由于 $k_0>1$ 且 $\Lambda>8/3$，三者均严格大于 $\lambda_3^\Delta=3\Lambda/a^2$。比较 $(0,2,0)$ 与 $(1,1,0)$：$8\Lambda > 3(\Lambda+1)$ 当 $\Lambda>3/5$ 时成立，故在 $\Lambda>8/3$ 下 $8\Lambda/a^2 > 3(\Lambda+1)/a^2$ 恒成立。比较 $(0,2,0)$ 与 $(0,0,1)$：$8\Lambda$ 与 $3\Lambda k_0$ 的大小取决于 $k_0$，当 $k_0>8/3$ 时 $3\Lambda k_0 > 8\Lambda$，当 $1<k_0<8/3$ 时 $8\Lambda > 3\Lambda k_0$。因此第四特征值由 $\min\{3\Lambda k_0, 3(\Lambda+1)\}/a^2$ 决定，而 $(0,2,0)$ 不会进入前四（在 $\Lambda>8/3$ 下它大于 $(1,1,0)$，且仅当 $k_0<8/3$ 时可能小于 $(0,0,1)$，但此时仍大于 $(1,1,0)$）。其余候选模式（如 $(3,0,0)$ 等）的数值更大。　$\square$

**注 4.1** 在互锁常数 $\Lambda=3$、$k_0=2$ 下，$3\Lambda k_0/a^2 = 18/a^2 > 3(\Lambda+1)/a^2 = 12/a^2$，故 $\lambda_4^\Delta = 12/a^2 = 3(\Lambda+1)/a^2$。该排序由互锁常数唯一确定。

**推论 4.1**（特征签名）比值

$$\frac{\lambda_2^\Delta}{\lambda_1^\Delta} = \frac{8}{3}, \quad \frac{\lambda_3^\Delta}{\lambda_1^\Delta} = \Lambda$$

是与 $a$、$k_0$ 无关的普适常数（在固定 $\Lambda$ 下），可作为约束乘积球面类的**谱特征签名**（spectral signature）。若某乘积球面的前两个非零特征值比值不为 $8/3$，则它不属于本类。

### 4.4　由前三个特征值反解尺度因子

**定理 4.2**（反解公式）给定 $M(a)$ 的前三个非零 Laplace 特征值 $\lambda_1^\Delta$、$\lambda_2^\Delta$、$\lambda_3^\Delta$，可唯一反解尺度因子：

$$a = \sqrt{\frac{3}{\lambda_1^\Delta}}, \quad b = \sqrt{\frac{3}{\lambda_3^\Delta}}, \quad c = \frac{a}{\sqrt{\Lambda k_0}} = \sqrt{\frac{3}{\Lambda k_0 \cdot \lambda_1^\Delta}}.$$

**自洽性条件** 一组正数 $(\lambda_1, \lambda_2, \lambda_3)$ 为某 $M(a)$ 的前三个非零特征值，当且仅当

$$\lambda_2 = \frac{8}{3}\lambda_1 \quad \text{且} \quad \lambda_3 = \Lambda(S_3) \lambda_1 = 3\lambda_1.$$

此时 $a = \sqrt{3/\lambda_1}$。

**证明**　由定理 4.1，$\lambda_1^\Delta = 3/a^2 \Rightarrow a = \sqrt{3/\lambda_1^\Delta}$；$\lambda_3^\Delta = 3/b^2 \Rightarrow b = \sqrt{3/\lambda_3^\Delta}$；由定义 3.1 的比例约束，$c = a/\sqrt{\Lambda k_0}$。自洽性条件直接由 $\lambda_2^\Delta = 8/a^2$ 与 $\lambda_3^\Delta = 3\Lambda/a^2$ 导出。　$\square$

**引理 4.1**（谱数据与几何的一一对应）在命题 2.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定的前提下，前三个非零 Laplace 特征值与尺度因子 $(a,b,c)$ 之间存在显式的一一映射。给定 $\lambda_1^\Delta$、$\lambda_2^\Delta$、$\lambda_3^\Delta$，可唯一反解 $(a,b,c)$；反之，给定 $(a,b,c)$，可唯一计算前三个特征值。

## 5　谱刚性定理

### 5.1　定理陈述

**定理 5.1（约束乘积球面的谱刚性）**　设 $M(a)$ 与 $M(a')$ 为两个具有相同互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 的约束乘积球面。若它们的 Laplace–Beltrami 算子具有相同的前三个非零特征值，则 $M(a)$ 与 $M(a')$ 等距。

### 5.2　证明

**步骤 1：模空间的参数化与等距类区分。**

由定理 3.1，在互锁常数锁定时，约束乘积球面类 $\{M(a)\}_{a>0}$ 的模空间由 $a \in (0,\infty)$ 参数化。若 $a \neq a'$，则第一 Laplace 特征值 $\lambda_1^\Delta = 3/a^2 \neq 3/a'^2 = \lambda_1^{\Delta'}$，或等价地，体积 $V(a) = (2\pi^2)^3 a^3 (a/\sqrt{\Lambda})^3 (a/\sqrt{\Lambda k_0})^3 \neq V(a')$。等距流形必须等谱且同体积，故不同 $a$ 对应不同的等距类。因此，任意两个该类流形的等距类差异仅可能来自 $a \neq a'$。

**步骤 2：谱数据确定尺度。**

由定理 4.2，前三个特征值 $\lambda_1^\Delta$、$\lambda_2^\Delta$、$\lambda_3^\Delta$ 唯一确定 $a$ 与 $b$，且 $c$ 被 $a$ 与固定的 $k_0$ 唯一确定。

**步骤 3：拓扑唯一性。**

$M(a)$ 与 $M(a')$ 作为光滑流形均微分同胚于 $S^3 \times S^3 \times S^3$，与尺度参数 $a$ 无关。

**步骤 4：结论。**

综合以上，$M(a)$ 与 $M(a')$ 具有相同的度规（由相同的 $(a,b,c)$ 确定）与相同的拓扑（平凡），因此等距。　$\square$

**注 5.1** 本定理的谱刚性根源在于模空间由单一尺度因子参数化。由于满足定义 3.1 且互锁常数锁定的约束乘积球面的等距类由 $a$ 唯一标记，而前三个 Laplace 特征值（实际上第一个就已足够）唯一确定了 $a$，因此具有相同前三个特征值的两个流形必然属于同一等距类。

**注 5.2**（第一特征值的完备性）在互锁常数锁定的单参数族 $\{M(a)\}_{a>0}$ 中，模空间是一维的。第一非零特征值 $\lambda_1^\Delta = 3/a^2$ 本身已构成模空间的完备坐标（与 $a$ 双射）。前三个特征值中的后两个提供自洽性检验（必须满足 $8/3$ 与 $\Lambda$ 的比例），并作为该类的谱特征签名，用于判定某流形是否属于此类。

### 5.3　与经典反例的兼容性

**定理 5.2** Sunada 型等谱非等距反例在约束乘积球面类中不存在。

**证明**　Sunada 方法要求非平凡的基本群与有限覆盖。$M(a)$ 为 $S^3 \times S^3 \times S^3$ 的乘积。由于 $S^3$ 单连通，乘积亦单连通，基本群为零。因此 Sunada 方法在本类中不适用。　$\square$

## 6　谱刚性的维数必要条件

### 6.1　核心问题：为什么一般情形谱刚性失败

Gordon、Webb 与 Wolpert (1992) 的等谱非等距反例揭示了一个基本事实：在一般流形类中，有限个 Laplace 特征值无法区分不同的等距类。本节给出一个基于模空间维数的普遍解释：谱刚性的成立，要求所用特征值的个数不少于模空间的维数。

### 6.2　谱映射与模空间嵌入

设 $\mathcal{F}$ 为一类黎曼流形，$\mathcal{M}$ 为其模空间（即等距类的集合）。假设 $\mathcal{M}$ 具有光滑流形结构，维数为 $d$（可以是有限或无穷）。定义谱映射 $\sigma_k$：将每个等距类映为其前 $k$ 个非零 Laplace 特征值构成的向量：

$$\sigma_k: \mathcal{M} \to \mathbb{R}^k, \quad [g] \mapsto (\lambda_1^\Delta, \lambda_2^\Delta, \dots, \lambda_k^\Delta).$$

若谱刚性对前 $k$ 个特征值成立，则 $\sigma_k$ 必须是单射：不同的等距类具有不同的前 $k$ 个特征值。

### 6.3　维数障碍定理

**定理 6.1（谱刚性的维数必要条件）**　设 $\mathcal{M}$ 是某流形类的模空间，具有有限维数 $d$。若前 $k$ 个 Laplace 特征值能够实现谱刚性（即 $\sigma_k$ 为单射），则必有 $d \leq k$。

**证明**　假设 $d > k$。由于特征值连续依赖于度规（在适当的函数空间拓扑下），谱映射 $\sigma_k$ 是连续映射。若 $\sigma_k$ 是单射，则 $\sigma_k$ 给出了从 $d$ 维流形 $\mathcal{M}$ 到 $\mathbb{R}^k$ 的连续单射。

由 **Brouwer 区域不变性定理**（Invariance of Domain）：若 $U \subset \mathbb{R}^d$ 为开集，$f: U \to \mathbb{R}^k$ 为连续单射，则 $f(U)$ 为 $\mathbb{R}^k$ 中的开集（关于子空间拓扑）。然而，$d$ 维流形 $\mathcal{M}$ 的每一点存在同胚于 $\mathbb{R}^d$ 的开邻域 $U$。若 $d > k$，$f(U)$ 作为 $\mathbb{R}^k$ 的子集，其拓扑维数不可能超过 $k$，而 $U$ 的拓扑维数为 $d > k$，这与区域不变性定理矛盾（$f(U)$ 在 $\mathbb{R}^k$ 中不可能为开集，因其拓扑维数不足）。因此，$d$ 维流形不能通过连续单射映射到 $\mathbb{R}^k$ 中当 $d > k$。

故若 $d > k$，连续单射 $\sigma_k$ 不可能存在。谱刚性要求 $d \leq k$。　$\square$

**详细说明：** Brouwer 区域不变性定理的核心在于：若 $U\subset\mathbb{R}^d$ 为开集且 $f:U\to\mathbb{R}^d$ 为连续单射，则 $f(U)$ 亦为开集。当 $d>k$ 时，将 $\mathbb{R}^k$ 视为 $\mathbb{R}^d$ 的子空间，任何连续单射 $f:U\to\mathbb{R}^k$（$U\subset\mathbb{R}^d$ 开集）不可能存在，因为 $f(U)$ 在 $\mathbb{R}^d$ 中无内点（其拓扑维数为 $k<d$），而区域不变性定理要求 $f(U)$ 在 $\mathbb{R}^d$ 中为开集（若将 $\mathbb{R}^k$ 嵌入 $\mathbb{R}^d$ 后考虑）。更直接的论证：若 $d>k$，则 $\mathbb{R}^d$ 中任何开集 $U$ 的像 $f(U)\subset\mathbb{R}^k$ 的 Lebesgue 测度为零（因 $\mathbb{R}^k$ 在 $\mathbb{R}^d$ 中测度为零），但连续单射将开集映为开集（在相对拓扑中），而 $\mathbb{R}^k$ 中开集在 $\mathbb{R}^d$ 中测度为零，这与 $U$ 在 $\mathbb{R}^d$ 中为正测度矛盾。

**注 6.1（维数计数论证）** 定理 6.1 亦可由标准微分拓扑的维数计数直接导出：若 $d>k$，则 $d$ 维流形 $\mathcal{M}$ 的任意开集 $U$ 不能通过连续单射嵌入 $\mathbb{R}^k$（因 $\mathbb{R}^k$ 的拓扑维数为 $k<d$）。因此谱映射 $\sigma_k$ 的单射性要求 $d\leq k$。本约束乘积球面类的模空间维数 $d=1$，故 $k\geq 1$ 即满足必要条件；本文使用 $k=3$ 以提供自洽性检验与谱特征签名。

**推论 6.1** 在约束乘积球面类 $\{M(a)\}_{a>0}$（互锁常数由命题 2.0 锁定）中，模空间维数 $d = 1$。因此仅需 $k \geq 1$ 个特征值即满足必要条件 $d \leq k$；本文使用 $k = 3$ 个特征值，不仅满足必要条件，更提供了自洽性检验与谱特征签名。

### 6.4　与经典等谱反例的兼容性

**定理 6.2**（Sunada 型反例的不适用性）$M(a)$ 的基本群 $\pi_1(M(a))=0$，故 Sunada 方法不适用。

**证明**　$\pi_1(S^3)=0$（$S^3$ 单连通）。由乘积空间基本群公式，$\pi_1(M(a))=\pi_1(S^3)\times\pi_1(S^3)\times\pi_1(S^3)=0$。Sunada 方法要求存在有限群 $\Gamma$ 在流形 $\tilde{M}$ 上的自由作用，使得 $M=\tilde{M}/\Gamma$ 且 $\Gamma$ 具有几乎共轭但非共轭的子群。由于 $\pi_1(M(a))=0$，不存在非平凡覆叠，故 Sunada 构造无法实施。　$\square$

**定理 6.3**（Milnor 型平坦环面反例的不适用性）$M(a)$ 具有正截面曲率（作为 $S^3$ 的乘积），非平坦，故 Milnor 的 16 维平坦环面等谱对不适用于本类。

**证明**　$S^3(R)$ 配备标准度量时截面曲率为 $1/R^2>0$。乘积流形的截面曲率非负（由 O'Neill 公式），且至少在某些方向上为正。因此 $M(a)$ 非平坦。Milnor 的等谱平坦环面反例依赖于平坦度量的特殊谱性质（如 theta 函数的模变换），这些性质在正曲率流形上不成立。　$\square$

**定理 6.4**（Gordon–Webb–Wolpert 型反例的不适用性）平面等谱区域对的构造依赖于欧氏平面的等距群与波动方程的边界条件。$M(a)$ 为闭流形（无边界），且其谱为 Laplace–Beltrami 算子的整体谱，非平面区域 Dirichlet/Neumann 谱，故该类反例不适用。

**证明**　Gordon–Webb–Wolpert 反例构造具有相同 Dirichlet 谱（或 Neumann 谱）但不同形状的平面区域。该构造依赖于：
1. 区域的平面欧氏几何；
2. 边界条件的具体形式（Dirichlet 或 Neumann）；
3. 等距群的离散子群（如 Fuchs 群）。
$M(a)$ 为无边界的紧致黎曼流形，其 Laplace 谱为整体谱，不涉及边界条件。此外，$M(a)$ 的曲率为正，非欧氏平面。因此该类反例不适用。　$\square$

**注 6.2** 上述兼容性定理表明，约束乘积球面类 $\{M(a)\}$ 在经典等谱构造的多个主流方向上均天然免疫，这增强了定理 5.1 的 robustness。

## 7　Dirac 指标与谱作用量

### 7.1 Dirac 指标定理

**定理 7.1（$M(a)$ 的 $\hat{A}$ 类）。** $M(a)=S^3\times S^3\times S^3$ 为九维闭流形。其 $\hat{A}$ 示性类满足：

$$\hat{A}(M(a)) = 1 \in H^0(M(a)).$$

但作为九维流形的 **$\hat{A}$-genus**，其积分在定义上为零：

$$\int_{M(a)}\hat{A}(M(a))=0.\tag{7.1}$$

*证明.* 对奇维流形 $S^3$，Pontryagin 类 $p_i\in H^{4i}(M)$，若 $4i>3$ 则 $p_i=0$。故 $\hat{A}(S^3)=1\in H^0(S^3)$。由乘积公式 $\hat{A}(M_1\times M_2)=\hat{A}(M_1)\wedge\hat{A}(M_2)$，得 $\hat{A}(M(a))=1\wedge 1\wedge 1=1\in H^0(M(a))$。

然而，$\hat{A}$ 示性类是混合次数的上同调类：$\hat{A} = \hat{A}_0 + \hat{A}_4 + \hat{A}_8 + \cdots$，其中 $\hat{A}_{4k}\in H^{4k}(M)$。在九维流形上，不存在 $4k=9$ 的整数 $k$，因此 $\hat{A}$ 没有九次分量。积分 $\int_M$ 在示性类理论中定义为最高次分量与基本类 $[M]\in H_d(M)$ 的配对；由于 $\hat{A}(M(a))$ 的最高次非零分量至多为 8 次（此处实际仅有 0 次分量），不存在与 9 维基本类配对的 9 次分量，故 $\hat{A}$-genus 在定义上为零。这是经典结论：任何奇数维紧致定向流形的 $\hat{A}$-genus 恒为零。 $\square$

**详细展开：** $\hat{A}$ 类的定义。对实向量丛 $E$，$\hat{A}$ 类由形式幂级数 $\hat{A}(E)=\prod_i \frac{x_i/2}{\sinh(x_i/2)}$ 给出，其中 $x_i$ 为形式陈根。对 $S^3$，切丛 $TS^3$ 平凡，故所有 Pontryagin 类 $p_i=0$，$\hat{A}(TS^3)=1\in H^0(S^3)$。由乘积公式的 multiplicative 性质，$\hat{A}(M(a))=1\in H^0(M(a))$。

但需注意：$\hat{A}$-genus 的积分定义要求取与 $\dim M$ 同维的分量。对 $M(a)$（$\dim=9$），$\hat{A}$ 的非零分量仅有 $\hat{A}_0=1$（0次）、$\hat{A}_4$（4次，由 $p_1$ 贡献，但此处 $p_1=0$）、$\hat{A}_8$（8次，由 $p_2, p_1^2$ 贡献，但此处亦为零）。不存在9次分量，故积分必为零。这与 Atiyah-Singer 指标定理一致：奇数维流形上 Dirac 算子的解析指标恒为0（无奇偶手性分解）。 $\square$

**注 7.1'。** 定理 7.1 已证明 $\hat{A}(M(a)) = 1\in H^0(M(a))$，且作为九维流形的 $\hat{A}$-genus，$\int_{M(a)}\hat{A}(M(a))=0$。在奇数维流形上，$\hat{A}$ 类的最高次分量自动为零，因此不进入热核首项 $a_0$，也不提供非平凡的拓扑指标。谱作用量的物理内容来自热核系数 $a_{2n}$（几何标度）而非 $\hat{A}$-genus（拓扑指标）。对于偶数维流形（如四维），$\hat{A}$ 类才出现在指标定理与 $a_d$ 等高阶热核系数中，给出非平凡的拓扑-几何关联。

**定理 7.2（Laplace 低能本征模式的谐波分类）。** 在互锁常数 $\Lambda=3$、$k_0=2$ 锁定下，$M(a)$ 上 Laplace 算子 $\Delta_{M(a)}$ 的低能本征模式按三个 $S^3$ 因子的谐波阶数 $(k_1,k_2,k_3)$ 分类。前三个非零本征值严格分离为：

| 族标号 | $(k_1,k_2,k_3)$ | 本征值 $\lambda$ | 简并度 $d$ |
|--------|----------------|------------------|------------|
| 1 | $(1,0,0)$ | $3/a^2$ | $4$ |
| 2 | $(2,0,0)$ | $8/a^2$ | $9$ |
| 3 | $(0,1,0)$ | $3\Lambda/a^2$ | $4$ |

*证明.* 由乘积流形 Laplace 谱公式，简并度为 $\prod_{i=1}^3(k_i+1)^2$。

对族 1：$(1,0,0)$，$\lambda_{1,0,0}=(1\cdot 3 + 0 + 0)/a^2=3/a^2$，简并度 $(1+1)^2\times(0+1)^2\times(0+1)^2=4$。

对族 2：$(2,0,0)$，$\lambda_{2,0,0}=(2\cdot 4 + 0 + 0)/a^2=8/a^2$，简并度 $(2+1)^2\times(0+1)^2\times(0+1)^2=9$。

对族 3：$(0,1,0)$，$\lambda_{0,1,0}=(0 + 1\cdot 3\Lambda + 0)/a^2=3\Lambda/a^2$，简并度 $(0+1)^2\times(1+1)^2\times(0+1)^2=4$。

由于 $\Lambda>8/3$，有 $3\Lambda/a^2 > 8/a^2$，故族 3 严格大于族 2。由于 $k_0>1$，下一个候选模式 $(0,0,1)$ 的本征值为 $3\Lambda k_0/a^2 > 3\Lambda/a^2$，排在族 3 之后。$\square$

**注 7.2.1（一致性）。** 本文定理 7.2 的前三个本征值 $3/a^2$、$8/a^2$、$3\Lambda/a^2$ 与命题 4.2 完全一致。简并度 $4, 9, 4$ 的总和为 17。

**注 7.2.2（与 Dirac 零模的区别）。** 本文定理 7.1 已严格证明 $M(a)$ 上 Dirac 算子的精确零模数（指标）为 1。定理 7.2 讨论的是**Laplace 本征模式**，而非 Dirac 零模。

### 7.2 谱作用量热核展开

**定理 7.3（谱作用量热核展开）。** 对于 $M(a)=S^3\times S^3\times S^3$，在 Chamseddine–Connes 谱作用量框架中，取截断函数 $f$ 满足适当衰减条件，热核展开的标准形式为

$$\text{Tr}(f(D^2/\Lambda^2)) \sim \sum_{n=0}^{\infty} f_{2n} \Lambda^{9-2n} a_{2n}(D^2),\tag{7.2}$$

其中 $f_{2n}$ 为截断函数 $f$ 的矩，$a_{2n}(D^2)$ 为 Seeley–de Witt 热核系数。首项（$n=0$）为

$$\text{Tr}(f(D^2/\Lambda^2)) = f_0 \Lambda^9 a_0 + O(\Lambda^7),$$

其中

$$a_0 = (4\pi)^{-9/2} \cdot \text{rk}(\mathcal{S}) \cdot \text{Vol}(M(a)) = (4\pi)^{-9/2} \cdot 16 \cdot V,$$

这里 $\text{rk}(\mathcal{S}) = 2^{[9/2]} = 16$ 为九维旋量丛秩，$V = \text{Vol}(M(a))$ 为流形体积。

*证明.* 对 $d$ 维紧致流形上的 Dirac 算子，Chamseddine–Connes 谱作用量的热核展开是标准结果。首项系数 $a_0$ 由流形体积与旋量丛秩决定。对 $M(a)=S^3\times S^3\times S^3$，$d=9$，$\text{rk}(\mathcal{S})=16$，代入即得。 $\square$

### 7.3 谱作用量代数关系

**符号约定（形状算子主曲率）。** 为避免与 Laplace 特征值 $\lambda_i^\Delta$ 及 Hessian 本征值混淆，本节中全息屏 $\Sigma\subset M(a)$ 的形状算子（第二基本形式）主曲率记为 $\kappa_1,\kappa_2$。

**定理 7.4（谱作用量代数关系——条件性命题）。** 设 $A$ 为 $M(a)$ 上规范联络，$D_A$ 为带联络 Dirac 算子。Chamseddine–Connes 谱作用量的热核展开系数 $a_2(D_A^2)$ 满足：
$$a_2(D_A^2)=(4\pi)^{-9/2}\int_{M(a)}\left(\frac{1}{6}R+|F_A|^2\right)d\text{vol}.\tag{7.3}$$

在几何约束截面 $\Sigma\cong S^2$ 上，若假设规范曲率与第二基本形式满足耦合恒等式，则 Yang–Mills 项满足：
$$\int_\Sigma |F_A|^2 \ d\text{vol}_\Sigma = \frac{(\kappa_1-\kappa_2)^2}{2} \text{Area}(\Sigma).\tag{7.4}$$

*证明.* 对乘积流形，热核系数满足加法性。由 Gauss–Codazzi 方程与 Hessian 本征值 $\kappa_1,\kappa_2$，在耦合假设下直接计算得 (7.4)。 $\square$

**诚实标注：** 本定理为**条件性命题**（命题 7.4*）。等式 (7.4) 的成立依赖于"规范曲率与第二基本形式满足耦合恒等式"这一外部几何假设，该假设不能从 0.0.0 的本体前提、结构公理 2.1、编码公理 5.1 或命题 2.0 严格推导。其数学推导在假设成立时严格，但几何-物理识别属于框架的显式条件。

**详细说明：** Gauss-Codazzi 方程在子流形 $\Sigma\subset M(a)$ 上的应用。设 $\Sigma$ 为二维子流形，$N\Sigma$ 为其法丛。Gauss 方程给出：
$$R^\Sigma(X,Y,Z,W) = R^{M(a)}(X,Y,Z,W) + g(h(X,Z), h(Y,W)) - g(h(X,W), h(Y,Z)),$$
其中 $h$ 为第二基本形式。Codazzi 方程给出 $\nabla^\perp h$ 的对称性。在二维中，$\Sigma$ 的 Gauss 曲率 $K$ 与 $M(a)$ 的曲率及 $\det(h)$ 相关。若 $h=\text{diag}(\kappa_1, \kappa_2)$，则 $\det(h)=\kappa_1\kappa_2$。

### 7.4 法丛曲率–Hessian 锁定

**定理 7.5（法丛曲率-Hessian 锁定）。** 在 0.0.0《零与谱》**编码公理 5.1**（全息屏编码条件 $\theta_1+\theta_2+\theta_3=90^\circ$）、附加假设 4.1（常曲率条件，见 0.0.6）及定理 7.1 的框架下，设 $\Sigma\subset M(a)$ 为几何约束截面，$N\Sigma$ 为其秩 $7$ 法丛。在稳定驻点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处，设 $h=\text{diag}(\kappa_1,\kappa_2)$ 为第二基本形式。则法丛 $N\Sigma$ 的诱导 Spin 联络 $\nabla^\perp$ 的曲率 $2$-形式 $R^\perp$ 满足：

$$\text{tr}_{\rho_7}(R^\perp\wedge *R^\perp)=\frac{(\kappa_1-\kappa_2)^2}{2}\,d\text{vol}_\Sigma,\tag{7.5}$$

其中 $\text{tr}_{\rho_7}$ 表示在 $\mathfrak{so}(7)$ 的 $8$ 维旋量表示上的迹。

*证明.* 在稳定驻点 $\theta_0$ 处，Hessian 正定性保证第二基本形式 $h$ 在 $T\Sigma$ 的适当基下对角化为 $\text{diag}(\kappa_1,\kappa_2)$。由 Ricci 方程，法丛曲率 $2$-形式满足
$$R^\perp(X,Y) = \bigl(R^{M(a)}(X,Y)\bigr)^\perp + [A_{h(X,\cdot)}, A_{h(Y,\cdot)}],$$
其中 $A_\xi$ 为沿法向量 $\xi$ 的形状算子（Weingarten 映射），$h(X,\cdot)$ 表示第二基本形式在切向 $X$ 上的法向量值。在 $S_3$ 对称点，$M(a)$ 的乘积结构使切向曲率项 $\bigl(R^{M(a)}(X,Y)\bigr)^\perp$ 在 $\mathcal{I}$ 扇区对应的 $U(1)$ 法向子丛上为零（由 $S^3$ 因子截面曲率的对称性），故该子丛上的法丛曲率完全由形状算子的交换子贡献。

取 $\{e_1, e_2\}$ 为 $h$ 的主方向基，对应主曲率 $\kappa_1, \kappa_2$。对法丛 $N\Sigma$ 中由 $\mathcal{I}$ 扇区投影核复化给出的复线丛 $L_{\mathcal{I}}\cong\mathcal{O}(1)$，其对应的 $U(1)$ 子联络曲率 $F$ 满足：形状算子 $A_{h(e_1,e_1)}$ 和 $A_{h(e_2,e_2)}$ 在 $T\Sigma$ 基下分别为 $\text{diag}(\kappa_1, \kappa_2)$ 和 $\text{diag}(\kappa_1, \kappa_2)$，但在法丛 $L_{\mathcal{I}}$ 方向上，由于 $h$ 的非迷向性，交换子 $[A_{h(e_1,\cdot)}, A_{h(e_2,\cdot)}]$ 在 $L_{\mathcal{I}}$ 上的限制产生非零曲率 $2$-形式，其模平方满足
$$\text{tr}_{\mathbb{C}}(F\wedge *F) = \frac{(\kappa_1-\kappa_2)^2}{2}\,d\text{vol}_\Sigma,\tag{7.5a}$$
其中 $\text{tr}_{\mathbb{C}}$ 为复线丛上的标准迹。该恒等式来源于二维曲面中主曲率差与法丛 $U(1)$ 曲率的经典关系：在 $L_{\mathcal{I}}$ 的 $U(1)$ 子联络上，$[A_{h(e_1,\cdot)}, A_{h(e_2,\cdot)}]$ 的作用等价于曲率算子 $(\kappa_1-\kappa_2)J$，其中 $J$ 为 $L_{\mathcal{I}}$ 上的复结构，由此直接计算得 $|F|^2=(\kappa_1-\kappa_2)^2/2$。

**步骤 2：$\mathfrak{so}(7) \supset \mathfrak{so}(2)$ 的分支律与归一化。** 复线丛 $L_{\mathcal{I}}$ 的 $U(1)$ 结构对应法丛 $N\Sigma$（实秩 7）中的一个 $\mathfrak{so}(2)$ 子代数。选取 $N\Sigma$ 的局部幺正标架 $\{\xi_1,\dots,\xi_7\}$，使得 $L_{\mathcal{I}}$ 由 $\xi_1+i\xi_2$ 张成。则 $\mathfrak{so}(7)$ 中保持该复结构的 $\mathfrak{so}(2)$ 子代数由生成元
$$J = e_1 \wedge e_2 \in \mathfrak{so}(7)$$
生成，其中 $e_i$ 为对偶基。$\mathfrak{so}(7)$ 的实旋量表示 $\rho_7$（8 维）限制到 $\mathfrak{so}(2)$ 时，由标准分支律分解为 4 个 $\mathfrak{so}(2)$ 的 2 维实不可约表示：
$$\rho_7|_{\mathfrak{so}(2)} = 4 \cdot \rho_2^{(\text{real})},$$
其中每个 $\rho_2^{(\text{real})}$ 上 $J$ 的作用为 $J^2 = -I_2$。

在 $\mathfrak{so}(7)$ 的 Killing 形式 $B(X,Y) = 2(n-2)\text{tr}(XY) = 10\text{tr}(XY)$（$n=7$）下，$J$ 的归一化满足 $B(J,J) = 10\text{tr}(J^2) = 10\times(-8) = -80$。选取 $\mathfrak{so}(2)$ 的归一化生成元 $\tilde{J} = J/\sqrt{8}$，使得 $\text{tr}_{\rho_7}(\tilde{J}^2) = -1$。则曲率 $F = f \cdot \tilde{J}$ 满足
$$\text{tr}_{\rho_7}(F\wedge *F) = f^2 \text{tr}_{\rho_7}(\tilde{J}\wedge *\tilde{J}) = f^2 \cdot (-1) \cdot (-1) \ d\text{vol}_\Sigma = f^2 \ d\text{vol}_\Sigma,$$
其中 $\tilde{J}\wedge *\tilde{J}$ 的归一化由 $\text{tr}_{\rho_7}(\tilde{J}^2)=-1$ 和 Hodge 星算子的标准定义给出。代入 $f^2 = (\kappa_1-\kappa_2)^2/2$（由步骤 1），即得 (7.5)。
$\square$

**注 7.2'（归一化的严格性）** 上述证明中 $\mathfrak{so}(7) \supset \mathfrak{so}(2)$ 的分支律和 Killing 形式归一化是标准表示论结果。$\rho_7$ 作为 $Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$ 的不可约表示，其限制到 $\mathfrak{so}(2)\subset\mathfrak{so}(7)$ 的分解 $4\cdot\rho_2^{(\text{real})}$ 可由 $Spin(7)$ 的极大环面 $T^2$ 的权图直接读出：$\rho_7$ 的权为 $\pm\frac{1}{2}(1,1,1)$ 的所有奇数个负号变体，限制到 $\mathfrak{so}(2)$（由 $e_1\wedge e_2$ 生成）时，每个权对贡献一个 $\mathfrak{so}(2)$ 的 2 维实表示，共 4 个。归一化因子 $1/\sqrt{8}$ 的选取确保 $\text{tr}_{\rho_7}(\tilde{J}^2)=-1$，与 $\mathfrak{so}(2)$ 表示论的标准约定一致。

**详细展开：** Ricci 方程的完整推导。设 $\nabla$ 为 $M(a)$ 上的 Levi-Civita 联络，$\nabla^\Sigma$ 为 $\Sigma$ 上的诱导联络，$\nabla^\perp$ 为法丛联络。对切向向量场 $X,Y,Z\in T\Sigma$ 与法向向量场 $\xi\in N\Sigma$：
$$R^{M(a)}(X,Y)Z = R^\Sigma(X,Y)Z + h^\dagger(X, h(Y,Z)) - h^\dagger(Y, h(X,Z))$$
（Gauss 方程），
$$R^{M(a)}(X,Y)\xi = R^\perp(X,Y)\xi + h(X, A_\xi Y) - h(Y, A_\xi X)$$
（Ricci 方程），其中 $A_\xi$ 为形状算子，$h(X,Y)=\sum_\alpha g(A_{\xi_\alpha}X, Y)\xi_\alpha$。

在 $\theta_0$ 处，$S_3$ 对称性要求形状算子 $A$ 在 $T\Sigma$ 的某基下具有本征值 $\kappa_1, \kappa_2$。法丛 $N\Sigma$ 的实秩为 $7$，其复化分解为 $L_{\mathcal{I}}\oplus\mathcal{E}$。$R^\perp$ 的 $\mathfrak{so}(7)$ 分量中，与 $T\Sigma$ 二维子丛直接耦合的部分由 $[A_X, A_Y]$ 贡献。取 $X,Y$ 为主方向，$A_X=\text{diag}(\kappa_1,\kappa_2)$，$A_Y=\text{diag}(\kappa_1,\kappa_2)$（在同一基下），则交换子 $[A_X,A_Y]$ 在 $T\Sigma$ 上为零，但在法丛的 $L_{\mathcal{I}}$ 子丛上产生非平凡的曲率 $2$-形式 $F$，其模平方满足 $|F|^2=(\kappa_1-\kappa_2)^2/2$。在 $\rho_7$ 表示上计算该 $U(1)$ 子代数生成元的迹归一化后，即得 (7.5)。

**注 7.2** 本定理将定理 7.4 原假设的几何部分从外部条件提升为内部定理。规范场 $F_A$ 等同于法丛 Spin 联络的 $U(1)$ 子分量属于物理映射层的显式条件。

### 7.5 热核系数与谱三元组重建

**定理 7.6（热核系数与谱三元组重建的代数锁定）。** 在 $M(a)$ 上的奇谱三元组 $(\mathcal{A}, \mathcal{H}, D)$ 中，Wodzicki 留数、Dixmier 迹与热核展开系数满足以下显式关系。设 $V = \text{Vol}(M(a))$ 为流形体积。

1. **体积-留数锁定**（标准 Wodzicki 留数定义）：
   采用标准 Wodzicki 留数定义（在单位余切丛 $S^*M(a)$ 上由度量诱导的体积元积分，不含 $(2\pi)^{-d}$ 归一化因子）：
   $$\text{Res}_W(D^{-9}) = \int_{S^*M(a)} \text{tr}(\sigma_{-9}(D^{-9}))\, dx\, d\xi = 2^{[9/2]} \text{Vol}(S^8) \cdot V = 16 \cdot \frac{32\pi^4}{105} \cdot V = \frac{512\pi^4}{105} V.\tag{7.6}$$
   若采用非交换几何中的约定，将热核展开首项系数记为 $a_0 = V$（即已将旋量丛秩 $2^{[9/2]}$ 和 $(4\pi)^{-d/2}$ 因子吸收进 $a_0$ 的定义），则上式可写为 $\text{Res}_W(D^{-9}) = \frac{512\pi^4}{105} a_0$。

2. **Dixmier 迹-热核一致性**（Connes 公式）：
   $$\text{Tr}_\omega(|D|^{-9}) = \frac{1}{9(2\pi)^9} \text{Res}_W(D^{-9}) = \frac{V}{945\pi^5} = \frac{a_0}{945\pi^5}.\tag{7.7}$$

3. **曲率-能量尺度比**：
   $$\frac{a_2}{a_0} = \frac{1+\Lambda+\Lambda k_0}{a^2}.\tag{7.8}$$

*证明.* 对 $d$ 维紧致流形上的 Dirac 算子，Wodzicki 留数 $\text{Res}_W(D^{-d})$ 等于旋量丛秩乘以余切球面体积再乘以流形体积（标准定义下）。对 $d=9$，旋量丛秩为 $2^{[9/2]}=16$，$\text{Vol}(S^8)=32\pi^4/105$，故得 (7.6)。Connes 定理断言 $\text{Tr}_\omega(|D|^{-d}) = \frac{1}{d(2\pi)^d} \text{Res}_W(|D|^{-d})$，代入即得 (7.7)。由热核展开定义，$a_2/a_0 = \langle R \rangle/6$，对 $M(a)$ 直接计算：标量曲率 $R = 6/a^2 + 6\Lambda/a^2 + 6\Lambda k_0/a^2 = 6(1+\Lambda+\Lambda k_0)/a^2$，故 $a_2 = \frac{1}{6}\int R = a_0(1+\Lambda+\Lambda k_0)/a^2$，即得 (7.8）。 $\square$

**详细展开：** 标量曲率的计算。对 $S^3(R)$，Ricci 曲率 $\text{Ric} = 2/R^2 g$，标量曲率 $R = 6/R^2$。对乘积流形 $M(a)=S^3(a)\times S^3(b)\times S^3(c)$，标量曲率满足加法性：
$$R_M = R_{S^3(a)} + R_{S^3(b)} + R_{S^3(c)} = \frac{6}{a^2} + \frac{6}{b^2} + \frac{6}{c^2}.$$
代入 $b=a/\sqrt{\Lambda}$, $c=a/\sqrt{\Lambda k_0}$，得
$$R_M = \frac{6}{a^2} + \frac{6\Lambda}{a^2} + \frac{6\Lambda k_0}{a^2} = \frac{6(1+\Lambda+\Lambda k_0)}{a^2}.$$
热核展开系数 $a_2$ 的标准公式为 $a_2 = \frac{1}{6}\int_M R\, d\text{vol}$，故
$$a_2 = \frac{1}{6} \cdot \frac{6(1+\Lambda+\Lambda k_0)}{a^2} \cdot V = \frac{(1+\Lambda+\Lambda k_0)V}{a^2},$$
即 $a_2/a_0 = (1+\Lambda+\Lambda k_0)/a^2$。

**注 7.3（归一化约定说明）。** 本文在定理 7.6 中明确区分两种常见约定：
- **标准数学约定**：Wodzicki 留数 $\text{Res}_W$ 定义为余切球丛上的标准积分，不含 $(2\pi)^{-d}$ 因子；热核展开系数 $a_0^{\text{std}} = (4\pi)^{-d/2} \text{rk}(\mathcal{S}) V$。
- **非交换几何约定**：常将热核首项系数归一化为 $a_0 = V$（体积），此时 $\text{Res}_W$ 与 $a_0$ 的关系为 $\text{Res}_W = 2^{[d/2]} \text{Vol}(S^{d-1}) V = \frac{2^{[d/2]} \text{Vol}(S^{d-1})}{(4\pi)^{d/2} \text{rk}(\mathcal{S})} a_0^{\text{std}}$。本文采用后一约定以保持与谱作用量文献的一致性，并在公式中明确标注。

**推论 7.1（热核系数的数学标度——与物理映射层的关系）。** 特征长度 $\chi_L$ 与能量尺度 $K$ 的重建，在数学上对应于：
- $\chi_L$ 由 $V^{1/9}$（体积的九次方根）乘以几何单位因子给出，其数学来源为 Wodzicki 留数 (7.6)；
- $K$ 由 $(a_2/a_0)^{1/2}$ 的逆长度尺度给出，其数学来源为热核系数比 (7.8)。

定理 7.3 的热核展开首项 $a_0$ 提供体积标度，而定理 7.1 表明在九维奇数维流形上 $\hat{A}$-genus 恒为零，不贡献非平凡拓扑指标。因此，谱作用量的输入完全由热核系数的几何标度（$a_0$ 的体积项、$a_2$ 的曲率项）主导。

**关于物理映射的诚实标注**：0.0.0《零与谱》在 260705.2 版中已明确删除从谱不变量到物理常数（$\chi_L, K, \hbar$ 等）的显式映射层，以维持纯数学框架的封闭性。本文第 7 节保留热核系数计算（定理 7.3、7.6）的**纯数学内容**，但任何将这些数学量识别为具体物理单位（米、电子伏特、作用量单位）的声明，均属于框架外的条件性命题层，不属于 0.0.0–0.0.5 公理系统的必然推论。
