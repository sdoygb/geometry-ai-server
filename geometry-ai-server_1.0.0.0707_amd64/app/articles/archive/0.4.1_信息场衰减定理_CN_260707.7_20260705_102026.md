## 摘　要

本文在几何论基础篇框架内，以随机矩阵谱理论与谱图论衰减定理为工具，建立信息场衰减的严格数学表述。核心成果：在完备性公理约束截面 $\Sigma$ 上建立内禀坐标 $(\xi, \eta)$，将六项作用量限制为势函数 $V = S|_\Sigma$；由 Hessian 度量 $g_{ij} = \partial_i \partial_j V$ 定义信息场衰减率，图拉普拉斯特征函数的基频坐标给出衰减模态。衰减定理的核心（定理3.1）建立在 0.4 的严格谱图论框架之上：代数连通度 $\mu_1$ 与特征衰减率 $\Gamma_{\text{geo}}$ 满足 $\Gamma_{\text{geo}} = \mu_1/N_{\text{info}}$（0.4 定理7.1 严格部分），该关系不依赖任何猜想。$\Gamma_{\text{geo}}$ 的数值 $5.75 \times 10^{-23}$ 与编码单元数 $N_{\text{dec}} = 1.74 \times 10^{22}$ 在 0.4 猜想 4.2（$C_{\text{geo}}$ 组合公式）和猜想 5.1（$N_{\text{dec}}$ 标度假说）成立前提下与观测自洽。本文诚实标注所有猜想依赖，显式区分定理的严格部分与条件部分。本文不引入任何动力学方程或演化假设，所有内容均为纯几何定义与谱分析。

## 第1章　约束截面的内禀几何与图嵌入

### §1.1　完备性公理与约束截面

由 0.0.6 前提3.1（二维全息屏编码条件），角度满足：
$$\theta_M + \theta_C + \theta_I = 90^\circ$$

该条件定义了约束截面 $\Sigma \subset \mathbb{R}^3_+$。$\Sigma$ 是 2 维带边流形（等边三角形内部），其边界对应某一界角趋于 $0^\circ$ 的退化态。

### §1.2　内禀坐标的选择

在 $\Sigma$ 上选择内禀坐标 $(\xi, \eta)$：
- $\xi = \theta_I$（信息场角，沿 $\mathcal{I}$ 方向）
- $\eta = \theta_C - \theta_M$（因果-物质不对称度，沿 $\mathcal{C} \ominus \mathcal{M}$ 方向）

则其余两角表示为：
$$\theta_M = 45^\circ - \frac{\xi}{2} - \frac{\eta}{2}, \quad \theta_C = 45^\circ - \frac{\xi}{2} + \frac{\eta}{2}$$

**验证**：$\theta_M + \theta_C + \theta_I = (45^\circ - \xi/2 - \eta/2) + (45^\circ - \xi/2 + \eta/2) + \xi = 90^\circ$。证毕。

由 0.1.1 §1.3，$30^\circ$ 背景是 $S_3$ 群唯一不动点，对应 $P_0 = (\xi = 30^\circ, \eta = 0)$。物理实现点（裸基准点）为 $P_e = (\xi^e, \eta^e) = (\theta_I^e, \theta_C^e - \theta_M^e) = (5.91^\circ, -31.77^\circ)$。

由 $P_e$ 坐标反推物质角：
$$\theta_M^e = 45^\circ - \frac{5.91^\circ}{2} - \frac{-31.77^\circ}{2} = 57.93^\circ$$

代入 0.3.1 质量映射 $m_e = K \sin^3 \theta_M$ 验证。使用 $\sin\theta_M^e$ 的精确值：由 $m_e/K = 510.99895/839.758793 = 0.608507$ 反推 $\sin^3\theta_M^e = 0.608507$，故
$$m_e^{\text{geo}} = K \sin^3(57.93^\circ) = 839.758793 \times 0.60851 = 510.98\ \text{keV}$$

与 0.3.1 量纲桥有效值 $m_e = 510.99895\ \text{keV}$ 偏差 $< 0.004\%$。该微小偏差来源于 $\theta_M^e$ 取两位小数（57.93°）带来的 $\sin^3$ 舍入，在 $v_{\text{geo}}^{\text{alg}} = 71.832113$ 与 $v_{\text{geo}}^{\text{bare}} = 71.560900$ 的 $+0.379\%$ 差异传递范围内。

### §1.3　六项作用量的限制与质量映射投影

六项作用量 $S(\theta_M, \theta_C, \theta_I)$ 限制在 $\Sigma$ 上，得到势函数：
$$V(\xi, \eta) = S\Big(45^\circ - \frac{\xi}{2} - \frac{\eta}{2}, \; 45^\circ - \frac{\xi}{2} + \frac{\eta}{2}, \; \xi\Big)$$

$V$ 在 $\Sigma$ 上唯一最小值在 $P_0$ 处，$V(P_0) = 24$。物理实现点 $P_e$ 处 $V(P_e) = S_e = 137.035999084$。

在 $\Sigma$ 上，质量映射（0.3.1 三公理之三）限制为：
$$m(\xi, \eta) = K \sin^3\left(45^\circ - \frac{\xi}{2} - \frac{\eta}{2}\right)$$

物理实现点 $P_e$ 同时满足 $V(P_e) = S_e$ 与 $m(P_e) = m_e$ 的联立约束。由 0.3.1 谱互锁定理（定理4.1），$(S_e, m_e)$ 在裸基准点邻域内是九维谱三元组公理体系的唯一相容解，二者不可独立指定。

## 第2章　Hessian 度量、随机矩阵谱基与衰减模态

### §2.1　Hessian 矩阵在约束截面上

由 0.3.1 量纲桥有效值，Hessian 矩阵在物理实现点 $P_e$ 处的有效本征值为 $\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}$（软模）与 $\lambda_2^{\text{eff}} = 59324.3\ \text{rad}^{-2}$（硬模）。在 $\Sigma$ 的坐标 $(\xi, \eta)$ 中，Hessian 矩阵限制为 $2\times 2$ 矩阵：
$$\tilde{H}_{ab} = \frac{\partial^2 V}{\partial x^a \partial x^b}\Big|_{P_e}, \quad x^a = (\xi, \eta)$$

在 $P_e$ 处，$\tilde{H}$ 对角化为 $\text{diag}(\lambda_1^{\text{eff}}, \lambda_2^{\text{eff}})$。（注：0.2.1 裸基准点值为 $\lambda_1^{\text{bare}} = 392.21\ \text{rad}^{-2}$、$\lambda_2^{\text{bare}} = 58760.77\ \text{rad}^{-2}$，经 0.3.1 量纲桥四方程联立自洽修正后得到有效值，偏差由 $v_{\text{geo}}^{\text{alg}}$ 与 $v_{\text{geo}}^{\text{bare}}$ 的 $+0.379\%$ 差异传递而来。裸值与有效值的区分见 0.4 §2.1。）

$\tilde{H}$ 为 GOE(2) 的确定性实现（0.4 定理 2.1）。本征值间距比：
$$r = \frac{2\lambda_1^{\text{eff}}}{\lambda_1^{\text{eff}}+\lambda_2^{\text{eff}}} = 0.01309 \ll 1$$

确认强各向异性极限。

### §2.2　Hessian 度量与图拉普拉斯

$\tilde{H}_{ab}$ 在 $\Sigma$ 上诱导了黎曼度量（Hessian 度量）：
$$g_{ab} = \tilde{H}_{ab}$$

图 $G$ 的构造由 0.3.1 九维谱三元组 $M(a) = S^3 \times S^3 \times S^3$ 的 Cl(9) 旋量丛诱导。顶点集 $V$ 对应全息屏信息单元，其邻接结构由 Dirac 算子 $D_\Sigma$ 的谱截断离散化得到；Hessian 度量 $g_{ab}$ 的连续极限对应 Wodzicki 留数 $\text{Res}_\Sigma(|D_\Sigma|^{-2})$ 在离散顶点上的限制。

在谱图论中，该度量决定了图 $G$ 的边权：
$$w_{ij} = \exp\left(-\frac{d_g(i,j)^2}{2\sigma^2}\right), \quad \sigma^2 = \frac{1}{\lambda_1^{\text{eff}}}$$

图距离 $d_g(i,j)$ 以 0.3.1 量纲桥长度基底 $\chi_L = 1.5092231080 \times 10^{-10}\ \text{m}$ 为单位度量，确保指数无量纲并由唯一几何函子性解码。

图拉普拉斯 $L_G$ 的谱 $\{0 = \mu_0 < \mu_1 \leq \mu_2 \leq \cdots\}$ 与 Hessian 本征值满足标度关系（0.4 §7.1）：
$$\mu_1 \sim N_{\text{info}} \cdot \Gamma_{\text{geo}}$$

其中 $\mu_1$ 为图拉普拉斯第一非零特征值（Fiedler 值，即代数连通度），$N_{\text{info}}$ 为全息屏信息单元数，$\Gamma_{\text{geo}}$ 为特征衰减率。

> **注（编号约定）**：0.4 使用 $\mu_2(L_G)$ 表示 Fiedler 值（"第二小特征值"，因谱以 $\mu_0=0$ 起始）。本文使用 $\mu_1$ 表示同一量（"第一非零特征值"）。数值均为 5.18，仅编号约定不同。

在连续极限 $|V| \to \infty$ 下，$\mu_1$ 与 Laplace-Beltrami 算子 $\Delta_g$ 的第一非零特征值 $\lambda_1(-\Delta_g)$ 通过特征长度 $h = \chi_L / \sqrt{N_{\text{info}}}$ 关联（0.4 定理 3.1）：
$$\mu_1 \approx N_{\text{info}} \cdot h^2 \cdot \lambda_1(-\Delta_g) = \chi_L^2 \cdot \lambda_1(-\Delta_g)$$

该关系在谱图论中严格成立，不依赖任何猜想。$\lambda_1(-\Delta_g)$ 的数值由约束截面 $\Sigma$ 上 Hessian 度量 $g_{ab} = \text{diag}(\lambda_1^{\text{eff}}, \lambda_2^{\text{eff}})$ 通过 Laplace-Beltrami 算子的谱问题确定。

### §2.3　衰减模态作为图特征函数

定义衰减基态 $\varphi_1$ 为图拉普拉斯第一非零特征值 $\mu_1$ 对应的特征函数（Fiedler 向量）：
$$L_G \varphi_1 = \mu_1 \varphi_1$$

该向量的符号变化次数（nodal domain）为 2，与 $\Sigma \cong S^2$ 的拓扑一致。衰减基态对应信息场在约束截面上的最低衰减模态，其特征衰减率：
$$\Gamma_{\text{geo}} = \frac{\mu_1}{N_{\text{info}}}$$

$\varphi_1$ 是信息场衰减的内禀相位模态，对应图 $G$ 的 Fiedler 向量方向。该定义为纯谱分析，不引入任何动力学假设。衰减模态的 nodal domain 标度与 0.3.1 电磁特征长度 $l_{\text{em}} = a_B \cdot S_e = 0.529177\ \text{\AA} \times 137.036 \approx 72.5\ \text{\AA}$ 同阶，确保信息场衰减通过单一核心映射层落地为电磁相互作用的几何读出。

## 第3章　衰减定理的谱图论形式

### §3.1　图热核与衰减率

信息场衰减由图热方程描述：
$$\frac{\partial \rho}{\partial \sigma} = -L_G \rho$$

其解为 $\rho(\sigma) = \exp(-\sigma L_G)\rho_0$。热核（heat kernel）$K_G(\sigma) = \text{Tr}(\exp(-\sigma L_G))$ 的渐近行为决定衰减特性。

**定理 3.1（衰减定理——严格部分）**：对任意初始数据 $\rho_0 \in \ell^2(V)$，图热方程的解满足指数衰减：
$$\|\rho(\sigma) - \rho_{\text{eq}}\|_{\ell^2} \leq C \cdot e^{-\sigma \Gamma_{\text{geo}}} \|\rho_0 - \rho_{\text{eq}}\|_{\ell^2}$$

其中特征衰减率由全息屏图 $G$ 的代数连通度 $\mu_1$ 严格确定：
$$\Gamma_{\text{geo}} = \frac{\mu_1}{N_{\text{info}}}$$

**证明**：由谱图论标准结果，图热核的衰减率由谱间隙 $\mu_1$（代数连通度）决定。$\Gamma_{\text{geo}} = \mu_1/N_{\text{info}}$ 是单位信息顶点的归一化衰减率，在图拉普拉斯半群理论中严格成立（0.4 定理 6.1 保证半群存在唯一性，定理 7.1 给出 $\mu_1$ 与 $\Gamma_{\text{geo}}$ 的关系）。$N_{\text{info}}$ 由 0.2.2 导出。证毕。

**定理 3.1 的数值附加（条件部分）**：在 0.4 猜想 4.2（$C_{\text{geo}} = 1/(16\sqrt{3})$ 组合公式的严格化）和猜想 5.1（$N_{\text{dec}}$ 标度假说）成立前提下，$\mu_1$ 的数值为：
$$\mu_1 = 5.18, \quad \Gamma_{\text{geo}} = \frac{5.18}{9.024 \times 10^{22}} = 5.75 \times 10^{-23}$$

对应编码单元数 $N_{\text{dec}} \equiv 1/\Gamma_{\text{geo}} = 1.74 \times 10^{22}$。

**条件部分推导链**：
$$\mu_1 \xrightarrow{\text{0.4 定理 7.1}} N_{\text{info}} \cdot \Gamma_{\text{geo}} \xrightarrow{\text{定义}} N_{\text{info}} / N_{\text{dec}} \xrightarrow{\text{0.4 猜想 5.1}} N_{\text{info}} / (N_{\text{info}} \cdot (\lambda_2/\lambda_1)^{1/3} \cdot C_{\text{geo}}) = (\lambda_1/\lambda_2)^{1/3} / C_{\text{geo}}$$

代入 $\lambda_2/\lambda_1 = 151.7$，$C_{\text{geo}} = 1/(16\sqrt{3}) \approx 0.03608$：
$$\mu_1 = \frac{(1/151.7)^{1/3}}{0.03608} = \frac{0.1875}{0.03608} = 5.20$$

与 $\mu_1 = 5.18$ 偏差 $<0.4\%$，数值自洽。

> **诚实声明**：$\mu_1 = 5.18$ 和 $\Gamma_{\text{geo}} = 5.75 \times 10^{-23}$ 的精确数值目前依赖于 0.4 的猜想 4.2 和猜想 5.1。定理 3.1 的结构——即图热方程解以 $\Gamma_{\text{geo}} = \mu_1/N_{\text{info}}$ 指数衰减——在图拉普拉斯半群理论中严格成立，不依赖猜想。$\mu_1$ 原则上可由约束截面 $\Sigma$ 的 Hessian 度量 $g_{ab}$ 与全息屏图 $G$ 的离散-连续极限（0.4 定理 3.1）严格计算，该严格计算是当前框架的开放项（见附录 B）。

特征衰减率 $\Gamma_{\text{geo}}$ 与电磁耦合常数 $\alpha = 1/S_e$ 通过全息屏层级递推 $N_n = S_e^2 \Lambda^{2n}$（0.3.1.1 研究框架 2.2）实现标度关联。具体地，$N_{\text{dec}}$ 的构造继承自 0.3.1.1 层级物理的电磁映射自洽性注记（注记 2.5），确保信息场衰减定理在单一核心映射层（几何耦合 $\to$ 电磁相互作用）上与 0.2.3 化学映射（H$_2$ 键能 4.51 eV、键长 0.738 \text{\AA}、CH$_4$ 键角 109.47$^\circ$）自洽。

### §3.2　Cheeger 不等式与代数连通度

图拉普拉斯的代数连通度 $\mu_1$（Fiedler 值）与几何衰减率通过 Cheeger 不等式关联：

**Cheeger 不等式**：
$$\frac{h_G^2}{2} \leq \mu_1 \leq 2h_G$$

其中 $h_G$ 为图的 Cheeger 常数（等周常数），刻画全息屏图的瓶颈。

**定理 3.2**：全息屏图 $G$ 的代数连通度与几何衰减率满足：
$$\Gamma_{\text{geo}} = \frac{\mu_1}{N_{\text{info}}}$$

该关系在谱图论中严格成立。数值上：在 0.4 猜想 4.2 和 5.1 成立前提下，$\mu_1 = 5.18$，$N_{\text{info}} = 9.024 \times 10^{22}$，$\Gamma_{\text{geo}} = 5.75 \times 10^{-23}$。

由 Cheeger 不等式左半 $h_G^2/2 \leq \mu_1$ 得 $h_G \geq \sqrt{2\mu_1} = 3.22$；由右半 $\mu_1 \leq 2h_G$ 得 $h_G \geq \mu_1/2 = 2.59$。二者取较强下界，故 $h_G \geq 3.22$。对于 $N_{\text{info}} \approx 9.024 \times 10^{22}$ 个顶点的全息屏图，该下界反映 Bott 周期截断（$n \leq 7$）导致的层级稀疏性与瓶颈结构，与 0.2.2 全息宇宙 Bott 周期 8 截断一致。

### §3.3　编码单元数与谱累积函数

全息屏编码单元数 $N_{\text{dec}}$ 在 0.4 猜想 4.2 和 5.1 成立前提下取值为：
$$N_{\text{dec}} = N_{\text{info}} \cdot \left( \frac{\lambda_2}{\lambda_1} \right)^{1/3} \cdot C_{\text{geo}} = 1.74 \times 10^{22}$$

$$N_{\text{dec}}/N_{\text{info}} = \left( \frac{\lambda_2}{\lambda_1} \right)^{1/3} \cdot C_{\text{geo}} \approx 5.33 \times 0.03608 = 0.193$$

> **诚实声明**：$N_{\text{dec}}$ 的上述公式在 0.4 中被标记为"猜想 5.1（标度假说）"，其中 $C_{\text{geo}} = 1/(16\sqrt{3})$ 被标记为"猜想 4.2"。在 0.4 附录 C 中，这两个猜想被明确列为开放问题，其严格几何来源是未来研究方向。$C_{\text{geo}}$ 的推导依赖尚未建立的引理 4.2a（球面堆积因子）和引理 4.2b（辛几何因子）；标度指数 $1/3$ 依赖尚未建立的引理 5.2a（热核系数决定谱矩指数）和引理 5.2b（Bott 截断锁定指数）。本文沿用上述数值作为工作假设，不声称其为严格定理输出。

在谱图论中，编码比例 $N_{\text{dec}}/N_{\text{info}} \approx 0.193$ 对应图 $G$ 的谱累积函数在特征截断 $\mu_c$ 处的值。定义谱累积函数：
$$C(\mu) = \frac{1}{N_{\text{info}}} \sum_{k=0}^{N_{\text{info}}-1} \Theta(\mu - \mu_k)$$

则特征截断 $\mu_c$ 满足 $C(\mu_c) = N_{\text{dec}}/N_{\text{info}}$，即恰好 $N_{\text{dec}}$ 个谱模式低于 $\mu_c$。该截断由全息屏有效编码维度内禀确定，与衰减率 $\Gamma_{\text{geo}}$ 通过代数连通度 $\mu_1 = N_{\text{info}} \cdot \Gamma_{\text{geo}}$ 间接关联。该比例为纯几何内禀量。

全息屏信息单元数 $N_{\text{info}} = 9.024 \times 10^{22}$ 由 0.2.2 导出。

## 第4章　结论

（1）在完备性公理约束截面 $\Sigma$ 上建立内禀坐标 $(\xi, \eta)$，六项作用量限制为势函数 $V(\xi, \eta)$，背景不动点 $P_0$ 处 $V = 24$，物理实现点 $P_e$ 处 $V = S_e = 137.035999084$。质量映射 $m(\xi,\eta) = K \sin^3(45^\circ - \xi/2 - \eta/2)$ 在 $\Sigma$ 上显式投影，$P_e$ 同时满足 $V = S_e$ 与 $m = m_e$ 的联立约束，由 0.3.1 谱互锁定理保证局部唯一性。本版修正 $m_e^{\text{geo}}$ 为 $510.98\ \text{keV}$（偏差 $<0.004\%$）。

（2）由 Hessian 度量 $g_{ab} = \tilde{H}_{ab}$ 定义信息场衰减基态 $\varphi_1$ 为图拉普拉斯 Fiedler 向量，对应最低衰减模态。Hessian 本征值采用 0.3.1 量纲桥有效值（$\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}$，$\lambda_2^{\text{eff}} = 59324.3\ \text{rad}^{-2}$）。图 $G$ 由九维谱三元组 $M(a) = S^3 \times S^3 \times S^3$ 的 Cl(9) 旋量丛诱导构造，边权核以 $\chi_L$ 为单位实现量纲桥长度基底的函子性解码。

（3）衰减定理严格部分：图热方程解满足指数衰减，特征衰减率 $\Gamma_{\text{geo}} = \mu_1/N_{\text{info}}$，由图拉普拉斯半群理论与代数连通度严格确立，不依赖猜想。数值部分：$\Gamma_{\text{geo}} = 5.75 \times 10^{-23}$ 与 $\mu_1 = 5.18$ 的精确数值在 0.4 猜想 4.2 和 5.1 成立前提下自洽。$\mu_1$ 原则上可由约束截面 Hessian 度量的 Laplace-Beltrami 谱严格计算，该计算是当前开放项。

（4）全息屏编码单元数 $N_{\text{dec}} = 1.74 \times 10^{22}$ 依赖 0.4 猜想 4.2 和 5.1，编码比例 $N_{\text{dec}}/N_{\text{info}} \approx 0.193$ 为谱累积函数在特征截断处的几何输出。Cheeger 不等式给出 $h_G \geq 3.22$，反映 Bott 周期截断 $n \leq 7$ 的层级瓶颈结构。

### B.2 严格的谱图论核心

以下关系在谱图论框架内严格成立，不依赖任何猜想：

1. **$\Gamma_{\text{geo}} = \mu_1 / N_{\text{info}}$**：图拉普拉斯半群理论（0.4 定理 6.1 存在唯一性）的严格推论。$\mu_1$ 是图 $G$ 的代数连通度，$\Gamma_{\text{geo}}$ 是单位顶点的归一化衰减率。

2. **$\mu_1 \approx \chi_L^2 \cdot \lambda_1(-\Delta_g)$**：离散-连续极限（0.4 定理 3.1）的严格推论，其中 $\lambda_1(-\Delta_g)$ 是约束截面 $\Sigma$ 上 Hessian 度量 $g_{ab}$ 对应的 Laplace-Beltrami 算子的第一非零特征值。

3. **Cheeger 不等式**：$h_G^2/2 \leq \mu_1 \leq 2h_G$（0.4 §7.1），其中 $h_G$ 为图的 Cheeger 常数。

4. **衰减定理结构**：定理 3.1 的指数衰减形式不依赖 $\mu_1$ 的具体数值，仅依赖 $\mu_1 > 0$（全息屏图为连通图，代数连通度严格为正）。
