## 摘要

本文探讨十方几何框架中三代带电轻子质量的刚性关系，对比丘成桐流形（CY）路径与小出（Koide）公式。核心发现：

1. **电子质量在框架内完全自足**——唯一达到"0.0.X + 单一核心映射即足够"标准的粒子。
2. **$W_{12}$ 显式代数形式已建立**（定理 5.2）——联合截面耦合的精确角度函数，零自由参数。
3. **有效投影角几何定义已建立**（定义 5.1）——主扇区 + 独占原则严格区分裸角度与质量映射输入。
4. **缪子-因果角对应**——$m_\mu^{\text{(th)}}(\theta_C^0) = 126$ MeV，偏差 19.6%，方向、量级和扇区归属均正确。
5. **$W_{12}$ 纯二次耦合不能解释 19.6% 偏差**（第7次修订：路径1负结果）——双线性结构在展开点一阶梯度为零，电子被 $\lambda_{\text{eff}}^{(1)} \sim 10^5$ 冻结。
6. **展开基准点修正（路径2）**——$\theta_M^{(2)} = \theta_C^{(2)} = 26.16^\circ$ 是方便的对称点，但缺乏第一性原理理由。层级自洽点 $\theta_C \approx 24.54^\circ$ 提供了候选锚点，但其独立推导需要框架扩展（E1–E3）。

Koide 公式在十方框架中最强的论据是 $Q$ 的 $C_n$ 抵消性质——若比例系数三代普适，$Q$ 是纯角度函数。但在当前 $C_n$ 不普适情形下，$Q = 2/3$ 等价于 $s_n$ 与 $C_n$ 的联合约束条件。

1. **CY 路径无法导出固定粒子代数**——仅提供 Yukawa 耦合的模空间参数化，三代数目来自拓扑 Euler 数而非动力学原理。

## §4 Koide 公式的几何对应

### 4.1 Koide 公式

$$Q = \frac{m_e + m_\mu + m_\tau}{\left(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau}\right)^2} \approx \frac{2}{3}.$$

实验值（PDG 2024）：$m_e = 0.51099895$ MeV, $m_\mu = 105.65837$ MeV, $m_\tau = 1776.86$ MeV, $Q = 0.66666(2)$。

### 4.2 假设 H1 下的 Koide 刚性

在假设 H1 下，$m_n = C_n s_n^3$（$s_n = \sin\theta_{\text{eff}}^{(n)}$）：

$$\boxed{Q = \frac{C_e s_1^3 + C_\mu s_2^3 + C_\tau s_3^3}{\left(\sqrt{C_e} s_1^{3/2} + \sqrt{C_\mu} s_2^{3/2} + \sqrt{C_\tau} s_3^{3/2}\right)^2}.}$$

**(A) $C_n$ 普适情形**：若 $C_e = C_\mu = C_\tau$，$Q$ 退化为纯角度函数 $Q = (s_1^3 + s_2^3 + s_3^3)/(s_1^{3/2} + s_2^{3/2} + s_3^{3/2})^2$。这是 Koide 公式最干净的几何解释——$Q = 2/3$ 成为角度的一个纯代数恒等式。

但 $C_n$ 普适被实验排除（$m_\mu/m_e \gg 1$）且与层级标度框架矛盾。

**(B) $C_n$ 不普适情形（实际情形）**：$Q$ 同时依赖 $s_n$ 和 $C_n$。$Q = 2/3$ 等价于 $(s_n, C_n)$ 的联合约束方程。不是恒等式，而是需要解释的数值巧合。

**(C) $C_n$ 抵消机制**：$Q = 2/3$ 的最强几何论据来自比例系数的抵消。若 $C_n$ 有确定的比值（如 $C_\mu/C_e = \Lambda S_e^{1/2}$, $C_\tau/C_\mu = \Lambda S_e^{1/2}$），则：

$$\frac{Q}{2/3} = f\left(\frac{C_\mu}{C_e}, \frac{C_\tau}{C_\mu}, s_1, s_2, s_3\right).$$

$C_n$ 比值在框架内确定时，$Q$ 退化为角度函数——Koide 刚性从拟合提升为可检验输出。

### 4.3 三角恒等式论证的局限

取 $\theta^{(n)} = \pi/6 - \delta, \pi/6, \pi/6 + \delta$ 且 $\theta^{(1)} + \theta^{(2)} + \theta^{(3)} = 90^\circ$（H2），此时 $Q \approx 2/3 + O(\delta^4)$。

**反例**：$\theta^{(1)} = \theta^{(2)} = \theta^{(3)} = 30^\circ$ 满足 H2，但 $Q = 1/3$，不是 $2/3$。H2（角度和 $= 90^\circ$）是 $Q = 2/3$ 的必要条件，不是充分条件——还需要特定的角度分布。

### 4.4 由实验质量反推的矛盾

在假设 H1 + H2 下，从实验 $m_e, m_\mu, m_\tau$ 和 $C_n = M_n$ 反推：

```text
m_e = K sin³θ^(1)    → θ^(1) = 57.93°（裸基准物质角，自洽）
m_μ = M_1 sin³θ^(2)  → θ^(2) 依赖 M_1 实际值
m_τ = M_2 sin³θ^(3)  → θ^(3) 依赖 M_2 实际值
```

若 H2 成立：
$$\theta^{(2)} + \theta^{(3)} = 90^\circ - 57.93^\circ = 32.07^\circ.$$

$\theta^{(2)}$ 和 $\theta^{(3)}$ 均 $\leq 32.07^\circ$，且若 $\theta^{(2)} \approx 24.54^\circ$（对应 $m_\mu$ 与 $M_1$ 的反推），则 $\theta^{(3)} \approx 7.53^\circ$。

若 $M_n$ 的成长因子为 $\Lambda S_e^{1/2}$ 且三代角度和为 90°，则 Koide 刚性反映了一个深层的几何约束——角度分布与层级标度在约束乘积球面上互锁。但这条链条的每个环节都需要独立验证。

## §5 联合截面耦合 $W_{12}$ 与驻点条件

### 5.1 单粒子 $H^W$ 结构

0.1.1(260626.6) 定理 4.2 给出跨扇区耦合 Hessian：

$$H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix}, \quad a' = 2w + w' = 2.580,\quad b' = 2w - w' = 0.867.$$

其中 $w = 0.862, w' = 0.857$ 为裸 Hessian 矩阵元（0.1 §3.4）。

本征值：$\lambda_+^W = a' + b' = 3.447$, $\lambda_-^W = a' - b' = 1.713$，本征向量 $v_+ = (1,1)$（对称）, $v_- = (1,-1)$（反对称）。

在物理识别点 $\theta^0$ 的切平面坐标 $(\xi, \eta)$ 上：
- $\xi = \theta_I - \theta_C$（信息-因果差）
- $\eta = \theta_C - \theta_M$（因果-物质差）

裸 Hessian 矩阵元：
$$\lambda_2^{\text{bare}} = 58760.77\ \text{rad}^{-2},\quad \lambda_1^{\text{bare}} = 392.21\ \text{rad}^{-2}.$$

### 5.2 $H^W$ 驻点分析：构型 A, B, C

**(A) 构型 A（裸基准点 $\theta^0$）**：$\theta_M = 57.93^\circ, \theta_C = 26.16^\circ, \theta_I = 5.91^\circ$。$\xi = -20.25^\circ, \eta = -31.77^\circ$。$S = S_e = 137.036$。**框架内定理级物理识别点**。

**(B) 构型 B（$\theta_M = \theta_C = 26.16^\circ$）**：$\theta_I = 37.68^\circ$。$\xi = 11.52^\circ, \eta = 0^\circ$。$S = 25.54 \ll S_e$。**数值便利对称点，非物理识别点**。

**(C) 构型 C（$\theta_M = \theta_I = 5.91^\circ$）**：$\theta_C = 78.18^\circ$。$S \gg S_e$，梯度极大。

### 5.3 单粒子 $H^W$ 微扰的灾难性失效（构型 B 处）

在构型 B（$\theta_M = \theta_C = 26.16^\circ$）做 $S$ 的泰勒展开：

$$\nabla_T S|_{\text{B}} \approx (-28.60, 0)\ \text{rad}^{-1}.$$

将 $\nabla_T S$ 投影到 $H^W$ 本征基：

$$\begin{pmatrix} \delta\xi_{H^W} \\ \delta\eta_{H^W} \end{pmatrix} = -\frac{1}{\lambda_-^W} v_- v_-^T \nabla_T S.$$

$$\delta\xi_{H^W} \approx -20.0\ \text{rad} \approx -1146^\circ.$$

**二阶展开完全崩溃**——$\Delta\eta \approx 300^\circ \gg 90^\circ$，超出角度单纯形的拓扑范围。构型 B 不是 $H^W$ 驻点的微扰邻域。

### 5.4 反对称模式-$\theta_C$ 耦合

$$v_- = \frac{1}{\sqrt{2}}\begin{pmatrix} 1 \\ -1 \end{pmatrix}, \quad \xi - \eta = \theta_I + \theta_M - 2\theta_C.$$

在角度和约束 $\theta_I + \theta_M = 90^\circ - \theta_C$ 下：
$$\xi - \eta = 90^\circ - 3\theta_C.$$

**定理 5.1**：$H^W$ 的反对称模式 $\lambda_-^W = 1.713$ 直接耦合 $\theta_C$，回复力 $-3\lambda_-^W \cdot \Delta\theta_C$。$H^W$ 结构天然"关注"因果角。

### 5.5 $d_{AB}$ 数值邻近

构型 A 到 B 的 $\xi$-$\eta$ 距离：$d_{AB} = 1.121\ \text{rad} = 64.24^\circ$。

反对称模式的特征标度：$1/\sqrt{\lambda_-^W} = 1/\sqrt{1.713} = 0.764\ \text{rad}$。在 $v_-$ 方向（$\xi-\eta$）：$d_{AB}$ 的 $v_-$ 投影 $= 1.121 \times (-1)/\sqrt{2} = -0.793\ \text{rad}$。

$$\frac{|d_{AB}^{v_-}|}{1/\sqrt{\lambda_-^W}} = \frac{0.793}{0.764} = 1.038, \quad \text{偏差 3.8\%}.$$

$$\boxed{d_{AB}^{v_-} \approx 1/\sqrt{\lambda_-^W}.}$$

$H^W$ 结构内禀编码了跨代构型间距——这是 $H^W$ 反对称模式最重要的推论之一。

### 5.6 双粒子联合截面耦合 $W_{12}$

#### 5.6.1 23 号文章的结构基础

23 号文章（$\Psi_{\text{joint}}$ 计算）§3.1 给出：双粒子联合截面总 Hessian 在对角基（$\xi^{(1)}, \xi^{(2)}, \eta^{(1)}, \eta^{(2)}$）上分块为 $4 \times 4$ 对称矩阵。在对称/反对称基下，$\xi$ 和 $\eta$ 方向的二阶导数矩阵分别为 $\lambda_+^W I_2$ 和 $\lambda_-^W I_2$。

#### 5.6.2 $W_{12}$ 的显式推导（定理 5.2）

**引理 5.1**（$W_{12}$ 提取）：$H_{W_{12}} = H_{\text{total}}^{(2)} - \operatorname{diag}(H^W, H^W)$。

其中 $H_{\text{total}}^{(2)}$ 是双粒子联合截面总 Hessian 的耦合部分，$\operatorname{diag}(H^W, H^W) = H^W \oplus H^W$ 是块对角单粒子 Hessian。

**引理 5.2**（$W_{12}$ 因子化）：在对称/反对称基下：

$$W_{12}^{(2)} = b' (\delta\xi^{(1)} - \delta\eta^{(2)})(\delta\xi^{(2)} - \delta\eta^{(1)}).$$

**定理 5.2**（$W_{12}$ 角度函数）：

$$\boxed{W_{12}^{(2)} = b' (\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)})(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)})}, \quad b' = 2w - w' = 0.867.$$

放松 M-扇区同步时（引入 $\Delta_M = \Delta\theta_M^{(1)} - \Delta\theta_M^{(2)}$）：

$$W_{12}^{(2)} = b' [(\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)}) + \Delta_M] [(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)}) - \Delta_M].$$

**$W_{12}$ 的结构**：粒子 1 的 $\theta_I$ 与粒子 2 的 $\theta_C$ **交叉配对**，粒子 2 的 $\theta_I$ 与粒子 1 的 $\theta_C$ 配对。乘积形式 $(\theta_I^{(1)} - \theta_C^{(2)})(\theta_I^{(2)} - \theta_C^{(1)})$ 在交换 $1 \leftrightarrow 2$ 下不变。

所有常数来自 $w, w'$（0.1.1），零自由参数。

#### 5.6.3 驻点条件代数化

联合作用量（二阶）：

$$S_{\text{joint}}^{(2)} = S^{(1)} + S^{(2)} + W_{12}^{(2)}.$$

驻点条件 $\delta S_{\text{joint}}^{(2)} = 0$：

$$\begin{cases} \nabla S^{(1)} + \nabla W_{12}^{(1)} = 0, \\ \nabla S^{(2)} + \nabla W_{12}^{(2)} = 0. \end{cases}$$

每个粒子一个梯度方程，$W_{12}$ 通过交叉导数耦合两者。

### 5.10 电子沿 $S=S_e$ 等值面移动的参数化

电子不是固定在 $\theta^0$——它可以在 $S = S_e$ 等值面上自由移动，同时满足角度和约束。

在构型 A 附近，$S$ 的梯度在 $(\xi^{(1)}, \eta^{(1)})$ 坐标中为：

$$\nabla S = \begin{pmatrix} \partial S/\partial\xi \\ \partial S/\partial\eta \end{pmatrix}.$$

等值面 $S = S_e$ 的切线方向 $v_T$ 满足 $v_T \cdot \nabla S = 0$：

$$\boxed{\delta\eta^{(1)} = -\alpha\,\delta\xi^{(1)},\quad \alpha = \frac{\partial S/\partial\xi}{\partial S/\partial\eta} \approx 23.38.}$$

在 $\theta^0$ 处直接计算：$\partial S/\partial\xi \approx 23163$, $\partial S/\partial\eta \approx -991$，比值 $\alpha = 23.38$。

电子在 $S=S_e$ 等值面上的位移 $\delta\xi^{(1)}$ 是路径 1 中唯一的电子自由度。

**电子角度参数化**（从 $\delta\xi^{(1)}$ 和 $\delta\eta^{(1)} = -23.38\,\delta\xi^{(1)}$ 加上角度和约束）：

$$\begin{aligned}
\Delta\theta_C^{(1)} &= -8.130\,\delta\xi^{(1)}, \\
\Delta\theta_M^{(1)} &= +15.262\,\delta\xi^{(1)}, \\
\Delta\theta_I^{(1)} &= -7.130\,\delta\xi^{(1)}.
\end{aligned}$$

**验证**：$\Delta\theta_M^{(1)} + \Delta\theta_C^{(1)} + \Delta\theta_I^{(1)} = (15.262 - 8.130 - 7.130)\delta\xi^{(1)} = 0$ ✓.

### 5.11 电子-缪子联合作用量（二阶完整展开）

$$\begin{aligned}
S_{\text{joint}}^{(2)} &= S_e + \frac{1}{2}\lambda_{\text{eff}}^{(1)}(\delta\xi^{(1)})^2 \quad (\text{电子沿 } S=S_e \text{ 等值面的代价})\\
&\quad + g_\xi^{(2)}\delta\xi^{(2)} + g_\eta^{(2)}\delta\eta^{(2)} \quad (\text{缪子 } S \text{ 梯度})\\
&\quad + \frac{1}{2}\lambda_2^{\text{bare}}(\delta\xi^{(2)})^2 + \frac{1}{2}\lambda_1^{\text{bare}}(\delta\eta^{(2)})^2 \quad (\text{缪子裸 Hessian})\\
&\quad + W_{12}(\delta\xi^{(1)}, \delta\xi^{(2)}, \delta\eta^{(2)}). \quad (\text{定理 5.2 放松形式})
\end{aligned}$$

**$\lambda_{\text{eff}}^{(1)}$ 的计算**（电子沿 $S=S_e$ 等值面的有效刚度）：

$\delta S^{(1)} = \frac{1}{2}\lambda_2^{\text{bare}}(\delta\xi^{(1)})^2 + \frac{1}{2}\lambda_1^{\text{bare}}(\delta\eta^{(1)})^2$，代入 $\delta\eta^{(1)} = -23.38\,\delta\xi^{(1)}$：

$$\lambda_{\text{eff}}^{(1)} = \lambda_2^{\text{bare}} + (23.38)^2\lambda_1^{\text{bare}} = 58760.77 + 546.6 \times 392.21 = 273166\ \text{rad}^{-2}.$$

**$\lambda_{\text{eff}}^{(1)} = 2.73 \times 10^5$**——电子被极强地锁定在 $S=S_e$ 等值面上。

### 5.12 $W_{12}$ 完整形式（电子自由 + 缪子自由）

从定理 5.2 放松形式，代入电子参数化（§5.10）和缪子 $(\delta\xi^{(2)}, \delta\eta^{(2)})$：

$$\boxed{W_{12} = b' \left[8.132\,\delta\xi^{(1)} + \frac{2\delta\xi^{(2)}+\delta\eta^{(2)}}{3}\right] \left[\frac{\delta\xi^{(2)}-\delta\eta^{(2)}}{3} - 7.132\,\delta\xi^{(1)}\right].}$$

其中缪子角度展开使用了：

### 5.13 $W_{12}$ 混合偏导数——双线性结构的致命性质

**定理 5.4**（$W_{12}$ 在展开点的混合偏导数）：在展开基准点 $(\delta\xi^{(1)} = \delta\xi^{(2)} = \delta\eta^{(2)} = 0)$：

$$\frac{\partial W_{12}}{\partial\delta\xi^{(1)}} = \frac{\partial W_{12}}{\partial\delta\xi^{(2)}} = \frac{\partial W_{12}}{\partial\delta\eta^{(2)}} = 0.$$

**证明**：$W_{12} \propto F_1 \cdot F_2$，其中 $F_1(0) = 8.132 \times 0 + 0 = 0$, $F_2(0) = 0 - 7.132 \times 0 = 0$。因此 $\partial W_{12}/\partial x = (\partial F_1/\partial x)F_2 + F_1(\partial F_2/\partial x)$，在 $(0,0,0)$ 处 $F_1 = F_2 = 0$，故所有一阶偏导数为零。$\square$

**物理含义**：$W_{12}$ 是纯二阶效应。在展开点无线性驱动力。电子和缪子在展开点通过 $W_{12}$ 完全解耦。

### 5.17 路径 2——展开基准点修正

#### 5.17.1 动机

路径 1 的负结果揭示了一个根本问题：**展开基准点 $\theta_M^{(2)} = \theta_C^{(2)} = 26.16^\circ$ 是方便的对称点（$\eta = 0$），但没有第一性原理的物理理由。** 缪子裸角度可能与 $\theta_C^0 = 26.16^\circ$ 不同——26.16° 是电子物理识别点的因果角，由 $S=S_e$ 锁定，没有理由认为缪子的裸因果角恰好等于此值。

路径 2 提出：**将 $\theta_M^{(2)}$ 和 $\theta_C^{(2)}$ 作为待定参数，寻找由层级约束截面确定的正确展开基准点。**

#### 5.17.2 层级自洽点

由层级质量映射（假设 H1）和实验值 $m_\mu \approx 105.66$ MeV：

$$\sin^3\theta_C^{(2)} = m_\mu / C_\mu.$$

$C_\mu$ 的框架候选值取决于 0.3.1.1 的精确解释。若 $C_\mu = M_1 = K \Lambda S_e^{1/2}$：

- 若 $\Lambda = 3$（框架锁定值）：$M_1 = 29.5$ MeV → $\sin^3\theta_C^{(2)} = 105.66/29.5 = 3.58$，超出实数域 ❌
- 若 $\Lambda_{\text{eff}} = 150$（0.3.1.1 识别值）：$M_1 \approx 1.47$ GeV → $\sin^3\theta_C^{(2)} = 0.0721 \to \theta_C^{(2)} \approx 24.54^\circ$ ✅

#### 5.17.3 对称自洽点的 $S$ 梯度

在对称自洽点 $\theta_M^{(2)} = \theta_C^{(2)} = 24.54^\circ$, $\theta_I^{(2)} = 40.92^\circ$：

$$\sin\theta_M = \sin 24.54^\circ = 0.4155,\quad \cos\theta_M = 0.9096,$$
$$\sin\theta_I = \sin 40.92^\circ = 0.6549,\quad \cos\theta_I = 0.7557.$$

$$\begin{aligned}
\frac{\partial S}{\partial\theta_M} &= -\frac{2\cos\theta_M}{\sin^3\theta_M} - \frac{\cos\theta_M}{\sin^2\theta_M\sin\theta_I} + \frac{2\cos\theta_I}{\sin\theta_M\sin^2\theta_I} \\
&= -2 \times 12.68 - 8.04 + 8.48 = -24.92\ \text{rad}^{-1}.
\end{aligned}$$

在 $(\xi, \eta)$ 坐标下，对称点 $g_\eta = 0$（由对称性），$g_\xi$ 计算：

$$g_\xi = \frac{\partial S}{\partial\theta_I} - \frac{\partial S}{\partial\theta_C} = \frac{\partial S}{\partial\theta_I} - \frac{\partial S}{\partial\theta_M}.$$

$$\frac{\partial S}{\partial\theta_I} = -\frac{2\cos\theta_I}{\sin^3\theta_I} - \frac{2\cos\theta_I}{\sin^2\theta_I\sin\theta_M} + \frac{2\cos\theta_M}{\sin\theta_I\sin^2\theta_M}.$$

数值代入后：
$$\frac{\partial S}{\partial\theta_I} \approx -5.38 - 8.66 + 26.41 = 12.37\ \text{rad}^{-1}.$$

$$\boxed{g_\xi(24.54^\circ) = 12.37 - (-24.92) = 37.29\ \text{rad}^{-1}.}$$

| 展开基准点 | $\theta_M = \theta_C$ | $S$ | $g_\xi$ | $g_\eta$ |
|:---|:---|:---|:---|:---|
| 对称便利点 | 26.16° | 25.54 | 28.60 | 0 |
| 层级自洽点 | 24.54° | 27.06 | **37.29** | 0 |
| 背景极小点 | 30.00° | 24.00 | 0 | 0 |

**关键观察**：层级自洽点的 $g_\xi$（37.29）比对称便利点（28.60）大 30%。更大的梯度意味着 S 对缪子的驱动力更强——但方向仍然是将 $\xi$（从而 $\theta_C$）推向更小的值。

## §6 $C_n$ 不普适条件下的 Koide 刚性

### 6.1 Koide 公式回顾

$$Q = \frac{m_e + m_\mu + m_\tau}{\left(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau}\right)^2} \approx \frac{2}{3}.$$

### 6.2 $C_n$ 普适情形

若 $C_n$ 三代普适（$C_e = C_\mu = C_\tau = C$），则 $Q$ 退化为纯角度函数：

$$Q = \frac{s_1^3 + s_2^3 + s_3^3}{\left(s_1^{3/2} + s_2^{3/2} + s_3^{3/2}\right)^2}, \quad s_n = \sin\theta_{\text{eff}}^{(n)}.$$

### 6.3 $C_n$ 不普适情形（当前框架实际情形）

在 H1 下：

$$Q = \frac{K s_1^3 + C_\mu s_2^3 + C_\tau s_3^3}{\left(\sqrt{K} s_1^{3/2} + \sqrt{C_\mu} s_2^{3/2} + \sqrt{C_\tau} s_3^{3/2}\right)^2}.$$

$Q = 2/3$ 等价于 $s_n$ 与 $C_n$ 的联合条件——P0 级开放问题。

### 6.4 $C_n$ 抵消机制
