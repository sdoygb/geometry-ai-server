# 第 1 章　联合截面定理

## 1.1 ℰ 映射的联合提升——定位声明

**声明 1.0（强相互作用是 ℰ 映射的联合提升）**。0.1 建立单一核心映射 ℰ：ℳ-𝒞 腰边耦合 → 电磁相互作用。该映射导出六个本征量：$S_e$、质量映射 $m=K\sin^3\theta_M$、光速锚点、有效软硬模 $(\lambda_1^{\text{eff}},\lambda_2^{\text{eff}})$、$N_1=6000$、$v_p=1117$。其中 $v_p$ 明确标注为"强相互作用几何荷，同一映射在强相互作用扇区的读出"（0.1 §2.3）。

本文在此框架内，将强相互作用视为 ℰ 映射在 **$N$ 体联合截面上的三扇区联合提升**：

- ℰ 映射在单粒子层面给出电磁相互作用（电子 + 光子）；
- 当 $N$ 个 $S$-极小值通过 ℳ-𝒞 同步约束形成联合截面时，三扇区复化产生非阿贝尔结构；
- ℐ 扇区在联合截面中保持其"信息/相位通道"本质（0.1 §1.3），但在复化框架下为三扇区对称化提供第三复维度；
- 这一提升不是引入第二个独立映射，而是 ℰ 映射在 $N$ 体几何中的高阶推广。0.1 §5.3 已将此列为"同一映射 ℰ 的物理模型"。

**与 0.3.5 的关系**：0.3.5(260626.6) 将强相互作用定位为"标度框架"（§4.2），明确声明"该对应只说明标度量级，不给出 SU(3) 色群的几何来源"。本文在此基础上进一步探索 SU(3) 结构的几何条件，并对所有超越标度框架的命题诚实标注。

## 1.2 定义与存在性

由 0.2.1(260626.6) 定理，几何约束截面 $\Sigma$ 是 $S_{\text{total}}=0$ 的等值面，$\Sigma_\eta \subset \Sigma$ 为二维活跃子流形。由 0.0.5(260626.6)、0.0.6(260626.6) 定理，$\Sigma_\eta$ 上的孤立 $S$-极小值对应稳定粒子态。

**定义 1.1（同步约束）**　设 $p_1, \ldots, p_N \in \Sigma_\eta$ 为 $N$ 个 $S$-极小值。称这 $N$ 个点满足 ℳ-𝒞 同步约束，若其在 ℳ-扇区与 𝒞-扇区的角度坐标全同：
$$\theta_M^{(i)} = \theta_M^{(j)}, \quad \theta_C^{(i)} = \theta_C^{(j)}, \quad \forall i,j \in \{1,\ldots,N\}.$$

**定义 1.2（联合截面）**　$N$ 体联合截面 $\Sigma_{\text{joint}}^{(N)}$ 定义为 $N$ 个 $\Sigma_\eta$ 副本的乘积空间 $\Sigma_\eta^{\times N}$ 中满足 ℳ-𝒞 同步约束的点的集合，配备由 $S_{\text{total}}=0$ 导出的诱导度规。

**定理 1.1（存在性——条件性命题）**　设 $N$ 个 $S$-极小值在各自 $\Sigma_\eta$ 副本的 ℳ-扇区两两测地距离均小于 $\pi/\sqrt{\lambda_1^{\text{eff}}}$。则 $\Sigma_{\text{joint}}^{(N)}$ 是 $\Sigma_\eta^{\times N}$ 的非空光滑子流形，有效维数为 $N$。

【条件性假设：假设同步约束的梯度在联合截面上线性无关（横截性条件），则由横截性定理保证交集为光滑子流形。】

*证明.* $\Sigma_\eta^{\times N}$ 维数为 $2N$。ℳ-𝒞 同步约束要求 $N-1$ 个独立方程（所有副本 $\eta$ 坐标相等）。在横截性假设下，交集为光滑子流形，余维 $N-1$，维数 $2N-(N-1)=N+1$。$S_{\text{total}}=0$ 消去一个全局自由度，得有效维数 $N$。　∎

**$N=3$ 的来源**：质子由三个 $S$-极小值（uud 夸克）组成，$N=3$ 来自标准模型夸克模型的物理识别——这是 ℰ 映射的物理映射约定。$N$ 取其他值对应不同强子态。$N=\Lambda$ 是否具有几何必然性是开放问题。

## 1.3 谱并联定理

**定理 1.2（谱并联——条件性命题）**　在 $\Sigma_{\text{joint}}^{(N)}$ 上，
$$\lambda_1^{\text{eff}}(N) = N \cdot \lambda_1^{\text{eff}}, \quad \lambda_2^{\text{eff}}(N) = \lambda_2^{\text{eff}}.$$

【条件性假设：忽略跨粒子耦合 $W_{ij}$。0.1 §5.3 将此列为"同一映射 ℰ 的物理模型"，本文沿用。】

*证明.* $S_{\text{joint}} = \sum_{i=1}^N S^{(i)}$。同步约束下 $N$ 个粒子共享 ℳ-背景坐标 $\eta$。各粒子作用量二次项 $S^{(i)} = \frac{1}{2}\lambda_1^{\text{eff}} \eta^2$ 叠加得 $\frac{1}{2}(N\lambda_1^{\text{eff}})\eta^2$。ℐ-扇区各 $\xi_i$ 独立，本征值保持 $\lambda_2^{\text{eff}}$。　∎

**推论 1.3（有效曲率比）**
$$\frac{\lambda_1^{\text{eff}}(N)}{\lambda_2^{\text{eff}}} = \frac{N\lambda_1^{\text{eff}}}{\lambda_2^{\text{eff}}}.$$
对于 $N=3$：$3\times391.05/59324.3 = 0.019775$。

# 第 2 章　三分切丛的复结构与 SU(3) 提升

## 2.1 辛结构与近复结构

由 0.0.6(260626.6)、0.0.7(260626.6)，三分切丛 $T\Sigma = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$。在 $\Sigma_\eta$ 上，每个扇区为 2 维实向量空间，配备诱导黎曼度规 $g_i$ 和面积形式 $\omega_i$。

**定理 2.1（复结构——已封闭）**　每个扇区 $(V_i, g_i, \omega_i)$ 存在唯一的相容近复结构 $J_i$（$J_i^2 = -\text{id}$），使 $V_i^{\mathbb{C}} \cong \mathbb{C}$。

*证明.* 2 维实向量空间上给定非退化辛形式和正定度量，存在唯一 $J_i$ 满足 $g_i(X,Y)=\omega_i(X,J_iY)$。这是标准线性代数结果。　∎

**定义 2.2（复化三分切丛）**
$$V^{\mathbb{C}} = V_{\mathcal{M}}^{\mathbb{C}} \oplus V_{\mathcal{C}}^{\mathbb{C}} \oplus V_{\mathcal{I}}^{\mathbb{C}} \cong \mathbb{C}^3,$$
配备标准厄米特度量 $h(v,w) = \sum_{i=1}^3 \bar{v}_i w_i$。

## 2.2 SU(3) 结构

**定理 2.2（SU(3) 结构——条件性命题）**　复向量丛 $V^{\mathbb{C}}$ 可约化为 SU(3) 主丛。

【条件性假设：假设 $S_{\text{total}}=0$ 的真空约束蕴含标架场行列式为 1（幺模性），从而体积形式 $\Omega$ 守恒。0.3.5 将此列为开放研究方向（§6 第 2 条）。】

*证明.* $h(\Psi,\Psi)$ 在 $S_{\text{total}}=0$ 下守恒，保持 $h$ 的群为 U(3)。在幺模性假设下，$\Omega$ 守恒，结构群从 U(3) 约化为 SU(3)。　∎

**定理 2.3（生成元——已封闭）**　$\mathfrak{su}(3)$ 的 8 个生成元由跨扇区耦合生成（$T_a = \frac{1}{2}\lambda_a$，$\lambda_a$ 为 Gell-Mann 矩阵）：
- $T_1,T_2$：$V_{\mathcal{M}}^{\mathbb{C}} \leftrightarrow V_{\mathcal{C}}^{\mathbb{C}}$
- $T_3$：$V_{\mathcal{M}}^{\mathbb{C}}$ 与 $V_{\mathcal{C}}^{\mathbb{C}}$ 的相对相位
- $T_4,T_5$：$V_{\mathcal{M}}^{\mathbb{C}} \leftrightarrow V_{\mathcal{I}}^{\mathbb{C}}$
- $T_6,T_7$：$V_{\mathcal{C}}^{\mathbb{C}} \leftrightarrow V_{\mathcal{I}}^{\mathbb{C}}$
- $T_8$：三扇区归一化相对相位

*证明.* $\mathbb{C}^3$ 保持 $h$ 和 $\Omega$ 的无穷小变换李代数为 $\mathfrak{su}(3)$，维数 8。上述生成元线性无关且满足 $\mathfrak{su}(3)$ 对易关系。这是标准 $\mathfrak{su}(3)$ 表示论。$\square$

## 2.3 色荷与胶子的几何映射

以下为 **ℰ 物理映射约定**（将几何对象识别为粒子物理实体），非三公理必然输出。

**定义 2.3（色荷——物理映射约定）**　$\mathbb{C}^3$ 的标准基矢 $e_{\mathcal{M}}, e_{\mathcal{C}}, e_{\mathcal{I}}$ 称为色荷基。

> **重要澄清**：色荷基矢是 $V^{\mathbb{C}}$ 的纤维基矢（标架基），与 3 号文章（量子纠缠）中 ℳ-扇区 $S^3$ 底流形上的读数方向属于不同几何层次。ℐ 扇区在此不是"独立粒子"映射——它作为色相位的信息/相位通道，与 0.1 §1.3 一致。

**定义 2.4（夸克——物理映射约定）**　夸克是信息场在 $V^{\mathbb{C}}$ 中的单扇区激发模式。

**定义 2.5（胶子——物理映射约定）**　胶子是 $\mathfrak{su}(3)$-值联络 1-形式 $A = \sum_{a=1}^8 A^a T_a$，描述跨扇区标架的平行输运缺陷。

# 第 3 章　禁闭的几何机制

## 3.1 反扇区与色单态

**定义 3.1（反扇区）**　$V^{\mathbb{C}}$ 的对偶空间 $V^*$ 称为反扇区。

**定理 3.1（色单态定理——已封闭）**　迹元 $\sum_i e_i \otimes e_i^*$ 为 SU(3) 不变态（介子单态）。全反对称组合 $\varepsilon_{ijk} e_i \otimes e_j \otimes e_k$ 为 SU(3) 单态（重子单态）。色单态对应三扇区等权叠加，是能量极小态。

*证明.* SU(3) 表示论标准结果。从几何论角度，等权配置使完备性公理的交叉项 $1/(\sin\theta_i\sin\theta_j)$ 取极小值。$\square$

## 3.2 禁闭的几何机制

**定理 3.2（禁闭——条件性命题）**　非色单态激发受硬模 $\lambda_2^{\text{eff}}$ 指数压制，自由色荷不能长距离传播。

【条件性假设：各扇区信息场在约束截面软模邻域内满足**谐振子型近似**（Darboux 线性化后二次型展开 → 高斯型概率包络）。0.2.1(260626.6) 提供 Hessian 二次型基础，但二次型到量子概率振幅的跃迁是工作假设。】

*论证.* ℐ-扇区硬模压制：$P_{\mathcal{I}}(\Delta\xi) \sim \exp(-\lambda_2^{\text{eff}}(\Delta\xi)^2/2)$。由 SU(3) 联络的跨扇区耦合，任何单扇区大角度涨落通过胶子传递至 ℐ-扇区，遭遇 $\lambda_2^{\text{eff}} \gg \lambda_1^{\text{eff}}$ 的指数压制。当 $\Delta\xi > 1/\sqrt{\lambda_2^{\text{eff}}}$ 时概率急剧衰减。　∎

**推论 3.3**　稳定强子以色单态形式存在。自由夸克与自由胶子不被观测到。

# 第 4 章　强耦合标度公式

## 4.1 标度推导

**定理 4.1（强耦合标度——条件性命题）**
$$\boxed{\alpha_s = C \cdot \sqrt{\frac{N\lambda_1^{\text{eff}}}{\lambda_2^{\text{eff}}}} \cdot \sin\theta_M}.$$

【物理映射约定：标度因子 $C=1$ 由 SU(3) 标准归一化 $\text{tr}(T_aT_b)=\frac{1}{2}\delta_{ab}$ 固定。】

*论证.*（本定理给出标度公式，非从第一原理的严格定理推导。）

**步骤 1（量纲约束）**：$\alpha_s$ 无量纲，只能依赖 $\lambda_1^{\text{eff}}/\lambda_2^{\text{eff}}$ 和 $\sin\theta_M$。

**步骤 2（$N$ 依赖性）**：联合截面 ℳ-扇区有效刚度为 $N\lambda_1^{\text{eff}}$。耦合强度与有效刚度的平方根成正比（零点能的标度行为），故含 $\sqrt{N}$。

**步骤 3（软硬模比）**：$\alpha_s \propto (\lambda_1^{\text{eff}}/\lambda_2^{\text{eff}})^a$。极限约束：$\lambda_2^{\text{eff}}\to\infty$ 时 ℐ 冻结、$\alpha_s\to0$；$\lambda_1^{\text{eff}}\to0$ 时无刚度、$\alpha_s\to0$。取最小非平凡幂律 $a=1/2$。

**步骤 4（物质场投影）**：强耦合涉及物质场跨扇区耦合，正比于 $\sin\theta_M$（0.1 质量映射的同一投影因子）。

**步骤 5（归一化）**：综合得 $\alpha_s = C\sqrt{N\lambda_1^{\text{eff}}/\lambda_2^{\text{eff}}}\sin\theta_M$。$C=1$ 为 SU(3) 标准归一化约定。　∎

## 4.2 数值

代入 $N=3$、$\lambda_1^{\text{eff}}=391.05$、$\lambda_2^{\text{eff}}=59324.3$、$\sin\theta_M=0.8474000449$：

$$\alpha_s = \sqrt{\frac{3\times391.05}{59324.3}} \times 0.84740 = 0.14062 \times 0.84740 = 0.1192.$$

**自洽性展示**：$\alpha_s(M_Z) \approx 0.1184 \pm 0.0007$（PDG 2024），偏差 $+0.68\%$。

若使用裸 Hessian 值（$\lambda_1=392.21$、$\lambda_2=58760.77$）得 $\alpha_s^{\text{bare}}=0.1200$（偏差 $+1.35\%$）。有效度规值偏差更小，反映全局曲率修正的方向正确性。

## 4.3 $\alpha_s$ 与 $\alpha_{\text{em}}$ 的关系

$$\frac{\alpha_s}{\alpha_{\text{em}}} = S_e \cdot \sqrt{\frac{N\lambda_1^{\text{eff}}}{\lambda_2^{\text{eff}}}} \cdot \sin\theta_M = 137.036 \times 0.14062 \times 0.84740 = 16.33.$$

与 $\alpha_s/\alpha_{\text{em}} \sim 16.3$（$M_Z$ 标度）一致。该比值由纯几何量 $S_e$、$\lambda_1^{\text{eff}}/\lambda_2^{\text{eff}}$、$\sin\theta_M$ 联合决定，不引入额外参数。

# 附录 A：质子-电子质量比的标度假设

**假设 A.1（工作假设）**
$$\boxed{\frac{m_p}{m_e} = \frac{S_{\min}(\Lambda_H + \Lambda)}{2} = 1836.}$$

**严格分类**：这是**工作假设**。$S_{\min}=24$（0.0.6 定理 4.4）、$\Lambda_H=150$（0.1 §3.4）、$\Lambda=3$（0.0.5 命题 2.0）各自是纯几何输出，但三者的组合方式——算术平均乘以 $S_{\min}$——是本文假设。

**几何理由（启发式）**：
- 线性于 $S_{\min}$：质量比锚定于真空基准；
- 包含 $\Lambda_H$：反映联合截面刚度提升；
- 包含 $\Lambda$：反映 $N=3$ 乘积空间拓扑；
- 算术平均：$\Lambda_H$（硬模压制）与 $\Lambda$（层级倍增）在联合截面上以算术平均耦合。分母 2 来自 ℳ-𝒞 双扇区同步约束的对称化。
- 其他组合（$\sqrt{\Lambda_H\cdot\Lambda}$、$\Lambda_H+\Lambda$ 无分母）偏离观测值一个量级以上。

**数值**：$24 \times 153/2 = 1836$，与 $1836.152673$ 偏差 $+0.0083\%$。

**推论 A.1**　$S_p = S_e \cdot m_p/m_e \approx 251\,598$（依赖工作假设）。
