## 摘　要

本文在《十方几何空间的数学理论》（0.X.X系列，260626.6）所建立的公理化框架下，对比丘成桐（Yau）的 Calabi-Yau 紧致化路径与小出义夫（Koide）的三代轻子质量公式，探讨三代带电轻子（电子、缪子、陶子）的质量关系是否可以从约束流形的几何结构获得解释。

**数值交叉验证的核心发现**：

- **电子**：C_e = K = 839.76 keV，θ^(1) = θ_M^0 = 57.93°（0.1 已锁定）→ m_e = 511 keV。框架内自足。
- **缪子**：若取 C_μ = M_1 = 1.47 GeV 且 θ^(2) 由跨扇区 Hessian 的 (ξ,η) 驻点条件确定，则 θ^(2) 的候选范围为 24°–28°。**该范围的上界（约 28°）与裸基准因果角 θ_C^0 = 26.16° 的量级和扇区归属一致**——这为"缪子对应因果扇区主导的物理识别"提供了框架内数值线索。在 θ^(2) = 26.16° 时，m_μ^(th) = M_1 sin³ θ_C^0 ≈ 126 MeV，与实验值 105.658 MeV 偏差约 19%。该偏差可能来自 (ξ,η) 驻点的非微扰修正，方向与量级合理。
- **陶子**：m_τ = 1777 MeV。若取 C_τ = M_1，要求 sin³θ^(3) > 1，在实数角度几何中不可能。当前最优候选为 C_τ = M_2 = 221 GeV，此时 sin³θ^(3) ≈ 0.0080 → θ^(3) ≈ 11.5°。该角度的几何起源（与 θ_I^0 = 5.91° 的关系）待澄清。

## 第 1 章　引言：质量的两种几何化与十方路径

### 1.1 丘成桐路径：从模空间到 Yukawa 耦合

丘成桐（Yau）证明的 Calabi-Yau 定理为弦理论提供了紧致化背景。在 10 维超弦框架下，4 维有效理论的 Yukawa 耦合由 CY 三维fold 上的三形式积分给出：

$$Y_{ijk}=\int_{CY}\Omega\wedge\omega_i\wedge\omega_j\wedge\omega_k$$

带电轻子质量通过 Higgs 机制获得：

$$m_n = y_n v,\quad y_n\propto Y_{njk}$$

其中 v ≈ 246 GeV 为 Higgs 真空期望值。

**该路径的根本困境**：Yᵢⱼₖ 是 CY 复结构模的函数。三代轻子对应三个独立的 Yukawa 耦合 yₑ, y_μ, y_τ，它们在模空间上无代数关联。你可以通过精细调节复结构模，使 yₑ : y_μ : y_τ 碰巧接近实验值，但没有任何几何原理强制其满足 Koide 关系。质量是模的函数，而非几何的刚性输出。

### 1.2 小出路径：精确的唯象巧合

小出义夫（Koide, 1981）发现带电轻子质量满足：

$$Q=\frac{m_e+m_\mu+m_\tau}{(\sqrt{m_e}+\sqrt{m_\mu}+\sqrt{m_\tau})^2}\approx\frac{2}{3}$$

代入实验值（PDG 2024）：

- mₑ = 0.51099895 MeV/c²
- m_μ = 105.6583745 MeV/c²
- m_τ = 1776.86 MeV/c²

得 Q = 0.666659 ± 0.00001，与 2/3 的偏差小于 10⁻⁵。

四十年来，标准模型、大统一理论（GUT）及弦理论均无法从第一性原理导出此关系。Koide 本人将其视为经验公式；后续研究者尝试将其归因于离散味对称性（如 A₄, S₃）或弦理论额外维的拓扑约束，但均无法自然解释 2/3 的精确数值刚性。

### 1.3 十方路径：层级约束截面上的三层物理识别

本文基于《十方几何空间的数学理论》（0.X.X 系列，260626.6）、《几何动力学》（0.1, 260626.6）及《层级约束截面与多尺度标度理论》（0.3.1.1, 260626.6）建立的公理化框架，提出以下研究路线：

## 第 2 章　Koide 公式的代数结构

### 2.1 公式重写

令 xₙ = √mₙ，Koide 公式等价于：

$$(x_1+x_2+x_3)^2=\frac{3}{2}(x_1^2+x_2^2+x_3^2)$$

展开得：

$$x_1^2+x_2^2+x_3^2+2(x_1x_2+x_2x_3+x_3x_1)=\frac{3}{2}(x_1^2+x_2^2+x_3^2)$$

即：

$$4(x_1x_2+x_2x_3+x_3x_1)=x_1^2+x_2^2+x_3^2$$

或等价地，向量 (x₁, x₂, x₃) 与全 1 向量 (1, 1, 1) 的夹角满足：

$$\cos^2\phi=\frac{(x_1+x_2+x_3)^2}{3(x_1^2+x_2^2+x_3^2)}=\frac{2}{3}$$

因此 cos φ = √(2/3)，φ ≈ 35.26°。

**关键观察**：35.26° 恰好是三维空间中线与坐标轴的等角投影角的补角相关值（arccos(1/√3) ≈ 54.74°，其补角为 35.26°）。这与十方几何中三分切丛 ℳ ⊕ 𝒞 ⊕ ℐ 的投影几何存在深层对应，暗示 Koide 公式可能具有几何根源。

### 2.2 标准模型的参数困境

在标准模型中，三代轻子质量是三个独立的拉格朗日参数：

$$\mathcal{L}_{\text{Yukawa}}=-y_e\bar{e}_L e_R H-y_\mu\bar{\mu}_L\mu_R H-y_\tau\bar{\tau}_L\tau_R H+\text{h.c.}$$

yₑ, y_μ, y_τ 通过实验测定，彼此之间无对称性约束。标准模型的规范群 SU(3)꜀ × SU(2)ₗ × U(1)ᵧ 不对味空间作任何限制。因此 Koide 关系在标准模型中是偶然。

### 2.3 CY 紧致化的模依赖

在 IIB 型弦理论的 CY 紧致化中，Yukawa 耦合由复结构模 τⁱ 和 Kähler 模 tᵃ 共同决定：

$$Y_{ijk}(\tau)=\int_{CY}\Omega(\tau)\wedge\omega_i\wedge\omega_j\wedge\omega_k$$

虽然特定 CY 的拓扑可以限制 Yukawa 耦合的结构（如零模式、对称性），但三个独立耦合的比值仍然是复结构模的函数。你可以在模空间上找到一个点使 Q ≈ 2/3，也可以在另一个点使 Q ≈ 0.5 或任何其他值。CY 几何不锁定 Koide 关系。

### 3.4 三代角度的约束结构

> **假设 H2（三代角度和约束——猜想）**：三个有效投影角满足
> $$\theta^{(1)}+\theta^{(2)}+\theta^{(3)}=90^\circ.$$

**诚实标注**：

1. **从扇区约束到三代粒子角度约束的推广**：公理 1（θ_M + θ_C + θ_I = 90°）是同一物理识别点的三个扇区投影角之间的约束。将其推广为三个不同粒子的有效投影角之和——这一步骤在现行框架中缺乏严格论证。

2. **层级框架下的重新诠释**：在 0.3.1.1 的层级嵌套结构中，每一层级的物理识别点有自己的 (θ_M^(n), θ_C^(n), θ_I^(n)) 三元组。θ^(n) 指代哪个扇区角取决于该代轻子的主导扇区归属。假设 H2 在不同代间做角度和约束，需要指定各代 θ^(n) 的扇区对应规则——该规则尚未建立。

3. **本文的处理**：假设 H2 作为概念约束保留（继承公理 1 的完备性精神），但 §3.5 的数值探索不依赖 H2——角度由层级质量标度和跨扇区 Hessian 的驻点条件独立确定。

### 4.2 缪子（n = μ）：层级质量标度 M_1 与因果角候选

**（A）质量标度候选**

取 C_μ = M_1 = K Λ S_e^(1/2) = 1.4745 GeV。由实验 m_μ = 105.658 MeV 反推：

$$\sin^3\theta^{(2)} = \frac{m_\mu}{M_1} = \frac{105.658}{1474.5} = 0.07167$$

$$\theta^{(2)} = \arcsin(0.07167^{1/3}) = \arcsin(0.4153) = 24.54^\circ$$

**（B）与裸基准因果角的数值交叉验证**

裸基准因果角 θ_C⁰ = 26.16°（0.1 §3.1）。若假设缪子的有效投影角即为 θ_C⁰：

$$m_\mu^{\text{(th)}} = M_1 \sin^3\theta_C^0 = 1474.5 \times \sin^3 26.16^\circ$$

计算：sin 26.16° = 0.4409，sin³ 26.16° = 0.08569

$$m_\mu^{\text{(th)}} = 1474.5 \times 0.08569 = 126.35\ \text{MeV}$$

与实验值的偏差：

$$\frac{|126.35 - 105.658|}{105.658} = 19.6\%$$

**（C）偏差分析**

| 量 | 符号 | 候选值 | 实验反推值 | 偏差 |
|:---|:---|:---|:---|:---|
| 有效投影角 | θ^(2) | 26.16°（θ_C⁰） | 24.54° | 1.62° |
| sin³ θ^(2) | — | 0.08569 | 0.07167 | 19.6%（相对） |
| 理论质量 | m_μ^(th) | 126.35 MeV | 105.658 MeV | 19.6% |

**（D）诚实评估**

1. **量级对**：m_μ^(th) = 126 MeV 与实验 106 MeV 在同一量级（百MeV），且都在 M₀（9.83 MeV）和 M₁（1.47 GeV）之间——缪子的质量量级与层级 n=1 的标度匹配。

2. **扇区归属对**：θ^(2) 的候选范围（24°–28°）与裸基准因果角 θ_C⁰ = 26.16° 的扇区归属一致（均为 𝒞 扇区）。这为"缪子对应因果扇区主导的物理识别"提供了框架内数值线索。

3. **19.6% 偏差的可能来源**：0.1.2 的跨扇区耦合前置因子 κ_w = 13.092、κ_w' = 14.929 在联合截面（Hessian 交叉块）上产生 5%–25% 量级的非微扰修正（参考 0.1 定理 4.2 的注记和 23 号文章 §3.1 的 Ψ_joint 讨论）。19.6% 在此范围内，方向合理（裸基准角略高估 sin³ 值，非微扰修正压低质量）。

4. **不能声称已锁定**：θ^(2) 的框架内确定需要联合截面上 H^W(θ) = 0 的全局驻点分析（参考 23 号文章 §4.1 的联合 Hessian 结构），该分析尚未完成。当前仅展示 θ_C⁰ 作为候选值的可行性。

### 4.3 陶子（n = τ）：二层级候选

**（A）层级 1 候选（C_τ = M_1）**

若 C_τ = M_1 = 1.4745 GeV：

$$\sin^3\theta^{(3)} = \frac{m_\tau}{M_1} = \frac{1776.86}{1474.5} = 1.205 > 1$$

这在标准三角函数定义域（sin θ ≤ 1，故 sin³ θ ≤ 1）中**不可能**。陶子质量超过 M_1 → sin³θ > 1 → 需要角度映射的定义域扩展（复化角度或双曲函数，见 §4.4 开放问题 7）。

**（B）层级 2 候选（C_τ = M_2）**

若 C_τ = M_2 = K Λ² S_e^(1/2) = 221.2 GeV：

$$\sin^3\theta^{(3)} = \frac{1776.86}{221200} = 8.034 \times 10^{-3}$$

$$\theta^{(3)} = \arcsin((8.034\times10^{-3})^{1/3}) = \arcsin(0.2002) = 11.55^\circ$$

| 量 | 符号 | 候选值 | 注记 |
|:---|:---|:---|:---|
| 质量映射系数 | C_τ | M_2 = 221.2 GeV | 0.3.1.1 研究框架输出 |
| 有效投影角 | θ^(3) | 11.55° | 从实验质量反推 |
| 裸基准信息角 | θ_I⁰ | 5.91° | 0.1 §3.1 |
| 偏差 | Δθ | 5.64° | 较大，扇区归属不清晰 |

**（C）诚实评估**

1. 陶子的质量标度（~1.8 GeV）落在 M_1（1.47 GeV）和 M_2（221 GeV）之间。用 M_1 则 sin³θ > 1（溢出），用 M_2 则 θ^(3) ≈ 11.5°——该角度在框架内无直接锚点（不像缪子的 θ^(2) ≈ θ_C⁰ 有明确的扇区对应）。

2. 若 θ^(3) 应接近 θ_I⁰ = 5.91°（信息扇区），则 m_τ = M_2 sin³ 5.91° = 221.2 × 0.00109 = 241 MeV——与 1777 MeV 差一个数量级。若改用更大的 n，M_3 = 33.2 TeV → sin³θ ≈ 5.4×10⁻⁵ → θ ≈ 2.2°，更偏离 θ_I⁰。

3. **当前最优候选**：C_τ = M_2，θ^(3) ≈ 11.5°。该角度的几何起源（与 θ_I⁰ = 5.91° 的关系、是否对应 ℐ 扇区的某个激发态）是未来研究方向。

### 4.4 Koide 关系在 C_n 不普适条件下的代数结构

**关键公式**：在假设 H1（m_n = C_n sin³θ^(n)）下，Koide Q 的表达式为：

$$Q = \frac{C_e s_1^3 + C_\mu s_2^3 + C_\tau s_3^3}{\left(\sqrt{C_e}\,s_1^{3/2} + \sqrt{C_\mu}\,s_2^{3/2} + \sqrt{C_\tau}\,s_3^{3/2}\right)^2}$$

其中 s_n ≡ sin θ^(n)。

**情形 1：C_n 普适（C_e = C_μ = C_τ）**——C_n 在分子和分母中精确抵消。Q 简化为纯角度函数：

$$Q = \frac{s_1^3 + s_2^3 + s_3^3}{(s_1^{3/2} + s_2^{3/2} + s_3^{3/2})^2}$$

这是 Koide 公式可被几何框架导出的最佳情形。但如 §3.3 所述，C_n 普适性在 M_n 候选值下不成立。

**情形 2：C_n 层级相关（C_e = K, C_μ = M_1, C_τ = M_2）**——C_n 不抵消。代入 M_n = K Λ^n S_e^(1/2)：

$$\sqrt{C_n} = \sqrt{K} \cdot (\Lambda S_e^{1/2})^{n/2}$$

$$\frac{C_\mu}{C_e} = \Lambda S_e^{1/2} \approx 1755,\quad \frac{C_\tau}{C_\mu} = \Lambda = 150,\quad \frac{C_\tau}{C_e} = \Lambda^2 S_e^{1/2} \approx 2.63\times10^5$$

Q 中 C_n 比值的贡献无法通过角度分布完全抵消——除非角度 s_n 的特定分布恰好产生 Q = 2/3。

**情形 3（开放研究）**：是否存在 C_n 的中间赋值，使得：(a) C_n 全部在框架内有几何起源（M_n 或其变体）；(b) C_n 比值与角度分布的联合效应恰好使 Q = 2/3；(c) 该代数条件在几何框架内被证明？这是 P0 级开放问题。

### 4.5 缪子-因果角对应的数值稳健性

本节检查 §4.2 的"缪子 θ^(2) ≈ θ_C⁰"对应的稳健性——该对应依赖两个框架内量：M_1 = K Λ S_e^(1/2) 和 θ_C⁰。

| 框架参数 | 数值 | 对 m_μ^(th) 的影响 |
|:---|:---|:---|
| K | 839.758793 keV | 线性：K ± 1% → m_μ^(th) ± 1% |
| Λ | 150（0.13% 偏差的近似识别） | 线性：Λ ± 1 → m_μ^(th) ± 0.67% |
| S_e^(1/2) | 11.706 | 线性：S_e^(1/2) 由 S_e 锁定，< 10⁻¹⁰ 不确定性 |
| θ_C⁰ | 26.16°（裸基准点输出） | 非线性：θ_C⁰ ± 0.1° → sin³θ ± 6.5% |

Λ 的 0.13% 偏差影响远小于 19.6% 的总偏差，因此 Λ 的近似识别不是偏差的主要来源。19.6% 的偏差最可能的来源是：(a) θ^(2) 与 θ_C⁰ 之间的非微扰偏移（裸基准角 vs 联合截面驻点角）；(b) M_1 作为 C_μ 的近似性（M_n 是研究框架输出，非定理）。

**保守结论**：缪子-因果角对应是值得关注的框架内数值线索，但尚不能作为定理化主张。

### 5.1 假设施设 H2（θ^(1)+θ^(2)+θ^(3) = 90°）的检验

## 第 7 章　结论与开放问题

### 7.1 主要结果

1. **Koide 公式在标准模型和 CY 弦理论中是不可解释的巧合**：标准模型将其视为三个独立拉格朗日参数；CY 紧致化将其视为模空间上的可调函数，无刚性约束。

2. **0.3.1.1 的层级质量标度 M_n 为三代轻子的 C_n 提供了框架内候选值**：C_e = K（电子，框架已锁定），C_μ = M_1 = 1.47 GeV（缪子候选），C_τ = M_2 = 221 GeV（陶子候选）。这三个候选值全部来自框架内参数（K, Λ, S_e），无需外部输入。

3. **缪子有效投影角与裸基准因果角 θ_C⁰ = 26.16° 的数值邻近（偏差 1.62°）是第3次修订的核心正向发现**。在 C_μ = M_1 的候选赋值下，m_μ^(th)(θ_C⁰) = 126 MeV，与实验 106 MeV 偏差 19.6%——方向、量级和扇区归属均合理。

4. **C_n 不普适**（C_e = K, C_μ = M_1, C_τ = M_2），Koide Q = 2/3 在 C_n 不普适的条件下需要角度分布与 C_n 比值的联合机制——这是 P0 级开放问题。

5. **电子的质量映射在框架内完全封闭**——这是三代轻子中唯一达到"0.0.X + 单一核心映射即足够"标准的。缪子部分封闭（C_n 有候选，θ^(2) 有候选取向），陶子最开放（θ^(3) 无清晰锚点）。

### 7.2 开放问题
