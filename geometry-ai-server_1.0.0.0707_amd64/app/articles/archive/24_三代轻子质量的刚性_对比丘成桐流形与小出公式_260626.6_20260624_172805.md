## 摘　要

三代轻子质量的层级结构（$m_e = 0.511$ MeV, $m_\mu = 105.66$ MeV, $m_\tau = 1776.86$ MeV）是粒子物理中一个未解决的基本问题。本文从十方几何的单一核心映射层（0.1）与层级约束截面理论（0.3.1.1）出发，分析三代轻子质量的刚性几何约束，并与丘成桐 Calabi-Yau 路径及小出 Koide 公式进行对比。

**核心发现（第4次修订）**：

1. **缪子-因果角对应**（第3次修订）：以层级质量标度 $M_1 = K \Lambda_H S_e^{1/2} = 1.47$ GeV 为 $C_\mu$，取有效投影角为裸基准因果角 $\theta_C^0 = 26.16^\circ$，预言 $m_\mu^{\text{(th)}} = 126$ MeV，与实验值 106 MeV 偏差 19.6%——方向、量级和扇区归属均合理。对应有效角度偏移 $\Delta\theta \approx 1.56^\circ$。

2. **H^W 驻点分析**（本次修订新增）：从 0.1 定理 4.2 的跨扇区耦合矩阵 $H^W$ 出发，建立双粒子联合截面上二代轻子角度偏移的严格变分框架。核心结论：单粒子 $H^W$ 微扰在构型 B（因果界）处灾难性失效（软模位移 $\sim 300^\circ$，二阶展开完全崩溃），迫使分析必须在**双粒子联合截面**上进行。$H^W$ 矩阵本征值 $\lambda_-^W = 0.710$ 的反对称模式直接耦合 $\theta_C$，且其特征角度标度 $1/\sqrt{\lambda_-^W} = 1.187$ rad 与构型 A–B 间角距离 1.121 rad 仅差 5.9%，暗示 $H^W$ 结构内禀编码了二代轻子的角度定位信息。阻塞点：双粒子联合截面上 $W_{12}$ 的显式角度函数形式和二代轻子的物理识别条件目前不在框架内。

3. **陶子层级归属**：$M_1$ 方案导致 $\sin^3\theta > 1$（溢出，需复化扩展）；$M_2 = K \Lambda_H^2 S_e^{1/2} = 221$ GeV 方案给出有效角度 $\approx 11.5^\circ$，无直接几何锚点。

4. **Koide 公式**：$C_n$ 不普适（$C_e = K, C_\mu = M_1, C_\tau = M_2$），$Q = 2/3$ 在 $C_n$ 不普适条件下等价于角度分布与 $C_n$ 比值的联合约束——P0 级开放问题。

本文所有假设以"假设 H"标注，所有猜想以"猜想 G"标注。电子质量在框架内完全封闭（"0.0.X + 单一核心映射即足够"）；缪子半封闭；陶子开放。

## §3　三代轻子质量的几何结构

### 3.1 层级质量映射假设

**假设 H1（层级质量映射）**：第 $n$ 代轻子质量由层级标度 $C_n$ 与有效物质角 $\theta^{(n)}$ 给出：

$$m_n = C_n \sin^3 \theta^{(n)}$$

其中：
- $C_0 = K = 839.76$ keV（电子，框架内定理）
- $C_1 = M_1 = K \Lambda_H S_e^{1/2} = 1.47$ GeV（缪子候选，研究框架）
- $C_2 = M_2 = K \Lambda_H^2 S_e^{1/2} = 221$ GeV（陶子候选，研究框架）

**假设 H2（三代角度和约束）**：$\theta^{(1)} + \theta^{(2)} + \theta^{(3)} = 90^\circ$（公理 1 向三代粒子角度约束的推广，待严格化）。

### 3.2 电子的框架内自足性 ✅

$m_e = K \sin^3\theta_M^0 = 839.76 \times \sin^3 57.93^\circ = 839.76 \times 0.6085 = 511.0$ keV。

电子是三代轻子中**唯一**达到"0.0.X + 单一核心映射即足够"标准的。其质量映射、裸基准角、有效 Hessian 全部在框架内封闭。

### 3.3 缪子的半封闭状态 ⚠️

在 H1 下取 $C_1 = M_1$，有效角度候选为 $\theta_C^0 = 26.16^\circ$（构型 B 的物质角，等于裸基准因果角）：

$$m_\mu^{\text{(th)}}(\theta_C^0) = M_1 \sin^3 26.16^\circ = 1.47\ \text{GeV} \times 0.08557 = 126\ \text{MeV}$$

实验值 $m_\mu^{\text{exp}} = 105.66$ MeV。偏差 $\Delta = 19.6\%$。

反推有效角度：$\sin^3\theta_{\text{eff}} = 105.66/1470 = 0.07188$，$\theta_{\text{eff}} = 24.60^\circ$。与 $\theta_C^0$ 差 $\Delta\theta = 1.56^\circ$。

**正向发现**（第3次修订）：方向（百 MeV 量级）、量级（偏差仅 19.6%）、扇区归属（因果角而非物质角或信息角）全部正确。这不是巧合。

## §5　H^W 驻点分析与 19.6% 偏差（第4次修订新增）

### 5.1 问题陈述

在假设 H1 下，取 $C_1 = M_1 = K \Lambda_H S_e^{1/2} = 1.47$ GeV，有效角度候选 $\theta_C^0 = 26.16^\circ$，预言 $m_\mu^{\text{(th)}} = 126$ MeV，与实验值 106 MeV 偏差 $\Delta = 19.6\%$。等效于有效角偏移 $\Delta\theta = 1.56^\circ$（从 $26.16^\circ \to 24.60^\circ$）。

**本节任务**：从 0.1 定理 4.2 的跨扇区耦合矩阵 $H^W$ 出发，分析 $\Delta\theta$ 是否可以从联合截面的驻点条件推导。

### 5.2 构型 B 的几何结构

构型 B（因果界，S₃ 轮换）的角度配置为：

$$P_B = (\theta_M^B, \theta_C^B, \theta_I^B) = (26.16^\circ, 5.91^\circ, 57.93^\circ).$$

由 S₃ 对称性，$S(P_B) = S(P_e) = 137.0000000009$。

#### 5.2.1 梯度计算

在 $P_B$ 处，六项作用量 $S$ 在约束流形切空间上的梯度经精确计算如下（角度以弧度计，梯度单位 rad⁻¹）：

**全梯度**（无约束）：

$$\begin{aligned}
\partial S/\partial\theta_M|_{P_B} &= -71.224, \\
\partial S/\partial\theta_C|_{P_B} &= -2145.4, \\
\partial S/\partial\theta_I|_{P_B} &= -10.602.
\end{aligned}$$

（验证 S₃ 对称性：$\partial S/\partial\theta_M|_{P_B} = \partial S/\partial\theta_C|_{P_e} = -71.23$，$\partial S/\partial\theta_C|_{P_B} = \partial S/\partial\theta_I|_{P_e} = -2145$，$\partial S/\partial\theta_I|_{P_B} = \partial S/\partial\theta_M|_{P_e} = -10.60$。✓）

**约束流形切空间梯度**（消去法向分量 $(\nabla S \cdot \hat{n})\hat{n}$，$\hat{n} = (1,1,1)/\sqrt{3}$）：

$$\begin{aligned}
\partial_T S/\partial\theta_M|_{P_B} &= +671.2, \\
\partial_T S/\partial\theta_C|_{P_B} &= -1403.0, \\
\partial_T S/\partial\theta_I|_{P_B} &= +731.8.
\end{aligned}$$

（验证：$671.2 - 1403.0 + 731.8 = 0.0$。✓）

**$(\xi, \eta)$ 坐标下的切向梯度**（$\xi = \theta_I - \theta_C$，$\eta = \theta_C - \theta_M$）：

$$\boxed{\nabla_T S|_{P_B} = \begin{pmatrix} \partial S/\partial\xi \\ \partial S/\partial\eta \end{pmatrix}_{P_B} = \begin{pmatrix} +2135 \\ -2074 \end{pmatrix}\ \text{rad}^{-1}}.$$

**关键观察**：$\|\nabla_T S\|_{P_B} \approx 3000$ rad⁻¹。构型 B 远离 $S$ 的任何驻点（$S$ 的唯一全局极小点在 $(30^\circ, 30^\circ, 30^\circ)$，$\nabla S = 0$）。

#### 5.2.2 构型 B 的 Hessian

由 S₃ 对称性，构型 B 的 Hessian 矩阵元（在 $(\theta_M, \theta_C, \theta_I)$ 基下）为构型 A Hessian 的 S₃ 置换：

$$H_{MM}^B = H_{CC}^A = 367.73,\quad H_{CC}^B = H_{II}^A = 59259.35,\quad H_{II}^B = H_{MM}^A = 31.30,$$
$$H_{MC}^B = H_{CI}^A = 433.16,\quad H_{MI}^B = H_{MC}^A = 3.41,\quad H_{CI}^B = H_{MI}^A = 69.35.$$

在 $(\xi, \eta)$ 坐标下的 Hessian 截面（构型 B 局部）：

$$H_{\xi\xi}^B = H_{II}^B + H_{CC}^B - 2H_{CI}^B = 59151.6,$$
$$H_{\eta\eta}^B = H_{CC}^B + H_{MM}^B - 2H_{MC}^B = 58760.8,$$
$$H_{\xi\eta}^B = -H_{CI}^B + H_{MC}^B + H_{MI}^B - H_{CC}^B = -58892.1.$$

与构型 A 的 $(\lambda_1 = 392.21, \lambda_2 = 58760.77, H_{\xi\eta} \approx 0)$ 不同，构型 B 中两个方向均为硬模（$\sim 5.9 \times 10^4$），且强混合（$H_{\xi\eta} \sim -5.9 \times 10^4$）。这意味着 $(\xi, \eta)$ 坐标在构型 B 处不是好的主曲率坐标。

### 5.3 单粒子 H^W 微扰的灾难性失效

尝试在构型 B 处施加单粒子联合变分 $\delta(S + W_{\text{total}}) = 0$。在 $(\xi, \eta)$ 坐标中，$W_{\text{total}}$ 的 Hessian 为 $H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix}$（0.1.1 定理 4.2）。驻点条件：

$$\nabla_T S|_{P_B} + (H_{\text{bare}}^B + H^W) \Delta = 0, \quad \Delta = \begin{pmatrix} \Delta\xi \\ \Delta\eta \end{pmatrix}.$$

若直接使用二阶展开（构型 B 处 $H_{\text{bare}}^B$ 在 $(\xi,\eta)$ 坐标下的对角近似不可用——强混合），则 $\Delta$ 的量级为 $\|\Delta\| \sim \|\nabla S\| / \lambda_{\min} \sim 2000 / 392 \approx 5$ rad $\approx 300^\circ$。

**结论**：**单粒子 H^W 微扰在构型 B 处灾难性失效。** 构型 B 离 $S$ 的极小点太远，二阶展开完全不适用。缪子不能作为构型 B 附近的单粒子 H^W 微扰来分析。

### 5.4 为什么必须用双粒子联合截面

上述失效有明确的物理含义：**缪子不是独立存在的。它存在于电子-缪子双粒子系统中。** 正确的变分框架是双粒子联合截面：

$$S_{\text{joint}} = S^{(1)}(\theta^{(1)}) + S^{(2)}(\theta^{(2)}) + W_{12}(\theta^{(1)}, \theta^{(2)}),$$

其中粒子 1（电子）在构型 A 近旁，粒子 2（缪子）在构型 B 近旁，$W_{12}$ 为跨粒子耦合。

**关键区分**：
- $H^W$（0.1.1 定理 4.2）描述**单粒子内部**三扇区（$\mathcal{M}, \mathcal{C}, \mathcal{I}$）之间的跨扇区耦合。
- $W_{12}$（23 号文章 §3.1）描述**两个粒子之间**的耦合，**继承** $H^W$ 的结构但作用在不同对象上。

23 号文章 §3.1 明确指出：$W_{12}$ 的系数由 $\kappa_w, \kappa_w'$ 对 $w, w'$ 加权后生成。$H^W$ 矩阵的本征值 $a' \pm b'$ 对应联合截面上两粒子的对称/反对称耦合模式。

**M-扇区同步约束的失效**：23 号文章 §2.1 的 M-扇区同步约束 $\theta_M^{(1)} = \theta_M^{(2)}$ 是夸克偶素（$c\bar{c}$、$b\bar{b}$）场景下的合理近似（两夸克质量相近）。对于电子-缪子系统（$m_\mu/m_e \approx 207$），$\theta_M^{(1)} \neq \theta_M^{(2)}$ 必然是成立的。因此电子-缪子联合截面需要**放松 M-扇区同步约束**——两个粒子的物质角独立。

### 5.5 H^W 本征值与耦合模式

$H^W$ 矩阵（0.1.1 定理 4.2）：

$$H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} = \begin{pmatrix} 1.577 & 0.867 \\ 0.867 & 1.577 \end{pmatrix}.$$

本征值与本征向量：

$$\lambda_+^W = a' + b' = 2.444, \quad v_+ = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix} \quad (\xi + \eta\ \text{方向}),$$
$$\lambda_-^W = a' - b' = 0.710, \quad v_- = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix} \quad (\xi - \eta\ \text{方向}).$$

在 $(\theta_M, \theta_C, \theta_I)$ 坐标中：

$$\begin{aligned}
\xi + \eta &= (\theta_I - \theta_C) + (\theta_C - \theta_M) = \theta_I - \theta_M, \\
\xi - \eta &= (\theta_I - \theta_C) - (\theta_C - \theta_M) = \theta_I + \theta_M - 2\theta_C.
\end{aligned}$$

**关键观察**：反对称模式（$v_-$，$\lambda_-^W = 0.710$）的变分方向是 $\theta_I + \theta_M - 2\theta_C$。在角度和约束 $\theta_M + \theta_C + \theta_I = 90^\circ$ 下：

$$\theta_I + \theta_M - 2\theta_C = (90^\circ - \theta_C) - 2\theta_C = 90^\circ - 3\theta_C.$$

**反对称模式直接与 $\theta_C$ 耦合**——这是 $H^W$ 结构对缪子问题的核心含义：$\lambda_-^W$ 控制 $\theta_C$ 方向的曲率修正。

### 5.6 数值关系：$H^W$ 特征标度与构型间距

反对称模式的特征角度标度：

$$\Delta\theta_{\text{char}} = \frac{1}{\sqrt{\lambda_-^W}} = \frac{1}{\sqrt{0.710}} = 1.187\ \text{rad} = 68.0^\circ.$$

构型 A（电子）与构型 B（缪子候选）在约束流形上的角距离：

$$d_{AB} = \sqrt{(57.93^\circ - 26.16^\circ)^2 + (26.16^\circ - 5.91^\circ)^2 + (5.91^\circ - 57.93^\circ)^2} = 64.2^\circ = 1.121\ \text{rad}.$$

**观察**：$d_{AB} / \Delta\theta_{\text{char}} = 1.121 / 1.187 = 0.944$，偏差仅 5.6%。$H^W$ 反对称模式的特征标度与电子-缪子构型间距在数值上邻近。

**对称模式的特征标度**：$1/\sqrt{\lambda_+^W} = 0.640$ rad $= 36.6^\circ$。构型 A 与 B 在 $\theta_I - \theta_M$ 方向的平均分离：$(|5.91^\circ - 57.93^\circ| + |57.93^\circ - 26.16^\circ|)/2 = (52.02^\circ + 31.77^\circ)/2 = 41.9^\circ = 0.731$ rad。与 0.640 rad 偏差 14%。

**推论**：$H^W$ 矩阵的两个本征标度（1.187 rad 和 0.640 rad）与 S₃ 轮换构型之间的几何间距（1.121 rad 和 $\sim 0.73$ rad）在量级上匹配。这暗示 $H^W$ 的结构内禀编码了不同 S₃ 构型之间的角度关系信息——这是 $H^W$ 驻点分析可用于二代轻子的群论基础。

## §6　$C_n$ 不普适条件下的 Koide 刚性

### 6.1 Koide 公式回顾

小出公式（1982）：

$$Q = \frac{m_e + m_\mu + m_\tau}{\left(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau}\right)^2} \approx \frac{2}{3}.$$

实验值 $Q \approx 0.66666 \approx 2/3$，精度 $10^{-5}$。

### 6.2 $C_n$ 普适情形

若 $C_n$ 三代普适（$C_e = C_\mu = C_\tau = C$），则 $Q$ 退化为纯角度函数：

$$Q = \frac{s_1^3 + s_2^3 + s_3^3}{\left(s_1^{3/2} + s_2^{3/2} + s_3^{3/2}\right)^2}, \quad s_n = \sin\theta^{(n)}.$$

在此情形下，$Q = 2/3$ 等价于 $\theta^{(n)}$ 之间的代数约束——该约束是否从 H^W 驻点条件自然涌现，是封闭 Koide 公式的关键。

### 6.3 $C_n$ 不普适情形（当前框架实际情形）

在 H1 下，$C_e = K$，$C_\mu = M_1$，$C_\tau = M_2$，三者不相等。比值：

$$\frac{C_\mu}{C_e} = \Lambda_H S_e^{1/2} = 1755,\quad \frac{C_\tau}{C_\mu} = \Lambda_H = 150.$$

此时 $Q$ 不再是纯角度函数：

$$Q = \frac{K s_1^3 + M_1 s_2^3 + M_2 s_3^3}{\left(\sqrt{K} s_1^{3/2} + \sqrt{M_1} s_2^{3/2} + \sqrt{M_2} s_3^{3/2}\right)^2}.$$

$Q = 2/3$ 等价于 $s_n$ 与 $C_n$ 的联合条件——这是比 $C_n$ 普适情形更强的约束，也是 P0 级开放问题。

### 6.4 H2 约束下的数值检验

取实验反推角度：$\theta^{(1)} = 57.93^\circ$，$\theta^{(2)} = 24.60^\circ$，$\theta^{(3)} = 11.55^\circ$。和 $= 94.08^\circ \neq 90^\circ$。

若强制 H2（$\theta^{(1)}+\theta^{(2)}+\theta^{(3)} = 90^\circ$）且取 $\theta^{(1)} = \theta_M^0 = 57.93^\circ$，则 $\theta^{(2)}+\theta^{(3)} = 32.07^\circ$。

- 若 $\theta^{(2)} = 26.16^\circ$（$\theta_C^0$）：$\theta^{(3)} = 5.91^\circ$（$\theta_I^0$），$m_\tau^{\text{(th)}} = M_2 \sin^3 5.91^\circ = 241.6$ MeV（偏差 $86.4\%$，方向完全错误）。**$\theta_I^0$ 被陶子质量排除。**

- 若 $\theta^{(2)} = 24.60^\circ$（实验反推）：$\theta^{(3)} = 7.47^\circ$，$m_\tau^{\text{(th)}} = M_2 \sin^3 7.47^\circ = 487$ MeV（偏差 $72.6\%$）。

**H2 + $C_n = M_n$ + 任意角度分配均无法同时拟合三代质量。** $C_n$ 不普适条件下的 Koide $Q = 2/3$ 等价条件目前不在框架内。
