# 重子生成的几何理论——Sakharov三条件的统一封闭

## 摘要

本文在几何论框架内给出重子生成（baryogenesis）的完整几何翻译。Sakharov三条件被映射为三个几何结构：(i) **重子数破坏** = 七级递推相变边界上约束流形拓扑扇区的瞬时连通；(ii) **C/CP破坏** = θ_C-共享截面间旋转的复相位（已在8号导出，δ_CP≈1.20 rad）；(iii) **热非平衡** = N₂→N₃递推乘子从6跳变至33.3（非平衡度量 ε_non-eq=4.56）。重子数B被证明是θ_C-共享截面上的拓扑缠绕数：B = (1/3) w_{θ_C}，夸克B=1/3、轻子B=0、重子B=1——均从截面类型的几何分类自然引出。净重子不对称度η_B=n_B/n_γ的计算被约化到相变边界的非平衡量子动力学——由I场Fokker-Planck方程在拓扑重构瞬间的非平衡解主导。当前定量估计给出η_B ~ 8×10⁻⁸（与实验值6×10⁻¹⁰差约140倍），精确闭合需等待I场非平衡动力学框架的完整建立。

## 1. 重子生成的Sakharov框架

### 1.1 标准模型的三个缺口

1967年，Sakharov [Sak67] 提出宇宙中重子不对称度η_B ≡ n_B/n_γ ≠ 0需要同时满足三个条件：

1. **重子数破坏**（B violation）——存在ΔB ≠ 0的过程
2. **C和CP破坏**（C/CP violation）——正反粒子过程不对称
3. **热非平衡**（thermal non-equilibrium）——偏离热平衡使净B不被后续逆过程抹平

标准模型原则上可以满足这三条件（电弱sphaleron提供B破坏，CKM提供CP破坏，一阶电弱相变提供非平衡），但定量上：(a) SM的CP破坏（Jarlskog不变量J_CP ~ 3×10⁻⁵）太弱；(b) 125 GeV Higgs质量下电弱相变是平滑过渡（crossover），不是一阶相变。因此标准模型不足以解释观测值η_B ≈ 6×10⁻¹⁰ [PDG24]。

这催生了大量超出标准模型的重子生成模型：GUT重子生成、轻子生成（leptogenesis）、电弱重子生成（需扩展Higgs扇区）、Affleck-Dine机制等。所有这些的共同特征是：**引入额外自由参数**。

### 1.2 几何论的方法论

本文不从标准模型的Sakharov条件出发。"重子数"、"CP破坏"、"热非平衡"必须从几何论的第一原理重新定义。问题不是"如何在几何论中实现Sakharov条件"——而是"Sakharov条件在几何论中对应什么"。

回答逐层展开：

- **重子数B** → θ_C-共享截面上的拓扑缠绕数（§2）
- **B破坏** → 约束流形拓扑扇区在相变边界的瞬时连通（§3）
- **C/CP破坏** → θ_C截面间旋转的复相位（§4，已在8号导出）
- **热非平衡** → N₂→N₃递推乘子的6→33.3跳变（§5，已在50号导出）

## 2. 重子数的几何定义

### 2.1 粒子截面类型的几何分类

从0.6.7 §6.3，三代轻子的完备分类基于截面类型：

| 截面类型 | 物理对应 | θ_C参与度 | 世代数 |
|:---|:---|:---|:---|
| 裸截面 | 电子、电子中微子（第一代） | 0 | 1 |
| θ_C-共享截面 | u/d, c/s, t/b 夸克 | 1 | 3 |
| M-同步截面 | μ, ν_μ, τ, ν_τ（第二、三代轻子） | 0 | 2 |
| θ_I-共享截面 | 被排除（§6.3.2：物理不可行） | — | 0 |

**关键结构区分**：夸克与轻子的唯一几何区别是θ_C参与度。夸克必然涉及θ_C扇区的共享截面；轻子不涉及θ_C共享。

### 2.2 重子数作为拓扑缠绕数

**定义 1**（重子数的几何定义）。设粒子构型Γ在约束流形M_const上对应于一组截面构型。其重子数B定义为θ_C-共享截面上的拓扑缠绕数w_{θ_C}(Γ)除以3：

$$B(\Gamma) \equiv \frac{1}{3}\, w_{\theta_C}(\Gamma)$$

其中$w_{\theta_C}$是I场编码在该截面上的缠绕数（映射度）。

**定理 1**（重子数量化）。$w_{\theta_C}$取整数值，因此$B \in \frac{1}{3}\mathbb{Z}$。具体地：

- $w_{\theta_C} = 0$ → $B = 0$：轻子（无θ_C参与）
- $w_{\theta_C} = 1$ → $B = \frac{1}{3}$：单个夸克
- $w_{\theta_C} = -1$ → $B = -\frac{1}{3}$：单个反夸克
- $w_{\theta_C} = 3$ → $B = 1$：重子（三夸克束缚态，所有颜色/代截面同时参与）
- $w_{\theta_C} = -3$ → $B = -1$：反重子

**证明。** 约束流形M_const在θ_C方向具有非平凡拓扑。截面分类定理（0.6.7 §6.3.1）给出三种截面类型的离散分类。θ_C-共享截面在拓扑上对应于M_const在θ_C方向的一个非平凡S¹纤维。I场在该纤维上的编码构型具有良定义的缠绕数w ∈ ℤ。因子1/3来自三代结构——单个夸克占用三色/三代空间中的一份，完整重子占用全部三份。■

**推论 1.1**（微扰重子数守恒）。在微扰论中，I场在θ_C纤维上的缠绕数是拓扑不变量——连续形变不能改变拓扑不变量。因此B在微扰过程中守恒。这与标准模型的微扰B守恒完全对应。

**推论 1.2**（B-L的几何意义）。B-L = (1/3)w_{θ_C} - L。轻子数L对应M-同步截面上的缠绕数w_M。B-L守恒等价于w_{θ_C} - 3w_M的拓扑不变性——这对应于约束流形上更高级的同调不变量。

### 2.3 重子数定义的自洽性检查

| 物理事实 | 几何论推导 | 自洽？ |
|:---|:---|:---|
| 夸克B=1/3 | 单θ_C截面 → w=1 → B=1/3 | ✓ |
| 轻子B=0 | 无θ_C参与 → w=0 → B=0 | ✓ |
| 介子B=0 | q\bar{q}：w=1+(-1)=0 → B=0 | ✓ |
| 重子B=1 | qqq：w=1+1+1=3 → B=1 | ✓ |
| 质子稳定（微扰） | 拓扑不变量 → ΔB=0不可行 | ✓ |
| 质子衰变（非微扰） | 拓扑扇区连通 → ΔB=1可能（见§3和18号） | ✓ |
| 中子是费米子 | 三夸克 → w=3奇数 → 费米统计 | ✓ |

**没有人设的重子数。B是θ_C拓扑的必然推论。**

## 3. Sakharov条件I：B破坏的几何机制

### 3.1 微扰B守恒的非微扰破缺

在约束流形结构稳定的情况下，$w_{\theta_C}$是拓扑不变量——B绝对守恒。质子稳定（τ_p > 10³⁴年，18号）正是这一拓扑保护的结果。

**但拓扑保护在约束流形重构时失效。** 在相变边界上，约束流形的维度、拓扑结构瞬时改变（七级递推的$N_{\text{info}}$跃变），旧的拓扑分类暂时失去意义。在此期间，不同$w_{\theta_C}$的构型可以连续过渡。

### 3.2 零温B破坏的被压制

首先计算零温（今日宇宙）下的B破坏率，以确认拓扑保护的有效性。

θ_C截面上的拓扑扇区由一个"势垒"分隔。该势垒的高度由约束流形在θ_C方向的Hessian刚度决定。在P₀点附近（0.2.1 §4.5）：

$$H_{\xi\xi} = 58760.77\ \text{rad}^{-2}$$

拓扑扇区间的欧几里得作用量（类比瞬子作用量）：

$$S_E^{\text{inst}} \sim \sqrt{H_{\xi\xi}} \cdot \Delta\theta_C \sim \sqrt{58761} \times \frac{\pi}{3} \sim 242.4 \times 1.047 \sim 253.8$$

零温B破坏率（指数压制）：

$$\Gamma_{B}(T=0) \sim \hbar_{\text{eff}}^{-1} \cdot e^{-S_E^{\text{inst}}} \sim 6000 \cdot e^{-254} \sim 6000 \times 1.5 \times 10^{-111} \sim 9 \times 10^{-108}$$

在物理时间中（$t_{\text{step}}^{-1} \sim 0.01\ \text{s}^{-1}$）：

$$\Gamma_{B}^{\text{phys}}(T=0) \sim 9 \times 10^{-110}\ \text{s}^{-1}$$

宇宙年龄 ~ 4×10¹⁷秒 → 在宇宙寿命内发生一次B破坏的概率 ~ 4×10⁻⁹²。**零温B破坏实际上不存在。** 这保证了今日质子的稳定性，与18号文章一致。

### 3.3 相变边界的拓扑重构

**定理 2**（相变边界的拓扑扇区连通）。在七级递推的层级跃迁$N_k \to N_{k+1}$期间，约束流形的有效维度从$\dim(N_k)$跃迁至$\dim(N_{k+1})$。在维度跃迁的瞬时（$\Delta t_{\text{trans}}$），旧的拓扑不变量分类失效，所有θ_C扇区等权连通。

**证明（概要）。** 约束流形M_const依赖于I场编码规模$N_{\text{info}}$：$M_{\text{const}} = M_{\text{const}}(N_{\text{info}})$。不同$N_{\text{info}}$下的M_const具有不同的同调群结构。在$N_k \to N_{k+1}$的离散跃迁（而非连续形变）中，M_const经历拓扑重构——某些同调类出现/消失。在此期间，定义在旧同调群上的缠绕数$w_{\theta_C}$暂时无定义。跃迁完成后，新的同调群确立，新的$w_{\theta_C}'$重新分类构型。在中间"无定义"阶段，旧分类下不同$w_{\theta_C}$的构型可以连续过渡。■

**关键物理图像：** 不是"翻越势垒"（隧穿），而是**势垒本身在相变瞬间不存在了**。这完全不同于标准模型中的sphaleron图像（热激发跨越鞍点）。在几何论中，B破坏不需要热激发——它发生于拓扑分类本身的瞬时消失。

### 3.4 B破坏的活跃窗口

从50号§5.6的递推乘子序列：

| 层级跃迁 | 乘子比 r_k/r_{k-1} | 拓扑重构强度 |
|:---|:---|:---|
| N₁→N₂ | 6/1 = 6 | 中等 |
| **N₂→N₃** | **33.3/6 = 5.56** | **最大** |
| N₃→N₄ | 10/33.3 = 0.30 | 收缩（反方向） |
| N₄→N₅ | 10/10 = 1.00 | 零（无重构） |
| N₅→N₆ | 1.125/10 = 0.1125 | 微弱 |
| N₆→N₇ | 2/1.125 = 1.78 | 中等 |

**定理 3**（重子生成的主导窗口）。N₂→N₃跃迁（乘子跳变5.56倍，相区I→II过渡）是唯一的显著B破坏窗口。原因：

1. **最大的拓扑重构强度**：乘子比5.56远大于其他跃迁（第二大的是N₁→N₂的6→1=6，但那是从真空开始的初始跃迁，粒子尚未大量凝结）。
2. **相区过渡**：从"剧烈展开"（相区I）到"有序重组"（相区II）的结构性变化。
3. **粒子已大量存在**：N₂ = 36000的编码规模足以支持丰富的夸克/轻子构型。

N₁→N₂跃迁期间乘子比为6，但此时$N_{\text{info}}$才从6000升到36000——粒子凝结可能尚未完成（凝结时间见50号§9.2，电子~7×10⁻¹⁶ s对应因果步，但N₁阶段的全息屏物理时间标度不同）。N₂→N₃是粒子大量存在后的首次——也是唯一一次——剧烈拓扑重构。

## 4. Sakharov条件II：C/CP破坏

### 4.1 条件已在上游封闭

C/CP破坏在几何论中已在8号文章（CKM矩阵的几何起源）中系统导出。本节仅提取与重子生成直接相关的结论，细节参见[8, §3-4]。

**定理 4**（几何CP破坏——来自8号）。θ_C-共享截面间存在不可约复相位，源自截面构型在约束流形上的非平凡复结构。夸克质量矩阵$M_q$在截面基$\{q_1, q_2, q_3\}$下具有复矩阵元：

$$(M_q)_{ij} = |m_{ij}| e^{i\phi_{ij}}$$

CP破坏相位$\delta_{CP}$是质量矩阵复相位的规范不变量：

$$\delta_{CP} \approx 1.20 \pm 0.08\ \text{rad}$$

对应Jarlskog不变量$J_{CP} \approx (3.0\pm0.3)\times10^{-5}$。

### 4.2 CP破坏在B破坏过程中的偏置作用

在N₂→N₃相变边界的拓扑扇区连通期间，夸克构型（w=1）与反夸克构型（w=-1）间的跃迁振幅为：

$$\mathcal{A}(w=1 \to w=-1) = \mathcal{A}_0 \cdot e^{i\delta_{CP}} \cdot f_{\text{loop}}$$

$$\mathcal{A}(w=-1 \to w=1) = \mathcal{A}_0 \cdot e^{-i\delta_{CP}} \cdot f_{\text{loop}} = \overline{\mathcal{A}(w=1 \to w=-1)}$$

其中$\mathcal{A}_0$是CP守恒部分，$f_{\text{loop}}$是CP破坏过程涉及的圈级因子。

跃迁概率之差（CP破坏偏置）：

$$\frac{\Delta\Gamma}{\Gamma} \equiv \frac{|\mathcal{A}(1\to-1)|^2 - |\mathcal{A}(-1\to1)|^2}{|\mathcal{A}(1\to-1)|^2 + |\mathcal{A}(-1\to1)|^2} \approx \sin\delta_{CP} \cdot f_{\text{loop}}$$

$$\frac{\Delta\Gamma}{\Gamma} \approx \sin(1.20) \times 3.0\times10^{-5} \approx 0.932 \times 3.0\times10^{-5} \approx 2.8\times10^{-5}$$

**物理意义**：在等权连通期间，每10⁵次B破坏跃迁中，净多出约3次"夸克→反夸克为夸克"方向的事件。这个微小的偏好——在拓扑分类恢复后被"冻入"——就是宇宙净重子数的起源。

## 5. Sakharov条件III：热非平衡

### 5.1 热非平衡已在50号封闭

"温度"在几何论中不是基本概念——它是M场涨落方差的统计度量。但"非平衡"在几何论中有精确的几何对应：**七级递推乘子的不连续性。**

从50号§5.6：

$$\text{递推乘子序列 } (r_1, \ldots, r_6) = (6, 33.3, 10, 10, 1.125, 2)$$

**定义 2**（非平衡度量）。层级跃迁$N_k \to N_{k+1}$的非平衡度量定义为乘子之比：

$$\epsilon_{\text{non-eq}}^{(k)} \equiv \frac{r_{k+1}}{r_k}$$

当$\epsilon_{\text{non-eq}} \neq 1$时，系统处于非平衡状态。

**定理 5**（重子生成的非平衡条件）。N₂→N₃跃迁的$\epsilon_{\text{non-eq}} = 33.3/6 = 5.56$是最大非平衡度量，满足Sakharov第三条件。

**证明**（来自50号§7.2和§7.3）。在$\epsilon_{\text{non-eq}} = 5.56$的冲击下：(i) I场编码速率瞬时飙升至平时的5.56倍；(ii) 约束流形拓扑扇区连通（§3.3）；(iii) B破坏过程活跃；(iv) CP破坏偏置（2.8×10⁻⁵）产生净B；(v) 跃迁完成后系统进入乘子稳定的相区II（r₃=r₄=10）→近平衡恢复→B破坏冻结→净B被保留。■

### 5.2 标准模型电弱相变的几何对应

**命题 1**（电弱相变与N₂→N₃跃迁的对应）。标准模型的电弱相变温度$T_{EW} \sim 100\ \text{GeV}$对应于几何论中的N₂→N₃相变边界。

**论据。** N₂→N₃跃迁是乘子从6跳至33.3的剧烈膨胀——对应于粒子物理中的"电弱对称性破缺"能量标度（v≈246 GeV）。在此标度以下，SU(2)×U(1)→U(1)_em，规范玻色子获得质量。几何论中，N₃阶段（$N_{\text{info}} \sim 1.2\times10^6$）是七级递推中"有序重组"相区的开始——与电弱相变后的物理结构重组对应。

此命题的严格证明需等待规范群涌现（缺口1）的闭合——那时N₂→N₃的几何跃迁将与SU(2)×U(1)→U(1)_em的规范对称性破缺精确对应。

## 6. 净重子不对称度η_B的定量估计

### 6.1 问题的物理图像

$$N_{\text{info}} = N_2 = 3.6\times10^4 \xrightarrow{\text{乘子 } 33.3} N_3 = 1.2\times10^6$$

在此期间：

1. 约束流形拓扑扇区连通（§3.3）
2. 夸克↔反夸克跃迁等权（势垒消失）
3. CP破坏偏置2.8×10⁻⁵使正向（q → q̄？实际上是净q多于q̄的方向）略多于反向
4. 跃迁完成后拓扑分类恢复
5. 净B被锁定

### 6.2 第一性计算（树级估计）

**步骤1：等权连通期间的夸克构型数**

N₂阶段的θ_C-共享截面上，夸克/反夸克构型（不计自旋简并——自旋在几何论中对应截面取向自由度）：

| 属性 | 可能的取值 | 数量 |
|:---|:---|:---|
| 粒子/反粒子 | q, q̄ | 2 |
| 代 | 1, 2, 3 | 3 |
| 颜色 | R, G, B | 3 |
| **总计** | | **18种夸克 + 18种反夸克 = 36** |

加上同位旋二重态（u-type / d-type）的区分，总夸克构型数为36×2=72（含u,d,c,s,t,b各色各代）。

为保守估计，使用36（暂不区分u/d——在相变能量标度下u/d质量分裂尚未完全建立）。

**步骤2：B破坏过程的总事件数**

在Δt_{2→3} ≈ 4.2×10⁶ s ≈ 1.6月（50号§5.6）期间，等权连通持续。I场编码步长Δt_step ~ 100 s。相变期间的总编码步数：

$$N_{\text{steps}} \sim \frac{\Delta t_{2\to3}}{\Delta t_{\text{step}}} = \frac{4.2\times10^6}{100} = 4.2\times10^4$$

每一步中，所有36种夸克/反夸克构型经历一次拓扑扇区检查（是否有跃迁）。但由于拓扑扇区完全连通，"跃迁"不是概率事件——**在连通期间所有构型都等权混合**。关键量是扇区恢复后夸克/反夸克的净分配。

**步骤3：净重子数**

在等权连通→扇区恢复的过程中，CP破坏偏置ΔΓ/Γ ≈ 2.8×10⁻⁵（§4.2）使恢复后夸克构型净多出：

$$\Delta N_q = N_q^{\text{total}} \cdot \frac{\Delta\Gamma}{\Gamma} = 36 \cdot 2.8\times10^{-5} \approx 1.0\times10^{-3}$$

这是**每步编码**的净夸克产量。在$N_{\text{steps}} \approx 4.2\times10^4$步中累积：

但这不对——扇区只恢复一次（在跃迁结束时）。ΔN_q是跃迁结束时的总净夸克数，而非每步累积。

修正：跃迁期间36种构型完全混合，跃迁结束时拓扑恢复+CP偏置给出：

$$\frac{n_B}{n_q} \Big|_{\text{freeze}} \sim \frac{\Delta\Gamma}{\Gamma} \sim 2.8\times10^{-5}$$

**步骤4：转换为n_B/n_γ**

在N₂→N₃跃迁期间，能量标度从N₂的特征能标跃至N₃。光子（θ_M扇区电磁映射的产物）被大量产生。光子数与夸克数的比率来自截面统计权重：

- θ_C-共享截面占据约束流形相空间的约1/3（三扇区等权）
- 但光子产生于θ_M扇区，其统计权重在膨胀期间因真空极化而增大

膨胀期间光子的熵产生：

$$\frac{n_\gamma}{n_q} \sim r_{2\to3} \cdot g_*^{\text{eff}} \sim 33.3 \times 10 \sim 333$$

其中g_*^{\text{eff}} ≈ 10是有效自由度计数（夸克+轻子+规范玻色子）。

$$\eta_B \equiv \frac{n_B}{n_\gamma} \sim \frac{2.8\times10^{-5}}{333} \sim \mathbf{8.4\times10^{-8}}$$

### 6.3 与实验值的比较

| | η_B | 偏差 |
|:---|:---|:---|
| 几何论（树级估计） | 8.4×10⁻⁸ | — |

| 比 | — | **约140倍** |

**树级估计高估η_B约两个数量级。**

### 6.4 偏差的可能来源

140倍偏差的可能修正方向（按可能性排序）：

1. **非平衡抑制因子**（最可能）。§6.2假设了跃迁期间夸克-反夸克完全混合。实际上非平衡度ε_non-eq=5.56是有限值——拓扑扇区连通可能不是100%完全的。引入非平衡抑制因子f_non-eq：

   $$\frac{n_B}{n_q}\Big|_{\text{effective}} = f_{\text{non-eq}} \cdot \frac{\Delta\Gamma}{\Gamma}$$

   若f_non-eq ~ 7×10⁻³，则η_B可从8.4×10⁻⁸降至6×10⁻¹⁰。f_non-eq的来源是约束流形在跃迁期间的有效Hessian不完全退化为零。

2. **CP破坏在N₂能标下的跑动**。J_CP = 3×10⁻⁵是低能（电弱标度）的Jarlskog不变量。在N₂能标（对应T ~ 10¹⁵ GeV？）下，有效的CP破坏可能更小。从8号CKM推导的RG跑动方向来看，CP破坏在高能标下可能被抑制。

3. **熵稀释效应**。跃迁后的熵产生（粒子产生、宇宙再加热）会稀释n_B/n_γ。50号已给出三阶段膨胀，但具体的熵产生因子需等待E3（冻结临界点的物理标度）封闭。

4. **夸克-反夸克初始不对称**。如果N₂阶段已有微弱的初始B不对称（来自更早期的N₁→N₂跃迁），η_B的推导起点需要修正。N₁→N₂的ε_non-eq=6（仅次于N₂→N₃），可能存在前驱B破坏。

### 6.5 精确计算的路径

η_B的精确计算需要以下框架的完整建立：

- **I场Fokker-Planck方程在拓扑重构边界的非平衡解**：约束流形维度跃迁→扩散系数突变→分布函数的非平衡演化→冻结时刻的n_q分布
- **CP破坏相位在N₂能标下的RG跑动**：从8号CKM推导到电弱标度的反向RG流
- **熵产生因子**：光子产生率与N_info增长率的关系

这些均属于I场非平衡动力学范畴，当前（0.6.9系列）尚在建立中。

## 9. 推论与可检验预言

### 9.1 直接可检验

1. **η_B ≈ 6×10⁻¹⁰**（27号）。已在CMB+BAO观测中验证。

2. **质子寿命~10³⁴年**（18号）。这来自零温B破坏的指数压制（§3.2）。当前实验下限τ_p > 10³⁴年（Super-Kamiokande）。如果几何论正确，(a) 质子衰变必然存在（因为B在拓扑上可破坏）；(b) 但寿命远长于当前探测极限——未来Hyper-Kamiokande（灵敏度~10³⁵年）可能探测到。

### 9.2 间接可检验

3. **CP破坏来源的单一性**：CKM CP破坏（夸克扇区）是宇宙重子不对称的唯一来源。轻子扇区的CP破坏（PMNS相角δ_PMNS，7号）应通过轻子生成机制间接贡献——但在几何论中，重子生成主导窗口是N₂→N₃而非轻子生成——因此PMNS CP破坏对η_B的贡献应可忽略。这与纯轻子生成模型形成可检验的对立。

4. **N₂→N₃能标**：如果N₂→N₃对应电弱标度（命题1），则重子生成能标~100 GeV——远低于GUT能标（~10¹⁶ GeV）。这意味着：
   - 没有GUT标度的B破坏（排除了GUT重子生成）
   - 质子衰变不由GUT规范玻色子介导（与18号一致——几何论的质子衰变有不同机制）
   - 高能对撞机（LHC、未来对撞机）中的B破坏搜索无果（能标太低）

5. **反物质缺失的几何解释**：宇宙中不存在大尺度反物质区域——不是因为"初始条件"，而是因为CP偏置的符号在整个约束流形上是一致的。不存在"正物质区"和"反物质区"的分隔——全宇宙的θ_C纤维具有统一的复结构取向。

### 9.3 不可检验的（理论的"硬核"）

6. 约束流形的拓扑重构是离散事件——发生在N₂→N₃跃迁的~1.6月内。没有连续可调参数可供扫描。这使得"重子生成的精确时刻"在原理上不可直接观测——它被138亿年的宇宙膨胀和结构形成完全覆盖。

## 10. 开放问题

### O1（🟡 定量）：η_B精确值的封闭

当前树级估计8.4×10⁻⁸与实验值6×10⁻¹⁰差约140倍。需要：

- 非平衡抑制因子f_non-eq的精确计算：约束流形在有限ε_non-eq下的不完全拓扑连通度
- CP破坏RG跑动：J_CP(N₂) vs J_CP(电弱标度)
- 熵稀释因子：N₂→N₃膨胀期间的光子产生率

### O2（🟡 结构）：N₂→N₃能标的精确对应

命题1（N₂→N₃ ↔ 电弱能标）的严格证明需要规范群涌现（缺口1闭合）。当前此对应为物理合理的假设，未达到定理级别。

### O3（🟡 边界）：N₁→N₂跃迁的前驱B破坏

ε_non-eq(N₁→N₂) = 6/1 = 6，仅次于N₂→N₃的5.56。如果N₁→N₂期间已有粒子凝结，可能存在前驱B破坏——影响η_B的初值。N₁阶段的粒子凝结程度需要从因果推进定理定量评估。

### O4（🟢 外延）：与轻子生成的关系

如果PMNS CP破坏（7号）通过轻子生成间接贡献η_B，需计算B-L转换的速率。在几何论中B-L由w_{θ_C}-3w_M的不变量描述，其转换率需独立推导。

### O5（🟢 系统）：η_B的RG不变量

η_B在宇宙演化中是否守恒？是否有晚期B破坏过程（如黑洞蒸发或拓扑缺陷衰变）？需证明今日η_B = 早期宇宙冻结η_B。
