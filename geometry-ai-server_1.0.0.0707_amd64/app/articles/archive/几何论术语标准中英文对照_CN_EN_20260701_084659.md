## §11 维护规则

### §11.1 新增术语
任何新文章引入术语前，必须先在本表中登记。登记需提供：
1. 中文名 + 英文名
2. 精确定义或公式
3. 出处（文章编号 + 节号/定理号）
4. 分类（公理 / 定理 / 定义 / 概念名称 / 构造性概念）

### §11.2 术语冲突解决
若多篇文章对同一术语给出不同定义：
- **优先级**：卷0（0.0.X、0.1）> 桥梁卷（0.2-0.5）> 应用篇（1+）> 构造性概念
- 卷0内的冲突以版本号高者为准
- 应用篇如需偏离卷0定义，必须显式标注"本文约定"

### §11.3 定理标签使用
- "定理"标签仅限：有明确陈述 + 完整证明的命题
- 概念名称（九素互扼、谱刚性等打包叫法）→ 归入 §8，不得加"定理"后缀
- 构造性假设 → 归入 §9，标注"构造性假设"

### §11.4 禁用词执法
§10 中的禁用词不得出现在新文章中。若旧文章使用了禁用词，应在下次修订时修正。
