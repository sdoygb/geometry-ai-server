## 第2章　数学层的不可计算性

### §2.0　定位声明：核心映射层的不可约副产品

### §2.1　Cl(8)的基态涨落残余

Cl(8)具有128维不可约旋量表示Δ₁₂₈。在约化链Cl(9)→Cl(8)→Cl(7)→Cl(3,1)中：
- Cl(9) → Cl(8)：通过约束法向e₉投影，Δ₅₁₂ → Δ₊₂₅₆ ⊕ Δ₋₂₅₆
- Cl(8) → Cl(7)：通过f₈反对易子投影，Δ₂₅₆ → Δ₁₂₈ ⊕ Δ₁₂₈'（其中Δ₁₂₈'在Cl(7)下是零映射）
- Cl(7) → Cl(3,1)：通过G₃ 7-膜约化，Δ₁₂₈在Cl(3,1)下分解为物理费米子多重态

因此，128维反交换残余A(f₈)在物理表示中**恒为零**。但其数学结构包含：
- Spin(8) triality自同构的完整轨道（三类8维表示通过f₈反对易子关联）
- 对应于物理费米子在八维横向空间中的完整旋量结构，因e₉投影而冻结

**命题2.1（定理级）**：Cl(8)的128维反交换残余A(f₈)在物理Hilbert空间ℋ_phys = Δ₁₂₈(Cl(7))上的表示为恒等零映射：π_phys(A(f₈)) = 0。A(f₈)是非平凡的数学对象，归属Cl(8)⊗ℝ的范畴，但不在0.X公理系统的物理表示论中出现。

**证明**：由Clifford代数约化链的标准表示理论，A(f₈) = {ψ ∈ Δ₂₅₆(Cl(8)) : f₈·ψ = −ψ}（取f₈反对易于手征算符的适当约定）。Cl(8)→Cl(7)的约化通过选取f₈为Cl(8)中的一个生成元并将其从物理代数中移除来实现。Cl(7)的物理表示为{f₈·ψ = +ψ}子空间（Δ₁₂₈'），而{f₈·ψ = −ψ}子空间（A(f₈)）在Cl(7)的Clifford作用下不封闭——Cl(7)的任意元素g ∈ Cl(7)与f₈对易，故g·(f₈·ψ = −ψ子空间)仍满足f₈·(g·ψ) = g·(f₈·ψ) = g·(−ψ) = −(g·ψ)，故A(f₈)在Cl(7)作用下是封闭子空间。但Cl(7)旋量表示Δ₁₂₈的构造恰为选取f₈本征值+1的子空间，故A(f₈)在物理旋量表示中为零。证毕。

### §2.2　Cl(9)约束法向的不可约模

Cl(9)的生成元{e₁, ..., e₉}满足{eᵢ, eⱼ} = 2δᵢⱼ。几何约束曲面Σ由方程f(θ) = 0定义，其法向量场∂/∂θ在Cl(9)中对应生成元e₉。

Cl(9)的约束法向理想定义为：
$$\mathcal{I}(e_9) = \{x \in Cl(9) : x \cdot e_9 = 0\}$$

这是Cl(9)中的256维线性子空间（Cl(9)总维度2⁹=512，正交于e₉的子空间维度256）。ℐ(e₉)在约束面上形成闭理想——任意y ∈ Cl(9)，y·ℐ(e₉) ⊆ ℐ(e₉)加上边界项。

**命题2.2（定理级）**：约束法向理想ℐ(e₉)在约束面投影π_Σ下的像为零：π_Σ(ℐ(e₉)) = 0。ℐ(e₉)是法丛NΣ的代数实现，用于定义约束面的横截结构，但不贡献物理自由度。

**证明**：约束面Σ由f(θ)=0定义，约束面的切空间TΣ = ker(df)。e₉生成法向，故TΣ = {v ∈ ℝ⁹ : ⟨v, e₉⟩ = 0}。约束面投影π_Σ : ℝ⁹ → TΣ将法向分量归零。Cl(9)中，ℐ(e₉)的元素对应于法向代数结构——它们与e₉正交，但任何从ℐ(e₉)到TΣ的线性映射需首先包含沿e₉方向的平移（因ℐ(e₉)本身不含e₉分量），而该平移被约束面吸收。因此，π_Σ(ℐ(e₉)) ≡ 0。证毕。

### §2.3　递推链的n ≥ 8定义域边界

七级递推通过角度θ_M在完备性约束下的离散逼近给出S₀→S₇。递推公式基于θ_I→0时1/sin²θ_I的发散，逐级修正S值。n=8级对应于θ_I ≈ 0.74°（低于5.91°/8），该角度下sin³θ_I ≈ 1.7×10⁻⁶，质量映射m_I = K sin³θ_I ≈ 1.4 eV，落入声子能量范围——信息场"粒子化"，不再满足纯相位通道的公设。

**命题2.3（定理级）**：递推链Sₙ在n ≥ 8处超出几何论的定义域。n ≥ 8对应的θ_I值违反公理2的S(θ)在θ_I → 0时的发散阶分析——七阶泰勒展开将S(θ)在θ_I展开至O(θ_I⁷)，该阶余项已小于10⁻⁹。n=8的O(θ_I⁸)修正对应的S偏移ΔS < 10⁻¹⁰，低于S_e的当前实验不确定度。故n ≥ 8在实验上不可分辨，在数学上超出七阶截断的规范精度。

**证明**：由0.0.3号文章七级递推构造，Sₙ+₁ = Sₙ + ΔSₙ，其中ΔSₙ ∝ (∂ⁿ⁺¹/∂θ_Iⁿ⁺¹)(1/sin²θ_I)|_{θ_I=5.91°}。对1/sin²θ_I在θ_I=5.91°≈0.1032 rad处做泰勒展开，八阶项系数为O(10⁸)，乘以θ_I⁸≈(0.1032)⁸≈1.4×10⁻⁸，贡献ΔS₈≈1.4，但该修正需乘以七级递推的衰减因子∏_{k=1}^{7}r_k⁻¹≈(6000×33.33×10×10×1.125×2)⁻¹≈2.2×10⁻¹⁰，故有效ΔS₈^eff≈3×10⁻¹⁰ < 10⁻⁹。证毕。
