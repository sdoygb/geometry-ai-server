## §10 维护规则

### 10.1 新增术语登记流程

1. **提出**：在任何文章中出现新术语，必须在此表 §1-§8 对应分类中登记
2. **审核**：检查是否与已有术语冲突（同义词、歧义、命名不一致）
3. **登记**：填写中文名、英文名、定义、出处（文章编号 + 节号/定理号）
4. **传播**：登记后其他文章方可引用该术语

### 10.2 不允许的操作

- 在未登记情况下创造新术语并跨文章使用
- 为已有术语创建同义词（如"电子锚定"替代"ℰ 映射与公理1 联立确定"）
- 为同一概念在不同文章中使用不同名称

### 10.3 冲突解决

当发现两篇文章使用不同术语指代同一概念时：

1. 优先采用**更早发表**的文章中的术语
2. 若同时间，优先采用**卷0**中的术语
3. 若同卷，优先采用**定理级**文章中的术语
4. 在此表中记录冲突和裁决

### 10.4 版本同步

此表独立于文章版本体系。当任何文章修改术语定义时，此表同步更新。此表的时间戳记录最后一次术语变更时间。
