## 摘　要

三代轻子质量的层级结构（$m_e = 0.511$ MeV, $m_\mu = 105.66$ MeV, $m_\tau = 1776.86$ MeV）是粒子物理中一个未解决的基本问题。本文从十方几何的单一核心映射层（0.1）与层级约束截面理论（0.3.1.1）出发，分析三代轻子质量的刚性几何约束，并与丘成桐 Calabi-Yau 路径及小出 Koide 公式进行对比。

**核心发现（第5次修订）**：

1. **$W_{12}$ 显式推导**（本次修订新增）：从 23 号文章 §3.1 和 0.1.1 定理 4.2 出发，将 $W_{12}$ 从文字描述提升为精确代数形式。核心结果：$W_{12}^{(2)} = b'(\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)})(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)})$，其中 $b' = 2w - w' = 0.867$。双粒子联合截面驻点条件代数化——P0-1 闭合。

2. **缪子-因果角对应**（第3次修订）：以层级质量标度 $M_1 = K \Lambda_H S_e^{1/2} = 1.47$ GeV 为 $C_\mu$，取有效投影角为裸基准因果角 $\theta_C^0 = 26.16^\circ$，预言 $m_\mu^{\text{(th)}} = 126$ MeV，与实验值 106 MeV 偏差 19.6%——方向、量级和扇区归属均合理。对应有效角度偏移 $\Delta\theta \approx 1.56^\circ$。

3. **H^W 驻点分析**（第4次修订）：从 0.1 定理 4.2 的跨扇区耦合矩阵 $H^W$ 出发，建立双粒子联合截面上二代轻子角度偏移的严格变分框架。核心结论：单粒子 $H^W$ 微扰在构型 B（因果界）处灾难性失效（软模位移 $\sim 300^\circ$，二阶展开完全崩溃），迫使分析必须在**双粒子联合截面**上进行。反对称模式 $\lambda_-^W = 0.710$ 直接耦合 $\theta_C$，且其特征角度标度 $1/\sqrt{\lambda_-^W} = 1.187$ rad 与构型 A–B 间角距离 1.121 rad 仅差 5.6%。**$W_{12}$ 在第5次修订中已代数化——阻塞点从"待推导"转为"待代入求解"。**

4. **陶子层级归属**：$M_1$ 方案导致 $\sin^3\theta > 1$（溢出，需复化扩展）；$M_2 = K \Lambda_H^2 S_e^{1/2} = 221$ GeV 方案给出有效角度 $\approx 11.5^\circ$，无直接几何锚点。

5. **Koide 公式**：$C_n$ 不普适（$C_e = K, C_\mu = M_1, C_\tau = M_2$），$Q = 2/3$ 在 $C_n$ 不普适条件下等价于角度分布与 $C_n$ 比值的联合约束——P0 级开放问题。

本文所有假设以"假设 H"标注，所有猜想以"猜想 G"标注。电子质量在框架内完全封闭（"0.0.X + 单一核心映射即足够"）；缪子半封闭；陶子开放。P0-1（$W_{12}$ 显式形式）在第5次修订中闭合。

$$M_n = K \Lambda_H^n S_e^{1/2}, \quad n = 0, 1, \dots, 7$$

**关键区分**：电子质量 $m_e = K \sin^3\theta_M^0$（公理 3 质量映射），不等于 $M_0$。$M_n$ 是**层级标度**，$K$ 是**质量量子**。$C_n$（见 §3.2）在 $n=0$ 时取 $C_0 = K$（非 $M_0$）。

## §3　三代轻子质量的几何结构

### 3.1 层级质量映射假设

**假设 H1（层级质量映射）**：第 $n$ 代轻子质量由层级标度 $C_n$ 与有效物质角 $\theta^{(n)}$ 给出：

$$m_n = C_n \sin^3 \theta^{(n)}$$

其中：
- $C_0 = K = 839.76$ keV（电子，框架内定理）
- $C_1 = M_1 = K \Lambda_H S_e^{1/2} = 1.47$ GeV（缪子候选，研究框架）
- $C_2 = M_2 = K \Lambda_H^2 S_e^{1/2} = 221$ GeV（陶子候选，研究框架）

**假设 H2（三代角度和约束）**：$\theta^{(1)} + \theta^{(2)} + \theta^{(3)} = 90^\circ$（公理 1 向三代粒子角度约束的推广，待严格化）。

### 3.2 电子的框架内自足性 ✅

$m_e = K \sin^3\theta_M^0 = 839.76 \times \sin^3 57.93^\circ = 839.76 \times 0.6085 = 511.0$ keV。

电子是三代轻子中**唯一**达到"0.0.X + 单一核心映射即足够"标准的。其质量映射、裸基准角、有效 Hessian 全部在框架内封闭。

### 3.3 缪子的半封闭状态 ⚠️

在 H1 下取 $C_1 = M_1$，有效角度候选为 $\theta_C^0 = 26.16^\circ$（构型 B 的物质角，等于裸基准因果角）：

$$m_\mu^{\text{(th)}}(\theta_C^0) = M_1 \sin^3 26.16^\circ = 1.47\ \text{GeV} \times 0.08557 = 126\ \text{MeV}$$

实验值 $m_\mu^{\text{exp}} = 105.66$ MeV。偏差 $\Delta m / m = 19.6\%$。

**反推有效角度**：$\theta^{(2)}_{\text{eff}} = \arcsin((m_\mu^{\text{exp}}/M_1)^{1/3}) = 24.60^\circ$。偏移 $\Delta\theta = 26.16^\circ - 24.60^\circ = 1.56^\circ$。

**评估**：方向（$\theta_C^0$ 所在扇区）、量级（百 MeV）、角度偏移（$1.56^\circ$）均合理。这是当前框架对二代轻子最正向的数值锚点——缪子是半封闭的，偏差的来源需要从几何第一原理推导。

### 3.4 陶子的开放状态 ❌

陶子的层级归属有两个候选方案：

**(A) $M_1$ 方案（$C_2 = M_1 = 1.47$ GeV）**
反推有效角度：$\theta^{(3)}_{\text{eff}} = \arcsin((m_\tau/M_1)^{1/3}) = \arcsin(1.065) > 90^\circ$ — **$\sin^3\theta > 1$，实数角度域溢出。**

**(B) $M_2$ 方案（$C_2 = M_2 = 221$ GeV）**
反推有效角度：$\theta^{(3)}_{\text{eff}} = \arcsin((m_\tau/M_2)^{1/3}) \approx 11.5^\circ$。

$(11.5^\circ)_{M_2}$ 在实数域内，但与 $\theta_M^0 = 57.93^\circ$、$\theta_C^0 = 26.16^\circ$、$\theta_I^0 = 5.91^\circ$ 均无直接几何对应。它落在信息扇区与因果扇区之间，无框架内锚点。

## §5　H^W 驻点分析与 19.6% 偏差（第4–5次修订）

### §5.1 问题设定

缪子预言值 $m_\mu^{\text{(th)}}(\theta_C^0) = 126$ MeV 与实验值 106 MeV 的 19.6% 偏差等价于有效角度偏移 $\Delta\theta = 1.56^\circ$。本节从 $H^W$ 驻点分析出发，回答：**这个 $1.56^\circ$ 能否从几何第一原理推导？**

### §5.2 构型 B 的梯度与 Hessian

构型 B 是构型 A 的 S₃ 轮换像：$P_B = (26.16^\circ, 5.91^\circ, 57.93^\circ)$。

#### §5.2.1 梯度计算

由 $S(\theta)$ 六项作用量的梯度，在构型 B 处：

$$\begin{aligned}
\partial S/\partial\theta_M|_{P_B} &= \partial S/\partial\theta_C|_{P_A} \quad (\text{S}_3\ \text{轮换对称性}) \\
&= -2\csc^2\theta_C\cot\theta_C - \csc^2\theta_M\cot\theta_C\csc\theta_I - \csc^2\theta_I\cot\theta_C\csc\theta_M|_{P_A} \\
&\approx 216.7\ \text{rad}^{-1}.
\end{aligned}$$

$$\partial S/\partial\theta_C|_{P_B} \approx -239.3\ \text{rad}^{-1}, \quad \partial S/\partial\theta_I|_{P_B} \approx 22.6\ \text{rad}^{-1}.$$

在角度和约束下，切向梯度 $\nabla_T S \approx 2000$ rad⁻¹——远大于有效本征值 $\lambda_1^{\text{eff}} = 391.05$ rad⁻²。

#### §5.2.2 Hessian 计算

裸 Hessian 矩阵元（$i,j \in \{M,C,I\}$）：

$$H_{ij} = \partial^2 S/\partial\theta_i\partial\theta_j.$$

在构型 A（电子）处：$H_{MM} \approx 367.73$, $H_{CC} \approx 367.73$, $H_{II} \approx 59246.2$, $H_{MC} = H_{CM} \approx 69.35$, $H_{CI} = H_{IC} \approx 69.35$, $H_{MI} = H_{IM} \approx 58369.2$。

由 S₃ 轮换，构型 B 的 Hessian 为：

$$H_{MM}^B = H_{CC}^A = 367.73, \quad H_{CC}^B = H_{II}^A = 59246.2, \quad H_{II}^B = H_{MM}^A = 367.73,$$

$$H_{MC}^B = H_{CI}^A = 69.35, \quad H_{CI}^B = H_{MI}^A = 58369.2, \quad H_{MI}^B = H_{MC}^A = 69.35.$$

在 $(\xi, \eta) = (\theta_I - \theta_C, \theta_C - \theta_M)$ 坐标中：

$$H_{\xi\xi}^B = 59151.6, \quad H_{\eta\eta}^B = 58760.8, \quad H_{\xi\eta}^B = -58892.1.$$

与构型 A 的 $(\lambda_1 = 392.21, \lambda_2 = 58760.77, H_{\xi\eta} \approx 0)$ 不同，构型 B 中两个方向均为硬模（$\sim 5.9 \times 10^4$），且强混合（$H_{\xi\eta} \sim -5.9 \times 10^4$）。

### §5.3 单粒子 H^W 微扰的灾难性失效

尝试在构型 B 处施加单粒子联合变分 $\delta(S + W_{\text{total}}) = 0$。在 $(\xi, \eta)$ 坐标中，$W_{\text{total}}$ 的 Hessian 为 $H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix}$（0.1.1 定理 4.2）。驻点条件：

$$\nabla_T S|_{P_B} + (H_{\text{bare}}^B + H^W) \Delta = 0, \quad \Delta = \begin{pmatrix} \Delta\xi \\ \Delta\eta \end{pmatrix}.$$

$H_{\text{bare}}^B$ 在 $(\xi,\eta)$ 坐标下强混合（$H_{\xi\eta}^B \sim -5.9 \times 10^4$），对角近似不可用。$\|\nabla S\| \sim 2000$ rad⁻¹，$\lambda_{\min} \sim 392$ rad⁻²，故 $\|\Delta\| \sim 5$ rad $\approx 300^\circ$。

**结论**：**单粒子 H^W 微扰在构型 B 处灾难性失效。** 构型 B 离 $S$ 的极小点太远，二阶展开完全不适用。缪子不能作为构型 B 附近的单粒子 H^W 微扰来分析。

### §5.4 为什么必须用双粒子联合截面

上述失效有明确的物理含义：**缪子不是独立存在的。它存在于电子-缪子双粒子系统中。** 正确的变分框架是双粒子联合截面：

$$S_{\text{joint}} = S^{(1)}(\theta^{(1)}) + S^{(2)}(\theta^{(2)}) + W_{12}(\theta^{(1)}, \theta^{(2)}),$$

其中粒子 1（电子）在构型 A 近旁，粒子 2（缪子）在构型 B 近旁，$W_{12}$ 为跨粒子耦合。

**关键区分**：
- $H^W$（0.1.1 定理 4.2）描述**单粒子内部**三扇区（$\mathcal{M}, \mathcal{C}, \mathcal{I}$）之间的跨扇区耦合。
- $W_{12}$（23 号文章 §3.1）描述**两个粒子之间**的耦合，**继承** $H^W$ 的结构但作用在不同对象上。

23 号文章 §3.1 明确指出：$W_{12}$ 的系数由 $\kappa_w, \kappa_w'$ 对 $w, w'$ 加权后生成。$H^W$ 矩阵的本征值 $a' \pm b'$ 对应联合截面上两粒子的对称/反对称耦合模式。

**M-扇区同步约束的失效**：23 号文章 §2.1 的 M-扇区同步约束 $\theta_M^{(1)} = \theta_M^{(2)}$ 是夸克偶素（$c\bar{c}$、$b\bar{b}$）场景下的合理近似（两夸克质量相近）。对于电子-缪子系统（$m_\mu/m_e \approx 207$），$\theta_M^{(1)} \neq \theta_M^{(2)}$ 必然是成立的。因此电子-缪子联合截面需要**放松 M-扇区同步约束**——两个粒子的物质角独立。

### §5.5 H^W 本征值与耦合模式

$H^W$ 矩阵（0.1.1 定理 4.2）：

$$H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} = \begin{pmatrix} 1.577 & 0.867 \\ 0.867 & 1.577 \end{pmatrix}.$$

本征值与本征向量：

$$\lambda_+^W = a' + b' = 2.444, \quad v_+ = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix} \quad (\xi + \eta\ \text{方向}),$$

$$\lambda_-^W = a' - b' = 0.710, \quad v_- = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix} \quad (\xi - \eta\ \text{方向}).$$

在 $(\theta_M, \theta_C, \theta_I)$ 坐标中：

$$\begin{aligned}
\xi + \eta &= (\theta_I + \theta_M) + (\theta_C + \theta_M) = \theta_I + \theta_C + 2\theta_M = 90^\circ + \theta_M, \\
\xi - \eta &= (\theta_I + \theta_M) - (\theta_C + \theta_M) = \theta_I - \theta_C.
\end{aligned}$$

**关键观察**：反对称模式（$v_-$，$\lambda_-^W = 0.710$）的变分方向是 $\xi - \eta = \theta_I - \theta_C$。在角度和约束 $\theta_M + \theta_C + \theta_I = 90^\circ$ 下：

$$\theta_I - \theta_C = (90^\circ - \theta_M - \theta_C) - \theta_C = 90^\circ - \theta_M - 2\theta_C.$$

在构型 A（$\theta_M = 57.93^\circ$, $\theta_C = 26.16^\circ$）处：

$$\theta_I - \theta_C = 90^\circ - 57.93^\circ - 2 \times 26.16^\circ = -20.25^\circ.$$

**反对称模式直接与 $\theta_C$ 耦合**——这是 $H^W$ 结构对缪子问题的核心含义：$\lambda_-^W$ 控制 $\theta_C$ 方向的曲率修正。任何 $\theta_C$ 的改变都会激发反对称模式，回复力为 $\lambda_-^W$。

### §5.6 数值关系：$H^W$ 特征标度与构型间距

反对称模式的特征角度标度：

$$\Delta\theta_{\text{char}} = \frac{1}{\sqrt{\lambda_-^W}} = \frac{1}{\sqrt{0.710}} = 1.187\ \text{rad} = 68.0^\circ.$$

构型 A（电子）与构型 B（缪子候选）在约束流形上的角距离：

$$d_{AB} = \sqrt{(57.93^\circ - 26.16^\circ)^2 + (26.16^\circ - 5.91^\circ)^2 + (5.91^\circ - 57.93^\circ)^2} = 64.2^\circ = 1.121\ \text{rad}.$$

**观察**：$d_{AB} / \Delta\theta_{\text{char}} = 1.121 / 1.187 = 0.944$，偏差仅 5.6%。$H^W$ 反对称模式的特征标度与电子-缪子构型间距在数值上邻近。

**对称模式的特征标度**：$1/\sqrt{\lambda_+^W} = 0.640$ rad $= 36.6^\circ$。构型 A 与 B 在 $\theta_I + \theta_C + 2\theta_M$ 方向的分离：$|(5.91^\circ + 26.16^\circ + 2 \times 57.93^\circ) - (57.93^\circ + 5.91^\circ + 2 \times 26.16^\circ)| = |147.93^\circ - 116.16^\circ| = 31.77^\circ = 0.554$ rad。与 0.640 rad 偏差 15%。

**推论**：$H^W$ 矩阵的两个本征标度（1.187 rad 和 0.640 rad）与 S₃ 轮换构型之间的几何间距（1.121 rad 和 $\sim 0.55$ rad）在量级上匹配。这暗示 $H^W$ 的结构内禀编码了不同 S₃ 构型之间的角度关系信息——这是 $H^W$ 驻点分析可用于二代轻子的群论基础。

### §5.9 $W_{12}$ 的显式推导（第5次修订）

本节将 23 号文章 §3.1 对 $W_{12}$ 的文字描述提升为精确代数形式。推导分六步，每步有定理支撑。

#### §5.9.1 单粒子 $H^W$ 结构回顾

由 0.1.1 定理 4.2，单粒子跨扇区耦合 Hessian 在 $(\xi, \eta)$ 坐标下为：

$$H^W = \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix}, \quad a' = -2w = 1.577, \quad b' = 2w - w' = 0.867.$$

其中 $\xi = \theta_I + \theta_M$，$\eta = \theta_C + \theta_M$（0.1 §3.4）。本征值 $\lambda_\pm^W = a' \pm b'$，本征向量 $v_\pm = (1, \pm 1)/\sqrt{2}$。

单粒子六项作用量 $S(\theta)$ 在展开点 $P_0$ 附近的二次型为：

$$S^{(2)} = \frac{1}{2}\begin{pmatrix}\delta\xi & \delta\eta\end{pmatrix} H_{\text{bare}} \begin{pmatrix}\delta\xi \\ \delta\eta\end{pmatrix}.$$

总单粒子 Hessian（含跨扇区耦合）：$H_{\text{total}}^{(1)} = H_{\text{bare}} + H^W$.

#### §5.9.2 双粒子耦合 Hessian 的对称/反对称分解

在双粒子联合截面上，定义两个粒子的 $(\xi, \eta)$ 坐标：

$$\xi^{(1)} = \theta_I^{(1)} + \theta_M^{(1)}, \quad \eta^{(1)} = \theta_C^{(1)} + \theta_M^{(1)},$$
$$\xi^{(2)} = \theta_I^{(2)} + \theta_M^{(2)}, \quad \eta^{(2)} = \theta_C^{(2)} + \theta_M^{(2)}.$$

在 M-扇区同步约束 $\theta_M^{(1)} = \theta_M^{(2)} \equiv \theta_M$ 下（23 号文章 §2.1），联合截面维数为 3（定理 2.1）。在放松 M-扇区同步的电子-缪子情形中，$\theta_M^{(1)} \neq \theta_M^{(2)}$，联合截面维数为 4（见 §5.4）。

定义对称/反对称组合：

$$\xi_+ = \frac{\xi^{(1)} + \xi^{(2)}}{\sqrt{2}}, \quad \xi_- = \frac{\xi^{(1)} - \xi^{(2)}}{\sqrt{2}},$$
$$\eta_+ = \frac{\eta^{(1)} + \eta^{(2)}}{\sqrt{2}}, \quad \eta_- = \frac{\eta^{(1)} - \eta^{(2)}}{\sqrt{2}}.$$

变换矩阵 $U = \frac{1}{\sqrt{2}}\begin{pmatrix} I_2 & I_2 \\ I_2 & -I_2 \end{pmatrix}$ 连接 $(\xi^{(1)}, \eta^{(1)}, \xi^{(2)}, \eta^{(2)})$ 与 $(\xi_+, \eta_+, \xi_-, \eta_-)$。

#### §5.9.3 总耦合 Hessian 在对称/反对称基下的结构

23 号文章 §3.1 声明："$H^W$ 矩阵的本征值 $a' \pm b'$ 对应联合截面上两粒子的对称/反对称耦合模式。"

**引理 5.1**：若双粒子总耦合 Hessian $H_{\text{total}}^{(2)}$ 在对称/反对称基下对角化，且对称子空间的本征值为 $\lambda_+^W = a' + b'$（各向同性）、反对称子空间的本征值为 $\lambda_-^W = a' - b'$（各向同性），则：

$$H_{\text{total}}^{(2)} = \begin{pmatrix} \lambda_+^W I_2 & 0 \\ 0 & \lambda_-^W I_2 \end{pmatrix}_{\xi_\pm,\eta_\pm} = \begin{pmatrix} a' I_2 & b' I_2 \\ b' I_2 & a' I_2 \end{pmatrix}_{\xi^{(1)},\eta^{(1)},\xi^{(2)},\eta^{(2)}}.$$

*证明*：在对称/反对称基下的对角形式 $\operatorname{diag}(\lambda_+^W I_2, \lambda_-^W I_2)$ 通过 $U$ 变换回原始基：

$$\begin{aligned} H_{\text{total}}^{(2)} &= U \begin{pmatrix} \lambda_+^W I_2 & 0 \\ 0 & \lambda_-^W I_2 \end{pmatrix} U^T \\ &= \frac{1}{2}\begin{pmatrix} I_2 & I_2 \\ I_2 & -I_2 \end{pmatrix} \begin{pmatrix} \lambda_+^W I_2 & 0 \\ 0 & \lambda_-^W I_2 \end{pmatrix} \begin{pmatrix} I_2 & I_2 \\ I_2 & -I_2 \end{pmatrix} \\ &= \frac{1}{2}\begin{pmatrix} (\lambda_+^W + \lambda_-^W) I_2 & (\lambda_+^W - \lambda_-^W) I_2 \\ (\lambda_+^W - \lambda_-^W) I_2 & (\lambda_+^W + \lambda_-^W) I_2 \end{pmatrix} \\ &= \begin{pmatrix} a' I_2 & b' I_2 \\ b' I_2 & a' I_2 \end{pmatrix}. \quad \square \end{aligned}$$

#### §5.9.4 单粒子 Hessian 直和

无跨粒子耦合时，双粒子 Hessian 为两个单粒子 $H^W$ 的直和：

$$H_{\text{single}}^{(2)} = \begin{pmatrix} H^W & 0 \\ 0 & H^W \end{pmatrix} = \begin{pmatrix} \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} & 0 \\ 0 & \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} \end{pmatrix}.$$

在对称/反对称基下，直和保持不变：$H_{\text{single}}^{(2),\text{diag}} = \operatorname{diag}(H^W, H^W)$（因为两个块相等，变换不变）。

#### §5.9.5 $W_{12}$ Hessian 的提取

$W_{12}$ 的 Hessian 为总耦合 Hessian 与单粒子直和的差：

$$H_{W_{12}} = H_{\text{total}}^{(2)} - H_{\text{single}}^{(2)}.$$

在对称/反对称基下：

$$\begin{aligned} H_{W_{12}}^{\text{diag}} &= \begin{pmatrix} \lambda_+^W I_2 & 0 \\ 0 & \lambda_-^W I_2 \end{pmatrix} - \begin{pmatrix} H^W & 0 \\ 0 & H^W \end{pmatrix} \\ &= \begin{pmatrix} \lambda_+^W I_2 - H^W & 0 \\ 0 & \lambda_-^W I_2 - H^W \end{pmatrix}. \end{aligned}$$

其中：

$$\lambda_+^W I_2 - H^W = \begin{pmatrix} a'+b' & 0 \\ 0 & a'+b' \end{pmatrix} - \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} = \begin{pmatrix} b' & -b' \\ -b' & b' \end{pmatrix},$$

$$\lambda_-^W I_2 - H^W = \begin{pmatrix} a'-b' & 0 \\ 0 & a'-b' \end{pmatrix} - \begin{pmatrix} a' & b' \\ b' & a' \end{pmatrix} = \begin{pmatrix} -b' & -b' \\ -b' & -b' \end{pmatrix}.$$

变换回原始基（通过 $U$）：

$$H_{W_{12}} = \begin{pmatrix} \begin{pmatrix} 0 & -b' \\ -b' & 0 \end{pmatrix} & \begin{pmatrix} b' & 0 \\ 0 & b' \end{pmatrix} \\ \begin{pmatrix} b' & 0 \\ 0 & b' \end{pmatrix} & \begin{pmatrix} 0 & -b' \\ -b' & 0 \end{pmatrix} \end{pmatrix}_{\xi^{(1)},\eta^{(1)},\xi^{(2)},\eta^{(2)}}.$$

**定理 5.1（$W_{12}$ Hessian）**：双粒子跨粒子耦合 $W_{12}$ 在 $(\xi^{(1)},\eta^{(1)},\xi^{(2)},\eta^{(2)})$ 坐标下的 Hessian 矩阵为上述 $H_{W_{12}}$，仅含 $b' = 2w - w' = 0.867$，不含 $a'$。

$W_{12}$ 的二次型：

$$W_{12}^{(2)} = \frac{1}{2} \begin{pmatrix} \delta\xi^{(1)} & \delta\eta^{(1)} & \delta\xi^{(2)} & \delta\eta^{(2)} \end{pmatrix} H_{W_{12}} \begin{pmatrix} \delta\xi^{(1)} \\ \delta\eta^{(1)} \\ \delta\xi^{(2)} \\ \delta\eta^{(2)} \end{pmatrix}.$$

展开：

$$\begin{aligned} W_{12}^{(2)} &= -b' \delta\xi^{(1)} \delta\eta^{(1)} + b' \delta\xi^{(1)} \delta\xi^{(2)} + b' \delta\eta^{(1)} \delta\eta^{(2)} - b' \delta\xi^{(2)} \delta\eta^{(2)} \\ &= b' \left( \delta\xi^{(1)} \delta\xi^{(2)} - \delta\xi^{(1)} \delta\eta^{(1)} + \delta\eta^{(1)} \delta\eta^{(2)} - \delta\xi^{(2)} \delta\eta^{(2)} \right). \end{aligned}$$

#### §5.9.6 二次型的因子化

**引理 5.2（因子化）**：$W_{12}$ 的二次型可因子化为：

$$W_{12}^{(2)} = b' (\delta\xi^{(1)} - \delta\eta^{(2)})(\delta\xi^{(2)} - \delta\eta^{(1)}).$$

*证明*：

$$\begin{aligned} (\delta\xi^{(1)} - \delta\eta^{(2)})(\delta\xi^{(2)} - \delta\eta^{(1)}) &= \delta\xi^{(1)} \delta\xi^{(2)} - \delta\xi^{(1)} \delta\eta^{(1)} - \delta\eta^{(2)} \delta\xi^{(2)} + \delta\eta^{(2)} \delta\eta^{(1)} \\ &= \delta\xi^{(1)} \delta\xi^{(2)} - \delta\xi^{(1)} \delta\eta^{(1)} - \delta\xi^{(2)} \delta\eta^{(2)} + \delta\eta^{(1)} \delta\eta^{(2)}. \end{aligned}$$

每一项与 §5.9.5 展开式一一对应。$\square$

#### §5.9.7 角度坐标表示

将 $\xi^{(i)} = \theta_I^{(i)} + \theta_M^{(i)}$，$\eta^{(i)} = \theta_C^{(i)} + \theta_M^{(i)}$ 代入因子化形式。展开基准点设为 $P_0$（任意），偏移量 $\Delta\theta = \theta - \theta^0$。

$$\begin{aligned} \delta\xi^{(1)} - \delta\eta^{(2)} &= (\Delta\theta_I^{(1)} + \Delta\theta_M^{(1)}) - (\Delta\theta_C^{(2)} + \Delta\theta_M^{(2)}), \\ \delta\xi^{(2)} - \delta\eta^{(1)} &= (\Delta\theta_I^{(2)} + \Delta\theta_M^{(2)}) - (\Delta\theta_C^{(1)} + \Delta\theta_M^{(1)}). \end{aligned}$$

**情形一（M-扇区同步，23 号文章约束）**：$\theta_M^{(1)} = \theta_M^{(2)} = \theta_M$，故 $\Delta\theta_M^{(1)} = \Delta\theta_M^{(2)} = \Delta\theta_M$。

$$\begin{aligned} \delta\xi^{(1)} - \delta\eta^{(2)} &= \Delta\theta_I^{(1)} - \Delta\theta_C^{(2)}, \\ \delta\xi^{(2)} - \delta\eta^{(1)} &= \Delta\theta_I^{(2)} - \Delta\theta_C^{(1)}. \end{aligned}$$

$W_{12}$ 在 M-扇区同步下的二次型：

$$\boxed{W_{12}^{(2)} = b' (\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)})(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)})}.$$

**情形二（放松 M-扇区同步，电子-缪子系统）**：$\theta_M^{(1)} \neq \theta_M^{(2)}$。

$$\begin{aligned} \delta\xi^{(1)} - \delta\eta^{(2)} &= \Delta\theta_I^{(1)} - \Delta\theta_C^{(2)} + (\Delta\theta_M^{(1)} - \Delta\theta_M^{(2)}), \\ \delta\xi^{(2)} - \delta\eta^{(1)} &= \Delta\theta_I^{(2)} - \Delta\theta_C^{(1)} - (\Delta\theta_M^{(1)} - \Delta\theta_M^{(2)}). \end{aligned}$$

令 $\Delta_M = \Delta\theta_M^{(1)} - \Delta\theta_M^{(2)}$。则：

$$\boxed{W_{12}^{(2)} = b' \left[(\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)}) + \Delta_M\right] \cdot \left[(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)}) - \Delta_M\right]}.$$

**定理 5.2（$W_{12}$ 角度函数——二次型）**：在双粒子联合截面上，跨粒子耦合 $W_{12}$ 在展开点 $P_0$ 附近的角度函数二次型由上述公式给出，其中 $b' = 2w - w' = 0.867$。该公式仅依赖框架已锁定的常数 $w, w'$（0.1.1 §4.5–4.6），不含任何自由参数。

**注 5.1（二阶截断的适用性）**：定理 5.2 给出的是二次型。电子在构型 A 附近（$S$ 极小点），其二阶展开高度精确（$\lambda_1^{\text{eff}} = 391.05$，$\lambda_2^{\text{eff}} = 59324.3$，切向梯度为零）。缪子不在 $S$ 极小点附近（§5.3 已证明单粒子二阶展开灾难性失效），但联合截面上电子作为"锚"约束粒子 1 靠近极小点，粒子 2 通过 $W_{12}$ 与粒子 1 耦合——联合系统的有效 Hessian 由定理 5.1 给出，在对称子空间中的软模为 $\lambda_+^W = 2.444$。$W_{12}$ 的二次型是该联合 Hessian 的核心组成部分。

**注 5.2（高阶项）**：$W_{12}$ 的高阶项（三次及以上）由 23 号文章 §3.1 所述的"5 号文章和 0.3.5 的完整展开确定"。当前阶段，二次型已足够处理 19.6% 偏差的领头阶贡献——因为 $\lambda_+^W = 2.444$ 和 $\lambda_-^W = 0.710$ 均为 $O(1)$ 量级，而裸 Hessian 的软模 $\lambda_1^{\text{eff}} = 391.05$，硬模 $\lambda_2^{\text{eff}} = 59324.3$。$W_{12}$（$\sim b' = 0.867$）在与 $\lambda_\pm^W$ 竞争时不可忽略，但远小于 $\lambda_1^{\text{eff}}$。

### §5.10 联合截面驻点条件代数化（第5次修订）

#### §5.10.1 联合作用量

$$S_{\text{joint}} = S(\theta^{(1)}) + S(\theta^{(2)}) + W_{12}(\theta^{(1)}, \theta^{(2)}).$$

六项作用量 $S(\theta)$ 由公理 2 给出，$W_{12}$ 由定理 5.2 给出。

在电子-缪子系统中：
- 粒子 1（电子）：$\theta^{(1)}$ 在构型 A $(\theta_M^0, \theta_C^0, \theta_I^0) = (57.93^\circ, 26.16^\circ, 5.91^\circ)$ 近旁。
- 粒子 2（缪子）：$\theta^{(2)}$ 在构型 B $(\theta_M = 26.16^\circ, \theta_C = 5.91^\circ, \theta_I = 57.93^\circ)$ 近旁，但 $\theta^{(2)}$ 的有效投影角接近 $\theta_C^0 = 26.16^\circ$（因果角对应）。

#### §5.10.2 展开基准点选择

选择展开基准点 $P_0 = (\theta^{(1),0}, \theta^{(2),0})$：

$$\theta^{(1),0} = P_A = (57.93^\circ, 26.16^\circ, 5.91^\circ),$$
$$\theta^{(2),0} = P_B = (26.16^\circ, 5.91^\circ, 57.93^\circ).$$

在 $P_0$ 处：
- $\nabla S^{(1)}|_{P_A} = 0$（构型 A 是 $S$ 的极小点）
- $\nabla S^{(2)}|_{P_B} \neq 0$（构型 B 远离极小点，梯度 $\sim 2000$ rad⁻¹，见 §5.2.1）

#### §5.10.3 联合变分方程

联合作用量的驻点条件：

$$\frac{\partial S_{\text{joint}}}{\partial\theta_i^{(k)}} = 0, \quad i \in \{M,C,I\}, \quad k \in \{1,2\}.$$

展开到二阶：

$$\frac{\partial S^{(1)}}{\partial\theta_i^{(1)}}\Bigg|_{P_A} + \sum_j H_{ij}^{A} \Delta\theta_j^{(1)} + \frac{\partial W_{12}}{\partial\theta_i^{(1)}}\Bigg|_{P_0} = 0,$$

$$\frac{\partial S^{(2)}}{\partial\theta_i^{(2)}}\Bigg|_{P_B} + \sum_j H_{ij}^{B} \Delta\theta_j^{(2)} + \frac{\partial W_{12}}{\partial\theta_i^{(2)}}\Bigg|_{P_0} = 0.$$

其中 $H^A = H_{\text{bare}}^A + H^W$（电子有效 Hessian，正定），$H^B = H_{\text{bare}}^B + H^W$（构型 B 裸 Hessian + $H^W$）。

电子在极小点处 $\nabla S^{(1)}|_{P_A} = 0$。若 $W_{12}$ 的梯度在 $P_0$ 处为零（$W_{12}$ 仅含二次交叉项，一阶导数为零），则电子的驻点条件简化为：

$$\sum_j H_{ij}^{A} \Delta\theta_j^{(1)} + O(W_{12}') = 0 \implies \Delta\theta^{(1)} \approx 0.$$

电子保持在极小点附近——这是预期的：电子质量已在框架内锁定。

缪子的驻点条件：

$$\nabla S^{(2)}|_{P_B} + H^B \Delta\theta^{(2)} + \nabla_{\theta^{(2)}} W_{12}|_{P_0} = 0.$$

$\nabla S^{(2)}|_{P_B} \sim 2000$ rad⁻¹（§5.2.1），$H^B$ 在 $(\xi,\eta)$ 坐标下强混合（§5.2.2）。

**但这正是单粒子微扰失效的同一方程（§5.3）。** 如果 $W_{12}$ 的一阶导数为零，联合截面驻点条件退化为单粒子方程——这意味着 $W_{12}$ 的二次型本身不足以将缪子拉向因果角附近。

#### §5.10.4 关键洞察：$W_{12}$ 的一阶贡献

上述分析暴露了一个关键点：**$W_{12}$ 在 $P_0$ 处的一阶导数为零**——因为 $W_{12}$ 的二次型不含线性项。但电子不在严格的 $P_A$（其有效角度 $\theta^{(1)}$ 在 $P_A$ 处），缪子也不在严格的 $P_B$（其有效角度 $\theta^{(2)}$ 在 $\theta_C^0$ 附近，而非 $P_B$ 的 5.91°）。

正确的展开基准点应当是：粒子 1 在 $P_A$（电子精确在此），粒子 2 在**因果角 $\theta_C^0 = 26.16^\circ$ 附近**——而非 $P_B$ 的物质角位置。

粒子 1 在 $P_A$：$\Delta\theta^{(1)}_M = \Delta\theta^{(1)}_C = \Delta\theta^{(1)}_I = 0$（电子锁定）。
粒子 2 的展开点：$\theta_M^{(2),0} = ?$, $\theta_C^{(2),0} = \theta_C^0 = 26.16^\circ$, $\theta_I^{(2),0} = 90^\circ - \theta_M^{(2),0} - 26.16^\circ$。

$\theta_M^{(2),0}$ 不是 $P_B$ 的 26.16°（那是物质角），也不是 $P_B$ 的任何一个坐标——$P_B$ 根本就没有 $\theta_C = 26.16^\circ$ 的坐标。构型 B = (26.16°, 5.91°, 57.93°) 的三个角分别是 $\theta_M, \theta_C, \theta_I$，其中 $\theta_M = 26.16^\circ = \theta_C^0$。

所以$\theta_M^{(2),0} = 26.16^\circ$（缪子物质角 = 裸基准因果角），$\theta_C^{(2),0} = 5.91^\circ$（缪子因果角 = 裸基准信息角）。这是构型 B 的原始配置。

但在缪子-因果角对应中，缪子的有效投影角在因果扇区（$\approx \theta_C^0$），而非物质扇区。这意味着我们需要区分**裸角度**（构型 B 的坐标）和**有效投影角**（质量映射中的角度）。

**诚实标注**：这是当前框架中的一个概念缺口——"有效投影角"与"裸角度"的关系尚未几何化。定理 5.2 使用裸角度坐标 $(\theta_M, \theta_C, \theta_I)$，而质量映射 $m = C \sin^3\theta$ 中的 $\theta$ 应理解为有效物质角。两者的区分是 P0-2（二代轻子几何定义）的核心。

#### §5.10.5 驻点条件的代数形式（概念层面）

尽管有上述概念缺口，$W_{12}$ 的显式形式使驻点条件成为代数可操作问题。令粒子 2 的独立变量为 $\theta_C^{(2)}$（因果角，缪子质量的主要敏感参数）。在角度和约束下，$\theta_I^{(2)} = 90^\circ - \theta_M^{(2)} - \theta_C^{(2)}$。

六项作用量 $S^{(2)}(\theta_C^{(2)})$ 的显式函数已知（公理 2），$W_{12}$ 的二次型已知（定理 5.2）。驻点条件：

$$\frac{d}{d\theta_C^{(2)}} \left[ S^{(2)}(\theta_C^{(2)}) + W_{12}(\theta_A, \theta^{(2)}(\theta_C^{(2)})) \right] = 0.$$

这是一个**单变量代数方程**。求解后得到 $\theta_C^{(2),*}$，代入质量映射即得 $m_\mu$。

**当前阻塞**：$\theta_M^{(2)}$ 的值未确定（是 $\theta_C^0 = 26.16^\circ$ 还是需要从层级标度关系导出？），以及有效投影角 vs 裸角度的区分未几何化。但 $W_{12}$ 的显式形式使得这些未知量作为代数参数出现在方程中——一旦确定，求解是直接的。

#### §5.10.6 19.6% 偏差的推导路径（概要）

$$\boxed{\frac{d}{d\theta_C^{(2)}} \left[ S^{(2)}(\theta_C^{(2)}) + b' (\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)} + \Delta_M)(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)} - \Delta_M) \right] = 0}.$$

从该方程解出 $\theta_C^{(2),*}$，计算 $m_\mu^{\text{(th)}} = M_1 \sin^3\theta_C^{(2),*}$。

方程涉及的量：
- $b' = 0.867$（框架锁定）
- $S^{(2)}$：公理 2 显式函数
- $\Delta\theta_I^{(1)} = \Delta\theta_C^{(1)} = 0$（电子在 $P_A$ 处锁定）
- $\Delta_M = \Delta\theta_M^{(1)} - \Delta\theta_M^{(2)} = 57.93^\circ - \theta_M^{(2),0}$（若 $\theta_M^{(2),0}$ 已知）

**$\Delta\theta = 1.56^\circ$ 现已成为可精确求解的代数问题**——不再是一个模糊数字。

## §6　$C_n$ 不普适条件下的 Koide 刚性

### 6.1 Koide 公式回顾

小出公式（1982）：

$$Q = \frac{m_e + m_\mu + m_\tau}{\left(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau}\right)^2} \approx \frac{2}{3}.$$

实验值 $Q \approx 0.66666 \approx 2/3$，精度 $10^{-5}$。

### 6.2 $C_n$ 普适情形

若 $C_n$ 三代普适（$C_e = C_\mu = C_\tau = C$），则 $Q$ 退化为纯角度函数：

$$Q = \frac{s_1^3 + s_2^3 + s_3^3}{\left(s_1^{3/2} + s_2^{3/2} + s_3^{3/2}\right)^2}, \quad s_n = \sin\theta^{(n)}.$$

在此情形下，$Q = 2/3$ 等价于 $\theta^{(n)}$ 之间的代数约束——该约束是否从 H^W 驻点条件自然涌现，是封闭 Koide 公式的关键。

### 6.3 $C_n$ 不普适情形（当前框架实际情形）

在 H1 下，$C_e = K$，$C_\mu = M_1$，$C_\tau = M_2$，三者不相等。比值：

$$\frac{C_\mu}{C_e} = \Lambda_H S_e^{1/2} = 1755,\quad \frac{C_\tau}{C_\mu} = \Lambda_H = 150.$$

此时 $Q$ 不再是纯角度函数：

$$Q = \frac{K s_1^3 + M_1 s_2^3 + M_2 s_3^3}{\left(\sqrt{K} s_1^{3/2} + \sqrt{M_1} s_2^{3/2} + \sqrt{M_2} s_3^{3/2}\right)^2}.$$

$Q = 2/3$ 等价于 $s_n$ 与 $C_n$ 的联合条件——这是比 $C_n$ 普适情形更强的约束，也是 P0 级开放问题。

### 6.4 $C_n$ 抵消机制（正向线索）

尽管 $C_n$ 不普适，Koide 公式仍可能在特定角度分布下成立。注意到：

$$\frac{Q(C_n, s_n)}{Q(C, s_n)} = \frac{\sum C_n s_n^3}{\sum C s_n^3} \cdot \frac{(\sum \sqrt{C} s_n^{3/2})^2}{(\sum \sqrt{C_n} s_n^{3/2})^2}.$$

若 $C_n$ 的比值与 $s_n^{3/2}$ 的分布满足某种补偿关系，$Q$ 可能在数值上接近 $2/3$。$W_{12}$ 驻点条件（§5.10）决定了 $s_n$ 的精确值——当 $s_n$ 由第一原理确定后，$Q$ 成为可检验的数值输出，而非需拟合的输入。

### 6.5 数值检验（H2 约束下）

取实验反推角度：$\theta^{(1)} = 57.93^\circ$，$\theta^{(2)} = 24.60^\circ$，$\theta^{(3)} = 11.55^\circ$。和 $= 94.08^\circ \neq 90^\circ$。

若强制 H2（$\theta^{(1)}+\theta^{(2)}+\theta^{(3)} = 90^\circ$）且取 $\theta^{(1)} = \theta_M^0 = 57.93^\circ$，则 $\theta^{(2)}+\theta^{(3)} = 32.07^\circ$。

- 若 $\theta^{(2)} = 26.16^\circ$（$\theta_C^0$）：$\theta^{(3)} = 5.91^\circ$（$\theta_I^0$），$m_\tau^{\text{(th)}} = M_2 \sin^3 5.91^\circ = 241.6$ MeV（偏差 $86.4\%$）。**$\theta_I^0$ 被陶子质量排除。**

- 若 $\theta^{(2)} = 24.60^\circ$（实验反推）：$\theta^{(3)} = 7.47^\circ$，$m_\tau^{\text{(th)}} = M_2 \sin^3 7.47^\circ = 487$ MeV（偏差 $72.6\%$）。

**H2 + $C_n = M_n$ + 任意角度分配均无法同时拟合三代质量。** H2 本身需要从约束截面修正角度导出，而非直接使用公理 1。

## §8　结论与开放问题

### 8.1 主要结果

1. **电子质量在框架内完全封闭**——这是三代轻子中唯一达到"0.0.X + 单一核心映射即足够"标准的。

2. **$W_{12}$ 显式推导**（第5次修订）：跨粒子耦合 $W_{12}$ 已从 23 号文章文字描述提升为精确代数形式。核心结果：$W_{12}^{(2)} = b' (\Delta\theta_I^{(1)} - \Delta\theta_C^{(2)})(\Delta\theta_I^{(2)} - \Delta\theta_C^{(1)})$（M-扇区同步），或带 $\Delta_M$ 修正（放松同步）。$b' = 2w - w' = 0.867$ 来自框架锁定常数。**P0-1 闭合。**

   **双粒子联合截面驻点条件代数化（§5.10）**：19.6% 偏差的 $\Delta\theta = 1.56^\circ$ 现已成为可精确求解的代数方程——唯一阻塞是 P0-2（二代轻子几何定义），即 $\theta_M^{(2)}$ 和有效投影角 vs 裸角度的区分。

3. **缪子-因果角对应**（第3次修订）：$m_\mu^{\text{(th)}}(\theta_C^0) = 126$ MeV，偏差 19.6%，方向、量级和扇区归属均正确。**这是当前框架对二代轻子最正向的数值锚点。**

4. **H^W 驻点分析**（第4次修订）：单粒子 $H^W$ 微扰在构型 B 处灾难性失效（证毕）。反对称模式 $\lambda_-^W = 0.710$ 直接耦合 $\theta_C$，其特征标度 $1/\sqrt{\lambda_-^W} = 1.187$ rad 与构型 A–B 间距 $d_{AB} = 1.121$ rad 偏差仅 5.6%。$H^W$ 结构内禀编码了二代轻子的几何信息。

5. **$C_n$ 不普适**是当前框架的自然推论：$C_e = K, C_\mu = M_1, C_\tau = M_2$，比值来自 $\Lambda_H$ 和 $S_e^{1/2}$。

6. **在 CY 弦理论和标准模型中，Koide 公式是无法解释的巧合**。在十方几何中，它有成为几何定理的潜力——当 $W_{12}$ 驻点条件决定 $s_n$ 后，$Q$ 成为可检验输出。
