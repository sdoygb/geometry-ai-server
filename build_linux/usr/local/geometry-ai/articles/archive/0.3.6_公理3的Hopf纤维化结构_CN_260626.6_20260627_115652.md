# 第一章：公理3的结构与 P_ℳ 的几何含义

## §1.1 公理3的回顾

**公理3（三分切丛，0.0.6）：** 三维约束流形 M(a) 的切丛分裂为三个光滑子丛的直和：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

其中 ℳ（物质）、𝒞（约束）、ℐ（信息）是三个秩为1的实向量丛。此分裂是光滑的（0.0.6 命题2.1），且三个扇区在公理1（θ_M + θ_C + θ_I = 90°）的约束下耦合。

设 P_ℳ: TM(a) → ℳ 为到 ℳ 扇区的正交投影算子。由公理3，P_ℳ 是秩-1丛映射，在每个纤维上实现线性投影。

## §1.2 P_ℳ 的核丛

由秩定理，投影的核是秩-2光滑子丛：

$$V := \ker(P_\mathcal{M}) = \mathcal{C} \oplus \mathcal{I} \subset TM(a)$$

在每点 p ∈ M(a)，V_p 是 T_pM(a) ≅ ℝ³ 的二维子空间。

## §1.3 M(a) 的双重含义（诚实澄清）

本文使用与 0.0.6 不同的 M(a) 定义，此处显式声明：

## §2.3 从 V 的可积性问题到 ℳ 的自动可积性

**旧版遗留（已移除）。** 本文早期版本试图证明 V（= 𝒞⊕ℐ）的 Frobenius 可积性，但该论证存在裂缝且结论与拓扑 Hopf 的 Euler 类非零矛盾。经第四次审计修复，论证重新定向：

- ℳ 的积分曲线给出 S¹ 纤维（§3.2），商空间是 S²（定理3.1）

**引理2.1（ℳ 的自动可积性）** 任何秩-1光滑分布自动 Frobenius 可积。

*证明* 设 ℳ 由非零向量场 X 局部张成。任何两个截面 Y = fX, Z = gX 满足 [Y,Z] = [fX,gX] = (f·X(g) − g·X(f))X ∈ Γ(ℳ)。故 ℳ 对 Lie 括号封闭。∎

# 第四章：拓扑 Hopf 纤维化的闭合

## §4.1 从叶状结构到纤维丛

定理3.1建立了商映射 π: M(a) → S²。由 ℱ_ℳ 的叶片为 1 维且 M(a) ≅ S³，每条纤维 π⁻¹(x) 是 ℳ 的积分曲线，即 S³ 中的 1 维闭子流形。Ehresmann 定理（紧致基空间上的浸没）⇒ π 是局部平凡纤维化，纤维为 S¹。

**定理4.1（S¹-纤维化）** π: S³_ℳ → S² 是光滑 S¹-纤维丛。

*证明* 商映射是浸没（ℳ 定义非退化叶状结构），基 S² 紧致。Ehresmann 定理 ⇒ 局部平凡性。纤维是紧致1维流形且连通（S³ 连通），故为 S¹。∎

## §4.2 Euler 类的确定（定理4.2）

**定理4.2（Euler 类 = ±1）** 定理4.1的 S¹-丛的 Euler 类 e(π) = ±1 ∈ H²(S²,ℤ) ≅ ℤ。

*证明* S¹-丛 S³ → S² 的分类由 H²(S²,ℤ) ≅ ℤ 给出。若 |e| ≥ 2，全空间为透镜空间 L(|e|,1)（基本群 ℤ/|e|），其中仅 |e|=1 给出单连通 S³。因 M(a) ≅ S³ 单连通，必有 |e| = 1。∎

**θ_M 无关性：** 定理4.1和4.2仅依赖 S³ 的拓扑和 ℳ 的左不变性（来自 CPS SU(2) 结构），与 §4.1 中的角度值无关。**在物理锁定点 θ_M=57.93° 有效。**

## §4.3 拓扑 Hopf 纤维化（定理4.3）

*证明* 综合定理4.1（S¹-纤维化）和定理4.2（Euler 类 = ±1）。∎

### §8.1.2 独立性的严格论证

两条推导链在以下意义上**独立**：

1. **不同的谱不变量：** c_m 使用 Dixmier trace（逆三次幂的奇异迹），c_{9,1} 使用热核系数（短时渐近展开的第一修正项）。两者探测谱的不同方面。

3. c_{9,1} 的推导中，CPS 半径比来自 S₃ 群论（Λ=3, k₀=2），不是为拟合 c_{9,1} 而"选择"的——等半径和角度投影比均被明确拒斥
