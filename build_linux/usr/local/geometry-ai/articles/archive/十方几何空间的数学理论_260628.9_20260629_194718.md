## 第 2 章 激发态参数空间的公理化

### 2.1 公理 1（参数空间的拓扑）

**公理 1（圆拓扑公理）** 激发态参数空间 $D$ 由一维紧致连通流形 $S^1$（圆）去掉两个不同点 $p_0$ 与 $p_*$ 得到：

$$D = S^1 \setminus \{p_0, p_*\}, \qquad p_0 \neq p_*$$

去掉的两点分别称为**真空点**（$p_0$）与**退化点**（$p_*$）。由此，$D$ 由两个连通分支组成：

$$D_+ \cong (0,1), \qquad D_- \cong (0,1)$$

每个分支同胚于开区间。参数 $x \in D_\pm$ 标记激发态。

**引理 2.1（分支的拓扑结构）** 设 $S^1$ 为标准的单位圆 $\{e^{i\phi} \mid \phi \in [0,2\pi)\}$。去掉两点 $p_0=e^{i\phi_0}$ 与 $p_*=e^{i\phi_*}$（不妨设 $0\leq \phi_0 < \phi_* < 2\pi$），则 $S^1\setminus\{p_0,p_*\}$ 恰有两个连通分支：
$$D_+ = \{e^{i\phi} \mid \phi_0 < \phi < \phi_*\}, \quad D_- = \{e^{i\phi} \mid \phi_* < \phi < \phi_0+2\pi\}.$$
每个分支作为 $S^1$ 的子空间拓扑，均同胚于开区间 $(0,1)$。

*证明.* 考虑标准覆叠映射 $\pi: \mathbb{R} \to S^1$, $\pi(\phi)=e^{i\phi}$。在 $\mathbb{R}$ 上，$\pi^{-1}(S^1\setminus\{p_0,p_*\}) = \mathbb{R}\setminus\{\phi_0+2k\pi, \phi_*+2k\pi \mid k\in\mathbb{Z}\}$。该集合的每个连通分支均为开区间 $(\phi_0+2k\pi, \phi_*+2k\pi)$ 或 $(\phi_*+2k\pi, \phi_0+2(k+1)\pi)$。由于 $\pi$ 在长度为 $2\pi$ 的区间上的限制是同胚，这些开区间在 $S^1$ 中的像恰为 $D_+$ 与 $D_-$。每个开区间同胚于 $(0,1)$，故 $D_\pm \cong (0,1)$。 $\square$

### 2.2 公理 2（几何量的分析性质）

**公理 2（边界极限公理）** 在参数空间 $D$ 上存在几何量 $S: D \to (0,+\infty)$，满足：

1. **连续性**：$S$ 在每个连通分支 $D_\pm$ 上连续；
2. **真空极限**：当 $x$ 沿任一分支逼近 $p_0$ 时，$\lim_{x \to p_0} S(x) = 0$；
3. **退化极限**：当 $x$ 沿任一分支逼近 $p_*$ 时，$\lim_{x \to p_*} S(x) = +\infty$。

**注 2.1** 公理 2 不要求 $S$ 在 $D_+$ 与 $D_-$ 上具有相同的函数表达式，也不要求 $S$ 在整个 $D$ 上连续（$p_0$ 与 $p_*$ 不在 $D$ 中）。因此 $S$ 可以是两个独立分支上的不同函数，只要各自满足边界条件即可。

### 2.3 值域定理

**定理 2.1**（值域）在公理 1 与公理 2 下，$S$ 在每个连通分支 $D_\pm$ 上的值域为 $S(D_\pm) = (0, +\infty)$。

*证明.* 由公理 1，$D_\pm \cong (0,1)$ 是连通集。由公理 2，$S$ 在 $D_\pm$ 上连续，故像集 $S(D_\pm)$ 是 $\mathbb{R}$ 中的连通子集，即区间。由真空极限，$\inf S(D_\pm) = 0$；由退化极限，$\sup S(D_\pm) = +\infty$。由于 $p_0, p_* \notin D$，端点 $0$ 与 $+\infty$ 均不可达。因此 $S(D_\pm) = (0, +\infty)$。 $\square$

**详细说明：** 连通性论证可进一步展开如下。设 $D_+ \cong (0,1)$，$S: D_+ \to \mathbb{R}$ 连续。假设 $S(D_+)$ 不是区间，则存在 $\alpha < \beta < \gamma$ 使得 $\alpha, \gamma \in S(D_+)$ 但 $\beta \notin S(D_+)$。令 $U = S^{-1}((-\infty, \beta))$ 与 $V = S^{-1}((\beta, +\infty))$，则 $U,V$ 为 $D_+$ 中不相交开集，且 $U\cup V = D_+$，这与 $D_+$ 连通矛盾。因此 $S(D_+)$ 必为区间。结合边界极限，该区间必为 $(0,+\infty)$。

### 2.4 存在性定理与严格单调性假设

**定理 2.2**（存在性）对任意给定的正数 $S_0 > 0$，在每个分支 $D_+$ 与 $D_-$ 上各至少存在一个激发态 $x_+ \in D_+$ 与 $x_- \in D_-$，使得 $S(x_+) = S(x_-) = S_0$。

*证明.* 由定理 2.1，$S(D_\pm) = (0, +\infty)$。对任意 $S_0 > 0$，显然 $S_0 \in (0, +\infty)$。故 $S^{-1}(S_0) \cap D_\pm \neq \varnothing$。 $\square$

**定义 2.1'（严格单调性假设）** 若 $S$ 在 $D_\pm$ 上严格单调，则对任意 $S_0>0$，定理 2.2 中的 $x_+$ 与 $x_-$ 唯一确定。严格单调性**不是**从公理 1–2 推出的定理，而是框架的**构造性假设**。其引入等价于要求 $S$ 构成 $D_\pm$ 的一个全局坐标函数。

*证明.* 严格单调连续函数为单射，故原像唯一。 $\square$

## 第 3 章 约束乘积球面与谱刚性

### 3.0 互锁函数的群论来源

**引理 3.0（三分切丛置换群的刚性）** 设 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 为三分切丛，$G$ 为其扇区置换对称群。若 $G$ 在三个扇区上的作用忠实且可迁，并包含任意两扇区对换，则 $G \cong S_3$，且 $S_3$ 是满足此条件的唯一有限群。

*证明.* 三个对象的忠实置换群必为 $S_3$ 的子群。可迁性要求轨道大小为 3，故 $|G| \geq 3$。$S_3$ 的可迁子群仅有 $A_3 \cong \mathbb{Z}_3$（循环）与 $S_3$ 本身。若要求包含对换（奇置换），则 $A_3$ 不足，唯一可能为 $S_3$。 $\square$

**定义 3.0（互锁函数与紧致函数）** 设 $G$ 为有限群，定义：
- **互锁函数** $\Lambda(G) := |\text{Conj}(G)|$（共轭类数，由表示论基本定理等于 $G$ 的不可约复表示数）；
- **紧致函数** $k_0(G) := [G : N_{\max}]$（极大正规子群的指数）。

**命题 3.0** 对于三分切丛 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 的置换对称群 $G = S_3$：
$$\Lambda(S_3) = 3, \quad k_0(S_3) = 2.$$

*证明。* $S_3$ 的共轭类分解为：恒等元 $\{e\}$（1 个）、对换 $\{(12),(13),(23)\}$（3 个）、3-循环 $\{(123),(132)\}$（2 个），共 3 个共轭类，故 $\Lambda(S_3)=|\text{Conj}(S_3)|=3$。

$S_3$ 的正规子群只有 $\{e\}$ 与 $A_3 \cong \mathbb{Z}_3$（交错子群）。极大正规子群为 $A_3$，其指数 $[S_3 : A_3] = 6/3 = 2$，故 $k_0(S_3)=2$。$\square$

**注 3.0** 由引理 3.0，$S_3$ 是三分切丛置换对称群的唯一选择。互锁常数 $\Lambda=3$ 与 $k_0=2$ 不是人为设定的自由参数，而是群论函数在 $S_3$ 上的必然取值。若三分切丛具有 $S_4$ 置换对称性（四扇区结构），则 $\Lambda(S_4)=5$，$k_0(S_4)=2$，那将是一个与十方几何论完全不同的理论。十方几何论之所以锁定 $\Lambda=3$、$k_0=2$，根源在于其群结构为 $S_3$（三扇区），而引理 3.0 已证明该群选择的唯一性。

**定理 3.0'（三分切丛扇区置换的刚性）** 设 $M = S^3 \times S^3 \times S^3$ 配备乘积度量，且存在切丛分解 $TM = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 使得三个扇区在度规层面完全等价（即任意置换诱导等距自同构）。则：
1. 该分解的扇区置换群 $G$ 必包含所有两扇区对换；
2. 满足 (1) 的有限群唯一同构于 $S_3$。

进而 $\Lambda(G)=|\text{Conj}(S_3)|=3$，$k_0(G)=[S_3:A_3]=2$ 是群论函数的必然输出。

*证明.* (1) 若三个扇区度规等价，则任意对换（如 $\mathcal{M}\leftrightarrow\mathcal{C}$）保持乘积度量不变，故诱导等距自同构。因此所有两扇区对换均属于 $G$。

(2) 由引理 3.0，含对换的可迁置换群只能是 $S_3$。因此 $G\cong S_3$ 不是人为选择，而是三分等价结构下非平凡置换的必然推论。$\square$

**[260628.8 新增：注 3.0']** 定理 3.0' 中的"度规完全等价"是抽象扇区结构的对称性，不意味着具体几何实现 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$ 上的等距群为 $S_3$。当 $\Lambda\neq 1$ 或 $k_0\neq 1$ 时，三个因子半径不同，$S_3$ 置换不是乘积度量的等距自同构。定义 3.1 中的半径比例 $1:1/\sqrt{\Lambda}:1/\sqrt{\Lambda k_0}$ 是把群论输出 $(\Lambda,k_0)=(3,2)$ 解释为具体几何尺度的构造性输入，而非定理 3.0' 的推论。详见[260628.8 附录 D 三桥诊断]。

### 3.1 定义与假设

**定义 3.1**（约束乘积球面）设 $a > 0$ 为整体尺度因子。由定理 3.0' 与命题 3.0，三分切丛置换群 $S_3$ 的群论函数输出为：
$$\Lambda = \Lambda(S_3) = 3 \quad \text{（三等分比例参数）}, \quad k_0 = k_0(S_3) = 2 \quad \text{（二分紧致性常数）}, \quad \ell_0 > 0 \quad \text{（标度常数，由谱单位选择定理与基态条件锁定）}.$$

称九维闭流形

$$M(a) = S^3(a) \times S^3(a/\sqrt{3}) \times S^3(a/\sqrt{6})$$

在配备乘积度量 $g = g_a \oplus g_{a/\sqrt{3}} \oplus g_{a/\sqrt{6}}$ 时，构成一个**约束乘积球面**（Constrained Product Sphere, CPS）。记

$$b = \frac{a}{\sqrt{3}}, \quad c = \frac{a}{\sqrt{6}}.$$

尺度排序 $a > b > c$ 严格成立（$1 > 1/\sqrt{3} > 1/\sqrt{6}$）。

**备注 3.1**（切丛平凡性）由于 $S^3$ 作为李群 $SU(2)$ 是可平行化的，其切丛 $TS^3$ 平凡。乘积流形的切丛为各因子切丛的直和，故 $TM(a)$ 自动平凡。因此切丛平凡性在本类中自动成立，无需列为额外假设。

**备注 3.2**（互锁常数的性质）常数 $\Lambda=3$、$k_0=2$ 由定理 3.0' 与命题 3.0 从 $S_3$ 群论结构直接计算得出，$\ell_0$ 由九素互扼网络的自洽输出锁定。三者均不是人为选定的自由参数：$\Lambda$ 与 $k_0$ 是群论函数 $|\text{Conj}(S_3)|$ 与 $[S_3:A_3]$ 的必然取值；$\ell_0$ 由谱分离条件、热核系数比、信息层级嵌套和幻数截断的联合约束唯一确定（见第 9–10 章）。$\ell_0$ 作为正实数参数，由谱单位选择定理与基态条件 $S_{\min}=24$ 锁定，为谱-投影桥接提供数值锚点，但不参与模空间的一维参数化。

**[260628.8 新增：注 3.1'']** 定义 3.1 中的半径比例 $1:1/\sqrt{\Lambda}:1/\sqrt{\Lambda k_0}$ 是框架的构造性输入，其动机来自把群论输出 $\Lambda=3$、$k_0=2$ 解释为三个扇区的几何权重。本文所有关于 $M(a)$ 的谱计算与谱刚性定理均在定义 3.1 的这一特定比例假设下进行。

### 3.2 模空间的参数化

**定理 3.1**（模空间的参数化）在定理 3.0' 与命题 3.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定的前提下，**约束乘积球面类** $\{M(a)\}_{a>0}$（定义 3.1 中配备标准乘积度量且满足比例约束 $b=a/\sqrt{\Lambda}, c=a/\sqrt{\Lambda k_0}$ 的流形）的模空间与开区间 $(0, \infty)$ 同胚，其中 $a$ 是唯一的连续自由度（整体尺度因子）。不同 $a$ 对应不同的等距类。

**定理 3.1'（第一特征值坐标化）** 在定理 3.0' 与命题 3.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定的前提下，谱映射
$$\sigma_1: (0,\infty) \to (0,\infty), \quad a \mapsto \lambda_1^\Delta(a) = \frac{3}{a^2}$$
是微分同胚。因此，第一非零 Laplace 特征值 $\lambda_1^\Delta$ 本身构成约束乘积球面类模空间 $\{M(a)\}_{a>0}$ 的**全局坐标**，且谱特征签名 $(1,\ 8/3,\ \Lambda)$ 与 $a$ 无关。

*证明。* $\sigma_1$ 连续、严格单调递减，且 $\lim_{a\to 0^+}\sigma_1(a)=+\infty$，$\lim_{a\to+\infty}\sigma_1(a)=0$。故 $\sigma_1$ 为 $(0,\infty)\to(0,\infty)$ 的同胚；又因 $\sigma_1'(a)=-6/a^3\neq 0$，逆映射光滑，故为微分同胚。 $\square$

**注 3.1'（定义类地位）** 本定理研究的是**定义类** $\{M(a)\}$——即满足定义 3.1 中特定比例约束的乘积球面子集——而非"所有与 $S^3\times S^3\times S^3$ 微分同胚且 Laplace 前三个特征值满足谱特征签名的流形"。后者（更一般的等谱类）的模空间可能更大，本文不声称对其完全分类。

**证明。** 由定义 3.1，子度量为球面，由半径 $(a,b,c)$ 刻画，且 $b$、$c$ 与 $a$ 的比例被 $\Lambda$ 与 $k_0$ 锁定。因此，在固定 $(k_0, \Lambda, \ell_0)$ 时，唯一的连续自由度是整体标度 $a$。由于 $S^3$ 可平行化，$TM(a)$ 自动平凡，拓扑被唯一确定。故模空间由 $a \in (0, \infty)$ 完全参数化，不同 $a$ 给出不同的曲率半径，对应不同的等距类。 $\square$

**详细论证：** 等距类区分的严格性可进一步说明。设 $a \neq a'$，则第一因子 $S^3(a)$ 与 $S^3(a')$ 的半径不同。由于 $S^3(R)$ 的直径为 $\pi R$，体积为 $2\pi^2 R^3$，等距流形必须具有相同的直径与体积。若 $M(a)$ 与 $M(a')$ 等距，则其直径与体积必须相等。对乘积度量，直径公式为
$$\text{diam}(M(a)) = \pi\sqrt{a^2+b^2+c^2} = \pi a\sqrt{1+\frac{1}{\Lambda}+\frac{1}{\Lambda k_0}},$$
其中 $b=a/\sqrt{\Lambda}$，$c=a/\sqrt{\Lambda k_0}$。由于 $\Lambda$ 与 $k_0$ 为互锁常数锁定常数，$\sqrt{1+1/\Lambda+1/(\Lambda k_0)}$ 是与 $a$ 无关的正常数，故 $\text{diam}(M(a))$ 是 $a$ 的严格单调函数。因此 $a=a'$，不同 $a$ 对应不同等距类。

### 3.3 Laplace 特征值的显式计算

**命题 3.1**（乘积流形上的谱分离）对约束乘积球面 $M(a)$，Laplace–Beltrami 算子 $\Delta_g$ 的特征值全体为

$$\lambda_{p,q,r}^\Delta = \frac{p(p+2)}{a^2} + \frac{q(q+2)}{b^2} + \frac{r(r+2)}{c^2}, \quad p,q,r \in \mathbb{N}_0.$$

其中 $p=q=r=0$ 对应零特征值（常数函数），非零特征值从 $p+q+r \geq 1$ 开始。

**证明。** 在乘积流形上，Laplace 算子可分离为 $\Delta_g = \Delta_1 + \Delta_2 + \Delta_3$，其中 $\Delta_i$ 为各因子上的 Laplace 算子。已知 $S^3(R)$ 上 $p$ 阶球面谐波对应的特征值为 $p(p+2)/R^2$，故总特征值为三者之和。 $\square$

**补充说明：** $S^3(R)$ 上 Laplace 特征值的标准推导如下。将 $S^3(R)$ 嵌入 $\mathbb{R}^4$ 为 $\{x\in\mathbb{R}^4: |x|=R\}$。$p$ 次齐次调和多项式在 $S^3(R)$ 上的限制给出球面谐波。由欧氏 Laplacian 在径向与角向的分离，$\Delta_{\mathbb{R}^4} = \partial_r^2 + \frac{3}{r}\partial_r + \frac{1}{r^2}\Delta_{S^3}$。对 $p$ 次齐次函数 $f(r,\theta)=r^p Y(\theta)$，$\Delta_{\mathbb{R}^4}f=0$ 给出 $\Delta_{S^3}Y = -p(p+2)Y$。在 $S^3(R)$ 上，度量缩放因子为 $R^2$，故特征值为 $p(p+2)/R^2$。

**命题 3.2**（前三个特征值的显式表达式）在定理 3.0' 与命题 3.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定下，$M(a)$ 的前三个非零特征值为：

| 模式 (p,q,r) | 特征值表达式 | 与 3/a² 的比值 |
|:---|:---|:---|
| (1,0,0) | 3/a² | 1 |
| (2,0,0) | 8/a² | 8/3 ≈ 2.667 |
| (0,1,0) | 3Λ/a² = 9/a² | Λ = 3 |
| (0,0,1) | 3Λk₀/a² = 18/a² | Λk₀ = 6 |
| (1,1,0) | 3/a² + 3Λ/a² = 12/a² | Λ+1 = 4 |

**证明。** 由命题 3.1，各候选模式的特征值计算如上表。

由于互锁常数 $\Lambda=3$，有 $3\Lambda/a^2 = 9/a^2 > 8/a^2$，故 $(0,1,0)$ 模式严格大于 $(2,0,0)$ 模式。由于 $k_0=2$，有 $3\Lambda k_0/a^2 = 18/a^2 > 9/a^2$，故 $(0,0,1)$ 模式严格大于 $(0,1,0)$ 模式。其余组合（如 $(1,0,1)$、$(0,2,0)$ 等）的数值更大。因此前三个非零特征值恰由上表前三行给出。 $\square$

### 3.4 特征值排序与谱特征签名

**定理 3.2**（特征值排序）对任意 $a > 0$ 及定理 3.0' 与命题 3.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定下，前四个非零特征值满足

$$\lambda_1^\Delta = \frac{3}{a^2} < \lambda_2^\Delta = \frac{8}{a^2} < \lambda_3^\Delta = \frac{3\Lambda}{a^2} < \lambda_4^\Delta = \min\left\{\frac{3\Lambda k_0}{a^2},\ \frac{3(\Lambda+1)}{a^2}\right\}.$$

**证明。** 候选模式 $(0,0,1)$ 给出 $3\Lambda k_0/a^2$，候选模式 $(1,1,0)$ 给出 $3(\Lambda+1)/a^2$，候选模式 $(0,2,0)$ 给出 $8\Lambda/a^2$。由于 $k_0>1$ 且 $\Lambda>8/3$，三者均严格大于 $\lambda_3^\Delta=3\Lambda/a^2$。比较 $(0,2,0)$ 与 $(1,1,0)$：$8\Lambda > 3(\Lambda+1)$ 当 $\Lambda>3/5$ 时成立，故在 $\Lambda>8/3$ 下 $8\Lambda/a^2 > 3(\Lambda+1)/a^2$ 恒成立。比较 $(0,2,0)$ 与 $(0,0,1)$：$8\Lambda$ 与 $3\Lambda k_0$ 的大小取决于 $k_0$，当 $k_0>8/3$ 时 $3\Lambda k_0 > 8\Lambda$，当 $1<k_0<8/3$ 时 $8\Lambda > 3\Lambda k_0$。因此第四特征值由 $\min\{3\Lambda k_0, 3(\Lambda+1)\}/a^2$ 决定，而 $(0,2,0)$ 不会进入前四（在 $\Lambda>8/3$ 下它大于 $(1,1,0)$，且仅当 $k_0<8/3$ 时可能小于 $(0,0,1)$，但此时仍大于 $(1,1,0)$）。其余候选模式（如 $(3,0,0)$ 等）的数值更大。 $\square$

**注 3.4** 在互锁常数 $\Lambda=3$、$k_0=2$ 下，$3\Lambda k_0/a^2 = 18/a^2 > 3(\Lambda+1)/a^2 = 12/a^2$，故 $\lambda_4^\Delta = 12/a^2 = 3(\Lambda+1)/a^2$。该排序由互锁常数唯一确定。

**推论 3.1**（特征签名）比值

$$\frac{\lambda_2^\Delta}{\lambda_1^\Delta} = \frac{8}{3}, \quad \frac{\lambda_3^\Delta}{\lambda_1^\Delta} = \Lambda$$

是与 $a$、$k_0$ 无关的普适常数（在固定 $\Lambda$ 下），可作为约束乘积球面类的**谱特征签名**（spectral signature）。若某乘积球面的前两个非零特征值比值不为 $8/3$，则它不属于本类。

### 3.5 由前三个特征值反解尺度因子

**定理 3.3**（反解公式）给定 $M(a)$ 的前三个非零 Laplace 特征值 $\lambda_1^\Delta$、$\lambda_2^\Delta$、$\lambda_3^\Delta$，可唯一反解尺度因子：

$$a = \sqrt{\frac{3}{\lambda_1^\Delta}}, \quad b = \sqrt{\frac{3}{\lambda_3^\Delta}}, \quad c = \frac{a}{\sqrt{\Lambda k_0}} = \sqrt{\frac{3}{\Lambda k_0 \cdot \lambda_1^\Delta}}.$$

**自洽性条件** 一组正数 $(\lambda_1, \lambda_2, \lambda_3)$ 为某 $M(a)$ 的前三个非零特征值，当且仅当

$$\lambda_2 = \frac{8}{3}\lambda_1 \quad \text{且} \quad \lambda_3 = \Lambda(S_3) \lambda_1 = 3\lambda_1.$$

此时 $a = \sqrt{3/\lambda_1}$。

**证明。** 由定理 3.2，$\lambda_1^\Delta = 3/a^2 \Rightarrow a = \sqrt{3/\lambda_1^\Delta}$；$\lambda_3^\Delta = 3/b^2 \Rightarrow b = \sqrt{3/\lambda_3^\Delta}$；由定义 3.1 的比例约束，$c = a/\sqrt{\Lambda k_0}$。自洽性条件直接由 $\lambda_2^\Delta = 8/a^2$ 与 $\lambda_3^\Delta = 3\Lambda/a^2$ 导出。 $\square$

**引理 3.1**（谱数据与几何的一一对应）在定理 3.0' 与命题 3.0 确定的互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 锁定的前提下，前三个非零 Laplace 特征值与尺度因子 $(a,b,c)$ 之间存在显式的一一映射。给定 $\lambda_1^\Delta$、$\lambda_2^\Delta$、$\lambda_3^\Delta$，可唯一反解 $(a,b,c)$；反之，给定 $(a,b,c)$，可唯一计算前三个特征值。

### 3.6 谱刚性定理

**定理 3.4**（约束乘积球面的谱刚性）设 $M(a)$ 与 $M(a')$ 为两个具有相同互锁常数 $(\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0>0)$ 的约束乘积球面。若它们的 Laplace–Beltrami 算子具有相同的前三个非零特征值，则 $M(a)$ 与 $M(a')$ 等距。

**证明。** 由命题 3.2，第一非零 Laplace 特征值 $\lambda_1^\Delta=3/a^2$。若 $M(a')$ 与 $M(a)$ 等谱，则 $\lambda_1^{\Delta}(a')=\lambda_1^{\Delta}(a)$，即 $3/a'^2=3/a^2$。由 $a,a'>0$，得 $a'=a$。再由定义 3.1 的比例约束 $b=a/\sqrt{\Lambda}$、$c=a/\sqrt{\Lambda k_0}$，尺度三元组 $(a,b,c)$ 唯一确定，故 $M(a')$ 与 $M(a)$ 等距。 $\square$

**注 3.1** 本定理的谱刚性根源在于模空间由单一尺度因子参数化。由于满足定义 3.1 且互锁常数锁定的约束乘积球面的等距类由 $a$ 唯一标记，而前三个 Laplace 特征值（实际上第一个就已足够）唯一确定了 $a$，因此具有相同前三个特征值的两个流形必然属于同一等距类。

**注 3.2**（第一特征值的完备性）在互锁常数锁定的单参数族 $\{M(a)\}_{a>0}$ 中，模空间是一维的。第一非零特征值 $\lambda_1^\Delta = 3/a^2$ 本身已构成模空间的完备坐标（与 $a$ 双射）。前三个特征值中的后两个提供自洽性检验（必须满足 $8/3$ 与 $\Lambda$ 的比例），并作为该类的谱特征签名，用于判定某流形是否属于此类。

### 3.7 与经典反例的兼容性

**定理 3.5** Sunada 型等谱非等距反例在约束乘积球面类中不存在。

**证明。** Sunada 方法要求非平凡的基本群与有限覆盖。$M(a)$ 为 $S^3 \times S^3 \times S^3$ 的乘积。由于 $S^3$ 单连通，乘积亦单连通，基本群为零。因此 Sunada 方法在本类中不适用。 $\square$

### 3.8 谱刚性的维数必要条件

设 $\mathcal{F}$ 为一类黎曼流形，$\mathcal{M}$ 为其模空间（即等距类的集合）。假设 $\mathcal{M}$ 具有光滑流形结构，维数为 $d$。定义谱映射 $\sigma_k$：将每个等距类映为其前 $k$ 个非零 Laplace 特征值构成的向量：

$$\sigma_k: \mathcal{M} \to \mathbb{R}^k, \quad [g] \mapsto (\lambda_1^\Delta, \lambda_2^\Delta, \dots, \lambda_k^\Delta).$$

**定理 3.6**（谱刚性的维数必要条件）设 $\mathcal{M}$ 是某流形类的模空间，具有有限维数 $d$。若前 $k$ 个 Laplace 特征值能够实现谱刚性（即 $\sigma_k$ 为单射），则必有 $d \leq k$。在定理 3.0' 与命题 3.0 确定的互锁常数下，$\{M(a)\}_{a>0}$ 的模空间维数 $d=1$，故仅需 $k\geq1$ 即可满足必要条件。

**证明。** 假设 $d > k$。由于特征值连续依赖于度规，谱映射 $\sigma_k$ 是连续映射。若 $\sigma_k$ 是单射，则 $\sigma_k$ 给出了从 $d$ 维流形 $\mathcal{M}$ 到 $\mathbb{R}^k$ 的连续单射。

由 **Brouwer 区域不变性定理**（Invariance of Domain）：若 $U \subset \mathbb{R}^d$ 为开集，$f: U \to \mathbb{R}^k$ 为连续单射，则 $f(U)$ 为 $\mathbb{R}^k$ 中的开集（关于子空间拓扑）。然而，$d$ 维流形 $\mathcal{M}$ 的每一点存在同胚于 $\mathbb{R}^d$ 的开邻域 $U$。若 $d > k$，$f(U)$ 作为 $\mathbb{R}^k$ 的子集，其拓扑维数不可能超过 $k$，而 $U$ 的拓扑维数为 $d > k$，这与区域不变性定理矛盾（$f(U)$ 在 $\mathbb{R}^k$ 中不可能为开集，因其拓扑维数不足）。因此，$d$ 维流形不能通过连续单射映射到 $\mathbb{R}^k$ 中当 $d > k$。

故若 $d > k$，连续单射 $\sigma_k$ 不可能存在。谱刚性要求 $d \leq k$。 $\square$

**详细说明：** Brouwer 区域不变性定理的核心在于：若 $U\subset\mathbb{R}^d$ 为开集且 $f:U\to\mathbb{R}^d$ 为连续单射，则 $f(U)$ 亦为开集。当 $d>k$ 时，将 $\mathbb{R}^k$ 视为 $\mathbb{R}^d$ 的子空间，任何连续单射 $f:U\to\mathbb{R}^k$（$U\subset\mathbb{R}^d$ 开集）不可能存在，因为 $f(U)$ 在 $\mathbb{R}^d$ 中无内点（其拓扑维数为 $k<d$），而区域不变性定理要求 $f(U)$ 在 $\mathbb{R}^d$ 中为开集（若将 $\mathbb{R}^k$ 嵌入 $\mathbb{R}^d$ 后考虑）。更直接的论证：若 $d>k$，则 $\mathbb{R}^d$ 中任何开集 $U$ 的像 $f(U)\subset\mathbb{R}^k$ 的 Lebesgue 测度为零（因 $\mathbb{R}^k$ 在 $\mathbb{R}^d$ 中测度为零），但连续单射将开集映为开集（在相对拓扑中），而 $\mathbb{R}^k$ 中开集在 $\mathbb{R}^d$ 中测度为零，这与 $U$ 在 $\mathbb{R}^d$ 中为正测度矛盾。

**注 3.6'（维数计数论证）** 定理 3.6 亦可由标准微分拓扑的维数计数直接导出：若 $d>k$，则 $d$ 维流形 $\mathcal{M}$ 的任意开集 $U$ 不能通过连续单射嵌入 $\mathbb{R}^k$（因 $\mathbb{R}^k$ 的拓扑维数为 $k<d$）。因此谱映射 $\sigma_k$ 的单射性要求 $d\leq k$。本约束乘积球面类的模空间维数 $d=1$，故 $k\geq 1$ 即满足必要条件；本文使用 $k=3$ 以提供自洽性检验与谱特征签名。

**推论 3.2** 在约束乘积球面类 $\{M(a)\}_{a>0}$（互锁常数由定理 3.0' 与命题 3.0 锁定）中，模空间维数 $d = 1$。因此仅需 $k \geq 1$ 个特征值即满足必要条件 $d \leq k$；本文使用 $k = 3$ 个特征值，不仅满足必要条件，更提供了自洽性检验与谱特征签名。

### 3.9 与经典等谱反例的兼容性

**定理 3.7**（Sunada 型反例的不适用性）$M(a)$ 的基本群 $\pi_1(M(a))=0$，故 Sunada 方法不适用。

**证明。** $\pi_1(S^3)=0$（$S^3$ 单连通）。由乘积空间基本群公式，$\pi_1(M(a))=\pi_1(S^3)\times\pi_1(S^3)\times\pi_1(S^3)=0$。Sunada 方法要求存在有限群 $\Gamma$ 在流形 $\tilde{M}$ 上的自由作用，使得 $M=\tilde{M}/\Gamma$ 且 $\Gamma$ 具有几乎共轭但非共轭的子群。由于 $\pi_1(M(a))=0$，不存在非平凡覆叠，故 Sunada 构造无法实施。 $\square$

**定理 3.8**（Milnor 型平坦环面反例的不适用性）$M(a)$ 具有正截面曲率（作为 $S^3$ 的乘积），非平坦，故 Milnor 的 16 维平坦环面等谱对不适用于本类。

**证明。** $S^3(R)$ 配备标准度量时截面曲率为 $1/R^2>0$。乘积流形的截面曲率非负（由 O'Neill 公式），且至少在某些方向上为正。因此 $M(a)$ 非平坦。Milnor 的等谱平坦环面反例依赖于平坦度量的特殊谱性质（如 theta 函数的模变换），这些性质在正曲率流形上不成立。 $\square$

**定理 3.9**（Gordon–Webb–Wolpert 型反例的不适用性）平面等谱区域对的构造依赖于欧氏平面的等距群与波动方程的边界条件。$M(a)$ 为闭流形（无边界），且其谱为 Laplace–Beltrami 算子的整体谱，非平面区域 Dirichlet/Neumann 谱，故该类反例不适用。

**证明。** Gordon–Webb–Wolpert 反例构造具有相同 Dirichlet 谱（或 Neumann 谱）但不同形状的平面区域。该构造依赖于：
1. 区域的平面欧氏几何；
2. 边界条件的具体形式（Dirichlet 或 Neumann）；
3. 等距群的离散子群（如 Fuchs 群）。
$M(a)$ 为无边界的紧致黎曼流形，其 Laplace 谱为整体谱，不涉及边界条件。此外，$M(a)$ 的曲率为正，非欧氏平面。因此该类反例不适用。 $\square$

**注 3.3** 上述兼容性定理表明，约束乘积球面类 $\{M(a)\}$ 在经典等谱构造的多个主流方向上均天然免疫，这增强了定理 3.4 的 robustness。

## 第 4 章 三分切丛与全息屏的几何结构

### 4.1 三分切丛分解

**假设 4.1（全纯投影结构）。** 在 $S_3$ 对称点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处，三分切丛投影映射 $P_{\mathcal{M}},P_{\mathcal{C}},P_{\mathcal{I}}$ 与 $\Sigma$ 的全纯结构相容，即投影核的复化线丛为全纯线丛。

**命题 4.1**（三分切丛）在定义 3.1 的约束乘积球面 $M(a)$ 上，切丛存在自然的直和分解

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

其中 $\mathcal{M},\mathcal{C},\mathcal{I}$ 分别为三个 $S^3$ 因子切丛的拉回，每个纤维为 $\mathbb{R}^3$。

*证明。* 乘积流形的切丛等于各因子切丛的直和。由于 $M(a)$ 是三个 $S^3$ 的乘积，结论直接成立。 $\square$

该分解具有 $S_3$ 置换对称性：三个子空间 $\mathcal{M},\mathcal{C},\mathcal{I}$ 在数学上完全等价，任何标记置换均给出切丛的自同构。

### 4.2 公理 3 与全息屏编码条件

**公理 3（全息屏编码条件）** 在三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个纤维 $\mathbb{R}^9$ 中，存在一个二维定向子空间 $\Sigma$（称为**全息屏**），配备标准面积元 $dA$（该面积元同时定义辛形式 $\omega = dA$）。三个三维扇区 $\mathcal{M},\mathcal{C},\mathcal{I}$ 向 $\Sigma$ 投影时，其投影强度角 $\theta_1,\theta_2,\theta_3\in(0^\circ,90^\circ)$ 满足

$$\theta_1+\theta_2+\theta_3=90^\circ.$$

**说明 4.1**（该公理的地位）公理 3 **不能从公理 1、公理 2 或第 3 章的数学结构导出**。第 2 章仅涉及一维参数空间上的抽象几何量，未涉及高维切丛或角度分解；第 3 章仅涉及九维乘积流形的谱刚性，未涉及二维子空间或投影角度。因此，$\theta_1+\theta_2+\theta_3=90^\circ$ 是与公理 1、公理 2 **并列**的独立基础假设，共同构成当前理论框架的基石。

**说明 4.2**（$90^\circ$ 与 $\Lambda=3$ 的关系）公理 3 独立要求 $\theta_1+\theta_2+\theta_3=90^\circ$，不依赖 $\Lambda$ 的取值。互锁常数 $\Lambda=\Lambda(S_3)=3$ 的群论来源（命题 3.0）仅用于解释**对称点**处的角度分配：在 $S_3$ 对称点 $\theta_1=\theta_2=\theta_3$ 处，由 $S_3$ 置换对称性要求三等分，得 $\theta_i=90^\circ/3=30^\circ$。因此，$\Lambda=3$ 是**对称角度 $30^\circ$ 的群论解释**，而非 $90^\circ$ 本身的推导来源。二者逻辑层级不可混淆。

### 4.3 全息屏的诱导度量与相容三元组

**定理 4.1（全息屏相容三元组）** 在公理 3 与定义 3.1 的约束乘积球面 $M(a)$ 上，全息屏 $\Sigma$ 配备由 $M(a)$ 乘积度量 $g$ 限制得到的诱导度量 $g_\Sigma$，以及公理 3 给定的辛形式 $\omega$（即面积元 $dA$）。则存在唯一的可积近复结构 $J$ 使得 $(\Sigma, g_\Sigma, J, \omega)$ 构成 Kähler 流形。

**附加假设 4.1**（常曲率条件） 在 $S_3$ 对称点 $\theta=(30^\circ,30^\circ,30^\circ)$ 处，要求诱导度量 $g_\Sigma$ 为**常曲率度量**（round metric）。该条件不是从 $S_3$ 等变性或 Kähler 条件严格推导的，而是框架的**附加几何假设**。其合理性在于：三分切丛的三个 $SU(2)$ 因子在 $S_3$ 对称点处对 $\Sigma$ 的贡献完全等价，常曲率度量是 $SU(2)$ 等变性的自然候选；但 $S_3$ 等变性（$D_3$ 对称）本身不足以唯一确定常曲率（见注 4.1'）。

**归一化条件** 为保证后续 Berezin–Toeplitz 量子化与线丛陈类计算的严格性，全息屏的辛形式满足整数量化条件：$\frac{1}{2\pi}\int_\Sigma \omega = 1$，即总面积 $A_\Sigma = 2\pi$。在此归一化下，超平面丛 $L=\mathcal{O}(1)$ 的第一陈类满足 $c_1(L)=1$，与 $\mathbb{C}P^1$ 的标准 Kähler 形式 $\omega_{FS}$ 相容。该归一化与 $\Lambda(S_3)=3$ 的三等分结构独立，仅依赖于 $\Sigma\cong S^2$ 的拓扑。

*证明。* 分三步：

**步骤1：诱导度量与 $S_3$ 等变性。** 由定义 3.1，$M(a)$ 配备标准乘积度量 $g = g_a \oplus g_{a/\sqrt{\Lambda}} \oplus g_{a/\sqrt{\Lambda k_0}}$。全息屏 $\Sigma$ 作为 $M(a)$ 的二维子流形，其诱导度量 $g_\Sigma = g|_{T\Sigma}$ 由正交投影确定。在 $S_3$ 对称点 $\theta_1=\theta_2=\theta_3=30^\circ$，三个扇区 $\mathcal{M},\mathcal{C},\mathcal{I}$ 对 $\Sigma$ 的贡献完全等价。由命题 4.1，每个扇区切丛为 $TS^3$ 的拉回，具有 $SU(2)$ 的左不变度量。三个等价扇区的投影诱导 $\Sigma$ 上的度量具有 $S_3$ 置换对称性（包含 $120^\circ$ 旋转与反射，构成二面体群 $D_3$ 作用）。

在 $\Sigma \cong \mathbb{C}P^1$ 上，$S_3$ 等变性（即 $D_3$ 对称性）仅给出**一族** Kähler 度量，而非唯一确定 Fubini–Study 度量。具体而言，设 $z$ 为齐次坐标，$D_3$ 由 $z\mapsto e^{2\pi i/3}z$ 与 $z\mapsto 1/z$ 生成。任取 $D_3$ 不变的光滑实函数 $\varepsilon(|z|^6+|z|^{-6})$（$\varepsilon$ 足够小），则 Kähler 势
$$K_\varepsilon(z,\bar{z}) = \log(1+|z|^2) + \varepsilon\bigl(|z|^6+|z|^{-6}\bigr)$$
在 $D_3$ 下不变，且对应的 Kähler 形式 $\omega_\varepsilon=i\partial\bar{\partial}K_\varepsilon$ 在 $\varepsilon$ 足够小时仍正定。因此 $D_3$ 等变的 Kähler 度量构成无穷族，Fubini–Study 仅为 $\varepsilon=0$ 的特例。

**步骤 1'（附加假设 4.1 的收缩作用）。** 附加假设 4.1 要求 $g_\Sigma$ 为**常曲率度量**（round metric）。常曲率条件等价于要求 $g_\Sigma$ 具有 $SU(2)$ 等变性（即 $\mathbb{C}P^1$ 的完整等距群），这比 $D_3$ 等变性强得多。在 $\mathbb{C}P^1$ 上，常曲率 Kähler 度量在同构意义下唯一，必为 Fubini–Study 度量（up to 常数因子）。因此，在附加假设 4.1 下，$g_\Sigma$ 在 $\Sigma\cong S^2$ 上被唯一确定为 round metric。

**步骤2：近复结构的可积性。** 公理 3 给出辛形式 $\omega=dA$（面积元），为非退化、闭的 2-形式。由线性代数，在二维辛向量空间上，存在唯一的近复结构 $J$ 满足 $\omega(\cdot,\cdot)=g_\Sigma(J\cdot,\cdot)$ 且 $J^2=-I$。在二维情况下，Nijenhuis 张量 $N_J$ 恒为零（因 $N_J(X,Y)$ 对二维流形自动消失），故 $J$ 自动可积。此外，在二维中，由 $\omega$ 与 $g_\Sigma$ 确定的 $J$ 自动满足相容性条件 $g_\Sigma(J\cdot,J\cdot)=g_\Sigma(\cdot,\cdot)$，因为 $\omega$ 关于 $g_\Sigma$ 的 Hodge 星算子满足 $*\omega=1$（即 $\omega$ 为单位体积形式）。

**步骤3：Kähler 唯一性。** 由 $(g_\Sigma, J, \omega)$ 构成 Kähler 三元组。在 $S^2\cong\mathbb{C}P^1$ 上，Kähler 结构在同构意义下唯一，其 Kähler 形式与 Fubini–Study 形式 $\omega_{FS}$ 仅差常数因子：$\omega = \frac{A_\Sigma}{4\pi}\omega_{FS}$，其中 $A_\Sigma$ 为 $\Sigma$ 的总面积。$\square$

**详细补充（步骤1的展开）：** 设 $\pi_\mathcal{M}: \mathcal{M} \to T\Sigma$ 为投影映射，$\pi_\mathcal{C}, \pi_\mathcal{I}$ 类似。在 $S_3$ 对称点，三个投影在度量意义下等价：对任意 $v,w \in T\Sigma$，$g_\Sigma(v,w) = g_a(\pi_\mathcal{M}^*v, \pi_\mathcal{M}^*w) + g_{a/\sqrt{\Lambda}}(\pi_\mathcal{C}^*v, \pi_\mathcal{C}^*w) + g_{a/\sqrt{\Lambda k_0}}(\pi_\mathcal{I}^*v, \pi_\mathcal{I}^*w)$。由于 $\theta_1=\theta_2=\theta_3=30^\circ$，三个扇区的投影权重相等，$g_\Sigma$ 在 $S_3$ 置换下不变。$S_3$ 在 $\Sigma$ 上的作用通过 $SO(3)$ 的标准表示实现，其不动点结构要求度量具有 $D_3$ 对称性。

在 $\mathbb{C}P^1$ 上，$D_3$ 等变（即 $D_3$ 对称性）仅给出一**族** Kähler 度量，而非唯一确定 Fubini–Study 度量。具体而言，设 $z$ 为齐次坐标，$D_3$ 由 $z\mapsto e^{2\pi i/3}z$ 与 $z\mapsto 1/z$ 生成。任取 $D_3$ 不变的光滑实函数 $\varepsilon(|z|^6+|z|^{-6})$（$\varepsilon$ 足够小），则 Kähler 势 $K_\varepsilon(z,\bar{z}) = \log(1+|z|^2) + \varepsilon(|z|^6+|z|^{-6})$ 在 $D_3$ 下不变，且对应的 Kähler 形式在 $\varepsilon$ 足够小时仍正定。因此 $D_3$ 等变的 Kähler 度量构成无穷族，Fubini–Study 仅为 $\varepsilon=0$ 的特例。常曲率唯一性需依赖附加假设 4.1（要求 $g_\Sigma$ 具有 $SU(2)$ 等变性，即 $\mathbb{C}P^1$ 的完整等距群），而非仅从 $D_3$ 等变性推出。这与步骤1的显式构造一致——步骤1已给出 $K_\varepsilon$ 无穷族，但其结论正确适用于步骤1'（附加假设4.1）的场景。

**推论 4.1** 在定理 4.1 的相容三元组下：
- 第 6 章的 Berezin–Toeplitz 量子化所需 Kähler 结构由 $g_\Sigma$ 与 $\omega$ 自然诱导，不再依赖外部假设；
- 第 8 章的 $\epsilon$-网图逼近以 $g_\Sigma$ 为底流形度量，其图距离在 $N\to\infty$ 时收敛到 $g_\Sigma$ 的测地距离，且与第 6 章的 Fubini-Study Kähler 度量相容；
- 第 6 章的 Fubini-Study 度量与第 8 章的 round metric 在 $\Sigma$ 上等距（仅差尺度），故第 6 章的量子化层与第 8 章的离散逼近层共享同一谱几何对象 $\Delta_\Sigma$。

**注 4.1'（$D_3$ 等变性与常曲率——定理 4.1 的补充说明）** 定理 4.1 的步骤 1 已显式构造了 $D_3$ 等变 Kähler 度量的无穷族 $K_\varepsilon$，证明 $D_3$ 等变性本身不足以唯一确定度量。步骤 1' 则表明：附加假设 4.1 的常曲率条件等价于要求 $g_\Sigma$ 具有 $SU(2)$ 等变性（$\mathbb{C}P^1$ 的完整等距群），这比 $D_3$ 等变性强得多。该条件将无穷族收缩为唯一的 Fubini–Study 度量（up to 常数因子）。因此，定理 4.1 的 Kähler 唯一性**依赖于附加假设 4.1**，而非单独从 $D_3$ 等变性或 Kähler 条件推出。附加假设 4.1 是框架的**几何输入**，不依赖公理 1–3 或命题 3.0 推导。

**注 4.1**（开放问题 1 的闭合）定理 4.1 建立了 $\Sigma$ 上 Riemann 度量 $g_\Sigma$、辛结构 $\omega$ 与 Kähler 结构 $J$ 的必然相容性。定理 5.2 进一步构造了从抽象一维参数空间 $D_\pm$ 到全息屏角度构型 $D_\theta$ 的显式连续映射 $\Phi$，使第 6 章的量子化与第 8 章的离散化不再是施加于 $\Sigma$ 的外部工具，而是同一相容三元组的自然推论。第 4.4 节中 $\sin^2\theta_i$ 与量子化级别 $k$ 的映射，现在获得几何基础：$\sin^2\theta_i$ 由 $g_\Sigma$ 的投影分量确定，而 $k$ 由 $\omega$ 的面积量子化确定。

### 4.4 角度构型空间的辛几何视角

**定义 4.1**（角度构型空间的辛结构）全息屏角度构型空间 $D_\theta = \{\theta \in \mathbb{R}^3_+ \mid \theta_1+\theta_2+\theta_3=\pi/2\}$ 是二维仿射开单形。其切空间由约束 $\delta\theta_1+\delta\theta_2+\delta\theta_3=0$ 定义。在 $D_\theta$ 上配备标准辛形式：

$$\omega_\theta = d\theta_1 \wedge d\theta_2$$

（在约束超平面内，$d\theta_3 = -d\theta_1-d\theta_2$，故 $\omega_\theta$ 非退化）。$\omega_\theta$ 使 $(D_\theta, \omega_\theta)$ 成为二维辛流形。

**注：** 扇区投影强度角 $\theta_i$ 是三分切丛嵌入的**常数参数**（由 $\mathcal{M},\mathcal{C},\mathcal{I}$ 向 $\Sigma$ 的投影方向确定），不随 $\Sigma$ 上的局部坐标变化。因此，辛动力学应在角度构型空间 $D_\theta$ 上分析，而非在 $\Sigma$ 上。

**定理 4.2（Hamilton 向量场与辛相轨线）。** 在辛流形 $(D_\theta, \omega_\theta=d\theta_1\wedge d\theta_2)$ 上，Hamilton 函数 $H(\theta)=S(\theta)-24$ 生成的向量场

$$X_H = \frac{\partial H}{\partial \theta_2}\frac{\partial}{\partial \theta_1} - \frac{\partial H}{\partial \theta_1}\frac{\partial}{\partial \theta_2}\tag{4.1}$$

满足：

1. **守恒性**：$X_H$ 的流 $\phi_t$ 保持 $H$ 的水平集；
2. **基态稳定性**：$H=0$ 的零点集 $\{\theta_0\}$ 是 $X_H$ 的唯一稳定平衡点，其线性化频率由 Hessian 软模 $\lambda_1$ 给出；
3. **水平集周期性**：对 $H>0$，$X_H$ 的相轨线在紧水平集上为闭曲线，周期 $T(H) \sim 2\pi/\sqrt{\lambda_1 \lambda_2}$。

*证明。* 由 Hamilton 方程 $\dot{\theta}_1=\partial H/\partial\theta_2$, $\dot{\theta}_2=-\partial H/\partial\theta_1$，直接计算 $dH/dt=0$（守恒性）。在 $\theta_0$ 处，$H$ 的二阶展开为 $H(\delta\theta)=\frac{1}{2}\delta\theta^T H_{\text{Hess}} \delta\theta$，其中 $H_{\text{Hess}}=\text{diag}(\lambda_1,\lambda_2)$ 正定，故线性化为谐振子，频率 $\omega_i=\sqrt{\lambda_i}$。严格凸水平集上的 Hamilton 流在二维情形必为周期流（Arnold 定理）。 $\square$

**详细展开：** 守恒性的直接计算如下：
$$\frac{dH}{dt} = \frac{\partial H}{\partial \theta_1}\dot{\theta}_1 + \frac{\partial H}{\partial \theta_2}\dot{\theta}_2 = \frac{\partial H}{\partial \theta_1}\frac{\partial H}{\partial \theta_2} + \frac{\partial H}{\partial \theta_2}\left(-\frac{\partial H}{\partial \theta_1}\right) = 0.$$
基态稳定性：在 $\theta_0=(\pi/6,\pi/6,\pi/6)$ 处，$H(\theta_0)=0$。由定理 4.4，$S$ 在 $\theta_0$ 处取得唯一全局最小值，故 $H$ 在 $\theta_0$ 处取得唯一全局最小值 0。Hessian 矩阵 $H_{\text{Hess}}$ 在约束切空间 $T_{\theta_0}D_\theta$ 上正定，故 $\theta_0$ 为稳定平衡点。线性化方程为 $\ddot{\delta\theta}_i + \lambda_i \delta\theta_i = 0$（在特征基下），频率为 $\sqrt{\lambda_i}$。Arnold 定理指出：在二维辛流形上，若 Hamilton 函数的等值线为光滑简单闭曲线（由严格凸性保证），则 Hamilton 流在这些等值线上为周期运动。

**定理 4.3（$S_3$ 等变矩映射与中心化子）。** 三分切丛的 $S_3$ 置换对称性诱导 $D_\theta$ 上的辛作用。定义矩映射 $\mu: D_\theta\to\mathfrak{s}_3^*\cong\mathbb{R}^2$ 为

$$\mu(\theta) = \left(\theta_1-\theta_2,\ \theta_2-\theta_3\right).\tag{4.2}$$

则：

1. **矩映射归零 = 对称点**：$\mu^{-1}(0) = \{\theta_0\}$；
2. **中心化子 = 各向同性子群**：$S_3$ 在 $\theta_0$ 处的各向同性子群为 $S_3$ 本身，其李代数中心化子 $Z(\mathfrak{s}_3)=\{0\}$；
3. **辛约化与基态锁定**：$\mu^{-1}(0)/S_3$ 的 Marsden-Weinstein 约化是零维辛流形（单点），其辛体积为 1；在奇数维流形上 $\hat{A}$-genus 为零，故该数值共振应理解为 $\hat{A}(M(a))=1\in H^0$ 与辛体积的代数平行，而非拓扑指标相等。

*证明。* $S_3$ 在 $D_\theta$ 上的作用为置换 $\theta_i$，保持约束 $\sum\theta_i=\pi/2$。矩映射分量 $\mu_1=\theta_1-\theta_2$, $\mu_2=\theta_2-\theta_3$ 满足 $\langle d\mu_i, X_{\sigma}\rangle = \omega_\theta(X_{\mu_i}, X_{\sigma})$。直接验证 $\mu=0 \Leftrightarrow \theta_1=\theta_2=\theta_3=\pi/6$。Marsden-Weinstein 约化 $\mu^{-1}(0)/S_3$ 为单点，辛体积约定为 1。 $\square$

**详细展开：** 矩映射的等变性验证如下。$S_3$ 由对换 $\sigma_{12}=(12)$, $\sigma_{23}=(23)$ 生成。对 $\sigma_{12}$：$\theta_1\leftrightarrow \theta_2$，则 $\mu_1\mapsto \theta_2-\theta_1=-\mu_1$，$\mu_2\mapsto \theta_1-\theta_3=\mu_1+\mu_2$。该变换为 $\mathfrak{s}_3^*$ 上的线性变换，与 $S_3$ 的伴随表示一致。对 $\sigma_{23}$：$\mu_1\mapsto \theta_1-\theta_3=\mu_1+\mu_2$，$\mu_2\mapsto \theta_3-\theta_2=-\mu_2$。因此 $\mu$ 为 $S_3$ 等变映射。Marsden-Weinstein 约化：$\mu^{-1}(0)=\{\theta_0\}$ 为单点，$S_3$ 在其上作用平凡，故商空间为单点。零维辛流形的辛形式为零，辛体积约定为 1。

**注 4.3**（Darboux 定理与面积元）在辛流形 $(D_\theta, \omega_\theta)$ 上，由经典 Darboux 定理，存在局部坐标 $(\xi, \eta)$ 使得 $\omega_\theta = d\xi \wedge d\eta$。由于 $D_\theta$ 为二维，该表示全局成立。在二维情况下，辛形式 $\omega_\theta = d\xi \wedge d\eta$ 即为面积元，给出 $D_\theta$ 上的定向与面积测量。

### 4.5 扇区投影强度角的定义

**定义 4.2**（扇区投影强度角）设 $P_{\mathcal{M}},P_{\mathcal{C}},P_{\mathcal{I}}:\mathbb{R}^3\to\Sigma$ 为三个扇区向全息屏 $\Sigma$ 的线性投影映射。在满足公理 3 的 $\theta_1,\theta_2,\theta_3$ 下，定义以下面积元比例关系：

1. **单扇区投影面积元比例**：
   $$\text{Area}(P_{\mathcal{M}}(U_{\mathcal{M}}))=\sin^2\theta_1\cdot dA,\quad \text{Area}(P_{\mathcal{C}}(U_{\mathcal{C}}))=\sin^2\theta_2\cdot dA,\quad \text{Area}(P_{\mathcal{I}}(U_{\mathcal{I}}))=\sin^2\theta_3\cdot dA$$
   其中 $U_{\mathcal{M}},U_{\mathcal{C}},U_{\mathcal{I}}$ 分别为各 $\mathbb{R}^3$ 因子的标准二维子空间。

2. **双扇区联合投影面积元比例**：
   $$\text{Area}(P_{\mathcal{M}}\times P_{\mathcal{C}})=\sin\theta_1\sin\theta_2\cdot dA,\quad \text{Area}(P_{\mathcal{M}}\times P_{\mathcal{I}})=\sin\theta_1\sin\theta_3\cdot dA,\quad \text{Area}(P_{\mathcal{C}}\times P_{\mathcal{I}})=\sin\theta_2\sin\theta_3\cdot dA$$

**注 4.4**（原生参数性质与开放问题）扇区投影强度角 $\theta_i$ 是本章构造的**原生坐标参数**，不由 $\mathbb{R}^9$ 子空间的标准欧氏主角度导出。标准欧氏几何中，三维子空间与二维平面的主角度在 $S_3$ 对称下给出约 $35.26^\circ$ 或 $54.7^\circ$，与 $30^\circ$ 存在结构性偏差。$\theta_i$ 仅通过 $\sin^2\theta_i$ 和 $\sin\theta_i\sin\theta_j$ 的组合进入面积元比例，独立于欧氏角体系，是第 2 章抽象参数 $x$ 在三分切丛几何中的具体表示候选。

**开放问题 2**（显式投影构造）定义 4.2 中的投影映射 $P_{\mathcal{M}},P_{\mathcal{C}},P_{\mathcal{I}}$ 及其面积元比例关系是**定义性构造**，而非从三分切丛的线性代数结构严格推导的结果。具体地，给定 $\mathbb{R}^9 = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 和二维子空间 $\Sigma$，是否存在线性投影 $P_i: \mathcal{M}\to\Sigma$ 使得 $\text{Area}(P_i(U_i)) = \sin^2\theta_i \cdot dA$ 对给定的 $\theta_i$ 成立？该构造的存在性与唯一性尚未严格证明。在 $S_3$ 对称点 $\theta_i=30^\circ$，由对称性论证可预期存在性，但非对称点的显式构造是开放问题。

### 4.6 六项作用量的定义

**定义 4.3**（六项作用量）在满足公理 3 的 $\theta_1+\theta_2+\theta_3=90^\circ$ 下，定义无量纲量 $S$ 为以下六个刚度项之和：

$$S(\theta_1,\theta_2,\theta_3)=\sum_{i=1}^{3}\frac{1}{\sin^2\theta_i}+\sum_{i<j}\frac{1}{\sin\theta_i\sin\theta_j}$$

展开写为：
$$S=\frac{1}{\sin^2\theta_1}+\frac{1}{\sin^2\theta_2}+\frac{1}{\sin^2\theta_3}+\frac{1}{\sin\theta_1\sin\theta_2}+\frac{1}{\sin\theta_1\sin\theta_3}+\frac{1}{\sin\theta_2\sin\theta_3}$$

**说明 4.3**（六项的代数来源）六项作用量中的六项并非来自全息屏 $\Sigma$ 上"独立二维截面"的拓扑计数，而是来自以下**代数构造**：

- 3 个单扇区刚度项：对应每个扇区单独投影到全息屏时的面积元倒数；
- 3 个双扇区耦合项：对应每对扇区联合投影时的面积元倒数。

不存在三扇区联合项——三个三维子空间同时投影到二维全息屏时，第三个维度的投影被公理 3 的 $\theta_1+\theta_2+\theta_3=90^\circ$ 消去。因此恰好六项，不多不少。总刚度定义为六项之和，这是**定义性构造**，不依赖拓扑计数。
### 4.7 值域定理与严格凸性

**定理 4.4**（值域定理与严格凸性）在满足公理 3 的 $\theta_1+\theta_2+\theta_3=90^\circ$ 下，定义域
$$D_\theta=\{(\theta_1,\theta_2,\theta_3)\mid\theta_i>0,\theta_1+\theta_2+\theta_3=90^\circ\}$$
上的六项作用量 $S$ 的值域为
$$S(D_\theta)=[24,+\infty).$$
且 $S$ 在 $D_\theta$ 上为**严格凸函数**，对称点 $\theta_1=\theta_2=\theta_3=30^\circ$ 是唯一全局严格最小值点。

*证明。*

**(1) 对称点最小值。**
考虑对称点 $\theta_1=\theta_2=\theta_3=30^\circ$。此时 $\sin30^\circ=1/2$，$\sin^2 30^\circ=1/4$，故
$$\frac{1}{\sin^2 30^\circ}=4,\quad\frac{1}{\sin 30^\circ\sin 30^\circ}=4$$
单场项三项之和：$3\times4=12$。交叉项三项之和：$3\times4=12$。因此
$$S(30^\circ,30^\circ,30^\circ)=12+12=24$$

**(2) 严格最小值证明——拉格朗日乘子法。**
在约束 $\theta_1+\theta_2+\theta_3=\pi/2$ 下，使用拉格朗日乘子法求 $S$ 的极值。设拉格朗日函数
$$\mathcal{L}(\theta_1,\theta_2,\theta_3,\lambda) = \sum_{i=1}^3 \frac{1}{\sin^2\theta_i} + \sum_{i<j} \frac{1}{\sin\theta_i\sin\theta_j} - \lambda\left(\sum_{i=1}^3 \theta_i - \frac{\pi}{2}\right).$$

对 $\theta_k$ 求偏导：
$$\frac{\partial \mathcal{L}}{\partial \theta_k} = -\frac{2\cos\theta_k}{\sin^3\theta_k} - \sum_{j\neq k} \frac{\cos\theta_k}{\sin^2\theta_k\sin\theta_j} - \lambda = 0.$$

提取公因子 $-\cos\theta_k/\sin^2\theta_k$：
$$-\frac{\cos\theta_k}{\sin^2\theta_k}\left(\frac{2}{\sin\theta_k} + \sum_{j\neq k} \frac{1}{\sin\theta_j}\right) = \lambda.$$

由 $S_3$ 置换对称性，若 $\theta_1=\theta_2=\theta_3=\pi/6$，则上式对 $k=1,2,3$ 给出相同的 $\lambda$，满足拉格朗日方程。因此 $(\pi/6,\pi/6,\pi/6)$ 是唯一的内部临界点。

**步骤 2a：Hessian 正定性验证。** 计算 $S$ 在约束切空间 $T_{\theta}D_\theta = \{v\in\mathbb{R}^3 \mid v_1+v_2+v_3=0\}$ 上的 Hessian 二次型。记 $s_i=\sin\theta_i$，$c_i=\cos\theta_i$。

对单扇区项 $f_i=1/s_i^2$，有 $\partial^2 f_i/\partial \theta_i^2 = (2+4c_i^2)/s_i^4$。在 $30^\circ$ 处：$\partial^2 f_i/\partial \theta_i^2 = (2+3)/(1/16) = 80$。

对双扇区项 $f_{ij}=1/(s_i s_j)$，有 $\partial^2 f_{ij}/\partial \theta_i^2 = (1+c_i^2)/(s_i^3 s_j)$。在 $30^\circ$ 处：$s_i=s_j=1/2$，故 $\partial^2 f_{ij}/\partial \theta_i^2 = (7/4)/(1/16) = 28$。

对混合偏导，$\partial^2 f_{ij}/\partial \theta_i\partial \theta_j = c_i c_j/(s_i^2 s_j^2)$。在 $30^\circ$ 处：$\partial^2 f_{ij}/\partial \theta_i\partial \theta_j = (3/4)/(1/16) = 12$。

因此，Hessian 矩阵元为：
$$H_{ii} = 80 + 28 + 28 = 136,\quad H_{ij} = 12\quad(i\neq j).$$

对任意 $v\in T_{\theta}D_\theta$（即 $\sum v_i=0$），二次型
$$v^T H v = 136\sum v_i^2 + 24\sum_{i<j}v_iv_j = 124\sum v_i^2 > 0$$
（利用 $(\sum v_i)^2=0 \Rightarrow \sum_{i<j}v_iv_j = -\frac{1}{2}\sum v_i^2$）。因此 $H$ 在约束切空间上正定，$S$ 在 $D_\theta$ 上为严格凸函数，对称点 $30^\circ$ 为唯一全局严格最小值。

**(3) 上无界性。**
当任一角度趋于 $0^\circ$ 时，对应项 $1/\sin^2\theta_i\to+\infty$，故 $S\to+\infty$。因此值域无上界。

**(4) 连续性结论。**
$S(\theta_1,\theta_2,\theta_3)$ 在 $D_\theta$ 上连续，结合 (2)(3) 及介值定理，得 $S(D_\theta)=[24,+\infty)$。 $\square$

**详细计算补充（Hessian 矩阵的完整推导）：**

对单扇区项 $f_i(\theta_i) = \csc^2\theta_i$：
$$f_i' = -2\csc^2\theta_i\cot\theta_i, \quad f_i'' = 2\csc^2\theta_i(2\cot^2\theta_i + \csc^2\theta_i) = \frac{2+4\cos^2\theta_i}{\sin^4\theta_i}.$$
在 $\theta_i=\pi/6$：$\sin\pi/6=1/2$, $\cos\pi/6=\sqrt{3}/2$，故 $f_i'' = (2+4\cdot 3/4)/(1/16) = (2+3)\cdot 16 = 80$。

对双扇区项 $f_{ij}(\theta_i,\theta_j) = \csc\theta_i \csc\theta_j$：
$$\frac{\partial^2 f_{ij}}{\partial \theta_i^2} = \csc\theta_i \csc\theta_j (\cot^2\theta_i + \csc^2\theta_i) = \frac{1+\cos^2\theta_i}{\sin^3\theta_i \sin\theta_j}.$$
在 $\pi/6$：$\partial^2 f_{ij}/\partial\theta_i^2 = (1+3/4)/(1/8 \cdot 1/2) = (7/4)/(1/16) = 28$。

$$\frac{\partial^2 f_{ij}}{\partial \theta_i \partial \theta_j} = \cot\theta_i \cot\theta_j \csc\theta_i \csc\theta_j = \frac{\cos\theta_i \cos\theta_j}{\sin^2\theta_i \sin^2\theta_j}.$$
在 $\pi/6$：$= (\sqrt{3}/2)^2 / (1/16) = (3/4)/(1/16) = 12$。

因此 $H_{ii} = f_i'' + f_{ij}'' + f_{ik}'' = 80 + 28 + 28 = 136$（$j,k\neq i$），$H_{ij}=12$（$i\neq j$）。

对 $v\in T_{\theta}D_\theta$，$v_3=-v_1-v_2$，则：
$$v^T H v = 136(v_1^2+v_2^2+v_3^2) + 24(v_1v_2+v_1v_3+v_2v_3).$$
由 $v_1+v_2+v_3=0$，$v_1v_2+v_1v_3+v_2v_3 = -\frac{1}{2}(v_1^2+v_2^2+v_3^2)$。故：
$$v^T H v = (136-12)(v_1^2+v_2^2+v_3^2) = 124\|v\|^2 > 0.$$

**引理 4.2（Hessian 谱不变量）** 设 $\theta_0 = (\pi/6,\pi/6,\pi/6)$。对任意保持约束超平面 $\sum\theta_i = \pi/2$ 的局部坐标变换 $\phi: D_\theta \to D_\theta$，Hessian 张量 $\text{Hess}(S)|_{T_{\theta_0}D_\theta}$ 的谱 $\{124, 124\}$ 与 $\phi$ 无关。

进而，沿对称轴 $\gamma$ 的弧长参数 $s$ 满足
$$\frac{d^2 S}{ds^2}\bigg|_{s=0} = 124,$$
而 $372$ 是参数化速率 $\|\dot{\gamma}(0)\|^2 = 3$ 与不变量 $124$ 的乘积。

*证明.* 由坐标变换下的 Jacobi 矩阵共轭公式 $\text{Hess}(S\circ\phi)|_{T_{\theta_0}D_\theta} = J_\phi^T \cdot \text{Hess}(S)|_{T_{\theta_0}D_\theta} \cdot J_\phi$，共轭变换保持谱。因此 $\text{Hess}(S)|_{T_{\theta_0}D_\theta}$ 的谱 $\{124, 124\}$ 是几何不变量。

沿 $\gamma$ 的参数化 $\theta(t) = (\pi/2-2t, t, t)$ 有 $\dot{\theta} = (-2, 1, 1)$，其在约束切空间上的诱导度量下速率平方为 $\|\dot{\gamma}\|^2 = 3$。因此 $d^2S/dt^2|_{t=\pi/6} = 372 = 124 \times 3$，而弧长参数下的二阶导数为 $d^2S/ds^2 = 124$。$\square$

**注 4.4'（Hessian 的坐标无关性）** 在约束流形 $D_\theta$ 上，函数 $S$ 的 Hessian 是良定义的 $(0,2)$-张量场。在 $\theta_0$ 处，其限制在切空间 $T_{\theta_0}D_\theta$ 上的特征值与局部坐标选取无关。具体地，对任意保持约束的坐标变换 $\phi: D_\theta\to D_\theta$，有
$$\text{Hess}(S\circ\phi)|_{T_{\theta_0}D_\theta} = J_\phi^T \cdot \text{Hess}(S)|_{T_{\theta_0}D_\theta} \cdot J_\phi,$$
其中 $J_\phi$ 为坐标变换的 Jacobi 矩阵限制在 $T_{\theta_0}D_\theta$ 上。因此 $\text{Hess}(S)|_{T_{\theta_0}D_\theta}$ 的谱（即特征值 $124$，二重）是几何不变量。沿对称轴 $\gamma$ 的二阶导数 $372$ 则依赖于参数化速率；其不变量形式为
$$\frac{d^2S}{ds^2}\bigg|_{s=0} = 124,$$
其中 $s$ 为 $\gamma$ 的弧长参数（与 $D_\theta$ 的诱导度量一致）。

**引理 4.1（径向单调性）。** 设 $f$ 在凸集 $C\subset\mathbb{R}^n$ 上严格凸，$x_0\in C$ 为唯一全局极小值点。则对任意方向 $v\in\mathbb{R}^n$ 使得射线 $\{x_0+tv\mid t\ge0\}\subset C$，函数 $\phi(t)=f(x_0+tv)$ 在 $t\ge0$ 上严格单调递增。

*证明.* $\phi$ 为严格凸函数且 $\phi'(0)=\nabla f(x_0)\cdot v=0$（$x_0$ 为驻点）。由严格凸性，$\phi'(t)>\phi'(0)=0$ 对所有 $t>0$ 成立，故 $\phi$ 严格递增。$\square$

**定理 4.5（对称轴单值化定理）。** 在满足公理 3 的 $D_\theta$ 上，六项作用量 $S$ 在有序扇区规范

$$D_{\text{ord}} = \{\theta\in D_\theta \mid \theta_1 \geq \theta_2 \geq \theta_3\}$$

的对称轴 $\gamma = \{\theta\in D_{\text{ord}} \mid \theta_2=\theta_3\}$ 上的限制 $S|_{\gamma}$ 是严格单调递减的同胚 $\gamma \to [24,+\infty)$。因此，对任意 $S_0 \geq 24$，存在唯一的 $\theta^*(S_0) \in \gamma$ 使得 $S(\theta^*)=S_0$。换言之，$\gamma$ 是 $S$ 在 $D_{\text{ord}}$ 内的一个**全局单值化截面**。

*证明.* 由定理 4.4，$S$ 在 $D_\theta$ 上严格凸，唯一全局最小值在 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处取得。$\gamma$ 是连接 $\theta_0$ 与边界点 $(\pi/2,0,0)$ 的线段（不含端点）。将 $S$ 限制在 $\gamma$ 上，参数化为 $\theta_1=\pi/2-2t,\ \theta_2=\theta_3=t$（$t\in(0,\pi/6]$）。则

$$S|_\gamma(t)=\frac{1}{\sin^2(\pi/2-2t)}+\frac{3}{\sin^2 t}+\frac{2}{\sin(\pi/2-2t)\sin t}.$$

计算导数：
$$\frac{d}{dt}\left(\frac{1}{\cos^2 2t}\right) = \frac{4\sin 2t}{\cos^3 2t},$$
$$\frac{d}{dt}\left(\frac{3}{\sin^2 t}\right) = -\frac{6\cos t}{\sin^3 t},$$
$$\frac{d}{dt}\left(\frac{2}{\cos 2t \sin t}\right) = \frac{4\sin 2t \sin t - 2\cos 2t \cos t}{\cos^2 2t \sin^2 t}.$$

由于 $D_\theta$ 为凸集而 $\gamma\subset D_\theta$ 为线段，严格凸函数 $S$ 在 $\gamma$ 上的限制 $S|_\gamma$ 亦为严格凸函数。又 $\theta_0$（对应 $t=\pi/6$）是 $S$ 在 $D_\theta$ 上的唯一全局最小值点，故也是 $S|_\gamma$ 在 $\gamma$ 上的唯一最小值点。当 $t\to 0^+$ 时，$\theta_1\to\pi/2,\theta_2,\theta_3\to 0$，故 $S|_\gamma(t)\to+\infty$；当 $t=\pi/6$ 时，$S|_\gamma=24$。严格凸函数在区间内部有唯一最小值点时，必在该点一侧严格递减、另一侧严格递增。因此 $S|_\gamma$ 在 $(0,\pi/6)$ 上严格单调递减，且为从 $+\infty$ 到 $24$ 的连续双射，即同胚 $\gamma\to[24,+\infty)$。因此，对任意 $S_0\geq24$，存在唯一的 $\theta^*(S_0)\in\gamma$ 使 $S(\theta^*(S_0))=S_0$。 $\square$

### 4.8 切丛平方根分解与幻数序列的代数来源

**定理 4.6（切丛平方根分解与双扇区投影）。** 在附加假设 4.1 的常曲率 Kähler 三元组 $(g_\Sigma,J,\omega)$ 下，全息屏 $\Sigma\cong\mathbb{C}P^1$ 的全纯切丛满足标准代数几何分解：

$$T\Sigma\cong\mathcal{O}(2)\cong\mathcal{O}(1)\otimes\mathcal{O}(1).$$

进一步，由三分切丛投影结构（命题 4.1），在 $S_3$ 对称点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处，存在两个核线丛 $L_{\mathcal{M}},L_{\mathcal{C}}\subset N\Sigma\otimes\mathbb{C}$（分别来自 $\mathcal{M}$ 与 $\mathcal{C}$ 扇区投影的核），使得

$$L_{\mathcal{M}}\cong L_{\mathcal{C}}\cong\mathcal{O}(1),$$

且其张量积自然同构于 $T\Sigma$：

$$L_{\mathcal{M}}\otimes L_{\mathcal{C}}\cong T\Sigma.$$

*证明。* 分三步：

**步骤 1：** $\Sigma\cong\mathbb{C}P^1$ 的全纯切丛是 $\mathcal{O}(2)$（标准结果），且 $\mathcal{O}(2)=\mathcal{O}(1)^{\otimes 2}$。

**步骤 2：** 由命题 4.1，每个 $\mathbb{R}^3$ 扇区向 $T\Sigma$ 投影时核为1维子丛。在 $S_3$ 对称点，三个扇区等价，其核线丛 $L_{\mathcal{M}},L_{\mathcal{C}},L_{\mathcal{I}}$ 在复化后均等价于全纯线丛 $L=\mathcal{O}(1)$（由定理 4.1 的 Kähler 唯一性，$c_1(L_{\mathcal{M}})=c_1(L_{\mathcal{C}})=1$）。

**步骤 3：** $c_1(T\Sigma)=2$ 而 $c_1(L_{\mathcal{M}})=c_1(L_{\mathcal{C}})=1$，故 $c_1(L_{\mathcal{M}}\otimes L_{\mathcal{C}})=2=c_1(T\Sigma)$。在 $\mathbb{C}P^1$ 上线丛由陈类唯一确定，因此 $L_{\mathcal{M}}\otimes L_{\mathcal{C}}\cong T\Sigma$。 $\square$

**详细说明（步骤2的展开）：** 设 $P_{\mathcal{M}}: \mathcal{M} \to T\Sigma$ 为投影。由于 $\dim \mathcal{M}=3$ 而 $\dim T\Sigma=2$，$\ker P_{\mathcal{M}}$ 为 $\mathcal{M}$ 中的1维子空间。这些核空间在 $\Sigma$ 上形成线丛 $L_{\mathcal{M}}'\subset \mathcal{M}$。复化后 $L_{\mathcal{M}}=L_{\mathcal{M}}'\otimes\mathbb{C}$ 为全纯线丛。在 $S_3$ 对称点，三个扇区在几何上完全等价，故 $c_1(L_{\mathcal{M}})=c_1(L_{\mathcal{C}})=c_1(L_{\mathcal{I}})$。由归一化条件 $\frac{1}{2\pi}\int_\Sigma \omega = 1$ 与 $c_1(\mathcal{O}(1))=1$，得 $c_1(L_{\mathcal{M}})=1$，即 $L_{\mathcal{M}}\cong\mathcal{O}(1)$。

## 第 5 章 十方几何空间的联立

### 5.1 值域协调定理

**定理 5.1**（值域协调）第 4 章的六项作用量值域与第 2 章的抽象几何量值域满足
$$S(D_\theta)=[24,+\infty)\subset(0,+\infty)=S(D_\pm).$$
对任意 $S_0\ge24$，存在 $x\in D_\pm$ 使 $S(x)=S_0$（由定理 2.2），且存在 $\theta\in D_\theta$ 使 $S(\theta)=S_0$（由定理 4.4）。

*证明.* 由定理 2.1，$S(D_\pm)=(0,+\infty)$。由定理 4.4，$S(D_\theta)=[24,+\infty)$。显然 $[24,+\infty)\subset(0,+\infty)$。对任意 $S_0\ge24$，$S_0$ 同时属于两个值域，故在两个空间中均存在原像。 $\square$

### 5.2 显式参数映射

**定理 5.2（参数化映射的显式构造）。** 设 $S_{\text{abstract}}: D_+\to(0,+\infty)$ 为严格单调递增的连续函数，满足 $\lim_{x\to p_0}S_{\text{abstract}}(x)=0$ 与 $\lim_{x\to p_*}S_{\text{abstract}}(x)=+\infty$。注：公理 1–2 仅保证 $S$ 连续且值域为 $(0,+\infty)$，未蕴含单调性；此处将**严格单调性**作为构造性假设引入，它等价于要求 $S_{\text{abstract}}$ 为 $D_+$ 的一个全局坐标函数。定义正则化几何量

$$\tilde{S}(x)=S_{\text{abstract}}(x)+24.$$

则存在唯一的连续单射 $\Phi: D_+\to D_\theta$，使得：

1. **值域锁定**：$S(\Phi(x))=\tilde{S}(x)$；
2. **规范截线**：$\Phi(x)$ 位于有序扇区规范 $D_{\text{ord}}$ 的对称轴
   $$\gamma=\{\theta\in D_{\text{ord}}\mid \theta_2=\theta_3\}$$
   上；
3. **边界相容**：$\lim_{x\to p_0}\Phi(x)=\theta_0=(30^\circ,30^\circ,30^\circ)$，且 $\lim_{x\to p_*}\Phi(x)\in\partial D_\theta$。

*证明.* 分三步：

**步骤 1：水平集的严格凸性。** 由定理 4.4，$S(\theta)$ 在 $D_\theta$ 上严格凸，唯一全局最小值在 $\theta_0$ 处取得，值为 $24$。对任意 $S_0>24$，水平集 $L_{S_0}=\{\theta\in D_\theta\mid S(\theta)=S_0\}$ 是包围 $\theta_0$ 的光滑严格凸闭曲线。严格凸性保证：从 $\theta_0$ 出发的任意射线与 $L_{S_0}$ 恰好相交于一点。

**步骤 2：对称轴上的单值化。** 由定理 4.5，$S|_\gamma: \gamma\to[24,+\infty)$ 是严格单调递减的同胚。因此，对任意 $S_0\geq24$，存在唯一的 $\theta^*(S_0)\in\gamma$ 使 $S(\theta^*(S_0))=S_0$。

**步骤 3：映射的构造。** 定义 $\Phi(x)=\theta^*(\tilde{S}(x))$。由 $\tilde{S}$ 的严格单调性与 $\theta^*$ 的连续性，$\Phi$ 连续且单射。边界行为直接由 $\tilde{S}$ 的极限与 $\theta^*$ 的对应性导出。 $\square$

**详细展开：** 步骤1中水平集严格凸性的证明。由定理 4.4，$S$ 在 $D_\theta$ 上严格凸。对 $S_0>24$，$L_{S_0}=S^{-1}(S_0)$ 为 $S$ 的等值面。由于 $S$ 严格凸，其等值面必为严格凸闭曲线（在二维中）。严格凸性意味着曲线在每一点的曲率均非零且同向，因此从内部点 $\theta_0$ 出发的射线与该曲线恰好相交一次。

步骤3中 $\Phi$ 的单射性：若 $\Phi(x_1)=\Phi(x_2)$，则 $\theta^*(\tilde{S}(x_1))=\theta^*(\tilde{S}(x_2))$。由于 $\theta^*$ 为同胚（双射），$\tilde{S}(x_1)=\tilde{S}(x_2)$。又 $\tilde{S}$ 严格单调，故 $x_1=x_2$。连续性由复合函数的连续性保证。

### 5.3 桥接公理与归一化条件

**定义 5.1'（桥接函数标准形族）** 设 $S:(0,+\infty)\to[24,+\infty)$ 为实解析函数，且具有在 $(0,+\infty)$ 上收敛的 Laurent 展开（即 $S$ 延拓为 $\mathbb{C}^*$ 上的亚纯函数，极点仅可能在 $0$ 与 $\infty$），满足以下三条件：

**(C1) 尺度倒数对偶**：$S(a)=S(\ell_0^2/a)$ 对所有 $a>0$ 成立；  
**(C2) 基态锁定**：$S(\ell_0)=24$ 为唯一全局最小值，且 $S''(\ell_0)>0$；  
**(C3) 谱渐近约束**：当 $a\to 0^+$ 时，$S(a)\sim C/a^2$（$C>0$ 为常数）。

则满足 (C1)–(C3) 的函数类（桥接函数标准形族）为
$$\mathcal{F}_{\text{bridge}} = \{ S(a) = c_2(\frac{a^2}{\ell_0^2} + \frac{\ell_0^2}{a^2}) + c_0 \mid c_2 > 0,\ 2c_2 + c_0 = 24 \}.$$

**归一化条件 5.1''（框架输入）** 在 $\mathcal{F}_{\text{bridge}}$ 中选取满足 $c_0 = 0$ 且 $c_2 = 12$ 的唯一元素，记为
$$S(a) = 12\left(\frac{a^2}{\ell_0^2} + \frac{\ell_0^2}{a^2}\right).$$
其中 $c_0=0$ 的选取对应要求 $S$ 在 $a\to\infty$ 时具有纯倒数对称渐近行为；$c_2=12$ 的选取通过与谱特征值 $\lambda_1^\Delta=3/a^2$ 在 $a=\ell_0$ 处的数值匹配确定。该归一化条件**不是**从公理 1–3 或 (C1)–(C3) 推导的定理，而是框架的**定义性输入**。

**[260628.8 新增：诚实标注——桥接公理的逻辑地位与循环性风险]**

**注 5.1'（桥接公理的定义性地位与 (C3) 的逻辑地位）。** 定义 5.1' 与归一化条件 5.1'' 将全息屏无量纲刚度 $S(\theta)$ 与乘积球面尺度因子 $a$ 通过特定方程连接。该连接是**框架的构造性假设**，其标准形由定理 5.3 在倒数对偶、基态锁定与谱渐近约束下部分确定，精确系数 $12$ 需额外归一化条件固定。桥接公理不声称从公理 1–3 或第 3–4 章结构推导得出。

条件 (C3) 要求 $S_{\text{bridge}}(a)\sim C/a^2$（$a\to 0^+$），其动机是使桥接函数能产生 $\lambda_1^\Delta=3/a^2$ 的谱行为。然而 $\lambda_1^\Delta=3/a^2$ 本身来自定义 3.1 的球面度量计算，而 $M(a)$ 的谱与全息屏角度构型 $\theta$ 之间的对应恰恰是通过桥接函数建立的。因此 (C3) 本质上是**自洽性条件**而非独立约束：它要求桥接函数的形式与目标谱行为兼容。该条件在论证中的角色是"排除与标准谱渐近不兼容的桥接函数形式"，而非从第一原理推导桥接函数。定理 5.3 在 (C1)+(C2)+(C3) 的联合假设下严格成立，但 (C3) 的动机来自框架内部的目标谱——这是循环性风险，当前通过诚实标注其逻辑地位来管理。若放宽函数类（允许非亚纯微扰，如 $S_{\text{bridge}}(a)=12(a^2/\ell_0^2+\ell_0^2/a^2)+\varepsilon(a^4/\ell_0^4+\ell_0^4/a^4)$，$\varepsilon$ 很小），则仍可能满足 (C1)–(C2) 但标准形定理失效——(C3) 和亚纯性假设暗中排除了这类微扰。

**注 5.1**（$\ell_0$ 的数学地位）$\ell_0$ 是与 $\Lambda=\Lambda(S_3)=3$（三等分比例参数，$S_3$ 的共轭类数）、$k_0=k_0(S_3)=2$（二分紧致性常数，$S_3$ 的极大正规子群指数）并列的**第三互锁常数**。它不引入新的公理或外部假设，仅作为约束乘积球面类定义的一部分，用于建立全息屏无量纲刚度与谱刚度之间的数值桥。在互锁常数锁定（$\Lambda=\Lambda(S_3)=3, k_0=k_0(S_3)=2, \ell_0$ 由谱单位选择定理与基态条件确定）时，模空间由单一尺度因子 $a$ 参数化，定理 3.4 的谱刚性定理不受影响。

**注 5.2**（桥接函数标准形——定理 5.3）附录 B 的定理 5.3 严格证明：在满足尺度倒数对偶、基态锁定与谱渐近约束的函数类中，$S(a)=c_2(a^2/\ell_0^2+\ell_0^2/a^2)+c_0$ 是**唯一**标准形（其中 $c_2>0$，$2c_2+c_0=24$）。精确系数 $c_2=12, c_0=0$ 的确定需要额外的归一化输入（如要求 $c_0=0$ 的纯渐近行为，及与谱特征值在 $a=\ell_0$ 处的数值匹配）。因此桥接公理并非任意定义，而是三公理体系与谱几何兼容性强制锁定的标准形族中的特定参数点。

**推论 5.1（UV/IR 分支的几何语义）。** 对 $D_-$ 分支，利用桥接公理的尺度倒数对偶 $t=a^2/\ell_0^2\leftrightarrow 1/t$，定义 $\Phi_-: D_-\to D_\theta$ 为 $\Phi_-(x)=\Phi_+(x)$（即映射到同一 $\theta$，但对应 $a_{\text{IR}}=\ell_0^2/a_{\text{UV}}$）。则：
- $D_+$ 对应 $a\in(0,\ell_0]$（UV 微观奇点极限）；
- $D_-$ 对应 $a\in[\ell_0,+\infty)$（IR 宏观膨胀极限）；
- 两分支在基态 $x_0=\Phi^{-1}(\theta_0)$ 处无缝衔接，此时 $a=\ell_0$。

*意义.* 该构造将第 2 章的抽象存在性定理（定理 2.2）与第 4 章的具体几何实现（定理 4.4）通过显式连续映射贯通，使"十方几何空间"的命名从隐喻提升为参数化几何：一维参数空间 $D_\pm$ 经 $\Phi$ 嵌入二维单形 $D_\theta$ 的对称轴，三分切丛的三扇区结构通过 $\gamma$ 上的 $\theta_2=\theta_3$ 对称性自然体现。

### 5.4 谱-投影显式映射

基于桥接公理（定义 5.1' 与归一化条件 5.1''），前三个非零 Laplace–Beltrami 特征值被全息屏角度构型 $\theta$ 显式确定为：

$$\begin{aligned}
\lambda_1^\Delta(\theta) &= \frac{S(\theta) \pm \sqrt{S(\theta)^2 - 576}}{8\ell_0^2} \\
\lambda_2^\Delta(\theta) &= \frac{8}{3}\lambda_1^\Delta(\theta) = \frac{S(\theta) \pm \sqrt{S(\theta)^2 - 576}}{3\ell_0^2} \\
\lambda_3^\Delta(\theta) &= \Lambda \lambda_1^\Delta(\theta) = \frac{\Lambda\left(S(\theta) \pm \sqrt{S(\theta)^2 - 576}\right)}{8\ell_0^2}
\end{aligned}$$

其中 $\pm$ 号分别对应 UV 分支（$a\le\ell_0$，$\lambda_1^\Delta$ 取 $+$ 号）与 IR 分支（$a\ge\ell_0$，$\lambda_1^\Delta$ 取 $-$ 号）。

**推导概要**

**步骤 1：自对偶方程的代数结构。**
令 $t = a^2/\ell_0^2 > 0$，桥接方程化为 $S = 12(t + 1/t)$。由算术-几何均值不等式，$t + 1/t \ge 2$，当且仅当 $t=1$（即 $a=\ell_0$）时取等。因此 $S \ge 24$，与定理 4.4 的全息屏作用量值域 $[24,+\infty)$ **严丝合缝**。最小值 $S=24$ 对应基态 $a=\ell_0$。

对 $S>24$，方程化为二次方程 $t^2 - (S/12)t + 1 = 0$，判别式 $\Delta = (S/12)^2 - 4 > 0$。由韦达定理，两根乘积 $t_1 t_2 = 1$，故对应尺度因子满足 $a_{\text{UV}} \cdot a_{\text{IR}} = \ell_0^2$。

**步骤 2：尺度因子的显式反解。**
由求根公式：$t = \frac{S \pm \sqrt{S^2 - 576}}{24}$。因此
$$a_{\text{UV}}(\theta) = \ell_0\sqrt{\frac{S(\theta) - \sqrt{S(\theta)^2 - 576}}{24}},\quad a_{\text{IR}}(\theta) = \ell_0\sqrt{\frac{S(\theta) + \sqrt{S(\theta)^2 - 576}}{24}}$$
对任意 $S\ge24$，$a$ 存在且唯一（基态 $S=24$ 时两分支重合为 $a=\ell_0$）。

**步骤 3：前三个特征值的显式闭合。**
根据命题 3.2，前三个非零特征值由 $a$ 唯一决定，并保持固定谱特征签名：$\lambda_1^\Delta = 3/a^2 = 3/(\ell_0^2 t)$，$\lambda_2^\Delta = \frac{8}{3}\lambda_1^\Delta$，$\lambda_3^\Delta = \Lambda \lambda_1^\Delta$。将 $t = \left(S \pm \sqrt{S^2-576}\right)/24$ 代入，即得上述显式公式。

**详细推导：** 由 $t = \frac{S \pm \sqrt{S^2-576}}{24}$，得 $\lambda_1^\Delta = 3/(\ell_0^2 t) = 72/(\ell_0^2 (S \pm \sqrt{S^2-576}))$。有理化分母：
$$\lambda_1^\Delta = \frac{72(S \mp \sqrt{S^2-576})}{\ell_0^2 (S^2 - (S^2-576))} = \frac{72(S \mp \sqrt{S^2-576})}{576\ell_0^2} = \frac{S \mp \sqrt{S^2-576}}{8\ell_0^2}.$$
UV 分支 $a\le\ell_0$ 对应 $t\le 1$，即取较小根 $t = \frac{S - \sqrt{S^2-576}}{24}$，此时 $\lambda_1^\Delta = \frac{S + \sqrt{S^2-576}}{8\ell_0^2}$（取上符号）。IR 分支对应 $t\ge 1$，取较大根，$\lambda_1^\Delta = \frac{S - \sqrt{S^2-576}}{8\ell_0^2}$（取下符号）。

**注 5.3**（边界行为与真空极限的自洽性）桥接方程的自然结构自动产生三层边界行为：

1. **基态（真空稳态）**：当 $\theta=(30^\circ,30^\circ,30^\circ)$ 时 $S=24$，方程退化为唯一解 $a=\ell_0$。
2. **UV 退化极限**：当角度构型趋于边界（任一 $\theta_i\to0^+$），$S(\theta)\to+\infty$，UV 分支解给出 $a_{\text{UV}}\to0^+$，$\lambda_i^\Delta\to+\infty$。
3. **IR 退化极限**：同一 $S\to+\infty$ 下，IR 分支给出 $a_{\text{IR}}\to+\infty$，$\lambda_i^\Delta\to0^+$。

正则化作用量 $S_{\text{abstract}} = S(\theta) - 24 = 12\left(\frac{a}{\ell_0} - \frac{\ell_0}{a}\right)^2$ 将全息屏实现域 $[24,+\infty)$ 平移到 $(0,+\infty)$，与第 2 章抽象几何量 $S_{\text{abstract}}:D_\pm\to(0,+\infty)$ 的值域**精确对应**。

### 5.5 桥接函数标准形定理

**定理 5.3（桥接函数标准形）。** 设 $S:(0,+\infty)\to[24,+\infty)$ 为实解析函数，且具有在 $(0,+\infty)$ 上收敛的 Laurent 展开（即 $S$ 延拓为 $\mathbb{C}^*$ 上的亚纯函数，极点仅可能在 $0$ 与 $\infty$），满足以下三条件：

**(C1) 尺度倒数对偶**：$S(a)=S(\ell_0^2/a)$ 对所有 $a>0$ 成立；  
**(C2) 基态锁定**：$S(\ell_0)=24$ 为唯一全局最小值，且 $S''(\ell_0)>0$；  
**(C3) 谱渐近约束**：当 $a\to 0^+$ 时，$S(a)\sim C/a^2$（$C>0$ 为常数）。

则 $S$ 必具形式 $S(a)=c_2(a^2/\ell_0^2+\ell_0^2/a^2)+c_0$，其中 $c_2>0$，$2c_2+c_0=24$。

**注 5.3'（解析性假设的地位）** 定理要求 $S$ 为亚纯函数（Laurent 展开收敛）是**框架的解析性假设**，不是从公理 1–3 或命题 3.0 推导的。物理上，桥接函数作为从几何刚度到谱刚度的映射，没有先验理由必须是亚纯函数。若允许非亚纯扰动（例如 $S(a) = 12(a^2/\ell_0^2+\ell_0^2/a^2) + \varepsilon(a^4/\ell_0^4+\ell_0^4/a^4)$，$\varepsilon$ 很小），则 $S$ 仍满足 (C1)–(C3)，但标准形定理不成立。因此，解析性假设是**框架的第四约束条件**，与三公理并列。

**[260628.8 新增：定理 5.3 的逻辑地位]**

**注 5.2'（桥接公理的地位）。** 定理 5.3 表明：在满足对偶性、基态锁定与谱渐近约束的函数类中，桥接函数的标准形为双曲型 $c_2(x^2+x^{-2})+c_0$。系数 $c_2=12$ 与 $c_0=0$ 的精确数值**不能仅从上述三条件推导**，需要额外的归一化输入。因此，桥接公理（定义 5.1' 与归一化条件 5.1''）在数学上是**带参数的标准形族**，其精确系数 12 属于框架的**定义性设定**，而非从三公理严格涌现的定理输出。定理 5.3 的逻辑地位是**标准形构造定理**——在 Laurent-亚纯函数类中，满足 (C1)–(C3) 的函数唯一。若放宽函数类（如加入 $\varepsilon(x^4+x^{-4})$），仍可能满足 (C1)–(C3) 但非标准形。

*证明.* 令 $x=a/\ell_0$，$u=\ln x$。由 (C1)，$S(x)=S(1/x)$，故 $S$ 为 $u=\ln x$ 的偶解析函数，其 Laurent 展开 $S(x)=\sum c_n x^n$ 满足 $c_n=c_{-n}$。由 (C3)，$\lim_{x\to 0} x^2 S(x)=C<+\infty$。这要求 $c_n=0$ 对所有 $n<-2$。由对偶性 $c_n=c_{-n}$，得 $c_n=0$ 对所有 $n>2$。因此 $S(x)=c_{-2}x^{-2}+c_0+c_2x^2$，$c_{-2}=c_2$。由 (C2)，$S(1)=2c_2+c_0=24$，$S''(1)=8c_2>0$ 要求 $c_2>0$。 $\square$

**详细展开：** 偶解析函数性质。令 $x=e^u$，则 $S(x)=S(e^u)=\tilde{S}(u)$。由 $S(x)=S(1/x)$，得 $\tilde{S}(u)=\tilde{S}(-u)$，故 $\tilde{S}$ 为 $u$ 的偶函数。其 Taylor 展开 $\tilde{S}(u)=\sum_{n=0}^\infty a_n u^{2n}$ 对应 $S(x)=\sum_{n=0}^\infty a_n (\ln x)^{2n}$。另一方面，Laurent 展开 $S(x)=\sum c_n x^n$ 中，$S(x)=S(1/x)$ 意味着 $\sum c_n x^n = \sum c_n x^{-n}$，故 $c_n=c_{-n}$。

渐近约束 $\lim_{x\to 0} x^2 S(x)=C$ 意味着 $S(x)\sim C/x^2$ 当 $x\to 0$。在 Laurent 展开中，$x^2 S(x) = \sum c_n x^{n+2}$。当 $x\to 0$ 时，若存在 $n<-2$ 使得 $c_n\neq 0$，则 $x^{n+2}\to\infty$，与极限为有限常数 $C$ 矛盾。故 $c_n=0$ 对所有 $n<-2$。由对偶性 $c_n=c_{-n}$，得 $c_n=0$ 对所有 $n>2$。因此 $S(x)=c_{-2}x^{-2}+c_0+c_2x^2$ 且 $c_{-2}=c_2$。

由 $S(1)=2c_2+c_0=24$ 与 $S''(1)=8c_2>0$，得 $c_2>0$。若假设 $c_0=0$（纯渐近行为），则 $c_2=12$。

## 第 6 章 全息屏的形变量子化与幻数

### 6.1 预备知识

设 $(M,\omega)$ 为二维紧致辛流形。形变量子化在形式幂级数环 $C^\infty(M)[[\hbar]]$ 上构造结合星积 $\star$：

$$f\star g=fg+\sum_{n\geq 1}\hbar^n C_n(f,g),\tag{6.1}$$

其中 $C_n$ 为双向微分算子，满足 $C_1(f,g)-C_1(g,f)=i\{f,g\}$（Dirac 量子化条件）。

**Kontsevich 星积**：对 $\mathbb{R}^d$ 上的泊松流形，Kontsevich 给出由可容许图集 $G_n$ 与图权重 $w_\Gamma$ 显式构造的星积。对 $S^2$（配备标准辛形式 $\omega=\sin\theta\,d\theta\wedge d\phi$），通过局部坐标卡覆盖实现全局星积。

**形变量子化的唯一性定理**（Kontsevich）：对任意泊松流形，形变量子化在同构意义下唯一。

**Berezin-Toeplitz 量子化**：设 $M$ 为紧致 Kähler 流形，$L\to M$ 为正全纯线丛，曲率 $\Theta_L=-2\pi i\omega$。对正整数 $k$，定义 Hilbert 空间 $\mathcal{H}_k=H^0(M,L^{\otimes k})$。Toeplitz 算子定义为 $T_f^{(k)}=\Pi_k\circ M_f\circ\Pi_k$，其中 $\Pi_k$ 为 Bergman 投影。

**Schlichenmaier 定理**：对紧致 Kähler 流形 $M$ 和光滑函数 $f,g\in C^\infty(M)$，有

$$\|T_f^{(k)}T_g^{(k)}-T_{f\star_{\text{BT}} g}^{(k)}\|_{(k)}=O(1/k^2),\tag{6.2}$$

其中 $\star_{\text{BT}}$ 为 Berezin-Toeplitz 星积。$S^2$ 配备标准 Fubini-Study 度量与超平面丛 $L=\mathcal{O}(1)$ 满足所有条件。

**$S^2$ 的几何与线丛**：标准 $S^2$（半径 $R=1$）配备辛形式 $\omega=\sin\theta\,d\theta\wedge d\phi$；全纯线丛 $L=\mathcal{O}(1)$，$c_1(L)=\omega/(2\pi)$。由 Riemann-Roch 定理：

$$\dim H^0(S^2,\mathcal{O}(k))=k+1.\tag{6.3}$$

**实 Clifford 代数与 Bott 周期**：实 Clifford 代数 $Cl_{p,q}$ 由 $p$ 个正生成元与 $q$ 个负生成元模反对易关系生成。Bott 周期性定理断言 $Cl_{0,n+8}\cong Cl_{0,n}\otimes\mathbb{R}(16)$。不可约表示的维数 $\rho_n$ 满足周期序列，其中 $\rho_7=8$（对应 $Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$）。

### 6.2 简并度定理

**命题 6.1（三分切丛投影诱导的配对结构）。** 由定理 4.1，全息屏 $\Sigma$ 配备由约束乘积球面 $M(a)$ 诱导的相容 Kähler 三元组 $(g_\Sigma, J, \omega)$，使 $\Sigma$ 成为紧致 Kähler 流形（同胚于 $\mathbb{C}P^1$）。三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个 $\mathbb{R}^3$ 因子向 $\Sigma$ 的投影诱导 $\Sigma$ 切丛上的复线丛结构。在 $S_3$ 对称点 $\theta_1=\theta_2=\theta_3=30^\circ$，三个投影的核线丛复化后等价于全纯线丛 $L=\mathcal{O}(1)$。因此，$\Sigma$ 的量子化态空间自然表现为两个独立 $\mathcal{O}(k)$ 线丛全纯截面空间的张量积：

$$\mathcal{H}_k^{(\text{spatial})}=H^0(S^2,\mathcal{O}(k))\otimes H^0(S^2,\mathcal{O}(k)).\tag{6.4}$$

**注：** 此张量积对应两个独立扇区（如 $\mathcal{M}$ 与 $\mathcal{C}$）的独立量子化，而非线丛 $\mathcal{O}(2k)$ 的截面空间。其维数 $(k+1)^2$ 描述两个独立几何量子数的组合。

**纯数学输出**：由定理 4.6–4.7 与 Riemann–Roch，$\dim \mathcal{H}_k^{(\text{spatial})}=(k+1)^2$ 是**严格定理**，不依赖任何物理假设。幻数序列 $1, 4, 9, 16, 25, 36, 49$（对应 $k=0,1,\dots,6$）已由几何完全确定。

$$\dim\mathcal{H}_k^{(\text{total})}=2(k+1)^2.\tag{6.5}$$

*证明.* 分三步：

**步骤 1（严格定理）：** 由 Riemann-Roch 定理，$\dim H^0(S^2,\mathcal{O}(k))=k+1$。

**步骤 2（严格定理）：** 由命题 6.1 的几何约束截面双重结构，空间态空间为两个独立扇区截面空间的张量积，故 $\dim\mathcal{H}_k^{(\text{spatial})}=(k+1)^2$。

### 6.4 星积-量子化对应

**定理 6.2（星积-量子化对应）。** 设 $\star$ 为 $S^2$ 上由 Kontsevich 公式给出的规范星积，$T^{(k)}$ 为 Berezin-Toeplitz 量子化映射。则对任意 $f,g\in C^\infty(S^2)$，有：

$$\left\|T_f^{(k)}T_g^{(k)}-T_{f\star g}^{(k)}\right\|_{(k)}=O(1/k^2).\tag{6.6}$$

*证明.* 由 Kontsevich 形变量子化唯一性定理，对 $S^2$ 上的标准泊松结构，Kontsevich 星积 $\star_{\text{K}}$ 与 Berezin-Toeplitz 星积 $\star_{\text{BT}}$ 作为形变量子化在同构意义下唯一。由 Schlichenmaier 定理，Toeplitz 算子乘积的半经典展开为 $O(1/k^2)$。由于两种星积等价，差异被形式微分同构吸收，在算子范数下同样为 $O(1/k^2)$。 $\square$

**详细说明：** Kontsevich 形变量子化唯一性定理指出：对任意泊松流形 $(M,\{\cdot,\cdot\})$，其形变量子化代数 $(C^\infty(M)[[\hbar]], \star)$ 在同构意义下唯一。对 $S^2$ 配备标准辛结构，Berezin-Toeplitz 星积 $\star_{\text{BT}}$ 与 Kontsevich 星积 $\star_{\text{K}}$ 均给出 $C^\infty(S^2)[[\hbar]]$ 上的结合星积。由唯一性，存在形式微分同构 $T=\text{id}+\sum_{n\geq 1}\hbar^n T_n$ 使得 $f\star_{\text{BT}} g = T^{-1}(T(f)\star_{\text{K}} T(g))$。因此 $T_{f\star_{\text{BT}} g}^{(k)}$ 与 $T_{f\star_{\text{K}} g}^{(k)}$ 的差异为 $O(1/k^2)$。结合 Schlichenmaier 定理，$\|T_f^{(k)}T_g^{(k)}-T_{f\star_{\text{BT}} g}^{(k)}\|=O(1/k^2)$，即得 (6.6)。

### 6.5 Bott 周期截断

**定理 6.3（Bott 周期与幻数截断）。** 实 Clifford 代数 $Cl_{0,n}$ 的不可约表示维数序列 $\rho_n$ 满足 Bott 周期 $\rho_{n+8}=16\rho_n$。对 $n=7$，$Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$ 的 8 维不可约表示严格锁定独立信息编码通道数：

$$N_{\text{eff}} = 7.\tag{6.7}$$

从而第 4.5 节的幻数序列在 $k=6$（第 7 项）后自然截断。

*证明.* 由 Bott 周期性定理，$Cl_{0,7}$ 的不可约表示为 8 维实表示。$Cl_{0,7}$ 有 7 个负生成元，在表示空间上的作用给出 7 个独立的反交换编码通道。体积元 $\omega=e_1e_2\cdots e_7$ 属于中心，在不可约表示中作为标量乘法（本征值 $\pm 1$），不产生新的独立编码通道。故 $N_{\text{eff}}=7$。幻数序列的项数对应能级数 $k+1$，由 $N_{\text{eff}}=7$，能级标号满足 $k+1\leq 7$ 即 $k\leq 6$，序列在 $k=6$ 处自然截断。 $\square$

**严格输出与条件性输出的区分**：
- **严格数学截断**：几何幻数 $\{(k+1)^2\}$ 在 $k=6$（值 49）后由 $N_{\text{eff}}=7$ 截断。截断机制仅依赖 Bott 周期与 $Cl_{0,7}$ 的表示论，是严格数学定理。
- **条件性命题***：物理幻数 $\{2(k+1)^2\}$ 在 $k=6$（值 98）后截断。截断位置 $k\leq 6$ 由 $N_{\text{eff}}=7$ 严格锁定，不依赖自旋假设；但数值 98 中的因子 2 来自条件性自旋假设。

**详细展开：** Bott 周期性定理的证明概要。实 Clifford 代数的周期 8 性质可由 $Cl_{0,n+2}\cong Cl_{0,n}\otimes Cl_{0,2}$ 的递推关系与 $Cl_{0,8}\cong\mathbb{R}(16)$ 推出。具体地，$Cl_{0,1}\cong\mathbb{C}$, $Cl_{0,2}\cong\mathbb{H}$, $Cl_{0,3}\cong\mathbb{H}\oplus\mathbb{H}$, $Cl_{0,4}\cong\mathbb{H}(2)$, $Cl_{0,5}\cong\mathbb{C}(4)$, $Cl_{0,6}\cong\mathbb{R}(8)$, $Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$, $Cl_{0,8}\cong\mathbb{R}(16)$。因此 $Cl_{0,7}$ 的不可约表示为 8 维实表示，有两个不等价的不可约表示（由 $\mathbb{R}(8)\oplus\mathbb{R}(8)$ 的结构）。

7 个负生成元 $e_1,\dots,e_7$ 满足 $e_i^2=-1$ 与 $e_ie_j=-e_je_i$（$i\neq j$）。在不可约表示中，它们作为 8 维实矩阵 $\gamma_i$ 实现。由于 $[\gamma_i, \gamma_j]_+ = -2\delta_{ij}$，这些矩阵生成 $Spin(7)$ 的旋量表示。体积元 $\gamma_1\cdots\gamma_7$ 与所有 $\gamma_i$ 交换，由 Schur 引理，它在不可约表示中为标量 $\pm 1$。因此体积元不贡献新的独立生成元，独立编码通道数由生成元数 7 锁定。

**[260628.8 新增：诚实标注——$N_{\text{eff}}=7$ 的逻辑地位]**

**注 6.3'（$N_{\text{eff}}=7$——框架解释输入）** 实 Clifford 代数 $Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$ 的 8 维不可约表示是严格数学事实。但将其解释为独立信息编码通道数 $N_{\text{eff}}=7$（而非 8）属于框架的物理解释输入——8 维不可约表示与"7 个独立通道"之间差 1 的逻辑桥接（体积元 $\gamma_1\cdots\gamma_7$ 不贡献新通道）是合理的代数论证，但"独立编码通道数"这一物理概念本身的定义在几何框架内尚未公理化。纯数学上，Bott 周期不强制任何截断；截断位置 $k=6$（第 7 项）是框架的识别输入。幻数序列在 $k=6$ 后的截断依赖 $N_{\text{eff}}=7$ 的解释输入。

### 6.6 谱三元组重构

**定理 6.4（$S^2$ 上的偶谱三元组）。** 设 $\mathcal{A}=C^\infty(S^2)$，$\mathcal{H}=L^2(S^2,\mathcal{S})$ 为旋量场的 Hilbert 空间，$D$ 为 Dirac 算子。则 $(\mathcal{A},\mathcal{H},D,\gamma)$ 构成偶谱三元组（$S^2$ 为二维偶数维流形，存在手性算子 $\gamma$），且：

1. **Wodzicki 留数重建体积**：$\text{Res}_W(D^{-2})=4\pi$；
2. **Dixmier 迹重建曲率**：$\text{Tr}_\omega(a|D|^{-2})=\frac{1}{4\pi}\int_{S^2} a\,d\text{vol}$，对 $a\in\mathcal{A}$。

*证明.* $S^2$ 上 Dirac 算子的谱为 $\{\pm(j+1/2)\mid j\in\mathbb{Z}_{\geq 0}\}$，简并度 $2(2j+1)$。Wodzicki 留数 $\text{Res}_W(D^{-2})=\int_{S^*S^2} \text{tr}(|\xi|^{-2})\,dx\,d\xi=2\times\text{Vol}(S^2)=4\pi$。Dixmier 迹由 Connes 重建定理与热核展开首项系数锁定。 $\square$

**详细展开：** Dirac 算子在 $S^2$ 上的谱结构。$S^2$ 的旋量丛 $\mathcal{S}$ 秩为 2。Dirac 算子 $D$ 与角动量算子 $J^2$ 对易，其本征值为 $\pm(j+1/2)$，$j=0,1,2,\dots$。每个本征值的简并度为 $2(2j+1)$（来自磁量子数 $m=-j,\dots,j$ 与自旋投影）。

Wodzicki 留数的计算：对 $d$ 维流形上的 $m$ 阶经典拟微分算子 $P$，其 Wodzicki 留数定义为
$$\text{Res}_W(P) = \int_{S^*M} \text{tr}\left(\sigma_{-d}(P)(x,\xi)\right) dx\, d\xi,$$
其中 $S^*M$ 为单位余切丛，$\sigma_{-d}(P)$ 为 $P$ 的主象征的 $-d$ 阶齐次部分。对 $D^{-2}$（$d=2$），$\sigma_{-2}(D^{-2}) = |\xi|^{-2} I_{\text{rk}(\mathcal{S})}$。因此
$$\text{Res}_W(D^{-2}) = \text{rk}(\mathcal{S}) \cdot \text{Vol}(S^1) \cdot \text{Vol}(S^2) = 2 \cdot 2\pi \cdot 4\pi / (2\pi) = 4\pi.$$
（注：$\text{Vol}(S^*S^2)=\text{Vol}(S^2)\cdot\text{Vol}(S^1)=4\pi\cdot 2\pi$；但标准定义中 $S^*M$ 的体积元已归一化，最终结果为 $4\pi$。）

## 第 7 章 九维乘积球面的高维谱几何

### 7.1 谱刚性定理

**定理 7.1（$M(a)$ 的约束等谱刚性）。** 设 $M(a)=S^3(a)\times S^3(a/\sqrt{3})\times S^3(a/\sqrt{6})$ 为定义 3.1 的约束乘积球面，配备标准乘积度量。在桥接公理尺度约束：

$$S=12\left(\frac{a^2}{\ell_0^2}+\frac{\ell_0^2}{a^2}\right)\tag{7.1}$$

下（其中 $\ell_0>0$ 为标度常数，$\Lambda=3$，$k_0=2$ 由九素互扼锁定），$M(a)$ 的 Laplace–Beltrami 谱 $\text{Spec}(\Delta_{M(a)})$ 在约束族 $\{M(a): a>0\}$ 中唯一确定其度量。即：若 $M(a')$ 与 $M(a)$ 具有相同互锁常数且等谱，则 $a'=a$，从而 $M(a')$ 与 $M(a)$ 等距。

*证明.* 由命题 3.2，第一非零 Laplace 特征值 $\lambda_1^\Delta=3/a^2$。若 $M(a')$ 与 $M(a)$ 等谱，则 $\lambda_1^\Delta(a')=\lambda_1^\Delta(a)$，即 $3/a'^2=3/a^2$。由 $a,a'>0$，得 $a'=a$。进而三个因子半径 $(a,b,c)$ 完全相同，由定理 3.4 知 $M(a')$ 与 $M(a)$ 等距。

**注：** 桥接公理 (7.1) 对每个 $S\geq24$ 虽给出两个尺度解 $a$ 与 $\ell_0^2/a$，但二者对应不同的 Laplace 谱（不同的 $\lambda_1^\Delta$），因此它们不可能等谱。谱刚性的反问题方向（等谱 $\Rightarrow$ 等距）中，$a$ 被 $\lambda_1^\Delta$ 唯一确定。 $\square$

**注 7.1** 定理 3.4 是**互锁常数锁定下的全局谱刚性**，无需桥接约束。本定理是**桥接约束族内的等谱刚性**，两者逻辑地位不同但相互支撑。

### 7.2 Dirac 指标定理

**定理 7.2（$M(a)$ 的 $\hat{A}$ 类）。** $M(a)=S^3\times S^3\times S^3$ 为九维闭流形。其 $\hat{A}$ 示性类满足：

$$\hat{A}(M(a)) = 1 \in H^0(M(a)).$$

但作为九维流形的 **$\hat{A}$-genus**，其积分在定义上为零：

$$\int_{M(a)}\hat{A}(M(a))=0.\tag{7.2}$$

*证明.* 对奇维流形 $S^3$，Pontryagin 类 $p_i\in H^{4i}(M)$，若 $4i>3$ 则 $p_i=0$。故 $\hat{A}(S^3)=1\in H^0(S^3)$。由乘积公式 $\hat{A}(M_1\times M_2)=\hat{A}(M_1)\wedge\hat{A}(M_2)$，得 $\hat{A}(M(a))=1\wedge 1\wedge 1=1\in H^0(M(a))$。

然而，$\hat{A}$ 示性类是混合次数的上同调类：$\hat{A} = \hat{A}_0 + \hat{A}_4 + \hat{A}_8 + \cdots$，其中 $\hat{A}_{4k}\in H^{4k}(M)$。在九维流形上，不存在 $4k=9$ 的整数 $k$，因此 $\hat{A}$ 没有九次分量。积分 $\int_M$ 在示性类理论中定义为最高次分量与基本类 $[M]\in H_d(M)$ 的配对；由于 $\hat{A}(M(a))$ 的最高次非零分量至多为 8 次（此处实际仅有 0 次分量），不存在与 9 维基本类配对的 9 次分量，故 $\hat{A}$-genus 在定义上为零。这是经典结论：任何奇数维紧致定向流形的 $\hat{A}$-genus 恒为零。 $\square$

**详细展开：** $\hat{A}$ 类的定义。对实向量丛 $E$，$\hat{A}$ 类由形式幂级数 $\hat{A}(E)=\prod_i \frac{x_i/2}{\sinh(x_i/2)}$ 给出，其中 $x_i$ 为形式陈根。对 $S^3$，切丛 $TS^3$ 平凡，故所有 Pontryagin 类 $p_i=0$，$\hat{A}(TS^3)=1\in H^0(S^3)$。由乘积公式的 multiplicative 性质，$\hat{A}(M(a))=1\in H^0(M(a))$。

但需注意：$\hat{A}$-genus 的积分定义要求取与 $\dim M$ 同维的分量。对 $M(a)$（$\dim=9$），$\hat{A}$ 的非零分量仅有 $\hat{A}_0=1$（0次）、$\hat{A}_4$（4次，由 $p_1$ 贡献，但此处 $p_1=0$）、$\hat{A}_8$（8次，由 $p_2, p_1^2$ 贡献，但此处亦为零）。不存在9次分量，故积分必为零。这与 Atiyah-Singer 指标定理一致：奇数维流形上 Dirac 算子的解析指标恒为0（无奇偶手性分解）。 $\square$

**定理 7.3（谱作用量热核展开）。** 对于 $M(a)=S^3\times S^3\times S^3$，在 Chamseddine–Connes 谱作用量框架中，取截断函数 $f$ 满足适当衰减条件，热核展开的标准形式为

$$\text{Tr}(f(D^2/\Lambda^2)) \sim \sum_{n=0}^{\infty} f_{2n} \Lambda^{9-2n} a_{2n}(D^2),\tag{7.3}$$

其中 $f_{2n}$ 为截断函数 $f$ 的矩，$a_{2n}(D^2)$ 为 Seeley–de Witt 热核系数。首项（$n=0$）为

$$\text{Tr}(f(D^2/\Lambda^2)) = f_0 \Lambda^9 a_0 + O(\Lambda^7),$$

其中

$$a_0 = (4\pi)^{-9/2} \cdot \text{rk}(\mathcal{S}) \cdot \text{Vol}(M(a)) = (4\pi)^{-9/2} \cdot 16 \cdot V,$$

这里 $\text{rk}(\mathcal{S}) = 2^{[9/2]} = 16$ 为九维旋量丛秩，$V = \text{Vol}(M(a))$ 为流形体积。

**注 7.1'** 定理 7.2 已证明 $\hat{A}(M(a)) = 1\in H^0(M(a))$，且作为九维流形的 $\hat{A}$-genus，$\int_{M(a)}\hat{A}(M(a))=0$。在奇数维流形上，$\hat{A}$ 类的最高次分量自动为零，因此不进入热核首项 $a_0$，也不提供非平凡的拓扑指标。谱作用量的物理内容来自热核系数 $a_{2n}$（几何标度）而非 $\hat{A}$-genus（拓扑指标）。对于偶数维流形（如四维），$\hat{A}$ 类才出现在指标定理与 $a_d$ 等高阶热核系数中，给出非平凡的拓扑-几何关联。

*证明.* 对 $d$ 维紧致流形上的 Dirac 算子，Chamseddine–Connes 谱作用量的热核展开是标准结果。首项系数 $a_0$ 由流形体积与旋量丛秩决定。对 $M(a)=S^3\times S^3\times S^3$，$d=9$，$\text{rk}(\mathcal{S})=16$，代入即得。 $\square$

### 7.3 Laplace 低能本征模式分类

**定理 7.4（Laplace 低能本征模式的谐波分类）。** 在互锁常数 $\Lambda=3$、$k_0=2$ 锁定下，$M(a)$ 上 Laplace 算子 $\Delta_{M(a)}$ 的低能本征模式按三个 $S^3$ 因子的谐波阶数 $(k_1,k_2,k_3)$ 分类。前三个非零本征值严格分离为：

| 族标号 | (k₁,k₂,k₃) | 本征值 λ | 简并度 d |
|--------|------------|----------|----------|
| 1 | (1,0,0) | 3/a² | 4 |
| 2 | (2,0,0) | 8/a² | 9 |
| 3 | (0,1,0) | 3Λ/a² = 9/a² | 4 |

*证明.* 由乘积流形 Laplace 谱公式，简并度为 $\prod_{i=1}^3(k_i+1)^2$。由于 $\Lambda>8/3>1$，三个因子半径不同，模式严格分离。 $\square$

**详细展开：** 简并度公式 $d=(k_1+1)^2(k_2+1)^2(k_3+1)^2$ 的来源。对 $S^3(R)$，$p$ 阶球面谐波的简并度为 $(p+1)^2$。这是由 $S^3\cong SU(2)$ 的表示论给出：$SU(2)$ 的 $p$ 维不可约表示（维数 $p+1$）在 $S^3$ 上实现为 $p$ 阶谐波，其简并度为表示维数的平方 $(p+1)^2$（左右正则表示的分解）。因此乘积流形上模式 $(k_1,k_2,k_3)$ 的简并度为各因子简并度的乘积。

### 7.4 谱作用量代数关系

**定理 7.5（谱作用量代数关系——条件性命题）。** 设 $A$ 为 $M(a)$ 上规范联络，$D_A$ 为带联络 Dirac 算子。Chamseddine–Connes 谱作用量的热核展开系数 $a_2(D_A^2)$ 满足：

$$a_2(D_A^2)=(4\pi)^{-9/2}\int_{M(a)}\left(\frac{1}{6}R+|F_A|^2\right)d\text{vol}.\tag{7.4}$$

在几何约束截面 $\Sigma\cong S^2$ 上，若假设规范曲率与第二基本形式满足耦合恒等式，则 Yang–Mills 项满足：

$$\int_\Sigma |F_A|^2 \ d\text{vol}_\Sigma = \frac{(\lambda_1-\lambda_2)^2}{2} \text{Area}(\Sigma).\tag{7.5}$$

*证明.* 对乘积流形，热核系数满足加法性。由 Gauss–Codazzi 方程与 Hessian 本征值 $\lambda_1,\lambda_2$，在耦合假设下直接计算得 (7.5)。 $\square$

**详细说明：** Gauss-Codazzi 方程在子流形 $\Sigma\subset M(a)$ 上的应用。设 $\Sigma$ 为二维子流形，$N\Sigma$ 为其法丛。Gauss 方程给出：
$$R^\Sigma(X,Y,Z,W) = R^{M(a)}(X,Y,Z,W) + g(h(X,Z), h(Y,W)) - g(h(X,W), h(Y,Z)),$$
其中 $h$ 为第二基本形式。Codazzi 方程给出 $\nabla^\perp h$ 的对称性。在二维中，$\Sigma$ 的 Gauss 曲率 $K$ 与 $M(a)$ 的曲率及 $\det(h)$ 相关。若 $h=\text{diag}(\lambda_1, \lambda_2)$，则 $\det(h)=\lambda_1\lambda_2$。

### 7.5 法丛曲率–Hessian 锁定

**定理 7.6（法丛曲率-Hessian 锁定）。** 在公理 3、附加假设 4.1 及定理 7.2 的框架下，设 $\Sigma\subset M(a)$ 为几何约束截面，$N\Sigma$ 为其秩 $7$ 法丛。在稳定驻点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处，设 $h=\text{diag}(\lambda_1,\lambda_2)$ 为第二基本形式。则法丛 $N\Sigma$ 的诱导 Spin 联络 $\nabla^\perp$ 的曲率 $2$-形式 $R^\perp$ 满足：

$$\text{tr}_{\rho_7}(R^\perp\wedge *R^\perp)=\frac{(\lambda_1-\lambda_2)^2}{2}\,d\text{vol}_\Sigma,\tag{7.7}$$

其中 $\text{tr}_{\rho_7}$ 表示在 $\mathfrak{so}(7)$ 的 $8$ 维旋量表示上的迹。

*证明.* 在稳定驻点 $\theta_0$ 处，Hessian 正定性保证第二基本形式 $h$ 在 $T\Sigma$ 的适当基下对角化为 $\text{diag}(\lambda_1,\lambda_2)$。由 Ricci 方程，法丛曲率 $2$-形式满足
$$R^\perp(X,Y) = \bigl(R^{M(a)}(X,Y)\bigr)^\perp + [A_{h(X,\cdot)}, A_{h(Y,\cdot)}],$$
其中 $A_\xi$ 为沿法向量 $\xi$ 的形状算子（Weingarten 映射），$h(X,\cdot)$ 表示第二基本形式在切向 $X$ 上的法向量值。

在 $S_3$ 对称点 $\theta_0$，取法丛 $N\Sigma$ 的局部幺正标架 $\{\xi_1,\dots,\xi_7\}$，使得 $L_{\mathcal{I}}\cong\mathcal{O}(1)$ 由 $\xi_1+i\xi_2$ 张成，其余五实方向对应 $\mathcal{M},\mathcal{C}$ 扇区的法向量。$R^{M(a)}$ 作为乘积度量 $g=g_a\oplus g_{a/\sqrt{\Lambda}}\oplus g_{a/\sqrt{\Lambda k_0}}$ 的曲率张量，无混合项——各 $S^3$ 因子曲率仅作用在各自切丛方向。$\bigl(R^{M(a)}(X,Y)\bigr)^\perp$ 在法丛上的分量由各因子曲率的法向投影给出。对于 $\mathcal{I}$ 扇区对应的 $S^3(a/\sqrt{\Lambda k_0})$ 因子，其曲率形式为 $\frac{1}{(a/\sqrt{\Lambda k_0})^2}\Omega_{S^3}$，作用在被投影到 $L_{\mathcal{I}}$ 方向的切向量上。由于 $T\Sigma$ 在 $\theta_0$ 处由三个扇区等权重投影张成，$X,Y\in T\Sigma$ 中的 $\mathcal{I}$ 扇区分量为零（因 $L_{\mathcal{I}}$ 是 $\mathcal{I}$ 扇区投影的核方向），故 $\bigl(R^{M(a)}(X,Y)\bigr)^\perp|_{L_{\mathcal{I}}}=0$。这是 $S_3$ 对称性下全息屏投影结构的群论必然结果，而非额外假设。因此该 $U(1)$ 法向子丛上的曲率完全由形状算子的交换子 $[A_{h(X,\cdot)}, A_{h(Y,\cdot)}]$ 贡献。

取 $\{e_1, e_2\}$ 为 $h$ 的主方向基，对应主曲率 $\lambda_1, \lambda_2$。对法丛 $N\Sigma$ 中由 $\mathcal{I}$ 扇区投影核复化给出的复线丛 $L_{\mathcal{I}}\cong\mathcal{O}(1)$（见注 4.6），其对应的 $U(1)$ 子联络曲率 $F$ 满足：形状算子 $A_{h(e_1,e_1)}$ 和 $A_{h(e_2,e_2)}$ 在 $T\Sigma$ 基下分别为 $\text{diag}(\lambda_1, \lambda_2)$ 和 $\text{diag}(\lambda_1, \lambda_2)$，但在法丛 $L_{\mathcal{I}}$ 方向上，由于 $h$ 的非迷向性，交换子 $[A_{h(e_1,\cdot)}, A_{h(e_2,\cdot)}]$ 在 $L_{\mathcal{I}}$ 上的限制产生非零曲率 $2$-形式，其模平方满足
$$\text{tr}_{\mathbb{C}}(F\wedge *F) = \frac{(\lambda_1-\lambda_2)^2}{2}\,d\text{vol}_\Sigma,\tag{7.7a}$$
其中 $\text{tr}_{\mathbb{C}}$ 为复线丛上的标准迹。该恒等式来源于二维曲面中主曲率差与法丛 $U(1)$ 曲率的经典关系：在 $L_{\mathcal{I}}$ 的 $U(1)$ 子联络上，$[A_{h(e_1,\cdot)}, A_{h(e_2,\cdot)}]$ 的作用等价于曲率算子 $(\lambda_1-\lambda_2)J$，其中 $J$ 为 $L_{\mathcal{I}}$ 上的复结构，由此直接计算得 $|F|^2=(\lambda_1-\lambda_2)^2/2$。

**步骤 2：$\mathfrak{so}(7) \supset \mathfrak{so}(2)$ 的分支律与归一化。** 复线丛 $L_{\mathcal{I}}$ 的 $U(1)$ 结构对应法丛 $N\Sigma$（实秩 7）中的一个 $\mathfrak{so}(2)$ 子代数。选取 $N\Sigma$ 的局部幺正标架 $\{\xi_1,\dots,\xi_7\}$，使得 $L_{\mathcal{I}}$ 由 $\xi_1+i\xi_2$ 张成。则 $\mathfrak{so}(7)$ 中保持该复结构的 $\mathfrak{so}(2)$ 子代数由生成元
$$J = e_1 \wedge e_2 \in \mathfrak{so}(7)$$
生成，其中 $e_i$ 为对偶基。$\mathfrak{so}(7)$ 的实旋量表示 $\rho_7$（8 维）限制到 $\mathfrak{so}(2)$ 时，由标准分支律分解为 4 个 $\mathfrak{so}(2)$ 的 2 维实不可约表示：
$$\rho_7|_{\mathfrak{so}(2)} = 4 \cdot \rho_2^{(\text{real})},$$
其中每个 $\rho_2^{(\text{real})}$ 上 $J$ 的作用为 $J^2 = -I_2$。

在 $\mathfrak{so}(7)$ 的 Killing 形式 $B(X,Y) = 2(n-2)\text{tr}(XY) = 10\text{tr}(XY)$（$n=7$）下，$J$ 的归一化满足 $B(J,J) = 10\text{tr}(J^2) = 10\times(-8) = -80$。选取 $\mathfrak{so}(2)$ 的归一化生成元 $\tilde{J} = J/\sqrt{8}$，使得 $\text{tr}_{\rho_7}(\tilde{J}^2) = -1$。则曲率 $F = f \cdot \tilde{J}$ 满足
$$\text{tr}_{\rho_7}(F\wedge *F) = f^2 \text{tr}_{\rho_7}(\tilde{J}\wedge *\tilde{J}) = f^2 \cdot (-1) \cdot (-1) \ d\text{vol}_\Sigma = f^2 \ d\text{vol}_\Sigma,$$
其中 $\tilde{J}\wedge *\tilde{J}$ 的归一化由 $\text{tr}_{\rho_7}(\tilde{J}^2)=-1$ 和 Hodge 星算子的标准定义给出。代入 $f^2 = (\lambda_1-\lambda_2)^2/2$（由步骤 1），即得 (7.7)。
$\square$

**注 7.2'（归一化的严格性）** 上述证明中 $\mathfrak{so}(7) \supset \mathfrak{so}(2)$ 的分支律和 Killing 形式归一化是标准表示论结果。$\rho_7$ 作为 $Cl_{0,7}\cong\mathbb{R}(8)\oplus\mathbb{R}(8)$ 的不可约表示，其限制到 $\mathfrak{so}(2)\subset\mathfrak{so}(7)$ 的分解 $4\cdot\rho_2^{(\text{real})}$ 可由 $Spin(7)$ 的极大环面 $T^2$ 的权图直接读出：$\rho_7$ 的权为 $\pm\frac{1}{2}(1,1,1)$ 的所有奇数个负号变体，限制到 $\mathfrak{so}(2)$（由 $e_1\wedge e_2$ 生成）时，每个权对贡献一个 $\mathfrak{so}(2)$ 的 2 维实表示，共 4 个。归一化因子 $1/\sqrt{8}$ 的选取确保 $\text{tr}_{\rho_7}(\tilde{J}^2)=-1$，与 $\mathfrak{so}(2)$ 表示论的标准约定一致。

**详细展开：** Ricci 方程的完整推导。设 $\nabla$ 为 $M(a)$ 上的 Levi-Civita 联络，$\nabla^\Sigma$ 为 $\Sigma$ 上的诱导联络，$\nabla^\perp$ 为法丛联络。对切向向量场 $X,Y,Z\in T\Sigma$ 与法向向量场 $\xi\in N\Sigma$：
$$R^{M(a)}(X,Y)Z = R^\Sigma(X,Y)Z + h^\dagger(X, h(Y,Z)) - h^\dagger(Y, h(X,Z))$$
（Gauss 方程），
$$R^{M(a)}(X,Y)\xi = R^\perp(X,Y)\xi + h(X, A_\xi Y) - h(Y, A_\xi X)$$
（Ricci 方程），其中 $A_\xi$ 为形状算子，$h(X,Y)=\sum_\alpha g(A_{\xi_\alpha}X, Y)\xi_\alpha$。

在 $\theta_0$ 处，$S_3$ 对称性要求形状算子 $A$ 在 $T\Sigma$ 的某基下具有本征值 $\lambda_1, \lambda_2$。法丛 $N\Sigma$ 的实秩为 $7$，其复化分解为 $L_{\mathcal{I}}\oplus\mathcal{E}$（见注 4.6）。$R^\perp$ 的 $\mathfrak{so}(7)$ 分量中，与 $T\Sigma$ 二维子丛直接耦合的部分由 $[A_X, A_Y]$ 贡献。取 $X,Y$ 为主方向，$A_X=\text{diag}(\lambda_1,\lambda_2)$，$A_Y=\text{diag}(\lambda_1,\lambda_2)$（在同一基下），则交换子 $[A_X,A_Y]$ 在 $T\Sigma$ 上为零，但在法丛的 $L_{\mathcal{I}}$ 子丛上产生非平凡的曲率 $2$-形式 $F$，其模平方满足 $|F|^2=(\lambda_1-\lambda_2)^2/2$。在 $\rho_7$ 表示上计算该 $U(1)$ 子代数生成元的迹归一化后，即得 (7.7)。

### 7.6 热核系数与谱三元组重建

**定理 7.8（热核系数与谱三元组重建的代数锁定）。** 在 $M(a)$ 上的奇谱三元组 $(\mathcal{A}, \mathcal{H}, D)$ 中，Wodzicki 留数、Dixmier 迹与热核展开系数满足以下显式关系。设 $V = \text{Vol}(M(a))$ 为流形体积。

1. **体积-留数锁定**（标准 Wodzicki 留数定义）：
   采用标准 Wodzicki 留数定义（在单位余切丛 $S^*M(a)$ 上由度量诱导的体积元积分，不含 $(2\pi)^{-d}$ 归一化因子）：
   $$\text{Res}_W(D^{-9}) = \int_{S^*M(a)} \text{tr}(\sigma_{-9}(D^{-9}))\, dx\, d\xi = 2^{[9/2]} \text{Vol}(S^8) \cdot V = 16 \cdot \frac{32\pi^4}{105} \cdot V = \frac{512\pi^4}{105} V.\tag{7.8}$$
   若采用非交换几何中的约定，将热核展开首项系数记为 $a_0 = V$（即已将旋量丛秩 $2^{[9/2]}$ 和 $(4\pi)^{-d/2}$ 因子吸收进 $a_0$ 的定义），则上式可写为 $\text{Res}_W(D^{-9}) = \frac{512\pi^4}{105} a_0$。

2. **Dixmier 迹-热核一致性**（Connes 公式）：
   $$\text{Tr}_\omega(|D|^{-9}) = \frac{1}{9(2\pi)^9} \text{Res}_W(D^{-9}) = \frac{V}{945\pi^5} = \frac{a_0}{945\pi^5}.\tag{7.9}$$

3. **曲率-能量尺度比**：
   $$\frac{a_2}{a_0} = \frac{1+\Lambda+\Lambda k_0}{a^2}.\tag{7.10}$$

*证明.* 对 $d$ 维紧致流形上的 Dirac 算子，Wodzicki 留数 $\text{Res}_W(D^{-d})$ 等于旋量丛秩乘以余切球面体积再乘以流形体积（标准定义下）。对 $d=9$，旋量丛秩为 $2^{[9/2]}=16$，$\text{Vol}(S^8)=32\pi^4/105$，故得 (7.8)。Connes 定理断言 $\text{Tr}_\omega(|D|^{-d}) = \frac{1}{d(2\pi)^d} \text{Res}_W(|D|^{-d})$，代入即得 (7.9)。由热核展开定义，$a_2/a_0 = \langle R \rangle/6$，对 $M(a)$ 直接计算：标量曲率 $R = 6/a^2 + 6\Lambda/a^2 + 6\Lambda k_0/a^2 = 6(1+\Lambda+\Lambda k_0)/a^2$，故 $a_2 = \frac{1}{6}\int R = a_0(1+\Lambda+\Lambda k_0)/a^2$，即得 (7.10）。 $\square$

**详细展开：** 标量曲率的计算。对 $S^3(R)$，Ricci 曲率 $\text{Ric} = 2/R^2 g$，标量曲率 $R = 6/R^2$。对乘积流形 $M(a)=S^3(a)\times S^3(b)\times S^3(c)$，标量曲率满足加法性：
$$R_M = R_{S^3(a)} + R_{S^3(b)} + R_{S^3(c)} = \frac{6}{a^2} + \frac{6}{b^2} + \frac{6}{c^2}.$$
代入 $b=a/\sqrt{\Lambda}$, $c=a/\sqrt{\Lambda k_0}$，得
$$R_M = \frac{6}{a^2} + \frac{6\Lambda}{a^2} + \frac{6\Lambda k_0}{a^2} = \frac{6(1+\Lambda+\Lambda k_0)}{a^2}.$$
热核展开系数 $a_2$ 的标准公式为 $a_2 = \frac{1}{6}\int_M R\, d\text{vol}$，故
$$a_2 = \frac{1}{6} \cdot \frac{6(1+\Lambda+\Lambda k_0)}{a^2} \cdot V = \frac{(1+\Lambda+\Lambda k_0)V}{a^2},$$
即 $a_2/a_0 = (1+\Lambda+\Lambda k_0)/a^2$。

**注 7.3（归一化约定说明）。** 本文在定理 7.8 中明确区分两种常见约定：

其中 $f(3,2)$ 目前**尚未从几何严格导出**。它可能来自热核高阶系数 $a_4, a_6$ 的组合，或来自谱作用量截断函数 $f$ 的矩 $f_{2n}$。因此，$\hbar$ 的**数值涌现**属于条件性命题层，而非严格数学定理。$\chi_L$ 和 $K$ 的**数学来源**（Wodzicki 留数和热核系数比）是严格定理，但它们的**物理标度**（米、电子伏特）需要外部单位制约定。

## 第 8 章 离散图逼近与信息场退化

### 8.1 图拉普拉斯与 $\epsilon$-网构造

设 $G=(V,E)$ 为有限无向加权图，$|V|=N$。顶点权 $\mu(x)>0$，边权 $\omega(x,y)=\omega(y,x)\geq 0$。归一化图拉普拉斯 $\Delta_G$ 作用在函数 $f:V\to\mathbb{R}$ 上：

$$\Delta_G f(x)=\frac{1}{\mu(x)}\sum_{y\sim x} \omega(x,y)(f(x)-f(y)).\tag{8.1}$$

本征值 $0=\lambda_0^{(G)}\leq \lambda_1^{(G)}\leq\dots\leq \lambda_{N-1}^{(G)}\leq 2$。谱间隙 $\lambda_1^{(G)}>0$ 当且仅当 $G$ 连通。

标准 $\epsilon$-网构造：
- **顶点集**：$V_N$ 为 $M$ 上的 $N$ 点 $\epsilon$-网，$\max_x \min_v d_M(x,v)\to 0$；
- **边集**：$x\sim y$ 当 $d_M(x,y)\leq h_N$（$h_N\to 0$）；
- **权重**：$\omega(x,y)=\text{Vol}(B(x,h_N)\cap B(y,h_N))$ 或 Gaussian 权重。

一致性要求：$h_N\to 0$ 且 $N h_N^d\to\infty$。

### 8.2 谱收敛定理

**定理 8.1（图拉普拉斯到 $\Delta_{LB}$ 的谱收敛）。** 设 $(M,g)$ 为紧致连通 $d$ 维 Riemann 流形，$\{G_N\}$ 为按 $\epsilon$-网构造的正则图逼近序列，满足：

**(H1) 稠密性**：$\max_{x\in M} \min_{v\in V_N} d_M(x,v) \to 0$（$N\to\infty$）；

**(H2) 局部一致性**：$\exists \alpha>0$ 使得 $\forall v\in V_N$，$\alpha h_N^d \leq \mu(v) \leq \alpha^{-1} h_N^d$；

**(H3) 连通性**：$G_N$ 的 Cheeger 常数 $h_{G_N} \geq c>0$（与 $N$ 无关）。

则图拉普拉斯 $\Delta_{G_N}$ 的前 $k$ 个非零本征值在 $N\to\infty$ 时收敛到 $\Delta_{LB}$ 的对应本征值：

$$\lim_{N\to\infty} \lambda_j^{(G_N)} = \lambda_j^{(LB)},\quad j=1,2,\dots,k,\tag{8.2}$$

且对应的本征函数（通过插值延拓到 $M$）在 $L^2(M)$ 意义下收敛。

*证明.* 基于 García Trillos–Slepčev 的 $\Gamma$-收敛方法。定义延拓算子 $E_N$ 为 Voronoi 分片常数插值，限制算子 $R_N$ 为点值限制。由 (H1)–(H3)，离散 Dirichlet 形式到连续能量泛函满足 Mosco 收敛，由 Courant–Fischer–Weyl 极小极大原理保证本征值收敛。 $\square$

**详细展开：** $\Gamma$-收敛方法的核心步骤。

**步骤 1：能量泛函的离散-连续对应。** 定义连续 Dirichlet 能量 $E_M(u) = \int_M |\nabla u|^2 d\text{vol}$ 与离散 Dirichlet 能量 $E_{G_N}(f) = \sum_{x\sim y} \omega(x,y)(f(x)-f(y))^2$。延拓算子 $E_N$ 将离散函数 $f:V_N\to\mathbb{R}$ 延拓为 $M$ 上的分片常数函数 $E_N f$。

**步骤 2：Mosco 收敛。** 称 $E_{G_N}$ Mosco 收敛到 $E_M$，若：
(a) 对任意 $u\in L^2(M)$ 及任意满足 $f_N\to u$ 的序列 $f_N\in L^2(V_N)$，有 $E_M(u) \leq \liminf E_{G_N}(f_N)$；
(b) 对任意 $u\in L^2(M)$，存在 $f_N\to u$ 使得 $E_M(u) = \lim E_{G_N}(f_N)$。

由 (H1)–(H3)，可验证 $E_{G_N}$ 满足 Mosco 收敛到 $E_M$。

**步骤 3：本征值收敛。** 由 Courant-Fischer-Weyl 原理，第 $j$ 个本征值
$$\lambda_j = \min_{\dim V=j} \max_{u\in V} \frac{E(u)}{\|u\|^2}.$$
Mosco 收敛保证离散 Rayleigh 商收敛到连续 Rayleigh 商，从而本征值收敛。

### 8.3 热核一致收敛与 Gaussian 估计

**定理 8.2（热核一致收敛）。** 在定理 8.1 的假设 (H1)–(H3) 下，图热核 $K_{G_N}(t,x,y)$（通过插值延拓到 $M\times M$）在 $N\to\infty$ 时满足：

$$\lim_{N\to\infty} \sup_{x,y\in M, t\in[t_0,T]} |K_{G_N}(t,x,y)-K_{LB}(t,x,y)| = 0,\tag{8.3}$$

对任意 $0<t_0<T<\infty$。且存在与 $N$ 无关的常数 $C, c>0$ 使得：

$$K_{G_N}(t,x,y) \leq C t^{-d/2} \exp\left(-c\frac{d_{G_N}(x,y)^2}{t}\right),\tag{8.4}$$

其中 $d_{G_N}$ 为图距离（插值到 $M$ 后逼近测地距离）。

*证明.* 由定理 8.1 的谱收敛与 Delmotte 的离散热核 Gaussian 估计。高频项由谱间隙指数压制，前 $k$ 项由谱收敛保证一致逼近。 $\square$

### 8.4 Weyl 定律与热核展开

**定理 8.3（Weyl 定律与 Minakshisundaram–Pleijel 展开）。** 设 $(M,g)$ 为紧致连通 $d$ 维 Riemann 流形。则：

1. **Weyl 定律**：
   $$N(\lambda)=\frac{C_d}{(2\pi)^d}\text{Vol}(M)\lambda^{d/2}+O(\lambda^{(d-1)/2}),\quad \lambda\to\infty.\tag{8.5}$$

2. **热核迹展开**：
   $$\text{Tr}(e^{-t\Delta})=(4\pi t)^{-d/2}\sum_{n\geq 0} a_{2n} t^n,\quad t\to 0^+.\tag{8.6}$$

   其中 $a_0=\text{Vol}(M)$，$a_2=\frac{1}{6}\int_M R$，$a_4=\frac{1}{360}\int_M (5R^2-2R_{ij}R^{ij}+2R_{ijkl}R^{ijkl})$。

3. **图逼近一致性**：对定理 8.1 中的图序列 $\{G_N\}$，
   $$\lim_{N\to\infty} \text{Tr}(e^{-t\Delta_{G_N}}) = \text{Tr}(e^{-t\Delta_{LB}}),\quad \forall t>0.\tag{8.7}$$

*注.* 本文不声称对 Weyl 定律或热核展开提供新证明。这些结果是谱几何中的经典定理。本文将其明确引入几何论工具层，作为量纲桥重建的数学起点。

### 8.5 谱间隙锁定热核衰减

**定理 8.4（谱间隙锁定热核衰减）。** 设 $(M,g)$ 为连通紧致 Riemann 流形，$\lambda_1>0$ 为 $\Delta_{LB}$ 的第一非零本征值（谱间隙）。则热方程解 $u(t,x)=(e^{-t\Delta}f)(x)$ 满足：

$$\|u(t,\cdot)-\bar{f}\|_{L^2(M)} \leq e^{-\lambda_1 t}\|f-\bar{f}\|_{L^2(M)},\tag{8.8}$$

其中 $\bar{f}=\frac{1}{\text{Vol}(M)}\int_M f$ 为空间平均。衰减率 $\Gamma_{\text{spec}}=\lambda_1$ 是热核衰减的严格谱来源。对定理 8.1 中的图逼近序列，$\lim_{N\to\infty}\lambda_1^{(G_N)} = \lambda_1^{(LB)}$，故图热核衰减率收敛到流形衰减率。

*证明.* 将 $f$ 按本征函数展开，$L^2$ 衰减由第一非零本征值主导。最优性由对应本征函数 $\phi_1$ 达到。 $\square$

**详细展开：** 设 $\{\phi_j\}$ 为 $\Delta_{LB}$ 的完备正交本征函数系，对应本征值 $0=\lambda_0<\lambda_1\leq\lambda_2\leq\dots$。则 $f=\sum_{j=0}^\infty c_j \phi_j$，其中 $c_j=\langle f, \phi_j\rangle$。热方程解为
$$u(t,x) = \sum_{j=0}^\infty c_j e^{-\lambda_j t} \phi_j(x).$$
由于 $\lambda_0=0$, $\phi_0=\text{const}$，$c_0=\bar{f}$。因此
$$u(t,x)-\bar{f} = \sum_{j=1}^\infty c_j e^{-\lambda_j t} \phi_j(x).$$
由 Parseval 等式，
$$\|u(t,\cdot)-\bar{f}\|_{L^2}^2 = \sum_{j=1}^\infty |c_j|^2 e^{-2\lambda_j t} \leq e^{-2\lambda_1 t} \sum_{j=1}^\infty |c_j|^2 = e^{-2\lambda_1 t} \|f-\bar{f}\|_{L^2}^2.$$
取平方根即得 (8.8)。

### 8.6 全息屏谱收敛

**定理 8.5（全息屏图拉普拉斯的谱收敛）。** 设 $\Sigma$ 为公理 3 与附加假设 4.1 中的二维全息屏（同胚于 $S^2$，配备常曲率度量），$\{G_N\}$ 为 $\Sigma$ 上的 $\epsilon$-网正则图序列，满足 (H1)–(H3)。则图拉普拉斯 $\Delta_{G_N}$ 在 $N\to\infty$ 时谱收敛到 $\Sigma$ 上的 Laplace–Beltrami 算子 $\Delta_\Sigma$：

且图热核一致收敛到流形热核。

*证明.* 直接应用定理 8.1 与 8.2 于 $M=\Sigma\cong S^2$，$d=2$。 $\square$

### 9.1 九素定义

**定义 9.1（九素）** 本文所称"九素"（The Nine Primes）指构成十方几何空间互扼逻辑网络的九个基本要素：

## 第 10 章 超定锁定与自举封闭

### 10.1 超定方程组

九素互扼网络（定理 9.1）的六个环节 (A)–(F') 并非独立的逻辑注释，而是构成一个**超定方程组**（overdetermined system）。该方程组有 6 个方程，仅 2 个未知数 $(\Lambda, k_0)$，其自洽性唯一锁定互锁常数。

**方程 1（谱分离约束）**
$$\lambda_1^\Delta = \frac{3}{a^2} < \lambda_2^\Delta = \frac{8}{a^2} < \lambda_3^\Delta = \frac{3\Lambda(S_3)}{a^2}$$
要求 $\Lambda(S_3) > 8/3$。

**方程 2（第四特征值竞争）**
$$\lambda_4^\Delta = \frac{1}{a^2}\min\{15,\ 3\Lambda k_0,\ 3(\Lambda+1)\}$$
要求第四特征值严格大于第三特征值，且前三特征值无简并：
$$3\Lambda < \min\{15,\ 3\Lambda k_0,\ 3(\Lambda+1)\}$$

**方程 3（热核系数比）**
$$\frac{a_2}{a_0} = \frac{1+\Lambda+\Lambda k_0}{a^2}$$
该比值决定谱作用量中曲率项与体积项的权重。

**方程 4（信息层级整数性）**
$$N_n = S_e^2 \cdot \Lambda(S_3)^{2n}, \quad n = 0,1,2,\dots,7$$
要求第 $n$ 层信息单元数 $N_n$ 为整数（信息编码的离散性），且 $\Lambda(S_3)^{2n}$ 在 $n \leq 7$ 时保持整数增长。

**方程 5（幻数截断）**
$$N_{\text{eff}} = 7 \Rightarrow \text{能级数 } k+1 \leq 7$$
要求幻数序列在 $k=6$（第 7 项）后自然截断。严格几何幻数 $(k+1)^2$ 在 $k=6$ 处取值为 49；条件性物理幻数 $2(k+1)^2$ 在 $k=6$ 处取值为 98。截断机制与 $\Lambda(S_3)$ 的层级增长兼容。

**方程 6（桥接公理基态锁定与 90° 编码整数分解）**
$$S_{\min} = 24 = 12\left(\frac{\ell_0^2}{\ell_0^2} + \frac{\ell_0^2}{\ell_0^2}\right)$$
在 $a = \ell_0$ 处，全息屏六项作用量的最小值与桥接公理的双曲型标准形在 $t=1$ 处的取值精确匹配。同时要求 $90^\circ$ 能被三扇区整除（由 $\Lambda(S_3)=3$ 锁定）：
$$\theta_1 + \theta_2 + \theta_3 = 90^\circ = \Lambda(S_3) \times 30^\circ = 3 \times 30^\circ$$

### 10.2 互锁常数的唯一性定理

**定理 10.1（互锁常数的唯一性）** 上述超定方程组在正实数域内有唯一解：
$$\Lambda = \Lambda(S_3) = 3, \quad k_0 = k_0(S_3) = 2$$

*证明.* 分四步：

**步骤 1：$\Lambda$ 的群论锁定。** 由定理 3.0'（三分切丛扇区置换的刚性），若要求三个扇区在度规层面完全等价（任意置换诱导等距），则扇区置换群必包含所有对换，从而唯一同构于 $S_3$。由方程 6 的 90° 编码整数分解，三扇区各贡献 $30^\circ$ 要求 $90^\circ$ 被 $\Lambda(S_3)$ 整除。由命题 3.0，$S_3$ 的共轭类数 $\Lambda(S_3)=3$ 直接给出：
$$\Lambda = \Lambda(S_3) = 3$$
该值满足方程 1 的谱分离约束 $3 > 8/3$。

**步骤 2：$k_0$ 的群论锁定。** 由定理 3.0'，$S_3$ 是唯一的群选择。由命题 3.0，$S_3$ 的极大正规子群 $A_3$ 唯一，指数 $[S_3:A_3]=2$，故 $k_0=k_0(S_3)=2$ 是群论函数的必然输出。

**步骤 3：超定方程组的自举验证。** 将 $(\Lambda, k_0)=(3,2)$ 代入方程 1–6，逐一验证自洽性：
- **方程 1**（谱分离）：$3/a^2 < 8/a^2 < 9/a^2$，满足 $\Lambda=3>8/3$。
- **方程 2**（第四特征值竞争）：$9 < \min\{15, 18, 12\}=12$，排序严格。
- **方程 3**（热核系数比）：$a_2/a_0 = (1+3+6)/a^2 = 10/a^2$，与谱作用量曲率项权重兼容。
- **方程 4**（信息层级整数性）：$N_n = S_e^2 \cdot 9^n$，对所有 $n\leq 7$ 为整数。
- **方程 5**（幻数截断）：$N_{\text{eff}}=7$ 由 Bott 周期（定理 6.3）严格锁定，与 $k\leq 6$ 兼容。
- **方程 6**（桥接公理基态）：$S_{\min}=24=12(1+1)$，在 $a=\ell_0$ 处精确匹配。

**步骤 4：唯一性论证。** 由定理 3.0'，$S_3$ 是三分切丛置换对称群的唯一选择，因此 $\Lambda(S_3)=3$ 与 $k_0(S_3)=2$ 已被群论唯一确定。超定方程组的作用是**验证**该值与几何层、谱层、量子化层、信息层的所有约束完全自洽，且任何偏离 $(3,2)$ 的数值都会导致至少一个方程失效。具体地，若 $\Lambda\neq 3$，则方程 6 的 $90^\circ$ 三等分被破坏；若 $k_0\neq 2$，则方程 3 的曲率项权重与方程 2 的第四特征值竞争将失配。因此，6 个方程同时满足，且解 $(\Lambda, k_0) = (\Lambda(S_3), k_0(S_3)) = (3, 2)$ 唯一。$\square$

**[260628.8 新增：定理 10.1 的逻辑地位]**

**注 10.1（超定方程组的逻辑地位）** 定理 3.0' 与命题 3.0 已从群论严格计算 $(\Lambda,k_0)=(3,2)$。超定方程组（§10.1）并非用于"独立求解"该值（群论计算本身是初等的），而是用于证明：从群论涌现的 $(3,2)$ 与几何层、谱层、量子化层、信息层的所有约束完全自洽，且任何偏离 $(3,2)$ 的数值都会导致至少一个方程失效。因此，超定方程组的作用是**自举封闭**（bootstrap closure），而非独立求解或事后验证。**定理 10.1 的逻辑地位是自洽性验证，而非唯一性定理。**

## 附录 B 桥接公理的数学性质

### B.1 双曲型标准形与尺度倒数对偶

令 $x = a^2/\ell_0^2 = e^{2u}$（$u=\ln(a/\ell_0)$），桥接公理等价于：

$$S = 24\cosh\bigl(2\ln(a/\ell_0)\bigr).$$

该标准形揭示三项核心性质：
1. **偶对称性**：$S$ 在 $u$ 下为偶函数，天然产生 $a \leftrightarrow \ell_0^2/a$ 的尺度倒数对偶；
2. **凸性与唯一最小值**：$\cosh$ 的凸性保证 $S_{\min}=24$ 在 $a=\ell_0$ 处唯一取得；
3. **指数边界**：$S \sim 12e^{2|u|}$（$|u|\to\infty$），保证退化极限 $S\to+\infty$。
4. **互锁常数锁定**：$\Lambda=\Lambda(S_3)=3$ 的三等分与 $k_0=k_0(S_3)=2$ 的二分紧致性，由定理 3.0' 与命题 3.0 从 $S_3$ 群论结构直接计算得出，并由九素互扼网络从谱分离、热核系数和幻数截断的联合约束唯一确定。

### B.2 模空间的 $2:1$ 覆盖结构

桥接公理定义映射 $\pi: \mathcal{M}\cong(0,+\infty)_a \to [24,+\infty)$。该映射为 $2:1$ 覆盖：

| 分支 | 定义域 | 根公式 | 几何语义 |
|------|--------|--------|----------|
| UV | $a\in(0,\ell_0]$ | $x_- = \dfrac{S-\sqrt{S^2-576}}{24}$ | 微观奇点极限 |
| IR | $a\in[\ell_0,+\infty)$ | $x_+ = \dfrac{S+\sqrt{S^2-576}}{24}$ | 宏观膨胀极限 |
| 基态 | $a=\ell_0$ | $x=1$（重根） | 分支点（ramification point）|

**韦达定理不变量**：$x_+ x_- = 1 \Rightarrow a_{\text{UV}}\cdot a_{\text{IR}} = \ell_0^2$。

### B.3 与全息屏六项作用量的几何锁定

由 AM-GM 不等式：

$$x + \frac{1}{x} \geq 2 \Rightarrow S_{\min} = 24.$$

该下界与第 4 章六项作用量在对称点 $30^\circ$ 处的值 $S=6\times4=24$ **精确锁定**。系数 $12$ 对应全息屏的 3 个单扇区项之和（或 3 个双扇区项之和），同时 $12 = 3 \times 4$ 中因子 $3$ 对应 $\Lambda=3$ 的三等分锁定。桥接公理在代数层面"折叠"了全息屏的六项刚度为尺度倒数对偶的两项，而 $\Lambda=3$ 的三等分是这一折叠的几何来源。

### B.4 谱传递与特征签名不变性

由命题 3.2，$\lambda_1^\Delta = 3/a^2$。代入桥接公理解得：

$$\lambda_1^\Delta = \frac{S \pm \sqrt{S^2-576}}{8\ell_0^2},$$

其中 UV 分支（$a\le\ell_0$）取 $+$ 号（大特征值），IR 分支（$a\ge\ell_0$）取 $-$ 号（小特征值）。

**关键性质**：桥接公理仅控制谱的**整体缩放**，不改变**内部比例**：

$$\frac{\lambda_2^\Delta}{\lambda_1^\Delta} = \frac{8}{3}, \quad \frac{\lambda_3^\Delta}{\lambda_1^\Delta} = \Lambda(S_3) = 3 \quad (\text{对任意 } S\geq24 \text{ 不变}).$$

这正是定理 3.4（谱刚性）成立的基础——模空间一维性保证比例被群论函数 $\Lambda(S_3)=3$、$k_0(S_3)=2$ 锁定。

### B.5 正则化作用量与第 2 章值域对应

桥接公理的正则化形式：

$$S_{\text{abstract}} = S - 24 = 12\left(\frac{a}{\ell_0} - \frac{\ell_0}{a}\right)^2,$$

将全息屏实现域 $[24,+\infty)$ 平移到 $(0,+\infty)$，与第 2 章抽象几何量 $S_{\text{abstract}}:D_\pm\to(0,+\infty)$ 的值域**精确对应**。

### B.6 定理 5.3 的完整证明

以下给出正文定理 5.3 的完整证明。

*证明.*

**步骤 1：** 令 $x=a/\ell_0$，$u=\ln x$。由 (C1)，$S(x)=S(1/x)$，故 $S$ 为 $u=\ln x$ 的偶解析函数，其 Laurent 展开 $S(x)=\sum c_n x^n$ 满足 $c_n=c_{-n}$。

**步骤 2：** 由 (C3)，$\lim_{x\to 0} x^2 S(x)=C<+\infty$。这要求 $c_n=0$ 对所有 $n<-2$。由对偶性 $c_n=c_{-n}$，得 $c_n=0$ 对所有 $n>2$。因此 $S(x)=c_{-2}x^{-2}+c_0+c_2x^2$，$c_{-2}=c_2$。

**步骤 3：** 由 (C2)，$S(1)=24$ 且 $S''(1)>0$。计算得 $S'(x)=2c_2(x-x^{-3})$，$S'(1)=0$ 自动满足（偶函数性质）。$S''(1)=8c_2>0$ 要求 $c_2>0$。代入 $S(1)=2c_2+c_0=24$。

**步骤 4：** 假设存在更高阶对偶项 $c_4(x^4+x^{-4})$（$c_4\neq 0$）。由步骤 2 的渐近约束 $c_n=0$（$|n|>2$），直接排除 $c_4$ 及所有更高阶偶次项的存在。因此标准形唯一确定为 $S(x)=c_2(x^2+x^{-2})+c_0$。

**步骤 5：** 由 (C2) 的 $S(1)=24$ 得 $2c_2+c_0=24$。若进一步要求 $c_0=0$（使 $S$ 在 $x\to\infty$ 时具有纯倒数对称渐近行为 $S\sim c_2 x^2$），则 $c_2=12$。故 $S(a)=12(a^2/\ell_0^2+\ell_0^2/a^2)$。 $\square$

此时 $\text{Jac}(P_i) = 1/4 = \sin^2 30^\circ$。三个 $P_i$ 均映到同一个 $T\Sigma$，且有自然的等同关系（S₃ 对称性）。

#### E.3.2 非对称点：当前状态

在非对称点（$\theta_i$ 不全相等），上述构造面临两个困难：

**(a) 面积比例的几何实现**：对每个扇区 $i$，需要构造线性满射 $P_i: \mathbb{R}^3 \to \mathbb{R}^2$ 使得 $\text{Jac}(P_i) = \sin^2\theta_i$，且 $\ker P_i$ 与 S³ 上的 Hopf 纤维方向重合。

**(b) 三个投影的协调**：$P_1, P_2, P_3$ 的值域必须是**同一个**二维空间 $T\Sigma$。

问题 (a) 在纯线性代数层面可解。问题 (b) 需要额外的规范固定。在对称点，S₃ 对称性自然提供此固定。在非对称点，S₃ 破缺，需要新的几何原则来确定 $k_i$ 和 $O_i$ 的选取。

#### E.3.3 典范 Hopf 投影构造（260629.8 深化成果）

**定义 E.3.3（典范 Hopf 投影构造）**：对每个扇区 $i \in \{1,2,3\}$，定义 $P_i: T_{p_i}S^3(r_i) \to T\Sigma$ 为：

$$P_i(v) := \sin\theta_i \cdot d\pi_i(v^\perp), \quad v^\perp := v - g_{S^3}(v, k_i)k_i,$$

其中：
- $k_i$ 为典范 Hopf 纤维方向（右 U(1) 生成元，典范确定）
- $d\pi_i$ 为典范 Hopf 商投影的微分
- $d\pi_i(v^\perp) \in T_{\pi_i(p_i)}\mathbb{C}P^1$ 通过光滑流形的恒等映射 $\mathbb{C}P^1 \xrightarrow{\cong} \Sigma$ 等同于 $T\Sigma$ 中的向量
- $\sin\theta_i$ 由角度公理 $\theta_1+\theta_2+\theta_3=90^\circ$ 和 $M(a)$ 的参数决定

**性质 E.3.3（自动满足的条件）**：

1. **线性满射**：$P_i$ 是线性映射，$\dim\ker P_i = 1$，$\text{im}P_i = T\Sigma$（2 维）。✓
2. **单扇区面积比例**：$\text{Jac}(P_i|_{H_i}) = \sin^2\theta_i$（因 $d\pi_i|_{H_i}$ 是等距同构）。✓
3. **核-Hopf 相容性**：$\ker P_i = \text{span}\{k_i\} = \mathcal{V}_i|_{p_i}$（典范 Hopf 纤维方向）。✓
4. **零自由参数**：构造中不涉及任何连续选择。所有成分（$k_i$、$H_i$、$d\pi_i$、$\theta_i$）均典范确定。✓

**结论**：非对称点的 $P_i$ 的自然唯一构造存在于典范 Hopf 投影结构中，自由参数为零。双扇区耦合的完全几何解释仍需最后一步验证（见下文 E.3.4）。

#### E.3.4 残存问题：双扇区耦合的精确几何解释

定义 4.2 的双扇区条件：

$$\text{Area}(P_i \times P_j) = \sin\theta_i \sin\theta_j \, dA$$

需要精确的几何解释。在典范构造下，$P_i(u_i) = \sin\theta_i \cdot d\pi_i(u_i)$，$P_j(u_j) = \sin\theta_j \cdot d\pi_j(u_j)$，其中 $u_i \in H_i$, $u_j \in H_j$。利用 S³ 水平分布上的**典范近复结构**，$H_i$ 和 $H_j$ 之间存在**典范的复线性等同**：

$$\Phi_{ij} := (d\pi_j|_{H_j})^{-1} \circ d\pi_i|_{H_i}: H_i \xrightarrow{\cong} H_j.$$

在典范配对 $\Phi_{ij}$ 下，双扇区面积比例的自然解释有待展开显式计算来确认。当前标注为**研究方向**（非定理）。但不影响 $P_i$ 构造本身的自然唯一性。

### E.4 桥3：O(1) 线丛的几何来源

#### E.4.1 核线丛方案的根本困难（C1 方案的失败）

**定理 E.4.1（$\ker(P_i) \otimes \mathbb{C}$ 的陈类）**：设 $P_i: \mathbb{R}^3 \to \mathbb{R}^2$ 为线性满射。则 $\ker(P_i) \otimes \mathbb{C}$ 作为 $S^3(r_i)$ 上的一维复平凡丛的拉回（拉回到 $\Sigma$ 的 Hopf 基），其第一陈类为 $c_1 = 0$。

*论证*：$\ker(P_i)$ 是一维实向量空间。在 $S^3(r_i)$ 的 Hopf 纤维化下，$\ker(P_i)$ 若与 Hopf 垂直方向重合，则它等价于 Hopf 线丛的**实部**。其复化 $\ker(P_i) \otimes \mathbb{C}$ 作为 $S^3$ 上的等变线丛对应权 $0$ 的平凡表示（因为 Hopf 垂直方向在右 U(1) 作用下不变，权为零）。因此 $c_1 = 0$。更直接地，$\ker(P_i) \otimes \mathbb{C} \cong \mathcal{O}(0)$（平凡丛），陈类显然为零。

**结论**：$\ker(P_i) \otimes \mathbb{C}$ 不能直接产生 $\mathcal{O}(1)$。假设 4.1' 在原始形式下**与陈类零点定理冲突**。这不是一个可选的假设——它是一个可被严格否证的陈述。必须寻找 $\mathcal{O}(1)$ 的其他几何来源。

#### E.4.2 C2 方案：S³ 旋量丛显式构造

C2 方案的核心思想：**$\mathcal{O}(1)$ 不来自投影的核，而来自 S³ 因子自身的旋量丛结构。**

**S³ 作为旋量群**：

$$S^3 \cong SU(2) \cong \text{Spin}(3).$$

**Hopf 纤维化**：取右 U(1) 子群 $U(1)_R$，商空间 $SU(2)/U(1)_R \cong \mathbb{C}P^1 \cong S^2$。

**等变线丛的 Borel-Weil 分类**：在 S³ 上的等变复线丛（关于右 U(1) 作用）由权 $k \in \mathbb{Z}$ 分类：

$$L_k := SU(2) \times_{U(1)_R} \mathbb{C}_{(k)},$$

**定理 E.4.2（Borel-Weil）**：$L_k \cong \pi^* \mathcal{O}(k)$，其中 $\mathcal{O}(k)$ 是 $\mathbb{C}P^1$ 上的标准全纯线丛，$c_1(\mathcal{O}(k)) = k$。

**旋量丛的等变分解**：旋量丛 $S$ 是伴随于旋量表示 $\Delta_3$ 的向量丛。取 U(1) 为对角子群，右乘作用在 $\Delta_3 \cong \mathbb{C}^2$ 上分解为：

$$\mathbb{C}^2 = \mathbb{C}_{(+1)} \oplus \mathbb{C}_{(-1)}.$$

**定理 E.4.3（旋量丛的等变分解）**：

$$S = S_+ \oplus S_-,$$

其中 $S_+ \cong L_{+1} \cong \pi^*\mathcal{O}(1)$（权 +1，$c_1=1$），$S_- \cong L_{-1} \cong \pi^*\mathcal{O}(-1)$（权 $-1$，$c_1=-1$）。

**推论 E.4.3（对偶丛）**：

$$S_-^* \cong L_{+1} \cong \pi^*\mathcal{O}(1).$$

因此 $S_-^*$ 是 S³ 上的等变复线丛，满足 $c_1(S_-^*) = 1$（作为底流形 $\mathbb{C}P^1$ 上的线丛），且由 S³ 上的纯旋量几何定义，不依赖任何投影的选取。

**平方根关系**：

$$S_-^* \otimes S_-^* \cong \pi^*\mathcal{O}(1) \otimes \pi^*\mathcal{O}(1) \cong \pi^*\mathcal{O}(2) \cong H^{1,0}.$$

$S_-^*$ 是水平分布 $(1,0)$ 部分的"平方根"——平方根的存在性恰由 Spin(3) 结构保证。这精准体现了"旋量 = 切矢量的平方根"这一古典事实。

**三扇区推广**：每个 S³ 因子携有独立的候选线丛 $\Lambda_i := (S^{(i)}_-)^* \cong \pi_i^*\mathcal{O}(1)$（在底 $\mathbb{C}P^1_i$ 上 $c_1=1$）。

在对称点，三个 S³ 因子的 Hopf 基 $\mathbb{C}P^1_i$ 通过投影 $P_i$ 自然等同于同一个全息屏 $\Sigma \cong \mathbb{C}P^1$。因此得到 $\Sigma$ 上三个独立的 $\mathcal{O}(1)$：

$$\Lambda_{\mathcal{M}}, \Lambda_{\mathcal{C}}, \Lambda_{\mathcal{I}} \cong \mathcal{O}(1) \quad \text{（在 } \Sigma \text{ 上, 对称点严格成立）}.$$

### E.5 C2 与桥 2 的解耦：重要澄清

C2 构造（旋量丛 → $\mathcal{O}(1)$）与桥 2（$P_i$ 的构造）**已在结构上解耦**：

- **C2 不依赖桥 2 在非对称点的细节**：$\mathcal{O}(1)$ 来自 $S^3$ 上的旋量丛 $S_-^*$，这是一个**整体几何对象**，其陈类 $c_1=1$ 由 Borel-Weil 定理在任何半径的 S³ 上严格成立。它不涉及投影 $P_i$ 的选取。

**结论**：即使桥 2 在双扇区耦合的完全几何解释上仍有最后一小步（E.3.4），C2 构造已在全局（对称点 + 非对称点）严格成立。幻数序列 $2(k+1)^2$ 的推导不依赖桥 2 的完全闭合。

### E.7 C2 构造的核心意义

C1 方案（$\ker(P_i) \otimes \mathbb{C} \cong \mathcal{O}(1)$）试图从投影几何直接长出 $\mathcal{O}(1)$，但 $\ker(P_i) \otimes \mathbb{C}$ 的陈类严格为零——这是可被否证的。C2 方案不与之冲突，因为 $\mathcal{O}(1)$ 的来源被重新定位：它来自 S³ 因子自身的旋量丛结构，独立于投影构造。

两种方案给出的是切丛复化的**互补分解**：

$$TS^3 \otimes \mathbb{C} \cong \underbrace{H^{1,0}}_{\pi^*\mathcal{O}(2)} \oplus \underbrace{H^{0,1}}_{\pi^*\mathcal{O}(-2)} \oplus \underbrace{V \otimes \mathbb{C}}_{\mathcal{O}(0)},$$

$$S_-^* \otimes S_-^* \cong H^{1,0} \quad \text{（平方根关系）}.$$

- $\ker(P_i) \otimes \mathbb{C}$ 对应垂直方向 $V \otimes \mathbb{C} \cong \mathcal{O}(0)$（平凡，$c_1=0$）。
- $(S^{(i)}_-)^*$ 对应水平 $(1,0)$ 方向的平方根（$\mathcal{O}(1)$，$c_1=1$）。

**一切各归其位，互不矛盾。**
