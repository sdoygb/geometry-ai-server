## 一、设计原则

1. **一条主线**：公理 → 数学基础 → 物理映射 → 约束截面 → 量纲桥 → 动力学 → 应用
2. **三条支线**：粒子物理 / 宇宙学 / 其他（水平发散，末端收敛至验证节点）
3. **一个闭环**：应用输出反馈至公理验证
4. **每条箭头标注逻辑状态**：[T]定理 / [P]命题 / [R]研究方向 / [H]假设

## 四、读图指南

### 纵向主线
从上往下读：**公理→数学→映射→约束→量纲桥→动力学→应用→闭环验证**。这条路径覆盖从几何论最底层到实验预测的完整逻辑链条。

### 横向支线
- **粒子物理域**：量纲桥 → 0.3.5相互作用框架 → 各粒子应用篇 → 19参数消解
- **宇宙学域**：约束截面 → 0.2.2全息宇宙 → 宇宙学应用篇 → Λ_res与CMB异常
- **其他域**：动力学核心 → 凝聚态/化学/生物应用篇

### 闭环逻辑
应用篇输出（S_e锁定、九素互扼自洽、谱刚性验证）通过验证节点指向公理确认——这不是推导循环，而是**元逻辑闭合**：公理保证推导一致性，应用输出反过来确认公理的有效性。
