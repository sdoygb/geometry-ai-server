## 第1章　引言：从点动力学到密度动力学

### 1.1　问题的提出

0.5号文章在约束截面Σ上成功建立了因果场的点动力学：

- **瞬态推进T_trans**：沿η方向向基态的单调映射，有限步N_cause≈10内收敛到对称基态θ^*=(30°,30°,30°)；
- **稳态环流T_ss**：在中心子流形E^c={ξ=0}上以无理旋转形式保测遍历。

然而，0.5的全部分析是在**几何点**的层面进行的——T_trans将一个点(ξ,η)推到另一个点，T_ss让点在E^c上旋转。这对"因果场是什么、它如何运动"给出了完美的几何描述。

但**计算**涉及的是**分布**。编码——无论是0.4的图热扩散编码还是53号的缺陷编码——将信息写入信息场时，产生的是Σ上的初始密度分布ρ(x,0)，而非单个几何点。演化是密度在时间中扩散、弛豫、读出的过程。如果C-场要参与计算——如三界联合计算的要求——它的"态"也必须是分布ρ_C(ξ,η,t)，而非单个点。

这就是本文的核心动机：**将0.5的点动力学提升为密度动力学。** 该提升在遍历论中是标准的Perron-Frobenius算子构造，但0.5未定义该算子，更未分析其谱、单调性和收敛速率。

### 1.2　提升的逻辑结构

给定点动力学T: Σ→Σ（可以是T_trans或T_ss），密度演化由**Perron-Frobenius算子**L_T给出：

$$(L_T \rho)(y) = \sum_{x \in T^{-1}(y)} \frac{\rho(x)}{|\det DT(x)|}$$

其中DT为T的Jacobian矩阵。L_T将初始密度ρ_0推前：ρ_n = L_T^n ρ_0（离散形式），或在连续极限下导出Liouville型方程∂_t ρ = ℒ ρ。

0.5已有：
- T_trans和T_ss的显式形式（0.5-§2.2、§2.3）
- DT的显式形式（0.5-T4.2：Jacobian = diag(e^{Λ_ξ}, 1)或diag(1,1)）
- 体积元dμ ≈ √(λ₁λ₂) dξ dη（0.5-§2.1）

因此Perron-Frobenius算子的构造是直接的，不依赖任何新假设。

### 1.3　本文与0.5的关系

本文不修改0.5的任何结论。本文为0.5的点动力学添加**密度层**，使其可与0.4的信息场密度动力学（图热扩散方程）在统一层面桥接。所有新定理在0.5已建立的结构上逐条封闭推导。

## 第2章　C-场密度态空间的严格定义

### §2.1　定义

**定义2.1**（C-场密度态空间）。在约束截面Σ上，给定Hessian度量g_{ij} ≈ diag(λ₂, λ₁)（0.5-§2.1），体积元dμ = √(det g) dξ dη ≈ √(λ₁λ₂) dξ dη。**C-场密度态空间**定义为：

$$\boxed{\mathcal{X}_C := \{\rho \in L^1(\Sigma, \mathrm{d}\mu) : \rho \geq 0\ \text{a.e.},\ \int_\Sigma \rho\,\mathrm{d}\mu = 1\}}$$

物理含义：ρ(ξ,η,t)是t时刻在(ξ,η)处找到因果场"活跃度"的概率密度。归一化条件∫_Σ ρ dμ = 1表示总因果量守恒。

### §2.2　正则性

**命题2.1**（𝒳_C的完备性与紧性）。𝒳_C在L¹范数下是完备的凸集。在L¹弱拓扑下，𝒳_C是序列紧的。

**证明**：L¹(Σ, dμ)是Banach空间（Σ紧致，μ有限）。𝒳_C是该Banach空间中单位球的闭凸子集。由Dunford-Pettis定理，L¹中有界集的弱序列紧性成立。证毕。

**注2.1**：命题2.1的序列紧性保证了后续稳态定理中极限对象的存在性——任何演化序列{ρ_n}在𝒳_C中有弱收敛子列。

### §2.3　与0.5点动力学的对应

在点动力学中，"态"是Σ上的一个点x=(ξ,η)。在密度动力学中，Dirac测度δ_x(ξ',η') = δ(ξ'-ξ)δ(η'-η)/√(λ₁λ₂) ∈ 𝒳_C是点态的密度表示。任意密度ρ可视为系综平均：ρ = ∫ δ_x dν(x)，其中ν为Σ上的Borel概率测度。

## 第3章　Perron-Frobenius算子与Liouville提升

### §3.1　Perron-Frobenius算子的良定义

**定义3.1**（Perron-Frobenius算子）。对可测变换T: Σ→Σ，其**Perron-Frobenius算子**L_T: 𝒳_C → 𝒳_C定义为：

$$\boxed{(L_T \rho)(y) := \sum_{x \in T^{-1}(y)} \frac{\rho(x)}{|\det DT(x)|}}$$

其中T^{-1}(y) = {x ∈ Σ : T(x) = y}为y的逆像集。

**定理3.1**（L_T的Markov性质）。对0.5中定义的T ∈ {T_trans, T_ss}，L_T是𝒳_C上的Markov算子，即满足：

**(i) 保正性**：ρ ≥ 0 ⇒ L_T ρ ≥ 0；
**(ii) 保范性**：∫_Σ (L_T ρ) dμ = ∫_Σ ρ dμ = 1；
**(iii) L¹-收缩性**：‖L_T ρ₁ − L_T ρ₂‖_{L¹} ≤ ‖ρ₁ − ρ₂‖_{L¹}。

**证明**：

**(i)** 由定义，L_T ρ是非负函数的加权和，权重1/|det DT| > 0（T_trans的det DT = e^{Λ_ξ} > 0，T_ss的det DT = 1 > 0），故保正性平凡成立。

**(ii)** 保范性由积分变量替换：

$$\int_\Sigma (L_T \rho)(y)\,\mathrm{d}\mu(y) = \int_\Sigma \sum_{x \in T^{-1}(y)} \frac{\rho(x)}{|\det DT(x)|}\,\mathrm{d}\mu(y)$$

对T_trans（单射），逆像集为单点{T^{-1}(y)}，变量替换y=T(x)给出dμ(y) = |det DT(x)| dμ(x)，故积分等于∫ ρ(x) dμ(x) = 1。

对T_ss（在E^c上为平移），T_ss在E^c上的限制为刚性平移（det DT_ss = 1，0.5-P2.3），且Σ \ E^c在T_ss下测度为零（ξ=0为中心子流形，0.5-T4.2的硬方向冻结保证ξ≠0的区域在T_trans阶段已被压缩为Lebesgue零测集）。因此保范性同样成立。

**(iii)** L¹-收缩性由保正保范及线性性推出，是Markov算子的标准性质。证毕。

### §3.2　对T_trans的Liouville提升

**定义3.2**（瞬态密度推进）。对初始密度ρ_0 ∈ 𝒳_C，瞬态密度推进为：

$$\rho_{n+1} = L_{T_{\text{trans}}} \rho_n,\quad n = 0,1,\dots,N_{\text{cause}}-1$$

其中N_cause=10（0.5-T5.1）。

**显式形式**：T_trans(ξ,η) = (e^{Λ_ξ}ξ, η+δη)，其逆为T_trans^{-1}(ξ',η') = (e^{-Λ_ξ}ξ', η'-δη)，det DT_trans = e^{Λ_ξ}（0.5-T4.2步骤5，e^{Λ_ξ} = √(λ₁/λ₂)）。由于T_trans是双射（Σ → T_trans(Σ)），Perron-Frobenius算子简化为：

$$\boxed{(L_{T_{\text{trans}}} \rho)(\xi, \eta) = e^{-\Lambda_\xi} \cdot \rho(e^{-\Lambda_\xi}\xi,\ \eta - \delta\eta)}$$

其中e^{-Λ_ξ} = √(λ₂/λ₁) ≈ 12.25（由0.5-T4.2，Λ_ξ = −½log(150) ≈ −2.51）。

**命题3.1**（L_T_trans的因子分解）。L_T_trans可分解为两个交换算子的复合：

$$L_{T_{\text{trans}}} = L_\eta \circ L_\xi$$

其中(L_η ρ)(ξ,η) = ρ(ξ, η-δη)为η方向刚性平移，(L_ξ ρ)(ξ,η) = e^{-Λ_ξ} ρ(e^{-Λ_ξ}ξ, η)为ξ方向缩放。

**证明**：直接验证。L_T_trans ρ(ξ,η) = e^{-Λ_ξ} ρ(e^{-Λ_ξ}ξ, η-δη) = L_ξ(L_η ρ)(ξ,η) = L_η(L_ξ ρ)(ξ,η)。交换性成立因为L_ξ仅作用于ξ变量，L_η仅作用于η变量。证毕。

**物理含义**：每步瞬态密度推进可理解为"先沿η平移→再沿ξ收缩"。ξ方向的收缩因子e^{-Λ_ξ}≈12.25将密度沿ξ方向挤向中心子流形E^c={ξ=0}，这对应0.5-T4.2的硬方向冻结效应在密度层面的体现。

### §3.3　对T_ss的Koopman-Perron-Frobenius对偶

**定义3.3**（稳态密度环流）。在中心子流形E^c={ξ=0}上，密度已集中于ξ=0邻域（经N_cause步瞬态推进后，见§5.2收敛速率）。稳态密度环流为：

$$\rho_{n+1}^{E^c} = L_{T_{\text{ss}}} \rho_n^{E^c},\quad n \geq N_{\text{cause}}$$

其中T_ss(0,η) = (0, η+δη mod Δη_ss)（0.5-§2.3），det DT_ss = 1。

**显式形式**：由于T_ss在E^c上为单参数平移群，且det=1：

$$\boxed{(L_{T_{\text{ss}}} \rho^{E^c})(0, \eta) = \rho^{E^c}(0, \eta - \delta\eta)}$$

即L_T_ss在E^c上是标准的平移Perron-Frobenius算子。

**注3.1**（Koopman对偶）。在L²(E^c, dμ_c)上，Perron-Frobenius算子L_T_ss与Koopman算子U_T_ss（0.5-§7.2，U_T f = f∘T）满足对偶关系：

$$\langle L_T \rho, f \rangle = \langle \rho, U_T f \rangle$$

此对偶关系是遍历论的标准结果，在本文中用于将0.5的Koopman谱分析（0.5-§7.2、§8）翻译为密度演化的谱性质。

## 第4章　C-场熵泛函与H-定理

### §4.1　KL散度定义与正则性

**定义4.1**（KL散度）。对ρ, σ ∈ 𝒳_C，定义Kullback-Leibler散度：

$$\boxed{D_{\text{KL}}(\rho \,\|\, \sigma) := \int_\Sigma \rho(\xi,\eta) \log\frac{\rho(\xi,\eta)}{\sigma(\xi,\eta)}\,\mathrm{d}\mu(\xi,\eta)}$$

约定0·log(0/σ)=0，且若ρ在σ零测集上非零则D_KL=+∞。

**命题4.1**（KL散度的正则性）。对ρ, σ ∈ 𝒳_C ∩ L^∞(Σ)，D_KL(ρ‖σ) ∈ [0, +∞]。D_KL(ρ‖σ)=0当且仅当ρ=σ a.e.。

**证明**：由Gibbs不等式，s log(s/t) ≥ s−t对所有s,t>0成立，等号仅当s=t。积分得D_KL ≥ ∫(ρ−σ)dμ = 1−1 = 0。证毕。

### §4.2　瞬态推进的严格H-定理

**定理4.1**（瞬态密度推进的H-定理）。设ρ_0 ∈ 𝒳_C为任意初始密度，ρ_n = L_T_trans^n ρ_0为瞬态密度推进。令ρ_C^*为稳态密度（定义见§5.1）。则KL散度严格单调递减：

$$\boxed{D_{\text{KL}}(\rho_{n+1} \,\|\, \rho_C^*) \leq D_{\text{KL}}(\rho_n \,\|\, \rho_C^*)}$$

且等号成立当且仅当ρ_n在T_trans作用下已达稳态（即ρ_n = ρ_C^*）。

**证明**：

**步骤1（分解）**：由命题3.1，L_T_trans = L_η ∘ L_ξ。逐项分析KL散度的变化。

**步骤2（L_ξ收缩的KL单调性）**：L_ξ(ρ)(ξ,η) = e^{-Λ_ξ} ρ(e^{-Λ_ξ}ξ, η)。设σ(ξ,η) = ρ_C^*(ξ,η)。由于ρ_C^*在ξ方向集中在{ξ=0}（§5.1将证明），对任意ρ：

$$D_{\text{KL}}(L_\xi \rho \,\|\, \sigma) = \int e^{-\Lambda_\xi} \rho(e^{-\Lambda_\xi}\xi,\eta) \log\frac{e^{-\Lambda_\xi} \rho(e^{-\Lambda_\xi}\xi,\eta)}{\sigma(\xi,\eta)}\,\mathrm{d}\mu$$

做变量替换ξ'=e^{-Λ_ξ}ξ，dξ=e^{Λ_ξ}dξ'，dμ=√(λ₁λ₂)dξdη→√(λ₁λ₂)·e^{Λ_ξ}dξ'dη = e^{Λ_ξ}dμ'。

$$= \int \rho(\xi',\eta) \log\frac{e^{-\Lambda_\xi} \rho(\xi',\eta)}{\sigma(e^{\Lambda_\xi}\xi',\eta)}\,\mathrm{d}\mu'$$

$$= D_{\text{KL}}(\rho \,\|\, \sigma \circ T_{\text{trans}}^{-1}) - \Lambda_\xi$$

由于−Λ_ξ = ½log(λ₂/λ₁) > 0（0.5-T4.2），此项为KL散度添加正常数偏移。但该偏移被L_η平移步骤中σ的性质所补偿——完整论证见步骤3。

**步骤3（联合单调性——需要σ的特殊形式）**：σ = ρ_C^*为T_ss的稳态密度。由§5.1（待证），ρ_C^*在E^c={ξ=0}上为均匀分布，在ξ≠0处为零（或近似为零）。因此对任意ρ：

$$\sigma(e^{\Lambda_\xi}\xi', \eta) = \sigma(\xi', \eta) \cdot \mathbf{1}_{\{\xi'=0\}} + \text{高阶小量}$$

这意味着在ξ'=0的支撑集上，σ(e^{Λ_ξ}ξ',η)=σ(ξ',η)。而在ξ'≠0处，ρ的分布被L_ξ的收缩持续压向ξ=0（命题3.1的物理含义）。严格的逐点不等式：

$$D_{\text{KL}}(L_{T_{\text{trans}}}\rho \,\|\, \sigma) - D_{\text{KL}}(\rho \,\|\, \sigma) = -\frac{1}{2}\log\frac{\lambda_2}{\lambda_1} + \int \rho \log\frac{\sigma(T_{\text{trans}}^{-1}(x))}{\sigma(x)}\,\mathrm{d}\mu(x)$$

由T_trans向基态的单调性（0.5-T2.1），σ(T_trans^{-1}(x)) ≥ σ(x)（稳态密度在基态方向浓度递增），故对数项非正，补偿−½log(λ₂/λ₁)项并产生净单调递减。

**步骤4（等号条件）**：等号成立仅当ρ_n = ρ_C^* a.e.，即密度已完全集中在E^c上且为均匀分布。证毕。

> **诚实标注**：步骤3中"σ(T_trans^{-1}(x)) ≥ σ(x)"的严格证明依赖于§5.1中稳态密度的显式确定。在§5.1完成之前，该不等式为合理物理推断。下文§5.1完成后，本节证明获得完全封闭。

### §4.3　稳态环流的Shannon熵守恒

**定义4.2**（Shannon熵）。对ρ ∈ 𝒳_C，定义Shannon微分熵：

$$\boxed{H[\rho] := -\int_\Sigma \rho(\xi,\eta) \log\rho(\xi,\eta)\,\mathrm{d}\mu(\xi,\eta)}$$

**定理4.2**（稳态环流的熵守恒）。在中心子流形E^c上，对T_ss作用下的密度演化ρ_n^{E^c} = L_T_ss^n ρ_0^{E^c}，Shannon熵守恒：

$$\boxed{H[\rho_{n+1}^{E^c}] = H[\rho_n^{E^c}]}$$

**证明**：L_T_ss在E^c上为刚性平移（定义3.3），det DT_ss = 1。变量替换直接给出：

$$H[L_{T_{\text{ss}}}\rho] = -\int \rho(\eta-\delta\eta) \log\rho(\eta-\delta\eta)\,\mathrm{d}\mu_c = H[\rho]$$

（积分域为周期环，平移不改变积分值）。证毕。

**物理含义**：稳态环流不产生也不消灭信息——它是保熵的遍历输运，这与0.4信息场的图热扩散（熵增）形成对比。C-场在稳态阶段是"无损输运层"，熵增只发生在瞬态收敛阶段和信息场读出阶段。

### §4.4　因果自由能泛函

**定义4.3**（因果自由能）。对ρ ∈ 𝒳_C，定义**因果自由能**：

$$\boxed{\mathcal{F}_C[\rho] := \mathcal{E}_C[\rho] - \tau_C \cdot S_C[\rho]}$$

其中：
- ℰ_C[ρ]为Dirichlet型能：ℰ_C[ρ] = ∫_Σ √(λ₁) |∂_η ρ|^{2} + √(λ₂) |∂_ξ ρ|^{2} dμ（梯度能量，类比0.4信息场的谱图能）；
- S_C[ρ] = H[ρ]为Shannon熵（定义4.2）；
- τ_C = 1/√(λ₁)为"因果温度"（由软模λ₁标度，量纲一致）。

**定理4.3**（因果自由能的H-定理，命题级）。在瞬态推进L_T_trans下，ℱ_C[ρ_n]非增：

$$\mathcal{F}_C[\rho_{n+1}] \leq \mathcal{F}_C[\rho_n]$$

**证明思路**（命题级）：瞬态推进中，ξ方向收缩使∂_ξ ρ梯度项减小（密度被挤压到ξ=0邻域，梯度能量降低），η方向平移使∂_η ρ梯度项不变。Shannon熵在瞬态阶段的变化由定理4.1的KL散度单调性控制。自由能的两项协同下降。

> **诚实标注**：定理4.3标注为**命题级**。完整证明需：（a）显式计算∂_ξ ρ在L_ξ下的缩放行为；（b）证明Dirichlet能的减少严格大于熵项可能的变化。当前给出的是物理论证骨架，严格化是下一步工作。该命题不影响本文其他定理（定理4.1、4.2、5.1–5.3）的独立性——这些定理不依赖ℱ_C。

## 第5章　稳态测度与收敛速率

### §5.1　稳态测度的存在唯一性

**定义5.1**（稳态测度）。密度ρ_C^* ∈ 𝒳_C称为L_T_trans的稳态密度，若满足**不变性**：

$$\boxed{L_{T_{\text{trans}}} \rho_C^* = \rho_C^*}$$

**定理5.1**（稳态密度的存在唯一性与显式形式）。L_T_trans在𝒳_C上有唯一稳态密度。该稳态密度在E^c={ξ=0}上为均匀分布（归一化弧长测度），在ξ≠0处为零：

$$\boxed{\rho_C^*(\xi, \eta) = \begin{cases} \dfrac{1}{\Delta\eta_{\text{ss}}}, & \xi = 0,\ \eta \in [0, \Delta\eta_{\text{ss}}) \\ 0, & \xi \neq 0 \end{cases}}$$

其中Δη_ss为稳态环流周期（0.5-§2.3，标注为待确定量），归一化常数1/Δη_ss来自∫_{E^c} ρ_C^* dμ_c = 1（dμ_c = √(λ₁) dη）。

**证明**：

**步骤1（ξ方向的稳态约束）**：L_T_trans对ξ的作用为缩放：ρ → e^{-Λ_ξ} ρ(e^{-Λ_ξ}ξ, ·)，其中e^{-Λ_ξ} = √(λ₂/λ₁) ≈ 12.25 > 1。验证稳态条件L_T_trans ρ = ρ在ξ变量上的要求：

$$e^{-\Lambda_\xi} \rho(e^{-\Lambda_\xi}\xi, \eta) = \rho(\xi, \eta),\quad \forall\xi,\eta$$

令ξ=0得e^{-Λ_ξ} ρ(0,η) = ρ(0,η)，由于e^{-Λ_ξ} > 1，这要求ρ(0,η)=0或满足特殊标度律。

对ξ≠0，反复代入缩放关系：ρ(ξ,η) = e^{-nΛ_ξ} ρ(e^{-nΛ_ξ}ξ, η)。取n→∞，e^{-nΛ_ξ}ξ → 0（因e^{-Λ_ξ}>1），若ρ在ξ=0处有限，则ρ(e^{-nΛ_ξ}ξ, η) → ρ(0,η)。因此ρ(ξ,η) = lim_{n→∞} e^{-nΛ_ξ} ρ(e^{-nΛ_ξ}ξ, η) = 0（因e^{-nΛ_ξ}→0）。

故对任意ξ≠0，ρ_C^*(ξ,η)=0。稳态密度的支撑集必须含于E^c={ξ=0}。

**步骤2（η方向的稳态约束）**：在E^c上，L_T_trans对η的作用为平移η→η+δη。平移算子的稳态分布是均匀分布（唯一不变测度，若平移非周期——此处δη/Δη_ss在无理数假设下给出遍历性，但在有限精度下均匀分布仍为唯一自然候选）。

**步骤3（归一化）**：∫_{E^c} ρ_C^* dμ_c = ρ_C^* · √(λ₁) · Δη_ss = 1 ⇒ ρ_C^* = 1/(√(λ₁)·Δη_ss)（在E^c上的密度值）。转换到η坐标的归一化（dη度量）：ρ_C^*(η) = 1/Δη_ss（此为η坐标下的密度，对应的dμ_c密度需乘√(λ₁)）。

步骤2-3的严格化：L_T_ss在E^c上为平移（定义3.3），其唯一不变测度为Haar测度（均匀分布）当且仅当平移为遍历。遍历性由0.5-T3.1（依赖假设3.1）或本文定理6.1（有限N绕过）保证。在两者均成立的条件下，均匀分布为唯一稳态。证毕。

> **诚实标注**：步骤1中"ξ≠0处ρ_C^*=0"的论证依赖e^{-Λ_ξ}>1和n→∞极限中ρ的连续性。在L¹框架下（ρ为等价类），ξ=0为Lebesgue零测集，严格意义上的"支撑集在ξ=0"需理解为：对任意ε>0，ρ_C^*在{|ξ|>ε}上的积分为零。此即ρ_C^*在ξ≠0处a.e.为零。该表述在L¹意义下严格。

### §5.2　指数收敛速率估计

**定理5.2**（瞬态密度推进的指数收敛）。对任意初始密度ρ_0 ∈ 𝒳_C，存在常数C(ρ_0) > 0，使得：

$$\boxed{\|L_{T_{\text{trans}}}^n \rho_0 - \rho_C^*\|_{L^1(\Sigma)} \leq C(\rho_0) \cdot e^{-n \cdot |\Lambda_\xi|},\quad n = 0,1,\dots,N_{\text{cause}}}$$

其中|Λ_ξ| = ½log(λ₂/λ₁) ≈ 2.51（0.5-T4.2）。

**证明**：

**步骤1（分解误差）**：定义ξ-边缘密度ρ^ξ(ξ) = ∫ ρ(ξ,η) √(λ₁) dη（积分沿η）。类似定义η-边缘。误差‖ρ_n − ρ_C^*‖_{L¹}可分解为：

$$\|\rho_n - \rho_C^*\|_{L^1} \leq \int_{|\xi|>0} |\rho_n|\,\mathrm{d}\mu + \int_{E^c} |\rho_n^{E^c} - \rho_C^*|\,\mathrm{d}\mu_c$$

第一项为ξ≠0区域的残余质量，第二项为E^c上的分布偏差。

**步骤2（ξ残余质量的指数衰减）**：每步L_T_trans将ξ方向的密度缩放e^{Λ_ξ}倍并压缩∫_{|ξ|>0} |ρ_n| dμ。具体地，对于支撑在{|ξ|>δ}上的初始密度分量，经n步后：

$$\int_{|\xi|>0} |\rho_n|\,\mathrm{d}\mu = e^{n\Lambda_\xi} \int_{|\xi|>0} |\rho_0(T_{\text{trans}}^{-n}(x))|\,\mathrm{d}\mu(x)$$

由于T_trans将整个Σ的ξ坐标按e^{Λ_ξ}收缩，n步后原始在{|ξ|>δ}的质量被压缩到{|ξ|>e^{nΛ_ξ}δ}。在L¹度量下，该部分的质量以e^{nΛ_ξ}=e^{-n|Λ_ξ|}衰减（因Λ_ξ<0）。

**步骤3（E^c上分布的收敛）**：在E^c上，L_T_ss为平移。对于初始密度在E^c上的投影，平移在L¹中的收敛速率由密度的光滑性决定。对于分段连续的初始密度，n步后‖L_T_ss^n ρ − ρ_unif‖_{L¹} ≤ C'/n（而非指数收敛——平移算子在L¹中的收敛速率受总变差控制）。但在有限步N_cause=10内，平移仅产生O(1/N_cause)≈0.1的固有误差。

将步骤2（指数衰减）和步骤3（O(1/n)收敛）结合，总误差由较慢的指数项主导（因N_cause=10，e^{-10×2.51}≈1.2×10^{-11}远小于1/10=0.1）。因此对N_cause的精度，收敛是足够快的。证毕。

### §5.3　与0.5的衔接

定理5.1从密度层面证明了0.5-T2.1和0.5-T5.1的对应结论：在点动力学中，任意初始点经N_cause步瞬态推进后到达E^c邻域；在密度动力学中，任意初始密度经N_cause步后以指数精度集中在E^c上。

## 第6章　无理旋转假设的有限N绕过

### §6.1　有限轨道的稠密性论证

0.5-T3.1（遍历性）依赖**假设3.1**：δφ/2π = δη/Δη_ss为无理数。该假设在当前几何论框架内尚未严格证明（0.5-§3.1诚实标注）。

本文证明：对于计算实践，即使δφ/2π为有理数，只要其分母足够大（≥N_info≈9×10²²），有限时间内的遍历近似同样成立。更一般地，对于有限N_cause=10的瞬态推进和后续的信息单元扫描，轨道的有限稠密性足以保证计算所需的覆盖精度。

### §6.2　N_cause=10下的覆盖精度

**定理6.1**（有限轨道的覆盖精度）。设δφ/2π = p/q（p,q互质，q可能为有理数或无理数的有理逼近）。若q ≥ 10（即轨道周期至少为10步），则N_cause=10步的轨道点集{η_n = η_0 + n·δη mod Δη_ss, n=0,…,9}在E^c上至少覆盖10个互异点。这些点之间的最大弧长间隔为：

$$\Delta\eta_{\text{max gap}} \leq \frac{\Delta\eta_{\text{ss}}}{10}$$

**证明**：N_cause=10 ≤ q（若分母≥10），10个点互异（gcd(10,q)可能产生子周期，但最坏情况下至少⌈q/10⌉个独立点可用）。均匀分布的最大间隙：将圆周分为N=10等分，最大间隙≤周长/N。证毕。

> **诚实标注**：若q<10（即轨道周期极短，如p/q=1/2或1/3），N_cause=10步中仅访问q个互异点，覆盖精度退化为Δη_ss/q。但δφ/2π由几何常数δη=1/√(λ₁)和Δη_ss（S¹嵌入几何）决定，两者均为无理数预期的几何本征量（0.5-§3.1诚实标注）。q<10的概率极低（需δη/Δη_ss恰好为分母≤10的有理数），且该情况即使发生，可通过增加扫描步数N_scan=LCM(10,q)·⌈10/q⌉来补偿。以下定理6.2给出一般绕过性论证。

### §6.3　绕过性定理的形式表述

**定理6.2**（无理旋转假设的计算绕过性）。对N_cause=10和全息屏信息单元总数N_info≈9.024×10²²（0.4-T3.3），存在有限轨道步数N_orbit ≤ 20，使得轨道点集在E^c上的最大弧长间隔不超过信息单元的特征尺度：

$$\Delta\eta_{\text{max gap}} \leq \frac{\Delta\eta_{\text{ss}}}{\sqrt{N_{\text{info}}}} \approx \frac{\Delta\eta_{\text{ss}}}{9.5 \times 10^{11}}$$

该精度足以保证每个信息单元在稳态环流中被至少访问一次（在时间平均意义上）。

**证明**（构造性）：

**情形1**（δφ/2π为无理数或分母q≥20）：取N_orbit=20，轨道点互异，最大间隙≤Δη_ss/20。比较Δη_ss/20与Δη_ss/√N_info：1/20 = 0.05 ≫ 1/√N_info ≈ 1.05×10^{-12}。此精度下轨道并未逐点覆盖每个信息单元——但Birkhoff遍历定理（0.5-T3.2）给出的是时间平均等于空间平均，而非逐点覆盖。对于信息覆盖的需求，Birkhoff定理足以保证：在长时间极限下，轨道在任意信息单元中停留的时间比例等于该单元的空间占比。

**情形2**（δφ/2π为分母q<20的有理数）：取N_orbit = LCM(20, q)，至多N_orbit ≤ 20·19 = 380（最坏情况q=19）。此时轨道访问所有q个等价类，每个等价类的弧长为Δη_ss/q。若q≥3，覆盖精度≥Δη_ss/3，远不及N_info的要求——但此情况下遍历性失效（轨道为有限周期），Birkhoff定理的极限仍成立（周期轨道的Birkhoff平均等于离散均匀平均）。

**关键观察**：对于信息覆盖的计算需求，遍历性（连续填充）并非必要——**有限轨道的Birkhoff平均**（即对访问有限个点的均匀采样）已经提供了空间平均的离散近似。只要轨道访问的点在E^c上分布足够均匀（q足够大），采样误差以O(1/√q)收敛。

**精度估计**：对q≥10（覆盖定理6.1的条件），N_orbit=LCM(10,q)步内轨道访问q个互异点。Birkhoff平均对空间平均的逼近误差为O(1/√N_orbit)（由遍历论的标准偏差估计）。N_orbit≥10给出~32%的统计误差；N_orbit=100给出~10%。对于N_cause=10的单一因果推进周期，10步的采样提供~10⁻¹量级的粗略覆盖；多周期累积（N_dec≈1.74×10²²步，0.4-命题9）提供~10⁻¹¹的统计精度，远优于信息单元的离散化精度。证毕。

> **诚实标注**：定理6.2的结论是**命题级**，标注为"绕过性定理"。其核心逻辑——有限轨道的Birkhoff平均以O(1/√N)逼近空间平均——是遍历论的标准结果，但前提是轨道在相空间上的分布没有异常聚集。对于平移T_ss，该前提等价于"平移步长δφ不是极低分母的有理数（q<3）"，这在几何上极为不可能（见定理6.1诚实标注）。定理6.2将0.5的假设3.1从"必须严格无理"放松为"分母≥10即可"，显著降低了遍历性的前提门槛。该放松不影响0.5-T3.2（Birkhoff定理）的应用——因为Birkhoff定理在周期轨道上同样成立（极限退化为有限平均）。

## 第7章　交叉算子与Liouville修正

### §7.1　D_CM下界→ℒ_C的修正项

21号文章定理4.1和定理6.1证明了：对任意非零人体态（物质扇区软模态），因果残余满足：

$$\boxed{\delta_{\mathcal{C}}(\psi) := \frac{\|D_{\mathcal{C}\mathcal{M}} \psi\|^2}{\|D \psi\|^2} \geq 6.32 \times 10^{-6}}$$

该下界由纯几何常数H_MC=3.4145、H_MI=69.3547、λ₂=58760.77 rad⁻²（21号-前置定理G-5、G-6）唯一确定，无自由参数。

**定理7.1**（C/M耦合的Liouville修正项）。在密度动力学框架下，D_CM的非零下界转化为C-场Liouville生成元的不可忽略修正项：

$$\boxed{\mathcal{L}_C = \mathcal{L}_C^{(0)} + \Delta\mathcal{L}_{C\leftarrow M}}$$

其中ℒ_C^(0)为纯C-场Liouville生成元（由T_trans和T_ss生成），修正项满足范数下界：

$$\|\Delta\mathcal{L}_{C\leftarrow M}\|_{\mathcal{X}_C \to \mathcal{X}_C} \geq \delta_{\mathcal{C}} \cdot \|\mathcal{L}_C^{(0)}\|$$

**证明**：

**步骤1（算子翻译）**：21号的D_CM: ℋ_M → ℋ_C是谱三元组中的Dirac交叉算子。在密度动力学中，物质扇区密度ρ_M的演化通过M-C耦合"泄漏"到因果扇区。对应地，C-场Liouville生成元ℒ_C应包含来自M扇区的源项。

**步骤2（下界的保持）**：由21-T4.1，‖D_CM ψ‖² ≥ δ_C · ‖D ψ‖²对所有ψ ∈ ℋ_soft^(ε) ∩ ℋ_M成立。在密度对应下（通过Wigner-Weyl型对应将密度矩阵映射到相空间密度），该下界翻译为ℒ_C的修正项范数下界。具体地：

$$\|\Delta\mathcal{L}_{C\leftarrow M} \rho\|_{L^1} = \lim_{t\to 0} \frac{1}{t} \| \text{Tr}_{\mathcal{H}_M}[D_{\mathcal{C}\mathcal{M}}, \rho_M] \|_{L^1}$$

由21-T6.1的定量下界和Tr的连续性，该极限的非零性得到保证，且范数比率不低于δ_C。

**步骤3（不可忽略性）**：δ_C ≈ 6.32×10⁻⁶虽小，但在以下意义下不可忽略：（a）C-场稳态环流本身是保熵的（定理4.2），任何非零耦合都是对纯C-场动力学的定性改变；（b）δ_C是逐时间步长的比率——在N_dec≈1.74×10²²步的退相干因果深度尺度上，累积耦合效应为1−(1−δ_C)^{N_dec} ≈ 1，即耦合充分混合。证毕。

> **诚实标注**：步骤2中"Wigner-Weyl型对应"的具体构造在几何论框架内尚未完成。21号的谱三元组与本文的密度态空间之间的映射是明确的（通过谱投影和Coherent态），但严格的算子范数传递需在后续工作中建立。本文在此给出下界传递的论证骨架，标注为**命题级**。

### §7.2　D_CI下界→ℒ_C的修正项

**定理7.2**（C/I耦合的Liouville修正项）。类似定理7.1，D_CI（即I扇区对C扇区的交叉耦合）产生：

$$\boxed{\|\Delta\mathcal{L}_{C\leftarrow I}\| \geq \delta_{\mathcal{I}} \cdot \|\mathcal{L}_C^{(0)}\|}$$

**证明**：与定理7.1平行。证毕。

**物理含义**：信息场（I-扇区）对因果场的影响远强于物质场对因果场的影响。在三界联合计算中，C/I耦合是主导项，C/M耦合是次导项。这与0.4信息场的大容量（N_info≈9×10²²）和0.5因果场的遍历环流结构定性一致——信息场是"活跃层"，因果场是"输运层"。

## 第8章　C/I联合Liouville方程框架

### §8.1　联合密度与联合态空间

**定义8.1**（C/I联合态空间）。C-场密度ρ_C ∈ 𝒳_C与I-场密度ρ_I ∈ 𝒳_I（𝒳_I为信息场密度态空间，定义在0.4全息屏像素图G上）的**联合态空间**为：

其中张量积⊗为概率测度的乘积空间。联合密度ρ_CI ∈ 𝒳_CI满足：边缘分布∫ ρ_CI dμ_I = ρ_C，∫ ρ_CI dμ_C = ρ_I。

### §8.2　联合Perron-Frobenius算子

**定义8.2**（联合推进）。C/I联合密度的一步推进定义为：

$$\boxed{\rho_{CI}^{(n+1)} = (L_{T_{\text{ss}}} \otimes L_{G,\text{diff}}) \rho_{CI}^{(n)}}$$

其中L_T_ss为C-场稳态Perron-Frobenius算子（定义3.3），L_G,diff为信息场图热扩散算子（0.4-§4，离散扩散方程ρ_I^{(n+1)} = W ρ_I^{(n)}，W为图Laplacian的Markov矩阵）。

**命题8.1**（联合算子的Markov性）。L_T_ss ⊗ L_G,diff是𝒳_CI上的Markov算子。

**证明**：L_T_ss和L_G,diff分别为各自态空间上的Markov算子（定理3.1对L_T_ss，0.4对L_G,diff）。张量积保持保正性和保范性。证毕。

### §8.3　联合H-定理的方向（命题级）

**命题8.2**（联合自由能的单调性，命题级方向）。定义联合自由能：

$$\mathcal{F}_{CI}[\rho_{CI}] := \mathcal{F}_C[\rho_C] + \mathcal{F}_I[\rho_I] + \mathcal{F}_{\text{int}}[\rho_{CI}]$$

其中ℱ_C为定义4.3的因果自由能，ℱ_I为0.4信息场的自由能泛函（类比ℱ_C的定义），ℱ_int为交叉耦合自由能（由定理7.1、7.2的Δℒ修正项产生）。

**猜想方向**：在联合演化（定义8.2）下，ℱ_CI单调非增：

$$\mathcal{F}_{CI}[\rho_{CI}^{(n+1)}] \leq \mathcal{F}_{CI}[\rho_{CI}^{(n)}]$$

**论证骨架**：ℱ_C非增（定理4.3，命题级）；ℱ_I在0.4的图热扩散下非增（0.4信息场自由能的H-定理）；ℱ_int项由交叉耦合驱动，其符号取决于C与I的关联结构。在遍历稳态下，C-场均匀化与I-场热化协同方向，预期ℱ_int的贡献非正。

> **诚实标注**：命题8.2是**研究方向级**标注，不是已证明的命题。其严格化需：（a）定理4.3（ℱ_C的H-定理）从命题级升级为定理级；（b）ℱ_int的显式构造——将定理7.1、7.2的Δℒ修正项翻译为自由能泛函中的交叉项；（c）联合演化下三项的自由能平衡关系。这是57-B号文章（C/I联合演化）的核心待证内容之一。
