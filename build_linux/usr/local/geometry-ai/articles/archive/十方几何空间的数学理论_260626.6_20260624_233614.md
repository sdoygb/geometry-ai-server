## 第 2 章 激发态参数空间的公理化

### 2.1 公理 1（参数空间的拓扑）

**公理 1（圆拓扑公理）** 激发态参数空间 $D$ 由一维紧致连通流形 $S^1$（圆）去掉两个不同点 $p_0$ 与 $p_*$ 得到：

$$D = S^1 \setminus \{p_0, p_*\}, \qquad p_0 \neq p_*$$

去掉的两点分别称为**真空点**（$p_0$）与**退化点**（$p_*$）。由此，$D$ 由两个连通分支组成：

$$D_+ \cong (0,1), \qquad D_- \cong (0,1)$$

每个分支同胚于开区间。参数 $x \in D_\pm$ 标记激发态。

**引理 2.1（分支的拓扑结构）** 设 $S^1$ 为标准的单位圆 $\{e^{i\phi} \mid \phi \in [0,2\pi)\}$。去掉两点 $p_0=e^{i\phi_0}$ 与 $p_*=e^{i\phi_*}$（不妨设 $0\leq \phi_0 < \phi_* < 2\pi$），则 $S^1\setminus\{p_0,p_*\}$ 恰有两个连通分支：
$$D_+ = \{e^{i\phi} \mid \phi_0 < \phi < \phi_*\}, \quad D_- = \{e^{i\phi} \mid \phi_* < \phi < \phi_0+2\pi\}.$$
每个分支作为 $S^1$ 的子空间拓扑，均同胚于开区间 $(0,1)$。

*证明.* 考虑标准覆叠映射 $\pi: \mathbb{R} \to S^1$, $\pi(\phi)=e^{i\phi}$。在 $\mathbb{R}$ 上，$\pi^{-1}(S^1\setminus\{p_0,p_*\}) = \mathbb{R}\setminus\{\phi_0+2k\pi, \phi_*+2k\pi \mid k\in\mathbb{Z}\}$。该集合的每个连通分支均为开区间 $(\phi_0+2k\pi, \phi_*+2k\pi)$ 或 $(\phi_*+2k\pi, \phi_0+2(k+1)\pi)$。由于 $\pi$ 在长度为 $2\pi$ 的区间上的限制是同胚，这些开区间在 $S^1$ 中的像恰为 $D_+$ 与 $D_-$。每个开区间同胚于 $(0,1)$，故 $D_\pm \cong (0,1)$。 $\square$

### 2.2 公理 2（几何量的分析性质）

**公理 2（边界极限公理）** 在参数空间 $D$ 上存在几何量 $S: D \to (0,+\infty)$，满足：

1. **连续性**：$S$ 在每个连通分支 $D_\pm$ 上连续；
2. **真空极限**：当 $x$ 沿任一分支逼近 $p_0$ 时，$\lim_{x \to p_0} S(x) = 0$；
3. **退化极限**：当 $x$ 沿任一分支逼近 $p_*$ 时，$\lim_{x \to p_*} S(x) = +\infty$。

**注 2.1** 公理 2 不要求 $S$ 在 $D_+$ 与 $D_-$ 上具有相同的函数表达式，也不要求 $S$ 在整个 $D$ 上连续（$p_0$ 与 $p_*$ 不在 $D$ 中）。因此 $S$ 可以是两个独立分支上的不同函数，只要各自满足边界条件即可。

### 2.3 值域定理

**定理 2.1**（值域）在公理 1 与公理 2 下，$S$ 在每个连通分支 $D_\pm$ 上的值域为 $S(D_\pm) = (0, +\infty)$。

*证明.* 由公理 1，$D_\pm \cong (0,1)$ 是连通集。由公理 2，$S$ 在 $D_\pm$ 上连续，故像集 $S(D_\pm)$ 是 $\mathbb{R}$ 中的连通子集，即区间。由真空极限，$\inf S(D_\pm) = 0$；由退化极限，$\sup S(D_\pm) = +\infty$。由于 $p_0, p_* \notin D$，端点 $0$ 与 $+\infty$ 均不可达。因此 $S(D_\pm) = (0, +\infty)$。 $\square$

**定理 2.2**（存在性）对任意给定的正数 $S_0 > 0$，在每个分支 $D_+$ 与 $D_-$ 上各至少存在一个激发态 $x_+ \in D_+$ 与 $x_- \in D_-$，使得 $S(x_+) = S(x_-) = S_0$。

*证明.* 由定理 2.1，$S(D_\pm) = (0, +\infty)$。对任意 $S_0 > 0$，显然 $S_0 \in (0, +\infty)$。故 $S^{-1}(S_0) \cap D_\pm \neq \varnothing$。 $\square$

**定义 2.1'（严格单调性假设）** 若 $S$ 在 $D_\pm$ 上严格单调，则对任意 $S_0>0$，定理 2.2 中的 $x_+$ 与 $x_-$ 唯一确定。严格单调性**不是**从公理 1–2 推出的定理，而是框架的**构造性假设**。

## 第 3 章 约束乘积球面与谱刚性

### 3.0 互锁函数的群论来源

**引理 3.0（三分切丛置换群的刚性）** 设 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 为三分切丛，$G$ 为其扇区置换对称群。若 $G$ 在三个扇区上的作用忠实且可迁，并包含任意两扇区对换，则 $G \cong S_3$，且 $S_3$ 是满足此条件的唯一有限群。

**定义 3.0（互锁函数与紧致函数）** 设 $G$ 为有限群，定义：
- **互锁函数** $\Lambda(G) := |\text{Conj}(G)|$（共轭类数）；
- **紧致函数** $k_0(G) := [G : N_{\max}]$（极大正规子群的指数）。

**命题 3.0** 对于三分切丛 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 的置换对称群 $G = S_3$：
$$\Lambda(S_3) = 3, \quad k_0(S_3) = 2.$$

**定理 3.0'（三分切丛扇区置换的刚性）** 设 $M = S^3 \times S^3 \times S^3$ 配备乘积度量，且存在切丛分解 $TM = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 使得三个扇区在度规层面完全等价（即任意置换诱导等距自同构）。则扇区置换群必包含所有两扇区对换，从而唯一同构于 $S_3$。进而 $\Lambda(G)=3$，$k_0(G)=2$ 是群论函数的必然输出。

### 3.1 定义与假设

**定义 3.1**（约束乘积球面）设 $a > 0$ 为整体尺度因子。由定理 3.0' 与命题 3.0，三分切丛置换群 $S_3$ 的群论函数输出为：
$$\Lambda = \Lambda(S_3) = 3, \quad k_0 = k_0(S_3) = 2, \quad \ell_0 > 0 \text{（谱几何单位的锚点）}.$$

称九维闭流形
$$M(a) = S^3(a) \times S^3(a/\sqrt{3}) \times S^3(a/\sqrt{6})$$
在配备乘积度量 $g = g_a \oplus g_{a/\sqrt{3}} \oplus g_{a/\sqrt{6}}$ 时，构成一个**约束乘积球面**（CPS）。

**备注 3.2**（互锁常数的性质）常数 $\Lambda=3$、$k_0=2$ 由定理 3.0' 与命题 3.0 从 $S_3$ 群论结构直接计算得出。$\ell_0$ 作为正实数参数，其数学地位是**谱几何单位的锚点**（等价于选择使 $M(\ell_0)$ 体积归一化的长度单位，见 §10.4），不引入新的物理常数，不参与模空间的一维参数化。

### 3.2 模空间的参数化

**定理 3.1**（模空间的参数化）约束乘积球面类 $\{M(a)\}_{a>0}$ 的模空间与 $(0, \infty)$ 同胚，$a$ 是唯一的连续自由度。不同 $a$ 对应不同的等距类。

### 3.3 Laplace 特征值的显式计算

**命题 3.1**（乘积流形上的谱分离）对 $M(a)$，Laplace–Beltrami 算子 $\Delta_g$ 的特征值全体为
$$\lambda_{p,q,r}^\Delta = \frac{p(p+2)}{a^2} + \frac{q(q+2)}{b^2} + \frac{r(r+2)}{c^2}, \quad p,q,r \in \mathbb{N}_0.$$

**命题 3.2**（前三个特征值的显式表达式）在互锁常数锁定下，$M(a)$ 的前三个非零特征值为：

| 模式 (p,q,r) | 特征值表达式 | 与 3/a² 的比值 |
|:---|:---|:---|
| (1,0,0) | 3/a² | 1 |
| (2,0,0) | 8/a² | 8/3 |
| (0,1,0) | 3Λ/a² = 9/a² | Λ = 3 |

### 3.4 特征值排序与谱特征签名

**定理 3.2**（特征值排序）前四个非零特征值满足
$$\lambda_1^\Delta = \frac{3}{a^2} < \lambda_2^\Delta = \frac{8}{a^2} < \lambda_3^\Delta = \frac{3\Lambda}{a^2} < \lambda_4^\Delta = \min\left\{\frac{3\Lambda k_0}{a^2},\ \frac{3(\Lambda+1)}{a^2}\right\}.$$

**推论 3.1**（特征签名）比值
$$\frac{\lambda_2^\Delta}{\lambda_1^\Delta} = \frac{8}{3}, \quad \frac{\lambda_3^\Delta}{\lambda_1^\Delta} = \Lambda$$
是与 $a$ 无关的普适常数，作为约束乘积球面类的**谱特征签名**。

### 3.5 由前三个特征值反解尺度因子

**定理 3.3**（反解公式）给定前三个非零 Laplace 特征值，可唯一反解尺度因子。自洽性条件：$\lambda_2 = \frac{8}{3}\lambda_1$ 且 $\lambda_3 = \Lambda \lambda_1 = 3\lambda_1$。

### 3.6 谱刚性定理

**定理 3.4**（约束乘积球面的谱刚性）设 $M(a)$ 与 $M(a')$ 为两个具有相同互锁常数的约束乘积球面。若它们的前三个非零 Laplace–Beltrami 特征值对应相等，则 $M(a)$ 与 $M(a')$ 等距。

### 3.7 与经典反例的兼容性

**定理 3.5** Sunada 型等谱非等距反例在约束乘积球面类中不存在（因 $M(a)$ 单连通，基本群为零）。Milnor 型平坦环面反例不适用（$M(a)$ 具有正截面曲率）。Gordon–Webb–Wolpert 型反例不适用（$M(a)$ 为无边界的闭流形）。

### 3.8 谱刚性的维数必要条件

**定理 3.6**（谱刚性的维数必要条件）设 $\mathcal{M}$ 是某流形类的模空间，具有有限维数 $d$。若前 $k$ 个 Laplace 特征值能够实现谱刚性（即谱映射 $\sigma_k$ 为单射），则必有 $d \leq k$。在互锁常数锁定下，$\{M(a)\}_{a>0}$ 的模空间维数 $d=1$，故仅需 $k\geq1$ 即可满足必要条件。

*证明。* 假设 $d > k$。由于特征值连续依赖于度规，$\sigma_k$ 是连续映射。若 $\sigma_k$ 是单射，则给出了从 $d$ 维流形 $\mathcal{M}$ 到 $\mathbb{R}^k$ 的连续单射。由 **Brouwer 区域不变性定理**：若 $U \subset \mathbb{R}^d$ 为开集，$f: U \to \mathbb{R}^k$ 为连续单射，则 $f(U)$ 为 $\mathbb{R}^k$ 中的开集（关于子空间拓扑）。然而 $d$ 维流形的每一点存在同胚于 $\mathbb{R}^d$ 的开邻域 $U$。若 $d > k$，$f(U)$ 作为 $\mathbb{R}^k$ 的子集，其拓扑维数不可能超过 $k$，而 $U$ 的拓扑维数为 $d > k$，矛盾。故 $d > k$ 时连续单射 $\sigma_k$ 不可能存在。$\square$

**详细说明：** Brouwer 区域不变性定理断言：若 $U\subset\mathbb{R}^d$ 为开集且 $f:U\to\mathbb{R}^d$ 为连续单射，则 $f(U)$ 为 $\mathbb{R}^d$ 中的开集。当 $d>k$ 时，设 $f:U\to\mathbb{R}^k$ 为连续单射（$U\subset\mathbb{R}^d$ 开集），将其与标准嵌入 $\iota:\mathbb{R}^k\hookrightarrow\mathbb{R}^d$ 复合，得连续单射 $\iota\circ f:U\to\mathbb{R}^d$。由区域不变性定理，$(\iota\circ f)(U)$ 应为 $\mathbb{R}^d$ 中开集。但 $(\iota\circ f)(U)\subset\iota(\mathbb{R}^k)$，而 $\iota(\mathbb{R}^k)$ 作为 $\mathbb{R}^d$ 的 $k$ 维线性子空间，其拓扑维数为 $k<d$，在 $\mathbb{R}^d$ 中无内点——任何包含于 $\iota(\mathbb{R}^k)$ 的集合在 $\mathbb{R}^d$ 中均不可能为开集（非空开集必含 $d$ 维开球，但 $\iota(\mathbb{R}^k)$ 不含任何 $d$ 维开球）。矛盾。因此 $d>k$ 时不存在从 $U$ 到 $\mathbb{R}^k$ 的连续单射。这是一个纯拓扑维数论证，仅依赖 Brouwer 区域不变性定理与拓扑维数的基本性质，不涉及 Lebesgue 测度。
