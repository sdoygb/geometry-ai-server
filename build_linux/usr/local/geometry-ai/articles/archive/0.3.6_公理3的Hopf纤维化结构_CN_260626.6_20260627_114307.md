# 第一章：公理3的结构与 P_ℳ 的几何含义

## §1.1 公理3的回顾

**公理3（三分切丛，0.0.6）：** 三维约束流形 M(a) 的切丛分裂为三个光滑子丛的直和：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

其中 ℳ（物质）、𝒞（约束）、ℐ（信息）是三个秩为1的实向量丛。此分裂是光滑的（0.0.6 命题2.1），且三个扇区在公理1（θ_M + θ_C + θ_I = 90°）的约束下耦合。

设 P_ℳ: TM(a) → ℳ 为到 ℳ 扇区的正交投影算子。由公理3，P_ℳ 是秩-1丛映射，在每个纤维上实现线性投影。

## §1.2 P_ℳ 的核丛

**定理1.1（核丛的秩-2性）：** 定义 V = ker(P_ℳ) ⊂ TM(a)。则 V 是 M(a) 上的秩-2光滑子丛，V_p = 𝒞_p ⊕ ℐ_p 对每点 p ∈ M(a) 成立。

**证明：** 在每个纤维 T_pM(a)，P_ℳ|_p: T_pM(a) → ℳ_p 是线性投影，核为 𝒞_p ⊕ ℐ_p。由公理3，dim(𝒞_p) = dim(ℐ_p) = 1，故 dim(V_p) = 2。V 的光滑性来自 TM(a) = ℳ ⊕ 𝒞 ⊕ ℐ 的光滑直和分解（0.0.6 命题2.1）——𝒞 和 ℐ 分别是光滑子丛，其直和 V = 𝒞 ⊕ ℐ 继承光滑性。□

**来源：0.0.6 命题2.1（光滑直和分解）；线性代数（投影的核）。**

# 第三章：ℳ 的左不变性与光滑 S² 商空间

**策略转变（G4闭合）：** 旧版试图通过 V-foliation（秩-2叶状结构）论证商空间 ≅ S²。这是维度错误——秩-2分布给出1维叶空间，不可能微分同胚于2维 S²。正确路径：**ℳ 是秩-1子丛，自动可积，其积分曲线是1维纤维，商空间是2维 S²。** 本章给出此路径的完整论证，并借助 ℳ 的 SU(2) 左不变性同时闭合叶空间光滑性缺口（G4）。

## §3.1 ℳ 的自动可积性

**引理3.1（秩-1子丛的自动可积性）：** 任何光滑流形上的秩-1光滑子丛都是 Frobenius 可积的。

**证明：** 设 L 是秩-1子丛，X, Y 是 L 的任意两个局部截面。由于 L 的纤维是1维的，存在局部光滑函数 f 使得 Y = fX。则 Lie 括号：

$$[X, Y] = [X, fX] = (X \cdot f)X \in \Gamma(L)$$

其中 X·f 是 X 对函数 f 的方向导数。故 L 的截面空间在 Lie 括号下封闭，满足 Frobenius 可积条件。□

**推论3.1：** ℳ 在 M(a) ≅ S³_ℳ 上定义了一个1维叶状结构 ℱ_ℳ——积分曲线（叶片）是 S³_ℳ 上处处与 ℳ 相切的光滑1维子流形。

**证明：** 由公理3，ℳ 是秩-1光滑子丛。引理3.1直接适用。□

**状态：定理（秩-1子丛自动可积是微分拓扑的标准结论）。**

## §3.2 ℳ 的左不变性与 S¹ 纤维化

**S³ ≅ SU(2) 的群结构：** 三维球面 S³ 微分同胚于特殊酉群 SU(2)。SU(2) 上的左平移作用给出 TS³ 的一个整体平行化——𝔰𝔲(2) 的一组基 {e₁, e₂, e₃} 对应的左不变向量场在每点张成切空间。

**定义 3.1（左不变子丛）：** TS³ 的秩-1子丛 L 称为**左不变**的，如果存在非零 X ∈ 𝔰𝔲(2) 使得 L_p = span{X(p)} 对所有 p ∈ S³ 成立，其中 X(p) 是 X 在 p 点的左不变向量场的值。

**引理3.2（左不变子丛的积分曲线）：** 若 ℳ 是 S³_ℳ 上的左不变秩-1子丛，则 ℱ_ℳ 的叶片是紧致的一维闭合子流形，微分同胚于 S¹。

**证明：** 设 ℳ 由左不变向量场 X ∈ 𝔰𝔲(2)\\{0} 张成。积分曲线是群乘法下的左平移：φ_t(p) = p · exp(tX)。由于 SU(2) 紧致，1-参数子群 {exp(tX) : t ∈ ℝ} 是 U(1) ≅ S¹ 的嵌入。因此每条积分曲线是 S¹ 在 SU(2) 中的左陪集，紧致且微分同胚于 S¹。□

**命题3.2（ℳ 的左不变性——来自 CPS 类）：** ℳ 是 S³_ℳ 上的左不变子丛。

**论证：** CPS 类 M(a) = S³(a) × S³(a/√Λ) × S³(a/√Λk₀) 是三个 S³ 因子的乘积，携带自然的 SU(2)³ 等距群——每个因子由其 SU(2) 左平移作用。公理3的三分切丛分解 TM(a) = ℳ₉ ⊕ 𝒞₉ ⊕ ℐ₉（各秩3）来自乘积流形的标准因子切丛分解，该分解在 SU(2)³ 作用下不变。

三维约束流形 S³_ℳ 是此九维乘积在公理3约束条件下的三维子流形，同胚于 Material S³ 因子。S³_ℳ 通过以下方式从 SU(2)³ 继承 SU(2) 左作用：取对角嵌入 Δ: SU(2) → SU(2)³，Δ(g) = (g, φ_𝒞(g), φ_ℐ(g))，其中 φ_𝒞, φ_ℐ 是由角度参数 θ_C, θ_I 决定的李群同态。S³_ℳ 即 Δ(SU(2)) 在某基点的轨道（或等价地，取 Δ 为包含映射 (g, e, e)）。

在此继承的 SU(2) 作用下，ℳ₉ 在 S³_ℳ 切空间上的投影 ℳ 保持左不变性——因为 ℳ₉ 是 Material S³ 因子切丛的拉回，在第一个 SU(2) 因子的左作用下不变，而 Δ 的第一个分量是恒等映射 g ↦ g。

**结论：** ℳ 是 S³_ℳ ≅ SU(2) 上的左不变秩-1子丛。∎

**状态：命题。** 上述论证的严格性依赖于对角嵌入 Δ 的显式构造（φ_𝒞, φ_ℐ 与角度参数的精确关系），这需要回到 0.0.6 的约束流形定义。在当前分析层次，ℳ 的左不变性作为 CPS 类 SU(2) 结构在约束子流形上的自然继承，具有几何上的必然性。

4. **CPS的不同使用方式：** c_m 完全不依赖 CPS 半径比（C_m = c_m · A_Σ 仅涉及 Σ 的面积，而 Σ 的面积仅依赖 χ_L）。c_{9,1} 则强烈依赖 CPS 半径比——1:1/√3:1/√6 是得到简洁有理数 40/27 的必要条件。

**唯一的共同输入是 Hopf 假设 r_ℳ = r_Σ。**

### §8.1.3 交叉检验的统计意义

两个独立物理常数（K 和 v_geo）分别通过两条独立谱几何路径，在同一个假设下，各自收敛到极其简洁的无量纲数字（1 和 40/27），且偏差分别 < 0.02% 和 < 0.002%。

**这不是数值巧合的合理候选。** 理由：

1. K 和 v_geo 是独立锁定的——它们来自不同的实验输入（电子质量 m_e 和精细结构常数 α = 1/S_e）
2. c_m 的推导中，分子 K·χ_L = 642.2 与分母 641.95 的接近不是调节出来的——144π^(3/2)·3^(1/4) 是纯几何因子，sin³θ_M 由 S 的全局极小独立确定
3. c_{9,1} 的推导中，CPS 半径比来自 S₃ 群论（Λ=3, k₀=2），不是为拟合 c_{9,1} 而"选择"的——等半径和角度投影比均被明确拒斥

**双重交叉检验对 Hopf 假设提供的联合支撑强度远超任何单一检验。**
