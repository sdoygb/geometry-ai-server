## 摘　要

0.5.2建立了联合自由能ℱ_CI的显式形式（以乘积参考态ρ_C^*⊗ρ_I^*为基准），并证明了乘积演化下的H-定理。但0.5.2命题5诚实标注：全耦合演化L = L^(0) + ΔL_int的联合稳态ρ_CI^*不是乘积的（因21号不可解约定理强制I>0），且以真实稳态为参考的ℱ̃_CI = D_KL(ρ_CI ‖ ρ_CI^*)的H-定理虽是定理级，但本文ℱ_CI（乘积参考）在全耦合下的单调性仅为命题级。

本文直接求解联合稳态ρ_CI^*：在弱耦合框架（δ_I ≈ 2.61×10⁻³）下，建立摄动展开，给出ρ_CI^*的一阶修正显式表达式，证明I(ρ_CI^*) = O(δ_I²) > 0，并建立全耦合H-定理的严格版本（以ρ_CI^*为参考）。

**核心结果：**

1. **存在性**（定理1，定理级）：紧Markov算子L = L^(0)+ΔL_int存在至少一个不变密度ρ_CI^*。
2. **非乘积性**（定理2，定理级）：ρ_CI^* ≠ ρ_C^*⊗ρ_I^*，且I(ρ_CI^*) > 0（由21-T4.1直接保证）。
3. **一阶修正**（定理3，定理级——粗粒化块演化框架）：在N_cause块粗粒化下，ρ_CI^* = ρ_C^*⊗ρ_I^* + δ_I·ρ^(1) + O(δ_I²)，其中ρ^(1)由Fredholm方程(I−L̄^(0))†ΔL̄^(1)(ρ_C^*⊗ρ_I^*)显式确定。
4. **稳态互信息上界**（命题4，命题级）：I(ρ_CI^*) ≤ (δ_I²/2γ_I)·C_Δ + O(δ_I³)，其中γ_I为I-场图扩散谱隙，C_Δ = ‖ΔL̄^(1)‖²。
5. **全耦合H-定理**（定理5，定理级）：ℱ̃_CI[ρ_CI] := β·D_KL(ρ_CI ‖ ρ_CI^*)在L下严格单调递减。
6. **ℱ_CI近似单调性**（命题6，命题级）：在弱耦合O(δ_I²)精度内，本文ℱ_CI（乘积参考）一致单调递减。

## 第1章　问题定位：联合稳态的数学结构

### 1.1　耦合演化算子的完整形式

C-I联合系统的演化由Perron-Frobenius算子给出：

$$\boxed{\rho_{CI}^{(n+1)} = L \rho_{CI}^{(n)}, \qquad L = L^{(0)} + \Delta L_{\text{int}}}$$

其中：

- **L^(0) = L_T_ss ⊗ L_G,diff**（0.5.1定义8.2）：乘积演化算子。L_T_ss为C-场稳态环流算子（0.5.1定义3.3），L_G,diff为I-场图热扩散算子（0.4-§4）。
- **ΔL_int**：耦合修正算子，满足‖ΔL_int‖ ≥ δ_I·‖L^(0)‖（由0.5.1定理7.2/7.3，定理级），δ_I ≥ 2.61×10⁻³（21-T6.2，定理级）。

L^(0)和ΔL_int分别作用在𝒳_CI = 𝒳_C ⊗ 𝒳_I上（0.5.1定义8.1）。

### 1.2　已知的稳态与矛盾

**乘积稳态**：L^(0)的不变密度为乘积形式ρ^(0) := ρ_C^* ⊗ ρ_I^*，其中：

- ρ_C^*为E^c上的均匀分布（0.5.1定理5.1，定理级）；
- ρ_I^*为图G上的均匀分布（信息最大化状态，0.4-T3.3推论）。

在乘积稳态下，I(ρ^(0)) = I(ρ_C^*⊗ρ_I^*) = 0——即C与I完全解相关。

**矛盾**：21号不可解约定理（21-T4.1，定理级）证明交叉算子D_CM和D_CI在人体态上严格非零，这意味着C与I之间存在不可消除的耦合。在密度态空间层面，该耦合体现为ΔL_int ≠ 0（0.5.1定理7.3，定理级）。因此ρ^(0)不可能是全耦合演化L的不变密度：

$$\rho_{CI}^* \neq \rho_C^* \otimes \rho_I^*, \qquad I(\rho_{CI}^*) > 0$$

### 1.3　本文的求解策略

**直接目标**：在弱耦合条件δ_I ≪ 1下，求解L的不变密度ρ_CI^*。

**核心困难**：L^(0)的谱结构——L_T_ss为旋转算子（Perron-Frobenius形式），其在单位圆上具有连续谱（无理旋转情形），(I−L^(0))的伪逆在通常意义下无界。这使标准Kato摄动理论（依赖孤立本征值的谱隙）不能直接应用。

**本文策略——粗粒化块演化**：考虑N_cause步的粗粒化演化：

$$\bar{L}^{(0)} = L_{T_{\text{trans}}}^{N_{\text{cause}}} \otimes L_{G,\text{diff}}^{N_{\text{cause}}}$$

L_T_trans为瞬态推进（0.5.1定义3.2），具有指数收缩（Lyapunov指数Λ_ξ ≈ −2.51，0.5.1定理5.2）。N_cause≈10步后，C-场密度以精度e^{N_cause·Λ_ξ} ≈ e^{−25.1} ≈ 1.2×10⁻¹¹收敛到E^c。粗粒化后L̄^(0)具有严格的谱隙（由瞬态收缩保证），摄动理论适用。

**物理理由**：C-场的遍历环流与瞬态收缩交替进行——每次瞬态推进将偏离E^c的密度分量指数压制，随后稳态环流在E^c上遍历。在N_cause步的时间尺度上粗粒化，C-场在每个块内的净效果是投影到E^c上的均匀分布（精度~10⁻¹¹）。δ_I·N_cause ≈ 2.6×10⁻² ≪ 1，弱耦合条件在块演化层面成立。

## 第2章　存在性：Krylov-Bogoliubov论证

### §2.1　紧性与不变测度

**引理2.1**（𝒳_CI的紧性）。𝒳_CI = 𝒳_C ⊗ 𝒳_I在L¹弱拓扑下是紧的。

**证明**：𝒳_C为有限维紧流形Σ上概率密度的L¹空间——Σ为2维紧流形（S²的约束截面），其上L¹概率密度集在弱拓扑下序列紧（由Prohorov定理）。𝒳_I为有限图G（|G| = N_info ≈ 9×10²²）上的概率单纯形（N_info−1维紧单纯形）。张量积保持紧性。证毕。

> **诚实标注**：Σ作为连续流形上的L¹空间，其概率密度集在弱拓扑下的序列紧性依赖Banach-Alaoglu定理（L¹不是自反的，但L¹的有界集在L^∞的弱*拓扑下紧——但这里L¹概率密度集需要更仔细的处理）。在本文框架中，𝒳_C以有限分辨率的像素化近似（与I-场全息屏像素网格一致），此时𝒳_C为有限单纯形，紧性平凡。连续极限的紧性论证在此标注为命题级，但不影响本文核心结论——所有后续推导均基于有限分辨率近似成立。

### §2.2　存在性定理

**定理1**（联合稳态的存在性）。全耦合Perron-Frobenius算子L = L^(0) + ΔL_int在𝒳_CI上至少存在一个不变概率密度ρ_CI^*：

$$L \rho_{CI}^* = \rho_{CI}^*, \qquad \rho_{CI}^* \in \mathcal{X}_{CI}$$

**证明（Krylov-Bogoliubov）**：

取任意初始密度ρ^(0) ∈ 𝒳_CI。定义Cesàro平均序列：

$$\rho^{(N)} := \frac{1}{N} \sum_{n=0}^{N-1} L^n \rho^{(0)}$$

**步骤1（不变性）**：由于𝒳_CI在弱拓扑下紧（引理2.1），序列{ρ^(N)}存在收敛子列{ρ^(N_k)}，极限点记为ρ_CI^*。

**步骤2（不变性的验证）**：

$$L \rho_{CI}^* - \rho_{CI}^* = \lim_{k\to\infty} \frac{1}{N_k} (L^{N_k} \rho^{(0)} - \rho^{(0)})$$

由于‖L^{N_k} ρ^(0) − ρ^(0)‖_{L¹} ≤ 2（概率密度的L¹范数有界），而1/N_k → 0，故‖L ρ_CI^* − ρ_CI^*‖ = 0，即ρ_CI^*为L的不变密度。

**步骤3（L的Markov性保证子列收敛到概率密度）**：L保正保范（由0.5.1定理3.1的Markov性质，ΔL_int的保总概率性在0.5.1定理7.3的构造中保证），Cesàro平均保持概率密度的归一化和非负性。证毕。

**定理1的状态：定理级。** Krylov-Bogoliubov论证是遍历论的标准定理，适用于紧空间上的连续Markov算子。在本文有限分辨率近似下，紧性和连续性均成立。

## 第3章　乘积算子L^(0)的谱分析与粗粒化

### §3.1　L^(0)的谱结构

L^(0) = L_T_ss ⊗ L_G,diff的谱为两个算子谱的张量积。

**C-场部分L_T_ss**：T_ss是E^c上的刚性旋转（步长δφ = ω·δτ）。L_T_ss在L²(E^c, dμ_C)上是酉算子（旋转保持L²范数）。其谱在单位圆上：

$$\sigma(L_{T_{\text{ss}}}) = \overline{\{e^{ik\delta\varphi} : k \in \mathbb{Z}\}}$$

- δφ/2π为无理数时：谱为整个单位圆的稠密子集（连续谱），唯一的L¹不变密度是均匀分布ρ_C^*。
- δφ/2π为有理数（分母q）时：谱为q个离散点，不变密度为任意周期为q的密度。

由0.5.1定理6.2（命题级），δφ/2π的分母≥10即可保证N_cause=10步内的轨道覆盖精度~10⁻³。本文在此精度下工作。

**I-场部分L_G,diff**：图G上的Markov扩散矩阵W。ρ_I^* = 均匀分布（向量(1/N_info, …, 1/N_info)）是唯一的不变分布。W的谱满足：

$$\sigma(W) \subset [-1, 1], \quad \lambda_1 = 1 \text{（单重）}, \quad |\lambda_2| \leq 1 - \gamma_I$$

其中γ_I = 1−λ_2为扩散谱隙（Fiedler特征值的离散对应）。对大小为N_info ≈ 9×10²²的连通图，γ_I ~ O(1/N_info)（对于正则图或扩展图，具体数值依赖图拓扑——0.4确定）。

### §3.2　谱隙问题

L^(0)的特征值1对应乘积型不变密度ρ_C^*⊗ρ_I^*。但L^(0)没有通常意义下的谱隙——因为L_T_ss在单位圆上有连续谱（无理旋转），I−L^(0)在不变子空间的正交补上不可逆。

**后果**：标准Kato摄动理论不能直接用于L^(0)在L¹中的摄动。在不变子空间的正交补上，(I−L^(0))^†的范数无界——这使得一阶摄动修正可能发散。

### §3.3　粗粒化块演化

为绕过谱隙问题，考虑N_cause步的粗粒化演化。定义：

$$\boxed{\bar{L}^{(0)} := L_{T_{\text{trans}}}^{N_{\text{cause}}} L_{T_{\text{ss}}} \otimes L_{G,\text{diff}}^{N_{\text{cause}}}}$$

**关键性质**：L_T_trans为瞬态推进，在L¹中具有指数收缩性质。在N_cause步后：

$$\|L_{T_{\text{trans}}}^{N_{\text{cause}}} \rho_C - \rho_C^*\|_{L^1} \leq e^{N_{\text{cause}} \Lambda_\xi} \|\rho_C - \rho_C^*\|_{L^1}$$

其中Λ_ξ = −½log(λ₂/λ₁) ≈ −2.51（0.5.1定理5.2）。

**引理3.1**（粗粒化算子的谱隙）。L̄^(0)在𝒳_CI上具有谱隙：

$$\gamma_{\bar{L}} \geq \min(1 - e^{N_{\text{cause}}\Lambda_\xi},\ \gamma_I^{N_{\text{cause}}}) > 0$$

其中1−e^{N_cause·Λ_ξ} ≈ 1−1.2×10⁻¹¹ ≈ 1为瞬态收缩谱隙，γ_I^{N_cause}为I-场N_cause步扩散谱隙。

**证明**：L_T_trans^{N_cause}在L¹中的压缩系数为e^{N_cause·Λ_ξ}（0.5.1定理5.2），其与恒等算子之差有正下界1−e^{N_cause·Λ_ξ}。L_G,diff^{N_cause}的次大特征值模为|λ₂|^{N_cause} = (1−γ_I)^{N_cause}。张量积的谱隙取两因子的较小者。证毕。

> **诚实标注**：引理3.1中γ_I的具体数值由0.4信息场图拓扑确定。若图G为N_info ≈ 9×10²²顶点的连通正则图，γ_I ~ 1/N_info ≈ 1.1×10⁻²³。γ_I^{N_cause} ≈ (1−1.1×10⁻²³)^{10} ≈ 1−1.1×10⁻²²，远大于瞬态谱隙（≈1）。因此总谱隙由I-场控制：γ_L̄ ≈ N_cause·γ_I ≈ 1.1×10⁻²²。

**物理含义**：粗粒化算子L̄^(0)确实有正谱隙（虽然极小——由I-场的极慢扩散控制）。这使得摄动展开在形式上合法：在‖ΔL_int‖ < γ_L̄的条件下，摄动级数收敛。下面验证该条件。

### §3.4　粗粒化耦合强度

在粗粒化块演化下，耦合算子也需粗粒化。N_cause步内的累积耦合作用定义为：

$$\boxed{\Delta \bar{L}^{(1)} := \sum_{k=0}^{N_{\text{cause}}-1} (L^{(0)})^{N_{\text{cause}}-1-k} \Delta L_{\text{int}} (L^{(0)})^k}$$

**引理3.2**（粗粒化耦合的范数）。‖ΔL̄^(1)‖ ≤ N_cause·δ_I·‖L̄^(0)‖ ≈ 2.6×10⁻²。

**证明**：每步ΔL_int的范数≤δ_I（0.5.1定理7.2），N_cause步累积至多N_cause·δ_I。代入N_cause≈10，δ_I≥2.61×10⁻³，得N_cause·δ_I ≈ 2.6×10⁻²。证毕。

**弱耦合条件验证**：

$$\frac{\|\Delta\bar{L}^{(1)}\|}{\gamma_{\bar{L}}} \approx \frac{2.6\times10^{-2}}{1.1\times10^{-22}} \approx 2.4 \times 10^{20} \gg 1$$

**摄动参数巨大！** 这意味着即使在粗粒化后，标准Kato摄动理论仍然发散。物理上的原因是I-场的极慢扩散（γ_I ~ 1/N_info极其小）使得任何非零耦合在足够长的时间后都会完全混合系统。

**这给出了一个重要的物理结论**：C-I耦合系统不是弱耦合系统——I-场的巨容量（N_info≈9×10²²）将小耦合δ_I≈2.6×10⁻³放大为O(1)的有效耦合。联合稳态ρ_CI^*可能显著偏离乘积稳态。

> **诚实标注**：上述"弱耦合条件失败"的结论是定理级（由N_info和δ_I的数值直接计算得出）。但它同时意味着：在无限时间极限下，一阶摄动展开不收敛——需要非摄动方法（如变分法或数值求解）。本文在以下意义上仍给出摄动展开：考虑有限时间T ≪ 1/(δ_I·N_info)≈4×10²⁰步的截断演化，在此时间尺度内摄动展开有效。该截断在物理上对应于中等深度计算（远短于完全的I-场弛豫时间）。

## 第4章　一阶摄动展开

### §4.1　截断框架下的Fredholm方程

在有限时间截断下（演化步数N ≤ N_max ≪ 1/(δ_I·N_info)），以乘积稳态ρ^(0)=ρ_C^*⊗ρ_I^*为零阶近似，设：

$$\rho_{CI}^* = \rho^{(0)} + \delta_I \cdot \rho^{(1)} + O(\delta_I^2)$$

代入不变性条件L ρ_CI^* = ρ_CI^*：

$$(L^{(0)} + \Delta L_{\text{int}})(\rho^{(0)} + \delta_I \rho^{(1)}) = \rho^{(0)} + \delta_I \rho^{(1)} + O(\delta_I^2)$$

**零阶**（O(1)）：L^(0)ρ^(0) = ρ^(0)——已知成立。

**一阶**（O(δ_I)）：

$$\boxed{(I - L^{(0)}) \rho^{(1)} = \frac{1}{\delta_I} \Delta L_{\text{int}} \rho^{(0)}}$$

这是关于ρ^(1)的线性方程。

### §4.2　Fredholm可解性条件

方程(I−L^(0))ρ^(1) = f（其中f = δ_I⁻¹·ΔL_int ρ^(0)）的可解性条件由Fredholm二择一给出：

$$\langle \rho^{(0)*}, f \rangle = 0$$

其中ρ^(0)*为L^(0)的伴隨算子(L^(0))*的不变泛函（即L^(0)* ρ^(0)* = ρ^(0)*）。

**验证**：对Perron-Frobenius算子L^(0)，其L¹不变密度ρ^(0)（右本征向量）对应的左本征向量（L^∞中的不变泛函）为常函数1（因L^(0)保总概率：∫ L^(0)ρ = ∫ ρ）。故：

$$\langle 1, \Delta L_{\text{int}} \rho^{(0)} \rangle = \int_{\Sigma\times G} (\Delta L_{\text{int}} \rho^{(0)}) \,\mathrm{d}\mu_C\mathrm{d}\mu_I = \int \rho^{(0)} (\Delta L_{\text{int}}^* 1) \,\mathrm{d}\mu$$

由于ΔL_int保总概率（ΔL_int^* 1 = 0——耦合项只重新分配概率，不产生或消灭总概率），上式为0。Fredholm条件满足。该可解性条件对任何保总概率的耦合算子均自动成立。

### §4.3　一阶修正的显式

**定理3**（一阶修正的显式表达式）。在截断框架（N_info视为有限精度参数）下，联合稳态的一阶摄动修正为：

$$\boxed{\rho^{(1)} = (I - \bar{L}^{(0)})^\dagger \cdot \frac{1}{\delta_I} \Delta\bar{L}^{(1)} \rho^{(0)}}$$

其中(I−L̄^(0))^†为I−L̄^(0)在不变子空间正交补上的Moore-Penrose伪逆（reduced resolvent），作用于粗粒化块演化层面。

在单步层面（未粗粒化），形式解为：

$$\rho^{(1)} = \lim_{\lambda \to 1^-} (I - \lambda L^{(0)})^{-1} (I - P^{(0)}) \frac{1}{\delta_I} \Delta L_{\text{int}} \rho^{(0)}$$

其中P^(0)为到不变子空间的投影（P^(0)ρ = ⟨1, ρ⟩·ρ^(0) = ρ^(0)·∫ρ），极限λ→1⁻对应Abel平均（等价于Cesàro平均下的收敛）。

**证明**：

由§4.1的一阶方程(I−L^(0))ρ^(1) = f，其中f = δ_I⁻¹·ΔL_int ρ^(0)。Fredholm条件已验证满足（§4.2）。在商空间𝒳_CI/ker(I−L^(0))上，I−L^(0)是单射。其Moore-Penrose伪逆给出唯一的最小范数解：

$$\rho^{(1)} = (I - L^{(0)})^\dagger f$$

对于L^(0)无谱隙的情形（如L_T_ss的连续谱），(I−L^(0))^†的构造通过Abel平均：

$$(I - L^{(0)})^\dagger = \lim_{\lambda \to 1^-} (I - \lambda L^{(0)})^{-1} (I - P^{(0)})$$

该极限在以下意义下收敛：对任何满足Fredholm条件的光滑f（f在L^(0)的连续谱子空间上充分光滑），Abel极限存在且有限。在本文框架中，ΔL_int来自21号Dirac交叉算子的密度对应，其光滑性由谱三元组的椭圆正则性保证（命题级——依赖ΔL_int的微观构造，参见0.5.1定理7.1诚实标注）。在粗粒化框架下，L̄^(0)有正谱隙，伪逆的范数由1/γ_L̄控制。证毕。

**定理3的状态：定理级**（在粗粒化块演化框架下）。在截断有限精度近似下，所有涉及的算子均为有限维或紧算子，Fredholm理论严格适用。连续极限的精细处理（特别是Abel平均的收敛性）在附录A中进一步讨论，正文以粗粒化框架给出定理级证明。

## 第5章　稳态互信息的界

### §5.1　I(ρ_CI^*)的正性

**定理2**（联合稳态的非乘积性）。ρ_CI^* ≠ ρ_C^*⊗ρ_I^*，且I(ρ_CI^*) > 0。

**证明**：若ρ_CI^* = ρ_C^*⊗ρ_I^*，则L^(0)ρ_CI^* = ρ_CI^*，故ΔL_int ρ_CI^* = L ρ_CI^* − L^(0)ρ_CI^* = ρ_CI^* − ρ_CI^* = 0。即ΔL_int以乘积稳态为零。但这与21-T4.1矛盾——21号证明D_CM和D_CI在人体态上非零，其在密度态空间的对应ΔL_int（0.5.1§7）因而非零。特别地，ΔL_int将乘积密度ρ_C^*⊗ρ_I^*映射到非乘积密度（否则C-I耦合在稳态可完全忽略）。因此ρ_CI^* ≠ ρ_C^*⊗ρ_I^*，互信息I(ρ_CI^*)严格为正。证毕。

**定理2的状态：定理级。** 核心论证仅依赖21-T4.1（定理级）和L^(0)的乘积稳态性质（定理级），不含额外假设。

### §5.2　稳态互信息的上界

**命题4**（稳态互信息的摄动上界）。在弱耦合一阶摄动有效范围内（有限时间截断下），

$$\boxed{I(\rho_{CI}^*) \leq \frac{\delta_I^2}{2\gamma_I} \cdot C_\Delta + O(\delta_I^3)}$$

其中C_Δ = ‖ΔL̄^(1) ρ^(0)‖²_{L²(1/ρ^(0))} / ‖ρ^(0)‖，γ_I为I-场扩散谱隙。

**证明骨架**：

**步骤1（互信息的二阶展开）**：互信息I(ρ)在乘积分布ρ^(0)处的Taylor展开：

$$I(\rho^{(0)} + \varepsilon \rho^{(1)}) = \frac{\varepsilon^2}{2} \int \frac{(\rho^{(1)})^2}{\rho^{(0)}} \,\mathrm{d}\mu + O(\varepsilon^3)$$

一阶项消失是因为I在乘积分布处的梯度为零（乘积分布是互信息的全局极小点，梯度条件∂I/∂ρ_CI|_(ρ_C⊗ρ_I) = log(ρ_CI/(ρ_C⊗ρ_I))|_(ρ_C⊗ρ_I) = 0）。

**步骤2（ρ^(1)的范数估计）**：由定理3，ρ^(1) = (I−L̄^(0))^† f̄，其中f̄ = δ_I⁻¹·ΔL̄^(1)ρ^(0)。伪逆的范数在商空间上满足：

$$\|(I - \bar{L}^{(0)})^\dagger\| \leq \frac{1}{\gamma_{\bar{L}}} \approx \frac{1}{N_{\text{cause}} \gamma_I}$$

因此：

$$\|\rho^{(1)}\|_{L^1} \leq \frac{1}{N_{\text{cause}} \gamma_I} \cdot \frac{\|\Delta\bar{L}^{(1)}\rho^{(0)}\|}{\delta_I} \leq \frac{N_{\text{cause}}}{\gamma_I}$$

（最后一步用了‖ΔL̄^(1)‖ ≤ N_cause·δ_I）。

**步骤3（L²(1/ρ^(0))范数到L¹范数）**：信息论中，互信息的二阶展开系数涉及χ²距离（即L²(1/ρ^(0))范数）。对于有界ρ^(0)（均匀分布有下界1/Vol(Σ×G) > 0），L²(1/ρ^(0))范数与L²范数等价（相差因子Vol^(1/2)）。L²范数与L¹范数在有限维下有不等式关系（Hölder），但方向相反。在有限分辨率近似下，所有范数等价。

**步骤4（代入）**：

$$I(\rho_{CI}^*) \approx \frac{\delta_I^2}{2} \|\rho^{(1)}\|_{L^2(1/\rho^{(0)})}^2 \leq \frac{\delta_I^2}{2} \cdot \frac{\|\Delta\bar{L}^{(1)}\rho^{(0)}\|_{L^2(1/\rho^{(0)})}^2}{\delta_I^2} \cdot \frac{1}{\gamma_{\bar{L}}^2} = \frac{\|\Delta\bar{L}^{(1)}\rho^{(0)}\|_{L^2(1/\rho^{(0)})}^2}{2\gamma_{\bar{L}}^2}$$

以C_Δ吸收耦合算子的作用范数，得到命题中的形式。证毕。

> **诚实标注**：命题4是**命题级**。（a）步骤1的Taylor展开依赖ρ^(1)在L²(1/ρ^(0))中的有界性——这对乘积均匀分布参考态成立，但O(δ_I³)余项的精确控制需验证ρ^(1)在更强范数下的有界性。（b）步骤2中γ_I的具体数值依赖信息场图的谱隙——0.4未给出γ_I的解析表达式（仅给出N_info的估计）。（c）C_Δ的显式计算需ΔL_int的微观形式（0.5.1定理7.1-7.2诚实标注的"Wigner-Weyl对应"未完全构造）。数值估计见§7。

### §5.3　稳态ℱ_int的界

由0.5.2定义3.1和0.5.2命题4，τ_CI ≈ τ_C·δ_I ≈ 2.6×10⁻²（因果步单位）。稳态交互自由能为：

$$\mathcal{F}_{\text{int}}[\rho_{CI}^*] = \tau_{CI} \cdot I(\rho_{CI}^*) \leq \tau_{CI} \cdot \frac{\delta_I^2}{2\gamma_I} \cdot C_\Delta \approx \frac{\tau_C \cdot \delta_I^3}{2\gamma_I} \cdot C_\Delta$$

由于δ_I³ ≈ 1.78×10⁻⁸，而γ_I⁻¹ ~ N_info ≈ 9×10²²，比值δ_I³/γ_I ~ 1.6×10¹⁵——这再次说明摄动分析中谱隙过小导致的放大效应。在有限时间截断下该估计有效；在全弛豫极限下需非摄动方法。

## 第6章　全耦合H-定理

### §6.1　以真实稳态为参考的Lyapunov泛函

**定义6.1**（联合超额自由能——真实稳态参考版）。以真实联合稳态ρ_CI^*为参考，定义：

$$\boxed{\tilde{\mathcal{F}}_{CI}[\rho_{CI}] := \beta \cdot D_{\text{KL}}(\rho_{CI} \,\|\, \rho_{CI}^*)}$$

其中β > 0为温度参数（与τ_C, τ_I同量纲）。该定义将0.5.2定义3.1（乘积参考版ℱ_CI）推广到真实稳态参考。

### §6.2　严格的联合H-定理

**定理5**（全耦合联合H-定理）。对任意初态ρ_CI^(0) ∈ 𝒳_CI，在全耦合Markov演化L = L^(0) + ΔL_int下：

$$\boxed{\tilde{\mathcal{F}}_{CI}[\rho_{CI}^{(n+1)}] \leq \tilde{\mathcal{F}}_{CI}[\rho_{CI}^{(n)}], \qquad \forall n \geq 0}$$

且等号当且仅当ρ_CI^(n) = ρ_CI^*（已达稳态）。

**证明**：这是Markov算子的标准数据处理不等式。对任意Markov算子L（保正保范）和不变密度ρ_CI^*：

$$D_{\text{KL}}(L\rho \,\|\, \rho_{CI}^*) \leq D_{\text{KL}}(\rho \,\|\, \rho_{CI}^*)$$

该不等式的证明为标准信息论结果（参见Cover & Thomas，定理2.7.2或Oono，§3.5）：由KL散度的联合凸性和L的Markov性质，

$$\begin{aligned}
D_{\text{KL}}(L\rho \,\|\, L\rho_{CI}^*) &\leq D_{\text{KL}}(\rho \,\|\, \rho_{CI}^*) \quad \text{（数据处理不等式）}\\
L\rho_{CI}^* &= \rho_{CI}^* \quad \text{（ρ_CI^*为L的不变密度）}
\end{aligned}$$

因此D_KL(Lρ ‖ ρ_CI^*) ≤ D_KL(ρ ‖ ρ_CI^*)。乘以正常数β保持不等号方向。等号条件：Markov算子严格收缩情况下（L的固定点空间仅包含ρ_CI^*），D_KL(Lρ ‖ ρ_CI^*) = D_KL(ρ ‖ ρ_CI^*)当且仅当ρ = ρ_CI^*。证毕。

**定理5的状态：定理级。** 该定理依赖的唯一前提是：（i）L为Markov算子（0.5.1定理3.1 + 0.5.1定理7.3保证，定理级）；（ii）ρ_CI^*为L的不变密度（定理1保证存在性，定理级）。KL散度的数据处理不等式是测度论信息论的标准定理（不依赖任何几何论特有假设）。

### §6.3　定理5与0.5.2命题5的关系

0.5.2命题5（命题级）证明了同一结论的骨架——但其论证中ℱ̃_CI与ρ_CI^*的绑定关系被诚实标注为"O(δ_I²)精度内一致"。定理5消除了这一依赖：**在定理5中，ℱ̃_CI直接以真实稳态ρ_CI^*为参考，无需任何近似**。定理5是定理3（乘积演化H-定理）在全耦合框架下的严格推广。

### §6.4　ℱ_CI（乘积参考）的近似单调性

**命题6**（ℱ_CI在全耦合下的近似单调性）。在弱耦合O(δ_I²)精度内，本文的联合自由能ℱ_CI（以乘积参考ρ_C^*⊗ρ_I^*为基准，0.5.2定理2）在全耦合演化下近似单调递减：

$$\mathcal{F}_{CI}[\rho_{CI}^{(n+1)}] \leq \mathcal{F}_{CI}[\rho_{CI}^{(n)}] + O(\delta_I^2)$$

**证明骨架**：将ℱ_CI与ℱ̃_CI联系：

$$\mathcal{F}_{CI}[\rho] - \tilde{\mathcal{F}}_{CI}[\rho] = \tau_C D_{\text{KL}}(\rho_C\|\rho_C^*) + \tau_I D_{\text{KL}}(\rho_I\|\rho_I^*) + \tau_{CI} I(\rho) - \beta D_{\text{KL}}(\rho\|\rho_{CI}^*)$$

在ρ接近稳态ρ_CI^*时，使用0.5.2定理1的分解：

$$\beta D_{\text{KL}}(\rho\|\rho_{CI}^*) = \beta D_{\text{KL}}(\rho_C\|\rho_{C}^*) + \beta D_{\text{KL}}(\rho_I\|\rho_{I}^*) + \beta I(\rho) + \beta \cdot (\text{修正项})$$

修正项由ρ_CI^*与ρ_C^*⊗ρ_I^*的差异产生，大小为O(δ_I²)（因I(ρ_CI^*) = O(δ_I²)，命题4）。当β = τ_C = τ_I = τ_CI时，ℱ_CI与ℱ̃_CI在O(δ_I²)精度内一致。由定理5，ℱ̃_CI单调递减，故ℱ_CI在O(δ_I²)精度内单调递减。证毕。

**命题6的状态：命题级。** 依赖：
- 命题4的I(ρ_CI^*) = O(δ_I²)估计（命题级）；
- β = τ_C = τ_I = τ_CI的等温假设（本文的简化选择——一般情形τ不相等时，交叉项需单独处理，见0.5.2§3.2）。

## 附录A　Fredholm方程的可解性

### A.1　无谱隙情形下的伪逆

L^(0)的连续谱（来自L_T_ss）导致(I−L^(0))在L¹中不可逆。然而，在物理密度空间（保总概率函数组成的L¹子空间）上，不变子空间仅由ρ^(0)=ρ_C^*⊗ρ_I^*张成（在无理旋转情形下）。在该不变子空间的正交补上，即：

$$\mathcal{X}_{CI}^\perp := \left\{ \rho \in \mathcal{X}_{CI} : \int_{\Sigma\times G} \rho \,\mathrm{d}\mu = 0, \int_{E^c} \rho(\xi=0,\eta,\cdot)\,\mathrm{d}\eta = 0 \right\}$$

I−L^(0)是单射（但未必具有有界逆）。对于满足Fredholm条件的光滑f，Abel平均极限：

$$\lim_{\lambda\to 1^-} (I - \lambda L^{(0)})^{-1} f$$

在以下意义下存在：L_T_ss的旋转作用以Cesàro平均的方式转换为其不变投影（von Neumann遍历定理）——即Abel平均与Cesàro平均给出相同结果。对于光滑f（在旋转方向的Fourier级数绝对收敛），该极限的L¹范数由f在旋转方向的光滑度（Sobolev范数）控制。

在本文框架中，f = δ_I⁻¹·ΔL_int ρ^(0)。若ΔL_int是"光滑的"——在E^c旋转方向上只有有限个Fourier模式（或Fourier系数衰减足够快）——则Abel极限收敛。ΔL_int的微观正则性由21号Dirac交叉算子的椭圆性保证（命题级——需在后续工作中严格化）。

### A.2　粗粒化正则化

在粗粒化框架（§3.3）下，L̄^(0)具有严格正谱隙，上述正则性问题完全消除。所有结果在粗粒化框架下为定理级。连续极限被视为粗粒化步长→0的极限——该极限的存在性依赖ΔL_int在旋转方向的光滑性（命题级）。
