# 一、几何膨胀的内禀定义

## 1.1　物质空间的大小：$\sin^3\theta_M$

在几何论中，不存在外部时间参数，也不存在预先给定的空间度规。所有几何资产均在几何约束截面 $\Sigma$ 上：三个角度 $\theta_M, \theta_C, \theta_I$，几何约束 $\theta_M+\theta_C+\theta_I=90^\circ$，局部坐标 $(\xi, \eta)$，以及 Hessian 矩阵 $H_{ij}$（0.1 §7）。可观测的宇宙膨胀，由这些角度构型在截面上的位置变化描述。

物理三维空间是物质场 $\mathcal{M}$ 的投影。$\mathcal{M}$ 在全局背景中的投影体积元（0.1 §4.1）：

$$\mathrm{d}V_{\text{phys}} = \sin^3\theta_M \cdot \mathrm{d}V_M$$

宇宙的大小正比于 $\sin^3\theta_M$。膨胀就是 $\theta_M$ 发生演化，导致 $\sin^3\theta_M$ 变化。

## 1.2　几何约束截面上的内禀几何时间

几何约束截面 $\Sigma$ 的线元由 Hessian 矩阵决定。在原始角度坐标 $(\theta_M, \theta_C, \theta_I)$ 下，Hessian 分量为（0.1 附录 A）：

$$H_{11}=31.30,\quad H_{22}=367.73,\quad H_{33}=59259.35\ \text{rad}^{-2}$$

由谱刚性定理，Hessian 正定，其本征值 $\lambda_1=392.21\ 	ext{rad}^{-2}$（软模）、$\lambda_2=58760.77\ 	ext{rad}^{-2}$（硬模）构成内禀度规。在 $(\xi, \eta)$ 本征基下，线元严格对角化：

$$\mathrm{d}\mathcal{T}^2 = \lambda_2 \mathrm{d}\xi^2 + \lambda_1 \mathrm{d}\eta^2$$

其中 $\xi$ 为硬模方向（信息场编码），$\eta$ 为软模方向（物质场演化）。本征基下交叉项严格为零：

$$H_{\xi\eta}=0$$

几何时间 $\mathcal{T}$ 是几何约束截面上沿实际演化路径的无量纲几何弧长。$\mathcal{T}$ 不是物理秒，而是截面上的内禀演化步数，由角度坐标的变化经 Hessian 结构加权累积而成。

## 1.3　内禀几何膨胀率 $\mathcal{H}$

在几何约束截面上定义：

$$\mathcal{H} \equiv \frac{1}{\sin^3\theta_M} \cdot \frac{\mathrm{d}(\sin^3\theta_M)}{\mathrm{d}\mathcal{T}}$$

计算导数：

$$\frac{\mathrm{d}(\sin^3\theta_M)}{\mathrm{d}\mathcal{T}} = 3\sin^2\theta_M \cos\theta_M \cdot \frac{\mathrm{d}\theta_M}{\mathrm{d}\mathcal{T}}$$

因此：

$$\mathcal{H} = 3\cot\theta_M \cdot \frac{\mathrm{d}\theta_M}{\mathrm{d}\mathcal{T}}$$

这就是几何论的无量纲内禀膨胀率——不涉及外部时空度规，仅由物质场角和沿无量纲几何弧长的演化速率决定。

## 1.4　用几何约束局部坐标表达

由几何约束反解（0.1 §6）：

$$\theta_M = \theta_M^0 - \eta$$

因此：

$$\frac{\mathrm{d}\theta_M}{\mathrm{d}\mathcal{T}} = -\frac{\mathrm{d}\eta}{\mathrm{d}\mathcal{T}}$$

代入得：

$$\mathcal{H} = -3\cot\theta_M \cdot \frac{\mathrm{d}\eta}{\mathrm{d}\mathcal{T}}$$

关键洞察：膨胀率由 $\eta$ 方向的几何时间导数决定。$\xi$ 方向不直接贡献物质场投影变化。

# 二、几何时间的实际演化方向

## 2.1　弧长元的对角化

在 $(\xi, \eta)$ 本征基下，Hessian 已严格对角化：

$$\mathrm{d}\mathcal{T}^2 = \lambda_2 \mathrm{d}\xi^2 + \lambda_1 \mathrm{d}\eta^2$$

## 2.2　路径A：$\eta$ 方向的软模演化（物质场膨胀通道）

当演化在 $\eta$ 方向进行（$\mathrm{d}\xi=0$）：

$$\mathrm{d}\mathcal{T}_\eta = \sqrt{\lambda_1}\ |\mathrm{d}\eta| = \sqrt{392.21}\ |\mathrm{d}\eta| = 19.8043\ |\mathrm{d}\eta|$$

因此：

$$\left|\frac{\mathrm{d}\eta}{\mathrm{d}\mathcal{T}_\eta}\right| = \frac{1}{19.8043} = 0.050494$$

对应膨胀率（$\theta_M^e=57.9300000000^\circ$）：

$$\cot\theta_M^e = \frac{\cos 57.93^\circ}{\sin 57.93^\circ} = \frac{0.5309549547}{0.8474000449} = 0.626569$$

$$\mathcal{H}_\eta = -3 \times 0.626569 \times 0.050494 = -0.09491\ \text{（无量纲）}$$

这是 $\eta$ 软模方向的无量纲内禀膨胀率。负号表示膨胀方向是 $\theta_M$ 减小的方向（从裸基准向电子基态演化）。

## 2.3　路径B：$\xi$ 方向的硬模演化（信息场编码通道）

当演化在 $\xi$ 方向进行（$\mathrm{d}\eta=0$）：

$$\mathrm{d}\mathcal{T}_\xi = \sqrt{\lambda_2}\ |\mathrm{d}\xi| = \sqrt{58760.77}\ |\mathrm{d}\xi| = 242.406\ |\mathrm{d}\xi|$$

此时：

$$\frac{\mathrm{d}\theta_M}{\mathrm{d}\mathcal{T}_\xi} = 0 \quad \text{（因为 } \theta_M=\theta_M^0-\eta \text{ 不依赖 } \xi\text{）}$$

因此：

$$\mathcal{H}_\xi = 0\ \text{（无量纲）}$$

纯 $\xi$ 方向的移动不产生物质场膨胀。这是关键——信息场方向的演化不改变物质空间的大小。

# 四、差异量级的推导

## 4.1　直接耦合的分析

在 $(\xi, \eta)$ 本征基下，Hessian 矩阵严格对角化：

$$H_{\xi\eta}=0$$

**直接耦合严格为零**：几何约束截面上 $\xi$ 和 $\eta$ 方向在本征基下完全解耦。这意味着信息场硬模对物质场软模膨胀的拖拽，不可能通过 Hessian 交叉项的线性力学传递实现。拖拽必须通过更高阶的机制——全息屏 $\Sigma_\eta$ 的全局编码结构。

## 4.2　共同基础与差异来源的分解

定义两个过程：

**过程 A（共同基础）**：信息场窗口的分辨率限制

$$\text{分辨率因子} = \sqrt{\frac{\lambda_1}{\lambda_2}} = \sqrt{\frac{392.21}{58760.77}} = \frac{1}{\sqrt{150}} \approx 0.0816$$

该因子出现在两种测量的共同基础中——晚期测量通过 $\eta$ 软模窗口观测，早期测量通过 $\xi$ 硬模窗口读取。两者共享同一分辨率限制，因此该因子不影响比值，只影响绝对数值。

**过程 B（差异来源）**：$\tau_{\text{cmb}}$ 时全息屏的信息展开状态

CMB 光子退耦时，信息场仅部分展开。晚期测量对应今天（$\tau=1$，信息场完全展开）。两者的差异来自信息场展开度的不同。

## 4.3　几何时间与信息完整度

定义几何时间 $\tau$ 为信息场展开的归一化进度：

$$\tau \equiv \frac{\xi}{\xi_e}$$

$\tau=0$：裸基准态，信息场冻结。$\tau=1$：电子基态，信息场展开。

定义全息屏信息完整度：

$$\mathcal{I}(\tau) \equiv \frac{\sum_{n=1}^7 \varepsilon_n(\tau) \ln N_n}{\ln M_7}$$

其中 $M_7 = \prod_{n=1}^7 N_n$ 为七级总编码容量，$\ln M_7 = \sum_{n=1}^7 \ln N_n = 97.2677$（0.2.2），$\varepsilon_n(\tau)$ 为第 $n$ 层在几何时间 $\tau$ 时的展开度（$0\leq\varepsilon_n\leq1$）。

## 4.4　$\varepsilon_n(\tau)$ 的连续形式

**信息场展开的连续化方程**：几何约束截面上，总作用量 $S(\xi)$ 是信息场位移 $\xi$ 的连续函数。将 $\theta_M=\theta_M^0$、$\theta_C=\theta_C^0-\xi$、$\theta_I=\theta_I^0+\xi$ 代入六项公式：

$$S(\xi) = \frac{1}{\sin^2\theta_M^0} + \frac{1}{\sin^2(\theta_C^0-\xi)} + \frac{1}{\sin^2(\theta_I^0+\xi)} + \frac{1}{\sin\theta_M^0 \sin(\theta_C^0-\xi)} + \frac{1}{\sin\theta_M^0 \sin(\theta_I^0+\xi)} + \frac{1}{\sin(\theta_C^0-\xi)\sin(\theta_I^0+\xi)}$$

证明 $S(\xi)$ 的单调性（0.1 已知资产）：

$$\frac{\mathrm{d}S}{\mathrm{d}\xi} = \frac{\partial S}{\partial \theta_I} \cdot 1 + \frac{\partial S}{\partial \theta_C} \cdot (-1) = -2145.16 - (-71.24) = -2073.92\ \text{rad}^{-1}$$

在 $\xi\in[\xi_e, 0]$ 区间（$\xi_e=-1.7354\times10^{-5}\ 	ext{rad}$），$\mathrm{d}S/\mathrm{d}\xi=-2073.92<<0$ 为常数（领头阶）。因此 $S(\xi)$ 单调递减。当 $\xi$ 从 $0$ 减小到 $\xi_e$ 时，$S$ 从 $S_0=137.0000000009$ 单调增加到 $S_e=137.035999084$。

**层级激活的 $S$ 阈值**：七级递推中，第 $n$ 层激活对应总作用量达到累积阈值：

$$S_n = S_0 + \sum_{k=1}^n \Delta S_k$$

阈值 $S_n$ 将连续区间 $[S_0, S_e]$ 划分为七个子区间，每个子区间对应一个层级的激活过程。

**$\varepsilon_n(\tau)$ 的形式**：第 $n$ 层的展开度由总作用量相对于阈值的连续插值给出：

$$\varepsilon_n(\tau) = \max\left(0, \min\left(1, \frac{S(\tau \cdot \xi_e) - S_{n-1}}{\Delta S_n}\right)\right)$$

当 $S < S_{n-1}$：分子为负，$\max(0,\cdots)=0$，第 $n$ 层尚未激活

当 $S > S_n$：$\min(1,\cdots)=1$，第 $n$ 层展开完成

由于 $S(\xi)$ 单调，上述分段定义覆盖所有 $\tau\in[0,1]$，且 $\varepsilon_n(\tau)$ 连续。

**线性近似的精确性**：由于 $S(\xi)$ 在 $\xi$ 方向的一阶主导性极强：

$$\text{二阶修正} = \frac{1}{2} \lambda_2 \xi_e^2 = \frac{1}{2} \times 58760.77 \times (1.7354\times 10^{-5})^2 \approx 8.84\times 10^{-6}$$

$$\text{一阶主导} = \left|\frac{\mathrm{d}S}{\mathrm{d}\xi} \cdot \xi_e\right| = 2073.92 \times 1.7354\times 10^{-5} \approx 0.03599$$

$$\text{非线性度} = \frac{8.84\times 10^{-6}}{0.03599} \approx 2.5\times 10^{-4} = 0.025\%$$

$$\varepsilon_n(\tau) = \min(1, \tau/\tau_n) + O(10^{-4})$$

## 4.6　全息屏编码截断与 $\sqrt{\mathcal{I}}$ 的维度论证

**步骤 1：全息屏的二维编码结构（0.2.2 全息宇宙）**

全息屏 $\Sigma_\eta$ 分形维数 $D_f=2.000$。编码面密度恒定 $\rho_n=N_1/A_1$（$A_1=27.06$ 已由 0.3.1 定义）。全息屏总面积 $A_{\text{holo}}$ 由七级展开后的结构固定。

**步骤 2：有效编码密度随 $\tau$ 演化**

有效编码数：$N_{\text{eff}}(\tau)=\mathcal{I}(\tau)\cdot M_7$

有效编码面密度：$\rho(\tau)=N_{\text{eff}}(\tau)/A_{\text{holo}}=\mathcal{I}(\tau)\cdot \rho_{\max}$

其中 $\rho_{\max}=M_7/A_{\text{holo}}$ 为展开时的最大面密度。

**步骤 3：从二维面密度到一维线性分辨率（维度约化）**

特征像素面积：$a_{\text{pixel}}(\tau) = 1/\rho(\tau) = a_{\text{pixel}}^{\max} / \mathcal{I}(\tau)$

特征像素边长（线性分辨率）：

$$\ell_{\text{pixel}}(\tau) = \sqrt{a_{\text{pixel}}(\tau)} = \frac{\ell_{\text{pixel}}^{\max}}{\sqrt{\mathcal{I}(\tau)}}$$

全息屏是二维编码面（$D_f=2.000$），$\xi$ 方向探针沿一维路径扫描该平面。探针的有效步长由像素边长决定，而像素边长与面密度的平方根成反比（$\sqrt{a} \propto 1/\sqrt{\rho}$）。这是从 2D 面密度到 1D 线分辨率的维度约化。

**步骤 4：从分辨率到有效膨胀率**

$\xi$ 方向探针通过读取全息屏编码来推算物质场膨胀率。探针的有效膨胀率与其单位步长内能读取的几何信息量成正比。信息量 $\propto$（步长内像素数），像素数 $\propto$（步长/$\ell_{\text{pixel}}$）$\propto \sqrt{\mathcal{I}}$。因此：

$$H_0^{\text{early}} \propto \sqrt{\mathcal{I}(\tau)}$$

$$H_0^{\text{late}} \propto 1\quad \text{（展开时 } \mathcal{I}=1\text{）}$$

$$f = \frac{H_0^{\text{early}}}{H_0^{\text{late}}} = \sqrt{\mathcal{I}(\tau_{\text{cmb}})}$$

边界条件验证：

- $\tau=1$（今天，信息场展开）：$\mathcal{I}(1)=1$，$f=1$ → 晚期测量，无差异
- $\tau<<1$（早期，信息场部分展开）：$\mathcal{I}(\tau)<1$，$f<<1$ → 早期测量，膨胀率被低估
- $\mathcal{I}\to0$（无编码）：$f\to0$ → 无法分辨膨胀
