## 第1章　引言与公理桥接

### §1.1　流体力学在几何论框架内的定位

几何约束截面理论（0.0.3–0.5系列）已建立从三公理到物理常数的零锚点映射。流体力学作为物质界 ℳ 在约束截面 Σ≈S² 上的连续介质极限，进入该框架是自然延伸。核心叙事如下：

物质界流 **u**∈TΣ 是信息场凝聚在 N_info→∞ 极限下涌现的连续切向量场。三分切丛 TΣ=ℳ⊕𝒞⊕ℐ 天然对应流体三大变量：速度（物质界 ℳ）、涡量（因果界 𝒞）、熵/温度（信息界 ℐ）。压力由六项作用量 S(θ) 的梯度驱动，粘性由 Hessian 硬模 λ₂ 的阻尼效应给出（公理3→0.2.1→Hessian本征值），密度由质量映射定理 m=K sin³θ_M 导出。

本文不引入新公理或新外部锚点。**三公理**（标准公理体系，来源：0.0.3 §2 + 0.0.6 §3.1）是唯一输入：

- **公理1（圆拓扑公理）**：参数空间为 $D = S^1 \setminus \{p_0, p_\infty\}$
- **公理2（边界极限公理）**：$S \to 0$（真空极限），$S \to +\infty$（信息退化极限）
- **公理3（全息屏编码条件）**：三分切丛 $TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$，二维全息屏 $\Sigma$ 上 $\theta_M + \theta_C + \theta_I = 90^\circ$

> **公理-定理区分**：六项作用量 $S = \sum 1/\sin^2\theta_i + \sum_{i<j} 1/(\sin\theta_i \sin\theta_j)$ 是由全息屏投影面积导出的**定义/构造**（0.0.6 定义 3.2），不是公理。质量映射 $m = K\sin^3\theta_M$（$K = 839.758793$ keV）是**定理 1.2（质量映射定理）**，由公理2–3导出（0.1 §1.1），不是公理。

### §1.3　三公理→流体方程的逐级调用链

> **本节新增（260626.6，260702.2修订公理编号）**：为响应审计报告关于"与三公理联系不够显式"的要求，以下列出全文推导中三公理的逐级调用路径。

**公理3（全息屏编码条件）θ_M+θ_C+θ_I=90°**：
- → 约束截面 Σ（2维带边流形，等边三角形内部）→ 流体定义域 S²
- → 内禀坐标 (ξ,η)（§2.1, 跟随0.4.1的规范选择）

**六项作用量（定义/构造）S = Σ1/sin²θ_i + Σ_{i<j}1/(sinθ_i sinθ_j)**：
- → Σ 上的势函数 V(ξ,η) → 几何压力 −∇S(θ)（§2.3 NS映射中驱动项）
- → Hessian度量 g_ab（0.2.1）→ 软硬模本征值 λ₁,λ₂ → 几何粘度 ν_geo=1/λ₂（§2.3）、几何惯性标度 √λ₁（§3.1）
- → 谱刚性 S_e → 升力归一化因子（§6.1）
- → S(θ)∼k^(−5/3) 渐近行为 → 能量级联谱幂律（§5.3）

**质量映射定理 m=K sin³θ_M**：
- → 物质界密度 ρ_ℳ=K sin³θ_M/χ_L³（§2.2）
- → 质量标度 K → 表面张力量纲转换（§8.1）
- → sin³θ_M 因子 → 声速投影 sinθ_M（§3.2）、升力面积投影（§6.1）

**无额外锚点声明**：上述链中所有物理量（速度、压力、粘度、密度、温度、长度、时间、力）均由 χ_L, χ_T, K 三个量纲桥基底与无量纲几何量组合唯一确定。光速 c 仅用于 χ_T=c·χ_L 的标度校准（0.3.1标准程序），非独立输入。
