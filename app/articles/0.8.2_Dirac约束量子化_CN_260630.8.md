## §1 策略

**核心思想**：将 M 场量子化视为约束系统的 Dirac-Bergmann 量子化问题。两个约束：公理1（$\phi_1$）和 $S$ 定义（$\phi_2$）。分析约束分类 → Dirac 括号 → 正则量子化。

---

## §2 未约化相空间

$$\mathcal{P}_0 = T^*(\mathcal{M}^3) = \{(\theta_i, p_i): i=M,C,I\}$$

典范泊松括号：

$$\{\theta_i, p_j\} = \delta_{ij}, \quad \{\theta_i,\theta_j\} = 0$$

---

## §3 约束分析

### 约束定义

$$\phi_1 = \theta_M + \theta_C + \theta_I - \frac{\pi}{2} \approx 0$$

$$\phi_2 = S(\theta_M,\theta_C,\theta_I) - S_e \approx 0$$

（第二式用于电子层；自由层用 $S - S_{\min}$）

### 定理 3.1（约束分类）

在未约化相空间 $\mathcal{P}_0$ 的典范括号下：

$$\{\phi_1,\phi_2\} = \sum_i \frac{\partial S}{\partial\theta_i} \cdot \frac{\partial\phi_1}{\partial p_i} - 0 = 0$$

因为 $\phi_1$ 不含动量。同理 $\{\phi_1,\phi_1\} = 0$，$\{\phi_2,\phi_2\} = 0$。

**结论**：$\phi_1$ 和 $\phi_2$ 在未约化相空间中均为**第一类**约束。

### 3.1 第一类约束的物理意义

- $\phi_1$：生成 $\Sigma$ 上的角度重参数化（U(1) 规范对称性）
- $\phi_2$：生成 $S$ 等值面内的平移

---

## §4 Dirac 括号

两个第一类约束需要规范固定以得到物理相空间。

### 规范固定

$$\chi_1 = p_I \approx 0 \quad \text{（固定 U(1) 规范）}$$

$$\chi_2 = p_s \approx 0 \quad \text{（固定 S-平移规范）}$$

### 约束矩阵

$$C = \begin{pmatrix} \{\phi_1,\chi_1\} & \{\phi_1,\chi_2\} \\ \{\phi_2,\chi_1\} & \{\phi_2,\chi_2\} \end{pmatrix}$$

其中 $\{\phi_1,\chi_1\} = \{\theta_I, p_I\} = 1$（非零，因为 $\phi_1$ 含 $\theta_I$），其余由具体形式决定。

### Dirac 括号定义

$$\{f,g\}_D = \{f,g\} - \sum_{a,b} \{f,\Phi_a\} (C^{-1})_{ab} \{\Phi_b, g\}$$

其中 $\Phi = (\phi_1,\phi_2,\chi_1,\chi_2)$。

在约束面 $\Sigma$ 上：

$$\{\theta_M,\theta_C\}_D = \frac{1}{A(\theta_M,\theta_C)}$$

$$A = \sum_i \frac{1}{\sin^2\theta_i}$$

**与 0.2.1 的 Hamilton 定理一致。**

### 详细推导

在约束面 $\Sigma$ 上，$\theta_I = 90^\circ - \theta_M - \theta_C$。计算 $\{\theta_M,\theta_C\}_D$：

$$\{\theta_M,\theta_C\}_D = \{\theta_M,\theta_C\} - \sum_{a,b} \{\theta_M,\Phi_a\} (C^{-1})_{ab} \{\Phi_b,\theta_C\}$$

由于典范括号下 $\{\theta_M,\theta_C\} = 0$，且只有包含 $p_M$ 或 $p_C$ 的约束（$\Phi$）才有非零贡献。在规范固定 $\chi_1 = p_I$、$\chi_2 = p_s$ 下，约束矩阵的逆给出有效 Poisson 括号。最终在约化后的 2 维物理相空间（坐标 $\theta_M,\theta_C$）上：

$$\{\theta_M,\theta_C\}_D = \frac{1}{A}$$

与 0.2.1 定理 3.5.1 的 Hamilton 定理给出的辛形式 $\omega = A d\theta_M \wedge d\theta_C$ 完全一致。$\square$

## §5 动量映射（部分构造）

### 构造 5.1（U(1) 动量映射候选）

$$\mu = p_I$$

满足 $\iota_{\xi}\omega = d\mu$，其中 $\xi = \partial/\partial\theta_I$ 是 $\phi_1$ 生成的规范向量场。

**证明**：$\iota_{\partial/\partial\theta_I} (d\theta_M\wedge dp_M + \cdots) = -dp_I = -d(-p_I) = d\mu$。∎

### 诚实标注

$\mu = p_I$ 是**数学上合法的动量映射**，但与物理量 $\tilde{S}_{\text{total}}$（5号附录C路径A）的对应关系仍需一个正则变换。标注为 O8.2.1。

## §6 量子化

### 命题 6.1（Dirac 量子化）

在 Dirac 括号 $\{\cdot,\cdot\}_D$ 下，正则量子化：

$$[\hat{\theta}_M,\hat{\theta}_C] = i\hbar_{\text{eff}} \{\theta_M,\theta_C\}_D = \frac{i\hbar_{\text{eff}}}{A}$$

基态能量：

$$E_0 = S(\theta_*) + \frac{\hbar_{\text{eff}}}{2A_*} + O(\hbar_{\text{eff}}^2)$$

**与出路 C 的星积 WKB 首阶一致。**

### 6.1 量子化的具体步骤

1. 在 Dirac 括号下，$(\theta_M,\theta_C)$ 成为正则共轭对（乘以 $A$ 因子）
2. 定义 Darboux 坐标 $Q = \sqrt{A}\theta_M$，$P = \sqrt{A}\theta_C$，满足 $\{Q,P\}_D = 1$
3. 提升为算子：$[\hat{Q},\hat{P}] = i\hbar_{\text{eff}}$
4. 由于 $[\hat{\theta}_M,\hat{\theta}_C] = i\hbar_{\text{eff}}/A \neq 0$，$(\hat{\theta}_M,\hat{\theta}_C)$ 不能同时对角化——需做 Bogoliubov 变换：

$$\hat{a} = \sqrt{\frac{A}{2}}(\hat{\theta}_M + i\hat{\theta}_C), \quad [\hat{a},\hat{a}^\dagger] = 1$$

基态涨落 $\langle 0|\hat{\theta}_M^2|0\rangle = 1/(2A)$，量子修正 $\Delta E_0 = \hbar_{\text{eff}}/(2A_*)$。
