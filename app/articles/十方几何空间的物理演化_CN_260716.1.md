# 第一部分　几何基础

## 第 1 章　公理体系与激发态参数空间

### 1.1　三条公理

10方几何空间建立于以下三条公理：

**公理 1（圆拓扑公理）**　激发态参数空间 $D = S^1\setminus\{p_0,p_*\}$ 由两个连通分支组成：
$$D_+\cong(0,1),\quad D_-\cong(0,1).$$

**公理 2（边界极限公理）**　几何量 $S: D_\pm\to(0,+\infty)$ 在每个分支上连续，且满足：
$$\lim_{x\to p_0}S(x)=0,\quad \lim_{x\to p_*}S(x)=+\infty.$$

**公理 3（全息屏编码条件）**　在三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个纤维 $\mathbb{R}^9$ 上，存在二维定向子空间 $\Sigma$（全息屏），三个三维扇区向其投影的强度角满足
$$\theta_1+\theta_2+\theta_3=90^\circ,\qquad \theta_i\in(0^\circ,90^\circ).$$

### 1.2　激发态参数空间的基本性质

**命题 1.1（圆拓扑）**　激发态参数空间 $D=S^1\setminus\{p_0,p_*\}$ 由两个连通分支组成：
$$D_+\cong(0,1),\quad D_-\cong(0,1).$$

**命题 1.2（几何量的分析性质）**　几何量 $S:D_\pm\to(0,+\infty)$ 在每个分支上连续，且满足：
$$\lim_{x\to p_0}S(x)=0,\quad \lim_{x\to p_*}S(x)=+\infty.$$

**定理 1.3（值域定理）**　$S$ 在每个分支上的值域为
$$S(D_\pm)=(0,+\infty).$$

**定理 1.4（存在性定理）**　对任意给定的正数 $S_0>0$，在每个分支 $D_+$ 与 $D_-$ 上各至少存在一个参数 $x_+\in D_+$ 与 $x_-\in D_-$，使得 $S(x_+)=S(x_-)=S_0$。

*证明*　由命题1.1，$D_\pm$ 同胚于 $(0,1)$，故连通。由命题1.2，$S$ 连续且边界极限分别为 $0$ 与 $+\infty$。根据连续函数的介值定理，对任意 $S_0\in(0,+\infty)$，水平集 $S^{-1}(S_0)$ 在每个分支内非空。$\square$

## 第 2 章　约束乘积球面与谱刚性

### 2.1　三分切丛置换群的刚性

**引理 2.0（三分切丛置换群的刚性）** 设 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 为三分切丛，$G$ 为其扇区置换对称群。若 $G$ 在三个扇区上的作用忠实且可迁，并包含任意两扇区对换，则 $G \cong S_3$，且 $S_3$ 是满足此条件的唯一有限群。

*证明.* 三个对象的忠实置换群必为 $S_3$ 的子群。可迁性要求轨道大小为 3，故 $|G| \geq 3$。$S_3$ 的可迁子群仅有 $A_3 \cong \mathbb{Z}_3$（循环）与 $S_3$ 本身。若要求包含对换（奇置换），则 $A_3$ 不足，唯一可能为 $S_3$。$\square$

**定理 2.0'（三分切丛扇区置换的刚性）** 设 $M = S^3 \times S^3 \times S^3$ 配备乘积度量，且存在切丛分解 $TM = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 使得三个扇区在度规层面完全等价（即任意置换诱导等距自同构）。则：
1. 该分解的扇区置换群 $G$ 必包含所有两扇区对换；
2. 满足 (1) 的有限群唯一同构于 $S_3$。

进而 $\Lambda(G)=|\text{Conj}(S_3)|=3$，$k_0(G)=[S_3:A_3]=2$ 是群论函数的必然输出。

*证明.* (1) 若三个扇区度规等价，则任意对换（如 $\mathcal{M}\leftrightarrow\mathcal{C}$）保持乘积度量不变，故诱导等距自同构。因此所有两扇区对换均属于 $G$。

(2) 由引理 2.0，含对换的可迁置换群只能是 $S_3$。因此 $G\cong S_3$ 不是人为选择，而是三分等价结构下非平凡置换的必然推论。$\square$

**命题 2.0（互锁函数的群论来源）** 对于三分切丛 $TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$ 的置换对称群 $G = S_3$：
$$\Lambda(S_3) = 3, \quad k_0(S_3) = 2.$$

*证明*　$S_3$ 的共轭类数为 $3$（单位元、对换、三轮换），故 $\Lambda(S_3)=3$。$S_3$ 的极大正规子群为交错群 $A_3$，指数 $[S_3:A_3]=2$，故 $k_0(S_3)=2$。$\square$

### 2.2　约束乘积球面

**定义 2.1（约束乘积球面）**　设 $a>0$ 为整体尺度因子，$\Lambda=\Lambda(S_3)=3$ 为三等分比例参数，$k_0=k_0(S_3)=2$ 为二分紧致性常数。称九维闭流形
$$M(a)=S^3(a)\times S^3(a/\sqrt{3})\times S^3(a/\sqrt{6})$$
在配备乘积度量时构成一个约束乘积球面（CPS）。

**定理 2.2（模空间参数化）**　在互锁常数 $\Lambda=3$、$k_0=2$ 锁定时，约束乘积球面类 $\{M(a)\}_{a>0}$ 的模空间与 $(0,\infty)$ 同胚，$a$ 是唯一的连续自由度。

*证明*　乘积度量由三个因子球面的半径 $(a,b,c)$ 决定。约束条件 $b=a/\sqrt{\Lambda}$、$c=a/\sqrt{\Lambda k_0}$ 将 $(a,b,c)$ 压缩至一维子流形 $(0,+\infty)_a$。不同的 $a$ 给出不等距的度量，因为体积 $V(a)\propto a^9$ 严格单调。故模空间同胚于 $(0,\infty)$。$\square$

### 2.3　谱刚性

**定理 2.3（谱刚性）**　设 $M(a)$ 与 $M(a')$ 为两个具有相同互锁常数的约束乘积球面。若它们的前三个非零 Laplace–Beltrami 特征值对应相等，则 $M(a)$ 与 $M(a')$ 等距。

*证明*　对 $S^3(r)$，Laplace–Beltrami 特征值为 $\lambda_{k}^{S^3}=k(k+2)/r^2$（$k\ge 1$）。乘积流形的特征值为各因子特征值之和。前三个非零特征值分别来自：
- 第一个因子第一激发态：$\lambda_1^\Delta = 3/a^2$
- 第二个因子第一激发态：$\lambda_2^\Delta = 3/(a^2/\Lambda) = 3\Lambda/a^2 = 9/a^2$（当 $\Lambda=3$）
- 第一个因子第二激发态：$\lambda_3^\Delta = 8/a^2$

在 $S_3$ 置换对称下，需对特征值排序。由 $\Lambda=3$、$k_0=2$，有 $3<8<9$，故：
$$\lambda_1^\Delta=\frac{3}{a^2},\quad \lambda_2^\Delta=\frac{8}{a^2},\quad \lambda_3^\Delta=\frac{9}{a^2}.$$

给定 $\lambda_1^\Delta$ 可唯一反解 $a=\sqrt{3/\lambda_1^\Delta}$。由比值约束 $\lambda_2^\Delta/\lambda_1^\Delta=8/3$ 与 $\lambda_3^\Delta/\lambda_1^\Delta=\Lambda=3$ 的自洽性，$a$ 被唯一确定。故等谱蕴含等距。$\square$

**定理 2.4（特征值反解）**　给定前三个非零特征值 $\lambda_1^\Delta,\lambda_2^\Delta,\lambda_3^\Delta$ 满足自洽性条件，尺度因子唯一确定为
$$a=\sqrt{\frac{3}{\lambda_1^\Delta}},\quad b=\sqrt{\frac{3}{\lambda_3^\Delta}},\quad c=\frac{a}{\sqrt{\Lambda k_0}}.$$
自洽性条件：$\lambda_2^\Delta/\lambda_1^\Delta=8/3$ 且 $\lambda_3^\Delta/\lambda_1^\Delta=\Lambda=3$。$\square$

## 第 3 章　全息屏几何

### 3.1　三分切丛与全息屏编码

**命题 3.1（三分切丛）**　在 $M(a)$ 上，切丛存在自然的直和分解
$$TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I},$$
其中各子空间分别为三个 $S^3$ 因子切丛的拉回。

**基本前提（全息屏存在性）**　在三分切丛的每个纤维 $\mathbb{R}^9$ 上，存在二维定向子空间 $\Sigma$（全息屏），三个三维扇区向其投影的强度角 $\theta_1,\theta_2,\theta_3\in(0^\circ,90^\circ)$。

**说明 3.1（$90^\circ$ 的逻辑地位）**　$90^\circ$ 作为公理体系的独立输入（公理 3），是二维全息屏完整编码三个扇区信息的最简线性规范。$\Lambda=\Lambda(S_3)=3$ 在 $S_3$ 对称点 $\theta_1=\theta_2=\theta_3$ 处解释三等分 $30^\circ$ 的群论来源，与 $3\times30^\circ=90^\circ$ 精确兼容，但不可反推 $90^\circ$——后者是公理 3 的独立内容。

### 3.2　六项作用量

**定义 3.4（六项作用量）**　在公理 3 的 $90^\circ$ 约束下，定义
$$S(\theta_1,\theta_2,\theta_3)=\sum_{i=1}^3\frac{1}{\sin^2\theta_i}+\sum_{i<j}\frac{1}{\sin\theta_i\sin\theta_j}.$$

**定理 3.5（值域定理）**　在定义域 $D_\theta=\{(\theta_1,\theta_2,\theta_3)\mid\theta_i>0,\theta_1+\theta_2+\theta_3=90^\circ\}$ 上，六项作用量的值域为
$$S(D_\theta)=[24,+\infty).$$
最小值 $24$ 由对称点 $\theta_1=\theta_2=\theta_3=30^\circ$ 实现。

*证明*　定义域 $D_\theta$ 是 $\mathbb{R}^3$ 中二维单形的内部。函数 $S(\theta)$ 是严格凸函数 $\csc^2\theta_i$ 与 $\csc\theta_i\csc\theta_j$ 的正系数线性组合，故在凸集 $D_\theta$ 上严格凸。严格凸函数在紧集闭包上的最小值唯一，且在内部临界点取得。由对称性，唯一临界点为 $\theta_0=(30^\circ,30^\circ,30^\circ)$。代入得：
$$S(\theta_0)=3\cdot\frac{1}{(1/2)^2}+3\cdot\frac{1}{(1/2)\cdot(1/2)}=3\cdot 4+3\cdot 4=24.$$
当任一 $\theta_i\to 0^+$ 时，$\csc\theta_i\to+\infty$，故 $S\to+\infty$。因此值域为 $[24,+\infty)$。$\square$

### 3.3　全息屏角度构型空间的辛结构与 Hamilton 动力学

**定义 3.5（角度构型空间的辛结构）** 全息屏角度构型空间 $D_\theta$ 是二维仿射开单形。其切空间由约束 $\delta\theta_1+\delta\theta_2+\delta\theta_3=0$ 定义。在 $D_\theta$ 上配备标准辛形式
$$\omega_\theta = d\theta_1 \wedge d\theta_2$$
（在约束超平面内，$d\theta_3 = -d\theta_1-d\theta_2$，故 $\omega_\theta$ 非退化）。$\omega_\theta$ 使 $(D_\theta, \omega_\theta)$ 成为二维辛流形。

**定理 3.6（Hamilton 向量场与辛相轨线）** 在辛流形 $(D_\theta, \omega_\theta=d\theta_1\wedge d\theta_2)$ 上，Hamilton 函数 $H(\theta)=S(\theta)-24$ 生成的向量场
$$X_H = \frac{\partial H}{\partial \theta_2}\frac{\partial}{\partial \theta_1} - \frac{\partial H}{\partial \theta_1}\frac{\partial}{\partial \theta_2}$$
满足：
1. **守恒性**：$X_H$ 的流 $\phi_t$ 保持 $H$ 的水平集；
2. **基态稳定性**：$H=0$ 的零点集 $\{\theta_0\}$ 是 $X_H$ 的唯一稳定平衡点，其线性化频率由约束切空间上 Hessian 的本征值 $\lambda_{\text{Hess},1}=\lambda_{\text{Hess},2}=124$ 给出；
3. **水平集周期性**：对 $H>0$，$X_H$ 的相轨线在紧水平集上为闭曲线，周期 $T(H) \sim 2\pi/\sqrt{124}$。

*证明.* 由 Hamilton 方程 $\dot{\theta}_1=\partial H/\partial\theta_2$, $\dot{\theta}_2=-\partial H/\partial\theta_1$，直接计算 $dH/dt=0$（守恒性）。在 $\theta_0$ 处，$H$ 的二阶展开为 $H(\delta\theta)=\frac{1}{2}\delta\theta^T H_{\text{Hess}} \delta\theta$，其中 $H_{\text{Hess}}$ 在约束切空间上正定，特征值为 $124,124$，故线性化为谐振子。严格凸水平集上的 Hamilton 流在二维情形必为周期流（Arnold 定理）。$\square$

**定理 3.7（$S_3$ 外尔房坐标与中心化子）** 三分切丛的 $S_3$ 置换对称性诱导 $D_\theta$ 上的辛作用。定义外尔房破缺坐标 $\mu: D_\theta\to\mathbb{R}^2$ 为
$$\mu(\theta) = \left(\theta_1-\theta_2,\ \theta_2-\theta_3\right).$$
则：
1. **外尔房中心 = 对称点**：$\mu^{-1}(0) = \{\theta_0\}$；
2. **中心化子 = 各向同性子群**：$S_3$ 在 $\theta_0$ 处的各向同性子群为 $S_3$ 本身；
3. **辛约化与基态锁定**：$\mu^{-1}(0)/S_3$ 的 Marsden-Weinstein 型约化是零维辛流形（单点），其辛体积约定为 1。

*证明.* $S_3$ 在 $D_\theta$ 上的作用为置换 $\theta_i$，保持约束 $\sum\theta_i=\pi/2$。$\mu$ 的分量 $\mu_1=\theta_1-\theta_2$, $\mu_2=\theta_2-\theta_3$ 在 $S_3$ 作用下按其伴随表示变换。直接验证 $\mu=0 \Leftrightarrow \theta_1=\theta_2=\theta_3=\pi/6$。$\mu^{-1}(0)/S_3$ 为单点，辛体积约定为 1。$\square$

### 3.4　值域协调

**定理 4.1（值域协调）**　六项作用量值域与抽象几何量值域满足
$$S(D_\theta)=[24,+\infty)\subset(0,+\infty)=S(D_\pm).$$
对任意 $S_0\ge 24$，存在 $x\in D_\pm$ 使 $S(x)=S_0$（由定理1.4），且存在 $\theta\in D_\theta$ 使 $S(\theta)=S_0$（由定理3.5）。

*证明*　由定理1.3，$S(D_\pm)=(0,+\infty)$。由定理3.5，$S(D_\theta)=[24,+\infty)$。显然 $[24,+\infty)\subset(0,+\infty)$。$\square$
