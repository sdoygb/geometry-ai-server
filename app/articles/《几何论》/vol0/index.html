0.1 几何论是什么

定义 0.1（几何论） 几何论是一个基于公理的数学框架，从三条几何公理出发，经过演绎推导，唯一地输出基本物理常数的数值和粒子质量谱。它主张：角度是比时空更基本的几何结构——物理常数不是测量值，而是谱几何的必然结论。

几何论不是一个"万物理论"猜想——它是一个已完成理论核心建设的数学-物理框架。截至版本260710，它已完成以下输出：

  • 精细结构常数 $S_e = 137.035999084$（与实验值 $137.035999206(11)$ 在 $8.9\times 10^{-10}$ 精度内一致）
  • 电子质量 $m_e = 510.99895\ \text{keV}$（与CODATA值 $510.99895000(15)\ \text{keV}$ 一致）
  • 三角度锁定 $(\theta_M, \theta_C, \theta_I) = (57.93^\circ, 26.16^\circ, 5.91^\circ)$——全部由谱刚性唯一确定
  • 弱混合角 $\sin^2\theta_W = \frac{\sin\theta_I}{\sin\theta_I + \sin\theta_C}$——几何形式而非经验拟合
  • 中微子味基映射——PMNS矩阵的几何根源
  • 规范群 $SU(3)\times SU(2)\times U(1)$——扇区结构的拓扑必然性

与主流理论的差异

维度主流物理几何论
基本对象时空流形上的场三分切丛上的角度结构
物理常数实验测量值，作为理论输入谱几何的输出，由公理唯一推导
外部参数约30个自由参数（标准模型）3条公理 + 0个自由参数
与实验关系参数后验拟合先验预言，后实验验证
空间/时间理论的前提背景理论的导出结构（从谱几何自举产生）
读前须知： 几何论的数学不依赖于广义相对论或量子力学——它从更基础的几何前提出发，导出了这些理论赖以成立的常数和结构。它不是对现有理论的修正，而是对它们共同根源的追寻。

0.2 一句话地图

从三条公理到物理常数，整个推导链条可以铺展在一页纸上。每个箭头表示"唯一确定"，没有回溯、没有自由参数、没有外部输入。

层：递进推导链

这是全书的主干。每一步都是前一步的逻辑后继，形成一条从公理到应用的清晰路径：

▸ 推导主干（递进方向：自上而下）
1三条公理：圆拓扑公理 + 边界极限公理 + 全息屏编码条件
↓ 公理3 → 六项作用量
2六项作用量与极值：拉格朗日乘子法 → $S_{\min}=24$，谱间隙 $\Lambda_H\approx 150$
↓ 桥接函数
3精细结构常数倒数 $S_e = 137.035999084$
↓ 谱单位选择定理
4谱单位 $K$ 与角度 $(\theta_M, \theta_C, \theta_I)$：谱刚性 → 唯一锁定
↓ 质量公式 $m = K\sin^3\theta$
5电子质量 $m_e = K\sin^3\theta_M = 510.99895\ \text{keV}$
↓ 三场动力学 I/C/M
6场耦合 → 质量层级：缪子 $m_\mu$，τ子 $m_\tau$，夸克质量谱
↓ 扇区结构 → 规范群
7粒子物理全部输出：三代、混合角、耦合常数、禁闭、弱混合角
↓ 宇宙学外推
8宇宙学预言：CMB、暗物质替代、暗能量替代、引力作为梯度

圈：互锁闭环

上述递进链在步骤2→4之间隐藏着一个环。这五个概念谁也不能绝对先于谁——你必须进入其中一个，绕一圈，才能完整理解全部：

▸ 谱几何互锁圈（卷3 §3.4–§3.8）
谱单位选择 ↔ 谱刚性定理 ↔ 自举闭环 ↔ 角度锁定 ↔ 质量公式
↺ 环形互锁：任一个的完整理解依赖其他四个

全书按"层"分卷（每一卷是递进链上的一环），但每卷内部用"圈"组织章节——卷3的§3.4–§3.8被设计为可循环阅读：第一遍正常流过，读完§3.8后再决定是否回看§3.5的证明细节。其他卷中也有类似的局部互锁结构，均会在章首明确标注。

关键点： 这个地图不是"路线规划"——它是理论建成后的结构总览。每一步的"唯一确定性"来自谱刚性定理和循环依赖的合法消解，不是来自选择性的数据拟合。理解双重结构（层为骨架，圈为关节）是读懂全书的关键。

0.3 三种阅读路径

几何论是一本有层次的书。不同背景的读者应该从不同入口进入。以下三条路径经过设计，每条都是完整自洽的阅读体验——不需要中途回头补前置知识。

⊞

数学家路线

关注结构、自洽性、定理证明。想知道"为什么这个理论成立"。

  • 卷2 数学基础（完整推导）
  • 卷1 公理与本体论
  • 卷3 谱几何与数值输出
  • 卷5 量子化
  • 卷4（选读）三场动力学
⊡

物理学家路线

关注预言、数值输出、与实验的比对。想知道"这个理论给出了什么结果"。

  • 卷1 公理与本体论（概述）
  • 卷3 谱几何与数值输出（核心）
  • 卷6 粒子物理（应用）
  • 卷7 宇宙学与引力
  • 卷2（需要时查阅）
⊠

结论优先路线

关心结果而非过程。想知道"这个理论预言了什么，准不准"。

  • 卷0（本章）→ 地图
  • 卷3 §3.5–§3.9（数值结果）
  • 卷6 §6.19（预言清单）
  • 卷A.3（实验验证状态）
  • 卷A.5（数值常数汇总）

阅读建议

  • 不要跳卷0——它是最短的，但包含了全书的地图和术语速查，是后续所有卷的索引。
  • 数学家和物理学家的路径在卷3汇合——卷3是全书的心脏，两条路径都必须经过这里。
  • 结论优先路径是单向的——如果你发现对某个结果好奇"为什么是这个数"，随时可以沿着引用链进入更深的卷。
  • 卷4（三场动力学）是全书最厚的卷——它整合了原本分散在四个子系列中的动力学内容。只有当你需要理解场如何耦合时再去读它。
对于新读者： 我建议从物理学家路线开始。先读卷1的概述（§1.1–§1.2，约10分钟），然后直接进入卷3的数值推导（约2小时）。当你看到 $S_e=137.035999084$ 从公理3一步步算出来时，你对整个理论的信任感会自然建立。之后再决定是否深入数学基础或场动力学。

双重结构：递进与互锁

几何论不是一棵树——它是一张网。全书的结构同时服务于两种逻辑关系：

层 · 递进

公理 → 数学 → 谱几何 → 场 → 粒子 → 宇宙

这是全书的分卷依据。每一卷是递进链上的一环，前一卷的结论是后一卷的前提。这是新读者的阅读顺序——从卷0开始，到卷1，再到卷3，以此类推。

圈 · 互锁

谱单位选择 ↔ 谱刚性 ↔ 自举 ↔ 角度 → 质量

这五个概念形成环形依赖——谁也不能绝对先于谁。卷3的§3.4–§3.8为此专门设计为可循环阅读：第一遍流过，读完闭环再决定是否回看细节。其他卷中也有规模更小的互锁结构。

理解双重结构的实际读法：层决定你读到哪一页，圈决定你怎么读那一页。当你遇到一个互锁圈时，不要期望一次线性读完就能完全理解——绕两圈是正常的，也是设计预期的。

建议阅读顺序

推荐顺序（从零到完整理解）：
  1. 卷0（30分钟）——了解本书结构、地图、术语
  2. 卷1 §1.1–§1.2（10分钟）——三条公理的直观理解
  3. 卷3 §3.1–§3.9（2小时）——从公理到 $S_e$ 和 $m_e$ 的完整推导
    ※ §3.4–§3.8 是互锁圈：第一遍通读即可，不必一次完全理解
  4. 卷6 §6.1–§6.12（3小时）——粒子物理应用与预言
  5. 卷2（需要时查阅）——数学基础细节
  6. 卷4（4小时）——三场动力学的完整叙述
  7. 卷5、卷7、卷8——按兴趣深入

技术说明

  • 搜索：调用浏览器 Ctrl+F（Mac: Cmd+F）可在全书范围内全文搜索。
  • 公式渲染：所有数学公式使用 KaTeX 渲染。如果公式未显示，请检查网络连接（KaTeX 在首次加载时需从 CDN 获取字体）。
  • 交叉引用：所有"见卷X §Y.Z"均为可点击的超链接，点击后跳转到对应章节。
  • 离线阅读：本书提供 PDF 版本供下载，适用于离线环境和打印。PDF 版本与 HTML 版本内容一致。
  • 源码索引：本书使用 122 篇原始文章作为素材。如果你对某个推导的原始版本感兴趣，可在卷A.7（文章索引）中找到每章对应的原始文章编号。

关于旧文章编号

本书之前的几何论文章使用分散的编号系统（0.0.3, 0.3.1, 63, 等）。本书重新组织了全部内容，使其成为一本结构清晰的书。如果你遇到某个概念或结果想要追溯到原始文章——或者反过来，你已经读过某篇文章想找到它在书中的位置——请使用卷A.7（旧文章索引），它提供了完整的双向对照表。

最后一句提醒： 这本书的目标不是"展示几何论很有趣"，而是让你——无论你是数学家、物理学家还是好奇的读者——能够亲手验证从三条公理到物理常数的每一个推导步骤。如果你在某一步发现不清晰、不完整或不正确的地方，那正是这本书需要被改进的地方。
