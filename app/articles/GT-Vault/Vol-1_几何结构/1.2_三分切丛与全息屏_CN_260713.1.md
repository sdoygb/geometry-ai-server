## 2.1　从第1章带来的问题

第1章我们做了这样一件事：从一个"约束乘积球面"

$$M(a) = S^3(a) \times S^3\!\left(\frac{a}{\sqrt{3}}\right) \times S^3\!\left(\frac{a}{\sqrt{6}}\right)$$

的**谱**（Laplace特征值）出发，反推出了三个球面的半径——分别是 $a$、$a/\sqrt{3}$、$a/\sqrt{6}$。

但第1章结束时你可能会问：

> **"我有三个球面，每个球面是三维的，总共九维。物理信息存储在这九维空间的哪里？怎么读取？"**

这个问题引出了本章的核心概念：**三分切丛**和**全息屏**。

直观图像是这样的：九维空间的全部信息，最终被"投影"到一个二维屏幕上。就像三维物体在相机感光元件上形成二维图像——丢失了一个维度的信息，但图像仍然携带了物体的全部可观测特征。

## 2.2　切丛的直和分解

### 2.2.1　什么是切丛？

先回忆一个基本概念。对于一个流形（比如球面 $S^3$），**切丛** $TS^3$ 就是在每个点 $p$ 上粘上一个切空间 $T_pS^3$（一个 $\mathbb{R}^3$），然后把所有这些切空间集合起来。

对于乘积流形 $M(a) = S^3_1 \times S^3_2 \times S^3_3$，切丛有一个漂亮的性质：

**GT-1.2.0.1（三分切丛分解）** 约束乘积球面 $M(a)$ 的切丛可以写为三个子丛的直和：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

其中 $\mathcal{M}$、$\mathcal{C}$、$\mathcal{I}$ 分别对应三个 $S^3$ 因子的切丛的拉回，每个纤维是 $\mathbb{R}^3$。

为什么叫 $\mathcal{M}$、$\mathcal{C}$、$\mathcal{I}$？这些字母代表三个扇区的不同角色：
- **$\mathcal{M}$**（物质扇区，Matter）——三个球面中**半径最大**的那个 $S^3(a)$
- **$\mathcal{C}$**（约束扇区，Constraint）——中间半径 $S^3(a/\sqrt{3})$
- **$\mathcal{I}$**（信息扇区，Information）——最小半径 $S^3(a/\sqrt{6})$

*证明*：乘积流形的切丛等于各因子切丛的直和。$M(a)$ 是三个 $S^3$ 的乘积，结论直接成立。$\square$

这个分解有一个非常强的对称性：三个子空间 $\mathcal{M}$、$\mathcal{C}$、$\mathcal{I}$ 在数学上**完全等价**——任何置换标记都不改变切丛的结构。这个 $S_3$ 对称性（6阶置换群）是整个框架的基石之一。

## 2.3　全息屏：二维信息窗口

### 2.3.1　全息屏的存在

在每个纤维 $\mathbb{R}^9 = \mathbb{R}^3_{\mathcal{M}} \oplus \mathbb{R}^3_{\mathcal{C}} \oplus \mathbb{R}^3_{\mathcal{I}}$ 中，存在一个**二维定向子空间** $\Sigma$，称为**全息屏**。三个 $\mathbb{R}^3$ 扇区向这个二维平面投影。

这就是"九维变二维"的核心机制：

```
     ℝ³_ℳ ──┐
             ├──→ ℝ²_Σ （全息屏）
     ℝ³_𝒞 ──┤
             │
     ℝ³_ℐ ──┘
```

每个扇区损失了一个维度的信息（因为从三维投影到二维），但**三个扇区合起来的信息**被完整地编码在 $\Sigma$ 上——如果我们知道投影的角度。

### 2.3.2　投影强度角

关键问题来了：**每个扇区向全息屏投影的"强度"是多少？**

这由**投影强度角** $\theta_1, \theta_2, \theta_3$ 描述。直观地说：

- 如果 $\theta_i = 90^\circ$，扇区 $i$ 完全投影到 $\Sigma$ 上（垂直投影，无信息损失）
- 如果 $\theta_i = 0^\circ$，扇区 $i$ 完全不投影到 $\Sigma$ 上（平行于屏，信息完全丢失）
- 介于两者之间时，投影面积的正比于 $\sin\theta_i$

**GT-1.2.0.2（投影面积元比例）** 设 $P_{\mathcal{M}}, P_{\mathcal{C}}, P_{\mathcal{I}}$ 为三个扇区向全息屏 $\Sigma$ 的线性投影映射。则单扇区投影面积元满足：

$$\text{Area}(P_{\mathcal{M}}(U_{\mathcal{M}})) = \sin^2\theta_1 \cdot dA, \quad
\text{Area}(P_{\mathcal{C}}(U_{\mathcal{C}})) = \sin^2\theta_2 \cdot dA, \quad
\text{Area}(P_{\mathcal{I}}(U_{\mathcal{I}})) = \sin^2\theta_3 \cdot dA$$

其中 $dA$ 是全息屏的标准面积元。

双扇区联合投影的面积元为：

$$\text{Area}(P_{\mathcal{M}} \times P_{\mathcal{C}}) = \sin\theta_1\sin\theta_2 \cdot dA$$

（另两个组合类似。）

*说明*：$\sin\theta_i$ 的平方来自"投影一次取面积元"——三维体积元投影到二维平面时，面积缩放因子是投影角的正弦。

## 2.4　全息屏编码条件

### 2.4.1　为什么正好是 $90^\circ$？

三个投影强度角 $\theta_1, \theta_2, \theta_3$ 并非自由参数。它们满足一个简单而深刻的线性约束：

**GT-1.2.0.3（全息屏编码条件）** 

$$\theta_1 + \theta_2 + \theta_3 = 90^\circ, \qquad \theta_i \in (0^\circ, 90^\circ).$$

为什么是 $90^\circ$？可以用一个几何论证来理解。

考虑三个角 $\theta_i$ 的取值范围 $[0^\circ, 90^\circ]$，它们的和 $\Sigma = \theta_1 + \theta_2 + \theta_3$ 是总投影量。我们需要一个**对称**的编码条件——三个扇区在 $S_3$ 置换下等价，所以编码条件本身必须是 $S_3$ 不变的。

- 如果 $\Sigma < 90^\circ$：某些扇区的投影被截断，信息丢失
- 如果 $\Sigma > 90^\circ$：投影"溢出"，定义域被非物理地截断
- **如果 $\Sigma = 90^\circ$**：对称点 $\theta_1 = \theta_2 = \theta_3 = 30^\circ$ 正好在定义域中心，每个扇区都可以在 $(0^\circ, 90^\circ)$ 全区间变化而不被截断

从另一个角度看：三个 $\mathbb{R}^3$ 的总维度是 $9$，全息屏的维度是 $2$，"损失"的信息维度是 $9-2=7$。但三个投影角之和为 $90^\circ$ 的选择来自 $S_3$ 对称性——对称点 $30^\circ$ 自动满足 $3 \times 30^\circ = 90^\circ$。

## 2.5　全息屏的 Kähler 结构

### 2.5.1　全息屏是一个 $\mathbb{C}P^1$

全息屏 $\Sigma$ 不仅仅是 $\mathbb{R}^2$——它还携带了**额外的几何结构**。

首先，$\Sigma$ 是 $M(a)$ 的一个二维子流形，因此继承了 $M(a)$ 的乘积度量 $g$ 作为诱导度量 $g_\Sigma$。

其次，$\Sigma$ 上有一个自然的**辛形式** $\omega = d\xi \wedge d\eta$，其中 $(\xi, \eta)$ 是 $\Sigma$ 上的坐标。在二维中，辛形式就是面积元。

第三，存在一个**近复结构** $J$（即满足 $J^2 = -I$ 的线性映射），使 $\omega$ 和 $g_\Sigma$ 相容。

**GT-1.2.0.4（全息屏的 Kähler 结构）** 全息屏 $\Sigma$ 配备诱导度量 $g_\Sigma$、辛形式 $\omega$ 和唯一的可积近复结构 $J$，三者构成 **Kähler 流形**。而且 $\Sigma$ 同胚于 $\mathbb{C}P^1$（复射影直线），即拓扑上的二维球面 $S^2$。

*证明概要*：
1. 在二维中，任意近复结构自动可积（Nijenhuis 张量恒为零）
2. $\omega$ 与 $g_\Sigma$ 通过 $J$ 相容，构成 Kähler 三元组
3. 定向二维闭流形的亏格由 Gauss-Bonnet 定理决定，计算得 $\chi(\Sigma) = 2$，即 $\Sigma \cong S^2 \cong \mathbb{C}P^1$

$\square$

**附加假设（常曲率条件）**：在 $S_3$ 对称点 $\theta_1 = \theta_2 = \theta_3 = 30^\circ$ 处，要求诱导度量 $g_\Sigma$ 为**常曲率度量**（round metric）。这个条件在这个阶段作为假设引入——它的合理性在于三个扇区在对称点处的完全等价。在后面的章节中，我们会看到这个假设与更深的拓扑结构自洽。

## 2.6　六项作用量：全局刚度函数

### 2.6.1　定义

现在我们可以问一个关键问题：**三个扇区向全息屏投影的"总刚度"有多大？**

答案是六项作用量 $S$，它由六项组成——三项来自单个扇区的投影刚度，三项来自两个扇区之间的耦合刚度：

**GT-1.2.0.5（六项作用量）** 在 $\theta_1 + \theta_2 + \theta_3 = 90^\circ$ 下，定义：

$$S(\theta_1, \theta_2, \theta_3) = \sum_{i=1}^{3} \frac{1}{\sin^2\theta_i} + \sum_{i<j} \frac{1}{\sin\theta_i\sin\theta_j}$$

展开写为六项：

$$S = \frac{1}{\sin^2\theta_1} + \frac{1}{\sin^2\theta_2} + \frac{1}{\sin^2\theta_3} + \frac{1}{\sin\theta_1\sin\theta_2} + \frac{1}{\sin\theta_1\sin\theta_3} + \frac{1}{\sin\theta_2\sin\theta_3}$$

每一项的几何意义很清楚：
- $\frac{1}{\sin^2\theta_i}$：扇区 $i$ 单独投影的刚度（投影面积越小 $\to$ $\sin\theta_i$ 越小 $\to$ 刚度越大）
- $\frac{1}{\sin\theta_i\sin\theta_j}$：扇区 $i$ 和 $j$ 联合投影的耦合刚度

为什么没有三扇区联合项？因为三个 $\mathbb{R}^3$ 同时投影到 $\mathbb{R}^2$ 时，第三个扇区的投影被编码条件 $\theta_1+\theta_2+\theta_3=90^\circ$ 消去了。**恰好六项，不多不少**。

---

### 2.6.2　值域定理

这个函数 $S$ 有什么性质？它的最小值在哪里？值域是什么？

**GT-1.2.0.6（值域定理与严格凸性）** 六项作用量 $S$ 在定义域 $D_\theta = \{(\theta_1, \theta_2, \theta_3) \mid \theta_i > 0,\ \theta_1+\theta_2+\theta_3=90^\circ\}$ 上的值域为：

$$S(D_\theta) = [24, +\infty)$$

且 $S$ 在 $D_\theta$ 上为**严格凸函数**，对称点 $\theta_1 = \theta_2 = \theta_3 = 30^\circ$ 是唯一的全局严格最小值点，最小值为 $S(30^\circ) = 24$。

*证明*：分三步。

**第一步：对称点的值。** 在 $\theta_i = 30^\circ$，$\sin 30^\circ = 1/2$，$\sin^2 30^\circ = 1/4$：

$$\frac{1}{\sin^2 30^\circ} = 4, \quad \frac{1}{\sin 30^\circ\sin 30^\circ} = 4$$

三项单扇区之和 $3 \times 4 = 12$，三项双扇区之和 $3 \times 4 = 12$，总和 $S = 24$。

**第二步：最小值的唯一性。** 使用拉格朗日乘子法。在约束 $\sum\theta_i = \pi/2$ 下求 $S$ 的极值，构造拉格朗日函数：

$$\mathcal{L} = S - \lambda\!\left(\sum\theta_i - \frac{\pi}{2}\right)$$

对 $\theta_k$ 求偏导：

$$\frac{\partial \mathcal{L}}{\partial \theta_k} = -\frac{2\cos\theta_k}{\sin^3\theta_k} - \sum_{j \neq k} \frac{\cos\theta_k}{\sin^2\theta_k\sin\theta_j} - \lambda = 0$$

提取公因子 $-\cos\theta_k / \sin^2\theta_k$：

$$-\frac{\cos\theta_k}{\sin^2\theta_k}\left(\frac{2}{\sin\theta_k} + \sum_{j \neq k} \frac{1}{\sin\theta_j}\right) = \lambda$$

当 $\theta_1 = \theta_2 = \theta_3 = \pi/6$ 时，上式对 $k=1,2,3$ 给出相同的 $\lambda$。由 $S_3$ 对称性，这是唯一的内部临界点。

**第三步：Hessian 正定性。** 在 $\theta_0$ 处计算 Hessian 矩阵。记 $s_i = \sin\theta_i$，$c_i = \cos\theta_i$。对单扇区项 $f_i = 1/s_i^2$：

$$\frac{\partial^2 f_i}{\partial \theta_i^2}\bigg|_{\theta_0} = \frac{2+4\cos^2(\pi/6)}{\sin^4(\pi/6)} = \frac{2+3}{1/16} = 80$$

对双扇区项 $f_{ij} = 1/(s_i s_j)$：

$$\frac{\partial^2 f_{ij}}{\partial \theta_i^2}\bigg|_{\theta_0} = \frac{1+\cos^2(\pi/6)}{\sin^3(\pi/6)\sin(\pi/6)} = \frac{7/4}{1/16} = 28$$

$$\frac{\partial^2 f_{ij}}{\partial \theta_i\partial \theta_j}\bigg|_{\theta_0} = \frac{\cos(\pi/6)\cos(\pi/6)}{\sin^2(\pi/6)\sin^2(\pi/6)} = \frac{3/4}{1/16} = 12$$

因此 Hessian 矩阵元为：

$$H_{ii} = 80 + 28 + 28 = 136, \quad H_{ij} = 12 \ (i \neq j)$$

在约束切空间 $T_{\theta_0}D_\theta = \{v \in \mathbb{R}^3 \mid v_1+v_2+v_3=0\}$ 上，二次型：

$$v^T H v = 136\sum v_i^2 + 24\sum_{i<j} v_i v_j$$

利用 $(\sum v_i)^2 = 0 \Rightarrow \sum_{i<j}v_i v_j = -\frac12 \sum v_i^2$，得：

$$v^T H v = (136-12)\sum v_i^2 = 124\sum v_i^2 > 0$$

**Hessian 谱为 $\{124, 124\}$**，严格正定，因此 $S$ 严格凸，$30^\circ$ 是唯一全局最小值。

**第四步：无上界。** 当任一角度 $\theta_i \to 0^\circ$ 时，$\sin\theta_i \to 0$，$1/\sin^2\theta_i \to +\infty$，故 $S \to +\infty$。

结合连续性，值域为 $[24, +\infty)$。$\square$

## 2.7　切丛平方根分解与幻数

### 2.7.1　问题：为什么简并度是 $2(k+1)^2$？

在第1章我们看到了谱特征签名 $(3/a^2,\ 8/a^2,\ 9/a^2)$。现在我们要回答一个更深刻的问题：**这些特征值的简并度为什么是特定的序列？**

### 2.7.2　全纯切丛的平方根

$\Sigma \cong \mathbb{C}P^1$ 的全纯切丛有一个标准结果：

$$T\Sigma \cong \mathcal{O}(2)$$

其中 $\mathcal{O}(k)$ 是 $\mathbb{C}P^1$ 上次数为 $k$ 的全纯线丛。$\mathcal{O}(2)$ 可以写成两个 $\mathcal{O}(1)$ 的张量积：

$$T\Sigma \cong \mathcal{O}(2) = \mathcal{O}(1) \otimes \mathcal{O}(1)$$

现在关键来了：三分切丛的投影核（每个 $\mathbb{R}^3$ 扇区投影到 $\Sigma$ 时丢失的那个纬度）给出了两个独立的 $\mathcal{O}(1)$ 线丛。在 $S_3$ 对称点处，三个扇区的投影核都等价于 $\mathcal{O}(1)$。选取其中两个作为切丛的平方根：

**GT-1.2.0.7（切丛平方根分解）** 在 $S_3$ 对称点处，存在两个独立的全纯线丛 $L_{\mathcal{M}}, L_{\mathcal{C}} \to \Sigma$，满足：

$$L_{\mathcal{M}} \cong L_{\mathcal{C}} \cong \mathcal{O}(1), \quad L_{\mathcal{M}} \otimes L_{\mathcal{C}} \cong T\Sigma$$

*说明*：不要把这与法丛分解混淆。$L_{\mathcal{M}}, L_{\mathcal{C}}$ 来自**切丛**的平方根构造，用于解释空间维度的简并度；第三个线丛 $L_{\mathcal{I}}$ 属于法丛，将在后续章节扮演不同的角色。

---

### 2.7.3　幻数定理

现在我们可以计算量子化能级的简并度了。

对第 $k$ 能级，态空间是两个独立 $\mathcal{O}(k)$ 线丛的全纯截面空间的张量积：

$$\mathcal{H}_k^{(\text{spatial})} = H^0(\Sigma, L_{\mathcal{M}}^{\otimes k}) \otimes H^0(\Sigma, L_{\mathcal{C}}^{\otimes k})$$

由 Riemann-Roch 定理，对 $\mathbb{C}P^1$ 上的 $\mathcal{O}(k)$：

$$\dim H^0(\mathbb{C}P^1, \mathcal{O}(k)) = k+1$$

因此：

$$\dim \mathcal{H}_k^{(\text{spatial})} = (k+1)^2$$

引入自旋 $1/2$ 自由度（一个独立的 $\mathbb{C}^2$ 因子，其合理性在后续章节论证），总 Hilbert 空间维数为：

**GT-1.2.0.8（幻数序列）** 

$$\dim \mathcal{H}_k^{(\text{total})} = 2(k+1)^2, \quad k = 0, 1, 2, \dots$$

展开为序列：

$$2,\ 8,\ 18,\ 32,\ 50,\ 72,\ 98,\ \dots$$

这就是**幻数序列**——它描述了全息屏 $\Sigma$ 上信息单元的壳层填充数。第 $k$ 壳层容量为 $2(k+1)^2$。

**重要区分**：
- **几何部分** $(k+1)^2$：从切丛平方根分解严格导出，不依赖任何物理假设
- **自旋因子 $2$**：条件性假设，来自自旋 $1/2$ 自由度

## 2.8　Bott 周期与自然截断

### 2.8.1　为什么在 98 处停止？

幻数序列不会无限延伸——它在 $2 \times 7^2 = 98$ 处自然截断。为什么？

答案是 **Bott 周期性**。

实 Clifford 代数 $Cl_{0,n}$ 的不可约表示维数 $\rho_n$ 满足 8 周期律 $\rho_{n+8} = 16\rho_n$。对 $n = 7$：

$$Cl_{0,7} \cong \mathbb{R}(8) \oplus \mathbb{R}(8)$$

其不可约表示为 8 维实表示。$Cl_{0,7}$ 有 7 个负生成元，在表示空间上的作用给出 **7 个独立的反交换编码通道**。

**GT-1.2.0.9（Bott 截断）** 幻数序列的有效编码通道数为：

$$N_{\text{eff}} = 7$$

因此能级标号满足 $k+1 \leq 7$，即 $k \leq 6$。序列在 $k = 6$（值 $2 \times 7^2 = 98$）处自然截断：

$$\{2, 8, 18, 32, 50, 72, 98\}$$

共 7 个幻数，对应 7 个有效编码通道。
