## §6.1　渗透函数 $\Phi$ 的椭圆方程

### 6.1.1　为什么需要渗透函数？

回忆第4章的有效 Hessian：

$$H^{\text{eff}} = H^{\text{bare}} + \Phi \otimes H^{\text{bg\_proj}} + H^W$$

其中 $H^{\text{bare}} = \text{diag}(392.21, 58760.77)$ 是裸 Hessian（第4章），$H^{\text{bg\_proj}}$ 是 $30^\circ$ 背景在 $(\xi,\eta)$ 坐标下的投影（1.3章），$H^W$ 是跨扇区耦合矩阵（本节将推导）。$\Phi$ 是**渗透函数**——它描述 $30^\circ$ 背景如何"渗透"到局域截面上。

直观地说：我们在第4章找到的裸基准点是一个"裸"的点——它没有包含背景对它的渗透效应。当我们把 $30^\circ$ 背景（六项作用量的全局极小点，$S=24$）的几何结构投影到局域截面上时，相当于在"碗"的表面覆盖了一层"渗透膜"。$\Phi$ 就是这层膜的厚度分布。

### 6.1.2　联合作用量变分到椭圆方程

引入**联合作用量**：

$$S_{\text{joint}}[\text{总宇宙}] = S[\text{局域扇区}] + S[\text{30°背景}] + W[\text{耦合}]$$

其中 $S[\text{30°背景}] = 24$ 是已知常数（1.3章），$W[\text{耦合}]$ 是跨扇区耦合项（6.2节推导）。对 $\Phi$ 做变分 $\delta S_{\text{joint}}/\delta\Phi = 0$，展开到 $\Phi$ 的二阶项，得到渗透函数满足的椭圆型偏微分方程：

$$L[\Phi] := -\left(\lambda_1^{\text{eff}}\frac{\partial^2}{\partial \xi^2} + \lambda_2^{\text{eff}}\frac{\partial^2}{\partial \eta^2}\right)\Phi + M^2(\xi,\eta)\Phi = Q(\xi,\eta) \tag{6.1}$$

这个方程的每一项都有明确的几何意义：

| 项 | 符号 | 意义 | 来源 |
|:---|:---:|:---|:---:|
| 二阶导数项 | $\lambda_{1,2}^{\text{eff}}$ | 有效软硬模（391.05, 59324.3 rad$^{-2}$） | 第4章 Hessian + 渗透修正 |
| 质量项 | $M^2$ | 法向曲率（$\geq 0$） | 约束流形法线方向的曲率 |
| 源项 | $Q$ | 跨扇区耦合的局域投影 | 6.2节的前置因子 |

**一致椭圆性**：由于 $\lambda_1^{\text{eff}}, \lambda_2^{\text{eff}} > 0$，算子 $L$ 的主部系数矩阵 $\text{diag}(\lambda_1^{\text{eff}}, \lambda_2^{\text{eff}})$ 为正定，因此 $L$ 是一致椭圆算子。这意味着解的存在性、唯一性和正则性有完备的 PDE 理论保证。

### 6.1.3　有效质量项 $M^2$ 的非负性

$M^2(\xi,\eta)$ 是联合作用量在法向 $n = (1,1,1)/\sqrt{3}$（垂直约束流形 $\theta_M+\theta_C+\theta_I=90^\circ$ 的方向）上的二阶导数。在裸基准点处：

$$n^{\mathsf{T}} H_S(\theta^0) n = 160\ \text{rad}^{-2} > 0$$

其中 $H_S$ 是六项作用量 $S$ 的 Hessian。由 $S$ 在约束流形上的严格凸性（1.3章定理3.5），$M^2$ 在整个截面 $\Sigma$ 上非负：

$$M^2(\xi,\eta) \geq 0 \quad \text{在 } \Sigma \text{ 上处处成立}$$

这个非负性不是额外假设——它是1.3章严格凸性定理的直接推论。它在 PDE 理论中的作用是保证方程 (6.1) 解的存在唯一性（通过 Lax-Milgram 引理）。

### 6.1.4　源项 $Q$ 的嵌入映射

源项 $Q$ 将跨扇区耦合 $H^W$ 的常数矩阵嵌入为截面 $\Sigma$ 上的光滑函数：

$$Q(\xi,\eta) = \frac{\rho_\varepsilon(r)}{A_\varepsilon} \sum_{i,j=1}^2 H^W_{ij}$$

其中 $\rho_\varepsilon(r)$ 是中心在裸基准点的鼓包函数（Fubini-Study 度量下的光滑截断），$A_\varepsilon$ 是归一化常数。这一构造的几何意义是：跨扇区耦合是"局域的"——它在裸基准点附近有支撑，而在远离裸基准点的地方指数衰减到零。

### 6.1.5　边界值与 Green 函数

全息屏 $\Sigma$ 紧致化后拓扑同胚于 $S^2$（1.2章定理2.1），$S^2$ 的面积为 $2\pi$。在紧致化模型下，方程 (6.1) 的边界条件退化为整体归一化条件：

$$\int_\Sigma \Phi\,d\Sigma = \Phi_0$$

其中 $\Phi_0$ 由 $30^\circ$ 背景投影给出。Green 函数 $G(p,p')$ 定义为：

$$L_p[G(p,p')] = \delta(p-p') - \frac{1}{A_\Sigma}$$

右端的 $-1/A_\Sigma$ 项保证了零均值可解性。解的积分表示为：

$$\Phi(p) = \Phi_0 + \int_\Sigma G(p,p')\, Q(p')\, d\Sigma'$$

**正则性保证**：由椭圆正则性理论，若 $Q \in C^\infty(\Sigma)$（确实如此，因为 $Q$ 是光滑鼓包函数），则 $\Phi \in C^\infty(\Sigma)$。这意味着渗透函数处处光滑，没有奇点。

### 6.1.6　球谐展开与超代数衰减

将 $\Phi$ 在 $S^2$ 的球谐基 $\widetilde Y_l^m$ 上展开：

$$\Phi(\xi,\eta) = \sum_{l=0}^\infty \sum_{m=-l}^l C_l^m \widetilde Y_l^m(\xi,\eta)$$

**定理 GT-1.6.0.1（超代数衰减）**。渗透函数的球谐展开系数 $C_l^m$ 具有超代数衰减率：对任意 $N>0$，存在常数 $C_N>0$，使得 $|C_l^m| \leq C_N \cdot l^{-N}$。

这个定理的含义是：**渗透函数 $\Phi$ 在紧致全息屏上无奇异点，所有角向涨落都被指数压制**。这意味着高阶角向模式（$l>7$）的贡献可以忽略——这与下一节要讨论的 Bott 周期截断 $N_{\text{eff}}=7$ 完全一致。

## §6.2　跨扇区耦合 $H^W$ 的前置因子

### 6.2.1　问题：三个扇区怎么耦合？

从1.2章我们知道，切丛分解为三个扇区：

$$TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

每个扇区 $\cong \mathbb{R}^3$。扇区之间通过约束截面 $\Sigma$ 的法丛 $N\Sigma$ 耦合——这就是跨扇区耦合 $H^W$。它的矩阵元 $a'=1.577$、$b'=0.86699$ 看起来像是自由参数，但实际上它们由法丛几何**唯一锁定**。

> 核心洞见：全息屏的法丛 $N\Sigma$ 不是一个任意向量丛——它具有 $Spin(8)$ 结构，其谱留数决定了耦合强度。

### 6.2.2　法丛 $N\Sigma$ 的 Spin(8) 结构

约束截面 $\Sigma \subset M^9$ 是二维的，因此法丛 $N\Sigma$ 的秩为：

$$L = 9 - 2 = 7$$

第5章证明了 $N\Sigma$ 的结构群可提升为 $Spin(8)$，其横截模型纤维为 $S^7 = Spin(8)/Spin(7)$。$Spin(8)$ 有三个 8 维不可约表示：向量表示 $V$、正旋量 $S^+$、负旋量 $S^-$。在三分切丛框架下，这三个表示与三个扇区一一对应：

| 扇区 | Spin(8) 表示 | 角色 |
|:---|:---:|:---:|
| $\mathcal{M}$（物质） | $V$（向量） | 源模，对应电子 |
| $\mathcal{C}$（因果） | $S^+$（正旋量） | 传递模，对应光子 |
| $\mathcal{I}$（信息） | $S^-$（负旋量） | 感知模，对应中微子 |

**腰边耦合** $\mathcal{M}$-$\mathcal{C}$ 对应 $V \leftrightarrow S^+$，由 $\kappa_w$ 归一化。  
**底边耦合** $\mathcal{M}$-$\mathcal{I}$ 对应 $V \leftrightarrow S^-$，由 $\kappa_w'$ 归一化。

### 6.2.3　谱留数与前置因子的代数关系

前置因子 $\kappa_w$、$\kappa_w'$ 由法丛上椭圆算子的**谱留数**（Wodzicki 留数）唯一确定。谱留数可以粗略理解为"算子在纤维上的有效作用面积"。经过严格推导（0.1.2 附录C/D），两个前置因子满足两个刚性代数关系：

**定理 GT-1.6.0.2（前置因子比定理）**。

$$\frac{\kappa_w'}{\kappa_w} = \frac{D-1}{L} - \frac{1}{L^2(D-1)} = \frac{8}{7} - \frac{1}{392} = \frac{447}{392} \tag{6.2}$$

其中 $D=9$（总维度），$L=7$（法丛秩）。

**定理 GT-1.6.0.3（前置因子和定理）**。

$$\kappa_w + \kappa_w' = 4L + \frac{\pi}{\Lambda_H} = 28 + \frac{\pi}{150} \tag{6.3}$$

其中 $4L = 28$ 是 $Spin(8)$ Lie 代数 $\mathfrak{so}(8)$ 的维数，$\pi/\Lambda_H$ 是软硬模比 $\Lambda_H = \lambda_2/\lambda_1 = 150$ 引入的 Dixmier 迹对数留数修正。

这两个定理的几何直观是：

- **比的关系** (6.2) 来自法丛的"形状"：腰边耦合（沿 $\mathcal{C}$ 方向）受法丛维数 $D-1=8$ 标度，底边耦合（横向）受法丛秩 $L=7$ 标度。高阶修正 $-1/392$ 来自 $S^7$ 热核的曲率修饰。
- **和的关系** (6.3) 来自法丛的"容量"：$4L=28$ 是法丛 Lie 代数的整数拓扑项，$\pi/150$ 是软硬模比引入的量子修正。

### 6.2.4　前置因子的显式数值

联立 (6.2) 和 (6.3)，代入 $L=7$、$\Lambda_H=150$：

$$\begin{cases}
\kappa_w + \kappa_w' = 28 + \dfrac{\pi}{150} = 28.02094395\ldots \\[6pt]
\dfrac{\kappa_w'}{\kappa_w} = \dfrac{447}{392} = 1.14030612\ldots
\end{cases}$$

解得：

$$\boxed{\kappa_w = 13.092026\ldots, \qquad \kappa_w' = 14.928918\ldots}$$

**验证**：
- 和：$13.092026 + 14.928918 = 28.020944\ldots$，与 $4L + \pi/\Lambda_H$ 偏差 $< 10^{-6}$
- 比：$14.928918/13.092026 = 1.14030612\ldots$，与 $447/392$ 偏差 $< 10^{-8}$

### 6.2.5　从前置因子到 $H^W$ 矩阵元

将前置因子代入跨扇区耦合的定标公式（0.1 单一物理映射导出）：

$$w = -\frac{F_{\xi\eta}}{4\pi} \cdot \frac{\delta_{MC}}{90^\circ} \cdot \sin^2 30^\circ \cdot \sqrt{\frac{\lambda_1}{\lambda_2}} \cdot \kappa_w$$

$$w' = -\frac{F_{\xi\eta}}{4\pi} \cdot \frac{\delta_{MI}}{90^\circ} \cdot \sin^2 30^\circ \cdot \kappa_w'$$

其中 $F_{\xi\eta}=107.6$、$\delta_{MC}=31.0^\circ$、$\delta_{MI}=6.88^\circ$ 都是几何参数。代入计算：

$$w = -0.78850, \quad w' = -2.44399$$

由同型关系 $a' = -2w$、$b' = 2w - w'$：

$$\boxed{a' = 1.57700, \qquad b' = 0.86699}$$

这个结果与第4章使用的锁定值完全一致。**这不是巧合——而是定理输出的自洽验证**：

| 验证项 | 定理预测 | 锁定值 | 偏差 |
|:---|:---|:---|:---:|
| $a'$ | 1.57700 | 1.577 | $< 0.001\%$ |
| $b'$ | 0.86699 | 0.867 | $< 0.001\%$ |
| $\lambda_1^{\text{eff}}$ | 391.05 | 391.05 | 零 |
| $\lambda_2^{\text{eff}}$ | 59324.30 | 59324.30 | 零 |

### 6.2.6　零阶近似：直观感受

忽略高阶修正（令 $\pi/\Lambda_H \to 0$ 和 $1/(L^2(D-1)) \to 0$），得到零阶近似解：

$$\kappa_w^{(0)} = \frac{4L}{1 + (D-1)/L} = \frac{28}{1 + 8/7} = \frac{196}{15} = 13.0666\ldots$$

$$\kappa_w'^{(0)} = \frac{D-1}{L} \cdot \kappa_w^{(0)} = \frac{8}{7} \times \frac{196}{15} = \frac{224}{15} = 14.9333\ldots$$

零阶近似与精确值的偏差仅约 $0.2\%$。这说明：**耦合强度的主要部分（~99.8%）来自法丛的"形状因子"$(D-1)/L$ 和 Lie 代数维数 $4L$，高阶曲率修正只贡献了约 $0.2\%$**。

## §6.3　收官：渗透-变分封闭与卷1总结

### 6.3.1　渗透函数的几何角色

渗透函数 $\Phi$ 在有效 Hessian 中扮演的角色可以这样理解：

$$H^{\text{eff}} = \underbrace{H^{\text{bare}}}_{\text{第4章}} + \underbrace{\Phi \otimes H^{\text{bg\_proj}}}_{\text{本章 §6.1}} + \underbrace{H^W}_{\text{本章 §6.2}}$$

- $H^{\text{bare}}$：裸截面处的曲率（"碗的形状"）
- $\Phi \otimes H^{\text{bg\_proj}}$：$30^\circ$ 背景的渗透效应（"碗上的膜"）
- $H^W$：扇区之间的耦合（"膜中的张力"）

三者合起来，构成了一个超定但自洽的代数系统——渗透函数 $\Phi$ 的椭圆方程 (6.1) 的解与跨扇区耦合 $H^W$ 的矩阵元互为验证条件。这就是**渗透-变分封闭**。

### 6.3.2　双模零误差的几何意义

所谓的"双模零误差"是指：有效 Hessian 的两个特征值

$$\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}, \qquad \lambda_2^{\text{eff}} = 59324.3\ \text{rad}^{-2}$$

在几何定理常数（$\kappa_w$、$\kappa_w'$）的推导中，不是作为输入条件，而是作为**输出验证**。这意味着整个几何框架——从谱刚性（1.1）到渗透函数（6.1）到 Spin(8) 法丛几何（6.2）——在数值上自洽到 $10^{-8}$ 精度，没有使用任何自由参数。

### 6.3.3　九素互扼网络中的位置

本章的渗透函数和前置因子是九素互扼网络的**最后一个闭环**：

1.1 谱刚性 → 1.2 三分切丛 → 1.3 十方空间 → 1.4 约束截面 → 1.5 拓扑纤维化 → **1.6 渗透函数与跨扇区耦合**

这六个环节构成一个**互锁链**：每个环节的输出是下一个环节的输入，最后一个环节的输出又验证了整个链条的自洽性。任何环节的改变都将导致整个系统的不一致。
