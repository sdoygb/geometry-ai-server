## 8B.3.1　问题定位：为什么需要未约化相空间

### 8B.3.1.1　出路B的核心策略

8B.2（出路A）在扩展相空间中做了Dirac约束分析——引入浴自由度 $(s, p_s)$，将非哈密顿的梯度流嵌入更大的保守系统。出路B的策略不同：**不在相空间中添加任何额外自由度，直接在原始未约化相空间中分析约束的代数结构**。

两者的区别是根本性的：

| 方面 | 出路A（8B.2） | 出路B（8B.3） |
|:---|:---|:---|
| **相空间** | 扩展相空间（6维）：$(\theta_M,\theta_C,s,p_M,p_C,p_s)$ | 未约化相空间（6维）：$(\theta_M,\theta_C,\theta_I,p_M,p_C,p_I)$ |
| **额外自由度** | $s$ 和 $p_s$（浴自由度，有物理意义） | 无——$(\theta_I,p_I)$ 本是系统的一部分 |
| **约束** | $\phi=s-S$ + $\chi=\theta_M+\theta_C+\theta_I-90^\circ$（第二类对） | $\phi_1=\theta_M+\theta_C+\theta_I-90^\circ$（第一类）+ $\phi_2=S-S_{\text{target}}$（第一类） |
| **核心操作** | 将梯度流嵌入保守哈密顿系统 | 分析约束代数 → 构造动量映射 → 辛约化 |
| **浴** | 显式（$s,p_s$） | 隐式（在约化后的有效动力学中涌现） |

出路A是"自顶向下"——从更大的保守系统投影出耗散子系统。出路B是"自底向上"——从更基础的约束代数出发，通过辛约化到达物理相空间。两者在物理上等价但数学路径不同。

### 8B.3.1.2　关键洞察：从"先约化"到"先分析约束"

Vol-2直接在约束截面 $\Sigma = \{(\theta_M,\theta_C,\theta_I):\theta_M+\theta_C+\theta_I=90^\circ\}$ 上工作——这是一个2维空间，$\theta_I$ 已被完备性约束消去。这种"先约化后分析"的策略在经典层次非常高效。

然而，Dirac约束量子化要求**先分析约束结构，再量子化，最后施加约束**。这要求回到未约化的全构型空间：

$$\mathcal{Q}_{\text{full}} = \{(\theta_M,\theta_C,\theta_I) \in (0,90^\circ)^3\} \tag{1.1}$$

理由有三：

1. **约束的泊松括号必须在未约化相空间中计算**——约化后的括号已经是Dirac括号，用它来分析约束的代数是循环论证。
2. **规范对称性（U(1)作用）在全构型空间中才显式可见**——约化后，U(1)轨道被商掉，对称性变成了隐式的。
3. **动量映射的构造需要全相空间中的辛形式**——$\iota_{\xi}\omega = d\mu$ 中的 $\omega$ 必须是全空间的辛形式。

### 8B.3.1.3　与Vol-5附录C的关系

Vol-5附录C在**联合截面** $\Sigma_{\text{joint}}^{(N)}$（$N$体同步约束截面）上进行了Marsden-Weinstein辛约化的分析。其核心结论：

- **路径B（已封闭）**：$\tilde{S}_{\text{total}}$ 在U(1)作用下单调 → 真空标记条件 $\tilde{S}_{\text{total}}=0$ 唯一确定U(1)参数 → SU(3)幺模性
- **路径A（待技术适配）**：真动量映射 $J: \Sigma \to \mathfrak{u}(1)^*$ 未构造——"在当前联合截面的乘积辛结构下，这是可操作的技术任务，但尚未完成"

本章（8B.3）的任务是：**在单粒子未约化相空间中推进路径A**——构造U(1)动量映射，完成Marsden-Weinstein约化，并以此为基础建立Dirac括号量子化。单粒子情形是多粒子联合截面的必要前置。

## 8B.3.2　未约化相空间与辛结构

### 8B.3.2.1　全构型空间与余切丛

定义全构型空间：

$$\mathcal{Q}_{\text{full}} = \{(\theta_M,\theta_C,\theta_I) : \theta_M,\theta_C,\theta_I \in (0, 90^\circ)\} \subset \mathbb{R}^3 \tag{2.1}$$

其维数为3。余切丛 $T^*\mathcal{Q}_{\text{full}}$ 为相空间，维数6。典范坐标记为：

$$(\theta_M, \theta_C, \theta_I, p_M, p_C, p_I) \tag{2.2}$$

其中 $p_i$ 为与 $\theta_i$ 典范共轭的动量。

**诚实标注**：在几何论中，这些动量当前无独立的物理对应物——M场梯度流是非哈密顿流，没有自然动量。本章引入 $p_i$ 作为数学工具以执行Dirac-Bergmann算法；其物理诠释将在Marsden-Weinstein约化后涌现（或诚实标注为缺失）。这与8B.2引入 $(s, p_s)$ 作为浴自由度的策略形成对照——出路B的额外变量是纯数学工具，出路A的额外变量有物理动机。

### 8B.3.2.2　典范辛形式

$T^*\mathcal{Q}_{\text{full}}$ 上的典范辛形式为：

$$\omega_{\text{can}} = d\theta_M \wedge dp_M + d\theta_C \wedge dp_C + d\theta_I \wedge dp_I \tag{2.3}$$

$\omega_{\text{can}}$ 是非退化的（$\omega_{\text{can}}^{\wedge 3} \neq 0$）且闭的（$d\omega_{\text{can}} = 0$），因此 $(T^*\mathcal{Q}_{\text{full}}, \omega_{\text{can}})$ 是合法辛流形。

典范泊松括号：

$$\{f, g\}_{\text{can}} = \sum_{i \in \{M,C,I\}} \left( \frac{\partial f}{\partial\theta_i}\frac{\partial g}{\partial p_i} - \frac{\partial f}{\partial p_i}\frac{\partial g}{\partial\theta_i} \right) \tag{2.4}$$

特别地：$\{\theta_i, p_j\} = \delta_{ij}$，$\{\theta_i, \theta_j\} = \{p_i, p_j\} = 0$。

### 8B.3.2.3　完备性约束的几何意义

完备性约束：

$$\phi_1(\theta_M,\theta_C,\theta_I) \equiv \theta_M + \theta_C + \theta_I - 90^\circ = 0 \tag{2.5}$$

定义了 $\mathcal{Q}_{\text{full}}$ 中的一个2维仿射平面——正是Vol-2和Vol-3C中直接使用的约束截面 $\Sigma$。

在相空间 $T^*\mathcal{Q}_{\text{full}}$ 中，$\phi_1 = 0$ 定义了一个5维子流形（余维1的约束面）：

$$\mathcal{C}_1 = \{(\theta,p) \in T^*\mathcal{Q}_{\text{full}} : \phi_1(\theta) = 0\} \tag{2.6}$$

$\phi_1$ 是仅含构型坐标的约束（不含动量），其梯度在 $\mathcal{C}_1$ 上非零：

$$d\phi_1 = d\theta_M + d\theta_C + d\theta_I \neq 0$$

因此 $\mathcal{C}_1$ 是光滑嵌入子流形。

## 8B.3.3　约束的Dirac-Bergmann分类

### 8B.3.3.1　两个候选约束

系统包含两个候选约束：

**主约束 $\phi_1$（完备性）**：

$$\phi_1 = \theta_M + \theta_C + \theta_I - 90^\circ \approx 0 \tag{3.1}$$

这是普适约束——对几何论的所有物理构型均成立。它来自公理1（全息屏编码条件）。

**次级约束 $\phi_2$（$S$等值面）**：

$$\phi_2 = S(\theta_M,\theta_C,\theta_I) - S_{\text{target}} \approx 0 \tag{3.2}$$

其中 $S$ 为六项作用量（公理2）。$\phi_2$ **仅在特定的物理构型中生效**：

| 构型 | $\phi_2$状态 | $S_{\text{target}}$ | 物理自由度 |
|:---|:---|:---|:---:|
| 电子基态 | **施加** | $S_e = 137.035999084$ | 1（谱互锁不动点） |
| 对称点 | **施加** | $S_{\min} = 24$ | 0（孤立点） |
| 一般M场演化 | **不施加** | — | 2（自由演化） |
| 缪子构型 | **不施加** | — | 2（无$S=S_e$锁定） |

**诚实标注**：$\phi_2$ 在几何论中不是普适约束；其施加依赖物理构型。这与8B.1 §1.5.2的概略性描述不同——8B.1将两约束并列处理，本章给出精确分类。

### 8B.3.3.2　约束代数与修正

**引理 3.1（典范泊松括号下的约束代数）**　在典范泊松括号（2.4）下：

$$\{\phi_1, \phi_1\} = 0, \quad \{\phi_2, \phi_2\} = 0 \tag{3.3}$$

$$\{\phi_1, \phi_2\} = 0 \quad \text{（恒成立）} \tag{3.4}$$

**证明：**

(3.3) 由泊松括号的反对称性直接给出。

对于(3.4)，关键观察是 $\phi_1$ 和 $\phi_2$ 都**仅含构型坐标**（不含动量 $p_i$）。在典范泊松括号下：

$$\{\phi_1, \phi_2\} = \sum_i \left( \frac{\partial\phi_1}{\partial\theta_i}\frac{\partial\phi_2}{\partial p_i} - \frac{\partial\phi_1}{\partial p_i}\frac{\partial\phi_2}{\partial\theta_i} \right)$$

由于 $\partial\phi_1/\partial p_i = \partial\phi_2/\partial p_i = 0$ 对所有 $i$ 成立，故 $\{\phi_1,\phi_2\} = 0$。$\square$

**重要澄清**：8B.1 §1.5.2曾给出 $\{\phi_1,\phi_2\} = \partial S/\partial\theta_M+\partial S/\partial\theta_C+\partial S/\partial\theta_I$。该计算基于一个**不同的泊松括号定义**——可能是约束截面 $\Sigma$ 上的辛括号（其中 $\theta_I$ 已消去）。在未约化相空间的典范泊松括号下，该计算不成立——因为 $\phi_1$ 和 $\phi_2$ 在动量坐标上的偏导均为零。本章对此进行修正：在未约化相空间中，$\phi_1$ 和 $\phi_2$ 对合（均为第一类）。

### 8B.3.3.3　GT-8B.3.1：第一类约束的完整分类

**定理 GT-8B.3.1（约束分类，定理级）**　在未约化相空间 $(T^*\mathcal{Q}_{\text{full}}, \omega_{\text{can}})$ 中：

**(i)** $\phi_1$ 是第一类约束。它生成以下规范变换（沿约束面的切方向）：

$$X_{\phi_1} = \frac{\partial}{\partial p_M} + \frac{\partial}{\partial p_C} + \frac{\partial}{\partial p_I} \tag{3.5}$$

即：$\phi_1$ 生成三动量的整体平移。物理意义：动量空间中对角方向的平移不改变可观测物理量（因为物理约束面 $\phi_1=0$ 仅限制构型坐标）。

**(ii)** $\phi_2$（当施加时）也是第一类约束。$\{\phi_1,\phi_2\}=0$ 确保两者互相对合。

**(iii)** 两个第一类约束各生成一个规范自由度。在6维相空间中，2个第一类约束消去 $2 \times 2 = 4$ 个相空间维度，剩余物理自由度为 $6 - 4 = 2$ 个相空间维度——即1个构型自由度及其共轭动量。

**(iv)** 当仅施加 $\phi_1$（无 $\phi_2$）时，物理自由度为 $6 - 2 = 4$（2个构型自由度 + 2个动量）——这正是约束截面 $\Sigma$ 的余切丛 $T^*\Sigma$ 的维数。

**证明：**

(i) 第一类的定义：与所有约束（包括自身）的泊松括号在约束面上为零。由引理3.1，$\{\phi_1,\phi_1\}=\{\phi_1,\phi_2\}=0$，故 $\phi_1$ 是第一类。

$\phi_1$ 的哈密顿向量场：$X_{\phi_1} = \{\cdot, \phi_1\}$。对任意函数 $f$：

$$\{f, \phi_1\} = \sum_i \frac{\partial f}{\partial\theta_i}\frac{\partial\phi_1}{\partial p_i} - \frac{\partial f}{\partial p_i}\frac{\partial\phi_1}{\partial\theta_i} = -\sum_i \frac{\partial f}{\partial p_i}$$

因此 $X_{\phi_1} = -(\partial/\partial p_M + \partial/\partial p_C + \partial/\partial p_I)$（整体符号不影响分类）。

(ii) 同理，$\{\phi_2,\phi_2\}=0$，$\{\phi_2,\phi_1\}=0$，故 $\phi_2$ 也是第一类。

(iii) 每个第一类约束消耗2个相空间维度（Dirac约束理论的标准计数：1个约束条件 + 1个伴随规范条件）。两约束独立（$d\phi_1$ 与 $d\phi_2$ 在一般点处线性独立——$\nabla S$ 不与 $(1,1,1)$ 平行），故总消耗 $2 \times 2 = 4$ 维。剩余 $6-4=2$ 维 → 1个物理构型自由度。

**数值验证**：在裸基准点 $P_0 = (57.93^\circ, 26.16^\circ, 5.91^\circ)$ 处，$\nabla S \approx (-10.60, -71.24, -2145.16) \neq \lambda(1,1,1)$ 对任意 $\lambda$，故 $d\phi_1$ 与 $d\phi_2$ 线性独立——两约束确实独立。

(iv) 仅 $\phi_1$ 时消耗 $1 \times 2 = 2$ 维，剩余 $4$ 维 → 2个物理构型自由度。这正是 $\Sigma$ 的维数（加上共轭动量共4维）。$\square$

### 8B.3.3.4　规范自由度的物理意义

$\phi_1$ 生成的规范变换 $X_{\phi_1}$ 沿 $\partial/\partial p_M + \partial/\partial p_C + \partial/\partial p_I$。在该变换下，构型坐标不变（$\{\theta_i, \phi_1\} = 0$），三动量整体平移：

$$p_i \mapsto p_i + \varepsilon \quad (\forall i)$$

物理意义：动量空间的"对角方向" $p_M+p_C+p_I$ 是不可观测的——它对应三场在相空间中的整体相位。这与Vol-5附录C.3识别的U(1)中心子群作用一致（在构型空间中 $\theta_I$ 平移，在相空间中动量对角平移）。

$\phi_2$ 生成的规范变换 $X_{\phi_2}$ 沿 $S$ 的梯度方向在动量空间中运动：

$$X_{\phi_2} = -\sum_i \frac{\partial S}{\partial\theta_i} \frac{\partial}{\partial p_i}$$

物理意义：沿 $S$ 梯度方向的动量分量是不可观测的——它对应 $S$ 等值面族的重新参数化。两规范方向在一般点处线性独立。规范固定需要两个附加条件，将6维相空间降至2维物理相空间。

## 8B.3.4　$S$等值面约束的动力学角色

### 8B.3.4.1　$\phi_2$ 的构型依赖

在无 $\phi_2$ 的情况下，系统在 $T^*\Sigma$（4维相空间）上演化，M场有充分的动力学自由度。这是Vol-3C梯度流的经典设定。当且仅当物理构型需要 $S=S_{\text{target}}$ 锁定时，$\phi_2$ 才被施加。

### 8B.3.4.2　从 $T^*\mathcal{Q}_{\text{full}}$ 到 $\Sigma$ 的部分约化

仅施加 $\phi_1$ 时，Marsden-Weinstein约化的步骤为：

**步骤1（约束面）**：$\mathcal{C}_1 = \phi_1^{-1}(0) \subset T^*\mathcal{Q}_{\text{full}}$（5维）。

**步骤2（U(1)作用）**：$\phi_1$ 生成的整体动量平移作用 $p_i \mapsto p_i + t$ 的轨道为1维。

**步骤3（约化相空间）**：$\mathcal{C}_1 / U(1) \cong T^*\Sigma$（4维）。

这是标准结果：$T^*\mathbb{R}^3$ 被线性约束 $\sum \theta_i = \text{const}$ 和其对角的动量平移作用约化后，得到约束平面的余切丛。

### 8B.3.4.3　与Vol-4约束释放分析的对接

Vol-4 §8.5.3指出：电子受 $S=S_e$ 约束锁定（谱互锁不动点），而缪子和τ子不受此约束。在Dirac框架中：

- **电子**：$\phi_1 + \phi_2$ 均施加 → 物理自由度1（$\theta_M$ 被唯一确定在 $\theta_M^e = 57.93^\circ$ 附近）
- **缪子/τ子**：仅 $\phi_1$ 施加 → 物理自由度2（$\theta_M$ 和 $\theta_C$ 可独立变化，受梯度流驱动）

这与8B.4（出路C，形变量子化）的图景一致：电子对应 $S$ 的一个孤立本征值（$S_e$），缪子和τ子对应 $V_{\text{eff}}$ 上的不同稳态。

## 8B.3.5　U(1)动量映射的构造

### 8B.3.5.1　U(1)在未约化相空间上的提升作用

Vol-5附录C.3在构型空间 $\Sigma$ 上识别了U(1)作用：所有粒子的 $\theta_I$ 等量平移。在未约化全构型空间 $\mathcal{Q}_{\text{full}}$ 上，该作用的自然提升为：

$$\theta_M \mapsto \theta_M, \quad \theta_C \mapsto \theta_C, \quad \theta_I \mapsto \theta_I + \varepsilon \tag{5.1}$$

在余切丛上，通过余切提升，动量不变：

$$p_M \mapsto p_M, \quad p_C \mapsto p_C, \quad p_I \mapsto p_I \tag{5.2}$$

该作用在相空间上生成的向量场为：

$$\xi_{U(1)} = \frac{\partial}{\partial\theta_I} \tag{5.3}$$

余切提升保证动量不变是自然的——U(1)作用在底流形 $\mathcal{Q}_{\text{full}}$ 上的提升唯一确定其在 $T^*\mathcal{Q}_{\text{full}}$ 上的辛作用。

### 8B.3.5.2　GT-8B.3.2：动量映射候选

**构造 GT-8B.3.2（U(1)动量映射候选，构造性命题）**　定义函数 $\mu: T^*\mathcal{Q}_{\text{full}} \to \mathbb{R} \cong \mathfrak{u}(1)^*$：

$$\boxed{\mu(\theta_M,\theta_C,\theta_I,p_M,p_C,p_I) = p_I} \tag{5.4}$$

则 $\mu$ 满足动量映射的两个定义条件：

**(i) 等变性**：$\mu$ 在U(1)作用下不变（$\xi_{U(1)}(\mu) = \partial p_I/\partial\theta_I = 0$）。因为 $\mathfrak{u}(1)$ 是阿贝尔李代数，余伴随作用平凡。

**(ii) 哈密顿性**：$\iota_{\xi_{U(1)}}\omega_{\text{can}} = d\mu$。验证：

$$\iota_{\partial/\partial\theta_I} (d\theta_M \wedge dp_M + d\theta_C \wedge dp_C + d\theta_I \wedge dp_I) = dp_I$$

而 $d\mu = d(p_I) = dp_I$。两者一致。

**推广候选**：更一般的动量映射为 $\mu = \alpha_M p_M + \alpha_C p_C + \alpha_I p_I$，其中 $(\alpha_M,\alpha_C,\alpha_I)$ 为U(1)作用在构型空间上的无穷小生成元系数。对 $\theta_I$-平移作用（5.1），$(\alpha_M,\alpha_C,\alpha_I) = (0,0,1)$，故 $\mu = p_I$。

若考虑"对角"U(1)作用 $\theta_i \mapsto \theta_i + \varepsilon$（对所有 $i$），则 $\mu = p_M + p_C + p_I$。这两种U(1)作用的区别在于它们在约束面 $\phi_1=0$ 上的投影——对角作用将点移出约束面（$\sum(\theta_i+\varepsilon) = 90^\circ + 3\varepsilon \neq 90^\circ$），而 $\theta_I$-平移通过调整 $\theta_I$ 的补偿保持完备性。

### 8B.3.5.3　$\mu$ 与 $\tilde{S}_{\text{total}}$ 的关系（开放阻塞）

Vol-5附录C.7.3将 $\tilde{S}_{\text{total}}$ 识别为"Noether型守恒荷"——$\tilde{S}_{\text{total}}$ 在U(1)作用下单调变化，固定 $\tilde{S}_{\text{total}}=0$ 唯一确定U(1)参数。

在动量映射语言中，我们期望 $\mu$ 与 $\tilde{S}_{\text{total}}$ 有直接关系——即 $\mu = f(\tilde{S}_{\text{total}})$ 对某函数 $f$ 成立。这等价于：

$$\iota_{\xi_{U(1)}}\omega_{\text{can}} \stackrel{?}{=} c \cdot d\tilde{S}_{\text{total}} \tag{5.5}$$

对某个常数 $c$。在未约化相空间中：

$$d\tilde{S}_{\text{total}} = d(S - S_{\min}) = dS = \sum_i \frac{\partial S}{\partial\theta_i} d\theta_i$$

$$\iota_{\xi_{U(1)}}\omega_{\text{can}} = dp_I$$

两者不恒等——$dp_I$ 是动量空间方向，而 $dS$ 是构型空间方向。这意味着在未约化相空间中，$\mu$ 和 $\tilde{S}_{\text{total}}$ 是**不同的函数**。

**开放问题 O8B.3.1（核心阻塞）**：$p_I$（或 $p_M+p_C+p_I$）与 $\tilde{S}_{\text{total}}$ 的泊松括号关系是什么？是否存在一个正则变换将 $(p_I, \tilde{S}_{\text{total}})$ 变为共轭对？如果是，该正则变换的显式形式是什么？

**当前状态**：在约束面 $\phi_1=0$ 上，$\tilde{S}_{\text{total}}$ 沿 $\xi_{U(1)}$ 的方向导数非零（Vol-5附录C.4），这表明 $\tilde{S}_{\text{total}}$ 是U(1)作用的某种"类动量映射"。但完整对应关系的建立依赖正则变换的构造——这是 O8B.3.1 要解决的问题。

## 8B.3.6　Marsden-Weinstein约化

### 8B.3.6.1　GT-8B.3.3：从6维到2维

尽管 O8B.3.1 未封闭，我们仍可执行 Marsden-Weinstein 约化的核心步骤。

**定理 GT-8B.3.3（Marsden-Weinstein约化恢复约束截面，定理级）**　在 $\phi_2$ 施加的条件下，动量映射 $\mu = p_I$ 的水平集 $\mu^{-1}(0)$ 与约束面 $\mathcal{C} = \phi_1^{-1}(0) \cap \phi_2^{-1}(0)$ 的交集的辛约化：

$$\mathcal{C} \cap \mu^{-1}(0) / U(1) \cong T^*\Sigma_{\text{reduced}} \tag{6.1}$$

恢复约束截面上施加 $\phi_2$ 后的物理相空间（2维）。在一般点处，U(1)作用是自由的，约化相空间是合法辛流形。

**证明概要：**

**步骤1**：$\mathcal{C} = \phi_1^{-1}(0) \cap \phi_2^{-1}(0)$。在 $\mathcal{C}$ 上，$\theta_M+\theta_C+\theta_I=90^\circ$ 且 $S = S_{\text{target}}$。$\mathcal{C}$ 的维数为4（6维相空间减去2个约束）。

**步骤2**：$\mu^{-1}(0) = \{p_I = 0\}$。$\mu^{-1}(0) \cap \mathcal{C}$ 是 $\mathcal{C}$ 中满足 $p_I=0$ 的子集，维数为3。

**步骤3**：商空间 $\mathcal{C} \cap \mu^{-1}(0) / U(1)$。U(1)作用的轨道为1维（沿 $\xi_{U(1)}$ 方向）。若U(1)在 $\mathcal{C} \cap \mu^{-1}(0)$ 上的作用是自由的（在一般点处成立），则约化相空间维数为 $3 - 1 = 2$。

这2维约化相空间正是 $T^*\Sigma_{\text{reduced}}$——在 $\Sigma$ 上施加 $\phi_2$ 后的1维约束截面的余切丛。约化后的物理自由度为1个构型坐标 + 1个动量——与GT-8B.3.1(iii)的计数一致。$\square$

### 8B.3.6.2　电子构型的显式构造

在 $\mathcal{C} \cap \mu^{-1}(0)$ 上，取坐标 $(\theta_M, \theta_C, p_M, p_C)$，其中 $\theta_I = 90^\circ - \theta_M - \theta_C$，$p_I = 0$。约束 $\phi_2=0$ 给出：

$$S(\theta_M,\theta_C,90^\circ-\theta_M-\theta_C) = S_{\text{target}}$$

这是 $\Sigma$ 上的一个1维曲线（当 $S_{\text{target}}$ 固定时）。再商掉U(1)（$\theta_I$-平移对应于 $\theta_M \mapsto \theta_M - \varepsilon$，$\theta_C \mapsto \theta_C$——即沿此曲线的运动），最终得到0维构型空间（孤立点）+ 1维动量。

这正是电子构型的几何：$S=S_e$ 在 $\Sigma$ 上锁定 $\theta_M^e$（在 $\theta_C$ 由 $S=S_e$ 确定的意义下），动量自由度描述围绕该锁定位点的量子涨落。

## 8B.3.7　Dirac括号与量子化

### 8B.3.7.1　规范固定条件

两个第一类约束需要两个规范固定条件以得到良定义的Dirac括号：

**规范固定1（与 $\phi_1$ 配对）**：

$$\chi_1 \equiv p_M + p_C + p_I \approx 0 \tag{7.1}$$

物理意义：三动量的对角分量为零——消除 $\phi_1$ 生成的规范自由度。验证：$\{\phi_1, \chi_1\} = \sum_i \{\theta_i, p_i\} \cdot 1 \cdot 1 = 3 \neq 0$，故 $(\phi_1, \chi_1)$ 构成第二类约束对。

**规范固定2（与 $\phi_2$ 配对，当 $\phi_2$ 施加时）**：

$$\chi_2 \equiv \sum_i \frac{\partial S}{\partial\theta_i} \cdot p_i \approx 0 \tag{7.2}$$

物理意义：沿 $S$ 梯度方向的动量分量为零——消除 $\phi_2$ 生成的规范自由度。验证：$\{\phi_2, \chi_2\} = \sum_i (\partial S/\partial\theta_i)^2 = \|\nabla S\|^2 \neq 0$（在非驻点处），故 $(\phi_2, \chi_2)$ 构成第二类约束对。

### 8B.3.7.2　约束矩阵与Dirac括号

总约束集合（当 $\phi_2$ 施加时）：

$$\Phi = (\phi_1, \phi_2, \chi_1, \chi_2) \tag{7.3}$$

4个第二类约束在6维相空间中消去4个维度，剩余2维物理相空间。

约束矩阵 $C_{ab} = \{\Phi_a, \Phi_b\}$（$4\times4$）的显式形式：

$$C = \begin{pmatrix}
0 & 0 & 3 & \Sigma_S \\
0 & 0 & \Sigma_S & \|\nabla S\|^2 \\
-3 & -\Sigma_S & 0 & 0 \\
-\Sigma_S & -\|\nabla S\|^2 & 0 & 0
\end{pmatrix} \tag{7.4}$$

其中 $\Sigma_S \equiv \sum_i \partial S/\partial\theta_i$。

$C$ 的行列式：

$$\det C = \left[ 3\|\nabla S\|^2 - \Sigma_S^2 \right]^2$$

在一般点处 $\det C \neq 0$（因为 $3\|\nabla S\|^2 > \Sigma_S^2$，由Cauchy-Schwarz不等式：$(1,1,1) \cdot \nabla S$ 的平方 ≤ $3\|\nabla S\|^2$，等号仅当 $\nabla S \propto (1,1,1)$——即对称点或边界）。

Dirac括号：

$$\{f, g\}_D = \{f, g\} - \sum_{a,b=1}^4 \{f, \Phi_a\} C^{ab} \{\Phi_b, g\} \tag{7.5}$$

其中 $C^{ab}$ 是 $C_{ab}$ 的逆矩阵。

### 8B.3.7.3　GT-8B.3.4：量子约束实现

**构造 GT-8B.3.4（Dirac括号量子化，构造性命题）**　将Dirac括号替换为量子对易子，约束作为算子方程施加于物理态：

**(i) 典范量子化**：将相空间坐标提升为算子：

$$[\hat{\theta}_i, \hat{p}_j] = i\hbar_{\text{eff}} \delta_{ij}, \quad [\hat{\theta}_i, \hat{\theta}_j] = [\hat{p}_i, \hat{p}_j] = 0 \tag{7.6}$$

其中 $\hbar_{\text{eff}} = 1/N_1 = 1/6000$（无量纲，Vol-2 §4.5）或 $\hbar_{\text{eff}} = A_{\text{geo}}/(2\pi)$（有量纲，8B.1 GT-8B.1.8）。

**(ii) 约束实现**：物理态 $|\psi_{\text{phys}}\rangle$ 满足：

$$\hat{\phi}_1 |\psi_{\text{phys}}\rangle = 0, \quad \hat{\phi}_2 |\psi_{\text{phys}}\rangle = 0 \quad (\text{当施加时}) \tag{7.7}$$

$$\hat{\chi}_1 |\psi_{\text{phys}}\rangle = 0, \quad \hat{\chi}_2 |\psi_{\text{phys}}\rangle = 0 \quad (\text{当施加时}) \tag{7.8}$$

**(iii) 在$\theta$-表示中**（波函数 $\psi(\theta_M,\theta_C,\theta_I)$），$\hat{p}_i = -i\hbar_{\text{eff}} \partial/\partial\theta_i$：

$$\hat{\chi}_1 \psi = -i\hbar_{\text{eff}} \left(\frac{\partial}{\partial\theta_M}+\frac{\partial}{\partial\theta_C}+\frac{\partial}{\partial\theta_I}\right)\psi = 0 \tag{7.9}$$

这意味着 $\psi$ 仅依赖 $\theta_i$ 的两种独立组合（而非三个独立变量）——与 $\phi_1=0$ 的约束一起，将波函数限制在 $\Sigma$ 上。

### 8B.3.7.4　与出路C的量子等价性

**命题 8B.3.1（Dirac量子化与星积量子化的等价性，命题级）**　在 $\phi_2$ 施加且规范固定为 $\chi_1$、$\chi_2$ 的方案下，物理Hilbert空间 $\mathcal{H}_{\text{phys}} = \{|\psi\rangle : \hat{\Phi}_a|\psi\rangle = 0\}$ 与出路C（8B.4）的星积本征值方程 $S \star_{\hbar_{\text{eff}}} \psi = s\psi$ 的解空间之间存在一一对应：

$$\mathcal{H}_{\text{phys}}^{(B)} \cong \mathcal{H}_{\text{star}}^{(C)} \tag{7.10}$$

在 $\hbar_{\text{eff}} \to 0$ 的经典极限下，两路的首阶量子修正一致。

**论证概要：**

1. $\hat{\phi}_1|\psi\rangle = 0$ 限制波函数在 $\Sigma$ 上
2. $\hat{\chi}_1|\psi\rangle = 0$ 消除U(1)规范自由度
3. $\hat{\phi}_2|\psi\rangle = 0$ 即 $\hat{S}|\psi\rangle = S_{\text{target}}|\psi\rangle$——这正是星积本征值方程在 $\hbar_{\text{eff}} \to 0$ 极限下的形式（星积退化为普通乘积，$\star_{\hbar} \to \cdot$）

在 $\hbar_{\text{eff}}$ 的一阶，两路的修正均来自同一来源：约束算子与哈密顿量的非对易性（出路B）或星积的 $B_1$ 项（出路C）。Kontsevich形式定理保证两路的首阶修正由相同的图求和给出。

**诚实标注**：完整的等价性证明需要检验所有阶的修正。当前论证限于首阶（$\hbar_{\text{eff}}^1$），高阶等价性是开放问题 O8B.3.2。但首阶等价已足以进行8B.5的谐振子基准交叉验证。

### 8B.3.7.5　与出路A的概念关系

出路A（8B.2）在扩展相空间中做了类似但不同的Dirac分析：其约束 $\phi = s - S$ 与 $\chi = \theta_M+\theta_C+\theta_I-90^\circ$ 被分类为第二类约束对。这与本章的分类（均为第一类）不同——因为：

- 8B.2 的 $\phi = s-S$ **含动量**（通过 $s$ 的共轭动量 $p_s$ 的动力学），其泊松括号 $\{\phi, \chi\}$ 非零
- 本章的 $\phi_2 = S-S_{\text{target}}$ **仅含构型坐标**，与 $\phi_1$ 的典范泊松括号为零

两路的分类差异不是矛盾，而是**不同扩展策略的反映**：
- 出路A扩展了构型空间（添加 $s$），使 $\phi_2$ 变为含动量的约束 → 第二类
- 出路B保留原始构型空间，$\phi_2$ 仍为纯构型约束 → 第一类

两种策略在物理上等价——它们在约化后的物理相空间上定义了相同的辛结构和相同的量子化条件。8B.5的 $\mathcal{N}=2$ 谐振子基准将对三路（A、B、C）的首阶量子修正进行定量交叉验证。

$$E_{n_1,n_2}^{(B)} = \hbar_{\text{eff}} \left[\sqrt{\lambda_1}\left(n_1+\tfrac{1}{2}\right) + \sqrt{\lambda_2}\left(n_2+\tfrac{1}{2}\right)\right] \tag{8.3}$$

## 附录A　约束矩阵与Dirac括号的显式计算

### A.1　各对泊松括号的详细计算

约束按顺序排列：$\Phi = (\phi_1, \phi_2, \chi_1, \chi_2)$。利用典范泊松括号 $\{\theta_i, p_j\} = \delta_{ij}$，$\{\theta_i, \theta_j\} = \{p_i, p_j\} = 0$：

**$\phi_1$–行**：
- $\{\phi_1, \phi_1\} = 0$
- $\{\phi_1, \phi_2\} = 0$（两者均不含动量）
- $\{\phi_1, \chi_1\} = \sum_i \{\theta_i, p_i\} \cdot 1 \cdot 1 = 3$
- $\{\phi_1, \chi_2\} = \sum_i \{\theta_i, \sum_j (\partial S/\partial\theta_j) p_j\} = \sum_i \partial S/\partial\theta_i \equiv \Sigma_S$

**$\phi_2$–行**：
- $\{\phi_2, \phi_1\} = 0$
- $\{\phi_2, \phi_2\} = 0$
- $\{\phi_2, \chi_1\} = \sum_i \partial S/\partial\theta_i = \Sigma_S$
- $\{\phi_2, \chi_2\} = \sum_i (\partial S/\partial\theta_i)^2 = \|\nabla S\|^2$

**$\chi_1$–行**（由反对称性）：
- $\{\chi_1, \phi_1\} = -3$
- $\{\chi_1, \phi_2\} = -\Sigma_S$
- $\{\chi_1, \chi_1\} = 0$
- $\{\chi_1, \chi_2\} = 0$（两$\chi$均只含动量，其泊松括号为零）

**$\chi_2$–行**：
- $\{\chi_2, \phi_1\} = -\Sigma_S$
- $\{\chi_2, \phi_2\} = -\|\nabla S\|^2$
- $\{\chi_2, \chi_1\} = 0$
- $\{\chi_2, \chi_2\} = 0$

### A.2　退化点分析

在对称点 $(30^\circ,30^\circ,30^\circ)$，$\nabla S = 0$，$\Sigma_S = 0$，$\|\nabla S\|^2 = 0$——$C = 0$，约束矩阵完全退化（O8B.3.4）。

在裸基准点 $P_0 = (57.93^\circ, 26.16^\circ, 5.91^\circ)$：
- $\nabla S \approx (-10.60, -71.24, -2145.16)$ rad⁻¹
- $\Sigma_S = -2227.0$ rad⁻¹
- $\|\nabla S\|^2 \approx 4.61 \times 10^6$ rad⁻²
- $\det C \approx (8.84\times10^6)^2 \approx 7.8\times10^{13} \neq 0$

### A.3　Dirac括号的约化形式

在规范固定方案 $(\phi_1, \chi_1, \phi_2, \chi_2)$ 下，Dirac括号的计算简化为先消去 $(\phi_1, \chi_1)$ 再消去 $(\phi_2, \chi_2)$。

消去 $(\phi_1, \chi_1)$ 后，有效括号将 $(\theta_I, p_I)$ 变为冗余变量——$\theta_I = 90^\circ - \theta_M - \theta_C$，$p_I = -p_M - p_C$（因为 $\chi_1=0$）。这恢复了约束截面 $\Sigma$ 上的辛结构。

消去 $(\phi_2, \chi_2)$ 后，进一步将物理自由度降至2维（当 $\phi_2$ 施加时）。最终Dirac括号在约化空间上与Vol-2 §3.5的辛括号一致。
