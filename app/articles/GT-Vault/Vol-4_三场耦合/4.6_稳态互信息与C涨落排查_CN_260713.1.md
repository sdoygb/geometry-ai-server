# 稳态互信息与C涨落排查

> **核心问题**：三场在 $P_0$ 处达到联合稳态后，场之间的信息共享有多大？C场（弛豫最快，~$10^3$ s）在稳态附近是否有不可忽略的残余涨落？这些涨落是否会通过耦合通道泄漏到M场或I场，从而破坏联合稳态的稳定性？本章以**定量排查**回答：C-M与C-I互信息约$3\times10^{-3}$ bits（弱相关），M-I互信息几乎为零；C场RMS涨落约$1.58^\circ$，但全部被动力学时间标度分离与弱耦合条件压制，不威胁稳态。

## §4.6.1　联合稳态的互信息结构

### 高斯近似

在 $P_0$ 的邻域 $\mathcal{U}$（半径 $\rho_0\approx5^\circ$，GT-4.2.4）内，六项作用量 $S$ 可二次展开：

$$S(\theta) = S(P_0) + \frac{1}{2}(\theta-P_0)^T H (\theta-P_0) + \mathcal{O}(\|\theta-P_0\|^3),$$

其中一阶项 $\nabla S(P_0)=0$ 由联合稳态条件（GT-4.2.0）保证。在最高熵近似下，$P_0$ 邻域内的稳态分布为：

$$p(\delta\theta) \propto \exp\left(-\frac{1}{2}\delta\theta^T H \delta\theta\right),\quad \delta\theta = \theta - P_0.$$

该分布的协方差矩阵为 $\Sigma = H^{-1}$。

**GT-4.6.1（联合稳态互信息定理）**：在三场联合稳态 $P_0$ 处，三对场之间的互信息由Hessian矩阵 $H$ 的逆矩阵 $\Sigma=H^{-1}$ 给出：

$$I(C;M) = -\frac{1}{2}\ln(1-\rho_{CM}^2) = 0.00231\ \text{nats},$$
$$I(C;I) = -\frac{1}{2}\ln(1-\rho_{CI}^2) = 0.00253\ \text{nats},$$
$$I(M;I) = -\frac{1}{2}\ln(1-\rho_{MI}^2) = 1.83\times10^{-5}\ \text{nats},$$

其中 $\rho_{XY} = \Sigma_{XY}/\sqrt{\Sigma_{XX}\Sigma_{YY}}$ 为皮尔逊相关系数。

## §4.6.3　排查结果汇总

| 排查项目 | 结果 | 依据 |
|:---|---|:---|
| C–M互信息 | $3.33\times10^{-3}$ bits | 弱相关 $\rho_{CM}=-0.0678$ |
| C–I互信息 | $3.65\times10^{-3}$ bits | 弱相关 $\rho_{CI}=-0.0710$ |
| M–I互信息 | $2.64\times10^{-5}$ bits | 近似独立 $\rho_{MI}=0.006$ |
| C场RMS涨落 | $1.58^\circ$（相对$6.0\%$） | $\Sigma_{CC}=7.626\times10^{-4}$ |
| 固有涨落占比 | $99\%$ | $\sigma_{C\|M,I}^2/\Sigma_{CC}=0.990$ |
| C→M泄漏 | **已排除** | 时间平均压制$10^2$倍 |
| C→I泄漏 | **已排除** | 弱耦合$\gamma=0.0708$+刚度抑制 |
| M–I解耦假设自洽性 | **确认自洽** | $I(M;I)\approx0$证实Born-Oppenheimer近似有效 |

**诚实标注**：本章的互信息计算基于高斯近似的二阶展开。当 $\|\theta-P_0\| > \rho_0 \approx 5^\circ$ 时，高阶项贡献不可忽略，互信息的精确值可能偏离高斯近似。但 $\rho_0$ 由 GT-4.2.4 确定为 $\Phi$ 局部微分同胚的有效半径，在此范围内二阶展开是自洽的。C场涨落排查依赖于时间标度分离（GT-4.1.3），该分离在 $P_0$ 附近由Hessian的本征谱严格保证。
