# 夸克禁闭的颜色单态定理

## 摘要

将夸克禁闭从动力学假设（谐振子近似、指数压制）升级为**拓扑定理**。核心论证链仅依赖约束流形 $\Delta^2$ 的拓扑结构：

- **主论证（顶点奇点）**：自由色荷 = 单纯形顶点构型 → $S$-泛函发散 → 能量无穷（谱刚性）→ 自由色荷不可能作为渐近态存在。
- **辅论证（相空间维度）**：Marsden-Weinstein 约化后，N=1 相空间 0D（无动力学），N=2 相空间 1D（离散谱），N=3 相空间 2D（可自由传播）。

**核心结论**：禁闭不是 QCD 的非微扰动力学特征——它是 $\Delta^2$ 有边界这一几何事实的直接推论。

## 5.9.1 约束流形与色空间的几何对应

### 5.9.1.1 约束流形的基本结构

由公理3（全息屏编码条件）：

$$\theta_M + \theta_C + \theta_I = 90^\circ, \quad \theta_M, \theta_C, \theta_I \geq 0$$

约束流形是三维角度空间中的 **2-单纯形**：

$$S = \{(\theta_M, \theta_C, \theta_I) \in \mathbb{R}^3_{\geq 0} \mid \theta_M + \theta_C + \theta_I = 90^\circ\}$$

三个顶点：
$$V_M = (90^\circ, 0, 0), \quad V_C = (0, 90^\circ, 0), \quad V_I = (0, 0, 90^\circ)$$

重心 $P_0 = (30^\circ, 30^\circ, 30^\circ)$ 为真空构型（GT-0.0.6.3.1）。

### 5.9.1.2 色空间的几何对应

三分切丛 $T\Sigma = M \oplus C \oplus I$ 的三扇区复化给出 $\mathbb{C}^3$，配备标准厄米特度量 $h(v,w) = \sum \bar{v}_i w_i$。三个标准基矢（GT-5.1.1）：

$$\mathbf{e}_\mathcal{M} = (1,0,0), \quad \mathbf{e}_\mathcal{C} = (0,1,0), \quad \mathbf{e}_\mathcal{I} = (0,0,1)$$

**色荷的几何对应**（GT-5.1.2–5.1.4）：

- **夸克** = 单扇区激发：态矢量沿 $\mathbf{e}_\mathcal{M}$、$\mathbf{e}_\mathcal{C}$ 或 $\mathbf{e}_\mathcal{I}$ 方向
- **胶子** = $\mathfrak{su}(3)$-值联络的跨扇区涨落
- **色单态** = 三扇区等权叠加（迹元 $\sum \mathbf{e}_i \otimes \mathbf{e}_i^*$）或全反对称组合（$\varepsilon_{ijk} \mathbf{e}_i \otimes \mathbf{e}_j \otimes \mathbf{e}_k$）

## 5.9.2 顶点-色态对应定理

**顶点-色态对应定理（GT-5.9.1）**

单扇区激发（纯色态）与约束流形 $S$ 的顶点一一对应：

- 纯 $\mathcal{M}$ 态 ↔ $V_M = (90^\circ, 0, 0)$
- 纯 $\mathcal{C}$ 态 ↔ $V_C = (0, 90^\circ, 0)$
- 纯 $\mathcal{I}$ 态 ↔ $V_I = (0, 0, 90^\circ)$

**证明**：

单扇区激发意味着态矢量沿 $\mathbb{C}^3$ 的一个基矢方向（GT-5.1.1）。在 ℰ 物理映射约定下（GT-2.0.1）：

- $\mathcal{M}$ 扇区 = 质量扇区，参数化为 $\theta_M$
- $\mathcal{C}$ 扇区 = 电荷扇区，参数化为 $\theta_C$
- $\mathcal{I}$ 扇区 = 信息/相位扇区，参数化为 $\theta_I$

当态矢量纯沿 $\mathbf{e}_\mathcal{M}$ 时，$\mathcal{C}$ 和 $\mathcal{I}$ 扇区对态没有贡献 → $\theta_C = \theta_I = 0$。由公理3 → $\theta_M = 90^\circ$。同理可证其余两个顶点。$\square$

**物理直觉**：一个纯色（如纯红）的夸克，其物理构型仅涉及一个扇区。在约束流形上，这对应走到一个顶点——只有一个角度非零，其余两个为零。

## 5.9.3 顶点奇点与禁闭

### 5.9.3.1 S-泛函的顶点发散

**顶点发散发散引理（GT-5.9.2）**

在任意顶点 $V_i$ 处，六项作用量 $S$-泛函发散至无穷：$S(V_i) \to +\infty$。

**证明**（以 $V_M = (90^\circ, 0, 0)$ 为例）：

六项作用量（公理2）：
$$S(\theta_M, \theta_C, \theta_I) = \sum_{i \in \{M,C,I\}} \frac{1}{\sin^2\theta_i} + \sum_{i<j} \frac{1}{\sin\theta_i \sin\theta_j}$$

代入 $\sin 90^\circ = 1$，$\sin 0^\circ = 0$：

$$S(V_M) = \frac{1}{1^2} + \frac{1}{0^2} + \frac{1}{0^2} + \frac{1}{1 \cdot 0} + \frac{1}{0 \cdot 0} + \frac{1}{1 \cdot 0}$$

每一项含 $1/0$ 或 $1/0^2$ 都发散 → $S \to +\infty$。$\square$

**发散阶数**：设 $\theta_C = \varepsilon_1$，$\theta_I = \varepsilon_2$，$\theta_M = 90^\circ - \varepsilon_1 - \varepsilon_2$（$\varepsilon_1, \varepsilon_2 \to 0^+$）。主导发散项来自 $1/\sin^2\theta_C$、$1/\sin^2\theta_I$ 和 $1/(\sin\theta_C \sin\theta_I)$：

$$S \sim \frac{1}{\varepsilon_1^2} + \frac{1}{\varepsilon_2^2} + \frac{1}{\varepsilon_1\varepsilon_2}$$

对称路径 $\varepsilon_1 = \varepsilon_2 = \varepsilon$ 下：$S \sim 3/\varepsilon^2$。

### 5.9.3.2 能量-几何谱刚性

**能量-几何谱刚性引理（GT-5.9.3）**

在几何论框架内，$S$-泛函的发散直接导致物理能量发散：

$$\lim_{S \to \infty} E_{\text{phys}} = \infty$$

**证明**：

谱刚性定理（GT-0.0.5）保证 $S$-泛函的 Hessian 谱与物理 Hamiltonian 谱之间是一一对应的，该对应由桥接函数唯一性（GT-2.5.1）确保严格。

$S \to \infty$ 意味着至少两个扇区的 $\sin\theta_i \to 0$ → 有效度规在这些方向上的 Hessian 分量 $\lambda_i^{\text{eff}} / \sin^2\theta_i \to \infty$ → Hamiltonian 期望值发散。$\square$

### 5.9.3.3 禁闭拓扑定理

**禁闭拓扑定理（GT-5.9.4）**

在几何论框架内，自由色荷（单扇区激发）不能作为渐近自由态存在。所有物理可观测的渐近态必须是色单态。

**证明**：

1. 自由色荷 = 单扇区激发 = 顶点构型（GT-5.9.1）
2. 顶点构型处 $S \to \infty$（GT-5.9.2）
3. $S \to \infty$ → $E \to \infty$（GT-5.9.3）
4. 无穷大能量的态不能作为物理渐近态存在（量子场论渐近态基本定义）
5. 因此自由色荷不存在于渐近谱中。所有可观测态必为色单态（重心附近远离边界的构型）。$\square$

**定理力度**：**定理**。证明链中每一步都是定义或更基础的定理，无谐振子近似、无 Darboux 线性化、无数值参量依赖。论证完全依赖于约束流形 $\Delta^2$ 有边界这个拓扑事实。

### 5.9.3.4 物理直觉：为什么是"拓扑的"

$\Delta^2$ 是一个**有边界的紧致空间**。三个顶点是边界上的三个奇点——$S$-泛函在顶点处发散。

- 色单态（重心附近的构型）远离边界 → $S$ 有限 → 能量有限 → 可作为自由粒子存在
- 纯色态试图走到顶点 → $S$ 发散 → 能量无穷 → 不可能

**这不是"势能囚禁"——没有势阱。这是"几何禁止"——$\Delta^2$ 的边界就是隔离自由色荷的拓扑壁垒。**

## 5.9.4 相空间维度论证

主定理证明了自由色荷因能量发散而无法存在。本节从 Marsden-Weinstein 约化的相空间维度提供互补的几何论证。

### 5.9.4.1 N=1：无动力学

**N=1 维度灾难定理（GT-5.9.5）**

对 $N=1$（单个夸克），Marsden-Weinstein 约化后的相空间维数为 $0$。自由夸克不能作为动力学系统存在。

**证明**：

1. $\Sigma_{\text{joint}}^{(1)}$：原始维数 $=2$（$\theta_M, \theta_I$，$\theta_C$ 由公理3确定）
2. $S_{\text{total}}=0$ 约束：固定一个组合 → $1$ 维构型空间
3. U(1) 中心子群 Marsden-Weinstein 商：生成元 $\xi_{\text{U}(1)} = \partial/\partial\theta_I$，轨道 $1$ 维 → 商空间 $0$ 维
4. $0$ 维相空间 → 无动力学自由度。$\square$

**物理含义**：单个夸克连"运动"都做不到——不是一个物理系统。这比"被囚禁"更强：根本不存在。

### 5.9.4.2 N=2：无连续自由度

**介子离散谱定理（GT-5.9.6）**

对 $N=2$（夸克-反夸克对），Marsden-Weinstein 约化后的有效相空间维数为 $1$（奇数）。无法承载非退化辛形式。介子只存在离散本征值谱，不能作为连续自由度传播。

**证明**：

1. $\Sigma_{\text{joint}}^{(2)}$：原始维数 $=3$（$\theta_M, \theta_I^{(1)}, \theta_I^{(2)}$）
2. $S_{\text{total}}=0$ 约束：$2$ 维构型空间
3. U(1) 商：$1$ 维构型空间
4. 有效相空间为 $1$ 维——U(1) 约束和色单态条件联合作用导致动量自由度被完全压制（GT-4.3.3 §2.1）
5. 奇数维相空间无法承载非退化辛形式 → 介子谱离散，不能连续传播。$\square$

**物理含义**：
- 介子可以在 $\Delta^2$ 内部（远离顶点）以低能量存在
- 它们不能携带**净色荷**——两个夸克的色荷相互抵消
- 介子谱离散——不是"势阱"导致，而是**相空间拓扑**导致

### 5.9.4.3 N=3：可传播性

**重子可传播性定理（GT-5.9.7）**

$N=3$ 色单态构型（重子）拥有 $2$ 维约化构型空间和 $4$ 维约化相空间——可以承载连续动力学。重子可以作为自由粒子传播。

**证明**：

1. $\Sigma_{\text{joint}}^{(3)}$：原始维数 $=4$（$\theta_M, \theta_I^{(1)}, \theta_I^{(2)}, \theta_I^{(3)}$）
2. $S_{\text{total}}=0$ 约束：$3$ 维构型空间
3. U(1) 商：$2$ 维构型空间
4. 相空间 $= 2 \times 2 = 4$ 维 → 可承载非退化辛形式（GT-4.3.3 §2.2）
5. 重子可承载连续动力学 → 可作为自由粒子传播。$\square$

### 5.9.4.4 三阶段的统一图景

**色单态必然性推论（GT-5.9.7.1）**

由 GT-5.9.4–5.9.7 联合得：

| $N$ | 构型 | 相空间维数 | 物理态 | 原因 |
|:---:|:---|:---:|:---|:---|
| 1 | 自由夸克 | 0 | ❌ 不存在 | 无动力学自由度 |
| 2 | 夸克-反夸克对 | 1（奇数） | ✅ 介子（离散谱） | 无连续自由度 + 色单态必然 |
| 3 | 三夸克 | 4（偶数+辛） | ✅ 重子（自由传播） | 连续自由度 + 色单态必然 |
| $>3$ | 多夸克 | $2(N-1)$ | ⚠️ 色单态可传播 | 多体截面（GT-4.5.1） |

**所有非色单态 → 能量发散 → 不存在（GT-5.9.4）。所有色单态的可传播性由相空间维度决定。**

## 5.9.5 Wilson 圈与面积律的几何翻译

### 5.9.5.1 标准定义回顾

在 QCD 中，Wilson 圈定义为：

$$W(C) = \frac{1}{N_c} \operatorname{tr} \mathcal{P} \exp\left(ig \oint_C A_\mu^a T^a dx^\mu\right)$$

面积律 $\langle W(C) \rangle \sim \exp(-\sigma \cdot \text{Area}(C))$ 是禁闭的标准判据，其中 $\sigma$ 为弦张量。

### 5.9.5.2 几何论中的翻译

在几何论框架内，Wilson 圈对应约束流形 $S$ 上一条闭合路径 $C$ 的 **Berry 相位累积**。对于非色单态构型，路径 $C$ 趋近 $S$ 的边界（顶点区域），Berry 联络的曲率发散。

**Wilson 圈面积律命题（GT-5.9.8）**

在约束流形 $S$ 上，非色单态构型的 Berry 相位累积满足面积律：

$$\mathcal{A}(C) \propto \oint_C \langle\psi|\mathrm{d}|\psi\rangle \propto \text{Area}_{\text{geo}}(C)$$

其中 $\text{Area}_{\text{geo}}(C)$ 是路径 $C$ 在约束流形度规下扫过的几何面积。

**论证构造**：

1. Berry 曲率 $F_{ij} = \partial_{[i} A_{j]}$ 在 $S$ 内部有限，在边界发散（顶点区域 $1/\sin^2\theta \to \infty$）
2. 非色单态构型的路径 $C$ 必然穿过高曲率区域（因接近顶点）
3. 穿过高曲率区域的面积积分贡献主导 → 面积律定性成立

**诚实标注**：此为**构造性命题**——定性结构正确，但从 Hessian 谱刚性到精确弦张量 $\sigma$ 的完整推导链尚未建立。这是 G5a 开放问题。

## 5.9.6 $\alpha_s$ 跑动不是禁闭的必要条件

### 5.9.6.1 QCD 的标准叙事

在 QCD 中，禁闭通常与 $\alpha_s$ 的红外发散绑定：
- 渐近自由（$\beta_0 > 0$）→ $\alpha_s(Q^2)$ 随 $Q^2$ 减小而增大
- 当 $\alpha_s \gtrsim 1$ 时，微扰展开失效 → 推断禁闭发生

但严格来说，$\alpha_s$ 的红外增长是**禁闭的充分条件，不是必要条件**。

### 5.9.6.2 几何论的不同策略

强耦合常数定理（GT-5.4.4）给出了 $\alpha_s$ 的几何标度公式：

$$\alpha_s = \sqrt{\frac{N\lambda_1^{\text{eff}}}{\lambda_2^{\text{eff}}}} \cdot \sin\theta_M \approx 0.1192$$

**这个 $\alpha_s$ 是常数**——它来自 Hessian 谱刚性，不随能标跑动。

禁闭不需要 $\alpha_s$ 跑动。禁闭来自 $\Delta^2$ 的拓扑（有边界），不是来自耦合常数的红外行为。$\alpha_s$ 是一个固定的几何常数，而禁闭仍然成立——因为它是拓扑的，不是动力学的。

### 5.9.6.3 $\alpha_s$ 跑动的几何翻译方向

**$\alpha_s$ 跑动方向（GT-5.9.9）**

$\alpha_s(Q^2)$ 的"跑动"不是耦合常数的本征变化，而是探测尺度的几何投影效应：
- 高 $Q^2$（短距离）：探测到约束流形 Hessian 的约化值 → $\alpha_s \approx 0.1192$（裸几何值）
- 低 $Q^2$（长距离）：色单态约束的投影效应使有效耦合增强 → 等效于 $\alpha_s$ 增大

**诚实标注**：此为**研究方向**，非定理。从 GT-5.4.4 的常数 $\alpha_s$ 到 QCD 跑动函数的完整推导链尚未建立。

**现状**：GT-5.9.8 给出了定性结构（面积律 = Berry 曲率在边界附近的累积）。精确 $\sigma$ 值需要：
1. 顶点附近 Berry 曲率的精确行为（量子修正）
2. 从联合截面 Hessian 谱到物理能标 $\sigma$ 的桥接

**优先度**：中等。GT-5.9.4 已足够证明禁闭——弦张量是锦上添花。
