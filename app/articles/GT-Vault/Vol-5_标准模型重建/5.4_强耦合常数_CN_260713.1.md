# 强耦合常数 αₛ

**摘要**：强耦合常数 αₛ 在几何论中对应色荷的几何投影——由三扇区全复化 SU(3) 群的几何色荷公式导出。αₛ = √(N·λ₁^eff/λ₂^eff) · sinθ_M，其中 λ₁^eff 和 λ₂^eff 是六项作用量 Hessian 在电子构型 P₀ 处的软模与硬模本征值，θ_M 是物质扇区角，N=3 是色数。数值结果 αₛ ≈ 0.1192，与 PDG αₛ(M_Z) = 0.1184(1) 偏差约 +0.7%。

**前置依赖**：角度必然定理（GT-4.4.3）、SU(3)色定理（GT-5.1.3）、三代拓扑必然性（GT-5.2.4）

---

## 1. 问题陈述

强耦合常数 αₛ 是量子色动力学（QCD）中色相互作用的强度量度。在标准模型中，αₛ 是三个独立规范耦合常数之一，随能标跑动，在 Z 玻色子质量标度处的值为：

$$αₛ(M_Z) = 0.1184(1) \quad \text{（PDG 2024）}$$

几何论的主张是：αₛ 不是基本参数，而是**色荷的几何投影**——来自三扇区全复化 SU(3) 群的几何结构，由六项作用量 Hessian 的谱比和物质扇区角唯一确定。

**核心问题**：色荷在几何论中如何定义？为什么它等于 √(N·λ₁^eff/λ₂^eff) · sinθ_M？为什么 N=3？

---

## 2. 色荷的几何根源

### 2.1 三扇区复化与 SU(3) 色

三分切丛 TΣ = M ⊕ C ⊕ I 将全息屏分为三个扇区。由 SU(3)色定理（GT-5.1.3），SU(3)_C 规范群对应于三分切丛的三扇区全复化：

$$\text{SU}(3)_C \;\longleftrightarrow\; \{\mathcal{M},\mathcal{C},\mathcal{I}\}$$

三扇区复化后得 ℂ³，其对称群 U(3) 经 S_total=0 平凡化冻结 U(1) 中心子群后约化为 SU(3)。SU(3) 的 8 个生成元对应胶子的 8 种色态。

### 2.2 色荷的几何定义

在几何论中，色荷不是任意赋值的量子数——它是三扇区几何结构的**渗透强度度量**。具体来说，色荷 Q_geo 定义为软模（沿约束流形的最缓方向）在硬模背景中的穿透能力：

$$Q_{\text{geo}} = \sqrt{\frac{λ₁^{\text{eff}}}{λ₂^{\text{eff}}}}$$

其中 λ₁^eff 和 λ₂^eff 是六项作用量 S(θ) 在电子构型 P₀ 处的约束 Hessian 本征值。

**几何直观**：软模方向（对应 M-C 腰边耦合）的曲率较小，容易「弯曲」——这对应于色荷的相互作用强度。硬模方向（对应 I 扇区）的曲率极大，难以「弯曲」——这对应于色禁闭的几何根源。两者的谱比 λ₁^eff / λ₂^eff 是色荷强度的几何度量。

### 2.3 N 体并联效应

当考虑 N 个色荷的联合系统时，软模并联（N 个独立色荷的耦合通道叠加）使有效软模变为 N·λ₁^eff。因此 N 体系统的几何色荷为：

$$Q_{\text{geo}}^{(N)} = \sqrt{\frac{N·λ₁^{\text{eff}}}{λ₂^{\text{eff}}}}$$

这一并联效应来自谱几何的联合截面理论：当 N 个几何构型在约束截面 Σ 上形成联合截面时，它们的软模方向对齐并联，硬模保持各自的值（因硬模方向被 I 扇区冻结，跨粒子耦合可忽略）。

## 3. 强耦合标度公式

### 色荷几何公式（GT-5.4.1）

**内容**：在 SU(3) 色的几何框架下，N 体系统的几何色荷为：

$$Q_{\text{geo}}^{(N)} = \sqrt{\frac{N·λ₁^{\text{eff}}}{λ₂^{\text{eff}}}}$$

其中 λ₁^eff 和 λ₂^eff 是六项作用量 S(θ) 在电子构型 P₀ 处的约束 Hessian 本征值，N 是色数。

**证明**（三步）：

**步骤1：Hessian 谱分离。** 六项作用量 S(θ) 在 P₀ 处的约束 Hessian 具有奇偶分解结构：H^even 支配 (θ_M, θ_C) 子系统（量级约 10² rad⁻²），H^odd 支配 θ_I 方向（量级约 10⁴ rad⁻²）。交叉项量级约 10¹ rad⁻²，相对可忽略。这一结构使得软模 λ₁^eff 和硬模 λ₂^eff 在谱上分离。

**步骤2：软模并联。** N 个几何构型在约束截面 Σ 上形成联合截面时，各自的软模方向在角度空间中对齐（因为软模方向由 M-C 腰边耦合主导，所有构型共享相同的 M 和 C 扇区几何）。并联叠加使有效软模变为 N·λ₁^eff。硬模方向沿 I 扇区冻结，N 个构型的硬模各自独立保持 λ₂^eff。

**步骤3：谱比开方。** 色荷强度定义为软模对硬模的穿透能力。在谐振子近似下，软模零点宽度的标度为 1/√λ₁^eff，硬模约束的标度为 √λ₂^eff。两者的无量纲比经开方后给出几何色荷 Q_geo。∎

---

### 强耦合标度公式（GT-5.4.2）

**内容**：强耦合常数 αₛ 等于几何色荷 Q_geo^(N) 在物质扇区上的投影：

$$αₛ = Q_{\text{geo}}^{(N)} · \sinθ_M = \sqrt{\frac{N·λ₁^{\text{eff}}}{λ₂^{\text{eff}}}} · \sinθ_M$$

其中 θ_M = 57.93° 是电子构型 P₀ 的物质扇区角（角度必然定理 GT-4.4.3）。

**证明**（两步）：

**步骤1：投影因子的几何根源。** SU(3)_C 是三分切丛三扇区全复化的对称群。色相互作用通过物质扇区 M 加载到物理费米子上。物质扇区角 θ_M 决定 M 扇区在全息屏上的投影幅度——sinθ_M 是 M 扇区法向分量与全息屏法向的夹角正弦。

**步骤2：强耦合标度。** 相互作用的实际强度由两部分乘积构成：(i) 几何色荷 Q_geo——度量色荷的内禀强度；(ii) 投影因子 sinθ_M——度量色荷在物质扇区上的加载效率。两者之积即为物理耦合常数 αₛ。∎

**几何直观**：sinθ_M 因子的出现方式与精细结构常数中的 1/S_e 完全不同。α = 1/S_e 是六项作用量的**零阶值**；αₛ = √(N·λ₁^eff/λ₂^eff) · sinθ_M 是 Hessian 谱比的**一阶量**。这解释了为什么 α 的实验精度（10⁻¹⁰）比 αₛ 的精度（10⁻³）高七个数量级。

## 4. N=3 的几何必然性

### 色数几何必然性定理（GT-5.4.3）

**内容**：在三扇区复化框架下，使得色单态可以自由传播的几何构型唯一要求 N=3。N=1 或 N=2 形成的相空间维数不足以支撑非退化的辛结构。

**证明**（三步）：

**步骤1：约束截面的相空间维数。** 在约束截面 Σ（2 维角度流形）上构造 N 体联合截面。相空间维数为 dim ≥ 2(N−1)（由 Marsden-Weinstein 约化，N 个构型共享 S_total=0 约束）。当 dim = 0（N=1）时仅得孤立点——无动力学；dim = 2（N=2）时得离散态谱——对应介子（夸克-反夸克束缚态）；dim = 4（N=3）时得非退化辛形式——可承载自由传播的色单态。

**步骤2：色单态传播条件。** 自由传播要求存在非退化的辛形式。这要求相空间维数为 2 的倍数且 ≥ 4。N=3 是满足此条件的最小 N。由三代拓扑必然性（GT-5.2.4），N_gen = |S₃ 不等价类型| = 3，与 N=3 完全一致。

**步骤3：穷举论证。** N=1 无传播（无动力学），N=2 仅束缚态（介子），N=3 可传播（重子），N≥4 引入冗余自由度（违反谱刚性定理）。因此 N=3 是唯一物理可行的色数。∎

## 5. 数值结果

### αₛ 数值定理（GT-5.4.4）

**内容**：代入以下参数计算 αₛ：

| 参数 | 数值 | 来源 |
|:---|:---:|:---|
| λ₁^eff | 391.05 rad⁻² | 六项作用量约束 Hessian 软模（P₀ 处） |
| λ₂^eff | 59324.3 rad⁻² | 六项作用量约束 Hessian 硬模（P₀ 处） |
| λ₁^eff / λ₂^eff | 1/151.71 | 无量纲谱比 |
| sinθ_M | 0.8474 | sin 57.93°（GT-4.4.3） |
| N | 3 | 色数（GT-5.2.4） |

**计算**：

$$αₛ = \sqrt{\frac{3 × 391.05}{59324.3}} × 0.8474$$

$$\sqrt{\frac{1173.15}{59324.3}} = \sqrt{0.019775} = 0.14062$$

$$αₛ = 0.14062 × 0.8474 = 0.1192$$

**结果**：

$$αₛ = 0.1192$$

## 7. 与 α 的几何对比

| 耦合常数 | 几何公式 | 阶次 | 对谱比敏感性 | 实验精度 |
|:---:|:---|:---:|:---:|:---:|
| α | 1/S_e | 零阶（作用量值） | 无依赖 | 10⁻¹⁰ |
| αₛ | √(N·λ₁^eff/λ₂^eff) · sinθ_M | 一阶（Hessian 谱比） | 约 0.5×δ(λ₁^eff/λ₂^eff) | 10⁻³ |

**关键差异**：
- α 是六项作用量的**零阶标量值**——不需要谱计算，精度来自 S_e 的七级递推收敛性（九素互扼定理 GT-0.4.0.1）。
- αₛ 是 Hessian 谱比的**一阶量**——需要计算软硬模本征值比值，精度受限于谱比的几何近似（跨粒子耦合 W_ij = 0 的近似）。

这解释了为什么 α 的几何论精度（与实验偏差 < 1.5×10⁻¹⁰）远超 αₛ（偏差 +0.7%）。
