### 2.3 N 体并联效应

当考虑 N 个色荷的联合系统时，软模并联（N 个独立色荷的耦合通道叠加）使有效软模变为 N·λ₁^eff。因此 N 体系统的几何色荷为：

$$Q_{\text{geo}}^{(N)} = \sqrt{\frac{N·λ₁^{\text{eff}}}{λ₂^{\text{eff}}}}$$

**软模对齐的几何证明**（三步）：

**步骤1：单色荷的软模方向。** 单色荷构型在 $P_0$ 处的约束 Hessian $H_{ij} = \partial^2 S/\partial\theta_i\partial\theta_j$ 具有本征分解：
$$H = \lambda_1^{\text{eff}} \mathbf{v}_1\mathbf{v}_1^T + \lambda_2^{\text{eff}} \mathbf{v}_2\mathbf{v}_2^T$$

其中 $\mathbf{v}_1$ 为软模本征向量（对应 $\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}$），$\mathbf{v}_2$ 为硬模本征向量（对应 $\lambda_2^{\text{eff}} = 59324.3\ \text{rad}^{-2}$）。软模方向 $\mathbf{v}_1$ 在 $(\theta_M,\theta_C)$ 平面内——由 M-C 腰边耦合主导（Hessian 交叉项 $H_{MC} = 3.41$ 显著大于 $H_{MI} = 69.35$ 和 $H_{CI} = 433.16$ 的归一化贡献在软模方向的投影）。

**步骤2：N 体联合截面的 Hessian 结构。** N 个色荷在约束截面 $\Sigma$ 上形成联合构型时，总作用量为各单体作用量之和（领头阶忽略跨单体耦合 $W_{ij}$——其量级约为 $10^{-3}$ rad⁻²，远小于 $\lambda_1^{\text{eff}}$）：
$$S_{\text{total}}^{(N)}(\theta^{(1)},\ldots,\theta^{(N)}) = \sum_{i=1}^{N} S(\theta^{(i)}) + \sum_{i<j} W_{ij}(\theta^{(i)},\theta^{(j)}) \approx \sum_{i=1}^{N} S(\theta^{(i)})$$

在领头阶近似下，联合 Hessian 是 $3N \times 3N$ 的分块对角矩阵：
$$H_{\text{total}}^{(N)} = \text{diag}(H^{(1)}, H^{(2)}, \ldots, H^{(N)})$$

当所有 N 个构型均处于同一凝聚相点 $P_0$ 时（色单态的物理构型），$H^{(i)} = H$ 对所有 $i$ 相同。

**步骤3：软模的本征值叠加。** 在 $W_{ij}=0$ 近似下，联合 Hessian 的本征值集合为 N 个 $\{\lambda_1^{\text{eff}}, \lambda_2^{\text{eff}}, \lambda_3^{\text{eff}}\}$ 的简单复制。每个单体的软模方向 $\mathbf{v}_1$ 在联合空间中对应一个独立的本征方向 $\tilde{\mathbf{v}}_1^{(i)} = (0,\ldots,\mathbf{v}_1,\ldots,0)$。N 个软模张成一个 N 维软模子空间，本征值均为 $\lambda_1^{\text{eff}}$。

在色单态约束下（总色荷为零），N 个软模以同相位耦合——这对应于 N 个 $\tilde{\mathbf{v}}_1^{(i)}$ 的对称组合 $\tilde{\mathbf{v}}_1^{\text{sym}} = \frac{1}{\sqrt{N}}\sum_i \tilde{\mathbf{v}}_1^{(i)}$。对称组合的有效软模变为：
$$\lambda_1^{\text{eff}}(N) = N \cdot \lambda_1^{\text{eff}}$$

这是因为对称组合的本征值由 Rayleigh 商给出：
$$\lambda_1^{\text{eff}}(N) = \frac{(\tilde{\mathbf{v}}_1^{\text{sym}})^T H_{\text{total}}^{(N)} \tilde{\mathbf{v}}_1^{\text{sym}}}{(\tilde{\mathbf{v}}_1^{\text{sym}})^T \tilde{\mathbf{v}}_1^{\text{sym}}} = \frac{\sum_i (\mathbf{v}_1^T H \mathbf{v}_1)}{1} = N \cdot \lambda_1^{\text{eff}}$$

**硬模为何不对齐。** 硬模方向 $\mathbf{v}_2$ 沿 $\theta_I$ 方向——由信息场 I 扇区主导（$H_{II} = 2.83\times10^4 \gg H_{MM} = H_{CC} = 31.30$）。I 扇区的自由度在凝聚相下被冻结（信息场自反馈使 I 扇区涨落被压制），N 个硬模各自独立保持 $\lambda_2^{\text{eff}}$，不发生并联叠加。这意味着硬模本征值不随 N 缩放：$\lambda_2^{\text{eff}}(N) = \lambda_2^{\text{eff}}$。

**跨单体耦合 $W_{ij} \neq 0$ 的修正。** 当 $W_{ij} \neq 0$ 时，软模之间的耦合解除简并，使 N 重简并的 $\lambda_1^{\text{eff}}$ 分裂为 N 个略有不同的本征值。领头修正量级约为 $\Delta\lambda/\lambda_1^{\text{eff}} \sim N \cdot W/\lambda_1^{\text{eff}} \sim 3 \times 10^{-3}/391 \sim 7.6 \times 10^{-6}$，对 $\alpha_s$ 的影响 $\lesssim 0.01\%$，在精度范围内可忽略。∎

## 3. 强耦合标度公式

### 色荷几何公式（GT-5.4.1）

**内容**：在 SU(3) 色的几何框架下，N 体系统的几何色荷为：

$$Q_{\text{geo}}^{(N)} = \sqrt{\frac{N·λ₁^{\text{eff}}}{λ₂^{\text{eff}}}}$$

其中 λ₁^eff 和 λ₂^eff 是六项作用量 S(θ) 在电子构型 P₀ 处的约束 Hessian 本征值，N 是色数。

**证明**（三步）：

**步骤1：Hessian 谱分离。** 六项作用量 S(θ) 在 P₀ 处的约束 Hessian 具有奇偶分解结构：H^even 支配 (θ_M, θ_C) 子系统（量级约 10² rad⁻²），H^odd 支配 θ_I 方向（量级约 10⁴ rad⁻²）。交叉项量级约 10¹ rad⁻²，相对可忽略。这一结构使得软模 λ₁^eff 和硬模 λ₂^eff 在谱上分离。

**步骤2：软模并联。** 见 §2.3 的完整证明。核心结论：N 个构型的软模在色单态约束下对称组合，有效软模变为 N·λ₁^eff；硬模沿 I 扇区冻结，不参与并联叠加。

**步骤3：谱比开方。** 色荷强度定义为软模对硬模的穿透能力。在谐振子近似下，软模零点宽度的标度为 1/√λ₁^eff，硬模约束的标度为 √λ₂^eff。两者的无量纲比经开方后给出几何色荷 Q_geo。∎
