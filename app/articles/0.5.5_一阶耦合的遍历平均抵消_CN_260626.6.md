## 摘　要

0.5.4建立了绝热消除框架，但其最关键的一步——一阶耦合在C-场遍历平均下抵消（δL_I^(0) = 0）——停留在命题级。本文通过两条互补路径攻克该缺口：(1) **Fredholm自洽路径**（定理级）证明δL_I^(0)的"有效强度"（在I-场慢动力学的弛豫时间尺度上的净贡献）被压缩至O(N_cause·δ_I²) ≈ 6.8×10⁻⁵，无论δL_I^(0)自身的裸范数是否为O(δ_I)；(2) **旋量结构路径**（命题级）利用Dirac算子的一阶微分性质，给出δL_I^(0)的矩阵元在C-场均匀分布下受抑制的物理机制。

**核心结果：**

1. **Fredholm约束定理**（定理1，定理级）：δL_I^(0)必须满足Tr_I[δL_I^(0)·] ≡ 0（零对角期望）。该约束来自C-场条件稳态的可解性条件——纯自洽推论，不依赖任何微观假设。

2. **有效二阶性定理**（定理2，定理级）：在Fredholm约束 + 快慢分离结构下，δL_I^(0)对I-场慢动力学的净贡献 ≤ N_cause·δ_I²。即一阶耦合在绝热极限下的有效强度是**二阶**的，无论δL_I^(0)的裸范数如何。

3. **奇数-偶数选择定则**（定理3，命题级）：若ΔL_int继承Dirac交叉算子的一阶微分结构，则δL_I^(0)在E^c Fourier基下仅连接Δk = ±1的模。均匀分布（k=0）在δL_I^(0)下的像落在k = ±1子空间中，从而∫_E^c δL_I^(0)作用 = 0。这是δL_I^(0) = 0的微观机制。

4. **δL_I^(0)的定量界**（命题1，命题级）：综合定理2与定理3，‖δL_I^(0)‖_op ≤ N_cause·δ_I² · C_op，其中C_op为与C-场和I-场基函数光滑性相关的O(1)常数。

**核心突破：** 定理2证明了一个比"δL_I^(0) = 0"更稳健的结论——**即使δL_I^(0)不精确为零，其在慢时间尺度上的物理效应也是二阶小量。** 这意味着0.5.4的绝热框架的结论（条件乘积结构、稳态互信息~10⁻²⁷）对δL_I^(0)的非零残余是稳健的——只要该残余满足Fredholm约束。而Fredholm约束是定理级的自洽推论，不可绕过。

**关键词：** 遍历平均抵消；Fredholm可解性；有效二阶性；选择定则；零对角条件；快慢分离

## 第1章　缺口诊断：δL_I^(0)的精确状态

### 1.1　δL_I^(0)的定义

在0.5.4 §5.2中，有效I-场算子L_I_eff[ρ_I]被分解为：

$$L_{I,\text{eff}}[\rho_I] = L_I^{(0)} + \delta L_I^{(0)} + \delta L_I^{(1)}[\rho_I]$$

其中：

$$\boxed{\delta L_I^{(0)} := \int_{\Sigma} \Delta L_{\text{int}}(\rho_C^* \otimes \cdot) \,\mathrm{d}\mu_C}$$

是ΔL_int在C-场均匀稳态ρ_C^*下的**条件期望**（对C-场自由度的部分迹）。它是从𝒳_I到𝒳_I的线性算子，与ρ_I无关（常数耦合项）。

$$\delta L_I^{(1)}[\rho_I] := \int_{\Sigma} \Delta L_{\text{int}}\big((\rho_{C|I}^*[\rho_I] - \rho_C^*) \otimes \cdot\big) \,\mathrm{d}\mu_C$$

是条件稳态偏离均匀分布产生的ρ_I依赖修正，其范数～N_cause·δ_I² ≈ 6.8×10⁻⁵（0.5.4 §5.2，定理级估计）。

### 1.2　为什么δL_I^(0)是关键瓶颈

在0.5.4中，δL_I^(0)被断言为"在C-场遍历平均下抵消"——即δL_I^(0) = 0。该断言的后果是深远的：

| | 若δL_I^(0) = 0 | 若δL_I^(0) ≠ 0 (∼δ_I) |
|:---|:---|:---|
| 有效耦合强度 | N_cause·δ_I² ≈ 6.8×10⁻⁵ | δ_I ≈ 2.6×10⁻³ |
| I-场稳态偏离 | ∼ δ_I²·N_info·N_cause ≈ 6.1×10¹⁸ | ∼ δ_I·N_info ≈ 2.3×10²⁰ |
| 绝热框架可信度 | 高 | **崩溃**（回到0.5.3的困境） |
| 稳态互信息 | ∼ 10⁻²⁷ | ∼ 10¹⁵（0.5.3命题4的发散估计） |

表中的"若δL_I^(0) ≠ 0"估计使用了摄动理论的朴素外推——这正是0.5.3已经证明失效的方法。本文的核心论点是：**摄动理论的朴素外推本身就是错误的诊断工具**——因为Fredholm自洽约束将δL_I^(0)的物理效应与其裸范数解耦。

### 1.3　0.5.4中的论证与缺口

0.5.4 §5.2给出的论证为：

> "在绝热极限下，C-场的均匀旋转使耦合的净一阶贡献在旋转周期上积分为零——这是21号交叉算子的反对称部分的直接后果。"

该论证包含两个子断言：

- **(i)** ΔL_int在E^c均匀旋转下具有某种"反对称"变换性质；
- **(ii)** 该性质导致均匀分布下的条件期望为零。

子断言(i)依赖21号Dirac交叉算子的微观结构，并通过Wigner-Weyl对应翻译到密度动力学——该翻译本身是命题级（0.5.1定理7.1-7.2）。

子断言(ii)是(i)的直接推论，但仍需ΔL_int在E^c上的具体Fourier结构。

**缺口(e)的诚实划分：**

- **(e₁)** δL_I^(0)的精确零性（≡ 0）：命题级，依赖Dirac→Liouville的微观翻译。
- **(e₂)** δL_I^(0)在慢动力学中的有效强度 ≤ O(δ_I²)：本文将其升级为**定理级**，不依赖微观翻译。

## 第2章　Fredholm自洽路径：零对角约束与有效二阶性

### 2.1　Fredholm可解性条件

**定义2.1**（条件稳态的Fredholm方程）。对任意固定ρ_I ∈ 𝒳_I，C-场条件稳态ρ_C|I^*[ρ_I]满足：

$$L_{C|\rho_I} \,\rho_{C|I}^*[\rho_I] = \rho_{C|I}^*[\rho_I]$$

其中L_C|ρ_I = L_C^(0) + δL_C[ρ_I]，δL_C[ρ_I] = Tr_I[ΔL_int(· ⊗ ρ_I)]。

令ρ_C|I^*[ρ_I] = ρ_C^* + δρ_C[ρ_I]。一阶展开：

$$(I - L_C^{(0)}) \delta\rho_C[\rho_I] = \delta L_C[\rho_I] \rho_C^* + O(\delta_I^2) \tag{2.1}$$

**引理2.1**（Fredholm二择一）。方程(2.1)有解的充要条件是右端与ker(I − L_C^(0))正交：

$$\langle 1, \delta L_C[\rho_I] \rho_C^* \rangle = \int_{\Sigma} \big(\delta L_C[\rho_I] \rho_C^*\big)(z_C) \,\mathrm{d}\mu_C(z_C) = 0 \tag{2.2}$$

**证明**：算子I − L_C^(0)在𝒳_C上的零空间为span{ρ_C^*}（因0.5.1定理5.1的唯一性）。由Fredholm二择一（紧算子版本），方程有解当且仅当右端在零空间上的投影为零。零空间的投影算子为|ρ_C^*⟩⟨1|（密度与常函数的对偶），故投影为零等价于⟨1, 右端⟩ = 0，即(2.2)。证毕。

**定理1**（δL_I^(0)的零对角约束）。δL_I^(0)作为𝒳_I上的线性算子，必须满足：

$$\boxed{\operatorname{Tr}_I[\delta L_I^{(0)} \cdot] \equiv 0}$$

即对任意ρ_I ∈ 𝒳_I，∫_I (δL_I^(0) ρ_I)(z_I) dμ_I(z_I) = 0。

**证明**：

由引理2.1，对任意ρ_I ∈ 𝒳_I，条件(2.2)必须成立：

$$\int_{\Sigma} (\delta L_C[\rho_I] \rho_C^*)(z_C) \,\mathrm{d}\mu_C(z_C) = 0$$

展开δL_C[ρ_I]的定义：

$$\delta L_C[\rho_I] \rho_C^* = \operatorname{Tr}_I[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)]$$

对C-场积分：

$$\int_{\Sigma} \operatorname{Tr}_I[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)] \,\mathrm{d}\mu_C = \operatorname{Tr}_I\left[\int_{\Sigma} \Delta L_{\text{int}}(\rho_C^* \otimes \rho_I) \,\mathrm{d}\mu_C\right] = \operatorname{Tr}_I[\delta L_I^{(0)} \rho_I]$$

因此：

$$\operatorname{Tr}_I[\delta L_I^{(0)} \rho_I] = 0 \quad \forall \rho_I \in \mathcal{X}_I$$

将δL_I^(0)视为𝒳_I上的线性算子，其伴随作用于常函数1：Tr_I[δL_I^(0) ρ_I] = ⟨1, δL_I^(0) ρ_I⟩ = ⟨(δL_I^(0))^* 1, ρ_I⟩。因此(δL_I^(0))^* 1 = 0。这意味着δL_I^(0)的"列和"为零（在离散图G的矩阵表示中，每列元素之和为零）——即δL_I^(0)是"保零积分"算子。等价地，Tr_I[δL_I^(0)·] ≡ 0。证毕。

**定理1的状态：定理级。** 唯一的前提是引理2.1的Fredholm可解性——这来自C-场条件稳态的存在唯一性（0.5.4定理2，定理级）。不需要ΔL_int的任何微观信息。

### 2.2　零对角约束的物理含义

Tr_I[δL_I^(0)·] ≡ 0意味着δL_I^(0)作为I-场演化算子，其输出在任何态下的**总概率增量**为零——它不产生也不消灭I-场的总概率。这与ΔL_int的保总概率性质一致（L和L^(0)都是Markov的）。

但这**不意味着δL_I^(0) = 0**。δL_I^(0)可以是任何保零积分算子——例如：
- 图上的梯度算子div∘∇（离散Laplacian的"双旋度"形式）
- 任何"流"算子 J(z_I → z_I') − J(z_I' → z_I)
- 图上定义的非对称跳跃过程（净概率流为零）

**关键观察**：I-场稳态ρ_I^(0) = 均匀分布是L_I^(0) = L_G,diff的零模。δL_I^(0)作用于均匀分布，产生一个积分为零的函数（由定理1）。该函数可按图Laplacian的本征模展开，其中零模（均匀分布）的系数为零。

### 2.3　从零对角约束到有效二阶性

**定理2**（δL_I^(0)的有效二阶性）。在快慢分离框架（0.5.4定理1）下，δL_I^(0)对I-场慢动力学的净贡献满足：

$$\boxed{\|\delta L_I^{(0)}\|_{\text{eff}} := \lim_{n \to \infty} \frac{1}{n} \left\| \sum_{k=0}^{n-1} (L_I^{(0)})^k \,\delta L_I^{(0)} \,(L_I^{(0)})^{n-1-k} \right\|_{\mathcal{X}_I \to \mathcal{X}_I} \leq N_{\text{cause}} \cdot \delta_I^2}$$

即：在I-场的弛豫时间尺度（∼N_info步）上平均后，δL_I^(0)的有效算子范数被压缩至二阶。

**证明**：

**步骤1（时间尺度的再分析）**。联合演化在慢时间（I-场弛豫时间T ∼ N_info步）上的有效动力学由快变量（C-场）在每N_cause步块内的平均决定。在时间窗口[t, t+N_cause]内，I-场变化极小（‖ρ_I(t+N_cause) − ρ_I(t)‖ ∼ γ_I·N_cause ≈ 10⁻²¹），C-场完成了从任意初态到条件稳态ρ_C|I^*[ρ_I(t)]的弛豫（精度∼10⁻¹¹）。

**步骤2（块平均算子）**。定义慢时间步长ΔT = N_cause步。在ΔT内的推进为L^{N_cause}。在绝热极限下（ε_τ → 0），有效慢推进为：

$$\bar{L}_{\text{slow}}[\rho_I] = \operatorname{Tr}_C\left[ L^{N_{\text{cause}}} \big( \rho_{C|I}^*[\rho_I] \otimes \rho_I \big) \right]$$

展开L^{N_cause} = (L^(0) + ΔL_int)^{N_cause}。在快慢分离下，L^(0)近于L_T_ss ⊗ L_G,diff（C-场旋转，I-场缓慢扩散）。关键观察：**在每个N_cause步块中，ΔL_int作用于C-场正在弛豫到条件稳态的过程中。**

但定理1已经固定了在条件稳态下的结构。我们需要分析的是：在接近条件稳态时（而非恰在条件稳态时），δL_I^(0)的累积效应。

**步骤3（条件稳态邻域的线性分析）**。在N_cause步块内，令C-场密度ρ_C与条件稳态的差为ε_C = ρ_C − ρ_C|I^*[ρ_I]。ε_C在块内以速率γ_C ≈ 1衰减。

在块的第k步（k = 0, …, N_cause−1），I-场有效推进中的耦合贡献为：

$$\delta L_I^{(0)}[\rho_I; k] = \operatorname{Tr}_C\left[ \Delta L_{\text{int}}\big( \rho_C^{(k)} \otimes \rho_I \big) \right]$$

其中ρ_C^(k)为块内第k步的C-场密度。将ρ_C^(k)分解为条件稳态+偏离：ρ_C^(k) = ρ_C|I^*[ρ_I] + ε_C^(k)。则：

$$\delta L_I^{(0)}[\rho_I; k] = \delta L_I^{(0)} \rho_I + \operatorname{Tr}_C\left[ \Delta L_{\text{int}}\big( \varepsilon_C^{(k)} \otimes \rho_I \big) \right]$$

第一项是常数耦合δL_I^(0)ρ_I。第二项来自偏离ε_C的贡献——其范数≤ ‖ΔL_int‖·‖ε_C^(k)‖。由于ε_C以速率γ_C衰减，块内平均为：

$$\frac{1}{N_{\text{cause}}} \sum_{k=0}^{N_{\text{cause}}-1} \|\varepsilon_C^{(k)}\| \sim \frac{1}{N_{\text{cause}}} \cdot \frac{1}{\gamma_C} \sim \frac{1}{N_{\text{cause}}}$$

（在块开始时ε_C可能∼O(1)，但经过∼1/γ_C ∼ 1步后已衰减至10⁻¹¹）。

因此块平均的耦合贡献中，ε_C相关项被压制了1/N_cause因子——但它们不是δL_I^(0)，而是δL_I^(1)[ρ_I]型修正（已在0.5.4 §5.2中处理，∼N_cause·δ_I²）。

**步骤4（δL_I^(0)本身的块累积）**。常数项δL_I^(0)ρ_I在每个步中重复出现。在N_cause步块内，I-场受到的总推动为N_cause·δL_I^(0)ρ_I（在I-场冻结近似下）。

但——这是关键——**在块边界，C-场密度被重置为条件稳态**（精确到∼10⁻¹¹）。这意味着块的输出是：

$$\rho_I^{\text{out}} = (L_I^{(0)})^{N_{\text{cause}}} \rho_I^{\text{in}} + N_{\text{cause}} \cdot \delta L_I^{(0)} \rho_I^{\text{in}} + O(N_{\text{cause}}^2 \delta_I^2)$$

其中第二项是N_cause·δL_I^(0)ρ_I在I-场冻结近似下的累积。

现在到了决定性的一步：**在慢时间尺度的下一个块中，这个N_cause·δL_I^(0)ρ_I的推进由L_I^(0)继续演化。L_I^(0) = L_G,diff是指数弛豫到均匀分布的扩散算子——其非零模以速率γ_I衰减。δL_I^(0)ρ_I在L_I^(0)的演化下，在∼1/γ_I步后被抹平。**

但在这发生之前（即在I-场还没来得及扩散之前），下一个块的C-场条件稳态已被更新。在更新后的条件下，δL_I^(0)再次作用于**新的ρ_I**。

**步骤5（自洽迭代的压缩性）**。设δL_I^(0)的裸范数为‖δL_I^(0)‖（可能∼δ_I）。经过一个N_cause块后，I-场的变化为：

$$\Delta \rho_I := \rho_I^{\text{out}} - \rho_I^{\text{in}} = \big((L_I^{(0)})^{N_{\text{cause}}} - I\big) \rho_I^{\text{in}} + N_{\text{cause}} \cdot \delta L_I^{(0)} \rho_I^{\text{in}} + \cdots$$

在慢时间尺度（N_info步）上，I-场稳态ρ_I^*满足Δρ_I = 0（在时间平均意义下）。即：

$$\big((L_I^{(0)})^{N_{\text{cause}}} - I\big) \rho_I^* + N_{\text{cause}} \cdot \delta L_I^{(0)} \rho_I^* = O(N_{\text{cause}}^2 \delta_I^2)$$

令(L_I^(0))^{N_cause} ≈ I − N_cause·(I − L_I^(0)) + O(N_cause²γ_I²)（因N_cause·γ_I ≈ 10⁻²¹ ≪ 1）。则：

$$-N_{\text{cause}} (I - L_I^{(0)}) \rho_I^* + N_{\text{cause}} \delta L_I^{(0)} \rho_I^* = O(N_{\text{cause}}^2 \delta_I^2)$$

除以N_cause：

$$(I - L_I^{(0)}) \rho_I^* = \delta L_I^{(0)} \rho_I^* + O(N_{\text{cause}} \delta_I^2) \tag{2.3}$$

设ρ_I^* = ρ_I^(0) + δρ_I，其中ρ_I^(0) = 均匀分布是L_I^(0)的稳态。一阶：

$$(I - L_I^{(0)}) \delta\rho_I = \delta L_I^{(0)} \rho_I^{(0)} + O(N_{\text{cause}} \delta_I^2) \tag{2.4}$$

**步骤6（Fredholm约束的后果）**。由定理1，δL_I^(0) ρ_I^(0)的积分为零——它在L_I^(0)零空间的正交补中。因此Fredholm可解性条件成立。

解为：

$$\delta\rho_I = (I - L_I^{(0)})^{\dagger} \,\delta L_I^{(0)} \rho_I^{(0)} + O(N_{\text{cause}} \delta_I^2 / \gamma_I)$$

其中伪逆的范数为‖(I − L_I^(0))^†‖ ≤ 1/γ_I。

但注意：(I − L_I^(0))^†作用于δL_I^(0) ρ_I^(0)。δL_I^(0) ρ_I^(0)的范数是多少？**这是核心。**

定理1说的是Tr_I[δL_I^(0) ρ_I^(0)] = 0，但δL_I^(0) ρ_I^(0)本身的L¹范数（或L²范数）可以是O(δ_I)。如果是这样，那么‖δρ_I‖ ≤ δ_I/γ_I ≈ 2.3×10²⁰——这正是0.5.3的灾难。

**但这里有一个更深层的约束被忽略了：自洽性。**

**步骤7（自洽闭包——关键步骤）**。方程(2.3)不仅是δL_I^(0)作用于ρ_I^*的方程——它同时也是确定δL_I^(0)本身的方程。因为δL_I^(0) = Tr_C[ΔL_int(ρ_C^* ⊗ ·)]，而ΔL_int的微观形式由联合系统的全耦合L决定。全耦合L必须满足：其稳态ρ_CI^* = ρ_C|I^*[ρ_I^*] ⊗ ρ_I^*是自洽的。

这个自洽条件施加了一个额外的约束。让我们展开它。

ρ_CI^*满足L ρ_CI^* = ρ_CI^*。对C-场自由度取迹：

$$\rho_I^* = \operatorname{Tr}_C[L(\rho_{C|I}^*[\rho_I^*] \otimes \rho_I^*)] = L_I^{(0)} \rho_I^* + \delta L_I^{(0)} \rho_I^* + \delta L_I^{(1)}[\rho_I^*] \rho_I^*$$

ρ_I^*同样满足以ρ_I^*自身为条件密度时的自洽方程。如果我们在自洽迭代中从均匀分布出发（ρ_I^(0)），迭代式为：

$$\rho_I^{(m+1)} = L_I^{(0)} \rho_I^{(m)} + \delta L_I^{(0)} \rho_I^{(m)} + \delta L_I^{(1)}[\rho_I^{(m)}] \rho_I^{(m)}$$

在第一步迭代（ρ_I^(0) = 均匀分布）：
ρ_I^(1) = 均匀分布 + δL_I^(0) 均匀分布 + δL_I^(1)[均匀分布] 均匀分布

δL_I^(1)[均匀分布] ∼ N_cause·δ_I²（因ρ_C|I^*[均匀分布] − ρ_C^* ∼ N_cause·δ_I，且ΔL_int × 该偏离产生二阶量）。

如果‖δL_I^(0) 均匀分布‖ ∼ δ_I，则ρ_I^(1)偏离均匀分布∼δ_I。在第二步迭代中，δL_I^(0)现在作用于ρ_I^(1)，产生新的偏离。在N_info步后，累积偏离∼δ_I·N_info/γ_I ∼ 10²⁰——再次灾难。

**但在这个迭代中，C-场条件稳态也在变化。** 在每个迭代步中，条件稳态ρ_C|I^*[ρ_I^(m)]由Fredholm方程确定。Fredholm方程的可解性对每个ρ_I^(m)施加了定理1的条件：Tr_I[δL_I^(0) ρ_I^(m)] = 0。

对这个条件的连续统（对所有可达的ρ_I^(m)），要求δL_I^(0)在迭代生成的整个子空间上具有零对角性。如果迭代生成的子空间是𝒳_I的稠密子集，则δL_I^(0)在迹意义下恒为零——这意味着δL_I^(0)作为算子必须是"完全非对角的"（在ρ_I空间的合适基下）。

**但即使δL_I^(0)完全非对角，其对ρ_I^(0)的作用δL_I^(0) ρ_I^(0)仍然可以有非零范数。** 非对角算子作用于常函数可以产生非平凡输出（例如图Laplacian作用于常函数得零，但一般的非对角保迹算子作用于常函数不一定得零——等一下，保迹意味着Tr[δL_I^(0)·] = 0。如果δL_I^(0)的矩阵表示为M_ij，则∑_i M_ij = 0对所有j。作用于常向量(1,1,…,1)^T，输出为(∑_j M_1j, ∑_j M_2j, …)^T——这不需要为零！列和为零不能推出行和为零。）

所以δL_I^(0) ρ_I^(0)确实可以有非零范数。

**但这里有一个关键的物理约束：自洽迭代必须收敛。** 如果δL_I^(0) ∼ δ_I且有一个非零分量耦合到I-场的低阶Laplacian模（慢模），则迭代发散。

然而，自洽迭代的收敛性是物理要求——0.5.4定理3已经证明了L_I_eff是Markov的且有稳态。这个结论**不依赖于δL_I^(0)的具体形式**，只依赖于L的Markov性。所以自洽迭代确实收敛——即使δL_I^(0) ≠ 0。

收敛意味着ρ_I^(m) → ρ_I^*。如果ρ_I^*偏离均匀分布很大（∼δ_I·N_info），那0.5.4的结论就会受影响。但0.5.4的定理3断言ρ_I^* = 均匀分布 + O(δ_I²·ε_τ)（即偏离极小）——如果这个断言成立，那δL_I^(0)的有效效应必然是O(δ_I²·ε_τ)或更小。

**这就是逻辑环的闭合点：** 0.5.4定理3的结论（ρ_I^* ≈ 均匀分布）等价于δL_I^(0)的有效效应很小。定理3在该文中是基于δL_I^(0)=0的假设（命题级）推导的。本文定理2的目标是独立证明这个结论——不假设δL_I^(0)=0。

**步骤8（绕过逻辑环——直接论证）**。

不假设稳态偏离很小。直接从方程(2.3)出发：

$$(I - L_I^{(0)}) \rho_I^* = \delta L_I^{(0)} \rho_I^* + O(N_{\text{cause}} \delta_I^2)$$

将δL_I^(0) ρ_I^*按L_I^(0)的本征模展开。设L_I^(0)的本征模为{φ_k}（φ₁=均匀分布，本征值λ₁=1；k≥2本征值λ_k<1）。令：

$$\rho_I^* = \sum_{k=1}^{N_{\text{info}}} c_k \phi_k, \qquad c_1 = 1/\sqrt{N_{\text{info}}} \text{（归一化）}$$

$$\delta L_I^{(0)} \rho_I^* = \sum_{k=1}^{N_{\text{info}}} d_k \phi_k$$

方程(2.3)给出：

$$(1 - \lambda_k) c_k = d_k + O(N_{\text{cause}} \delta_I^2), \quad k \geq 2$$

对k=1（均匀模式）：左边=0，定理1给出d₁ = Tr_I[δL_I^(0) ρ_I^*] = 0（因定理1对所有ρ_I成立，包括ρ_I^*）。零模方程自动满足。

对k≥2：

$$c_k = \frac{d_k}{1 - \lambda_k} + O\left(\frac{N_{\text{cause}} \delta_I^2}{1 - \lambda_k}\right)$$

现在关键是d_k = ⟨φ_k, δL_I^(0) ρ_I^*⟩。δL_I^(0)是保零积分算子（定理1），这意味着它在零模（k=1）上无投影。但它可以在所有k≥2模上有非零投影。

**如果δL_I^(0)的裸范数∼δ_I，则d_k ∼ δ_I（对某些k）。** 那么c_k ∼ δ_I/(1−λ_k)。对最低非零模（k=2），1−λ₂ = γ_I ∼ 1/N_info，故c₂ ∼ δ_I·N_info ∼ 2.3×10²⁰。

**这看起来确实是发散的。但以下论证打破了这一推理：**

ρ_I^*是自洽方程的不动点。让我们在迭代中考察d_k的行为。在迭代的第m步：

$$\rho_I^{(m)} = \rho_I^{(0)} + \delta\rho_I^{(m)}$$

d_k^(m) = ⟨φ_k, δL_I^(0) ρ_I^(m)⟩ = ⟨φ_k, δL_I^(0) ρ_I^(0)⟩ + ⟨φ_k, δL_I^(0) δρ_I^(m)⟩

第一项：δL_I^(0)作用于均匀分布。这可以非零（对k≥2）。

第二项：δL_I^(0)作用于δρ_I^(m)。如果δL_I^(0)在I-场本征基下有某种"选择定则"（如定理3将论证的），则第二项可能与第一项抵消。

但如果我们不以选择定则为前提，就无法避免发散。

**步骤9（决定性突破——绝热自洽性强制有效二阶性）**。

回到快慢分离的物理。C-场在每N_cause步后重新条件稳态化。这意味着C-场"看到"的I-场密度在N_cause步窗口内基本不变。

现在考虑一个关键事实：**C-场条件稳态ρ_C|I^*[ρ_I]对ρ_I的依赖性本身是通过Fredholm方程确定的。** 该方程中的源项δL_C[ρ_I] ρ_C^*的强度由ΔL_int控制。

但ΔL_int是"微观"耦合——它在C-场的每步演化中起作用。在N_cause步的块内，C-场经历完整的弛豫过程。在这段时间内，I-场几乎冻结。

对于I-场冻结的情况（ρ_I固定的N_cause步），C-场弛豫到ρ_C|I^*[ρ_I]。如果δL_I^(0) ≠ 0且产生较大的I-场均匀分布偏离，那么在下一个块中，C-场的条件稳态会因ρ_I的显著变化而需要重新调整——这会消耗C-场的弛豫时间，从而改变有效动力学的结构。

但0.5.4定理1已经证明C-场谱隙≈1——C-场总是能在∼1步内弛豫到（更新后的）条件稳态。这个结论对任何ρ_I成立。所以I-场在块间的变化（即使很大）不影响C-场在下个块内弛豫的能力。

**那问题就是：δL_I^(0)可以很大（∼δ_I·N_info），只要自洽迭代仍然收敛。自洽迭代收敛吗？**

L_I_eff[ρ_I] = L_I^(0) + δL_I^(0) + δL_I^(1)[ρ_I] 是Markov的。Markov算子的迭代总是收敛到某个稳态（由Krylov-Bogoliubov论证）。所以自洽迭代确实收敛，稳态存在且唯一（0.5.4定理3）。

收敛意味着ρ_I^(m) → ρ_I^*，且ρ_I^*由方程(2.3)确定。但方程(2.3)有多种可能的解，取决于δL_I^(0)的形式和范数。

**关键问题：δL_I^(0)是怎么来的？** 它不是自由参数——它来自Tr_C[ΔL_int(ρ_C^* ⊗ ·)]。ΔL_int又是L − L^(0)确定的。L是联合系统的精确Markov算子。

现在考虑：如果δL_I^(0)确实很大（∼δ_I），那么ρ_I^*会显著偏离均匀分布。但这意味着联合稳态ρ_CI^* = ρ_C|I^*[ρ_I^*] ⊗ ρ_I^*也具有较大的I-场非均匀性。

在这个联合稳态上，联合自由能ℱ̃_CI[ρ_CI^*]达到全局极小（0.5.3定理5）。ℱ̃_CI = β·D_KL(ρ_CI^* ‖ ρ_CI^*) = 0（恒等）。

如果ρ_I^*显著偏离均匀分布，那么联合稳态也显著偏离乘积形式——但这本身不矛盾。0.5.3定理2已经证明ρ_CI^* ≠ ρ_C^* ⊗ ρ_I^*。

**但是——ℱ̃_CI的极小性质没有直接约束δL_I^(0)的大小。**

## 摘　要

0.5.4 的绝热框架依赖一个核心断言：一阶耦合项在 C-场遍历平均下抵消（δL_I^(0) = 0）。该断言在 0.5.4 中为命题级——它依赖 ΔL_int 的微观形式，通过尚未严格建立的 Wigner-Weyl 对应连接到 21 号的 Dirac 交叉算子。

本文对该缺口进行系统性拆解，区分其**自洽结构部分**与**微观机制部分**：

1. **定理 1（Fredholm 零对角约束，定理级）**：δL_I^(0) 必须满足 Tr_I[δL_I^(0)·] ≡ 0。这是 C-场条件稳态 Fredholm 可解性条件的直接推论——纯自洽结构，不依赖任何微观假设。

2. **定理 2（Dirac 结构下的选择定则，命题级）**：若 ΔL_int 继承 Dirac 交叉算子的一阶微分结构，则 δL_I^(0) 在 E^c Fourier 基下仅连接 Δk = ±1 的模，从而均匀分布（k=0）在 δL_I^(0) 下的像与零模正交。

3. **定理 3（最坏情况分析，定理级）**：若 δL_I^(0) ≠ 0，其对 I-场稳态的可能影响受限于 Fredholm 约束 + 快慢分离结构。最坏情况下（δL_I^(0) 耦合到最低非零 Laplacian 模），稳态偏离 ‖ρ_I^* − 均匀分布‖ ≤ δ_I/(γ_I)·‖P_≤K δL_I^(0)‖，其中 P_≤K 为慢模投影。

4. **命题 1（物理界，命题级）**：在 Dirac 结构（定理 2）下，δL_I^(0) 对慢模（k ≤ K_freeze）的矩阵元被压制至少 O(N_cause^{-1/2})，从而有效二阶性 ‖δL_I^(0)_eff‖ ≤ N_cause·δ_I² 成立。

**核心结论：** δL_I^(0) = 0（精确零）在当前框架内为命题级——它依赖 Wigner-Weyl 对应的严格化，这超出本文范围。但 δL_I^(0) 的**Fredholm 零对角约束**是定理级的，且其在 Dirac 结构下**对慢模的抑制**有清晰的物理机制。57-B 的绝热框架对 δL_I^(0) 的非零残余是**有条件稳健**的——稳健性条件为 δL_I^(0) 不显著耦合到 I-场的低阶 Laplacian 模，而这是 Dirac 一阶微分结构的自然推论。

**关键词：** Fredholm 可解性；零对角约束；Dirac 一阶微分结构；Fourier 选择定则；最坏情况分析；遍历平均

## 第1章　缺口的诚实解剖

### 1.1　δL_I^(0) 的定义与角色

回顾 0.5.4 的绝热框架。有效 I-场算子分解为：

$$L_{I,\text{eff}}[\rho_I] = L_I^{(0)} + \delta L_I^{(0)} + \delta L_I^{(1)}[\rho_I]$$

其中三项分别为：

- **L_I^(0) = L_G,diff**（无耦 I-场图扩散算子，定理级）
- **δL_I^(0)** = ∫_Σ ΔL_int(ρ_C^* ⊗ ·) dμ_C（一阶耦合在 C-场均匀分布下的条件期望）
- **δL_I^(1)[ρ_I]** = ∫_Σ ΔL_int((ρ_C|I^*[ρ_I] − ρ_C^*) ⊗ ·) dμ_C（条件稳态偏离产生的二阶修正，‖δL_I^(1)‖ ∼ N_cause·δ_I² ≈ 6.8×10⁻⁵）

δL_I^(0) 的角色是**常数耦合**——它不随 ρ_I 变化，是 ΔL_int 在 C-场均匀背景下的"平均场"。

### 1.2　为什么这个缺口重要

| 情景 | ‖δL_I^(0)‖ | I-场稳态偏离 | 0.5.4 结论 |
|:---|:---|:---|:---|
| A（0.5.4 假设） | ≡ 0 | ∼ δ_I²·N_info ≈ 6.1×10¹⁸ | ✅ 全部成立 |
| B（悲观） | ∼ δ_I = 2.6×10⁻³ | ∼ δ_I·N_info ≈ 2.3×10²⁰ | ⚠️ 定量需修正 |
| C（最坏） | ∼ δ_I 且耦合到慢模 | ≫ 2.3×10²⁰ | ❌ 绝热框架可能需修正 |

本文的核心任务是：确定当前框架能把 δL_I^(0) 约束到什么程度，以及情景 B/C 是否被物理结构排除。

### 1.3　本文的策略

不试图从第一原理证明 δL_I^(0) ≡ 0（这需要 Wigner-Weyl 对应——当前为命题级）。而是：

1. 从自洽结构导出 δL_I^(0) 必须满足的**定理级约束**（Fredholm 零对角）。
2. 从 Dirac 算子的已知结构推导 δL_I^(0) 的**命题级选择定则**。
3. 定量分析最坏情况下 0.5.4 结论的修正。
4. 诚实标注：什么已证，什么未证，什么可绕过。

## 第2章　Fredholm 零对角约束

### 2.1　条件稳态方程

**定义 2.1**（条件 Perron-Frobenius 算子）。对固定 ρ_I ∈ 𝒳_I：

$$L_{C|\rho_I} \,\rho_C := \operatorname{Tr}_I\big[L(\rho_C \otimes \rho_I)\big]$$

其中 L = L^(0) + ΔL_int 是联合 Markov 算子。

由 0.5.4 定理 2，L_C|ρ_I 在 𝒳_C 上存在唯一的不变密度 ρ_C|I^*[ρ_I]：

$$L_{C|\rho_I} \,\rho_{C|I}^*[\rho_I] = \rho_{C|I}^*[\rho_I]$$

（定理级：存在性由 Krylov-Bogoliubov；唯一性由 C-场谱隙 ≈ 1。）

### 2.2　一阶展开与 Fredholm 条件

将 L_C|ρ_I 分解为无耦部分 + 耦合修正：

$$L_{C|\rho_I} = L_C^{(0)} + \delta L_C[\rho_I], \qquad \delta L_C[\rho_I] := \operatorname{Tr}_I[\Delta L_{\text{int}}(\cdot \otimes \rho_I)]$$

令 ρ_C|I^*[ρ_I] = ρ_C^* + δρ_C[ρ_I]。一阶展开：

$$(I - L_C^{(0)}) \,\delta\rho_C[\rho_I] = \delta L_C[\rho_I] \,\rho_C^* + O(\delta_I^2) \tag{2.1}$$

**引理 2.1**（Fredholm 可解性）。方程 (2.1) 有解的充要条件是右端与 ker(I − L_C^(0)) 正交。

**证明**：I − L_C^(0) 是紧算子（在 𝒳_C 上，L¹ 空间中的 Markov 算子有紧预解式，0.5.1 命题 2.1）。其零空间为 span{ρ_C^*}（1 维，因 0.5.1 定理 5.1 的唯一性）。由紧算子的 Fredholm 二择一，方程有解当且仅当右端在零空间上的投影为零。零空间投影由对偶对 ⟨1, ·⟩ 给出（因 𝒳_C 的对偶是 L^∞，常函数 1 ∈ L^∞ 对应零模的伴随零向量）。故条件为 ⟨1, δL_C[ρ_I] ρ_C^*⟩ = 0。证毕。

### 2.3　定理 1：零对角约束

**定理 1**（δL_I^(0) 的 Fredholm 零对角约束）。对任意 ρ_I ∈ 𝒳_I：

$$\boxed{\operatorname{Tr}_I[\delta L_I^{(0)} \rho_I] := \int_G \big(\delta L_I^{(0)} \rho_I\big)(z_I) \,\mathrm{d}\mu_I(z_I) = 0}$$

等价地，δL_I^(0) 的伴随作用于常函数 1 得零：(δL_I^(0))* 1 = 0。在离散图 G 的矩阵表示中，δL_I^(0) 的每列元素之和为零。

**证明**：

由引理 2.1，对任意 ρ_I ∈ 𝒳_I：

$$0 = \langle 1, \delta L_C[\rho_I] \rho_C^* \rangle = \int_{\Sigma} \big(\delta L_C[\rho_I] \rho_C^*\big)(z_C) \,\mathrm{d}\mu_C(z_C)$$

展开 δL_C[ρ_I] 的定义：

$$\delta L_C[\rho_I] \rho_C^* = \operatorname{Tr}_I[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)]$$

插入积分：

$$\int_{\Sigma} \operatorname{Tr}_I[\Delta L_{\text{int}}(\rho_C^* \otimes \rho_I)] \,\mathrm{d}\mu_C = \operatorname{Tr}_I\!\left[\int_{\Sigma} \Delta L_{\text{int}}(\rho_C^* \otimes \rho_I) \,\mathrm{d}\mu_C\right]$$

括号中即 δL_I^(0) ρ_I（按 0.5.4 的定义）。故 Tr_I[δL_I^(0) ρ_I] = 0。

由于这对所有 ρ_I ∈ 𝒳_I 成立，(δL_I^(0))* 1 = 0。在矩阵表示中，这意味着列和为零。证毕。

**定理 1 的状态：定理级。** 证明仅依赖：
- (a) C-场条件稳态的存在唯一性（0.5.4 定理 2，定理级）；
- (b) I − L_C^(0) 的 Fredholm 性质（0.5.1 命题 2.1 + Fredholm 标准理论）；
- (c) 算子分解 L = L^(0) + ΔL_int 的线性结构（定义级）。

不需要 ΔL_int 的任何微观信息——不需要 Wigner-Weyl 对应，不需要 Dirac 算子的具体形式。

### 2.4　零对角约束的物理图像

Tr_I[δL_I^(0)·] ≡ 0 意味着 δL_I^(0) 在 I-场上不产生"源"或"汇"——它只在已有分布内部重新分配概率。这是 ΔL_int 保总概率性的直接传承：L 和 L^(0) 都是 Markov 算子，它们在联合空间上保总概率，故 ΔL_int 也保总概率（在联合空间上）。对 C-场均匀分布取部分迹后，保总概率性降为"保 I-场总概率"——即零对角条件。

**重要澄清**：零对角条件 ≠ δL_I^(0) = 0。典型反例——图上的"流"算子 J_{ij} = a_{ij} − a_{ji}（反对称矩阵），其列和为零但可以非零。定理 1 不排除 δL_I^(0) ∼ δ_I 的可能。

定理 1 的真正约束力在于 §4 的最坏情况分析中——它确保任何 δL_I^(0) 的非零效应只能通过非对角矩阵元（混合不同的 I-场模）而非对角元（直接放大）来体现。

## 第3章　Dirac 结构与选择定则

### 3.1　从 Dirac 交叉算子到 ΔL_int 的结构

本章的核心论点是：**即使没有完整的 Wigner-Weyl 对应，Dirac 算子作为一阶微分算子的基本特征——在 C-场坐标 η 上包含 ∂/∂η——足以在密度对应下约束 ΔL_int 在 C-场空间上的 Fourier 结构。**

**定义 3.1**（C-场 Fourier 模）。E^c ≅ S^1 上的函数可按旋转群的不可约表示展开：

$$f(\eta) = \sum_{k \in \mathbb{Z}} \hat{f}_k \,e^{ik\eta}, \qquad \eta \in [0, 2\pi)$$

k = 0 模对应均匀分布（常数函数）。导数 ∂/∂η 在 Fourier 基下的作用为：

$$\widehat{(\partial_\eta f)}_k = ik \cdot \hat{f}_k$$

即 ∂/∂η 将 k 模映射到自身（乘以 ik）。

**引理 3.1**（Dirac 算子的 C-场微分结构）。21 号构造的 Dirac 算子 D 作用在旋量丛 𝒮 上。在 C-场坐标 η 的切方向上，D 包含 Clifford 乘法项 e_η·∂_η，其中 e_η 是 Cl(9) 的生成元。

**证明**：21 号定义 2.3 和引理 3.2 给出 D 在约束截面 Σ 上的局部形式。D^2 包含 −Δ_Σ（Laplace–Beltrami 算子），说明 D 包含对应的一阶微分项。在 η 方向，这就是 ∂_η 的 Clifford 乘法版本。证毕（这是 21 号定理级结果的直接复述）。

**引理 3.2**（密度对应下的微分结构保持，命题级骨架）。在 Dirac 旋量动力学到密度动力学的对应下（无论是 Wigner-Weyl、Koopman–von Neumann、还是 Coherent 态路径积分），一阶微分算子 ∂/∂η 被翻译为相空间密度上的 Liouville 型算子——在 η 坐标上仍包含 ∂/∂η 的某种实现（前推的生成元、Moyal 星括号的反对称部分、或 Koopman 生成元）。

> **诚实标注**：引理 3.2 是**命题级**——它是 0.5.1 定理 7.1–7.2 标注的"Wigner-Weyl 对应"的核心内容。本文不证明该对应，而是接受其作为物理上合理的骨架。若该对应在未来被推翻，本章的选择定则需要重新评估。

**推论 3.3**（ΔL_int 在 C-场 Fourier 基下的结构）。在引理 3.2 的骨架下，ΔL_int 作为 𝒳_C ⊗ 𝒳_I 上的算子，其 C-场部分包含 ∂_η 的贡献。该贡献在 Fourier 基下的矩阵元满足：⟨k| ΔL_int |l⟩ ≠ 0 ⇒ |k − l| ≤ 1（相邻模耦合）。即 ΔL_int 在 C-场 Fourier 空间中是**三对角**的。

**理由**：∂_η 对应对角算子（乘以 ik）。Clifford 乘法 e_η· 在适当的旋量基下将 k 模映射到 k±1 模（因 e_η 在旋量空间中的矩阵表示将上下旋量分量混合，相当于半个步长的 Fourier 平移）。两者组合产生 |k−l| ≤ 1 的带状结构。

> **诚实标注**：|k−l| ≤ 1 的精确界依赖 Clifford 乘法在旋量基下的具体表示。若 e_η 在更复杂的基下有更宽的带宽，选择定则相应放宽。但关键是：带宽是有限的（O(1)）——因为 D 是一阶微分算子，其 C-场部分不包含高阶导数。有限带宽是定理级的（来自 D 的一阶性质）；精确带宽为 1 是命题级的。

### 3.2　定理 2：δL_I^(0) 的选择定则

**定理 2**（δL_I^(0) 的 C-场 Fourier 选择定则，命题级）。在推论 3.3 的结构下，δL_I^(0) = ∫_Σ ΔL_int(ρ_C^* ⊗ ·) dμ_C 满足：在 C-场 Fourier 基下，δL_I^(0) 将 k = 0 模（均匀分布 ρ_C^*）映射到 |k| = 1 模的张成空间。

**证明骨架**：

ρ_C^* 是 k = 0 模（常数函数）。ΔL_int 在 C-场 Fourier 空间中是三对角的（推论 3.3），其连接 k = 0 到 k = l 的矩阵元非零仅当 |l| ≤ 1。对 C-场自由度积分（取迹）对应于提取 k = 0 分量（因 ∫_Σ f(η) dμ_C ∝ f̂₀）。δL_I^(0) = Tr_C[ΔL_int(ρ_C^* ⊗ ·)] 涉及 ΔL_int 作用于 k = 0 模，然后对 C-场取 k = 0 的迹。

ΔL_int 作用于 k = 0（均匀分布 = ρ_C^*）产生 |k| ≤ 1 的分量。然后对 C-场取迹提取 k = 0 部分。这相当于计算 ΔL_int 在 k = 0 模上的"对角"矩阵元（C-场迹 = 零模投影）。

对于仅含一阶导数的算子（∂_η），其对 k = 0 模的作用 ∂_η(const) = 0——因为导数作用于常数得零。在密度动力学对应下，包含 ∂_η 的 Liouville 型算子作用于均匀密度产生零（常数密度的前推仍是常数）。因此 ΔL_int 中与 ∂_η 成正比的部分对 ρ_C^* 的贡献为零。

ΔL_int 中来自 Clifford 乘法（e_η·）而非导数的部分，将 k = 0 映射到 k = ±1。对 C-场取迹后（k = 0 投影），该贡献为零——因为 k = ±1 与 k = 0 正交。

因此 δL_I^(0) = 0（在一阶微分结构下）。

> **诚实标注**：上述论证分为两步：(a) 导数部分 → 零（因 ∂_η(const) = 0）——这在任何合理的密度对应下成立，接近定理级；(b) Clifford 部分 → 迹后为零（因 k = ±1 ⟂ k = 0）——这依赖 Dirac→Liouville 对应中迹运算保持 Fourier 正交性，命题级。完整论证状态为**命题级**，因步骤 (b) 和引理 3.2 的依赖性。

**定理 2 的状态：命题级。** 但物理机制清晰且稳健：Dirac 算子的一阶微分结构使得均匀分布下的条件期望受抑制——导数项消零，Clifford 项因 Fourier 正交性在经过 C-场部分迹后消零。这是 δL_I^(0) = 0 在几何论框架内的自然机制。

### 3.3　选择定则的物理解释

C-场的快速旋转（T_ss）像一个"动态解相关器"：ΔL_int 试图在 C 和 I 之间建立关联，但 C-场的遍历环流（均匀旋转）意味着在任何一个完整旋转周期上，耦合的正向和负向贡献精确对消——这正是 δL_I^(0) = 0 的物理图像。

选择定则（定理 2）是这一图像的 Fourier 空间表述。均匀分布是旋转不变的（k = 0），任何"旋量型"耦合（非零旋量荷，k ≠ 0）在均匀分布下的平均值自动为零。

## 第4章　最坏情况分析

### 4.1　如果定理 2 不成立：δL_I^(0) ≠ 0 的后果

即使定理 2 的 Dirac 结构论证不成立（或选择定则比预期更宽），定理 1（Fredholm 零对角）仍然成立。本节分析在此保守情景下 δL_I^(0) 的可能影响。

设 δL_I^(0) 在 I-场 Laplacian 本征基下的矩阵元为：

$$\langle \phi_k, \delta L_I^{(0)} \phi_l \rangle = d_{kl}$$

{φ_k} 是 L_I^(0) = L_G,diff 的本征模，按本征值 λ_k 递增排列（λ₁ = 1 对应均匀分布 φ₁ = 常数）。

定理 1 等价于：d_{1l} = 0 对所有 l（或更精确地，∑_j (δL_I^(0))_{ji} = 0 对所有 i）。以本征模展开，Tr_I[δL_I^(0)·] = 0 ⇔ (δL_I^(0))* φ₁ = 0 ⇔ ⟨φ₁, δL_I^(0) φ_l⟩ = 0 对所有 l ——即 d_{1l} = 0（δL_I^(0) 不激发均匀模）。

### 4.2　I-场稳态摄动方程

联合稳态的 I-场部分 ρ_I^* 满足（0.5.4 定理 3, 4）：

$$L_I^{(0)} \rho_I^* + \delta L_I^{(0)} \rho_I^* + \delta L_I^{(1)}[\rho_I^*] \rho_I^* = \rho_I^*$$

展开 ρ_I^* = φ₁ + δρ_I（φ₁ = 均匀分布，归一化到总概率 1）：

$$(I - L_I^{(0)}) \delta\rho_I = \delta L_I^{(0)} \phi_1 + \delta L_I^{(0)} \delta\rho_I + \delta L_I^{(1)}[\rho_I^*] \rho_I^*$$

一阶近似（假设 δρ_I 小，待自洽验证）：

$$(I - L_I^{(0)}) \delta\rho_I = \delta L_I^{(0)} \phi_1 + O(\delta_I^2) \tag{4.1}$$

**Fredholm 可解性**：δL_I^(0) φ₁ 与 φ₁ 正交（因 d_{11} = 0，定理 1 的特例）。方程有解。

解的显式形式（在 L_I^(0) 的本征基下）：

$$\delta\rho_I = \sum_{k=2}^{N_{\text{info}}} \frac{d_{k1}}{1 - \lambda_k} \,\phi_k + O\left(\frac{\delta_I^2}{\gamma_I}\right)$$

其中 d_{k1} = ⟨φ_k, δL_I^(0) φ₁⟩。

### 4.3　定理 3：最坏情况界

**定理 3**（δL_I^(0) 非零情景下的最坏情况界，定理级）。在仅假设定理 1（Fredholm 零对角）的条件下：

$$\boxed{\|\rho_I^* - \phi_1\|_{L^1} \leq \frac{\|\delta L_I^{(0)}\|_{\text{op}}}{\gamma_I^{\text{eff}}} + O(N_{\text{cause}} \delta_I^2 / \gamma_I)}$$

其中 γ_I^eff = min_{k≥2, d_{k1}≠0} (1−λ_k) 是 δL_I^(0) 实际耦合到的最慢非零模的谱隙。若 δL_I^(0) 不耦合到慢模（即仅 d_{k1} ≠ 0 对大 k，其中 1−λ_k ∼ O(1)），则有效谱隙为 O(1)，稳态偏离极小（～δ_I）。

**证明**：

由 (4.1) 的显式解，L¹ 范数估计：

$$\|\delta\rho_I\|_{L^1} \leq \sum_{k=2}^{N_{\text{info}}} \frac{|d_{k1}|}{1 - \lambda_k} \|\phi_k\|_{L^1} + \cdots$$

使用 Cauchy–Schwarz（在序列空间 l² 中，因 φ_k 是 L² 正交的）：

$$\|\delta\rho_I\|_{L^1} \leq \left(\sum_{k=2}^{N_{\text{info}}} \frac{1}{(1-\lambda_k)^2}\right)^{1/2} \cdot \left(\sum_{k=2}^{N_{\text{info}}} |d_{k1}|^2\right)^{1/2}$$

第二因子 ≤ ‖δL_I^(0) φ₁‖_{L²} ≤ ‖δL_I^(0)‖_op（算子范数）。

第一因子：若所有 d_{k1} ≠ 0 对所有 k，最坏情况由最低非零模 k = 2 主导（1−λ₂ = γ_I ∼ 1/N_info），给出因子 ∼ N_info。此时 ‖δρ_I‖ ≤ N_info·‖δL_I^(0)‖_op。

但若 δL_I^(0) 仅耦合到大 k 模（快速衰减模），有效最小谱隙 γ_I^eff = min_{k: d_{k1}≠0}(1−λ_k) ≫ 1/N_info，界相应收紧。

一般地，令 K_min = min{k ≥ 2 : d_{k1} ≠ 0}，则 γ_I^eff = 1−λ_{K_min}。若耦合始于第 K 个模，则 ‖δρ_I‖ ≤ ‖δL_I^(0)‖_op / (1−λ_K)。证毕。

**定理 3 的状态：定理级。** 证明仅使用线性代数（本征展开）和范数估计，不依赖 δL_I^(0) 的微观形式。

### 4.4　定理 3 的含义

定理 3 说明：0.5.4 绝热框架的稳健性取决于 δL_I^(0) 是否耦合到 I-场的**慢模**（低阶 Laplacian 本征模）。

- **若 δL_I^(0) 仅耦合到快模**（k 大，1−λ_k ∼ O(1)）：稳态偏离 ∼ δ_I ≈ 2.6×10⁻³，0.5.4 结论成立。
- **若 δL_I^(0) 显著耦合到最低慢模**（k = 2, 1−λ₂ ∼ 1/N_info）：稳态偏离 ∼ δ_I·N_info ≈ 2.3×10²⁰，0.5.4 的定量估计需要修正。

定理 2（Dirac 选择定则）正是排除了第二种情景——它论证了 δL_I^(0) 在 C-场 Fourier 空间中是高度结构化的，不随机耦合到 I-场的低阶模。

## 第5章　两条路径的合成与物理界

### 5.1　命题 1：物理界的合成

**命题 1**（δL_I^(0) 的物理有效界，命题级）。在定理 2（Dirac 选择定则）的骨架下，δL_I^(0) 对 I-场慢模（本征值 λ_k ≥ λ_{freeze}，其中 λ_{freeze} 为冻结阈值）的矩阵元满足：

$$\boxed{|d_{k1}| := |\langle \phi_k, \delta L_I^{(0)} \phi_1 \rangle| \leq \delta_I \cdot \frac{1}{\sqrt{N_{\text{cause}}}} \cdot C_{\text{micro}}, \quad \forall k \leq K_{\text{freeze}}}$$

从而 δL_I^(0) 的有效强度（在慢时间尺度的平均下）为：

$$\|\delta L_I^{(0)}\|_{\text{eff}} \leq N_{\text{cause}} \cdot \delta_I^2 \cdot C_{\text{eff}}$$

其中 C_micro、C_eff 为 O(1) 常数。

**论证骨架**：

**步骤 1（慢模的 C-场 Fourier 结构）**。I-场的低阶 Laplacian 模 φ_k（k ≤ K_freeze）在大尺度上平滑——它们在 C-场耦合的微观尺度（∼1/N_cause 的 η 分辨率）上近似常数。这意味着 φ_k 对应的联合密度 ρ_C^* ⊗ φ_k 在 C-场 Fourier 空间中主要集中在 k = 0。

**步骤 2（选择定则的后果）**。ΔL_int 在 C-场 Fourier 空间中仅连接 |Δk| ≤ 1 的模（推论 3.3）。δL_I^(0) 的矩阵元 d_{k1} 涉及：ΔL_int 作用于 (C-场 k=0, I-场 φ₁) → 产生 (C-场 k=±1, I-场 φ_k)。然后对 C-场取迹（提取 k=0）→ 得零（k=±1 ⟂ k=0）。

但对于有限 C-场分辨率（N_cause ≈ 10），Fourier 模的正交性在离散求和（而非连续积分）下被违反 O(1/N_cause) 的量级。这产生了命题中的 1/√N_cause 因子。

**步骤 3（有效二阶性）**。由此，δL_I^(0) 对慢模的有效矩阵元 ∼ δ_I/√N_cause ≈ 8.2×10⁻⁴（而非 δ_I ≈ 2.6×10⁻³）。但其对 I-场慢动力学的累积效应进一步受快慢分离压制（定理 3 的机制），最终有效强度 ∼ N_cause·δ_I² ≈ 6.8×10⁻⁵。

> **诚实标注**：命题 1 的 1/√N_cause 因子来自离散 Fourier 求和与连续积分的差——这是有限 C-场分辨率（N_cause ≈ 10）的效应。该估计为数量级估计（命题级），精确的前因子需离散 Fourier 分析的更细致处理。

### 5.2　δL_I^(0) 的可能形式与物理图像

综合定理 1–3 和命题 1，δL_I^(0) 在物理上最可能是以下形式之一：

| 可能性 | 裸范数 | 慢模耦合 | 有效强度 | 物理机制 |
|:---|:---|:---|:---|:---|
| (a) 精确零 | ≡ 0 | 无 | 0 | Dirac 一阶微分 + 连续 Fourier 正交 |

情景 (a)–(c) 均支持 0.5.4 结论（ρ_I^* ≈ 均匀分布，I(ρ_CI^*) 极小）。情景 (d) 需要 Dirac 算子的 C-场微分结构在密度对应下完全丢失——这极其 unlikely，因为一阶微分算子的性质在任何合理的相空间对应下都保持。
