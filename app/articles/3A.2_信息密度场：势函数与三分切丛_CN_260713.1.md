# 第2章 信息密度场：势函数与三分切丛

**摘要：** 本章在约束截面 $\Sigma$ 上构造信息密度场 $\Phi$，通过势函数 $V(\xi,\eta)$ 的六项展开刻画三扇区耦合结构。信息场方程由谱三元组的 Dirac 算子介导，在局域谐振子近似下可解。

---

## 2.1 势函数的六项展开

**定义 2.1（势函数 $V$，GT-3A.2.1.1）。** 约束截面 $\Sigma$ 上的势函数 $V: \Sigma \to \mathbb{R}_{\ge 0}$ 定义为信息场自相互作用势，在 $(\xi,\eta)$ 坐标中展开为六项：

$$V(\xi,\eta) = V_S(\xi) + V_B(\eta) + V_{\text{coup}}(\xi,\eta) + V_{\text{int}} + V_{\text{stab}} + V_{\text{bound}}$$

**（1）$\mathcal{I}$-扇区软模势 $V_S$（沿 $\xi$ 方向）：**

$$V_S(\xi) = \frac{1}{2}\lambda_2^{\text{eff}}\xi^2 + \frac{1}{4}\beta_\xi\xi^4$$

其中 $\lambda_2^{\text{eff}} = 59324.3\ \text{rad}^{-2}$ 为有效硬模，$\beta_\xi$ 为四次项系数（由 $S_e$ 自洽锁定）。

**（2）$\mathcal{M}\oplus\mathcal{C}$ 扇区硬模势 $V_B$（沿 $\eta$ 方向）：**

$$V_B(\eta) = \frac{1}{2}\lambda_1^{\text{eff}}\eta^2 + \frac{1}{4}\beta_\eta\eta^4$$

其中 $\lambda_1^{\text{eff}} = 391.05\ \text{rad}^{-2}$ 为有效软模，$\beta_\eta$ 为四次项系数。

**（3）跨扇区耦合势 $V_{\text{coup}}$：**

$$V_{\text{coup}}(\xi,\eta) = \gamma\,\xi^2\eta^2$$

其中 $\gamma$ 为耦合常数，由九素互扼约束锁定量级 $\gamma \approx \sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}$。

**（4）信息自相互作用 $V_{\text{int}}$：**

$$V_{\text{int}} = \alpha\, S_e^2 \cdot \Phi^4$$

其中 $\alpha$ 为无量纲耦合强度，$\Phi$ 为信息密度场。

**（5）稳定性势 $V_{\text{stab}}$：**

$$V_{\text{stab}} = \kappa\left(\theta_M + \theta_C + \theta_I - \frac{\pi}{2}\right)^2$$

确保扇区角完备性始终成立。

**（6）边界势 $V_{\text{bound}}$（紧致性保证）：**

$$V_{\text{bound}} = -\mu\exp\left(-\frac{r^2}{\sigma^2}\right), \quad r^2 = \xi^2 + \eta^2$$

其中 $\sigma$ 为截断宽度，由 $\Sigma$ 的特征尺度决定。

---

## 2.2 三扇区耦合结构

**定理 2.2（三扇区耦合结构，GT-3A.2.2.1）。** 势函数 $V$ 的最小值条件等价于以下三扇区耦合方程组：

$$\begin{cases}
\displaystyle\frac{\partial V}{\partial\xi} = \lambda_2^{\text{eff}}\xi + \beta_\xi\xi^3 + 2\gamma\xi\eta^2 - \frac{2\mu}{\sigma^2}\xi\exp\left(-\frac{r^2}{\sigma^2}\right) = 0 \\[10pt]
\displaystyle\frac{\partial V}{\partial\eta} = \lambda_1^{\text{eff}}\eta + \beta_\eta\eta^3 + 2\gamma\xi^2\eta - \frac{2\mu}{\sigma^2}\eta\exp\left(-\frac{r^2}{\sigma^2}\right) = 0
\end{cases}$$

**极小点位置。** 在 $(\xi,\eta)$ 平面内，存在三类临界点：

1. **原点 $O=(0,0)$：** 对应参考点 $P_e$，为 **鞍点**（$\eta$ 方向软模不稳定）
2. **$\xi$-轴上的两个极小点：** $Q_{1,2}=(\pm\xi_0,0)$，其中 $\xi_0$ 满足 $\lambda_2^{\text{eff}}\xi_0 + \beta_\xi\xi_0^3 \approx 0$，数值解 $\xi_0 = \pi/2 - \theta_I^0 \approx 1.4678\ \text{rad}$
3. **$\eta$-轴上的不稳定点：** 由于 $\lambda_1^{\text{eff}} > 0$，$\eta=0$ 为 $\xi$-轴上极小值方向

**定理 2.2.2（全局最小值性质，GT-3A.2.2.2）。** $V$ 在 $\Sigma$ 上的全局最小值位于 $Q_1$ 和 $Q_2$，且 $V(Q_1)=V(Q_2)=V_{\min}$。势垒高度为：

$$\Delta V = V(O) - V_{\min} = \frac{(\lambda_2^{\text{eff}})^2}{4\beta_\xi} + \mathcal{O}(\exp(-R_\Sigma^2/\sigma^2))$$

由 $\lambda_2^{\text{eff}}/\beta_\xi$ 的谱刚性锁定，$\Delta V \approx 4.27 \times 10^5\ \text{rad}^{-4}$。

## 2.3 信息场定义与方程

**定义 2.3（信息密度场 $\Phi$，GT-3A.2.3.1）。** 信息密度场 $\Phi: \Sigma \to \mathbb{C}$ 是约束截面上的复值函数，满足：

1. **归一化条件：** $\int_\Sigma |\Phi|^2 d\mu_g = 1$
2. **$S_e$ 耦合：** $\Phi$ 的零模由 $S_e^{-1}$ 控制
3. **谱分解：** $\Phi = \sum_{n=0}^\infty a_n \psi_n$，其中 $\{\psi_n\}$ 为 $\Delta_\Sigma$ 的特征函数

**定理 2.3（信息场方程，GT-3A.2.3.2）。** 信息密度场 $\Phi$ 满足非线性薛定谔型方程：

$$\left[-\frac{1}{2}\Delta_\Sigma + V(\xi,\eta) + \alpha S_e^2|\Phi|^2\right]\Phi = E\Phi$$

其中 $\Delta_\Sigma$ 为 §1.3 定义的 Laplace-Beltrami 算子，$E$ 为本征能量。

**证明：** 由 Vol-1 谱三元组构造，Dirac 算子 $D$ 的平方 $D^2 = \Delta_\Sigma + \frac{1}{4}R_\Sigma$（Lichnerowicz 公式）。$V$ 项从九素互扼约束的扇区耦合导出，非线性项 $\alpha S_e^2|\Phi|^2$ 从信息场自作用 $V_{\text{int}}$ 的变分得到。$\square$

**推论 2.3.1（基态方程）。** 在基态近似 $\Phi_0 = \sqrt{\rho_0}e^{i\chi}$ 下，信息场方程分解为连续方程和 Euler 方程：

$$\nabla\cdot(\rho_0\nabla\chi) = 0 \quad\text{(连续性)}$$

$$-\frac{1}{2}\Delta_\Sigma\sqrt{\rho_0} + V\sqrt{\rho_0} + \alpha S_e^2\rho_0^{3/2} = E\sqrt{\rho_0} \quad\text{(Euler)}$$

## 2.4 局域谐振子近似

在 $Q_1$ 附近展开 $V$，忽略四阶以上项，得到局域谐振子近似：

**定理 2.4（局域谐振子近似，GT-3A.2.4.1）。** 在 $Q_1=(\xi_0,0)$ 附近，信息场方程近似为：

$$\left[-\frac{1}{2}\Delta_\Sigma + \frac{1}{2}\omega_\xi^2 (\xi-\xi_0)^2 + \frac{1}{2}\omega_\eta^2 \eta^2\right]\Phi = E_{\text{loc}}\Phi$$

其中局域频率为：

$$\omega_\xi^2 = \left.\frac{\partial^2 V}{\partial\xi^2}\right|_{Q_1}, \quad \omega_\eta^2 = \left.\frac{\partial^2 V}{\partial\eta^2}\right|_{Q_1}$$

数值值待第3章计算。

**特征模式。** 二维谐振子解为：

$$\Phi_{n,m}(\xi,\eta) = H_n(\sqrt{\omega_\xi}(\xi-\xi_0))\, H_m(\sqrt{\omega_\eta}\eta) \exp\left(-\frac{\omega_\xi(\xi-\xi_0)^2 + \omega_\eta\eta^2}{2}\right)$$

本征值为 $E_{n,m} = (n+1/2)\omega_\xi + (m+1/2)\omega_\eta$。

**定义 2.4.2（像素定义，GT-3A.2.4.2）。** $\Sigma$ 上的信息像素由 $\mathcal{I}$-扇区谐振子基态波函数的局域化区域定义：

$$\Delta\xi_{\min} = \frac{1}{\sqrt{\omega_\xi}} = \lambda_2^{-1/4}\sqrt{2}$$

最小可分辨角 $\Delta\xi_{\min} \approx 0.160\ \text{rad}$。
