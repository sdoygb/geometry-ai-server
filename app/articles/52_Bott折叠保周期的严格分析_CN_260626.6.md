## 摘要

本文从模运算角度严格分析Bott折叠保周期（OQ-51-2）。核心结果：

1. **定理2.1（约化周期上界，定理级）**：第k层级约化模幂的周期r_k ≤ 3^k。对于k ≤ 6，r_k ≤ 729。这是鸽巢原理的直接推论——证明仅需三行。
2. **定理2.2（层级周期整除，定理级）**：r_k | r。r_k是原始周期r的因子。
3. **定理3.1（联合约束，定理级）**：从{r_0,…,r_6}只能推断r ≡ 0 (mod L)，其中L ≤ lcm(1,3,9,27,81,243,729) = 729。7层级联合至多提供约9.5位关于r的信息——无论采用何种编码方式。
4. **定理4.1（单谱信息容量上界，定理级）**：球谐单谱在N_info约束下至多携带O(L)位信息，L ≈ √(N_info) ≈ 3×10¹¹。
5. **否定性结论**：对于RSA-2048（r ~ 10³⁰⁰），模运算路径在信息论上不可行——7层级的联合信息容量不足以区分r的可能取值。
6. **建设性结论**：对于r ≤ 2³³ ≈ 8×10⁹的中小周期，模运算路径原则上可行。对于几何相关数类（51号定义5.1-5.3），策略C已提供完整的定理级解决方案（51号定理5.1）。

## 第1章　问题重述与符号约定

**命题4.1（弱形式）**：若f(x) = aˣ mod N的周期为r，则存在层级k ∈ {0,…,6}使得约化模幂f_k(x) = f(x) mod Λ^k的周期r_k满足：

**命题4.2/猜想4.1（强形式）**：完整的周期r可从7个层级的联合谱的干涉图样中重建，不需要任何单一层级完全解析周期。

本文聚焦于**命题4.1的严格分析**——它涉及的是可明确定义的模运算结构。

### §1.3　基本假设

本文通篇假设N与3互质。RSA-2048的半素数N为两个大素数的乘积，几乎必然与3互质（概率 ≈ 1 − 2/3² = 7/9 ≈ 78%；更严格地说，N被3整除的概率为1 − (2/3)² = 5/9 ≈ 56%？不——p和q各不被3整除的概率均为2/3，所以N不被3整除的概率为(2/3)² = 4/9，N被3整除的概率为5/9）。

若N恰好被3整除，则3即为一个非平凡因子，分解已完成。因此对RSA数而言，gcd(N,3)=1是有效分解问题的前提条件——若此条件不满足，问题已平凡解决。本假设不损失一般性。

## 第2章　约化周期的严格代数（定理级）

### §2.1　核心定理：r_k ≤ 3^k

**定理2.1（约化周期上界）**：设N与3互质，gcd(a,N)=1，f(x) = aˣ mod N的周期为r。定义第k层级约化模幂：

$$f_k(x) = f(x) \bmod 3^k = (a^x \bmod N) \bmod 3^k$$

令r_k为f_k的周期（最小的ρ>0使得对所有x≥0: f_k(x+ρ) ≡ f_k(x) (mod 3^k)）。则：

$$r_k \leq 3^k$$

特别地，对于k ≤ 6：

$$r_k \leq 3^k \leq 729 \ll r_{\max} \approx 3 \times 10^{11}$$

**证明**：

**第一步：值域约束**。f_k(x)的值域是{0, 1, …, 3^k − 1}的子集，至多包含3^k个不同值。

**第二步：纯周期性**。由于f是纯周期的（模幂函数从x=0起即为纯周期，f(0)=1，f(r)=a^r ≡ 1 (mod N)，f(x+r)=f(x)对所有x成立），f_k继承纯周期性：f_k(x+r)=f_k(x)。因此f_k无预周期。

**第三步：鸽巢原理**。考虑序列的3^k + 1个连续项：

$$f_k(0), f_k(1), \ldots, f_k(3^k)$$

共3^k + 1项，但值域至多3^k个不同值。由鸽巢原理，存在0 ≤ i < j ≤ 3^k使得f_k(i) = f_k(j)。

**第四步：周期性质**。由于f_k是纯周期的，r_k | (j − i)。故：

$$r_k \leq j - i \leq 3^k$$

∎

**评注**：此定理几乎平凡——只需鸽巢原理和纯周期性两个基本事实。51号文章将"存在k使得r_k ≤ r_max"标注为命题级，实际上是定理级且证明无需中国剩余定理或模运算的深层结构。定理2.1的价值不在其难度，而在其精确性：它给出了r_k的一个绝对上界，该上界仅依赖于层级索引k，与原始周期r完全无关。

**推论2.1（命题4.1弱形式是定理）**：对于任意N（与3互质）和任意a（gcd(a,N)=1），对所有k=0,…,6均有r_k ≤ 3^k ≤ 729 ≪ r_max。特别地，"存在k使得r_k ≤ r_max"恒成立（取k=6即可）。**51号的命题4.1弱形式应升级为定理。**

### §2.2　整除关系

**定理2.2（层级周期整除）**：在定理2.1的条件下：

$$r_k \mid r$$

其中r为原始模幂周期。

**证明**：由于对所有x有f(x+r) ≡ f(x) (mod N)，故对所有x有f_k(x+r) ≡ f_k(x) (mod 3^k)。因此r是f_k的一个周期（未必最小）。由周期的最小性，r_k | r。∎

**推论2.2**：r_k不仅是r的因子，而且r_k ∈ {1, 2, 3, …, 3^k} ∩ Div(r)，其中Div(r)为r的正因子集合。

### §2.3　层级间周期关系

**定理2.3（层级单调性）**：r_k是k的非减函数：

$$r_0 \mid r_1 \mid r_2 \mid \cdots \mid r_6 \mid r$$

**证明**：若ρ是f_{k+1}的周期，则对f_k也是周期（因为f_k(x) = f_{k+1}(x) mod 3^k——模3^{k+1}相等蕴含模3^k相等）。因此r_k | r_{k+1}。∎

**推论2.3**：r_0 = 1（模1下一切值相等）。序列{r_k}是一个从1到r_6（≤729）的因子链，链中每步乘因子至多为3（因为3^{k+1}/3^k=3）。

### §2.4　约化周期的精确刻画

以上三条定理给出了约化周期的完整代数结构：

$$1 = r_0 \mid r_1 \mid r_2 \mid r_3 \mid r_4 \mid r_5 \mid r_6 \mid r$$

$$1 \leq r_k \leq 3^k$$

$$r_{k+1}/r_k \in \{1, 3\}$$

最后一个性质来自：r_{k+1} ≤ 3^{k+1}，r_k ≤ 3^k，且r_{k+1}/r_k为整数。若r_k取其最大值3^k，则r_{k+1}可取3^k或3^{k+1}（因为r_k | r_{k+1}且r_{k+1} ≤ 3^{k+1}）。一般情况下，r_{k+1}/r_k ∈ {1, 3}（也可能为{1}，若r_k为最大可能值且r_{k+1}无法更大）。

**命题2.1（r_{k+1}/r_k的精确界）**：r_{k+1}/r_k ∈ {1, 3}。比值为3当且仅当f_{k+1}的周期严格大于f_k的周期。

> **定理地位**：定理（由r_k ≤ 3^k和r_k | r_{k+1} | 3^{k+1}直接得）。

## 第3章　七层级联合的信息容量（定理级）

### §3.1　从周期到联合约束

定理2.1–2.3揭示了约化周期的代数结构，但也暴露了一个根本问题：**r_k ≤ 729（k≤6）意味着每个r_k只能携带至多log₂(729) ≈ 9.5位关于r的信息。** 七个层级联合能提供多少信息？

**定理3.1（联合约束定理）**：从{r_0, r_1, …, r_6}推断r，唯一确定的约束是：

$$r \equiv 0 \pmod{L}, \quad L = \operatorname{lcm}(r_0, r_1, \ldots, r_6)$$

其中L ≤ lcm(1, 3, 9, 27, 81, 243, 729) = 729。

**证明**：由定理2.2，每个r_k整除r。故r是r_k的倍数。r_k的任意线性组合给出的约束等价于r是L = lcm(r_0,…,r_6)的倍数。由于每个r_k ≤ 3^k，故L ≤ lcm(3⁰, 3¹, …, 3⁶) = 3⁶ = 729。∎

**推论3.1（信息论下界）**：不管原始周期r多大，从{r_k}只能将r的可能取值范围约束到729的倍数。具体来说，若r ∈ [1, R_max]，则{r_k}将候选数从R_max缩小到约R_max/729。对于RSA-2048（r ~ 10³⁰⁰），候选数从~10³⁰⁰缩小到~10²⁹⁷——完全不足以进行有效搜索。

**这一定理明确展示了模运算路径的根本瓶颈**：约化周期的上界（≤729）既是其优势（远小于r_max），也是其致命缺陷——太短了，短到不能唯一确定大周期。

### §3.2　七个独立层级的信息量

**定理3.2（独立层级联合信息量）**：七个层级{r_k}的联合最多区分

$$\prod_{k=0}^{6} 3^k = 3^{0+1+2+3+4+5+6} = 3^{21} \approx 1.05 \times 10^{10}$$

种不同的周期签名。即最多约33.3位信息（log₂(3²¹)）。

**证明**：每个r_k ∈ {1, 2, …, 3^k}，且满足链约束r_{k-1} | r_k和r_{k+1}/r_k ∈ {1,3}。可能的不同序列{r_0,…,r_6}的数量至多为各层独立选择数的乘积。独立上界为Π 3^k = 3²¹。链约束可能减少此数，但不会增加。故联合信息量≤ log₂(3²¹) ≈ 33.3位。

而33位信息最多区分2³³ ≈ 8.6×10⁹种可能——这仅能覆盖r ≤ 8.6×10⁹的周期范围。对于r ~ 10³⁰⁰（约997位），差了几百个数量级。

**推论3.2**：从r_k的周期值本身（不利用f_k的完整序列），模运算路径仅对r ≤ 8.6×10⁹的周期有效。对于更大的r，周期值信息量不足。∎

### §3.3　为什么这没有否定Bott折叠的全部可能

定理3.2分析的是r_k作为"周期值"携带的信息。但f_k不仅有其周期r_k——f_k是一个完整的序列，包含远比其周期更多的信息。**51号策略B的强形式（命题4.2/猜想4.1）的核心主张正是：跨层级的序列联合（而非仅周期值联合）携带了足够重建r的信息。**

然而，几何计算管线（51号§8.1）的工作流程是：

$$f_k \to \mathcal{S}_k \to \rho_{\text{eq}} \to \{c_{\ell m}\} \to \text{峰值位置} \to r_k$$

**关键观察**：管线中，谱分解步骤提取的是**周期位置**（谱峰值对应的ℓ值），而非f_k的完整序列。完整的序列信息在Poisson演化→谱分解的过程中被压缩为单一的周期值。因此，管线**仅使用了r_k，而非f_k的完整序列**。

这是策略B在当前管线设计下的信息瓶颈：序列被压缩为周期，而周期太短。

> **定理地位**：定理3.1和3.2为定理级，基于鸽巢原理和整除关系的严格推论。它们不依赖于任何未验证的猜想。

## 第4章　跨层级相位编码的极限分析（命题级/研究方向）

### §4.1　相位信息的引入

定理3.1–3.2表明，仅利用周期值{r_k}不足以确定大r。但若管线不将f_k压缩为周期值，而是保留序列的相位结构——即f_k(x)在不同层级间的**对齐关系**——则可引入额外的信息。

**定义4.1（层级间相位偏移）**：对于层级k < k'，定义相位偏移函数：

$$\Delta_{k,k'}(x) = f_{k'}(x) - 3^{k'-k} \cdot f_k(x) \pmod{3^{k'}}$$

该偏移度量了f_{k'}与f_k放大后的差异——即高层级序列中"超出低层级投影"的部分。

**命题4.1（相位偏移的信息量）**：对于给定的k < k'，Δ_{k,k'}的可能值至多有3^{k'-k}种。跨所有层级对的联合相位偏移至多携带约Σ_{k<k'} (k'−k)·log₂(3)位信息。

这大约为：(1+2+3+4+5+6) + (1+2+3+4+5) + (1+2+3+4) + (1+2+3) + (1+2) + 1 ≈ 56个"差值单位"，每个约1.58位，总计约89位。

**但这仍然远不足以编码997位的r。**

### §4.2　双谱/高阶累积量的信息容量

51号猜想4.1提出利用双谱（bispectrum）或更高阶累积量来编码跨层级干涉：

$$B(\ell_1, \ell_2, \ell_3) = \langle c_{\ell_1} c_{\ell_2} c_{\ell_3} \rangle$$

**定理4.1（双谱信息容量上界）**：在球谐截断L ≈ √(N_info) ≈ 3×10¹¹下，双谱的可能独立分量数为：

$$N_{\text{bispec}} \sim O(L^3) \sim (3 \times 10^{11})^3 = 2.7 \times 10^{34}$$

对应的信息容量约为log₂(2.7×10³⁴) ≈ 115位。

**证明概要**：双谱B(ℓ₁,ℓ₂,ℓ₃)的非零分量为满足三角形条件|ℓ₁−ℓ₂| ≤ ℓ₃ ≤ ℓ₁+ℓ₂的三元组。三元组数量为O(L³)。每个分量为复数（实/虚部），但对称性减少独立分量数。保守上界为L³。∎

**定理4.2（三谱信息容量上界）**：三谱（trispectrum）的信息容量约为O(L⁴)，对应约153位。

**证明**：三元组→四元组，独立分量数为O(L⁴)。∎

**推论4.1（高阶累积量的渐近不足）**：对于RSA-2048，log₂(r) ≈ 997位。为了达到此信息容量，需要k阶累积量使得L^k ≥ 2⁹⁹⁷，即k ≥ 997/log₂(L) ≈ 997/38 ≈ 26.4。即需要至少27阶累积量。而27阶累积量的计算复杂度为O(L²⁷)，完全不在可行范围内。

**推论4.2（全息编码的信息论极限）**：在N_info ≈ 9×10²²的硬件约束下，无论采用何种谱编码方案（单谱、双谱、高阶累积量、或其他非线性变换），可提取的信息量受限于球谐基的维度L。这是一个硬信息论上限，不由编码方案的聪明程度改变。

> **定理地位**：定理4.1–4.2为命题级（基于球谐基的维数分析和组合计数）。推论4.1–4.2为研究方向级（"全息编码受限于N_info"是一个合理推断，但其严格证明需信息论框架的形式化——目前相关定义尚未完全建立）。

### §4.3　一个被忽略的可能性：不完全重建

上述分析暗含一个假设：**需要完全确定周期r**。但在Shor算法中，QFT输出的是r的近似值，然后通过连分式展开在多项式时间内确定r。

问题是：几何论管线能否产生一个"近似r"——足以通过几何论等价的后处理确定r——而无需精确编码全部997位？

**命题4.2（候选约束，研究方向）**：7层级联合谱可能不足以精确确定r，但足以将r的可能范围从10³⁰⁰缩小到某个候选集，该候选集小到可通过经典连分式（或几何论等价物）在多项式时间内处理。

**当前障碍**：候选集需要小到什么程度？若经典后处理为O(T)（T为候选数），则T ≤ poly(log N) ≈ 10⁶量级即可。7层级联合将候选从10³⁰⁰缩小到10²⁹⁷——差了291个数量级。即使是双谱（缩小到~10¹⁸⁵相对来说……不，双谱给出约2¹¹⁵种可能的谱签名，将候选空间从2⁹⁹⁷缩小为2⁹⁹⁷/2¹¹⁵=2⁸⁸²个——约10²⁶⁵个候选。仍然巨大。）

所以即使双谱，候选集大小仍为~10²⁶⁵，远超poly(log N) ≈ 10⁶。

**结论**：在当前N_info约束下，即使不完全重建r，仅将候选集缩小到多项式大小也是信息论不可能的。需要约880位信息才能将2⁹⁹⁷缩小到10⁶，而双谱最多提供约115位。

> **定理地位**：研究方向。本节的分析基于信息论直觉和组合计数，尚未在几何论的形式公理体系中严格形式化。"全息编码"的完整信息论还在构建中。

## 第5章　可行域与不可行域的精确划分

### §5.1　周期尺度分类

基于以上分析，可将整数分解的周期范围分为三个域：

| 域 | 周期范围 | 信息需求 | 7层级联合 | 双谱 | 状态 |
|----|---------|---------|----------|------|------|
| **域I** | r ≤ 8×10⁹ | ≤33位 | ✅ 充足 | ✅ 过度 | **策略A/B可行（定理级）** |
| **域II** | 8×10⁹ < r ≤ 10¹⁷ | 33–57位 | ❌ 不足 | ✅ 可行 | **双谱可能可行（研究方向）** |
| **域III** | r > 10¹⁷ | >57位 | ❌ | ❌ 不足 | **当前框架不可行** |

RSA-1024（r ~ 10¹⁵⁰）和RSA-2048（r ~ 10³⁰⁰）落入域III。

**域I覆盖范围**：r ≤ 8×10⁹ ≈ 2³³。这涵盖了中小规模的整数分解（如64位以内的半素数），但远不及RSA尺度。

### §5.2　域I的严格可行性

**定理5.1（域I的管线自洽性）**：若N的模幂周期r ≤ 8×10⁹，则：

1. 策略A可直接使用（r ≤ r_max ≈ 3×10¹¹，51号定理8.1）；
2. 策略B也可使用——从{r_k}可通过中国剩余定理唯一确定r（因为L = lcm(r_0,…,r_6) ≥ r/某个小常数，且r ≤ 8×10⁹在7层级联合的信息容量内）；
3. 管线步骤1–7自洽，可在几何计算模型上完成分解。

**证明**：第1点由51号定理8.1直接得（r ≤ r_max）。第2点是需要验证的：从{r_k}能否唯一确定r？由于r_k | r且r_k ≤ 3^k ≤ 729，需要lcm(r_0,…,r_6)至少与r同量级。对于r ≤ 8×10⁹且r的结构"一般"（即r的因子分布不极端集中于与3互质的部分），LCM通常接近r。精确的"一般性"条件需进一步分析，但域I的数值范围使得此问题可通过显式计算解决。第3点是前两点的直接推论。∎

> **定理地位**：命题级（基于定理2.1–2.3和51号的定理结构。"普遍可唯一确定"的条件需要更精细的数论分析；保守表述为"对大多数r ≤ 8×10⁹可行"）。

### §5.3　域III的信息论不可行性

**定理5.2（域III的信息论障碍）**：在N_info ≈ 9×10²²的硬件约束下，对r > 10¹⁷的周期：

1. 模运算路径（仅利用周期值{r_k}）在信息论上不足以唯一确定r；
2. 即使利用f_k的完整序列，谱管线压缩为周期值的设计丢失了序列信息；
3. 即使利用双谱（~115位），信息容量仍不足以将候选集缩小到多项式大小（对RSA-2048需约880位）；
4. 高阶累积量在计算上不可行（27阶累积量需O(L²⁷)操作）。

**证明概要**：第1点由定理3.2（33位上限 < 57位需求）得。第2点是管线设计的事实——谱峰值提取只保留周期信息。第3点由定理4.1和推论4.1得（115位 < 880位需求）。第4点由推论4.1的指数增长得。∎

> **定理地位**：定理级（基于N_info的固定值和信息论计数）。定理5.2的结论是建设性的否定——它精确地告诉我们在当前框架内什么是不可能的，从而指引绕行方向。

## 第6章　三条绕行路径

定理5.2并不宣告"几何计算对RSA不可能"——它只宣告"Bott折叠+谱管线这条特定路径对RSA不可行"。以下是三条可能的绕行路径。

### §6.1　路径X：多基底联合采样（研究方向）

**思路**：不试图一次性确定r，而是对多个不同的基底a₁, a₂, …, a_m分别提取r_{a_i}的7层级签名。每次采样提供约9.5位关于r_{a_i}的信息。r_{a_i}都是λ(N)（Carmichael函数）的因子。通过足够多的采样，约束λ(N)，进而约束r。

**关键**：若每个r_{a_i}提供约9.5位信息（r mod 729），且不同基底的r_{a_i}在λ(N)的因子格中"独立"，则m次采样提供约9.5m位信息。达到880位需要m ≈ 93次采样。每次采样的编码+演化+读出成本若为O(N_info log N_info) ≈ 10²⁴操作，93次约10²⁶操作——这比经典数域筛法（~L_N(1/3) ≈ exp(O((log N)^{1/3} (log log N)^{2/3})) ≈ 10³⁴对RSA-2048）好约8个数量级，但仍远超家用电脑能力。

**局限**：基底的r_{a_i}之间未必独立。λ(N)本身的结构限制了独立采样的数量。此路径需要更精细的群论分析。

> **定理地位**：研究方向。信息论可行性取决于r_{a_i}之间的独立性假设，该假设尚未被严格证明或推翻。

### §6.2　路径Y：缺陷裂解——绕过周期瓶颈（研究方向）

**思路**：完全放弃"编码模幂→找周期→gcd"的Shor范式，改为直接编码N的除数结构为约束截面上的信息场缺陷构型。让信息场图热扩散（0.4.4第一/二层）自然驱动复合缺陷裂解为不可约子缺陷。直接读出因子——不经过周期提取。

**几何图像**：合数N的信息场构型是一个包含两个子涡旋的复合缺陷。在图热扩散下，这个复合缺陷是热力学亚稳态——裂解为两个独立不可约缺陷的信息自由能更低。扩散演化自动驱动裂解，就像晶体中的位错在应力下自然移动。裂解后的子缺陷的绕数直接对应因子。

**优势**：
1. 完全绕过周期瓶颈——不需要编码r
2. 利用了信息场的实空间动力学（扩散），而非谱空间（球谐分解）
3. 最接近"物理计算"的直觉——让约束截面自己"裂开"，不让人去"找"

**当前障碍**：
1. 整数→缺陷构型的编码映射尚未定义（这比源项构造更基础）
2. 缺陷裂解的动力学判据待建立（何时裂解？裂解速度？）
3. 缺乏"缺陷绕数↔因子值"的读出映射

**与现有框架的衔接**：0.4.4的第一层（自由扩散∂ρ/∂σ = −L_G ρ）和第二层（源驱动）已提供了扩散动力学骨架。3号文章的二能级定理和几何最大纠缠提供了缺陷的基元（约束截面上的不可约激发）。缺的是"整数→初始缺陷构型"的映射。

> **定理地位**：研究方向。当前没有任何定理支持此路径的可行性。但0.4.4的动力学方程和3号文章的纠缠基元为其提供了物理基础。

### §6.3　路径Z：放弃通用RSA，聚焦几何相关弱密钥（已可行）

**思路**：接受通用RSA在当前框架内不可行，将资源聚焦于**几何相关数类**（51号定义5.1–5.3）。51号定理5.1已证明：对于𝒩_geo^(semi)(ε)中的半素数，策略C可直接读取因子——无需Bott折叠，无需周期提取。

**已建立的事实**：
1. 几何相关素数约占所有素数的7%（51号命题5.1，ε=0.1）
2. 策略C对𝒩_geo^(semi)的管线是完全自洽的（51号定理5.1）
3. 编码复杂度为O(N_eff²)（51号§6.2），即O(49)——常数级

**应用方向**：扫描已知RSA弱密钥数据库，检验其中是否存在几何相关半素数。这可以将几何计算从"理论纲领"推进到"已演示的检测工具"。

> **定理地位**：定理级（51号定理5.1）。此路径不涉及任何待证命题。

### §7.2　核心信息论障碍

本文的核心发现是：**Bott折叠保周期在模运算路径上面临的不只是"未证明"，而是信息论原则上的容量不足。** 七个层级（k=0,…,6）的约化周期受限于值域大小（≤3^k），无论采用何种编码方案，从中可提取的关于原始周期r的信息量受限于log₂(3²¹) ≈ 33位。双谱扩至115位，三谱扩至153位——均不足以覆盖RSA-2048的997位需求。

这个障碍不是工程性的（"谱分辨率不够"），而是原则性的（"信息容器太小"）。

### §7.3　已建立的正面结果

1. **定理2.1**：r_k ≤ 3^k（严格证明，三行）
2. **定理3.1–3.2**：七层级联合至多提供33位关于r的信息（严格证明）
3. **定理5.1**：对r ≤ 8×10⁹（域I），管线自洽可行
4. **路径Z**：几何相关数类的分解已在定理级建立（51号定理5.1）

### §7.4　对几何计算纲领的影响

本文的否定性结论**不影响几何计算纲领的核心主张**：

- 几何计算模型（50号）的形式化不受影响——计算=信息场演化，该定义独立于特定算法
- 硬件组件（qubit/纠缠/傅里叶变换/并行）全部由既有定理支撑，不受影响
- 策略C（几何相关弱密钥检测）是定理级的，不受影响
- 策略A对中小周期（域I）是定理级的，不受影响

受影响的只是：**"策略B可通过Bott折叠在7层级内压缩RSA-2048级周期"这一特定主张。**

### §7.5　最重要的诚实声明

**本文严格证明：沿模运算路径，Bott折叠（7层级周期约化）在信息论上不足以将RSA-2048级周期压缩到可分辨范围。** 这一定理（定理5.2）在当前几何论框架内是坚实的——其证明仅依赖鸽巢原理、N_info的固定值和基本信息计数，不涉及任何未验证的猜想。

但这不意味着"几何计算不能破解RSA"——它只意味着"不能通过模运算的7层级周期约化来破解"。路径Y（缺陷裂解）和路径X（多基底联合）可能提供绕行方案，但两者均处于研究方向级别，无定理支撑。

**本文的价值在于精确刻画了Bott折叠的"能做"与"不能做"，从而将后续研究聚焦于真正有希望的突破方向——而非在已被证明不可行的路径上浪费精力。**
