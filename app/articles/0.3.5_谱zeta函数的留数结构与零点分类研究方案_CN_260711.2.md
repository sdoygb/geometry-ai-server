# 0.3.5 谱zeta函数的留数结构与零点分类研究方案

**版本：** 0.3.5 | **日期：** 2026-07-11 | **状态：** 研究方案（步骤A：定理级草案；步骤B：数值路线图） | **依赖：** 0.0.5（热核与指标定理）、0.3.1（谱三重态，主库已验证）、0.3.4（谱zeta函数构造）

## 摘要

本文承接 0.3.4 对谱zeta函数 $\zeta_D(s)=\mathrm{Tr}(|D|^{-s})$ 的合法化构造，将其从"定义存在"推进到"内部几何分析"。全文分为两条并行进路：

- **步骤A（留数-曲率字典）：** 建立 $\zeta_D(s)$ 在极点处的Wodzicki留数与Minakshisundaram-Pleijel热核展开系数之间的精确对应，将留数表达为 $M(a)=S^3\times S^3\times S^3$ 的曲率不变量多项式。目标是形成可提交主库验证的定理链。
- **步骤B（零点分类）：** 对 $M(a)$ 上八种自旋结构下的Dirac谱做系统数值研究，分类 $\zeta_D(s)$ 在复平面上的零点分布。明确不预设其与黎曼 $\zeta(s)$ 零点的任何对应关系；任何结构呼应必须是数据驱动的事后发现，而非先验假设。

**诚实声明：** 本文不引用任何关于"黎曼猜想是公理3推论"或"$\theta_M=\theta_C$ 对称轴"的断言，因为文章库中不存在支撑这些断言的定理（参见此前对 0.577:0.577:1 比例及对称轴定义的审核）。本文只做 0.3.4 之后逻辑上最扎实的延伸。

## 1. 引言：从合法化到内部几何

在 0.3.4 中，我们已在几何论语境内完成了以下定理级构造：

- **定理2.1（解析延拓）：** 对紧9维流形 $M(a)=S^3\times S^3\times S^3$ 上的Dirac算符 $D$，$\zeta_D(s)$ 在 $\mathrm{Re}(s)>9$ 绝对收敛，且可亚纯延拓至 $\mathbb{C}$，极点仅出现在 $s=9,7,5,3,1$。
- **定理2.2（极点结构）：** 这些极点的留数由热核 $\mathrm{Tr}(e^{-tD^2})$ 在 $t\to 0^+$ 时的渐近展开系数唯一决定。
- **定理2.3（函数方程雏形）：** 由Dirac谱的 $\lambda\leftrightarrow -\lambda$ 对称性，$\zeta_D(s)$ 满足偶性 $s\mapsto -s$ 的某种残余形式（具体取决于零模处理）。

但 0.3.4 停留在**通用定理**层面：它适用于任何满足谱三元组条件的紧流形，尚未触及 $M(a)$ 的具体几何。本文的任务是：**把 $M(a)$ 的特定曲率数据写进留数公式，并系统探索其谱zeta函数的零点地貌。**

## 2. 步骤A：留数-热核-曲率精确对应

### 2.1 已知输入：热核展开（来自 0.0.5）

0.0.5 §7 已确立：对闭Riemannian流形 $(M,g)$ 上的广义Dirac-Laplace算符 $D^2$，存在渐近展开

$$\mathrm{Tr}(e^{-tD^2}) \sim \sum_{k=0}^{\infty} a_k(D^2)\, t^{(k-\dim M)/2}, \qquad t\to 0^+.$$

对于 $M(a)$，$\dim M(a)=9$，故

$$\mathrm{Tr}(e^{-tD^2}) \sim a_0 t^{-9/2} + a_1 t^{-7/2} + a_2 t^{-5/2} + a_3 t^{-3/2} + a_4 t^{-1/2} + O(t^{1/2}).$$

系数 $a_k(D^2)$ 是 $M(a)$ 上的局部曲率不变量积分（Minakshisundaram-Pleijel系数）。0.0.5 默认了该展开对指标定理和 $\hat{A}$-类的应用，但未给出 $M(a)$ 的显式 $a_k$。

### 2.2 Mellin 桥：从热核到谱zeta

由 Mellin 变换（已在 0.3.4 §2.1 合法化）：

$$\zeta_D(s) = \frac{1}{\Gamma(s/2)}\int_0^{\infty} t^{s/2-1}\,\mathrm{Tr}(e^{-tD^2})\,dt.$$

将热核展开代入，对积分在 $t=0$ 附近做渐近分析，可得极点-留数公式：

**命题A.1（留数-热核对应，待证）。** 对 $M(a)=S^3\times S^3\times S^3$（等半径 $a$，标准乘积度量），谱zeta函数 $\zeta_D(s)$ 在 $s=9-2k$（$k=0,1,2,3,4$）处的留数为

$$\mathrm{Res}_{s=9-2k}\,\zeta_D(s) = \frac{2\,a_k(D^2)}{\Gamma\!\left(\frac{9-2k}{2}\right)}.$$

其中 $a_k(D^2)$ 由以下曲率数据完全确定：

| $k$ | $a_k$ 的主导几何量 | 对 $S^3\times S^3\times S^3$ 的预测形式 |
|:---:|:---|:---|
| 0 | 体积 $\mathrm{Vol}(M(a))$ | $a_0 = (4\pi)^{-9/2}\,\mathrm{Vol} = (4\pi)^{-9/2}(2\pi^2 a^3)^3$ |
| 1 | 标量曲率积分 $\int R\,\sqrt{g}\,d^9x$ | $a_1 \propto a^7$（需显式计算） |
| 2 | $|Riem|^2, |Ric|^2, R^2$ 的组合 | $a_2 \propto a^5$（需显式计算） |
| 3 | 曲率张量及其协变导数的二次/四次组合 | $a_3 \propto a^3$（需显式计算） |
| 4 | 更高阶曲率不变量 | $a_4 \propto a^1$（需显式计算） |

**证明策略：**
1. 对单因子 $S^3$（半径 $a$，常截面曲率 $K=1/a^2$），其热核系数 $a_k^{(S^3)}$ 是经典结果（Gilkey, 1995）。
2. 对乘积流形 $M_1\times M_2$，热核满足 $e^{-t\Delta_{M_1\times M_2}} = e^{-t\Delta_1}\otimes e^{-t\Delta_2}$，因此 $a_k$ 是各因子系数的**卷积和**：
   $$a_k^{(M_1\times M_2)} = \sum_{i+j=k} a_i^{(M_1)} a_j^{(M_2)}.$$
3. 对Dirac-Laplace $D^2$（而非标量Laplace），需引入旋量曲率修正（$\frac{1}{4}R$ 项），这在 0.0.5 的指标定理语境中已标准化。
4. 将三步推广至 $S^3\times S^3\times S^3$，得到 $a_0$ 到 $a_4$ 的显式 $a$-依赖多项式。

**状态：** $k=0$ 为体积项，已确定；$k\ge 1$ 的显式组合公式需草稿验证。预计可形成 **2–3 条可提交主库的几何定理**。

### 2.3 Wodzicki 非交换留数的几何论语境

Wodzicki留数是伪微分算符迹的唯一（ up to 常数）非平凡延拓。对 $|D|^{-s}$，其在 $s=9$ 处的留数与 $a_0$ 的关系为：

$$\mathrm{Res}(|D|^{-9}) = 2a_0 = \frac{2}{(4\pi)^{9/2}}\mathrm{Vol}(M(a)).$$

对乘积流形，这给出**体积-留数对偶**：$M(a)$ 的体积完全编码在谱zeta的最高阶极点中。

**定理草案A.2（体积-留数对偶）。** 设 $M(a)=S^3\times S^3\times S^3$ 配备标准乘积度量，$D$ 为Dirac算符，则

$$\mathrm{Res}_{s=9}\,\zeta_D(s) = \frac{(2\pi^2)^3}{(4\pi)^{9/2}}\,a^9 = \frac{\pi^{3/2}}{2^{3/2}}\,a^9.$$

**状态：** 此式可直接从 $S^3$ 体积 $2\pi^2 a^3$ 和乘积公式导出，属于**低难度定理**，适合作为步骤A的首个主库提交。

## 3. 步骤B：零点分类的数值研究方案

### 3.1 核心对象：$\zeta_D(s)$ 的零点是什么？

在 0.3.4 中，$\zeta_D(s)$ 被定义为

$$\zeta_D(s) = \sum_{k} |\Lambda_k|^{-s},$$

其中 $\{\Lambda_k\}$ 是 $D$ 的非零特征值（含重数）。对紧流形，$\zeta_D(s)$ 是整函数（扣除极点后），其**零点**由以下因素决定：
- 谱的**整体分布**（Weyl定律给出的渐近密度）；
- 谱的**局部聚簇**（重数、近简并）；
- 低能谱的**具体数值**（前几十个特征值对零点位置有决定性影响）。

**关键事实：** 标准Dirac算符的谱zeta函数零点**一般不落在一个临界线上**。历史上只有非常特殊的算符（如环面 $T^n$ 上经Poisson求和构造的）才会产生类似黎曼零点的分布。因此，本步骤的诚实目标是**先分类、再判断**，而非先假设。

### 3.2 Dirac 谱的显式构造（$M(a)$ 的输入数据）

#### 3.2.1 单因子 $S^3$ 的谱

对半径为 $a$ 的 $S^3$（标准度量，常截面曲率 $1/a^2$），Dirac算符的特征值为（参见标准谱几何文献，如 Bär 1996）：

$$\lambda_n = \pm\frac{1}{a}\left(n + \frac{3}{2}\right), \qquad n = 0, 1, 2, \dots$$

重数为

$$m_n = 2(n+1)(n+2).$$

**自旋结构的影响：** $S^3$ 有两种自旋结构：
- **周期型（untwisted）：** 谱如上，$n\in\mathbb{Z}_{\ge 0}$。
- **反周期型（twisted / 旋量结构改变）：** 最低模式偏移，$\lambda_n = \pm\frac{1}{a}(n+1)$，重数相应改变。

**诚实标注：** 几何论 0.3.1 构造了谱三元组 $(\mathcal{A},\mathcal{H},D,J,\gamma)$，但**未显式声明**采用哪种自旋结构。本文的数值方案将**覆盖全部 $2^3=8$ 种组合**，并在结果中标注依赖性。

#### 3.2.2 乘积流形的扭结谱

对 $M(a)=S^3_1\times S^3_2\times S^3_3$，Dirac算符在乘积度量下的标准扭结形式为

$$D = D_1\otimes\gamma^2\otimes\gamma^3 + \mathbb{I}\otimes D_2\otimes\gamma^3 + \mathbb{I}\otimes\mathbb{I}\otimes D_3,$$

其中 $\gamma^i$ 是第 $i$ 个因子Clifford代数上的自伴表示元。由于各因子度量相容且Clifford表示满足 $\{\gamma^i,\gamma^j\}=2\delta^{ij}$，$D^2$ 的谱满足

$$\Lambda^2 = (\lambda_{n_1}^{(1)})^2 + (\lambda_{n_2}^{(2)})^2 + (\lambda_{n_3}^{(3)})^2.$$

（严格来说，扭结结构会导致额外的交叉项，但在等半径、标准表示下，$D^2$ 的谱恰好是平方和——这是乘积流形Dirac算符的经典性质。）

因此，$M(a)$ 的Dirac谱（平方后）是三个 $S^3$ 谱的**平方和格点**：

$$\Lambda_{n_1,n_2,n_3} = \pm\sqrt{\frac{(n_1+3/2)^2 + (n_2+3/2)^2 + (n_3+3/2)^2}{a^2}}.$$

**重数**为各因子重数之积：$M_{n_1,n_2,n_3} = m_{n_1}m_{n_2}m_{n_3}$。

**若采用反周期自旋结构**，将 $n_i+3/2$ 替换为相应的偏移量（如 $n_i+1$ 或 $n_i+2$）。

### 3.3 数值算法设计

#### 3.3.1 截断策略

由于重数 $m_n\sim 2n^2$ 增长，三重求和的总项数随截断 $N_{\max}$ 以 $O(N_{\max}^9)$ 增长。必须采用**分层截断**与**渐近尾补**：

1. **低能区（$n_i \le N_{\mathrm{cut}}$）：** 精确求和。取 $N_{\mathrm{cut}}\sim 50$，产生约 $51^3\approx 1.3\times 10^5$ 个格点，每个格点重数可高达 $O(10^6)$，需高效处理。
2. **高能尾（$n_i > N_{\mathrm{cut}}$）：** 用Weyl定律的积分近似代替离散求和。对 $s$ 在 $\mathrm{Re}(s)>9$ 的收敛区，尾部可解析估计；对解析延拓后的区域，利用热核展开的Euler-Maclaurin校正。

#### 3.3.2 零点搜索

目标：在复平面区域

$$\Omega = \{s\in\mathbb{C} : -10 \le \mathrm{Re}(s) \le 10,\; -T_{\max} \le \mathrm{Im}(s) \le T_{\max}\}$$

内搜索 $\zeta_D(s)=0$。取 $T_{\max}\sim 100$ 作为初步扫描。

**算法：**
- **步骤1：** 在 $\Omega$ 上建立网格（初始粗网格 $0.5\times 0.5$），计算 $|\zeta_D(s)|$。
- **步骤2：** 对局部极小值点使用 Newton-Raphson 或复变函数 Argument Principle（围道积分 $\\frac{1}{2\pi i}\oint \\frac{\zeta_D'}{\zeta_D}ds$ 计数零点）。
- **步骤3：** 验证零点的**重数**与**对称性**。由于 $\zeta_D(s)$ 在实轴上取实值（对实谱），零点关于实轴共轭对称：若 $s_0$ 是零点，则 $\bar{s}_0$ 也是。

### 3.4 输出目标与分类标准

对每种自旋结构 $(\varepsilon_1,\varepsilon_2,\varepsilon_3)$ 和半径参数 $a$，输出：

1. **平凡零点/消去零点：** 是否在某些负偶数 $s=-2,-4,\dots$ 处因Gamma函数极点与分子零点抵消而出现？（对标量Laplace的谱zeta常见，对Dirac谱zeta需重新检验。）
2. **实轴零点：** 在 $\mathrm{Im}(s)=0$ 上的零点位置。这些通常由低能谱的"竞争"决定。
3. **非实零点：** 共轭成对出现的复零点。记录其实部分布 $\mathrm{Re}(s)$ 的直方图。
4. **统计检验：** 若存在非实零点，检验其最近邻间距分布是否偏离泊松分布（量子混沌vs可积的判别）。

**诚实标注：** 上述第4项是对 Montgomery-Odlyzko 定律的**测试**，而非**预期**。如果分布呈泊松型（可积系统），则与黎曼零点的GUE统计截然不同；如果呈Wigner-Dyson型，则值得进一步研究。

## 4. 与黎曼 $\zeta(s)$ 的关系：接口而非桥梁

此前上传的多份纲领将几何论与黎曼猜想的关系描述为"三重桥梁"，并声称 $\theta_M=\theta_C$ 对应临界线 $\mathrm{Re}(s)=1/2$。经审核，这些断言存在以下问题：

- 几何论的物理锁定构型为 $(\theta_M,\theta_C,\theta_I)=(57.93^\circ, 26.16^\circ, 5.91^\circ)$，不满足 $\theta_M=\theta_C$；
- 0.0.7e 定理4.3（主库#12）定义的对称轴为 $\theta_C=\theta_I$，而非 $\theta_M=\theta_C$；
- 比例 $0.577:0.577:1$ 在文章库中无任何出处；
- 文章库中不存在编号66的文章，亦不存在"黎曼猜想是公理3必然推论"的定理。

因此，本文采取**更保守、更诚实**的表述：

> 黎曼 $\zeta(s)$ 与几何论谱zeta $\zeta_D(s)$ 之间目前**不存在已验证的数学对应**。步骤A和B的目标是**先建立 $\zeta_D(s)$ 的自身定理体系**。如果数值研究（步骤B）发现 $\zeta_D(s)$ 的零点具有某种直线分布规律（例如实部集中于某常数 $c$），则可在后续工作中提出"临界线接口"猜想；否则，几何论对黎曼猜想的贡献将停留在**谱几何方法论**层面（即提供构造Hilbert-Pólya算符的候选框架），而非直接证明。

## 附录A：$S^3$ 热核系数的显式值（参考）

对半径 $a$ 的 $S^3$（标量Laplace算符 $\Delta$），经典热核系数为：

$$a_0^{(S^3)} = (4\pi)^{-3/2}\cdot 2\pi^2 a^3 = \frac{\sqrt{\pi}}{2\sqrt{2}} a^3,$$
$$a_1^{(S^3)} = (4\pi)^{-3/2}\int R\,\sqrt{g}\,d^3x = \frac{6}{a^2}\cdot\frac{\sqrt{\pi}}{2\sqrt{2}}a^3 = 3\sqrt{2\pi}\,a,$$

其中 $R=6/a^2$ 为 $S^3$ 标量曲率。对Dirac-Laplace $D^2=\Delta+\frac{1}{4}R$，热核系数有熟知的旋量修正。乘积流形 $S^3\times S^3\times S^3$ 的系数 $a_k$ 是上述单因子系数的**三重卷积**：

$$a_k^{(M)} = \sum_{i+j+\ell=k} a_i^{(S^3_1)}a_j^{(S^3_2)}a_\ell^{(S^3_3)}.$$

例如：
$$a_0^{(M)} = [a_0^{(S^3)}]^3 \propto a^9,$$
$$a_1^{(M)} = 3[a_0^{(S^3)}]^2 a_1^{(S^3)} \propto a^7.$$

更高阶系数的显式多项式需在草稿中逐项核对。

## 附录B：自旋结构对谱的影响

$S^3\cong SU(2)$ 具有自然的双覆盖自旋结构。对**反周期自旋结构**（由 $SU(2)$ 的非平凡一维表示给出），Dirac特征值的偏移量为：

$$\lambda_n^{(\mathrm{tw})} = \pm\frac{1}{a}(n+1), \qquad m_n^{(\mathrm{tw})} = 2(n+1)^2.$$

$M(a)$ 的8种自旋结构对应三个因子各自独立选择周期/反周期，共产生8种不同的"格点偏移+重数修正"。数值研究必须遍历这8种情形，并在最终报告中以表格形式对比其谱zeta的零点差异。

## 附录C：数值误差估计

截断误差上界：对 $\mathrm{Re}(s)>9$，尾部可用积分估计

$$\left|\sum_{n>N_{\mathrm{cut}}} m_n |\lambda_n|^{-s}\right| \le C\int_{N_{\mathrm{cut}}}^{\infty} n^2\cdot n^{-\mathrm{Re}(s)}\,dn = O(N_{\mathrm{cut}}^{3-\mathrm{Re}(s)}).$$

对解析延拓至 $\mathrm{Re}(s)\le 9$ 的区域，需利用热核展开前 $K$ 项做**减除-补回**：

$$\zeta_D(s) = \sum_{|\Lambda_k|<\Lambda_{\mathrm{cut}}} |\Lambda_k|^{-s} + \frac{1}{\Gamma(s/2)}\int_0^{\infty} t^{s/2-1}\left[\mathrm{Tr}(e^{-tD^2}) - \sum_{k=0}^{K} a_k t^{(k-9)/2}\right]dt,$$

其中方括号内在 $t\to 0$ 时以 $O(t^{(K+1-9)/2})$ 衰减，保证积分在更大范围内收敛。
