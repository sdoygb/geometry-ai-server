## §1　错误诊断：0.0.7e的作用量形式错误

### 1.1 0.0.7e的声明

0.0.7e §1.3声称六项作用量S的耦合项为csc²(θ_X+θ_Y)：

$$S_{\text{0.0.7e}} = \sum_X \csc^2\theta_X + \csc^2(\theta_M+\theta_C) + \csc^2(\theta_M+\theta_I) + \csc^2(\theta_C+\theta_I).$$

基于此形式推导出的Hessian闭式为：

$$H_{MM}^{\text{0.0.7e}} = \frac{2(1+2\cos^2\theta_M)}{\sin^4\theta_M} + \frac{2(1+2\cos^2(\theta_M+\theta_C))}{\sin^4(\theta_M+\theta_C)} + \frac{2(1+2\sin^2\theta_C)}{\cos^4\theta_C},$$
$$H_{MI}^{\text{0.0.7e}} = \frac{2(1+2\sin^2\theta_C)}{\cos^4\theta_C}.$$

并由此得出裸Hessian下H_MI/H_MM→0，从而(c₁+c₂)/2→3。

### 1.2 正确的作用量——对照

0.0.6定义4.3（公理2的精确表述）给出：

$$S = \sum_{i=1}^{3}\frac{1}{\sin^2\theta_i} + \sum_{i<j}\frac{1}{\sin\theta_i\sin\theta_j}.$$

展开写为六项：

$$S = \frac{1}{\sin^2\theta_M} + \frac{1}{\sin^2\theta_C} + \frac{1}{\sin^2\theta_I} + \frac{1}{\sin\theta_M\sin\theta_C} + \frac{1}{\sin\theta_M\sin\theta_I} + \frac{1}{\sin\theta_C\sin\theta_I}.$$

**关键区别**：

| 项 | 正确形式 | 0.0.7e错误形式 |
|:---|:---|:---:|
| M-C耦合 | 1/(sinθ_M sinθ_C) | csc²(θ_M+θ_C) |
| M-I耦合 | 1/(sinθ_M sinθ_I) | csc²(θ_M+θ_I) |
| C-I耦合 | 1/(sinθ_C sinθ_I) | csc²(θ_C+θ_I) |

这两个函数在几何上完全不同：

- 1/(sinθ_X sinθ_Y)：当θ_X,θ_Y→0时→∞（极点在对角线θ_X=0或θ_Y=0处）
- csc²(θ_X+θ_Y)：当θ_X+θ_Y→0时→∞（极点沿整条线θ_X+θ_Y=0）

在基准点θ_M=57.93°, θ_C=26.16°处，两者的数值也显著不同：
- 1/(sin57.93° sin26.16°) = 1/(0.8476×0.4409) = 2.677
- csc²(57.93°+26.16°) = csc²84.09° = 1/sin²84.09° = 1/0.9896² = 1.021

量级差异约2.6倍。在倒伏相极限下，差异更大。

**结论**：0.0.7e使用了与0.0.6公理2不一致的作用量形式，其全部推导无效。

## §2　正确作用量下的Hessian闭式

### 2.1 预备：0.2.1的Hessian公式

0.2.1 §3.3已给出正确的Hessian计算公式。为方便后续推导，在此整理为闭式形式。

**引理2.1（基本导数公式，正确形式）**（0.2.1 §3.3）：

对单场项f(θ)=1/sin²θ：

$$f'(\theta) = -\frac{2\cos\theta}{\sin^3\theta}, \quad f''(\theta) = \frac{2}{\sin^2\theta} + \frac{6\cos^2\theta}{\sin^4\theta} = \frac{2+4\cos^2\theta}{\sin^4\theta}.$$

对耦合项g(θ,φ)=1/(sinθ sinφ)：

$$\frac{\partial g}{\partial\theta} = -\frac{\cos\theta}{\sin^2\theta}\cdot\frac{1}{\sin\varphi}, \quad \frac{\partial^2 g}{\partial\theta^2} = \frac{1+\cos^2\theta}{\sin^3\theta\sin\varphi},$$
$$\frac{\partial^2 g}{\partial\theta\partial\varphi} = \frac{\cos\theta\cos\varphi}{\sin^2\theta\sin^2\varphi}.$$

*证明*：直接求导。单场项f(θ)=csc²θ的二阶导已由0.2.1§3.3给出。耦合项g(θ,φ)=cscθ·cscφ，求导后得上述形式。∎

### 2.2 六个Hessian矩阵元的闭式

**定理2.1（正确Hessian闭式）**：六项作用量S在约束点(θ_M,θ_C,θ_I)（θ_M+θ_C+θ_I=90°）处的Hessian矩阵元的显式闭式为：

$$\boxed{H_{MM} = \left[\frac{2}{\sin^2\theta_M} + \frac{6\cos^2\theta_M}{\sin^4\theta_M}\right] + \frac{1+\cos^2\theta_M}{\sin^3\theta_M\sin\theta_C} + \frac{1+\cos^2\theta_M}{\sin^3\theta_M\sin\theta_I}},$$

$$\boxed{H_{CC} = \left[\frac{2}{\sin^2\theta_C} + \frac{6\cos^2\theta_C}{\sin^4\theta_C}\right] + \frac{1+\cos^2\theta_C}{\sin^3\theta_C\sin\theta_M} + \frac{1+\cos^2\theta_C}{\sin^3\theta_C\sin\theta_I}},$$

$$\boxed{H_{II} = \left[\frac{2}{\sin^2\theta_I} + \frac{6\cos^2\theta_I}{\sin^4\theta_I}\right] + \frac{1+\cos^2\theta_I}{\sin^3\theta_I\sin\theta_M} + \frac{1+\cos^2\theta_I}{\sin^3\theta_I\sin\theta_C}},$$

$$\boxed{H_{MC} = \frac{\cos\theta_M\cos\theta_C}{\sin^2\theta_M\sin^2\theta_C}},$$

$$\boxed{H_{MI} = \frac{\cos\theta_M\cos\theta_I}{\sin^2\theta_M\sin^2\theta_I}},$$

$$\boxed{H_{CI} = \frac{\cos\theta_C\cos\theta_I}{\sin^2\theta_C\sin^2\theta_I}}.$$

*证明*：对角元由单场项的二阶导（引理2.1第一式）加上耦合项的二阶对角导（引理2.1第二式）相加。例如H_MM：单场项csc²θ_M贡献第一项大括号；耦合项1/(sinθ_M sinθ_C)对θ_M的二阶导贡献第二项；耦合项1/(sinθ_M sinθ_I)对θ_M的二阶导贡献第三项。交叉项直接来自耦合项的混合偏导（引理2.1第三式）。∎
