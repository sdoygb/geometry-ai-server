# 第1章 几何预备：约束截面与内禀坐标

**摘要：** 本章为信息场动力学建立完整的几何舞台。从 Vol-1 三分切丛的约束截面出发，引入内禀坐标 $(\xi,\eta)$，计算诱导度量与 Laplace-Beltrami 算子，为后续信息密度场方程给出严格的谱几何基础。

---

## 1.1 约束截面

设 $(\Sigma, g, \mathcal{A})$ 为 Vol-1 构造的几何约束截面，其中 $\Sigma$ 为 $C^\infty$ 紧致无边的 2 维可定向 Riemann 流形，$g$ 为 Riemann 度量，$\mathcal{A}$ 为谱三元组 $(\mathcal{H}, D, J, \gamma)$ 诱导的 Clifford 模结构。

**定义 1.1（约束截面，GT-3A.1.1.1）。** 几何约束截面 $\Sigma$ 是满足以下条件的子流形：

1. **维数条件：** $\dim(\Sigma)=2$，嵌入于 3 维球面 $S^3_\Sigma$；
2. **三分解切丛：** $T\Sigma = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$，其中各扇区纤维维数为 $(\dim\mathcal{M},\dim\mathcal{C},\dim\mathcal{I})=(1,1,0)$；
3. **完备性约束：** 扇区角满足 $\theta_M + \theta_C + \theta_I = \pi/2$；
4. **谱刚性条件：** $\Sigma$ 是谱三元组 $(\mathcal{H}, D)$ 的 Dirac 算子在迹类光滑函数作用下的极小化子。

由 Vol-1 §1.5 谱刚性定理，满足上述条件的 $\Sigma$ 在同胚意义下唯一，其 Gauss 曲率 $K_\Sigma$ 为常数。

**定义 1.2（参考点 $P_e$，GT-3A.1.1.2）。** 在 $\Sigma$ 上指定参考点 $P_e = (\theta_M^0, \theta_C^0, \theta_I^0)$，其中：

- $\theta_M^0 = 57.93^\circ$（物质扇区基准角）
- $\theta_C^0 = 26.16^\circ$（因果扇区基准角）
- $\theta_I^0 = 5.91^\circ$（信息扇区基准角）

该三点满足 $\theta_M^0+\theta_C^0+\theta_I^0=90^\circ$。

**定义 1.3（$S_e$ 定位，GT-3A.1.1.3）。** 几何本征量 $S_e = 137.035999084$ 定位 $\Sigma$ 的谱零点：

$$S_e = \sum_{n=0}^\infty \frac{1}{\lambda_n + S_e^{-1}}$$

其中 $\{\lambda_n\}$ 为 $\Sigma$ 上 Dirac 算子的特征值谱。$S_e$ 的量纲由 $[\text{rad}]^0$ 确定，是纯几何常数。

---

## 1.2 内禀坐标

**定义 1.2.1（$(\xi,\eta)$ 坐标，GT-3A.1.2.1）。** 在参考点 $P_e$ 附近，引入局部坐标：

- $\xi$：沿 $\mathcal{I}$-扇区冻结方向的法向偏离，量纲为 $[\text{rad}]$
- $\eta$：沿 $\mathcal{M}\oplus\mathcal{C}$ 截面的切向偏离，量纲为 $[\text{rad}]$

坐标原点 $(\xi,\eta)=(0,0)$ 对应 $P_e$。

坐标变换由三分切丛的扇区角参数化给出：

$$\xi = \sin(\theta_I - \theta_I^0) \cdot R_\Sigma, \quad \eta = \cos(\theta_I - \theta_I^0) \cdot R_\Sigma$$

其中 $R_\Sigma$ 为 $\Sigma$ 的截面曲率半径，$R_\Sigma = 1/\sqrt{K_\Sigma}$。

极值点定位：$\Sigma$ 上存在两个全局极小点 $Q_1, Q_2$（$S$-极小值），在 $(\xi,\eta)$ 坐标系中位于：

$$Q_1: (\xi, \eta) = (+\xi_0, 0), \quad Q_2: (\xi, \eta) = (-\xi_0, 0)$$

其中 $\xi_0 = \pi/2 - \theta_I^0 = 84.09^\circ$（以弧度计为 1.4678 rad）。

---

## 1.3 诱导度量

**定理 1.3（诱导度量，GT-3A.1.3.1）。** 在 $(\xi,\eta)$ 坐标系中，$\Sigma$ 的诱导度量 $g$ 取对角形式：

$$g = d\xi^2 + h(\xi)^2 d\eta^2$$

其中 $h(\xi) = \cos(\xi/R_\Sigma)$ 为径向函数，满足 $h(0)=1, h(\pm\xi_0)=0$。

**证明：** 由 $\Sigma \subset S^3_\Sigma$ 的等距嵌入，$S^3_\Sigma$ 上的标准度量在 $\Sigma$ 上的拉回给出：

$$g = R_\Sigma^2\left(d\theta^2 + \sin^2\theta\, d\phi^2\right)$$

经过坐标变换 $\xi = R_\Sigma(\theta - \pi/2)$，$\eta = R_\Sigma\phi$，得到对角形式。$\square$

**推论 1.3.1（Laplace-Beltrami 算子）。** $\Sigma$ 上的 Laplace-Beltrami 算子为：

$$\Delta_\Sigma = \frac{\partial^2}{\partial\xi^2} + \frac{h'(\xi)}{h(\xi)}\frac{\partial}{\partial\xi} + \frac{1}{h(\xi)^2}\frac{\partial^2}{\partial\eta^2}$$

其中 $h'(\xi) = -\sin(\xi/R_\Sigma)/R_\Sigma$。

在参考点 $P_e=O$ 附近的小邻域内，$h(\xi) \approx 1$，算子简化为平直形式：

$$\Delta_\Sigma \approx \frac{\partial^2}{\partial\xi^2} + \frac{\partial^2}{\partial\eta^2}$$

## 1.4 三分切丛回顾

由 Vol-1 §1.2 三分切丛定理，$\Sigma$ 的切丛三分解 $T\Sigma = \mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 满足：

1. **正交分解：** 三个子丛在 $g$ 下两两正交
2. **扇区角：** $\mathcal{M}$-扇区角 $\theta_M$ 为 $\mathcal{M}$ 纤维与 $\mathcal{I}$-冻结方向的夹角
3. **互扼条件：** $\theta_C = \pi/2 - \theta_M - \theta_I$ 锁定扇区间的几何关系

在 $(\xi,\eta)$ 坐标下，三扇区对应的方向为：

$$\mathcal{M}: \text{沿 } \partial_\eta, \quad \mathcal{C}: \text{沿 } \partial_\xi \text{（绕 $\eta$ 轴旋转 $\theta_C$）}, \quad \mathcal{I}: \text{沿 } \partial_\xi \text{（硬模冻结方向）}$$

**定义 1.4（约束截面上的势盆地，GT-3A.1.4.1）。** $\Sigma$ 上的 $S_e$ 约束函数 $F_{S_e}: \Sigma \to \mathbb{R}$ 定义为：

$$F_{S_e}(p) = \left|\sum_{n=0}^\infty \frac{1}{\lambda_n(p) + S_e^{-1}} - S_e\right|^2$$

近零点近似下，$F_{S_e}$ 在 $Q_1, Q_2$ 处取全局极小值，形成一个双井势盆地结构。
