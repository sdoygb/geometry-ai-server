## 第 1 章 引言与公理系统

### 1.1 十方几何空间的总体框架

十方几何空间是一个基于以下三个层次结构的公理化数学构造：

**定义层**：激发态参数空间 $D = S^1\setminus\{p_0,p_*\}$ 与其两个分支 $D_\pm$；约束乘积球面 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$；三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 及其上的全息屏 $\Sigma$。

**公理层**：三条基础公理——圆拓扑公理、边界极限公理、全息屏编码条件——连同互锁常数 $( \Lambda, k_0, \ell_0 )$ 与桥接构造。

**结构层**：抽象几何量值域定理、谱刚性定理、六项作用量唯一最小值定理、桥接标准形定理、幻数序列定理。

### 1.2 三条基础公理

**公理 1（激发态参数空间的圆拓扑公理）** 激发态参数空间 $D$ 同胚于圆 $S^1$ 去掉两个基点 $p_0$ 与 $p_*$。其上存在连续几何量 $S_{\text{abs}}: D \to (0,+\infty)$，在 $D$ 的两个连通分支 $D_+$ 与 $D_-$ 上分别严格单调。

**公理 2（边界极限公理）** 连续几何量 $S_{\text{abs}}$ 满足以下极限行为：
$$\lim_{x \to p_0} S_{\text{abs}}(x) = 0, \qquad \lim_{x \to p_*} S_{\text{abs}}(x) = +\infty.$$

$p_0$ 与 $p_*$ 分别对应 $S_{\text{abs}}\to 0$ 和 $S_{\text{abs}}\to +\infty$ 的极限点。$S_{\text{abs}}$ 在 $D_+$ 上严格递增，在 $D_-$ 上严格递减（或反之，由约定决定）。

**公理 3（全息屏编码条件）** 在三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个纤维上，存在二维定向子空间 $\Sigma$（称为**全息屏**），以及从每个 $\mathbb{R}^3$ 扇区到 $\Sigma$ 的投影映射，使得扇区投影强度角 $\theta_1,\theta_2,\theta_3$（分别对应 $\mathcal{M},\mathcal{C},\mathcal{I}$ 扇区）满足：

$$\theta_1+\theta_2+\theta_3=90^\circ,\qquad \theta_i>0.$$

**公理 3 的精确内涵**：公理 3 所定义的 $\Sigma$ 是 $TM(a)$ 每个纤维中的一个二维定向线性子空间。在公理层面，$\Sigma$ **仅具有**：维度 2、定向。它**不携带**度规、拓扑、复结构或曲率。任何超出这些属性的附加结构（如 $S^2$ 拓扑、Fubini-Study 度量、Kähler 形式）均不是公理 3 的推论，而是需要进一步论证的涌现性质或框架假设。这一区分是全文几何论证的诚实性基础。

### 1.3 三个互锁常数

**命题 1.0（群论输出常数）** 令 $G = S_3$（三分置换群）。定义
$$\Lambda(G) = |\text{Conj}(G)| = 3, \quad k_0(G) = [G : G'] = 2,$$
其中 $G' = A_3$ 为 $G$ 的换位子群。$\Lambda=3$ 与 $k_0=2$ 为 $S_3$ 群论结构的输出，将在第 3–4 章中锁定几何比例与谱特征签名。

第三常数 $\ell_0$（标度常数）由谱单位选择定理（定理 10.2'）通过体积归一化约定 $V(\ell_0)=1$ 确定，不参与模空间的一维参数化。

## 第 2 章 激发态参数空间的公理化

### 2.1 参数空间的拓扑

由公理 1，$D = S^1\setminus\{p_0,p_*\}$ 是两个不相交开弧之并：
$$D = D_+ \sqcup D_-,$$
其中 $D_+$ 与 $D_-$ 各同胚于开区间 $(0,1)$。此分解由绕 $S^1$ 的定向自然给出——从 $p_0$ 到 $p_*$ 有两条路径，分别对应 $D_+$ 与 $D_-$。

### 2.2 抽象几何量的值域定理

**定理 2.1**（值域定理）$S_{\text{abs}}$ 在 $D_+$ 与 $D_-$ 上的值域均为 $(0,+\infty)$。

*证明.* 由公理 2，$\lim_{x\to p_0}S_{\text{abs}}(x)=0$，$\lim_{x\to p_*}S_{\text{abs}}(x)=+\infty$。公理 1 保证 $S_{\text{abs}}$ 在 $D_+$ 上连续且严格单调，故值域为 $(0,+\infty)$。$D_-$ 同理。$\square$

**定理 2.2**（存在性定理）对任意 $s \in (0,+\infty)$，存在唯一的 $x_+ \in D_+$ 与唯一的 $x_- \in D_-$ 使得 $S_{\text{abs}}(x_\pm) = s$。

*证明.* 由定理 2.1 的满射性与公理 1 的严格单调性，反函数唯一存在。$\square$

## 第 3 章 约束乘积球面的谱刚性

### 3.1 约束乘积球面的定义

**定义 3.1**（约束乘积球面）设 $a>0$ 为正实数（全局尺度因子）。定义九维黎曼流形：
$$M(a) = S^3(a) \times S^3(b) \times S^3(c),$$
其中 $S^3(R)$ 为半径 $R$ 的三维球面，配备从 $\mathbb{R}^4$ 诱导的标准 round 度量，且

$$b = \frac{a}{\sqrt{\Lambda}}, \quad c = \frac{a}{\sqrt{\Lambda k_0}}.$$

在互锁常数 $\Lambda=3$、$k_0=2$ 锁定的前提下，$b = a/\sqrt{3}$，$c = a/\sqrt{6}$。模空间由 $a \in (0,+\infty)$ 一维参数化。

**注 3.0'** 半径比例 $1 : 1/\sqrt{3} : 1/\sqrt{6}$ 是一个几何定义，由群论输出 $\Lambda=3$、$k_0=2$ 嵌入度量比例而确定。它不是群论的必然几何实现——群论仅提供数值 $3$ 与 $2$，将其嵌入半径比例是一种构造性选择。

### 3.2 模空间的参数化

**定理 3.1**（模空间的参数化）约束乘积球面类 $\{M(a)\}_{a>0}$ 的模空间与开区间 $(0,\infty)$ 同胚，其中 $a$ 是唯一的连续自由度。不同 $a$ 对应不同的等距类。

*证明.* 由定义 3.1，$b$ 与 $c$ 被 $\Lambda$ 与 $k_0$ 锁定，唯一的连续自由度是整体标度 $a$。$S^3$ 可平行化使 $TM(a)$ 自动平凡。若 $a \neq a'$，直径 $\text{diam}(M(a)) = \pi a\sqrt{1+1/\Lambda+1/(\Lambda k_0)}$ 不同，因此不是等距的。$\square$

**定理 3.1'**（第一特征值坐标化）谱映射
$$\sigma_1: (0,\infty) \to (0,\infty), \quad a \mapsto \lambda_1^\Delta(a) = \frac{3}{a^2}$$
是微分同胚。第一非零 Laplace 特征值 $\lambda_1^\Delta$ 构成模空间的全局坐标。

*证明.* $\sigma_1$ 连续、严格单调递减，且 $\lim_{a\to 0^+}\sigma_1(a)=+\infty$，$\lim_{a\to+\infty}\sigma_1(a)=0$，故为同胚；$\sigma_1'(a)=-6/a^3\neq 0$，逆映射光滑，故为微分同胚。$\square$

### 3.3 Laplace 特征值的显式计算

**命题 3.1**（乘积流形上的谱分离）$M(a)$ 的 Laplace–Beltrami 特征值全体为
$$\lambda_{p,q,r}^\Delta = \frac{p(p+2)}{a^2} + \frac{q(q+2)}{b^2} + \frac{r(r+2)}{c^2}, \quad p,q,r \in \mathbb{N}_0.$$

**命题 3.2**（前三个特征值的显式表达式）在互锁常数锁定下：

| 模式 (p,q,r) | 特征值表达式 | 与 3/a² 的比值 |
|:---|:---|:---|
| (1,0,0) | 3/a² | 1 |
| (2,0,0) | 8/a² | 8/3 ≈ 2.667 |
| (0,1,0) | 3Λ/a² = 9/a² | Λ = 3 |
| (0,0,1) | 3Λk₀/a² = 18/a² | Λk₀ = 6 |
| (1,1,0) | 3/a² + 3Λ/a² = 12/a² | Λ+1 = 4 |

由于 $\Lambda=3>8/3$，有 $9/a^2 > 8/a^2$，前三个非零特征值恰为 $(1,0,0)$、$(2,0,0)$、$(0,1,0)$。

### 3.4 特征值排序与谱特征签名

**定理 3.2**（特征值排序）对任意 $a>0$，
$$\lambda_1^\Delta = \frac{3}{a^2} < \lambda_2^\Delta = \frac{8}{a^2} < \lambda_3^\Delta = \frac{3\Lambda}{a^2} < \lambda_4^\Delta = \frac{3(\Lambda+1)}{a^2}.$$

**推论 3.1**（特征签名）比值
$$\frac{\lambda_2^\Delta}{\lambda_1^\Delta} = \frac{8}{3}, \quad \frac{\lambda_3^\Delta}{\lambda_1^\Delta} = \Lambda = 3$$
是与 $a$、$k_0$ 无关的普适常数，作为约束乘积球面类的识别签名。

### 3.5 反解公式与自洽性

**定理 3.3**（反解公式）给定前三个非零特征值，可唯一反解：
$$a = \sqrt{\frac{3}{\lambda_1^\Delta}}, \quad b = \sqrt{\frac{3}{\lambda_3^\Delta}}, \quad c = \frac{a}{\sqrt{\Lambda k_0}}.$$

一组正数 $(\lambda_1,\lambda_2,\lambda_3)$ 为某 $M(a)$ 的前三个非零特征值，当且仅当 $\lambda_2 = \frac{8}{3}\lambda_1$ 且 $\lambda_3 = 3\lambda_1$。

### 3.6 谱识别定理

**定理 3.4**（约束乘积球面类的谱识别）设 $M(a)$ 与 $M(a')$ 为两个具有相同互锁常数的约束乘积球面。若它们的前三个非零 Laplace 特征值相同，则 $M(a)$ 与 $M(a')$ 等距。

*证明.* $\lambda_1^\Delta=3/a^2$，等谱即 $a=a'$。由定义 3.1 的比例约束，$(a,b,c)$ 唯一确定，故等距。$\square$

**注 3.1** 这不是广义的"听谱辨形"——这是单参数定义族中的谱可识别性。第一特征值 $\lambda_1^\Delta = 3/a^2$ 本身已构成完备坐标。

### 3.7 与经典反例的兼容性

**定理 3.5**（Sunada 型反例的不适用性）约束乘积球面 $M(a)$ 单连通。对于本文的单参数定义族 $\{M(a)\}_{a>0}$，等距分类由第一特征值 $\lambda_1^\Delta=3/a^2$ 完全确定，不需借助 Sunada 路线的排除——$\lambda_1^\Delta$ 已直接区分不同的 $a$。

**定理 3.7**（Milnor 型反例的不适用性）Milnor 反例基于平坦环面，而 $M(a)$ 具有正截面曲率，几何不兼容。

**定理 3.8**（GWW 型反例的不适用性）GWW 反例基于带边界平面区域，而 $M(a)$ 为无边界的闭流形。

### 3.8 谱刚性定理的封闭性

**定理 3.10**（定义类内的谱封闭性）在约束乘积球面类 $\{M(a)\}_{a>0}$ 内部，等谱 $\Leftrightarrow$ 等距。

## 第 4 章 三分切丛与全息屏结构

### 4.1 三分切丛分解

**命题 4.1**（三分切丛分解）设 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$。其切丛自然分解为三个秩 3 子丛的直和：
$$TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I},$$
其中 $\mathcal{M}$、$\mathcal{C}$、$\mathcal{I}$ 分别对应三个球面因子的切丛。该分解在乘积度量下正交，且每个子丛平凡（因 $S^3$ 可平行化）。

### 4.2 公理 3：全息屏编码条件

公理 3 的完整陈述见 §1.2。$\Sigma$ 在公理层面是纤维中的二维定向线性子空间——不携带度规、拓扑、复结构或曲率。任何超出这些属性的附加结构均非公理 3 的推论，而是需要进一步论证的涌现性质。

### 4.3 全息屏的拓扑识别与度规结构

本节对 $\Sigma$ 的附加几何结构进行分层：先确立拓扑识别（作为框架假设），再讨论度规的涌现机制。

#### 4.3.1 $\Sigma$ 的拓扑识别

**假设 4.1a（$\Sigma$ 的拓扑识别——全局化假设）** 将 $TM(a)$ 上每点纤维中的二维定向子空间 $\Sigma$ 拼合为 $TM(a)$ 的一个二维分布，且该分布可积（满足 Frobenius 条件），从而 $\Sigma$ 叶状结构给出一个全局二维子流形。进一步假设该子流形紧致、连通、可定向，且拓扑同胚于 $S^2$（从而全纯等价于 $\mathbb{CP}^1$）。

**注 4.1a** 假设 4.1a 包含两个非公理的跳跃：(i) 从纤维中的线性子空间到 $TM(a)$ 上的光滑分布——需验证对合性；(ii) 从分布到 $S^2$ 拓扑——在对称点可由 $S_3$ 对称性论证，在非对称点需要构造。假设 4.1a 属于框架的几何输入，而非公理 3 的推论。

#### 4.3.2 $\Sigma$ 的度规与复结构——对称点涌现

**定理 4.1（对称点上 $\Sigma$ 的度规涌现与 Kähler 识别）** 在 $\Sigma \cong \mathbb{CP}^1$（由假设 4.1a）的前提下：

**(a) 对称点涌现**：在 $S_3$ 对称点 $\theta_0 = (30^\circ, 30^\circ, 30^\circ)$，三个扇区在几何上完全等价。由 $S_3$ 对称性，$\Sigma$ 上的任意张量结构若由三分切丛投影诱导，则必为 $SO(3)$-不变的。二维定向流形上的 $SO(3)$-不变 Riemann 度量必为常曲率度量（差尺度因子）。因此，$\Sigma$ 在对称点自然继承一个常曲率 Riemann 度量 $g_\Sigma$。

**(b) 常曲率条件下的 Kähler 识别**：在常曲率下，二维定向流形上可由 $g_\Sigma$ 与定向构造殆复结构 $J$（旋转 $90^\circ$），二维情形下 $J$ 自动可积。定义 Kähler 形式 $\omega(X,Y) = g_\Sigma(X, JY)$，则 $(g_\Sigma, J, \omega)$ 构成 Kähler 三元组。

**(c) 唯一性**：在整体归一化 $\frac{1}{2\pi}\int_\Sigma \omega = 1$ 下，紧致单连通二维常曲率 Kähler 流形上的 Kähler 度量唯一为 Fubini-Study 度量（均匀化定理）。

**(d) 非对称点**：在非对称点 $\theta \neq \theta_0$，三个扇区半径不等，$S_3$ 对称性破缺，定理 4.1(a) 的 $SO(3)$-不变论证失效。$\Sigma$ 上的有效度规是否为常曲率（乃至是否为 FS 度量）是框架的开放问题。定理 4.1(b)(c) 仅在对称点（或对称点的小邻域内以微扰形式）严格成立。

*证明.* (a) $S_3$ 可迁地作用于三个扇区标签。投影结构由公理 3 定义，$S_3$ 对称性迫使 $\Sigma$ 上的诱导结构在 $SO(3)$ 作用下不变。二维 + $SO(3)$ 不变 → 常曲率。(b) 二维时 $J$ 由度规和定向唯一确定，Nijenhuis 张量恒为零。(c) 均匀化定理的直接推论。(d) 见声明。$\square$

**注 4.1b** 与早期版本不同，常曲率在此不是先验假设，而是对称点涌现的性质。这澄清了"常曲率从何而来"的问题——它来自 $S_3$ 对称性，而非外部输入。

### 4.4 辛结构与 Hamilton 流

**定义 4.1**（角度构型空间辛形式）在满足公理 3 的角度构型空间
$$D_\theta = \{(\theta_1,\theta_2,\theta_3)\in\mathbb{R}_{>0}^3 \mid \theta_1+\theta_2+\theta_3=\pi/2\}$$
上，定义标准辛形式 $\omega_\theta = d\theta_1 \wedge d\theta_2$（$\theta_3$ 由约束确定）。

**定理 4.2**（Hamilton 向量场与周期流）设 $H = S_{\text{holo}}$ 为六项作用量（定义 4.3）。在 $(D_\theta, \omega_\theta)$ 上，$H$ 生成 Hamilton 向量场 $X_H$。$H$ 在 $S_3$ 对称点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处取得唯一全局最小值 $24$，该点为 $X_H$ 的唯一平衡点。若 $H=c$ 的规则水平集为紧的简单闭曲线，则 Hamilton 流在其上周期。

**定理 4.3**（离散群等变映射）映射
$$\mu: D_\theta \to \mathbb{R}^3, \quad \mu(\theta_1,\theta_2,\theta_3) = (\sin^2\theta_1, \sin^2\theta_2, \sin^2\theta_3)$$
是 $S_3$ 等变映射，满足 $\mu(g\cdot\theta)=g\cdot\mu(\theta)$。

### 4.5 扇区投影强度角与面积比例

**定义 4.2**（扇区投影强度角）在公理 3 的全息屏 $\Sigma$ 上，定义扇区投影强度角 $\theta_i$（$i=1,2,3$）如下：对扇区 $i$ 的投影 $P_i$，其像域面积元与全息屏面积元 $dA$ 之比为
$$\mathrm{Area}(P_i(U_i)) = \sin^2\theta_i \, dA,$$
双扇区联合投影的面积比例为
$$\mathrm{Area}(P_i\times P_j) = \sin\theta_i\sin\theta_j \, dA.$$

**引理 4.4'**（对称点处的投影存在性）在 $S_3$ 对称点 $\theta_0=(30^\circ,30^\circ,30^\circ)$ 处，存在从各扇区 $\mathbb{R}^3$ 到全息屏切空间 $T\Sigma\cong\mathbb{R}^2$ 的线性斜投影 $P_i$，使其面积元 Jacobian 精确等于 $\sin^2 30^\circ = 1/4$。

*证明.* 取 $P(x,y,z) = (x/2, y/2): \mathbb{R}^3\to\mathbb{R}^2$，Jacobian 为 $1/4$。在对称点三个扇区等价，取相同形式的投影即得。$\square$

**注 4.4'** 引理 4.4' 仅证明对称点处存在满足面积比例的线性斜投影。对非对称点 $\theta\neq\theta_0$（即 $\theta_i$ 不全相等），满足面积比例 $\sin^2\theta_i$ 的投影 $P_i$ 的显式构造及其存在性，需要从 $M(a)$ 的黎曼几何（乘积度量、Hopf 纤维化）中自然导出。本文附录 D 给出了典范 Hopf 投影构造——在对称点该构造严格，在非对称点构造主体已完成，双扇区耦合的精确几何公式是残存的验证任务。对称点的存在性为理论提供了锚点。

### 4.6 六项作用量

**定义 4.3**（六项作用量）在角度构型空间 $D_\theta$ 上，定义六项作用量
$$S_{\text{holo}}(\theta) = \sum_{i=1}^3 \csc^2\theta_i + \sum_{1\leq i<j\leq 3} \csc\theta_i \csc\theta_j.$$
该函数为 $S_3$ 不变函数。上下文明确时简记为 $S(\theta)$。

### 4.7 值域定理与唯一全局最小值

**定理 4.4**（六项作用量的值域与唯一全局最小值）在 $D_\theta$ 上，$S_{\text{holo}}$ 的值域为 $[24, +\infty)$，且在 $D_\theta$ 上具有唯一全局最小值点 $\theta_0=(30^\circ, 30^\circ, 30^\circ)$，$S(\theta_0)=24$。

*证明.* 分四步：

**步骤 1：** 对称点取值。代入 $\theta_0=(30^\circ,30^\circ,30^\circ)$，$\csc 30^\circ=2$，得 $S(\theta_0)=3\cdot 4 + 3\cdot 4 = 24$。

**步骤 2：** 临界点验证。使用拉格朗日乘子法，约束 $g(\theta)=\theta_1+\theta_2+\theta_3-\pi/2=0$。
$$\frac{\partial S}{\partial \theta_i} = -2\csc^2\theta_i\cot\theta_i - \sum_{j\neq i}\csc\theta_i\csc\theta_j\cot\theta_i = -\cot\theta_i\csc\theta_i\left(2\csc\theta_i + \sum_{j\neq i}\csc\theta_j\right).$$
在 $D_\theta$ 内 $\theta_i\in(0,\pi/2)$，$\csc\theta_i>0$，括号内为正。故 $\nabla S \parallel \nabla g$ 当且仅当 $\cot\theta_1=\cot\theta_2=\cot\theta_3$，即 $\theta_1=\theta_2=\theta_3=\pi/6$。因此 $\theta_0$ 是 $D_\theta$ 上唯一的临界点。

**步骤 3：** 对称化递减引理。设 $\theta=(\theta_1,\theta_2,\theta_3)\in D_\theta$ 且存在某对分量不等，不妨设 $\theta_1\neq\theta_2$。定义对称化点 $\theta' = \left(\frac{\theta_1+\theta_2}{2}, \frac{\theta_1+\theta_2}{2}, \theta_3\right)$。令 $u=(\theta_1+\theta_2)/2$，$\delta=(\theta_1-\theta_2)/2\neq 0$。

由 $\csc x$ 在 $(0,\pi/2)$ 上的严格凸性（二阶导 $\csc x(\cot^2 x + \csc^2 x)>0$）及 Jensen 不等式，
$$\csc^2\theta_1 + \csc^2\theta_2 > 2\csc^2 u.$$

对交叉项，考虑 $h(\delta)=\csc(u+\delta)\csc(u-\delta)$。计算
$$h'(\delta) = \csc(u+\delta)\csc(u-\delta)\big[\cot(u-\delta) - \cot(u+\delta)\big].$$
$\cot$ 在 $(0,\pi/2)$ 上严格递减，故 $\cot(u-\delta) - \cot(u+\delta)$ 与 $\delta$ 同号，因此 $h'(\delta)\neq 0$ 对 $\delta\neq 0$，且 $\delta=0$ 为唯一极小值点。故 $\csc\theta_1\csc\theta_2 > \csc^2 u$。同理 $(\csc\theta_1+\csc\theta_2)\csc\theta_3 > 2\csc u \csc\theta_3$。综合以上，$S(\theta) > S(\theta')$。

**步骤 4：** 唯一全局最小值。对任意 $\theta\neq\theta_0$，至少有两个分量不等。通过有限步对称化（每次将不等的一对分量平均化），可构造严格递减序列 $\theta^{(0)}=\theta, \theta^{(1)}, \dots$ 最终趋于对称点 $\theta_0$。由步骤 2，$\theta_0$ 是唯一临界点；由步骤 3 的严格递减性，任何非对称点的 $S$ 值均严格大于 $S(\theta_0)$。结合边界行为（当任一 $\theta_i\to 0^+$ 时 $\csc\theta_i\to+\infty$，故 $S\to+\infty$），$\theta_0$ 是 $D_\theta$ 上的唯一全局最小值点。值域由 $S$ 的连续性和边界发散确定为 $[24,+\infty)$。$\square$

**推论 4.1**（Hessian 正定性）$S$ 在 $\theta_0$ 处的 Hessian 限制在约束切空间 $T_{\theta_0}D_\theta$ 上正定，特征值为 $124$（二重）。

**定理 4.5**（对称轴单值化定理）在 $D_{\text{ord}} = \{\theta\in D_\theta \mid \theta_1 \geq \theta_2 \geq \theta_3\}$ 的对称轴 $\gamma = \{\theta\in D_{\text{ord}} \mid \theta_2=\theta_3\}$ 上，$S|_{\gamma}$ 是严格单调递减的同胚 $\gamma \to [24,+\infty)$。

*证明.* 在 $\gamma$ 上参数化 $\theta_1=\pi/2-2t,\ \theta_2=\theta_3=t$（$t\in(0,\pi/6]$）。则
$$S|_\gamma(t)=\frac{1}{\sin^2(\pi/2-2t)}+\frac{3}{\sin^2 t}+\frac{2}{\sin(\pi/2-2t)\sin t}.$$

计算导数：
$$\begin{aligned}
\frac{d}{dt}S|_\gamma(t) &= \frac{4\cos(\pi/2-2t)}{\sin^3(\pi/2-2t)} - \frac{6\cos t}{\sin^3 t} \\
&\quad - \frac{2\cos(\pi/2-2t)}{\sin^2(\pi/2-2t)\sin t} + \frac{2\cos t}{\sin(\pi/2-2t)\sin^2 t} \\
&= \frac{4\sin 2t}{\cos^3 2t} - \frac{6\cos t}{\sin^3 t} - \frac{2\sin 2t}{\cos^2 2t\sin t} + \frac{2\cos t}{\cos 2t\sin^2 t}.
\end{aligned}$$

对 $t\in(0,\pi/6)$，各项符号分析：
- 第一项为正（$\sin 2t>0$, $\cos 2t>0$），而第三项为负；
- 第二项为负（$\cos t>0$），第四项为正。
- 通分后分子符号可通过在 $t=\pi/6$ 处展开确认：$S|_\gamma(\pi/6)=24$ 为最小值。由于 $S$ 在 $\gamma$ 上仅有一个临界点（对称点），且该点为全局最小值，$S|_\gamma$ 在 $(0,\pi/6)$ 上严格递减。$\frac{d}{dt}S|_\gamma(t)<0$ 对 $t\in(0,\pi/6)$ 通过检查端点行为（$t\to 0^+$ 时 $S\to+\infty$，$t\to\pi/6$ 时 $S\to 24$）及唯一临界点性质严格成立。

由 $S|_\gamma$ 的连续性、严格单调性和边界行为，它为 $\gamma\to[24,+\infty)$ 的同胚。$\square$

### 4.8 切丛平方根分解与幻数序列的几何来源

本节建立幻数序列 $2(k+1)^2$ 的几何基础。核心进展：$\mathcal{O}(1)$ 线丛的来源从构造性假设升级为 $S^3$ 旋量丛的 Borel-Weil 定理。

#### 4.8.1 旋量丛构造（C2 方案）

**定理 4.6.1**（$S^3$ 旋量丛的手征分解）对每个 $S^3(r_i) \cong SU(2) \cong \text{Spin}(3)$，其旋量丛 $S^{(i)}$ 在右 $U(1)$ 等变分解下为：
$$S^{(i)} = S^{(i)}_+ \oplus S^{(i)}_-,$$
其中 $S^{(i)}_+ \cong \pi_i^*\mathcal{O}(1)$（权 $+1$），$S^{(i)}_- \cong \pi_i^*\mathcal{O}(-1)$（权 $-1$），$\pi_i: S^3(r_i) \to \mathbb{CP}^1_i$ 为 Hopf 商投影。因此：
$$(S^{(i)}_-)^* \cong \pi_i^*\mathcal{O}(1), \quad c_1((S^{(i)}_-)^*) = 1.$$

*证明.* $S^3 \cong SU(2)$ 作为 Lie 群，旋量丛 $S = SU(2) \times_{\text{Spin}(3)} \Delta_3$ 全局平凡。取右 $U(1)_R = \{\text{diag}(e^{i\phi}, e^{-i\phi})\}$，旋量表示 $\Delta_3 \cong \mathbb{C}^2$ 分解为权 $+1$ 与 $-1$ 的一维子表示。由 Borel-Weil 定理，齐性线丛 $SU(2) \times_{U(1)_R} \mathbb{C}_{(k)} \cong \pi^*\mathcal{O}(k)$。因此 $S^{(i)}_+ \cong \pi_i^*\mathcal{O}(1)$，$S^{(i)}_- \cong \pi_i^*\mathcal{O}(-1)$。取对偶得 $(S^{(i)}_-)^* \cong \pi_i^*\mathcal{O}(1)$。$\square$

**注 4.6.1** 此为严格定理。$\mathcal{O}(1)$ 不来自投影核（$\ker P_i$ 的陈类为零，见附录 D），而来自 $S^3$ 因子自身的旋量丛结构——这是古典事实"旋量 = 切矢量的平方根"在 Hopf 纤维化下的精确体现：$(S^{(i)}_-)^* \otimes (S^{(i)}_-)^* \cong H^{1,0} \cong \pi_i^*\mathcal{O}(2)$。

#### 4.8.2 全息屏上的线丛识别

**命题 4.6**（扇区线丛与切丛分解）设 $\Sigma \cong \mathbb{CP}^1$（假设 4.1a）。对每个扇区 $i \in \{\mathcal{M},\mathcal{C},\mathcal{I}\}$，定义扇区线丛
$$\Lambda_i := (S^{(i)}_-)^* \quad \text{（在底 } \mathbb{CP}^1_i \text{ 上）}.$$

**(a) 对称点**：在 $S_3$ 对称点 $\theta_0 = (30^\circ,30^\circ,30^\circ)$，三个 $S^3$ 因子等距。通过投影 $P_i$ 诱导的等同 $\mathbb{CP}^1_i \cong \Sigma$（引理 4.4'），三个扇区线丛推前为 $\Sigma$ 上的 $\mathcal{O}(1)$：
$$\Lambda_{\mathcal{M}}, \Lambda_{\mathcal{C}}, \Lambda_{\mathcal{I}} \cong \mathcal{O}(1) \quad \text{（在 } \Sigma \text{ 上，对称点严格成立）}.$$

**(b) 张量积分解**：选取 $\mathcal{M}$ 与 $\mathcal{C}$ 两个扇区，其线丛张量积重构全纯切丛：
$$\Lambda_{\mathcal{M}} \otimes \Lambda_{\mathcal{C}} \cong \mathcal{O}(1) \otimes \mathcal{O}(1) \cong \mathcal{O}(2) \cong T\Sigma.$$
第三个扇区线丛 $\Lambda_{\mathcal{I}} \cong \mathcal{O}(1)$ 归入法丛 $N\Sigma$ 的复化子丛，为法丛 Spin 联络提供 $U(1)$ 分量。

**(c) 非对称点**：$\Lambda_i \cong \pi_i^*\mathcal{O}(1)$ 在各自的 $S^3(r_i)$ 上由定理 4.6.1 严格成立。其在 $\Sigma$ 上的等同需要投影 $P_i$ 诱导的同构 $\mathbb{CP}^1_i \cong \Sigma$——该同构在非对称点的自然唯一构造是附录 D 的研究内容。在对称点，该同构由 $S_3$ 对称性严格确立。命题 4.6 的核心结论（$\Lambda_{\mathcal{M}} \otimes \Lambda_{\mathcal{C}} \cong T\Sigma$）在对称点具有定理地位。

*证明.* (a) 定理 4.6.1 + 对称点 $P_i$ 构造（引理 4.4'）。(b) $\mathcal{O}(1)^{\otimes 2} = \mathcal{O}(2) = T\mathbb{CP}^1$。(c) 见声明。$\square$

**注 4.6（双扇区选择的 $S_3$ 对称性）** 命题 4.6 中选取 $\mathcal{M}$ 与 $\mathcal{C}$ 扇区构造 $T\Sigma$。由 $S_3$ 可迁地作用于扇区标签，任意两个扇区的线丛张量积均同构于 $\mathcal{O}(2)$。双扇区选择等价于在 $S_3$ 的三个二元素子群 $\{\langle(12)\rangle,\langle(23)\rangle,\langle(13)\rangle\}$ 中选取一个——这是离散规范选择，不改变最终的全息屏结构。

**命题 4.7**（幻数定理）在 $\Sigma \cong \mathbb{CP}^1$（假设 4.1a）与命题 4.6 的线丛识别下，两个独立扇区（$\mathcal{M}$ 与 $\mathcal{C}$）的量子化态空间为：
$$\mathcal{H}_k^{(\text{spatial})} := H^0(\Sigma, \Lambda_{\mathcal{M}}^{\otimes k}) \otimes H^0(\Sigma, \Lambda_{\mathcal{C}}^{\otimes k}).$$

由 Riemann–Roch 定理，$\dim H^0(\mathbb{CP}^1, \mathcal{O}(k)) = k+1$，故
$$\dim \mathcal{H}_k^{(\text{spatial})} = (k+1)^2.$$

引入自旋 $1/2$ 自由度（假设），总 Hilbert 空间维数为
$$\dim \mathcal{H}_k^{(\text{total})} = 2(k+1)^2.$$

**幻数序列**：$2, 8, 18, 32, 50, 72, 98$（$k = 0,1,2,3,4,5,6$）。

**关键分层声明**：
- 幻数序列的代数推导**仅依赖 $\Sigma \cong \mathbb{CP}^1$ 的复结构**，不依赖 $\Sigma$ 上的具体 Kähler 度量（FS 或其他）。$\mathbb{CP}^1$ 作为紧致黎曼面的复结构是唯一的，因此该推导在 $\Sigma \cong \mathbb{CP}^1$ 的前提下具有全局严格性。
- 命题 4.7 在对称点具有完整定理地位（由附录 D 的旋量丛构造严格支撑）；在非对称点，$\Lambda_i$ 在 $\Sigma$ 上的等同依赖于投影构造的闭合。

*证明.* 由定理 4.6.1，每个 $S^3$ 因子的旋量丛对偶分量 $(S^{(i)}_-)^*$ 给出 $\mathcal{O}(1)$（$c_1=1$，Borel-Weil 定理）。在对称点，投影 $P_i$ 将 $\mathbb{CP}^1_i$ 等距地等同于 $\Sigma$，从而 $\Lambda_i$ 推前为 $\Sigma$ 上的 $\mathcal{O}(1)$。Riemann-Roch 给出 $\dim H^0(\mathcal{O}(k)) = k+1$。双扇区张量积 + 自旋 $1/2$ 得 $2(k+1)^2$。$\square$
