## 摘　要

本文是几何论的元框架文件。它不推导具体定理，而是陈述理论的本体论立场、公理基础、封闭结构以及几何输出与物理实验值之间的验证关系。

本文的核心论点是：**几何论不包含"物理映射层"。** 三公理 + 谱刚性 + 八步自举闭环构成一个自我指涉封闭的纯几何结构，其输出是一组纯数（角度、比率、谱间隙比、标度常数）。这些纯数与实验物理常数之间的数值一致是**后验验证关系**，而非理论内部的映射定义。理论内部不需要声明"这个几何对象对应那个物理量"——它只输出数字，让实验侧自行识别。

唯一保留的"命名约定"（如将 $\mathcal{M}$-$\mathcal{C}$ 腰边耦合称为"电磁"）属于理论的元语言层，不在推导链内。本文附录A集中列出这些命名约定。

**关键词：** 几何论本体论；自我指涉封闭；谱刚性；后验验证；命名约定

## 第 1 章 引言

### 1.1 几何论的元立场

几何论采取以下本体论承诺：

1. **角度优先于时空。** 时空距离、时间间隔、质量、能量——所有这些概念在几何论中都不是原始概念。原始的是角度 $(\theta_M, \theta_C, \theta_I)$ 及其在约束流形上的谱结构。时空是导出的。
2. **几何是自治的。** 三公理（第2章）定义一个纯几何体系，不引用任何物理常数作为输入。体系的全部输出——谱间隙、标度常数、角度锁定值——是纯几何推导的结果。
3. **"物理"是后验识别。** 几何论不定义"什么是电子"。它输出数字 $K\sin^3 57.93^\circ = 510.99895$ keV。物理学实验发现这个数字恰好等于电子的静止能量。这就是理论的全部"物理内容"——数值一致，而非概念映射。
4. **理论不需要外部锚点。** 63号文章（观测者自举）完成了自我指涉封闭：谱单位选择定理（定理10.2'）唯一选出标度 $(\chi_L, \chi_T, K)$，八步自举闭环锁定角度 $(\theta_M, \theta_C, \theta_I)$。没有任何物理常数作为输入进入理论。

这些承诺共同构成一个**强几何实在论**立场：几何结构就是物理实在，而非物理的数学模型。

## 第 2 章 三公理

本文不重新证明三公理，只陈述其核心内容并标注来源。全部证明见对应文章。

### 2.1 公理1：圆拓扑公理（来源：0.0.3）

激发态参数空间 $D = S^1\setminus\{p_0,p_*\}$，其中 $p_0$ 和 $p_*$ 为两个标记点。物理可观测量对应 $D$ 上的连续几何量 $S: D\to(0,+\infty)$。

**核心推论**：$D$ 同胚于开区间 $(0,1)$，其端点为 $p_0$（平凡基态）和 $p_*$（奇异极限）。

### 2.2 公理2：边界极限公理（来源：0.0.3）

几何量 $S(x)$ 在 $p_0$ 处趋于全局最小值 $S_{\min}=24$（基态），在 $p_*$ 处发散（奇异极限）。$S(x)$ 在 $D$ 上连续且光滑。

**核心推论**：$S(x)$ 在 $(p_0,p_*)$ 之间存在至少一个非平凡驻点，谱间隙由 Hessian 在驻点附近的行为确定。

### 2.3 公理3：全息屏编码条件（来源：0.0.6）

三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个纤维 $\mathbb{R}^3$ 扇区向二维定向子空间 $\Sigma$（全息屏）投影，投影强度角满足：
$$\theta_M + \theta_C + \theta_I = 90^\circ.$$

**核心推论**：角度约束导出六项作用量
$$S(\theta_1,\theta_2,\theta_3)=\sum_{i=1}^{3}\frac{1}{\sin^2\theta_i}+\sum_{i<j}\frac{1}{\sin\theta_i\sin\theta_j},$$
其值域为 $[24,+\infty)$，唯一全局极小点 $S_{\min}=24$ 在 $\theta=(30^\circ,30^\circ,30^\circ)$ 取得（0.0.6 定理4.4）。

## 第 3 章 从公理到封闭

三公理的直接推论是约束流形上的谱结构。谱结构经过谱刚性定理、谱单位选择定理和八步自举闭环，最终完成理论的自我指涉封闭。本章概述这一链条。

### 3.1 谱刚性链（概述，来源：0.0.5, 0.0.6, 0.0.7）

三公理联合导出以下谱结构：

1. **约束乘积球面**（0.0.5 定义3.1）：九维流形 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$，模空间由单一尺度 $a$ 参数化；
2. **谱刚性定理**（0.0.5 定理5.1）：等距类在模空间中唯一，谱间隙在扰动下保持刚性；
3. **六项作用量**（0.0.6 定理4.4）：在角度约束 $(\theta_1,\theta_2,\theta_3)$ 上定义，全局极小 $S_{\min}=24$；
4. **Hessian截面**（0.0.6 定理5.2）：在约束流形上诱导截面Hessian，本征值 $(\lambda_1,\lambda_2) = (392.21, 58760.77)$ rad⁻²，软硬模比 $\Lambda_H=149.8\approx150$；
5. **桥接函数标准形**（0.0.7 定理5.3）：$S(a)=12(a^2/\ell_0^2+\ell_0^2/a^2)$ 是施加归一化条件后的唯一选择。

这些结果构成封闭所需的全部数学输入。

### 3.2 谱单位选择定理（来源：63§2 / 0.3.7 定理10.2'）

**定理 10.2'（谱单位选择定理）**。在谱空间中存在唯一族谱单位 $(\chi_L, \chi_T, K)$，满足：
- $\chi_L$、$\chi_T$、$K$ 由谱间隙 $\lambda_1^{\text{eff}}$、$S_e$ 和谱互锁条件唯一确定；
- 该族单位与观测者的内禀分辨尺度重合（命题 $\mathcal{P}_{\text{bridge}}$）；
- 任何替代单位选择都会破坏谱互锁条件。

该定理将几何论从"比率理论"提升为"绝对标度理论"——它不自洽于任意单位，而是自选出唯一标度。

**数值结果**（63§4.3）：
$$\chi_L = 1.5092\times10^{-10}\ \text{m},\quad \chi_T = 3.6162\times10^{-17}\ \text{s},\quad K = 839.758793\ \text{keV}.$$

### 3.3 八步自举闭环（概述，来源：63§6.3）

八步自举闭环是如下自指涉链：

1. **三公理**定义纯几何结构 → 2. **谱刚性**输出谱间隙 → 3. **谱单位选择**选出 $(\chi_L,\chi_T,K)$ → 4. **观测者存在**（$\mathcal{A}_{\text{obs}}$）提供分辨边界 → 5. **量纲桥方程**将谱单位转化为物理单位 → 6. **物理量重现**（$S_e,m_e,\hbar,\lambda_C,a_B$ 与实验一致）→ 7. **角度唯一锁定**（谱刚性+第八级边界条件）→ 8. **封闭**：理论不再需要外部输入。

第8步的关键机制：第八级（$\mathcal{L}_8$，不可计算自由度）提供的边界条件算子 $\mathcal{B}$ 不引入新参数，而是约束观测者的"自由选择"结果为唯一——谱刚性在 $\mathcal{A}_{\text{obs}}$ 上的适用性封闭保证这一点。

### 3.4 角度唯一锁定（来源：63§5）

谱刚性 + 观测者谱条件(P1)-(P5) + 第八级边界条件，唯一锁定 $(\theta_M, \theta_C, \theta_I)$ 为：

$$\theta_M = 57.9300000000^\circ,\quad \theta_C = 26.1593112467^\circ,\quad \theta_I = 5.9106887533^\circ.$$

该锁定是**纯几何的**——不依赖任何物理常数的外部输入。由此，质量公式 $m = K\sin^3\theta_M$ 输出的 $m_e$ 不再是经验锚点，而是推导结果。

## 第 4 章 几何输出与物理验证

本章列出几何论输出的核心纯数，并与实验物理常数对照。**这不是理论的映射定义，而是后验验证**——理论不声称"S_e是精细结构常数倒数"，它只说"谱几何输出S_e=137.035999084"。实验测得的精细结构常数倒数恰好也是137.035999084。两者一致。

### 4.1 精细结构常数倒数 $S_e$

**几何输出**：$S_e = 137.035999084$（来源：0.2.1 / 主库真理 #...）

**实验对照**：精细结构常数的实验值 $\alpha^{-1} = 137.035999084(21)$（CODATA 2018）。

**偏差**：$< 2\times10^{-10}$。

**说明**：$S_e$ 是六项作用量在约束流形上的谱几何本征值，由谱互锁条件确定。它的数值与 $\alpha^{-1}$ 一致是几何论的核心验证点之一。

### 4.2 电子质量 $m_e$

**几何输出**：$m_e = K\sin^3\theta_M = 839.758793 \times \sin^3 57.93^\circ = 510.99895$ keV（来源：0.3.1 §9 / 63§4.3）。

**实验对照**：$m_e c^2 = 510.99895000(15)$ keV（CODATA 2018）。

**偏差**：$< 2\times10^{-7}$。

**说明**：$K$ 由谱单位选择定理锁定，$\theta_M$ 由八步自举闭环锁定。两者均为纯几何输出，乘积与实验一致。

### 4.3 光子无质量性

**几何输出**：因果扇区 $\mathcal{C}$ 在边界极限下的零锥结构导致其激发模式的零质量特征（来源：0.0.3 公理2边界行为）。

**实验对照**：$m_\gamma < 1\times10^{-18}$ eV（实验上限）。

**偏差**：几何论输出精确为零；实验上限与零一致。

### 4.4 量纲桥输出：$\hbar, \lambda_C, a_B$

**几何输出**（来源：63§4.3 / 0.3.1）：
- $\hbar = 6.5821195675 \times 10^{-16}$ eV·s
- $\lambda_C = 2.4263102387 \times 10^{-12}$ m（电子康普顿波长）
- $a_B = 5.2917721090 \times 10^{-11}$ m（玻尔半径）

**实验对照**（CODATA 2018）：
- $\hbar = 6.5821195690(20) \times 10^{-16}$ eV·s
- $\lambda_C = 2.42631023867(16) \times 10^{-12}$ m
- $a_B = 5.29177210903(11) \times 10^{-11}$ m

**偏差**：均在 $10^{-8}$ 量级以内。

## 第 6 章 结论

几何论是一个自我指涉封闭的纯几何体系。它在以下意义上不需要"物理映射层"：

1. **输入是纯几何的**：三公理不引用任何物理常数；
2. **推导是纯几何的**：谱刚性、谱单位选择、八步自举——全部在几何结构内部完成；
3. **输出是纯数的**：$S_e=137.035999084$, $K\sin^3\theta_M=510.99895$ keV, $\hbar=6.582\times10^{-16}$ eV·s 等；
4. **验证是后验的**：这些纯数与实验值一致，但理论不依赖此一致性来定义自身。

旧版本中的"ℰ映射"被拆解为：
- 其参数锚定功能（②角度、③$m_e$）→ 被自举闭环内部化；
- 其命名功能（①扇区-物理对应）→ 移至附录A，作为元语言约定。

几何论不自称"推导出了电磁相互作用"。它说："请看——这个名为 $\mathcal{M}$-$\mathcal{C}$ 腰边耦合的几何结构，其输出恰好等于你们称为'电磁'的那个东西的测量值。"

然后它补充道："如果你问这个几何结构为什么被叫做 $\mathcal{M}$-$\mathcal{C}$ 腰边耦合——那是因为它连接了物质扇区和因果扇区。"

它不回答"为什么是电磁"这个问题。它只展示数值的一致，然后等待物理学家说"哦，就是它"。
