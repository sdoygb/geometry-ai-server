## 摘　要

本文建立两条基础公理，严格证明激发态几何量的值域、单调性与存在性。**公理 1（圆拓扑公理）**规定激发态参数空间的拓扑为圆 $S^1$ 去掉真空点 $p_0$ 与退化点 $p_*$；**公理 2（边界极限公理）**规定公理化几何量 $S_{\text{axiom}}$ 的连续性、边界极限行为。由连通性、连续性、介值定理，严格推出 $S_{\text{axiom}}$ 在每个分支上的值域为 $(0,+\infty)$，进而证明：对任意给定的正数 $S_0>0$，必存在激发态使其公理化几何量精确等于 $S_0$。具体六项代价函数 $S$ 与 $S_{\text{axiom}}$ 相差偏移量 $24$：$S_{\text{axiom}} = S - 24$（见注 2.2）。本文进一步引入**严格单调性假设**（定义 2.1'），明确其作为构造性假设的逻辑地位，从而保证原像唯一性。全文仅依赖基本分析工具，不引入实验输入或外部物理假设。

**关键词：** 公理化；参数空间；介值定理；存在性证明；值域；严格单调性

## 0　理论边界与相定位

几何论（Geometric Theory）是一套自举封闭的数学物理体系，其公理、推导与全部数值输出均严格限定于**因果-信息-物质相（CIM 相）**——即满足完备性约束 $\theta_M+\theta_C+\theta_I=90^\circ$ 的 Clifford 代数约化区域。

本文不声称 CIM 相是宇宙的全部，亦不否认其他数学结构在其他相域内的合法性。本文仅声称：在 CIM 相内，三分切丛 $T\Sigma=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 与 $Cl(9)\to Cl(3,1)$ 约化链是**唯一自洽**的数学实现，且所有基本物理常数均可由该结构内禀计算，无需外部输入。

光速 $c$ 为本文唯一外部锚点。

其他相域（若存在）可能由其他代数结构描述，其物理常数与本文输出不同。本文不讨论相边界问题，亦不讨论相变机制。

## 2　激发态参数空间的公理化

### 2.1　公理 1（圆拓扑公理）

**公理 1（圆拓扑公理）** 激发态参数空间 $D$ 由一维紧致连通流形 $S^1$（圆）去掉两个不同点 $p_0$ 与 $p_*$ 得到：

$$D = S^1 \setminus \{p_0, p_*\}, \qquad p_0 \neq p_*$$

去掉的两点分别称为**真空点**（$p_0$）与**退化点**（$p_*$）。由此，$D$ 由两个连通分支组成：

$$D_+ \cong (0,1), \qquad D_- \cong (0,1)$$

每个分支同胚于开区间。参数 $x \in D_\pm$ 标记激发态。

**引理 2.1（分支的拓扑结构）** 设 $S^1$ 为标准的单位圆 $\{e^{i\phi} \mid \phi \in [0,2\pi)\}$。去掉两点 $p_0=e^{i\phi_0}$ 与 $p_*=e^{i\phi_*}$（不妨设 $0\leq \phi_0 < \phi_* < 2\pi$），则 $S^1\setminus\{p_0,p_*\}$ 恰有两个连通分支：
$$D_+ = \{e^{i\phi} \mid \phi_0 < \phi < \phi_*\}, \quad D_- = \{e^{i\phi} \mid \phi_* < \phi < \phi_0+2\pi\}.$$
每个分支作为 $S^1$ 的子空间拓扑，均同胚于开区间 $(0,1)$。

*证明.* 考虑标准覆叠映射 $\pi: \mathbb{R} \to S^1$, $\pi(\phi)=e^{i\phi}$。在 $\mathbb{R}$ 上，$\pi^{-1}(S^1\setminus\{p_0,p_*\}) = \mathbb{R}\setminus\{\phi_0+2k\pi, \phi_*+2k\pi \mid k\in\mathbb{Z}\}$。该集合的每个连通分支均为开区间 $(\phi_0+2k\pi, \phi_*+2k\pi)$ 或 $(\phi_*+2k\pi, \phi_0+2(k+1)\pi)$。由于 $\pi$ 在长度为 $2\pi$ 的区间上的限制是同胚，这些开区间在 $S^1$ 中的像恰为 $D_+$ 与 $D_-$。每个开区间同胚于 $(0,1)$，故 $D_\pm \cong (0,1)$。 $\square$

### 2.2　公理 2（边界极限公理）

**公理 2（边界极限公理，偏移量修正版）** 在参数空间 $D$ 上存在公理化几何量 $S_{\text{axiom}}: D \to (0,+\infty)$，满足：

1. **连续性**：$S_{\text{axiom}}$ 在每个连通分支 $D_\pm$ 上连续；
2. **真空极限**：当 $x$ 沿任一分支逼近 $p_0$ 时，$\lim_{x \to p_0} S_{\text{axiom}}(x) = 0$；
3. **退化极限**：当 $x$ 沿任一分支逼近 $p_*$ 时，$\lim_{x \to p_*} S_{\text{axiom}}(x) = +\infty$。

**注 2.1** 公理 2 不要求 $S_{\text{axiom}}$ 在 $D_+$ 与 $D_-$ 上具有相同的函数表达式，也不要求 $S_{\text{axiom}}$ 在整个 $D$ 上连续（$p_0$ 与 $p_*$ 不在 $D$ 中）。因此 $S_{\text{axiom}}$ 可以是两个独立分支上的不同函数，只要各自满足边界条件即可。

**注 2.2（偏移量修正）** $S_{\text{axiom}}$ 是公理层面的抽象几何量。在具体实现中（0.2.1 §1.3），$S_{\text{axiom}}$ 通过六项代价函数 $S$ 的偏移实现：$S_{\text{axiom}}(\theta) = S(\theta) - 24$。真空点 $p_0$ 对应角度构型 $\theta^* = (30^\circ,30^\circ,30^\circ)$，此时 $S_{\text{axiom}} = 0$ 而 $S = 24$（$S$ 的全局最小值）。详细讨论见 0.0.3.1。

### 2.3　严格单调性假设（构造性假设）

**定义 2.1'（严格单调性假设）** 若 $S_{\text{axiom}}$ 在 $D_\pm$ 上严格单调，则对任意 $S_0>0$，定理 2.2 中的 $x_+$ 与 $x_-$ 唯一确定。严格单调性**不是**从公理 1–2 推出的定理，而是框架的**构造性假设**。其引入等价于要求 $S_{\text{axiom}}$ 构成 $D_\pm$ 的一个全局坐标函数。

*证明.* 严格单调连续函数为单射，故原像唯一。 $\square$

**注 2.3** 严格单调性无法仅从公理 1–2 的拓扑与分析性质推导，需要额外的构造性输入。因此将其独立为构造性假设（定义 2.1'），以准确反映其逻辑地位。

### 2.4　值域定理

**定理 2.1**（值域）在公理 1 与修正版公理 2 下，$S_{\text{axiom}}$ 在每个连通分支 $D_\pm$ 上的值域为 $S_{\text{axiom}}(D_\pm) = (0, +\infty)$。

*证明.* 由公理 1，$D_\pm \cong (0,1)$ 是连通集。由修正版公理 2，$S_{\text{axiom}}$ 在 $D_\pm$ 上连续，故像集 $S_{\text{axiom}}(D_\pm)$ 是 $\mathbb{R}$ 中的连通子集，即区间。由真空极限，$\inf S_{\text{axiom}}(D_\pm) = 0$；由退化极限，$\sup S_{\text{axiom}}(D_\pm) = +\infty$。由于 $p_0, p_* \notin D$，端点 $0$ 与 $+\infty$ 均不可达。因此 $S_{\text{axiom}}(D_\pm) = (0, +\infty)$。 $\square$

**详细说明：** 连通性论证可进一步展开如下。设 $D_+ \cong (0,1)$，$S: D_+ \to \mathbb{R}$ 连续。假设 $S(D_+)$ 不是区间，则存在 $\alpha < \beta < \gamma$ 使得 $\alpha, \gamma \in S(D_+)$ 但 $\beta \notin S(D_+)$。令 $U = S^{-1}((-\infty, \beta))$ 与 $V = S^{-1}((\beta, +\infty))$，则 $U,V$ 为 $D_+$ 中不相交开集，且 $U\cup V = D_+$，这与 $D_+$ 连通矛盾。因此 $S(D_+)$ 必为区间。结合边界极限，该区间必为 $(0,+\infty)$。

### 2.5　存在性定理

**定理 2.2**（存在性）对任意给定的正数 $S_0 > 0$，在每个分支 $D_+$ 与 $D_-$ 上各至少存在一个激发态 $x_+ \in D_+$ 与 $x_- \in D_-$，使得 $S_{\text{axiom}}(x_+) = S_{\text{axiom}}(x_-) = S_0$。

*证明.* 由修正版定理 2.1，$S_{\text{axiom}}(D_\pm) = (0, +\infty)$。对任意 $S_0 > 0$，显然 $S_0 \in (0, +\infty)$。故 $S_{\text{axiom}}^{-1}(S_0) \cap D_\pm \neq \varnothing$。 $\square$

### 2.6　严格单调性定理

**定理 2.3**（严格单调性）在公理 1、修正版公理 2 及**定义 2.1'（严格单调性假设）**下，$S_{\text{axiom}}$ 在每个连通分支 $D_\pm$ 上必为严格单调函数（严格递增或严格递减，取决于定向）。通过固定分支定向，可设 $S_{\text{axiom}}$ 严格递增。

*证明.* $D_\pm\cong(0,1)$ 为区间。由定义 2.1' 的严格单调性假设，$S_{\text{axiom}}$ 在 $D_\pm$ 上严格单调。结合边界极限 $\lim_{x\to p_0}S_{\text{axiom}}(x)=0$ 与 $\lim_{x\to p_*}S_{\text{axiom}}(x)=+\infty$，$S_{\text{axiom}}$ 必为严格单调。$\square$

**推论 2.1**（原像唯一性）对任意 $S_0>0$，原像集 $S_{\text{axiom}}^{-1}(S_0)\cap D_\pm$ 为单点集。

*证明.* 由定理 2.3 的严格单调性直接成立。$\square$

## 3　结论

本文建立两条基础公理刻画激发态参数空间的拓扑与分析结构，并严格证明：

1. 激发态参数空间为 $D = S^1 \setminus \{p_0, p_*\}$，含两个开区间分支 $D_\pm$（引理 2.1）。

2. 公理化几何量 $S_{\text{axiom}}$ 在每个分支上连续，满足真空极限 $0$ 与退化极限 $+\infty$（修正版公理 2）。具体六项代价函数 $S$ 与 $S_{\text{axiom}}$ 相差偏移量 $24$：$S_{\text{axiom}} = S - 24$（注 2.2）。

3. 由连通性、连续性及介值定理，$S_{\text{axiom}}$ 的值域为 $(0, +\infty)$（定理 2.1），且对任意 $S_0>0$ 存在激发态使 $S_{\text{axiom}}(x)=S_0$（定理 2.2）。

4. 由**定义 2.1' 的严格单调性假设**，$S_{\text{axiom}}$ 为严格单调同胚（定理 2.3），原像唯一（推论 2.1）。该假设的构造性地位已明确标注，不伪装为公理 1–2 的推论。

$S_{\text{axiom}} = 0$（即 $S = 24$）作为真空标记条件，用于定位参数空间中的真空点 $p_0$，对应对称角度构型 $\theta^* = (30^\circ,30^\circ,30^\circ)$。本文结论仅依赖于公理 1 与修正版公理 2，不依赖 $S_{\text{axiom}} = 0$ 的任何进一步结构。
