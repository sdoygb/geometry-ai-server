## 1. 基础前提与定义

### 1.1 三公理

**公理 1（圆拓扑公理）**　激发态参数空间 D = S¹ \ {p₀, p*} 由两个连通分支组成：
$$D_+ \cong (0,1), \quad D_- \cong (0,1).$$
真空标记 S_total = 0 对应 p₀ 处的极限行为，不参与主要推导。

**公理 2（边界极限公理）**　几何量 S: D_± → (0, +∞) 在每个分支上连续，且满足：
$$\lim_{x \to p_0} S(x) = 0, \quad \lim_{x \to p_*} S(x) = +\infty.$$
该公理由 0.0.7 命题1.1–1.2 严格确立。

**公理 3（全息屏编码条件）**　在九维约束乘积球面 M(a) = S³(a) × S³(a/√Λ) × S³(a/√(Λk₀)) 的三分切丛 TM(a) = ℳ ⊕ 𝒞 ⊕ ℐ 上，每个 ℝ³ 扇区向二维全息屏 Σ 投影，扇区投影强度角 θ_M, θ_C, θ_I ∈ (0°, 90°) 满足完备性约束：
$$\theta_M + \theta_C + \theta_I = 90°.$$
该约束的合理性源于二维全息屏编码的完整性要求（0.0.3 公理3）。

## 3. 局部梯度与 Hessian 矩阵

### 3.1 一阶偏导数

利用 d/dθ csc²θ = −2cosθ/sin³θ，并计入耦合项，对代价函数公式求全偏导并代入裸基准角：

$$\frac{\partial S}{\partial \theta_M} = -\frac{2\cos \theta_M^0}{\sin^3 \theta_M^0} - \frac{\cos\theta_M^0}{\sin^2\theta_M^0\sin\theta_C^0} - \frac{\cos\theta_M^0}{\sin^2\theta_M^0\sin\theta_I^0} = -10.602445\ \mathrm{rad}^{-1},$$

$$\frac{\partial S}{\partial \theta_C} = -\frac{2\cos \theta_C^0}{\sin^3 \theta_C^0} - \frac{\cos\theta_C^0}{\sin^2\theta_C^0\sin\theta_M^0} - \frac{\cos\theta_C^0}{\sin^2\theta_C^0\sin\theta_I^0} = -71.243155\ \mathrm{rad}^{-1},$$

$$\frac{\partial S}{\partial \theta_I} = -\frac{2\cos \theta_I^0}{\sin^3 \theta_I^0} - \frac{\cos\theta_I^0}{\sin^2\theta_I^0\sin\theta_M^0} - \frac{\cos\theta_I^0}{\sin^2\theta_I^0\sin\theta_C^0} = -2145.162236\ \mathrm{rad}^{-1}.$$

∂S/∂θ_I 远大于其他两个，因为信息场角 θ_I⁰ ≈ 5.91° 很小，sin θ_I⁰ ≈ 0.103，所以 1/sin²θ_I⁰ ≈ 94.3 主导了 S 的值，其导数也最大。

### 3.2 几何约束下的局部梯度

由几何约束 θ_M = 90° − θ_C − θ_I，得全导数：
$$\frac{dS}{d\theta_C} = \frac{\partial S}{\partial \theta_C} - \frac{\partial S}{\partial \theta_M} = -71.243155 - (-10.602445) = -60.640710\ \mathrm{rad}^{-1},$$
$$\frac{dS}{d\theta_I} = \frac{\partial S}{\partial \theta_I} - \frac{\partial S}{\partial \theta_M} = -2145.162236 - (-10.602445) = -2134.559791\ \mathrm{rad}^{-1}.$$

$$g_{\mathrm{local}} = \frac{dS}{d\theta_C} - \frac{dS}{d\theta_I} = \frac{\partial S}{\partial \theta_C} - \frac{\partial S}{\partial \theta_I} = 2073.919082\ \mathrm{rad}^{-1}.$$
g_local 表征几何约束下因果场与信息场的梯度竞争。

### 3.3 Hessian 矩阵

利用二阶导公式。对单场项 1/sin²θ_X，其二阶导数为：
$$\frac{d^2}{d\theta_X^2}\left(\frac{1}{\sin^2\theta_X}\right) = \frac{2}{\sin^2\theta_X} + \frac{6\cos^2\theta_X}{\sin^4\theta_X}.$$

对耦合项 1/(sin θ_X sin θ_Y)，其关于 θ_X 的二阶导包含两部分：

（a）对角贡献：
$$\frac{d^2}{d\theta_X^2}\frac{1}{\sin\theta_X \sin \theta_Y} = \frac{1+\cos^2\theta_X}{\sin^3\theta_X \sin \theta_Y}.$$

（b）混合贡献（X ≠ Y）：
$$\frac{d^2}{d\theta_X d\theta_Y}\frac{1}{\sin\theta_X \sin \theta_Y} = \frac{\cos\theta_X \cos\theta_Y}{\sin^2\theta_X \sin^2\theta_Y}.$$

因此 Hessian 对角元 H_XX 必须同时包含单场项和两项耦合项的对角贡献：
$$\begin{aligned}
H_{MM} &= \left[\frac{2}{\sin^2\theta_M} + \frac{6\cos^2\theta_M}{\sin^4\theta_M}\right] + \frac{1+\cos^2\theta_M}{\sin^3\theta_M \sin\theta_C} + \frac{1+\cos^2\theta_M}{\sin^3\theta_M \sin\theta_I}, \\
H_{CC} &= \left[\frac{2}{\sin^2\theta_C} + \frac{6\cos^2\theta_C}{\sin^4\theta_C}\right] + \frac{1+\cos^2\theta_C}{\sin^3\theta_C \sin\theta_M} + \frac{1+\cos^2\theta_C}{\sin^3\theta_C \sin\theta_I}, \\
H_{II} &= \left[\frac{2}{\sin^2\theta_I} + \frac{6\cos^2\theta_I}{\sin^4\theta_I}\right] + \frac{1+\cos^2\theta_I}{\sin^3\theta_I \sin\theta_M} + \frac{1+\cos^2\theta_I}{\sin^3\theta_I \sin\theta_C}.
\end{aligned}$$

代入裸基准角的三角函数值：
$$H_{MM} = 31.3012\ \mathrm{rad}^{-2}, \quad H_{CC} = 367.7347\ \mathrm{rad}^{-2}, \quad H_{II} = 59259.35\ \mathrm{rad}^{-2},$$
$$H_{MC} = 3.4145\ \mathrm{rad}^{-2}, \quad H_{MI} = 69.3547\ \mathrm{rad}^{-2}, \quad H_{CI} = 433.1578\ \mathrm{rad}^{-2}.$$

### 3.4 约束流形上的 Hessian

在约束流形（θ_C、θ_I 独立，θ_M 由几何约束消去）上，Hessian 矩阵元为：
$$\begin{aligned}
H_{CC}^{\mathrm{con}} &= H_{CC} - 2H_{MC} + H_{MM} = 367.7347 - 2\times 3.4145 + 31.3012 = 392.2069\ \mathrm{rad}^{-2}, \\
H_{II}^{\mathrm{con}} &= H_{II} - 2H_{MI} + H_{MM} = 59259.35 - 2\times 69.3547 + 31.3012 = 59151.9418\ \mathrm{rad}^{-2}, \\
H_{CI}^{\mathrm{con}} &= H_{CI} - H_{MC} - H_{MI} + H_{MM} = 433.1578 - 3.4145 - 69.3547 + 31.3012 = 391.6898\ \mathrm{rad}^{-2}.
\end{aligned}$$

约束流形 Hessian 矩阵：
$$H^{\mathrm{con}} = \begin{pmatrix} 392.2069 & 391.6898 \\
 391.6898 & 59151.9418 \end{pmatrix}\quad (\mathrm{rad}^{-2}).$$

特征值求解：
$$\mathrm{Tr} = 59544.1487,\quad \det \approx 23046993,\quad \Delta = \sqrt{\mathrm{Tr}^2 - 4\det} \approx 58764.89.$$

$$\lambda_2^{\mathrm{con}} = \frac{59544.1487 + 58764.89}{2} = 59154.52\ \mathrm{rad}^{-2}\quad (\text{硬模}),$$
$$\lambda_1^{\mathrm{con}} = \frac{59544.1487 - 58764.89}{2} = 389.63\ \mathrm{rad}^{-2}\quad (\text{软模}).$$

> **注**：此处的 (θ_C, θ_I) 坐标系与 §4.5 的 (ξ, η) 坐标系度规不同，故特征值数值与截面 Hessian 谱（392.21, 58760.77）存在坐标变换差异，属预期之内。

曲率不对称：H_II/H_CC = 59259.35/367.73 = 161.2，H_II/H_MM = 59259.35/31.30 = 1893.2。信息场方向的曲率远大于其他方向，这是信息场主导总作用量的几何体现。

## 3.5 辛几何结构与 Hamilton 定理

### 3.5.1 约束相空间与辛形式

**工具层声明**。本章启用的辛几何工具（(𝒫_η, ω)、Hamiltonian 流、Darboux 线性化）由三公理自然诱导：公理3（全息屏编码条件）提供二维辛流形 (Σ, ω)，公理2（边界极限公理）保证 Hamiltonian 的凸性，公理1（圆拓扑公理）提供参数空间的周期结构。全部辛几何结构不引入超越三公理的新数学假设。

**定义 3.5.1（约束相空间）**。设 Σ_η 为几何约束截面（由完备性公理 θ_M + θ_C + θ_I = π/2 定义的2维子流形）。其局部坐标为 (θ_M, θ_C, θ_I)。定义**约束相空间** 𝒫_η ⊂ T*Σ_η 为余切丛中满足完备性约束的 Lagrange 子流形：
$$\mathcal{P}_\eta = \{(q,p)\in T^*\Sigma_\eta : q=(\theta_M,\theta_C,\theta_I),\ \theta_M+\theta_C+\theta_I=\pi/2,\ p_I=-(p_M+p_C)\}.$$

**定义 3.5.2（辛形式）**。在 𝒫_η 上定义2-形式 ω ∈ Ω²(𝒫_η)：
$$\omega = \frac{1}{\sin^2\theta_M}\mathrm{d}\theta_M\wedge\mathrm{d}\theta_C + \frac{1}{\sin^2\theta_C}\mathrm{d}\theta_C\wedge\mathrm{d}\theta_I + \frac{1}{\sin^2\theta_I}\mathrm{d}\theta_I\wedge\mathrm{d}\theta_M.$$

**定理 3.5.1（辛结构）**。(𝒫_η, ω) 为辛流形，即 ω 满足闭性 dω = 0 与非退化性。

*证明*。在 𝒫_η 的局部坐标 (θ_M, θ_C) 下，θ_I = π/2 − θ_M − θ_C。代入得：
$$\omega = \left(\frac{1}{\sin^2\theta_M}+\frac{1}{\sin^2\theta_C}+\frac{1}{\sin^2\theta_I}\right)\mathrm{d}\theta_M\wedge\mathrm{d}\theta_C \equiv A(\theta_M,\theta_C)\,\mathrm{d}\theta_M\wedge\mathrm{d}\theta_C.$$
闭性：dω = dA ∧ dθ_M ∧ dθ_C = 0（由 dθ_M ∧ dθ_M = 0）。非退化性：在2维流形上，ω ≠ 0 等价于 A ≠ 0。在几何约束截面的有效定义域 θ_i ∈ (0, π/2) 内，A > 0 处处成立。 □

**推论 3.5.2（体积形式）**。ω 给出典范体积形式 Ω = ω²/2! = A(θ_M, θ_C) dθ_M ∧ dθ_C ∧ dp_M ∧ dp_C。

### 3.5.2 六项代价函数作为 Hamiltonian

**定义 3.5.3（几何 Hamiltonian）**。定义 H: 𝒫_η → ℝ：
$$H(\theta_M,\theta_C,\theta_I) = S(\theta) - S_3,$$
其中 S(θ) 为六项代价函数，S₃ = S(π/6, π/6, π/6) = S_min = 24 为 30° 不动点基准值。

**定理 3.5.4（Hamilton 流频率）**。在辛流形 (𝒫_η, ω) 上，Hamiltonian H 生成的 Hamilton 流在不动点 (θ_M⁰, θ_C⁰, θ_I⁰) 附近的线性化频率为 √(det H^{(MC)})/A₀，其中 H^{(MC)} 为 (θ_M, θ_C) 子空间的 Hessian 子矩阵

$$H^{(MC)} = \begin{pmatrix} H_{MM} & H_{MC} \\ H_{CM} & H_{CC} \end{pmatrix}.$$

*证明*。在 (θ_M, θ_C) 坐标下，Poisson 括号为：
$$\{f,g\}_\omega = \frac{1}{A}\left(\frac{\partial f}{\partial\theta_M}\frac{\partial g}{\partial\theta_C}-\frac{\partial f}{\partial\theta_C}\frac{\partial g}{\partial\theta_M}\right).$$
Hamilton 流方程：
$$\dot{\theta}_M = \frac{1}{A}\frac{\partial H}{\partial\theta_C}, \quad \dot{\theta}_C = -\frac{1}{A}\frac{\partial H}{\partial\theta_M}.$$
在不动点处线性化，令 δθ_i = θ_i − θ_i⁰。线性化系统：
$$\frac{\mathrm{d}}{\mathrm{d}\tau}\begin{pmatrix}\delta\theta_M\\ \delta\theta_C\end{pmatrix} = \frac{1}{A_0}\begin{pmatrix} 0 & 1 \\
 -1 & 0 \end{pmatrix}\begin{pmatrix} H_{MM} & H_{MC} \\
 H_{CM} & H_{CC} \end{pmatrix}\begin{pmatrix}\delta\theta_M\\ \delta\theta_C\end{pmatrix}.$$
特征方程 μ² = −det(H^{(MC)})/A₀²。因此线性化频率为 √(det H^{(MC)})/A₀。 □

**结构注释 3.5.5**。定理3.5.4证明 Hamilton 流的线性化频率由 (θ_M, θ_C) 子矩阵的行列式决定。后续量子化中，硬模与软模方向近似解耦，可分别定义独立激发频率（见定理3.5.10）。

### 3.5.3 完备性公理的 Lagrange 子流形解释

**命题 3.5.6（Lagrange 子流形）**。完备性公理 θ_M + θ_C + θ_I = π/2 定义了辛流形 (𝒫_η, ω) 上的 Lagrange 子流形 ℒ_η ⊂ 𝒫_η，即 dim ℒ_η = (1/2) dim 𝒫_η 且 ω|_{ℒ_η} = 0。

*证明*。在 𝒫_η 的局部坐标 (θ_M, θ_C) 下，θ_I = π/2 − θ_M − θ_C。辛形式 ω = A(θ_M, θ_C) dθ_M ∧ dθ_C 已是2维体积形式。约束流形由 θ_I 的消去定义，其切空间由 ∂_{θ_M} − ∂_{θ_I} 与 ∂_{θ_C} − ∂_{θ_I} 张成。直接计算得 ω 在该切空间上的限制为零，故为 Lagrange 子流形。 □

**推论 3.5.7（维数锁定）**。Lagrange 子流形 ℒ_η 维数为2，与 0.0.7 定理中 D_f = 2.000 的 Hausdorff 维数一致。

**九素互扼关联**。完备性公理 θ_M + θ_C + θ_I = π/2 定义 Lagrange 子流形 ℒ_η，此即 0.0.7 §6.2 九素互扼环节 (A) 的外尔房边界锁死在截面层级的实现。辛结构将九维流形的 9 个几何素压缩至二维全息屏的 3 个宏角度素，剩余 6 个素（曲率+渗透）通过 Hessian 软硬模比与 Bott 周期截断锁定。

### 3.5.4 局部谐振子定理

**定理 3.5.10（局部谐振子定理）**。在不动点 (θ_M⁰, θ_C⁰, θ_I⁰) 的 Darboux 邻域内，Hamiltonian H = S(θ) − S₃ 的二次展开在 (ξ, η) 坐标下近似解耦：
$$\hat{H} \approx -\frac{\hbar^2}{2}\Delta_{\Sigma_\eta} + \frac{1}{2}\lambda_1 \eta^2 + \frac{1}{2}\lambda_2 \xi^2,$$
其中 Δ_{Σ_η} 为 Laplace-Beltrami 算子。在硬模方向 ξ 与软模方向 η 近似独立的极限下，得到两个解耦的谐振子方程，其频率分别为 √λ₁（软模）与 √λ₂（硬模）（几何单位 ℏ = 1）。

*证明*。由 Darboux 定理，不动点邻域内辛形式 ω = dq ∧ dp。坐标函数映为算子 q̂ = q，动量函数映为 p̂ = −iℏ∂_q。Hamiltonian H 在不动点处的 Taylor 展开：
$$H(\theta) = H(\theta^0) + \frac{1}{2}\sum_{i,j} H_{ij}\delta\theta_i\delta\theta_j + O(\delta\theta^3).$$
由 §4.5 的解耦定理，ℐ-扇区与 ℳ-𝒞 子空间独立。令 ℏ = 1（几何单位），即得谐振子方程。该推导仅依赖不动点邻域的局部辛结构。 □

**注 3.5.11（谐振子方程的地位）**。局部谐振子方程（定理3.5.10）是**几何约束截面在不动点附近的二次量子化模型**，其合法性由局部辛结构的线性化保证。

## 4. 几何约束截面上的二阶响应与双重投影

### 4.1 几何约束截面坐标系

引入截面坐标：
$$\xi = \theta_I - \theta_I^0, \quad \eta = (\theta_I + \theta_C) - (\theta_I^0 + \theta_C^0).$$

反解原始角度（关键步骤，不跳步）：
$$\theta_I = \theta_I^0 + \xi, \quad \theta_C = \theta_C^0 + \eta - \xi, \quad \theta_M = \theta_M^0 - \eta.$$

验证几何约束：
$$(\theta_M^0-\eta) + (\theta_C^0+\eta-\xi) + (\theta_I^0+\xi) = \theta_M^0+\theta_C^0+\theta_I^0 = 90°.$$

### 4.2 雅可比矩阵

由上述反解公式，各角度对 (ξ, η) 的偏导为：
$$\frac{\partial \theta_M}{\partial \xi} = 0, \quad \frac{\partial \theta_M}{\partial \eta} = -1, \quad \frac{\partial \theta_C}{\partial \xi} = -1, \quad \frac{\partial \theta_C}{\partial \eta} = 1, \quad \frac{\partial \theta_I}{\partial \xi} = 1, \quad \frac{\partial \theta_I}{\partial \eta} = 0.$$

### 4.3 一阶偏导数的坐标变换

由链式法则 ∂S/∂ξ = Σ_i (∂S/∂θ_i)(∂θ_i/∂ξ)：
$$\frac{\partial S}{\partial \xi} = (-2145.162236)(1) + (-71.243155)(-1) + (-10.602445)(0) = -2145.162236 + 71.243155 = -2073.919082\ \mathrm{rad}^{-1}.$$

由链式法则 ∂S/∂η = Σ_i (∂S/∂θ_i)(∂θ_i/∂η)：
$$\frac{\partial S}{\partial \eta} = (\frac{\partial S}{\partial \theta_M})(-1) + (\frac{\partial S}{\partial \theta_C})(1) + (\frac{\partial S}{\partial \theta_I})(0) = (-10.602445)(-1) + (-71.243155)(1) = 10.602445 - 71.243155 = -60.640710\ \mathrm{rad}^{-1}.$$

截面梯度向量：
$$\nabla S = \left(\frac{\partial S}{\partial \xi}, \frac{\partial S}{\partial \eta}\right) = (-2073.919082, -60.640710)\ \mathrm{rad}^{-1}.$$

重要发现：∂S/∂η|₀ = −60.64 rad⁻¹ ≠ 0。这意味着裸基准点不是 η 方向的极值点。然而电子基态的实际路径沿 η = 0（即 θ_M = θ_M⁰ 锁定）进行，由一阶梯度 ∂S/∂ξ 主导能量变化（见 §6）。

### 4.4 相互作用方向 v̂

v̂ 定义为与 ∇S 正交的单位向量（∇S · v̂ = 0）：
$$\hat{v}_\xi = 0.029227, \quad \hat{v}_\eta = -0.999573.$$

即 v̂ = (0.029227, −0.999573)。v̂ 几乎沿 η 方向（−0.9996），因为 η 方向梯度远小于 ξ 方向梯度，所以正交方向自然靠近 η 轴。

### 4.5 (ξ, η) 坐标系下的截面 Hessian 矩阵

由链式法则（θ_i 对 (ξ, η) 为线性变换，二阶导 ∂²θ_i/∂ξ² 等均为零）：
$$\begin{aligned}
H_{\xi\xi} &= H_{CC} + H_{II} - 2H_{CI} = 367.7347 + 59259.35 - 2\times 433.1578 = 58760.77\ \mathrm{rad}^{-2}, \\
H_{\eta\eta} &= H_{MM} + H_{CC} - 2H_{MC} = 31.3012 + 367.7347 - 2\times 3.4145 = 392.21\ \mathrm{rad}^{-2}, \\
H_{\xi\eta} &= -H_{CC} + H_{MC} - H_{MI} + H_{CI} = -367.7347 + 3.4145 - 69.3547 + 433.1578 = -0.52\ \mathrm{rad}^{-2}.
\end{aligned}$$

截面 Hessian 矩阵：
$$H = \begin{pmatrix} 58760.77 & -0.52 \\
 -0.52 & 392.21 \end{pmatrix}\quad (\mathrm{rad}^{-2}).$$

特征值：软模 λ₁ = 392.21 rad⁻²（η 方向），硬模 λ₂ = 58760.77 rad⁻²（ξ 方向）。特征值比：
$$\frac{\lambda_2}{\lambda_1} = \frac{58760.77}{392.21} = 149.8 \approx 150.$$

### 4.6 投影得到二阶响应系数

沿相互作用方向 v̂，将总作用量 S 在裸基准构型附近展开：
$$S(\text{沿 } \hat{v} \text{ 微小位移 } \varepsilon) = 137 + \frac{1}{2} S_{\hat{v}} \cdot \varepsilon^2 + O(\varepsilon^3).$$

一阶项因 ∇S · v̂ = 0 而消失。这就是选择相互作用方向的原因——让一阶效应哑火，让二阶效应主导。

二阶导数 S_v̂ 由截面 Hessian 矩阵向 v̂ 方向投影给出：
$$S_{\hat{v}} = \hat{v}^T \cdot H \cdot \hat{v} = 442.10\ \mathrm{rad}^{-2}.$$

单位换算：
$$442.10\ \mathrm{rad}^{-2} = \frac{442.10}{(180/\pi)^2}\ \mathrm{deg}^{-2} = \frac{442.10}{3282.8} = 0.1347\ \mathrm{deg}^{-2}.$$

该常数由裸基准角的三角函数经一阶偏导、坐标变换、梯度正交方向、Hessian 计算和方向投影五个步骤导出，不含任何自由参数。它描述等势方向 v̂ 上的二阶曲率响应，用于精细结构能级分裂（见 §5）。α 小数位的生成则沿 §6 所述的 ξ 方向（一阶梯度主导）进行。

### 4.7 平方律关系

$$\Delta S = \frac{1}{2} \cdot S_{\hat{v}} \cdot \varepsilon^2,$$

其中 ε 为沿 v̂ 方向的微小位移（单位：rad）。该平方律是层级递推的几何基础。

## 5. 层级几何的双重投影

在等势方向 v̂ 上，第 N 层级构型分裂为一对对称构型：

- 质心偏移 ε_N —— 构型对整体沿 v̂ 方向的净位移；
- 分裂半间距 δ_N —— 两个构型关于质心的对称张裂幅度。

构型位置：沿 v̂ 方向位移 +δ_N + ε_N 和 −δ_N + ε_N。

双重投影的物理对应：

- 质心偏移 ε_N → α⁻¹ 的该位小数 ΔS_N：
  $$\Delta S_N = \frac{1}{2} S_{\hat{v}} \varepsilon_N^2.$$
- 分裂半间距 δ_N → 精细结构能级分裂 ΔE_fine：
  $$S_A - S_B = 2 S_{\hat{v}} \delta_N \varepsilon_N.$$

当 ε_N = 0（纯粹对称张裂）时，构型对能量简并。

两个构型的作用量：
$$S_A = S_{N-1} + \frac{1}{2} S_{\hat{v}} (\delta_N+\varepsilon_N)^2,$$
$$S_B = S_{N-1} + \frac{1}{2} S_{\hat{v}} (-\delta_N+\varepsilon_N)^2.$$

质心作用量：
$$S_{\mathrm{质心}} = \frac{S_A+S_B}{2} = S_{N-1} + \frac{1}{2} S_{\hat{v}} (\delta_N^2+\varepsilon_N^2).$$

能级分裂：
$$S_A - S_B = 2 S_{\hat{v}} \delta_N \varepsilon_N.$$

在几何约束截面上，第 N 层级的几何变化包含两种独立模式：质心偏移 ε_n（贡献 α 小数）和对称分裂 δ_n（贡献精细结构分裂）。二者共享 S_v̂ = 442.10 rad⁻²，α 的小数序列与精细结构分裂是同一组底层几何运动的两个不同投影。

## 8. 非对称扭曲碗的定性图像

基于前七章的数学结构，几何约束截面 Σ 在裸基准点附近的二阶展开（平直近似）取得了成功。然而，从更全局的几何视角审视，约束截面并非严格的欧氏平面，而是一个**非对称扭曲碗**：

- **碗底平坦区**：30° 背景点 (θ_M = θ_C = θ_I = 30°) 附近，Hessian 软硬模差异极大（Λ_H ≈ 150），但 S 取全局最小值 S_min = 24。该点对应背景信息场的均匀态，是碗底。
- **坡腰区**：裸基准点 (θ_M⁰ = 57.93°, θ_C⁰ = 26.16°, θ_I⁰ = 5.91°) 位于坡腰区，S ≈ 137。远离背景点时，软硬模的强各向异性导致截面曲率逐渐显现，二阶 Taylor 展开的余项不可忽略。
- **悬崖边缘**：在信息扇区 θ_I → 0 的极限附近，sin²θ_I 的奇异性导致截面出现陡峭的梯度奇点，对应中微子质量等悬崖效应。

该定性图像的数学来源是约束流形上诱导度规的非均匀性：Hessian 本征值 λ₁, λ₂ 在裸基准点处为常数，但其高阶导数（三阶及以上）在全局上非零，导致截面本质上是一个弯曲流形。平直近似仅在裸基准点的足够小邻域内有效。

碗底平坦区对应 S_min = 24（30° 背景），坡腰区对应电磁耦合常数 S_e ≈ 137 的激发态，悬崖边缘对应 θ_I → 0 的信息场奇点。该图像的全局结构由互锁常数 Λ = 3, k₀ = 2 与 Bott 周期截断 N_eff = 7 共同锁定。
