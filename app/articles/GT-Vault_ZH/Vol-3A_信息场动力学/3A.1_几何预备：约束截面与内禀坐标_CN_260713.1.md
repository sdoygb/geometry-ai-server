# 第1章　几何预备：约束截面与内禀坐标

## §1.1　从三扇区到两维约束截面

**核心问题**：Vol-0 建立了三扇区整体框架（物质扇区角 $\theta_M$、因果扇区角 $\theta_C$、信息扇区角 $\xi$），Vol-1 建立了乘积球面 $S^3_\eta \times S^3_\chi$ 上的切丛三分割。信息场动力学的问题域是：**当三个扇区角的和固定为 $\pi/2$ 时，有效自由度从三维退化到二维，这个二维约束截面 $\Sigma$ 的几何性质决定了信息场的全部动力学**。

### 1.1.1　三扇区角的完整约束

**定义（三扇区角，引用 GT-0.3.0.1~0.3.0.3）**：几何论的基础框架定义三个扇区角：

$$\boxed{\theta_M,\ \theta_C,\ \xi \in (0,\pi/2)}$$

分别对应物质（Matter）、因果（Causality）和信息（Information）三个基本扇区。三者的**完满约束**来自乘积球面的纤维化结构（GT-1.5.0.1）：

$$\boxed{\theta_M + \theta_C + \xi = \pi/2}$$

**几何物理解释**：三个扇区角之和恒等于 $\pi/2$ 这一事实，源自 Vol-1 中乘积球面 $S^3_\eta \times S^3_\chi$ 上三分切丛的全息屏编码条件（GT-1.2.0.7）。它与量子力学中的互补原理有深刻类比——三个扇区无法同时完全确定，总和固定意味着每个扇区的精确值都以其他两个扇区的不确定为代价。

### 1.1.2　约束截面 $\Sigma$ 的定义

**定义（约束截面 $\Sigma$，GT-3A.1.1.1）**：定义三维参数空间 $(\theta_M,\theta_C,\xi) \in (0,\pi/2)^3$ 中的二维子流形

$$\boxed{\Sigma = \{\,(\theta_M,\theta_C,\xi) \in (0,\pi/2)^3 \mid \theta_M + \theta_C + \xi = \pi/2 \,\}}$$

称为**约束截面**（Constrained Section）。

**命题 1.1.1（$\Sigma$ 的拓扑，GT-3A.1.1.2）**：约束截面 $\Sigma$ 同胚于一个开二维单纯形（即开三角形）：

$$\Sigma \cong \Delta^\circ = \{\, (x,y) \in \mathbb{R}^2_{>0} \mid x + y < \pi/2 \,\}$$

**证明**：取坐标变换：令 $x = \theta_M$, $y = \theta_C$，则 $\xi = \pi/2 - x - y$。$x>0$, $y>0$, $x+y<\pi/2$ 给出了 $\mathbb{R}^2$ 中的一个开三角形区域。映射 $(\theta_M,\theta_C) \mapsto (\theta_M,\theta_C)$ 是双连续的双射，故 $\Sigma \cong \Delta^\circ$。证毕。□

$\Sigma$ 的闭包 $\overline{\Sigma}$ 是一个闭三角形，边界由以下三条线段构成：

1. **M-C 边界**：$\xi = 0$（信息扇区退化），即 $\theta_M + \theta_C = \pi/2$
2. **M-I 边界**：$\theta_C = 0$（因果扇区退化），即 $\theta_M + \xi = \pi/2$
3. **C-I 边界**：$\theta_M = 0$（物质扇区退化），即 $\theta_C + \xi = \pi/2$

在边界上，退化扇区角的 $\csc$ 函数发散（因为当 $\theta \to 0^+$ 时 $\csc\theta \to +\infty$），导致势函数 $V$ 在边界处发散到无穷（见第2章）。这一性质确保了物理实现点位于 $\Sigma$ 的内部而非边界。

### 1.1.3　物理实现点 $P_e$

**定义（物理实现点 $P_e$，GT-3A.1.1.3）**：$\Sigma$ 上存在一个唯一的特殊点

$$P_e = (\theta_M^e,\ \theta_C^e,\ \xi^e) \in \Sigma$$

称为**物理实现点**（Physical Realization Point），其坐标由 Vol-2 量纲桥的自洽重建确定（GT-2.4.0.6~2.4.0.8）：

$$\boxed{\theta_M^e = 60.0^\circ,\quad \theta_C^e = 24.09^\circ,\quad \xi^e = 5.91^\circ}$$

验证完满约束：$60.0^\circ + 24.09^\circ + 5.91^\circ = 90.0^\circ = \pi/2$。□

$P_e$ 的物理意义：它是量纲桥在标度重建过程中锁定的唯一自洽点——在该点处，从 Wodzicki 留数、热核系数和 Dixmier 迹三条独立路径重建的物理标度全部一致（GT-2.5.0.1~2.5.0.4）。$\xi^e = 5.91^\circ$ 远小于 $\theta_M^e$ 和 $\theta_C^e$，这是信息扇区"柔软性"的第一个信号。

## §1.2　内禀坐标 $(\xi,\eta)$

### 1.2.1　从三角坐标到正交坐标

在 Vol-1 的乘积球面纤维化中（GT-1.5.0.2），约束截面 $\Sigma$ 自然地与 $S^3_\eta$ 的叶空间相关联。但在动力学分析中，直接用 $(\theta_M,\theta_C)$ 对角坐标在表述上并不方便，因为势函数的 Hessian 在 $\theta_M$ 和 $\theta_C$ 方向高度耦合。更好的选择是引入一组正交内禀坐标。

**定义（内禀坐标 $(\xi,\eta)$，GT-3A.1.2.1）**：在约束截面 $\Sigma$ 上定义一组正交坐标为

$$\boxed{\xi = \xi}$$

$$\boxed{\eta = \frac{\theta_M - \theta_C}{\sqrt{2}}}$$

其中 $\theta_M$ 和 $\theta_C$ 是物质和因果扇区角。坐标变换的 Jacobian 为：

$$J = \frac{\partial(\xi,\eta)}{\partial(\theta_M,\theta_C)} = \begin{pmatrix} 0 & 0 & 1 \\ 1/\sqrt{2} & -1/\sqrt{2} & 0 \end{pmatrix}$$

实际上，三个原始坐标 $(\theta_M,\theta_C,\xi)$ 通过完满约束 $\theta_M + \theta_C + \xi = \pi/2$ 退化为两个独立自由度。$(\xi,\eta)$ 坐标选取 $\xi$ 自身作为一个坐标轴，另一个坐标轴 $\eta$ 沿 $\theta_M - \theta_C$ 方向，与 $\xi$ 方向正交。□

**$\eta$ 坐标的物理意义**：$\eta$ 度量的是**物质与因果之间的不对称程度**。当 $\theta_M = \theta_C$ 时 $\eta = 0$，此时物质和因果完全对称。物理实现点处：

$$\eta^e = \frac{60.0^\circ - 24.09^\circ}{\sqrt{2}} = \frac{35.91^\circ}{\sqrt{2}} \approx 25.39^\circ$$

$\eta^e \neq 0$ 反映了自然界中物质-因果对称性的破缺——这正是自发对称性破缺的几何论等价。

### 1.2.2　$\Sigma$ 在 $(\xi,\eta)$ 坐标中的表示

在 $(\xi,\eta)$ 坐标中，约束截面 $\Sigma$ 的表达式为：

$$\theta_M = \frac{1}{2}\left(\frac{\pi}{2} - \xi + \sqrt{2}\,\eta\right)$$

$$\theta_C = \frac{1}{2}\left(\frac{\pi}{2} - \xi - \sqrt{2}\,\eta\right)$$

$$\xi = \xi$$

**$\Sigma$ 的区域条件**：三个扇区角都必须在 $(0,\pi/2)$ 内，由此导出 $(\xi,\eta)$ 的允许范围：

1. $\xi > 0$：信息扇区不退化
2. $\frac12(\pi/2 - \xi + \sqrt{2}\,\eta) > 0$ 且 $< \pi/2$：$\theta_M$ 不退化
3. $\frac12(\pi/2 - \xi - \sqrt{2}\,\eta) > 0$ 且 $< \pi/2$：$\theta_C$ 不退化

这些条件共同定义了 $(\xi,\eta)$ 平面上的一个凸六边形区域（开集），物理实现点 $P_e$ 位于其内部：

$$\xi^e = 5.91^\circ,\quad \eta^e = 25.39^\circ$$

### 1.2.3　坐标变换的动机

选择 $(\xi,\eta)$ 坐标而不直接用 $(\theta_M,\theta_C)$ 的原因有三：

1. **物理分离**：$\xi$（信息扇区）和 $\eta$（物质-因果差异）对应于 Vol-1 全息屏三分割中性质不同的两个方向。信息扇区的涨落直接对应信息场的动力学，而 $\eta$ 方向对应物质-因果相互作用的平衡。

2. **Hessian 近似对角**：在物理实现点 $P_e$ 处，势函数的 Hessian 矩阵在 $(\xi,\eta)$ 坐标中**近似对角**（见第3章），交叉项 $V_{\xi\eta}$ 远小于对角项 $V_{\xi\xi}$ 和 $V_{\eta\eta}$。这使得分析可以自然地分解为"软模（$\xi$ 方向）"和"硬模（$\eta$ 方向）"。

3. **与 Vol-1 几何结构的对应**：$\xi$ 方向直接对应叶空间叶的"横向"涨落，$\eta$ 方向对应叶空间内的"纵向"标度。这一几何对应为后续的全息屏编码提供了自然的数学框架。

## §1.3　约束截面的度量与测地线

### 1.3.1　诱导度量

约束截面 $\Sigma$ 作为 $\mathbb{R}^3$ 的子流形，继承了 $\mathbb{R}^3$ 的欧几里得度量。在 $(\xi,\eta)$ 坐标中，诱导度量为：

$$ds^2_\Sigma = d\theta_M^2 + d\theta_C^2 + d\xi^2$$

由 $\theta_M = (\pi/2 - \xi + \sqrt{2}\eta)/2$，$\theta_C = (\pi/2 - \xi - \sqrt{2}\eta)/2$，计算微分：

$$d\theta_M = -\frac{1}{2}d\xi + \frac{1}{\sqrt{2}}d\eta$$

$$d\theta_C = -\frac{1}{2}d\xi - \frac{1}{\sqrt{2}}d\eta$$

代入得到：

$$ds^2_\Sigma = \left(-\frac{1}{2}d\xi + \frac{1}{\sqrt{2}}d\eta\right)^2 + \left(-\frac{1}{2}d\xi - \frac{1}{\sqrt{2}}d\eta\right)^2 + d\xi^2$$

$$= \frac14 d\xi^2 - \frac{1}{\sqrt{2}}d\xi d\eta + \frac12 d\eta^2 + \frac14 d\xi^2 + \frac{1}{\sqrt{2}}d\xi d\eta + \frac12 d\eta^2 + d\xi^2$$

$$= \left(\frac14 + \frac14 + 1\right)d\xi^2 + \left(\frac12 + \frac12\right)d\eta^2 + \left(-\frac{1}{\sqrt{2}} + \frac{1}{\sqrt{2}}\right)d\xi d\eta$$

$$= \frac32 d\xi^2 + d\eta^2$$

**命题 1.3.1（$\Sigma$ 的度量形式，GT-3A.1.3.1）**：约束截面 $\Sigma$ 在 $(\xi,\eta)$ 坐标中的诱导度量是平坦的：

$$\boxed{ds^2_\Sigma = \frac32 d\xi^2 + d\eta^2}$$

该度量平坦（曲率 $K \equiv 0$），坐标 $(\xi,\eta)$ 并非正交归一——$\xi$ 方向的度量因子为 $3/2$，$\eta$ 方向为 $1$。物理上，这意味着 $\xi$ 方向的几何距离是坐标距离的 $\sqrt{3/2} \approx 1.225$ 倍。□

### 1.3.2　测地线与 $\Sigma$ 上的最短路径

由于 $\Sigma$ 的诱导度量是平坦的，其测地线即为 $\mathbb{R}^2$ 中的直线。在 $(\xi,\eta)$ 坐标中，测地线方程为：

$$\frac{d^2\xi}{dt^2} = 0,\quad \frac{d^2\eta}{dt^2} = 0$$

即 $\xi(t) = \xi_0 + v_\xi t$，$\eta(t) = \eta_0 + v_\eta t$，其中 $t$ 是仿射参数。

**意义**：信息场在 $\Sigma$ 上的扩散路径——在没有势函数的阻碍时——是 $(\xi,\eta)$ 平面中的直线。势函数 $V$ 的存在将"弯曲"这些直线，产生等效的"力"。

### 1.3.3　面积元与积分测度

在 $(\xi,\eta)$ 坐标中，$\Sigma$ 的面积极元为：

$$dA_\Sigma = \sqrt{\det g}\,d\xi d\eta = \sqrt{\frac32 \cdot 1}\,d\xi d\eta = \sqrt{\frac32}\,d\xi d\eta$$

即 $\Sigma$ 上的函数积分需要乘以 $\sqrt{3/2}$ 的面密度因子。但对于信息场动力学中的**路径积分度量**，真正起作用的不是诱导度量，而是由势函数 Hessian 定义的**谱度量**（见第3章）。
