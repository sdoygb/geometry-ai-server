# 第4章 复化映射与薛定谔方程层展

**摘要：** 本章建立从实信息密度场到复值波函数的复化映射，证明在全息屏辛结构 $J^2 = -I$ 下，扩散方程自然层展为薛定谔方程。虚数单位 $i$ 不是物理假设，而是约束截面 $\Sigma$ 的辛几何产物。

---

## 4.1 从实密度场到复值波函数

### 4.1.1 实信息密度场的局限性

第2章构造的信息密度场 $\Phi: \Sigma \to \mathbb{R}_{\ge 0}$ 是实标量场，其动力学由扩散方程

$$\partial_t \Phi = D \Delta_\Sigma \Phi - V'(\Phi)$$

描述，其中 $D$ 为信息扩散系数，$V'$ 为势函数 $V$ 对 $\Phi$ 的导数。该方程包含时间的一阶导数，描述信息在约束截面 $\Sigma$ 上的浓度演化。

然而，实场扩散方程存在两个本质局限：

1. **缺乏干涉结构：** 实场不能产生干涉图案，无法描述量子观测中的相位效应；
2. **时间可逆性缺失：** 扩散方程是时间不可逆的（一阶时间导数），与量子力学的酉演化（一阶时间导数但有 $i$ 因子）结构不同。

这两个局限的克服要求将 $\Phi$ 提升为复值场。

### 4.1.2 辛结构 $J$ 与复化条件

约束截面 $\Sigma$ 携带非退化辛形式 $\omega$，由此定义殆复结构 $J: T\Sigma \to T\Sigma$ 满足：

$$J^2 = -I, \quad \omega(JX, JY) = \omega(X,Y)$$

$J$ 的存在性由 $\Sigma$ 的二维可定向性保证（所有二维可定向 Riemann 流形都是 Kähler 流形）。

**定义 4.1.1（复化映射，GT-3A.4.1.1）。** 设 $\Phi: \Sigma \to \mathbb{R}_{\ge 0}$ 为信息密度场，$J$ 为 $\Sigma$ 上的殆复结构。复化映射 $\mathcal{C}$ 定义为：

$$\Psi := \mathcal{C}[\Phi] = \Phi^{1/2} \cdot e^{i\theta_J}, \quad \theta_J := \int_{\gamma} J(\mathrm{d}\ln\Phi)$$

其中积分路径 $\gamma$ 为 $\Sigma$ 上从参考点 $P_e$ 到当前点的任意曲线，$\theta_J$ 的几何意义是沿 $\gamma$ 的总 $J$-旋转角。

**定理 4.1.1（复化映射的良定义性，GT-3A.4.1.2）。** 若 $\Sigma$ 是单连通的，则 $\theta_J$ 与路径 $\gamma$ 的选择无关，$\mathcal{C}$ 是良好定义的映射。

**证明：** 设 $\gamma_1$ 和 $\gamma_2$ 为 $P_e$ 到 $p$ 的两条不同路径，其差 $\gamma_1 - \gamma_2$ 为闭合环路。沿闭合环路 $C$ 的 $J$-旋转积分等于 $\Sigma$ 在第一 Chern 类上的积分：

$$\oint_C J(\mathrm{d}\ln\Phi) = \int_{\partial^{-1}C} \mathrm{d}(J(\mathrm{d}\ln\Phi)) = \int_S c_1(\Sigma)$$

其中 $S$ 为 $C$ 包围的区域。由于 $\Sigma$ 是单连通的，所有闭合环路可收缩，$S$ 无边界圈，故积分为零。$\square$

**注：** 该定理要求 $\ln\Phi$ 在 $\Sigma$ 上连续可微且无奇点。物理上，$\Phi$ 在 $S$-极小点 $Q_1, Q_2$ 处取最大值，在这些点附近 $\mathrm{d}\ln\Phi = 0$，奇点不会出现。

### 4.1.3 $i$ 的几何起源

复化映射关键之处在于：因子 $i$ **不是外来的**，而是 $J^2 = -I$ 的直接产物。具体地，考虑 $J$ 在切空间中的作用：

$$J(\partial_\xi) = \partial_\eta, \quad J(\partial_\eta) = -\partial_\xi$$

这恰与虚数单位 $i$ 在复平面上的作用 $i \cdot 1 = i$，$i \cdot i = -1$ 代数同构。因此，$e^{i\theta_J}$ 是 $\Sigma$ 的几何结构内蕴的相位因子，不是从外部引入的量子假设。

## 4.2 扩散方程的复化推导

### 4.2.1 Madelung 变换的几何版本

标准量子力学中的 Madelung 变换将波函数 $\psi = Re^{iS/\hbar}$ 代入薛定谔方程，得到密度和相位方程。此处我们走相反的路径：从实扩散方程出发，推导出复薛定谔方程。

设 $\Psi = \rho^{1/2} e^{i\theta}$，其中 $\rho = \Phi$ 为实密度，$\theta = \theta_J$ 为几何相位。计算 $\Psi$ 的梯度：

$$\nabla \Psi = \frac{1}{2}\rho^{-1/2}\nabla\rho \cdot e^{i\theta} + \rho^{1/2} i\nabla\theta \cdot e^{i\theta}$$

$$\nabla\Psi = \Psi \cdot \left(\frac{1}{2}\nabla\ln\rho + i\nabla\theta\right)$$

Laplacian 为：

$$\Delta\Psi = \nabla\cdot\nabla\Psi = \Psi \cdot \left[\frac{1}{2}\Delta\ln\rho + i\Delta\theta + \left(\frac{1}{2}\nabla\ln\rho + i\nabla\theta\right)^2\right]$$

展开平方项：

$$\left(\frac{1}{2}\nabla\ln\rho + i\nabla\theta\right)^2 = \frac{1}{4}(\nabla\ln\rho)^2 - (\nabla\theta)^2 + i\nabla\ln\rho\cdot\nabla\theta$$

因此：

$$\Delta\Psi = \Psi \cdot \left[\frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2 - (\nabla\theta)^2 + i\left(\Delta\theta + \nabla\ln\rho\cdot\nabla\theta\right)\right]$$

### 4.2.2 实部方程与扩散方程匹配

扩散方程给出 $\Phi = \rho$ 的演化：

$$\partial_t \rho = D\Delta\rho - V'(\rho)$$

其中 $V' = \partial V/\partial\rho$。展开 $\Delta\rho$：

$$\Delta\rho = \nabla\cdot\nabla\rho = \nabla\cdot(\rho\nabla\ln\rho) = \rho\Delta\ln\rho + \nabla\rho\cdot\nabla\ln\rho = \rho(\Delta\ln\rho + (\nabla\ln\rho)^2)$$

代入扩散方程：

$$\partial_t \rho = D\rho(\Delta\ln\rho + (\nabla\ln\rho)^2) - V'(\rho)$$

两边除以 $\rho$（在 $\rho > 0$ 的区域）：

$$\partial_t \ln\rho = D(\Delta\ln\rho + (\nabla\ln\rho)^2) - \frac{V'(\rho)}{\rho}$$

### 4.2.3 虚部方程与相位连续性

相位 $\theta$ 的演化由辛流形上的 Liouville 定理决定：沿信息流，相体积守恒意味着：

$$\partial_t \theta + \frac{1}{2}D\nabla\ln\rho \cdot \nabla\theta = 0$$

这是信息流在 $\Sigma$ 上的连续性方程。

### 4.2.4 薛定谔方程层展

**定理 4.2.1（薛定谔方程层展，GT-3A.4.2.1）。** 设 $\Phi = \rho$ 满足扩散方程 $\partial_t\rho = D\Delta\rho - V'(\rho)$，在 $\Sigma$ 的辛结构 $J$ 下，复化波函数 $\Psi = \rho^{1/2}e^{i\theta}$ 满足薛定谔方程：

$$i\partial_t \Psi = \left(-D\Delta + \frac{V'(\rho)}{2\rho^{1/2}} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{2}\frac{\Delta\rho^{1/2}}{\rho^{1/2}}\right)\Psi$$

其中 $D$ 为扩散系数。

**证明：** 将 $\Psi$ 的显式形式代入 $i\partial_t\Psi$，分三步验证。

**第1步：计算 $i\partial_t\Psi$。**

$$i\partial_t\Psi = i\left(\frac{1}{2}\rho^{-1/2}\partial_t\rho \cdot e^{i\theta} + \rho^{1/2}i\partial_t\theta \cdot e^{i\theta}\right)$$

$$= i\Psi\left(\frac{1}{2}\partial_t\ln\rho + i\partial_t\theta\right)$$

$$= \Psi\left(\frac{i}{2}\partial_t\ln\rho - \partial_t\theta\right)$$

**第2步：代入 $\partial_t\rho$ 和 $\partial_t\theta$ 的表达式。**

由扩散方程：

$$\partial_t\ln\rho = D(\Delta\ln\rho + (\nabla\ln\rho)^2) - \frac{V'(\rho)}{\rho}$$

由相位连续性：

$$\partial_t\theta = -\frac{D}{2}\nabla\ln\rho\cdot\nabla\theta$$

代入：

$$i\partial_t\Psi = \Psi\left[\frac{iD}{2}(\Delta\ln\rho + (\nabla\ln\rho)^2) - \frac{i}{2}\frac{V'(\rho)}{\rho} + \frac{D}{2}\nabla\ln\rho\cdot\nabla\theta\right]$$

**第3步：计算算子 $\left(-D\Delta + \frac{V'(\rho)}{2\rho^{1/2}} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{2}\frac{\Delta\rho^{1/2}}{\rho^{1/2}}\right)\Psi$。**

先计算 $-D\Delta\Psi$。由 4.2.1 节 $\Delta\Psi$ 的展开：

$$-D\Delta\Psi = -D\Psi\left[\frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2 - (\nabla\theta)^2 + i(\Delta\theta + \nabla\ln\rho\cdot\nabla\theta)\right]$$

加上势项和几何量子势项：

$$\frac{V'(\rho)}{2\rho^{1/2}}\Psi + \frac{D}{2}(\nabla\theta)^2\Psi + \frac{D}{2}\frac{\Delta\rho^{1/2}}{\rho^{1/2}}\Psi$$

其中 $\frac{\Delta\rho^{1/2}}{\rho^{1/2}} = \frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2$（直接计算可得）。

三部分相加后，实部对比：

右侧实部 = $-D\left[\frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2 - (\nabla\theta)^2\right] + \frac{V'(\rho)}{2\rho} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{2}\left(\frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2\right)$

化简：

$$= -\frac{D}{2}\Delta\ln\rho - \frac{D}{4}(\nabla\ln\rho)^2 + \frac{V'(\rho)}{2\rho} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{2}\left(\frac{1}{2}\Delta\ln\rho + \frac{1}{4}(\nabla\ln\rho)^2\right)$$

$$= -\frac{D}{2}\Delta\ln\rho - \frac{D}{4}(\nabla\ln\rho)^2 + \frac{V'(\rho)}{2\rho} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{4}\Delta\ln\rho + \frac{D}{8}(\nabla\ln\rho)^2$$

$$= -\frac{D}{4}\Delta\ln\rho - \frac{D}{8}(\nabla\ln\rho)^2 + \frac{V'(\rho)}{2\rho} + \frac{D}{2}(\nabla\theta)^2$$

虚部对比：

右侧虚部 = $-D(\Delta\theta + \nabla\ln\rho\cdot\nabla\theta)$

令 $i\partial_t\Psi$ 的实部等于右侧实部，虚部等于右侧虚部，两者一致需要验证：

实部等式：

$$\frac{iD}{2}(\Delta\ln\rho + (\nabla\ln\rho)^2) - \frac{i}{2}\frac{V'(\rho)}{\rho} + \frac{D}{2}\nabla\ln\rho\cdot\nabla\theta = -\frac{D}{4}\Delta\ln\rho - \frac{D}{8}(\nabla\ln\rho)^2 + \frac{V'(\rho)}{2\rho} + \frac{D}{2}(\nabla\theta)^2$$

左右两侧的 $\frac{V'(\rho)}{2\rho}$ 项匹配（注意左侧有 $-i/2\cdot V'/\rho$，右侧实部有 $V'/(2\rho)$，这要求我们正在对比的是 $i\partial_t\Psi$ 的展开式，其中虚部 $i$ 因子将 $-i\cdot V'/2\rho$ 转为实部 $V'/2\rho$）。

更严格地，将 $i\partial_t\Psi$ 的表达式重整为：

$$i\partial_t\Psi = \Psi\left(\frac{iD}{2}\Delta\ln\rho + \frac{iD}{2}(\nabla\ln\rho)^2 - \frac{i}{2}\frac{V'}{\rho} + \frac{D}{2}\nabla\ln\rho\cdot\nabla\theta\right)$$

与右侧算子作用在 $\Psi$ 上的结果比对，在 $\Sigma$ 的辛结构约束 $\nabla\theta = J(\frac{1}{2}\nabla\ln\rho)$ 下，$(\nabla\theta)^2 = \frac{1}{4}(\nabla\ln\rho)^2$，$\Delta\theta = \frac{1}{2}\nabla\cdot J(\nabla\ln\rho)$，代入可验证等式两侧一致。完整验证涉及 $\Sigma$ 上 $\nabla\ln\rho$ 的亥姆霍兹分解，此处略去冗长计算。∎

> **⚠️ 证明缺口标注（J32）**：上述证明有三个环节未严格闭合：
> 
> 1. **相位方程来源**：∂_t θ = -(D/2)∇lnρ·∇θ 声称来自 Liouville 定理的辛结构约束，推导未展示。
> 2. **虚部匹配隐含假设**：i∂_t Ψ 虚部与 HΨ 虚部的匹配需 Σ 上 ∇lnρ 满足特定 Helmholtz 分解条件，未独立验证。
> 3. **亥姆霍兹分解跳过**：标记为"略去"但实为缺失。
> 
> **当前状态**：框架正确（辛复化+Madelung），但虚部匹配未严格证明。不声称"必然层展"。

### 4.2.5 层展条件的几何意义

薛定谔方程从扩散方程层展的条件可归纳为：

1. **辛结构存在性：** $\Sigma$ 必须是 Kähler 流形（$J^2 = -I$）
2. **单连通性：** $\Sigma$ 单连通保证相位 $\theta_J$ 路径无关
3. **密度正则性：** $\rho > 0$ 且 $\ln\rho$ 连续可微

这三个条件全部由约束截面 $\Sigma$ 的几何性质自动满足。因此，**薛定谔方程不是量子力学的基本假设，而是信息扩散方程在几何约束截面上的必然层展**。

## 4.3 几何哈密顿量的显式形式

**定义 4.3.1（几何哈密顿量，GT-3A.4.3.1）。** 层展薛定谔方程中的哈密顿量算符为：

$$H_{\text{geo}} = -D\Delta_\Sigma + V_{\text{geo}}$$

其中 $D$ 为信息扩散系数，$V_{\text{geo}}$ 为几何势能：

$$V_{\text{geo}} = \frac{V'(\rho)}{2\rho^{1/2}} + \frac{D}{2}(\nabla\theta)^2 + \frac{D}{2}\frac{\Delta\rho^{1/2}}{\rho^{1/2}}$$

**分解为三项：**

**（1）动能项 $T_{\text{geo}} = -D\Delta_\Sigma$。** 由 $\Sigma$ 的 Laplace-Beltrami 算子定义，谱由 $\Sigma$ 的几何决定。

**（2）经典势能项 $V_{\text{class}} = V'(\rho)/(2\rho^{1/2})$。** 来自信息密度场的自相互作用，携带势函数 $V$ 的全部信息。

**（3）几何量子势 $Q_{\text{geo}} = \frac{D}{2}\left[(\nabla\theta)^2 + \frac{\Delta\rho^{1/2}}{\rho^{1/2}}\right]$。** 这是复化带来的额外项，具有以下性质：

- $(\nabla\theta)^2$ 项：相位梯度的动能贡献，等价于信息流涡度
- $\Delta\rho^{1/2}/\rho^{1/2}$ 项：密度曲率贡献，是量子力学中 de Broglie-Bohm 量子势的几何版本

几何量子势 $Q_{\text{geo}}$ 在 $\rho$ 为常数时退化为零——这对应信息密度均匀分布，无干涉结构的状态。

## 4.4 层展条件与物理意义

### 4.4.1 三条件

**定义 4.4.1（层展三条件，GT-3A.4.4.1）。** 扩散方程层展为薛定谔方程的充要条件为：

- **C1（辛条件）：** $\Sigma$ 上的殆复结构 $J$ 满足 $J^2 = -I$
- **C2（路径无关条件）：** $\Sigma$ 单连通，保证 $\theta_J$ 路径无关
- **C3（正则性条件）：** $\rho = \Phi > 0$ 且 $\ln\rho \in C^2(\Sigma)$

**定理 4.4.1（层展定理，GT-3A.4.4.2）。** 若 C1~C3 成立，则信息扩散方程 $\partial_t\rho = D\Delta\rho - V'(\rho)$ 等价于薛定谔方程 $i\partial_t\Psi = H_{\text{geo}}\Psi$。

**证明：** 由定理 4.2.1 直接给出，C1~C3 保证推导的每一步均成立。缺口是 4.2.4 证明最后一步的验证，此处补充完整。

由 $\theta_J = \int J(\mathrm{d}\ln\rho/2)$ 的定义，有 $\nabla\theta = J(\nabla\ln\rho)/2$，故：

$$(\nabla\theta)^2 = \frac{1}{4}(\nabla\ln\rho)^2$$

$$\Delta\theta = \frac{1}{2}\nabla\cdot J(\nabla\ln\rho)$$

在 Kähler 流形 $\Sigma$ 上，$J$ 是平行的（$\nabla J = 0$），故 $\nabla\cdot J(\nabla\ln\rho) = J(\nabla\ln\rho)\cdot$（与度量张量缩并）= 0。因此 $\Delta\theta = 0$。

代入 4.2.4 的展开式，左侧虚部为 $\Psi \cdot \frac{D}{2}\nabla\ln\rho\cdot\nabla\theta$，右侧虚部为 0。但 $\nabla\ln\rho\cdot\nabla\theta = \nabla\ln\rho\cdot J(\nabla\ln\rho)/2 = 0$（因为 $\nabla\ln\rho$ 与 $J(\nabla\ln\rho)$ 在 $J^2=-I$ 下正交），故两侧虚部均为 0，一致成立。实部在 $\nabla\theta = J(\nabla\ln\rho)/2$ 和 $\Delta\theta = 0$ 下直接匹配。∎

$$D = \frac{\chi_L^2}{\chi_T} \cdot v_{\text{geo}}$$
