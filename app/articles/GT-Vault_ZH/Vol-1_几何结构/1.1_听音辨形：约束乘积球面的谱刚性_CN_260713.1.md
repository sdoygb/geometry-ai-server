# 第1章 听音辨形：约束乘积球面的谱刚性

---

## 1.1 引言：一个古老的问题

1966年，Mark Kac 在一篇著名文章中提出了一个看似简单的问题：

**"Can one hear the shape of a drum?"**
（能否听出鼓的形状？）

意思是：如果你只知道一个鼓发出的所有音高（即 Laplace 算子的特征值），能不能唯一确定鼓面的形状？

这个问题在数学上等价于：**Laplace 谱是否能唯一确定黎曼流形的等距类？**

### 1.1.1 为什么这个问题重要

这不是一个单纯的谜题。Laplace 算子的特征值对应物理系统的**共振频率**——从鼓面振动到氢原子光谱，再到宇宙微波背景辐射的温度涨落，都是同一个数学结构在起作用。如果谱能唯一确定几何，那就意味着"测谱即测形"：只要测准频率，就能反推出产生这些频率的几何结构。

但现实是反直觉的：**一般来说，不能。**

### 1.1.2 反例的震撼

1992年，Gordon、Webb 与 Wolpert 构造了第一对**等谱非等距**的平面区域——两个形状不同的鼓面，却发出完全相同的音高。从此，"听音辨形"问题被证明在一般情况下没有肯定答案。

但等等——"一般情况"不等于"所有情况"。

本文研究的是一个**高度约束**的特殊流形类：三个球面的乘积，且它们的半径被群论锁定在固定比例。在这种约束下，谱刚性（spectral rigidity）意外地成立。而且只需要**前三个特征值**就足以唯一确定几何。

这个结果不是靠运气，而是有一个深刻的数学原因：**模空间的维数足够低**。

## 1.2 预备知识：从一个球面开始

在进入九维之前，先搞清楚一维的情况。

### 1.2.1 单个 S³ 上的谱

$S^3(R)$ 是半径为 $R$ 的三维球面。作为一个紧致流形，它的 Laplace–Beltrami 算子的谱是完全已知的。

**事实 1.2.1（$S^3(R)$ 的 Laplace 谱）** 设 $S^3(R)$ 配备标准度量。其 Laplace 算子的特征值为

$$\lambda_p = \frac{p(p+2)}{R^2}, \qquad p = 0, 1, 2, \dots$$

其中 $p=0$ 对应常数函数（零特征值），$p=1$ 给出第一非零特征值 $\lambda_1 = 3/R^2$。

**为什么是这个公式？** 推导思路如下。将 $S^3(R)$ 嵌入 $\mathbb{R}^4$ 为 $\{x \in \mathbb{R}^4 : |x| = R\}$。$p$ 次齐次调和多项式在球面上的限制给出球面谐波。将 $\mathbb{R}^4$ 的 Laplacian 分离为径向和角向部分：

$$\Delta_{\mathbb{R}^4} = \frac{\partial^2}{\partial r^2} + \frac{3}{r}\frac{\partial}{\partial r} + \frac{1}{r^2}\Delta_{S^3}.$$

对 $p$ 次齐次函数 $f(r,\theta) = r^p Y(\theta)$，$\Delta_{\mathbb{R}^4}f = 0$ 给出 $\Delta_{S^3}Y = -p(p+2)Y$。缩放到半径 $R$，特征值为 $p(p+2)/R^2$。

**关键观察：** 如果只知道 $\lambda_1 = 3/R^2$，能不能唯一确定 $R$？能：

$$R = \sqrt{\frac{3}{\lambda_1}}.$$

一个特征值→一个半径。一一对应，明确无误。

### 1.2.2 三个球面的乘积

现在考虑三个 $S^3$ 的乘积流形：

$$M = S^3(a) \times S^3(b) \times S^3(c),$$

配备乘积度量 $g = g_a \oplus g_b \oplus g_c$。它的 Laplace 谱是什么？

**事实 1.2.2（乘积流形上的谱分离）** 在乘积流形上，Laplace 算子分离为各因子上的算子和：

$$\Delta_g = \Delta_1 + \Delta_2 + \Delta_3,$$

其中 $\Delta_i$ 作用于第 $i$ 个因子。因此特征值是各因子特征值之和：

$$\lambda_{p,q,r} = \frac{p(p+2)}{a^2} + \frac{q(q+2)}{b^2} + \frac{r(r+2)}{c^2}, \qquad p,q,r \in \mathbb{N}_0.$$

$p=q=r=0$ 对应零特征值（常数函数），非零特征值从 $p+q+r \geq 1$ 开始。

现在问题来了：三个半径 $(a,b,c)$ 需要多少个特征值才能唯一确定？

如果只知道**前两个**非零特征值（分别为 $3/a^2$ 和 $3/b^2$），可以解出 $a$ 和 $b$，但 $c$ 悬而未决。**两个特征值不足以确定三个几何参数。**

这就是 Kac 问题在乘积球面上的具体体现：谱信息不够用。

## 1.3 $S_3$ 群论：把三个参数变成一个

现在引入本文最关键的输入：**三分切丛的置换对称性**。

### 1.3.1 三扇区结构的必然性

几何论的基本设定是：九维切丛 $TM$ 分解为三个扇区的直和：

$$TM = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}.$$

这三个扇区在度规层面**完全等价**——即任意置换扇区标签诱导一个等距自同构。

**置换群的唯一性（GT-1.1.0.1）** {#GT-1.1.0.1}

设 $G$ 为三分切丛的扇区置换对称群。若 $G$ 在三个扇区上的作用忠实、可迁，且包含任意两扇区对换，则 $G \cong S_3$，且 $S_3$ 是满足此条件的唯一有限群。

*证明*：三个对象的忠实置换群必为 $S_3$ 的子群。可迁性要求轨道大小为 3，故 $|G| \geq 3$。$S_3$ 的可迁子群仅有 $A_3 \cong \mathbb{Z}_3$（循环群）与 $S_3$ 本身。若要求包含对换（奇置换），则 $A_3$ 不足，唯一可能为 $S_3$。 ∎

所以 $S_3$ 是唯一选择——不是假设，而是从"三扇区等价"导出的必然结论。

### 1.3.2 两个关键数字：Λ = 3，k₀ = 2

$S_3$ 作为群，有两个与谱刚性直接相关的群论不变量：

**互锁函数与紧致函数（GT-1.1.0.2）** {#GT-1.1.0.2}

设 $G$ 为有限群。定义：
- **互锁函数** $\Lambda(G) := |\text{Conj}(G)|$ —— $G$ 的共轭类数（等于不可约复表示数）；
- **紧致函数** $k_0(G) := [G : N_{\max}]$ —— $G$ 对其极大正规子群的指数。

**S₃的群论函数（GT-1.1.0.3）** {#GT-1.1.0.3}

对于 $G = S_3$：

$$\Lambda(S_3) = 3, \qquad k_0(S_3) = 2.$$

*证明*：$S_3$ 的共轭类分解为：恒等元 $\{e\}$（1 个）、对换 $\{(12),(13),(23)\}$（3 个）、3-循环 $\{(123),(132)\}$（2 个），共 3 个共轭类，故 $\Lambda(S_3) = 3$。

$S_3$ 的正规子群仅有 $\{e\}$ 与 $A_3 \cong \mathbb{Z}_3$。极大正规子群为 $A_3$，其指数 $[S_3 : A_3] = 6/3 = 2$，故 $k_0(S_3) = 2$。 ∎

这两个数字 $\Lambda=3$ 和 $k_0=2$ 不是自由参数。如果三分切丛是四扇区结构（置换群 $S_4$），则 $\Lambda(S_4)=5$，$k_0(S_4)=2$——那将是一个完全不同的理论。

### 1.3.3 从群论权重到几何比例

$S_3$ 有一个正规子群链：

$$\{e\} \lhd A_3 \lhd S_3.$$

三层子群的阶数分别为 $|\{e\}| = 1$，$|A_3| = 3$，$|S_3| = 6$。取阶数的倒数并归一化：

$$\frac{1}{1} : \frac{1}{3} : \frac{1}{6} = 6 : 2 : 1.$$

这个 **6:2:1 权重** 将决定三个球面因子的半径比。具体规则是：若某一层由剩余对称性 $H$ 控制，则其特征长度平方与 $|H|^{-1}$ 成正比（等价于与权重 $w_H = [S_3:H]$ 成正比）。

取倒数平方根（因为特征长度 $\propto 1/\sqrt{\text{权重}}$），得到半径比：

$$R_1 : R_2 : R_3 = 1 : \frac{1}{\sqrt{3}} : \frac{1}{\sqrt{6}}.$$

三个球面的半径比例被锁定了。

## 1.4 约束乘积球面（CPS）

现在可以给出本章的核心定义。

### 1.4.1 定义

**约束乘积球面（GT-1.1.0.4）** {#GT-1.1.0.4}

设 $a > 0$ 为整体尺度因子。称九维闭流形

$$M(a) = S^3(a) \times S^3\!\left(\frac{a}{\sqrt{3}}\right) \times S^3\!\left(\frac{a}{\sqrt{6}}\right)$$

在配备乘积度量 $g = g_a \oplus g_{a/\sqrt{3}} \oplus g_{a/\sqrt{6}}$ 时，构成一个**约束乘积球面**（Constrained Product Sphere, CPS）。

记 $b = a/\sqrt{3}$，$c = a/\sqrt{6}$。则 $a > b > c$ 严格成立。

**比例锁定的来源：** 三个因子半径的比值 $1 : 1/\sqrt{3} : 1/\sqrt{6}$ 来自于 $S_3$ 群论权重 $6:2:1$。参数 $\Lambda=3$ 与 $k_0=2$ 是 $S_3$ 群论函数的必然输出，不是人为选择的。

参数关系可简洁表达为：

$$b^2 = \frac{a^2}{\Lambda},\qquad c^2 = \frac{a^2}{\Lambda k_0}.$$

### 1.4.2 模空间：一维的

整个 CPS 类 $\{M(a)\}_{a>0}$ 只有一个自由参数：整体尺度 $a$。所有几何量（曲率、体积、谱）都随 $a$ 变化，但比例关系不変。

**模空间的参数化（GT-1.1.0.5）** {#GT-1.1.0.5}

CPS 类 $\{M(a)\}_{a>0}$ 的模空间与开区间 $(0, \infty)$ 同胚，其中 $a$ 是唯一的连续自由度。不同 $a$ 对应不同的等距类。

*证明*：设 $a \neq a'$。$M(a)$ 与 $M(a')$ 若等距，则其直径与体积必须相等。对乘积度量：

$$\text{diam}(M(a)) = \pi\sqrt{a^2 + b^2 + c^2} = \pi a \sqrt{1 + \frac{1}{\Lambda} + \frac{1}{\Lambda k_0}},$$

其中 $\sqrt{1 + 1/\Lambda + 1/(\Lambda k_0)}$ 是与 $a$ 无关的正常数。故直径是 $a$ 的严格单调函数，$a \neq a'$ 给出不同等距类。 ∎

**关键洞察**：模空间是**一维的**。这意味着只需要**一个特征值**就足以确定 $a$。前三个特征值中的后两个是自洽性检验——它们不是确定几何所必需的，但它们提供了检验"这个流形是否确实属于 CPS 类"的判据。

## 1.5 Laplace 特征值的显式计算

### 1.5.1 前三特征值

由谱分离公式和已知的 $S^3(R)$ 特征值，可以直接计算。

**CPS的前三Laplace特征值（GT-1.1.0.6）** {#GT-1.1.0.6}

对 $M(a)$，前三个非零特征值为：

| 模式 (p,q,r) | 特征值表达式 | 与 λ₁ 的比值 |
|:---:|:---|:---:|
| (1,0,0) | 3/a² | 1 |
| (2,0,0) | 8/a² | 8/3 ≈ 2.667 |
| (0,1,0) | 3/b² = 3Λ/a² | Λ = 3 |

*证明*：由谱分离公式（事实1.2.2），各模式的特征值为：

- $(1,0,0)$：$1(1+2)/a^2 = 3/a^2$。
- $(2,0,0)$：$2(2+2)/a^2 = 8/a^2$。
- $(0,1,0)$：$1(1+2)/b^2 = 3/b^2 = 3\Lambda/a^2$。

需要验证这三者确实是**前三个**非零特征值。候选的竞争模式有 $(0,0,1)$（给出 $3/c^2 = 3\Lambda k_0/a^2 = 18/a^2$）和 $(1,1,0)$（给出 $3/a^2 + 3/b^2 = 3(\Lambda+1)/a^2 = 12/a^2$）。由于 $\Lambda = 3 > 8/3$，有 $3\Lambda/a^2 = 9/a^2 > 8/a^2$；又 $k_0 = 2$，故 $3\Lambda k_0/a^2 = 18/a^2$ 和 $3(\Lambda+1)/a^2 = 12/a^2$ 均大于 $9/a^2$。因此前三非零特征值严格如上表所示。 ∎

### 1.5.2 谱特征签名

**谱特征签名（GT-1.1.0.7）** {#GT-1.1.0.7}

比值

$$\frac{\lambda_2}{\lambda_1} = \frac{8}{3}, \qquad \frac{\lambda_3}{\lambda_1} = \Lambda = 3$$

是与 $a$ 无关的普适常数，称为 CPS 类的**谱特征签名**。

如果你测量一个流形的 Laplace 谱，发现前两个非零特征值的比值不是 $8/3$，那么它肯定**不是** CPS。谱特征签名就像身份证——不是"因为这个流形存在，所以比值是某个数"，而是"因为比值是这些数，所以这个流形属于 CPS 类"。

### 1.5.3 反解：从谱到几何

**反解公式（GT-1.1.0.8）** {#GT-1.1.0.8}

给定 $M(a)$ 的前三个非零 Laplace 特征值 $\lambda_1, \lambda_2, \lambda_3$，尺度因子被唯一确定：

$$a = \sqrt{\frac{3}{\lambda_1}}, \qquad b = \sqrt{\frac{3}{\lambda_3}}, \qquad c = \frac{a}{\sqrt{\Lambda k_0}} = \sqrt{\frac{3}{\Lambda k_0 \lambda_1}}.$$

**自洽性条件**：一组正数 $(\lambda_1, \lambda_2, \lambda_3)$ 为某个 CPS 的前三个特征值，当且仅当

$$\lambda_2 = \frac{8}{3}\lambda_1 \quad \text{且} \quad \lambda_3 = 3\lambda_1.$$

*证明*：由 $\lambda_1 = 3/a^2$ 直接解出 $a = \sqrt{3/\lambda_1}$。由 $\lambda_3 = 3/b^2$ 解出 $b = \sqrt{3/\lambda_3}$。再由 $b^2 = a^2/\Lambda$ 知 $\lambda_3 = 3\Lambda/a^2 = \Lambda \lambda_1$，故 $\lambda_3/\lambda_1 = \Lambda = 3$。同理 $\lambda_2/\lambda_1 = 8/3$。 ∎

## 1.6 谱刚性定理

铺垫至此，核心定理呼之欲出。

### 1.6.1 定理陈述

**CPS的谱刚性（GT-1.1.0.9）** {#GT-1.1.0.9}

设 $M(a)$ 与 $M(a')$ 为两个约束乘积球面。若它们的 Laplace 算子具有相同的**前三个非零特征值**，则 $M(a)$ 与 $M(a')$ 等距。

### 1.6.2 证明

*证明* 分为四个步骤。

**步骤 1：模空间由 $a$ 唯一参数化。** 由[模空间的参数化（GT-1.1.0.5）](#GT-1.1.0.5)，CPS 类 $\{M(a)\}_{a>0}$ 的模空间是 $a \in (0,\infty)$。不同 $a$ 对应不同等距类。

**步骤 2：前三个特征值确定 $a$。** 由[反解公式（GT-1.1.0.8）](#GT-1.1.0.8)，$\lambda_1 = 3/a^2$ 唯一确定 $a$。$\lambda_2$ 和 $\lambda_3$ 提供自洽性检验：若 $M(a)$ 和 $M(a')$ 的前三个特征值相同，则 $a = a'$。

**步骤 3：拓扑唯一。** 所有 CPS 均微分同胚于 $S^3 \times S^3 \times S^3$，与 $a$ 无关。

**步骤 4：等距。** $a = a'$ 意味着相同的半径 $(a,b,c)$，进而相同的乘积度量，因此 $M(a)$ 与 $M(a')$ 等距。 ∎

### 1.6.3 为什么只用了三个特征值？

注意：在步骤 2 中，实际上只需要**第一特征值** $\lambda_1$ 就足以确定 $a$（因为 $a = \sqrt{3/\lambda_1}$）。那为什么还要提"前三个"？

有两个原因：

1. **自洽性检验**：$\lambda_2$ 和 $\lambda_3$ 提供了检验"这个流形是否确实是 CPS"的判据。如果 $\lambda_2/\lambda_1 \neq 8/3$，则它根本不属此流形类——谱刚性定理的前提不成立。

2. **模空间维数论证**：CPS 类模空间的维数 $d=1$，谱刚性维数必要条件要求使用至少 $k \geq 1$ 个特征值。这里用了 $k=3$，远超必要，但也因此更鲁棒。

## 1.7 谱刚性的维数必要条件

### 1.7.1 为什么一般情形谱刚性失败？

Gordon-Webb-Wolpert 的等谱非等距反例告诉了我们一个残酷的事实：**在一般情况下，无穷个特征值都不足以确定几何。**

为什么？因为一般流形类的模空间是**无穷维**的。谱映射

$$\sigma_k: \mathcal{M} \to \mathbb{R}^k, \quad [g] \mapsto (\lambda_1, \dots, \lambda_k)$$

是从无穷维空间到 $k$ 维空间的映射——不可能是单射。这就是谱刚性失败的根本原因。

### 1.7.2 维数障碍定理

**谱刚性的维数必要条件（GT-1.1.0.10）** {#GT-1.1.0.10}

设 $\mathcal{M}$ 是一个 $d$ 维模空间（即流形类由 $d$ 个连续参数描述）。若谱刚性（前 $k$ 个特征值确定等距类）在 $\mathcal{M}$ 上成立，则必有 $k \geq d$。当 $k < d$ 时，谱刚性必然失败。

*证明*：谱映射 $\sigma_k: \mathcal{M} \to \mathbb{R}^k$ 是连续映射。若 $\sigma_k$ 是单射（即谱刚性成立），则 $\mathcal{M}$ 可嵌入 $\mathbb{R}^k$，故 $\dim \mathcal{M} = d \leq k$。若 $d > k$，则连续映射 $\sigma_k$ 不可能为单射，谱刚性必不成立。 ∎

这个定理给出了谱刚性的一个**纯粹拓扑的障碍**，完全不依赖于流形的具体几何——它只依赖模空间的维数。

### 1.7.3 在 CPS 上的应用

将维数必要条件应用于 CPS 类：$d = 1$（因为模空间由单一尺度参数 $a$ 参数化），谱刚性要求 $k \geq 1$。我们使用 $k = 3 \geq 1$，因此维数障碍**不**阻止谱刚性的成立。

事实上，$d=1$ 是谱刚性可能成立的**最有利情形**——只需一个特征值就可以确定整个等距类。这是 CPS 类能够满足谱刚性的深层原因。

### 1.7.4 与经典反例的兼容性

**反例不适用性（GT-1.1.0.11）** {#GT-1.1.0.11}

Sunada 型等谱非等距构造不适用于 CPS 类。$M(a)$ 的基本群 $\pi_1(M(a)) = 0$（平凡），且 CPS 类的模空间维数 $d=1$ 满足维数必要条件 $d \leq k$。故 Gordon-Webb-Wolpert 及 Sunada 的反例不能应用于 CPS 类。

*证明*：$S^3$ 单连通，故 $\pi_1(M(a)) = \pi_1(S^3) \times \pi_1(S^3) \times \pi_1(S^3) = 0$。CPS 类模空间维数 $d=1$，使用 $k=3 \geq 1$ 个特征值，满足维数必要条件。因此已知等谱非等距构造均不适用。 ∎

### 1.7.5 本节的核心教训

谱刚性定理告诉我们三件事：

1. **约束越多，谱越刚性**——将模空间从无穷维压缩到一维，谱刚性就从不成立变为成立。
2. **$S_3$ 群论的神奇之处**——它不仅给出了谱权重比例，还通过 $\Lambda=3$ 和 $k_0=2$ 将九维流形的模空间压缩到一维。
3. **谱刚性不是自动成立的**——它依赖于特定几何结构（约束乘积球面）。
