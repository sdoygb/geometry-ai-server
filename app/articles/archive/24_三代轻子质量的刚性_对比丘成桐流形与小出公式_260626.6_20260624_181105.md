## 摘要

本文在几何论框架内探索三代带电轻子（$e, \mu, \tau$）质量的代数结构及其对 Koide 公式的几何解释。核心工作包括：(1) 将三代轻子定位于几何约束截面上的三层物理识别点；(2) 建立层级质量映射 $C_n$ 与层级有效作用量 $S^{(n)}$ 的联合框架；(3) 推导电子-缪子联合截面 $W_{12}$ 的显式代数形式及其驻点条件；(4) 探索 $H^W$ 反对称模式与因果角的耦合；(5) 通过 E1（层级角度和修正 $\varepsilon_n$）和 E2（层级有效作用量 $S^{(n)}$）联合约束三代有效投影角。

本文严格区分四个命题层级：**严格定理**（仅依赖三公理或 0.X 系列已证明结果）、**本文定义**（几何框架内新引入的概念）、**数值观察**（框架内计算发现的规律，待严格证明）、**猜想/假设**（框架内未建立但可检验的命题）。

**核心结论**：电子质量在框架内自足锁定（$m_e = 511$ keV）。缪子质量通过层级质量标度 $M_1$ 与因果角 $\theta_C^0$ 的对应给出 126 MeV（偏差 19.6%），偏差来源已从 $W_{12}$ 耦合排除（定理 5.4），转向层级约束截面的展开基准点修正。$W_{12}$ 双线性结构的发现——在任意共同展开点电子与缪子完全线性解耦——是本文最硬的定理级结果。E1+E2 联合框架将 3.94° 的角度和偏差分解为层级修正与 $C_n$ 修正的联合约束。

**关键词**：三代轻子质量；Koide 公式；层级质量映射；联合截面驻点；$W_{12}$ 双线性结构；有效投影角；层级角度和修正；层级有效作用量

## §1 引言：Koide 公式的谜题与几何论策略

### 1.1 问题的几何本质

三代带电轻子质量满足 Koide 公式（1981）：

$$Q = \frac{m_e + m_\mu + m_\tau}{(\sqrt{m_e} + \sqrt{m_\mu} + \sqrt{m_\tau})^2} \approx \frac{2}{3}$$

这个比值对实验值（$Q_{\text{exp}} \approx 0.66666$）的吻合达到了 $10^{-5}$ 量级精度——远超一般唯象拟合的期望。标准模型将三代质量归因于 Yukawa 耦合常数的层级差异，Koide 公式在该框架中是一个"数值巧合"。

几何论的立场不同：若质量谱由约束流形的几何结构刚性锁定，则三代质量关系应当从约束截面的谱刚性中自然涌现。本文探索这一可能性的边界——哪些部分可以严格推导，哪些部分仍是开放假设。

### 1.2 几何论的核心主张（条件性）

**猜想 G1（三代物理识别点存在性——条件性命题*）** 在约束流形 $D_\theta = \{\theta_1 + \theta_2 + \theta_3 = 90^\circ\}$ 上，跨扇区耦合结构 $H^W$ 支持三个物理识别点，分别对应三代带电轻子。该猜想由以下结构要素启发：

1. **全息屏约束截面**（公理 1）：$\theta_M + \theta_C + \theta_I = 90^\circ$ 定义了三分切丛投影的二维单纯形。
2. **跨扇区耦合 $H^W$**（0.1.1 定理 4.2）：$2\times 2$ 矩阵结构编码了扇区间的几何渗透。
3. **Bott 周期上界**（0.0.6 定理 6.3）：$N_{\text{eff}} \le 7$ 限制了可解析信息层级数。
4. **七层截断**（0.3.1.1 研究框架 2.3）：物质界可解析信息层级 $n \le 7$。

> \* **条件性标注**：上述要素在现行框架（260626.6）中均未直接证明"约束流形上存在三个稳定驻点"。G1 是本文的**几何猜想**，其严格化需要：(a) 将 0.1 的单一物理识别点框架推广到多粒子联合截面；(b) 证明 $H^W$ 在多粒子联合截面上存在多个稳定驻点。当前框架仅确立了单一物理识别点（0.1 定理 3.1 的存在唯一性）。

### 1.3 十方路径

- **(A) 公理基础**：公理 1（$\theta_M + \theta_C + \theta_I = 90^\circ$）、公理 2（$S = \sum 1/\sin^2\theta_i + \sum_{i<j} 1/(\sin\theta_i \sin\theta_j)$）、公理 3（$m = K \sin^3\theta_M$）

## §3 三代轻子的几何定位

### 3.1 单一物理识别点（电子）

由 0.1(260626.6) 定理 3.1，约束截面 $D_\theta$ 上存在唯一物理识别点 $P_1$：

$$\boxed{(\theta_M^{(1)}, \theta_C^{(1)}, \theta_I^{(1)}) = (57.93°, 26.16°, 5.91°)}$$

该点由三个方程联合锁定：角度和约束 $+ S = S_e + m_e = K\sin^3\theta_M$。质量输出 $m_e = K\sin^3 57.93° = 510.99895$ keV（0.1 定理 7.3），与实验值 510.998950 keV 偏差 $\sim 10^{-5}$。

**这是框架内自足的定理级结果**——不依赖三代假设、不依赖 Koide 公式。

### 3.2 猜想 G1：三代物理识别点

**猜想 G1（三代物理识别点存在性——条件性命题*）** 在约束流形 $D_\theta$ 的联合截面上，跨扇区耦合 $H^W$ 支持三个物理识别点 $P_1, P_2, P_3$，分别对应 $e, \mu, \tau$。

**当前状态**：G1 在框架内未建立。0.1(260626.6) 定理 3.1 仅证明了单一物理识别点的存在唯一性——三个方程（角度和 + $S=S_e$ + $m=K\sin^3\theta_M$）只有一个解。多粒子联合截面的驻点存在性是开放问题。

**本文的策略**：不假设 G1 为真，而是自底向上构建——电子自足，缪子通过层级约束截面逐层确定，陶子作为第三层。每层有其独立的有效投影角、层级质量映射 $C_n$ 和（若 E2 成立）层级有效作用量 $S^{(n)}$。

### 3.3 层级质量映射

#### 3.3.1 公理 3 的层级推广

公理 3（$m = K \sin^3\theta_M$）在 0.1 中仅对电子层级严格成立。对于 $\mu$ 和 $\tau$，质量远超 $K = 839.76$ keV 的上限（$\sin^3\theta \le 1$），需要层级放大因子。

**假设 H1（层级质量映射）** 第 $n$ 代轻子质量由 $m_n = C_n \sin^3\theta_{\text{eff}}^{(n)}$ 确定，其中 $C_n$ 是层级 $n$ 的质量标度常数，$\theta_{\text{eff}}^{(n)}$ 是有效投影角。$C_1 \equiv K$ 由框架锁定（0.3.1 定理 9.1）。

#### 3.3.3 $\sin^3\theta > 1$ 与层级映射的必要性

若 $C_n = K$ 对所有 $n$（公理 3 普适），则：

$$m_\mu = K \sin^3\theta_{\text{eff}}^{(2)} \le K = 839.76 \text{ keV}$$

### 3.4 缪子-因果角对应（数值观察）

取 $C_2 = M_1 = 1.47$ GeV，$\theta_{\text{eff}}^{(2)} = \theta_C^0 = 26.16°$：

$$m_\mu^{\text{(th)}} = M_1 \sin^3 26.16° = 1.47 \times 10^6 \times 0.0857 = 126 \text{ MeV}$$

$$m_\mu^{\text{(exp)}} = 105.66 \text{ MeV}$$

**偏差 19.6%**。但**方向和量级都对**：百 MeV 量级（正确）、有效投影角落在因果扇区（正确——$\theta_C^0 = 26.16°$ vs 反推 $24.54°$，仅差 $1.62°$）。三扇区（$\mathcal{M}, \mathcal{C}, \mathcal{I}$）中只有因果扇区的角度量级能配合 $M_1$ 产生百 MeV。

**数值观察 3.1（缪子-因果角对应）**：$m_\mu \approx M_1 \sin^3\theta_C^0$，偏差 19.6%。该偏差的几何来源是本文的核心追问。

### 3.5 层级自洽点（第7次修订）

从 $m_\mu^{\text{exp}} = 105.66$ MeV 和 $C_2 = M_1 = 1.47$ GeV 反推：

$$\sin^3\theta_{\text{eff}}^{(2)} = \frac{105.66}{1470} = 0.07188 \Rightarrow \theta_{\text{eff}}^{(2)} \approx 24.54°$$

这个 $24.54°$ 是**层级自洽点**——若 $C_2 = M_1$ 正确，则 $\theta_{\text{eff}}^{(2)}$ 必须取此值才能匹配实验。它不是第一性原理推导，而是实验反推。

同理，$\theta_{\text{eff}}^{(3)} \approx 11.47°$（从 $C_3 = M_2 = 221$ GeV 和 $m_\tau = 1776.86$ MeV 反推）。

三代有效投影角（$C_n = M_{n-1}$ 假设下）：

| 代 | 粒子 | $C_n$ | $\theta_{\text{eff}}^{(n)}$ | 扇区 |
|:--|:---|:---|:---|:---|
| 1 | $e$ | $K = 839.76$ keV | $57.93°$ | $\mathcal{M}$ |
| 2 | $\mu$ | $M_1 = 1.47$ GeV | $24.54°$ | $\mathcal{C}$ |
| 3 | $\tau$ | $M_2 = 221$ GeV | $11.47°$ | $\mathcal{I}$ |

**三代角度和**：$57.93° + 24.54° + 11.47° = 93.94° \neq 90°$。偏差 $\Delta = 3.94°$。

这个 $3.94°$ 是 E1+E2 联合框架的核心约束——见 §6。

## §4 Koide 公式的几何分析

### 4.1 Koide 比值在 $C_n$ 不普适条件下的代数结构

若 $C_1 = K$，$C_2 = M_1$，$C_3 = M_2$，则 Koide $Q$ 为：

$$\boxed{Q = \frac{K s_1^3 + M_1 s_2^3 + M_2 s_3^3}{\left(\sqrt{K} s_1^{3/2} + \sqrt{M_1} s_2^{3/2} + \sqrt{M_2} s_3^{3/2}\right)^2}}$$

其中 $s_n = \sin\theta_{\text{eff}}^{(n)}$。当 $C_1 = C_2 = C_3$（普适）时，$Q$ 退化为纯角度函数。

**$C_n$ 不普适条件下的 $Q = 2/3$ 等价条件**：

$$\frac{K s_1^3 + M_1 s_2^3 + M_2 s_3^3}{\left(\sqrt{K} s_1^{3/2} + \sqrt{M_1} s_2^{3/2} + \sqrt{M_2} s_3^{3/2}\right)^2} = \frac{2}{3}$$

这是 $s_n$ 的非线性代数约束。已知 $C_n$ 比值（$M_1/K = 1755$，$M_2/M_1 = 150$），若 $s_n$ 能同时满足角度和约束（$\sum \theta_{\text{eff}}^{(n)} = 90° + \varepsilon$），则 $Q=2/3$ 可从几何约束推导。

### 4.2 角度和 = 90° 非 Koide 的充分条件

反例：$(\theta_{\text{eff}}^{(1)}, \theta_{\text{eff}}^{(2)}, \theta_{\text{eff}}^{(3)}) = (30°, 30°, 30°)$ 满足角度和 = 90°，但 $s_n = 1/2$，代入（设 $C_n$ 普适）：

$$Q = \frac{3 \times (1/8)}{(\sqrt{3} \times (1/2)^{3/2})^2} = \frac{3/8}{3 \times 1/8} = 1 \neq \frac{2}{3}$$

角度和 = 90° 是必要非充分条件。$Q=2/3$ 要求特定的角度分布。

### 4.3 $\theta_{\text{eff}}^{(1)} = 57.93°$ 后 Koide 的等价条件

将 $\theta_{\text{eff}}^{(1)} = 57.93°$ 代入 $Q=2/3$：

$$s_2^3\left(\frac{2}{3} - \frac{M_1^2}{(\sqrt{K}s_1^{3/2} + \sqrt{M_1}s_2^{3/2} + \sqrt{M_2}s_3^{3/2})^2}\right) + \cdots$$

这是一个关于 $s_2, s_3$ 的隐式约束。给定 $\theta_{\text{eff}}^{(2)}, \theta_{\text{eff}}^{(3)}$ 和 $C_n$ 候选值，可直接数值验证——**这是将来定理化的操作路径**。

### 5.2 构型 B 的单粒子 $H^W$ 微扰：灾难性失效（定理 5.1）

**定理 5.1（单粒子 $H^W$ 微扰在构型 B 处失效——第4次修订，严格证明）**

### 5.3 $H^W$ 的反对称模式-因果角耦合（定理 5.2）

**证明**：直接代数推导。角度和约束 $\theta_M + \theta_C + \theta_I = 90°$ 下，$\theta_I + \theta_M = 90° - \theta_C$，代入即得。$H^W$ 在 $v_-$ 方向的投影矩阵元为 $\lambda_-^W$。$\square$

#### 5.6.1 双粒子联合 Hessian 结构

### 6.3 E1：层级角度和修正 $\varepsilon_n$

#### 6.3.1 框架动机

0.3.1.1 中层级信息标度 $N_n = S_e^2 \cdot \Lambda^{2n}$ 意味着第 $n$ 层全息屏的信息容量是基态的 $\Lambda^{2n}$ 倍。约束截面 $D_\theta$ 从全息屏编码条件导出。信息容量的层级扩张可能导致有效角度跨度偏离严格 $90°$。

**研究框架 E1（层级角度和修正）** 第 $n$ 代轻子的层级约束截面满足：

$$\boxed{\theta_M^{(n)} + \theta_C^{(n)} + \theta_I^{(n)} = 90^\circ + \varepsilon_n}$$

其中 $\varepsilon_1 = 0$（电子层级由框架锁定），$\varepsilon_n$（$n \ge 2$）待定。

> 注：E1 是**研究框架**，不是定理。当前框架（260626.6）中公理 1 仅对基态全息屏 ($n=0$) 严格成立。$\varepsilon_n \neq 0$ 意味着层级的全息屏编码条件偏离基态约束——这需要从全息屏层级嵌套的几何结构中严格导出。0.3.1.1 提供了层级信息标度 $N_n$ 作为候选出发点，但尚未建立 $N_n \to \varepsilon_n$ 的推导链。

#### 6.3.2 候选形式

**候选 E1-A（角度和严格不变）**：

$$\varepsilon_n = 0 \quad \forall n$$

角度和保持 $90°$。$3.94°$ 的偏差全部由 $C_n \neq M_n$ 解释。这是最保守的假设。

**候选 E1-B（信息容量标度）**：

$$\varepsilon_n = \varepsilon_1 \cdot (\Lambda^{n-1} - 1)$$

其中 $\Lambda = 150$。$\varepsilon_2 \approx 149\varepsilon_1$，$\varepsilon_3 \approx 22499\varepsilon_1$。增长极快——若 $\varepsilon_2 \sim 1°$，则 $\varepsilon_3 \sim 150°$（不可能）。此候选物理上不合理。

**候选 E1-C（对数标度）**：

$$\varepsilon_n = \varepsilon_0 \cdot (n-1) \ln \Lambda$$

其中 $\ln \Lambda = \ln 150 \approx 5.01$。$\varepsilon_2/\varepsilon_3 = 1/2$，温和增长。若 $\varepsilon_0 \approx 0.4°$，则 $\varepsilon_2 \approx 2.0°$，$\varepsilon_3 \approx 4.0°$——量级合理。

**候选 E1-D（层级饱和标度）**：

$$\varepsilon_n = \varepsilon_{\max} \cdot (1 - \Lambda^{-(n-1)})$$

其中 $\varepsilon_{\max} \sim 4°$。$\varepsilon_2 \approx 3.97°$（接近饱和），$\varepsilon_3 \approx 4.00°$。物理图像：角度和修正随层级迅速饱和。

#### 6.3.3 实验约束

在 $C_n = M_{n-1}$ 假设下，$\theta_{\text{eff}}^{(n)}$ 由实验反推（§6.2），因此：

$$\sum_{n=1}^3 \theta_{\text{eff}}^{(n)} - 90° = 3.94°$$

若有效投影角严格对应各代的主扇区角（即 $\theta_{\text{eff}}^{(1)} = \theta_M^{(1)}$，$\theta_{\text{eff}}^{(2)} = \theta_C^{(2)}$，$\theta_{\text{eff}}^{(3)} = \theta_I^{(3)}$），且各代内部满足 $90° + \varepsilon_n$，则：

$$3.94° = \varepsilon_2 + \varepsilon_3 + (\theta_M^{(2)} + \theta_I^{(2)} - \theta_C^{(2)} - \theta_M^{(2)}) + \cdots$$

更诚实的表述：$\theta_{\text{eff}}^{(n)}$ 的"代内角度和"与 $\varepsilon_n$ 的关系需要通过联合截面驻点条件建立，当前这一步尚未完成。

**E1 的当前状态**：给出了候选函数形式和数值约束。E1-C（对数标度）和 E1-D（层级饱和标度）在量级上合理。但 $\varepsilon_n$ 的严格推导需要从全息屏层级嵌套的微分几何出发——这是 0.3.1.1 研究框架 3.2 中"递推生成的严格化"的自然延伸。

### 6.4 E2：层级有效作用量 $S^{(n)}$

#### 6.4.1 框架动机

在 0.1 中，电子的物理识别点由 $S = S_e$ 精确锁定。类似地，若层级 $n$ 存在独立于 $S_e$ 的几何有效作用量 $S^{(n)}$，则该层级的物理识别点由 $S = S^{(n)}$ 锁定。

**研究框架 E2（层级有效作用量）** 第 $n$ 代轻子的层级约束截面满足：

$$\boxed{S(\theta_M^{(n)}, \theta_C^{(n)}, \theta_I^{(n)}) = S^{(n)}}$$

其中 $S^{(1)} \equiv S_e = 137.035999084$（电子层级由框架锁定），$S^{(n)}$（$n \ge 2$）是层级 $n$ 的几何有效作用量。

> 注：E2 是**研究框架**，不是定理。$S^{(n)}$ 的存在性和唯一性尚未在框架内证明。关键区别：$S^{(n)}$ 是几何约束截面上的作用量值，**不是**物理精细结构常数——后者始终保持 $S_e$。

#### 6.4.2 $S^{(n)}$ 的候选来源

**候选 E2-A（联合截面驻点——本文首倡）**：

$S^{(n)}$ 不是从信息标度 $N_n$ 外推的，而是双粒子/多粒子联合截面 $H^W$ 驻点条件的**输出**。层级 $n$ 的物理识别点是联合截面上 $S$ 的鞍点——沿 $S$ 等值面的 $H^W$ 最小梯度方向。

$$S^{(n)} = S(\theta_M^{(n)}, \theta_C^{(n)}, \theta_I^{(n)}) \quad \text{其中 } (\theta_M^{(n)}, \theta_C^{(n)}, \theta_I^{(n)}) \text{ 是联合截面驻点}$$

此候选的优势：$S^{(n)}$ 是联合截面几何结构的自然输出，不引入新参数。劣势：需要先确定联合截面驻点——这是 G1 的核心内容。

**候选 E2-B（信息容量路径）**：

由 $N_n = S_e^2 \Lambda^{2n}$，$S^{(n)} \sim N_n^{1/2} = S_e \Lambda^n$。但 $S^{(2)} = 20550$ 过大——六项作用量在约束截面上虽然可达（让某个角趋近 0），但对应的角度配置与缪子的有效投影角 $24.54°$ 不一致。

考虑次线性标度 $S^{(n)} = S_e \cdot \Lambda^{\alpha(n-1)}$：
- $\alpha = 1/7$：$S^{(2)} \approx 259$，$S^{(3)} \approx 380$
- $\alpha = 2/7$：$S^{(2)} \approx 490$，$S^{(3)} \approx 1040$

**候选 E2-C（几何最小值标度）**：

$$S^{(n)} = S_{\min} + (S_e - S_{\min}) \cdot \Lambda^{-\gamma(n-1)}$$

其中 $S_{\min} = 24$（天平点）。物理图像：层级越高，信息场越接近饱和（$\theta_I \to 72.53°$），$S$ 值趋向 $S_{\min}$。

#### 6.4.3 E1+E2 耦合方程

对第 $n$ 代轻子（$n \ge 2$），三扇区角由以下联立方程确定：

$$\begin{cases}
\theta_M^{(n)} + \theta_C^{(n)} + \theta_I^{(n)} = 90^\circ + \varepsilon_n & \text{(E1: 层级角度和)}\\[4pt]
S(\theta_M^{(n)}, \theta_C^{(n)}, \theta_I^{(n)}) = S^{(n)} & \text{(E2: 层级有效作用量)}\\[4pt]
C_n \sin^3\theta_{\text{eff}}^{(n)} = m_n^{\text{exp}} & \text{(质量锁定)}
\end{cases}$$

其中 $\theta_{\text{eff}}^{(n)}$ 是主扇区角（定义 5.1）。三个方程，三个未知数 $(\theta_M^{(n)}, \theta_C^{(n)}, \theta_I^{(n)})$——**原理上可解**，若 $\varepsilon_n$、$S^{(n)}$ 和 $C_n$ 已知。

**关键耦合**：$\varepsilon_n$ 和 $S^{(n)}$ 不是独立的。改变 $\varepsilon_n$ 改变角度和，从而改变 $S$ 的取值范围——进而影响 $S^{(n)}$ 的可行域。

#### 6.4.4 缪子层级的显式约束（$n=2$）

给定 $\theta_C^{(2)} = 24.54°$（从 $C_2 = M_1$ 和 $m_\mu^{\text{exp}}$ 反推）：

$$\begin{cases}
\theta_M^{(2)} + \theta_I^{(2)} = 65.46° + \varepsilon_2 \\[4pt]
S(\theta_M^{(2)}, 24.54°, \theta_I^{(2)}) = S^{(2)}
\end{cases}$$

两个方程确定 $\theta_M^{(2)}$ 和 $\theta_I^{(2)}$。

**若 $\varepsilon_2 = 0$（E1-A）**：
- $\theta_M^{(2)} + \theta_I^{(2)} = 65.46°$
- $S^{(2)}$ 沿此线的值域需要计算
- $S = 137$ 的等值线在 $(\theta_M, \theta_I)$ 平面上的轨迹与 $\theta_M + \theta_I = 65.46°$ 的交点给出 $\theta_M^{(2)}, \theta_I^{(2)}$

**若 $S^{(2)} = S_e = 137$（E2 最保守假设）**：
- $S(\theta_M^{(2)}, 24.54°, \theta_I^{(2)}) = S_e$
- 此方程加上 $\theta_M^{(2)} + \theta_I^{(2)} = 90° - 24.54° = 65.46°$（若 $\varepsilon_2 = 0$）
- 两个方程两个未知数，可直接求解

#### 6.4.5 数值探索：$S^{(2)} = S_e$ 假设下的缪子配置

假设 $\varepsilon_2 = 0$，$S^{(2)} = S_e = 137.036$，$\theta_C^{(2)} = 24.54°$：

$$\begin{cases}
\theta_M^{(2)} + \theta_I^{(2)} = 65.46° \\
S(\theta_M^{(2)}, 24.54°, \theta_I^{(2)}) = 137.036
\end{cases}$$

数值扫描：在 $\theta_M \in [1°, 64°]$ 范围内搜索满足 $S(\theta_M, 24.54°, 65.46°-\theta_M) = 137.036$ 的解。

**观察**：$S(\theta_M, \theta_C, \theta_I)$ 在 $\theta_C$ 固定时，沿 $\theta_M + \theta_I = \text{const}$ 线通常有一个最小值（对称点 $\theta_M = \theta_I$ 处），两端趋向无穷。$S_e = 137$ 远大于 $S_{\min} = 24$，因此通常有两个解（$\theta_M < \theta_I$ 和 $\theta_M > \theta_I$）。

由于缪子主扇区是 $\mathcal{C}$（定义 5.1），$\theta_M^{(2)}$ 和 $\theta_I^{(2)}$ 的角色对称——具体数值分配由联合截面 $H^W$ 驻点条件决定。此计算留待后续修订完成。

### 6.5 E1+E2 联合：核心洞察

**E1 和 E2 通过层级约束截面的几何结构耦合——它们是同一问题的两个侧面。**

$$\boxed{\text{层级 } n \text{ 的物理识别点} \Longleftrightarrow \begin{cases} \text{不同的全息屏信息容量} \Rightarrow \varepsilon_n \neq 0 & (\text{E1}) \\ \text{不同的几何有效作用量} \Rightarrow S^{(n)} \neq S_e & (\text{E2}) \end{cases}}$$

**E2 的方向纠正（第8次修订关键洞察）**：$S^{(n)}$ 不应从 $N_n$ 或 $\rho_n$ 等信息标度量外推——因为 $S^{(n)}$ 的候选值（$S_e \Lambda^{\alpha n}$）与角度约束下的可行域可能冲突。正确的方向是：$S^{(n)}$ 是双粒子/多粒子联合截面 $H^W$ 驻点条件的**输出**，而非独立输入。

### 6.6 3.94° 偏差的分解

三代有效投影角和 $= 93.94°$ 与 $90°$ 的偏差 $3.94°$ 可以在以下两者间分配：
