## 摘 要

72号文章封闭了Bi₂Se₃七螺旋纳米线Tc = 438 K的五个理论缺口，将预言从"启发式构造"升级到"定理级"。但**一个材料不够**——如果几何论的Tc机制是真实的，它应该预言**一族材料**的转变温度，而非孤例。

本文基于几何论的Tc公式：

$$k_B T_c = \frac{E_{\text{topo}} \cdot g_{\text{pair}} \cdot \delta_0^2(N)}{\ln\Omega_0(N, d_{\text{IR}})}$$

构建一个系统化的拓扑超导材料批量筛选方案。该方案：

1. **识别关键材料参数**：哪些是普适的（g_pair, δ₀^single, λ₁^eff, S_e），哪些是材料依赖的（v_F, 晶体结构、螺旋链配置）
2. **给出初始候选列表**：基于已知拓扑绝缘体家族，用几何论公式预测每个材料的Tc
3. **提供Python脚本**：可连接到公开的拓扑材料数据库（Topological Quantum Chemistry Database 38,184种材料、TopoMat 13,628种材料），自动筛选候选物并计算预测Tc
4. **讨论扩展路径**：从Bi₂Se₃族→三元合金→拓扑晶体绝缘体→半Heusler化合物

最终目标：**几何论不再只有一个438 K预言——而是一张Tc预测谱，实验物理学家可以系统性地去验证。**

## 1. 几何论Tc公式的材料依赖参数分析

### 1.1 普适参数（所有材料共享，不依赖材料选择）

| 参数 | 数值 | 几何来源 | 锁定级别 |
|:---|:---:|:---|:---:|
| g_pair | 4.282 | sinθ_C^e/sinθ_I^e | 公理1-3 + 电子基态 |
| δ₀^single | 0.1729 | cot(5.91°)/(√2·λ₁^{1/4}) | 约束截面谱几何 |
| λ₁^eff | 391.05 rad⁻² | Hessian软模（0.2.1） | 谱刚性定理 |
| λ₂^eff | 59324.3 rad⁻² | Hessian硬模（0.2.1） | 谱刚性定理 |
| S_e | 137.036 | 全息屏极值（0.0.5/0.0.6） | 谱刚性定理 |
| sin³θ_M | 0.8479 | 电子基态角度（公理3） | 公理级 |
| χ_L | 1.509×10⁻¹⁰ m | 长度量纲桥（0.3.1） | 定理级 |
| k_B | 8.617×10⁻⁵ eV/K | 标准物理常数 | 外部输入 |

### 1.2 材料依赖参数（决定Tc差异）

| 参数 | 符号 | Tc依赖关系 | 获取途径 |
|:---|:---:|:---|:---|
| 费米速度 | v_F | E_topo ∝ v_F | ARPES实验 / DFT计算 |
| 螺旋链数 | N_eff | T_c(N) ~ N_eff / lnΩ₀(N) | 晶体结构分析 |
| 物理半径 | r_phys | E_topo ∝ 1/r_phys | 纳米线制备参数 |
| 表面态拓扑保护 | Z₂ | 必要条件（否则表面态不鲁棒） | 拓扑材料数据库 |
| 红外截断 | d_IR | lnΩ₀依赖d_IR | Spin(8) triality根系（材料维度） |

### 1.3 材料依赖参数的具体Tc计算公式

**第一步：几何半径**

$$r_{\text{geo}} = \frac{r_{\text{phys}}}{\chi_L}$$

其中χ_L = 1.509×10⁻¹⁰ m为长度量纲桥因子。对于r_phys = 10 nm（标准纳米线直径），r_geo = 66.3。

**第二步：拓扑能标**

$$E_{\text{topo}} = \frac{v_F}{r_{\text{geo}}} \cdot \frac{\lambda_1^{\text{eff}}}{S_e^2} \cdot \sin^3\theta_M$$

代入普适常数：

$$E_{\text{topo}} = \frac{v_F}{r_{\text{geo}}} \cdot 0.02084$$

**第三步：lnΩ₀计算**

d_IR = 2（拓扑表面态，2维Dirac锥）：

$$\ln\Omega_0(N_{\text{eff}}, 2) = \ln 2 + \frac{1}{4}\ln\frac{\lambda_1^{\text{eff}}}{\lambda_2^{\text{eff}}} + \ln\frac{S_e}{4\pi} - \frac{1}{2}\ln N_{\text{eff}}$$

代入数值后的显式形式：

$$\ln\Omega_0(N_{\text{eff}}, 2) = 1.8270 - \frac{1}{2}\ln N_{\text{eff}}$$

**第四步：Tc计算**

$$T_c(N_{\text{eff}}) = \frac{E_{\text{topo}} \cdot 4.282 \cdot (0.1729)^2 \cdot N_{\text{eff}}}{k_B \cdot \ln\Omega_0(N_{\text{eff}}, 2)}$$

代入k_B = 8.617×10⁻⁵：

$$T_c(N_{\text{eff}})\ [\mathrm{K}] = E_{\text{topo}}\ [\mathrm{meV}] \cdot \frac{N_{\text{eff}} \cdot 4.282 \cdot 0.02989}{8.617\times10^{-5} \cdot (1.8270 - 0.5\ln N_{\text{eff}})}$$

简化：

$$T_c(N_{\text{eff}})\ [\mathrm{K}] = E_{\text{topo}}\ [\mathrm{meV}] \cdot \frac{1486 \cdot N_{\text{eff}}}{1.8270 - 0.5\ln N_{\text{eff}}}$$

对于七螺旋（N_eff = 7）：

$$T_c(7)\ [\mathrm{K}] = E_{\text{topo}}\ [\mathrm{meV}] \times 12.17$$

## 4. Python筛选脚本

以下Python脚本可从公开的拓扑材料数据库自动筛选候选材料并计算Tc预测值。使用Topological Quantum Chemistry Database的在线API和Materials Project数据库。

```python
#!/usr/bin/env python3
"""
拓扑超导材料几何论筛选器 v1.0
基于几何论Tc公式：k_B*T_c = E_topo * g_pair * δ₀²(N) / lnΩ₀(N, d_IR)

用法：
    python3 ti_screener.py --radius 10 --max_chain 7 --output candidates.csv

依赖：
    pip install requests numpy pandas matplotlib
"""

import requests
import numpy as np
import pandas as pd
import argparse
import time
import sys
from typing import Dict, List, Tuple, Optional

# ============================================================
# 几何论普适常数（定理级锁定，不可调整）
# ============================================================
G_PAIR = 4.282           # sinθ_C/sinθ_I，公理1-3导出
DELTA0_SINGLE = 0.1729   # 单链投影收窄量，约束截面谱几何
LAMBDA1_EFF = 391.05     # 有效软模 (rad⁻²)
LAMBDA2_EFF = 59324.3    # 有效硬模 (rad⁻²)
S_E = 137.036            # 全息屏编码常数
SIN3_THETA_M = 0.8479    # sin³θ_M，电子基态
CHI_L = 1.509e-10        # 长度量纲桥因子 (m)
K_B_eV = 8.617333262e-5  # 玻尔兹曼常数 (eV/K)
K_B_meV = 0.08617333262  # 玻尔兹曼常数 (meV/K)


def calc_ln_omega(N_eff: int, d_IR: int = 2) -> float:
    """
    计算lnΩ₀(N_eff, d_IR)
    
    参数:
        N_eff: 有效螺旋链数 (1-7)
        d_IR: 红外截断 (1=体材料, 2=拓扑表面态)
    
    返回:
        lnΩ₀的无量纲值
    """
    ln2 = np.log(2)
    ln_ratio = 0.25 * np.log(LAMBDA1_EFF / LAMBDA2_EFF)
    ln_se = np.log(S_E / (2 * np.pi * d_IR))
    ln_N = -0.5 * np.log(N_eff)
    return ln2 + ln_ratio + ln_se + ln_N


def calc_E_topo(v_F: float, r_phys_nm: float) -> float:
    """
    计算拓扑能标 E_topo
    
    参数:
        v_F: 费米速度 (m/s)
        r_phys_nm: 纳米线物理半径 (nm)
    
    返回:
        E_topo (meV)
    """
    r_phys = r_phys_nm * 1e-9          # nm → m
    r_geo = r_phys / CHI_L              # m → 无量纲几何半径
    eV = (v_F / r_geo) * (LAMBDA1_EFF / S_E**2) * SIN3_THETA_M
    return eV * 1000  # eV → meV


def calc_Tc(v_F: float, r_phys_nm: float, N_eff: int, d_IR: int = 2) -> Tuple[float, float, float]:
    """
    计算转变温度 Tc
    
    参数:
        v_F: 费米速度 (m/s)
        r_phys_nm: 纳米线物理半径 (nm)
        N_eff: 有效螺旋链数
        d_IR: 红外截断
    
    返回:
        (E_topo (meV), lnΩ₀, T_c (K))
    """
    E_topo = calc_E_topo(v_F, r_phys_nm)
    ln_omega = calc_ln_omega(N_eff, d_IR)
    
    numerator = E_topo * G_PAIR * (DELTA0_SINGLE**2) * N_eff
    denominator = ln_omega * K_B_meV
    
    Tc = numerator / denominator
    return E_topo, ln_omega, Tc


# ============================================================
# 数据库接口
# ============================================================

def query_topological_quantum_chemistry(max_results: int = 100) -> pd.DataFrame:
    """
    从Topological Quantum Chemistry Database获取拓扑绝缘体列表
    
    API: https://topologicalquantumchemistry.com/api/
    
    注意：此API可能需要调整
    """
    base_url = "https://topologicalquantumchemistry.com/api/v1/materials"
    params = {
        "topological": "ti",    # 拓扑绝缘体
        "limit": max_results,
        "format": "json"
    }
    
    try:
        response = requests.get(base_url, params=params, timeout=30)
        response.raise_for_status()
        data = response.json()
        return pd.DataFrame(data)
    except Exception as e:
        print(f"[WARN] TQC API查询失败: {e}")
        print("[INFO] 回退到本地内置候选列表")
        return pd.DataFrame()


def query_materials_project(api_key: str = None, max_results: int = 50) -> pd.DataFrame:
    """
    从Materials Project查询拓扑材料
    需要API key: https://next-gen.materialsproject.org/api
    
    参数:
        api_key: Materials Project API key
        max_results: 最大返回条数
    """
    if not api_key:
        print("[WARN] 未提供Materials Project API key，跳过")
        return pd.DataFrame()
    
    base_url = "https://api.materialsproject.org/materials/v1/materials"
    headers = {"X-API-KEY": api_key}
    params = {
        "_fields": "material_id,formula_pretty,band_gap,is_metal,is_topological",
        "limit": max_results
    }
    
    try:
        response = requests.get(base_url, headers=headers, params=params, timeout=30)
        response.raise_for_status()
        data = response.json()
        return pd.DataFrame(data["data"])
    except Exception as e:
        print(f"[WARN] Materials Project API查询失败: {e}")
        return pd.DataFrame()

# ============================================================
# Tc计算引擎
# ============================================================

def calculate_all_Tc(df: pd.DataFrame, r_phys_nm: float = 10.0,
                     N_eff_list: List[int] = [1, 4, 5, 7]) -> pd.DataFrame:
    """
    为候选材料列表计算Tc
    
    参数:
        df: 候选材料DataFrame（必须包含'v_F_ms'列）
        r_phys_nm: 纳米线物理半径 (nm)
        N_eff_list: 需要计算的螺旋链数列表
    
    返回:
        添加了Tc列的DataFrame
    """
    results = []
    
    for _, row in df.iterrows():
        v_F = row['v_F_ms']
        formula = row['formula']
        
        for N in N_eff_list:
            E_topo, ln_omega, Tc = calc_Tc(v_F, r_phys_nm, N)
            
            results.append({
                "formula": formula,
                "family": row.get('family', ''),
                "topo_type": row.get('topo_type', ''),
                "v_F (10⁵ m/s)": v_F / 1e5,
                "r_phys (nm)": r_phys_nm,
                "N_eff": N,
                "E_topo (meV)": round(E_topo, 2),
                "lnΩ₀": round(ln_omega, 4),
                "T_c (K)": round(Tc, 1),
                "T_c (°C)": round(Tc - 273.15, 1),
            })
    
    return pd.DataFrame(results)
```

## 7. 诚实标注与开放问题

### 7.1 理论开放性

1. **Tc公式的普适性**：几何论的Tc公式从Bi₂Se₃推导，对同族材料（Bi₂Te₃、Sb₂Te₃）的适用性高（同晶体结构、同Dirac锥类型）。但对其他族的适用性需验证。
2. **对称性保护类型**：Z₂保护（Bi₂Se₃族）vs 镜面对称保护（SnTe族）——几何论的软模失稳机制是否依赖特定的对称性保护类型？这是一个开放问题。
3. **纳米线合成限制**：不是所有拓扑绝缘体都能生长为多螺旋纳米线。例如半Heusler族尚未有纳米线合成报道。

### 7.2 实验开放性

1. **螺旋链绕制**：多螺旋纳米线需要精确的绞合工艺（N根链均匀分布在圆周上），目前只有Bi₂Se₃有初步的纳米线绞合尝试。
2. **表面态纯度**：材料的体导电性会"短路"表面态超导。Bi₂Te₂Se因体电阻率高（>1 Ω·cm）可能比Bi₂Se₃更适合表面态超导研究。
3. **Fermi能级调谐**：许多拓扑绝缘体（如Sb₂Te₃）的Fermi能级不在表面Dirac点附近，需要通过掺杂或栅压调谐。

### 7.3 诚实残余

1. **Tc公式本身尚待主库验证**：72号文章Tc公式的熵稀释推导是定理级的，但主库验证还在进行中。
2. **TCI的适用性**：SnTe族使用镜面对称保护而非时间反演对称保护——几何论的约束截面软模失稳机制是否需要时间反演保护？这需要额外的理论分析。
3. **v_F取值的精度**：限于内置候选列表的v_F来自不同文献（ARPES或DFT），精度不一。精确筛选需要系统的v_F测量或计算。

## 9. 结论

几何论对拓扑超导的预言**不是孤例**——Bi₂Se₃ 438 K只是冰山一角。同族的Bi₂Te₃（预测346 K）、Sb₂Te₃（276 K）、Bi₂Te₂Se（388 K）为直接验证目标；跨族的SnTe（382 K）、Bi₀.₉Sb₀.₁（475 K）为扩展验证目标。

70号文章指出宇宙学ΛCDM模型有问题；71号文章指出几何论Σm_ν=0.118 eV是刚性的；72号文章封闭了438 K的缺口。**现在，73号文章把438 K从"一个数值"变成了"一张图谱"**——实验物理学家将有系统性工作去做。
