## 第 4 章 三分切丛与全息屏结构

### 4.1 三分切丛分解

**命题 4.1**（三分切丛分解）设 $M(a)=S^3(a)\times S^3(a/\sqrt{\Lambda})\times S^3(a/\sqrt{\Lambda k_0})$ 为约束乘积球面。其切丛自然分解为三个秩 3 子丛的直和：

$$TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I},$$

其中 $\mathcal{M}$、$\mathcal{C}$、$\mathcal{I}$ 分别对应三个球面因子的切丛。该分解在乘积度量下正交，且每个子丛平凡（因 $S^3$ 可平行化）。

*证明.* 乘积流形切丛的标准分解。$\square$

### 4.2 公理 3：全息屏编码条件

**公理 3（全息屏编码条件）** 在三分切丛 $TM(a)=\mathcal{M}\oplus\mathcal{C}\oplus\mathcal{I}$ 的每个纤维上，存在二维定向子空间 $\Sigma$（称为**全息屏**），以及从每个 $\mathbb{R}^3$ 扇区到 $\Sigma$ 的投影映射，使得扇区投影强度角 $\theta_1,\theta_2,\theta_3$（分别对应 $\mathcal{M},\mathcal{C},\mathcal{I}$ 扇区）满足：

$$\theta_1+\theta_2+\theta_3=90^\circ,\qquad \theta_i>0.$$

**注 4.0（Σ 的公理内涵——路径 B 阐明）** 公理 3 所定义的 Σ 是 TM(a) 每个纤维中的一个二维定向线性子空间。在公理层面，Σ **不携带**度规、拓扑、复结构或曲率。它仅具有：
- 维度：2
- 定向：有
- 嵌入：TM(a) 纤维中的子空间

任何超出这些属性的附加结构（度规、拓扑 $S^2$、复结构、Kähler 形式）均不是公理 3 的推论，而是需要进一步论证的**涌现性质**或**理论假设**。区分"公理赋予的结构"与"公理之外附加的假设"是本章及后续所有几何论证的诚实性基础。

### 4.3 全息屏的拓扑识别与度规涌现

本节取代版本 260628.8 中的假设 4.1 与定理 4.1。原假设 4.1 将常曲率 Kähler 结构作为先验条件赋予 Σ；路径 B 将其拆分为两个独立层次。

#### 4.3.1 Σ 的拓扑识别

**假设 4.1a（Σ 的拓扑识别——全局化假设）** 将 TM(a) 上每点纤维中的二维定向子空间 Σ 拼合为 TM(a) 的一个二维分布，且该分布可积（满足 Frobenius 条件），从而 Σ 叶状结构给出一个全局二维子流形。进一步假设该子流形紧致、连通、可定向，且拓扑同胚于 $S^2$（从而全纯等价于 $\mathbb{CP}^1$）。

**注 4.1a（假设 4.1a 的逻辑地位）** 假设 4.1a 包含两个非公理的跳跃：
1. **从纤维到全局**：公理 3 仅在每点纤维中定义了 Σ 为线性子空间。将其拼合为 TM(a) 上的光滑分布并证明可积性，需要额外论证（Frobenius 定理的条件——对合性——需验证）。
2. **从分布到 $S^2$ 拓扑**：即使分布可积给出叶状结构，该叶状结构的叶面（leaf）的拓扑为 $S^2$ 并非自动成立。在对称点，S₃ 对称性可提供论证；在非对称点，需要构造。

因此假设 4.1a 属于**框架的几何输入**，而非公理 3 的推论。它在本文中被赋予"假设"地位，其成立依赖于 $M(a)$ 的乘积球面结构及后续投影构造。

#### 4.3.2 Σ 的度规与复结构——对称点涌现

**定理 4.1（对称点上的度规涌现与 Kähler 识别——路径 B 版）** 在 $\Sigma \cong \mathbb{CP}^1$（由假设 4.1a）的前提下：

**(a) 对称点涌现**：在 $S_3$ 对称点 $\theta_0 = (30^\circ, 30^\circ, 30^\circ)$，三个扇区在几何上完全等价。由 S₃ 对称性，Σ 上的任意张量结构若由三分切丛投影诱导，则必为 SO(3)-不变的。二维定向流形上的 SO(3)-不变 Riemann 度量必为常曲率度量（差尺度因子）。因此，Σ 在对称点自然继承一个常曲率 Riemann 度量 $g_\Sigma$。

**(b) 常曲率条件下的 Kähler 识别**：在常曲率假设下，二维定向流形上可由 $g_\Sigma$ 与定向构造殆复结构 $J$（旋转 $90^\circ$），且二维情形下 $J$ 自动可积（Nijenhuis 张量恒为零）。定义 Kähler 形式 $\omega(X,Y) = g_\Sigma(X, JY)$，则 $(g_\Sigma, J, \omega)$ 构成 Kähler 三元组，且 $d\omega = 0$。

**(c) 整体归一化下的唯一性**：在整体归一化条件 $\frac{1}{2\pi}\int_\Sigma \omega = 1$ 下，紧致单连通二维常曲率 Kähler 流形上的 Kähler 度量唯一为 Fubini-Study 度量（由均匀化定理）。

**(d) 非对称点的诚实声明**：在非对称点 $\theta \neq \theta_0$，三个扇区半径不等（$r_1 : r_2 : r_3 = 1 : 1/\sqrt{3} : 1/\sqrt{6}$），S₃ 对称性破缺。定理 4.1(a) 的 SO(3)-不变论证失效。此时 Σ 上的有效度规是否为常曲率（乃至是否为 FS 度量）是框架的**开放问题**。当前尚无非对称点上 Σ 度规的严格构造。定理 4.1(b)(c) 仅在对称点（或对称点的小邻域内以微扰形式）严格成立。

*证明.* (a) 对称点处 θ₁ = θ₂ = θ₃，三个 S³ 因子半径相等。S₃ 可迁地作用于三个扇区标签。投影结构由公理 3 定义，S₃ 对称性迫使 Σ 上的诱导结构在 SO(3) 作用下不变。二维 + SO(3) 不变 → 常曲率。(b) 二维定向流形上 $J$ 由度规和定向唯一确定，二维时 Nijenhuis 张量恒为零。(c) 均匀化定理的直接推论。(d) 见声明。$\square$

**注 4.1b（定理 4.1 与版本 260628.8 的差异）** 版本 260628.8 将常曲率作为假设 4.1 的先验条件，定理 4.1 在该假设下成立（逻辑上无误，但假设本身的来源未论证）。路径 B 版将常曲率解释为对称点涌现性质——从公理 3 的裸 Σ + S₃ 对称性推导常曲率，消除"常曲率从何而来"的疑问。在非对称点，常曲率的成立性保持开放，而非默认成立。这是框架诚实性的提升。

### 4.8 切丛平方根分解与幻数序列的代数来源

本节取代版本 260628.8 的命题 4.6–4.7。核心变更：将幻数序列的严格性从依赖 FS 度规改为仅依赖 $\mathbb{CP}^1$ 的复结构。

**命题 4.6**（核线丛的复结构与陈类——路径 B 诚实版）设 $\Sigma \cong \mathbb{CP}^1$（由假设 4.1a），其全纯切丛满足标准分解：

$$T\Sigma \cong \mathcal{O}(2) \cong \mathcal{O}(1) \otimes \mathcal{O}(1).$$

由三分切丛投影结构（命题 4.1），在 $S_3$ 对称点 $\theta_0 = (30^\circ, 30^\circ, 30^\circ)$，存在两个核线丛 $L_{\mathcal{M}}, L_{\mathcal{C}} \subset N\Sigma \otimes \mathbb{C}$（分别来自 $\mathcal{M}$ 与 $\mathcal{C}$ 扇区投影的核），满足：

$$L_{\mathcal{M}} \cong L_{\mathcal{C}} \cong \mathcal{O}(1),$$

且其张量积自然同构于 $T\Sigma$：

$$L_{\mathcal{M}} \otimes L_{\mathcal{C}} \cong T\Sigma.$$

*证明与诚实标注.* 分三层：

**严格层（不依赖 FS 度规）**：
- $\Sigma \cong \mathbb{CP}^1$ 的全纯切丛为 $\mathcal{O}(2)$，且 $\mathcal{O}(2) = \mathcal{O}(1)^{\otimes 2}$。这是 $\mathbb{CP}^1$ 作为复流形的基本事实，**不依赖** Σ 上的具体 Kähler 度量——任何与 $\mathbb{CP}^1$ 标准复结构相容的 Kähler 度量都给出相同的全纯线丛同构类。
- 由 Riemann–Roch，$\dim H^0(\mathbb{CP}^1, \mathcal{O}(k)) = k+1$ 同样仅依赖复结构。

**半严格层（依赖对称点 + 假设 4.1' 的几何输入）**：
- 核线丛 $L_{\mathcal{M}}, L_{\mathcal{C}}$ 的复化为全纯线丛且第一陈类 $c_1 = 1$（几何假设 4.1'）——该假设的成立依赖于投影映射的显式构造。在对称点，附录 D 的典范 Hopf 构造（定义 D.8.2）给出严格支持；在非对称点，构造尚未完成（见注 4.4''）。

**开放层**：
- 命题 4.6 的结论（$L_{\mathcal{M}} \cong L_{\mathcal{C}} \cong \mathcal{O}(1)$）在对称点可借助 C2 旋量丛构造（附录 D.7）获得严格证明；在非对称点，它是开放问题。

$\square$

**命题 4.7**（幻数定理——路径 B 诚实版）接受命题 4.6 的分解（对称点严格，非对称点待证），幻数序列由以下推理确定：

两个核线丛 $L_{\mathcal{M}} \cong L_{\mathcal{C}} \cong \mathcal{O}(1)$ 对应三分切丛两个独立扇区向全息屏的投影。其量子化态空间为两个独立 $\mathcal{O}(k)$ 线丛全纯截面空间的张量积：

$$\mathcal{H}_k^{(\text{spatial})} := H^0(\Sigma, L_{\mathcal{M}}^{\otimes k}) \otimes H^0(\Sigma, L_{\mathcal{C}}^{\otimes k}).$$

由 Riemann–Roch，$\dim H^0(\Sigma, \mathcal{O}(k)) = k+1$，故

$$\dim \mathcal{H}_k^{(\text{spatial})} = (k+1)^2.$$

引入自旋 $1/2$ 自由度（假设），总 Hilbert 空间维数为

$$\dim \mathcal{H}_k^{(\text{total})} = 2(k+1)^2.$$

从而幻数序列 $2, 8, 18, 32, 50, 72, 98$ 由 $\Sigma \cong \mathbb{CP}^1$ 的复结构与三分切丛双扇区独立投影结构（结合自旋假设）确定。

**关键诚实声明（路径 B 新增）**：
- 幻数序列的代数推导仅依赖 $\Sigma \cong \mathbb{CP}^1$ 的**复结构**，不依赖 Σ 上的具体 Kähler 度量（FS 或其他）。这意味着：即使非对称点上 Σ 的有效度规偏离 FS 度量，只要其复结构仍为 $\mathbb{CP}^1$ 的标准复结构，幻数序列 $2(k+1)^2$ 仍然严格成立。
- 定理 4.1 中 FS 度规唯一性的成立范围（对称点严格，非对称点开放）**不影响**命题 4.7 的幻数序列——两者的逻辑依赖关系是解耦的：幻数 ← 复结构 ← 拓扑 $S^2$ ← 假设 4.1a；FS 度规 ← 常曲率 ← S₃ 对称性 ← 对称点。
- 命题 4.7 在对称点具有严格定理地位（由附录 D 的 C2 构造支撑）；在非对称点，它降级为"假设 4.1a + 几何假设 4.1' 下的推论"，其成立性最终依赖桥 2（非对称点投影构造）的闭合。

### D.8 典范 Hopf 投影构造（路径 B 兼容版）

#### D.8.1 路径 B 下的构造框架

本节将典范 Hopf 构造置于路径 B 的框架中。核心变更：Σ 不携带先验 FS 度规。典范 Hopf 投影 $d\pi_i: S^3(r_i) \to \mathbb{CP}^1_i$ 给出三个独立的 $\mathbb{CP}^1_i$（作为光滑流形均微分同胚于 $S^2$）。通过微分同胚 $\varphi_i: \mathbb{CP}^1_i \to \Sigma$ 将投影推前到 Σ 上。

**定义 D.8.2**（典范 Hopf 投影——路径 B 版）设 $k_i \in \mathfrak{su}(2)$ 为右不变向量场（右 U(1) 作用的生成元），$H_i = k_i^\perp$ 为 Hopf 联络的水平分布（典范确定），$d\pi_i: TS^3(r_i) \to T\mathbb{CP}^1_i$ 为标准 Hopf 商投影的微分。定义投影 $P_i$ 为：

$$P_i(v) = \sin\theta_i \cdot (\varphi_i)_* \circ d\pi_i\big(v - \langle v, k_i\rangle k_i\big),$$

其中 $\varphi_i: \mathbb{CP}^1_i \to \Sigma$ 为光滑微分同胚，$(\varphi_i)_*$ 为其推前映射。

**成分分析（路径 B 版）**：

| 成分 | 典范性 | 路径 B 下的状态 |
|---|---|---|
| $k_i$（核方向） | ✅ 典范 | 右 U(1) 生成元，群论确定，零自由参数 |
| $H_i$（水平分布） | ✅ 典范 | Hopf 联络典范确定 |
| $d\pi_i$（水平→$T\mathbb{CP}^1_i$） | ✅ 典范 | Hopf 商投影典范确定 |
| $\sin\theta_i$ 因子 | ✅ 典范 | 由 $\theta_i$ 公理确定 |
| $\varphi_i$（$\mathbb{CP}^1_i \to \Sigma$） | ⚠️ 非典范 | 微分同胚选取需额外几何原则 |

**注 D.8.2a（$\varphi_i$ 的非典范性与路径 B 的关系）** 在路径 A（Σ 携带先验 FS 度规）下，$\varphi_i$ 必须为等距映射——这在非对称点是不可能的（三个 $\mathbb{CP}^1_i$ 的 FS 曲率不同，无法等距嵌入同一个 FS-Σ）。路径 B 消除了这一矛盾：Σ 无先验度规，$\varphi_i$ 仅需为光滑微分同胚，存在性无阻碍。$\varphi_i$ 的选取自由度是 Diff(S²) 族的维度，但非唯一性不是理论的缺陷——它反映了非对称点上几何结构的丰富性。双扇区耦合条件 $\text{Area}(P_i \times P_j) = \sin\theta_i \sin\theta_j dA$ 可部分约束 $\varphi_i$ 的相对选取。

#### D.8.3 构造的严格性与残存问题

**(a) 已确立的部分**（严格定理层面）：
- $k_i, H_i, d\pi_i$ 的构造零自由参数，在所有点（对称+非对称）严格成立
- $\sin\theta_i$ 因子由公理 3 的角度约束唯一确定
- C2（$\mathcal{O}(1)$ 来源 = 旋量丛 $S_-^*$）在每个 S³ 因子上独立成立，不依赖 $\varphi_i$

**(b) 开放问题**（待验证/待补充）：
- $\varphi_i$ 的选取原则：需要几何条件（如双扇区耦合的面积公式）固定 $\varphi_i$ 的相对选取
- 全局化：$P_i$ 当前定义为点态构造，其在 $M(a)$ 上的全局光滑性需验证
- 双扇区耦合的精确公式：$\text{Area}(P_i \times P_j)$ 在典范近复结构配对下的显式计算

#### D.8.4 路径 B 的兼容性总结

| | 路径 A（Σ 先验 FS 度规） | 路径 B（Σ 裸 S²，度规涌现） |
|---|---|---|
| **对称点** | ✅ 严格 | ✅ 严格（FS 涌现为定理 4.1） |
| **非对称点 $P_i$ 存在性** | ❌ 硬矛盾（曲率不匹配） | ✅ 存在（$\varphi_i$ 存在） |
| **非对称点 $P_i$ 唯一性** | ❌ 不可能 | ⚠️ $\varphi_i$ 待固定 |
| **C2（O(1) 来源）** | ✅ 严格 | ✅ 严格（与路径无关） |
| **幻数序列** | ✅ 对称点 / ❓ 非对称点 | ✅ 对称点 / ⚠️ 非对称点（待桥 2 闭合） |
| **FS 度规地位** | 先验假设 | 对称点涌现定理 |

**结论**：路径 B 是唯一与三个 S³ 半径不等这一几何事实兼容的框架。典范 Hopf 构造在路径 B 下存在且基本唯一（$\varphi_i$ 的剩余自由度有物理意义），而在路径 A 下在非对称点遭遇硬矛盾。D.8 的构造与路径 B 完全兼容。
