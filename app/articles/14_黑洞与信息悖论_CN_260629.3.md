# 摘　要

本文在10方几何空间三公理体系（0.0.7）及信息场冻结定理（0.4）的基础上，推导黑洞的几何构型、视界半径、温度公式、辐射谱框架及信息守恒定理。核心推导不使用牛顿引力常数 $G$，不使用爱因斯坦场方程，不使用热力学第二定律，不使用量子场论外部假设。

**框架基础（九素互扼）**：所有结构常数由九素互扼锁定——三公理（圆拓扑、边界极限、全息屏编码）+ 三互锁常数（$\Lambda=3$, $k_0=2$, $\ell_0$）+ 三工具层（量子化层、谱刚性层、离散逼近层）构成六互扼环节的自举封闭链条。任何偏差超过万分之一均被谱刚性定理排除（0.0.7 §8）。

**单一核心映射层**：电磁相互作用由三分切丛 $\mathcal{M}$-$\mathcal{C}$ 腰边耦合结构唯一确定（0.1声明2.1），无第二独立映射。$\mathcal{I}$ 扇区不映射独立粒子，仅作为信息/相位通道。引力在几何论中的定位为开放问题（0.3.5 §4.4、§6 #5），本文不使用 $G$。

**闭环状态诚实声明**：几何论的结构常数（$S_e$, $\lambda_1^{\text{eff}}$, $\lambda_2^{\text{eff}}$, $v_p$, $v_{\text{geo}}$, $\chi_L$, $\chi_T$, $\ell_P^{\text{geo}}$, $K$, $k$）均由0.X.X内生导出。**重子数 $N$ 不是框架内生量**——$N = M_{\text{obs}}/m_p$（$M_{\text{obs}}$ 为天体物理观测质量，$m_p = k \cdot S_p$ 为几何论内生质子质量），取太阳质量示例 $N = 1.189 \times 10^{57}$。以下结构关系不依赖 $N$ 的具体数值：$\theta_M = \theta_C = 45^\circ$（冻结极限构型）、$R_{\text{BH}} \propto N^{1/3}$、$T_{\text{BH}} \propto N^{-1/3}$、$S_{\text{geo}} \propto N$、七条辐射谱线位置（主导阶）。**所有数值示例以 $N = M_\odot/m_p$ 为演示**——框架自身不预测特定天体应有多少核子，仅预测一旦 $N$ 已知（从观测质量除以几何导出 $m_p$ 得到），所有可观测量即由几何关系唯一确定。电磁几何映射 $\mathcal{E}$ 为唯一锚定映射，$c$ 为其传播模零锥结构固定点。

# 一、黑洞的几何构型

## 1.1　信息冻结相：$\theta_I \to 0^+$

基础篇 0.1 将粒子按物质场角 $\theta_M$ 分为展开相（$\theta_M > 0.5^\circ$）和倒伏相（$\theta_M \ll 1^\circ$）。黑洞的初始直觉可能将其归为倒伏相极限，但这与黑洞的巨大质量矛盾——由公理3（$m = K \sin^3\theta_M$，0.1导出项2.2.2）及互锁定理（0.3.1），$\theta_M \to 0$ 时质量急剧收缩至 eV 量级，对应中微子而非黑洞。

黑洞的真正特征是信息场的冻结。信息场角 $\theta_I$ 表征信息子空间 $I^3$ 与全局背景的倾斜程度。$\theta_I$ 越小，信息场越接近正交，其内部编码越局域化，与外部空间的耦合越弱。

**【黑洞构型定理】**　在10方几何空间三公理体系下，黑洞是信息场冻结极限 $\theta_I \to 0^+$ 的稳定构型。

当 $\theta_I \to 0^+$ 时，信息场单场项 $I = 1/\sin^2\theta_I \to \infty$，信息耦合项 $W_{MI} = 1/(\sin\theta_M \sin\theta_I) \to \infty$，$W_{CI} = 1/(\sin\theta_C \sin\theta_I) \to \infty$。信息被锁死在信息场内部，无法通过有限能量过程传递到物质场或因果场。

## 1.2　$\theta_M = \theta_C = 45^\circ$ 的极小值定理

由三公理（0.0.7）之全息屏编码条件 $\theta_M + \theta_C + \theta_I = 90^\circ$，当 $\theta_I \to 0^+$ 时：

$$\theta_M + \theta_C = 90^\circ$$

作用量 $S$ 的有效有限部分（不含发散的信息场项）为：

$$S_{\text{eff}} = \frac{1}{\sin^2\theta_M} + \frac{1}{\sin^2\theta_C} + \frac{1}{\sin\theta_M \sin\theta_C}$$

代入 $\theta_C = 90^\circ - \theta_M$，$\sin\theta_C = \cos\theta_M$：

$$S_{\text{eff}}(\theta_M) = \frac{1}{\sin^2\theta_M} + \frac{1}{\cos^2\theta_M} + \frac{1}{\sin\theta_M \cos\theta_M}$$

求导并令导数为零：

$$\frac{\mathrm{d}S_{\text{eff}}}{\mathrm{d}\theta_M} = -\frac{2\cos\theta_M}{\sin^3\theta_M} + \frac{2\sin\theta_M}{\cos^3\theta_M} + \frac{\sin^2\theta_M - \cos^2\theta_M}{\sin^2\theta_M \cos^2\theta_M} = 0$$

在 $\theta_M = 45^\circ$ 处：$\sin 45^\circ = \cos 45^\circ = \sqrt{2}/2$，三项分别为 $-2(\sqrt{2}/2)/(\sqrt{2}/2)^3 + 2(\sqrt{2}/2)/(\sqrt{2}/2)^3 + 0 = 0$。二阶导数 $\mathrm{d}^2S_{\text{eff}}/\mathrm{d}\theta_M^2|_{45^\circ} = 16 > 0$，因此 $\theta_M = 45^\circ$ 是极小值点。由对称性 $\theta_C = 45^\circ$。

进一步验证总作用量（含发散项）在 $\theta_I = \varepsilon \to 0^+$ 时的行为：
$$S = \frac{1}{\varepsilon^2} + \frac{1}{\varepsilon}\left(\frac{1}{\sin\theta_M} + \frac{1}{\cos\theta_M}\right) + S_{\text{eff}}(\theta_M)$$
其一阶导数在 $45^\circ$ 处因对称性严格为零，二阶导数含 $\varepsilon^{-1}$ 正发散项，故 $45^\circ$ 是总作用量在冻结极限下的严格极小值点。

**【黑洞几何约束构型定理】**　在信息场冻结极限下，物质场角和因果场角为 $\theta_M = \theta_C = 45^\circ$。该构型使电磁耦合 $\alpha = 1/S \to 0$，黑洞内部无电磁相互作用。九素互扼（0.0.7 §8）保证 $\Lambda=3$ 的层级标度在此极限下不发生漂移。

## 1.3　黑洞构型的作用量层级

黑洞构型的总作用量 $S$ 在 $\theta_I \to 0^+$ 时发散。物理黑洞不是数学奇点，而是 $\theta_I$ 极小但非零的极端构型。设黑洞的信息场角为 $\theta_I^{\text{BH}} = \varepsilon$，$\varepsilon \to 0^+$，则：

$$S_{\text{BH}} = \frac{1}{\varepsilon^2} + \frac{2\sqrt{2}}{\varepsilon} + 6$$

主导项为 $1/\varepsilon^2$。黑洞的巨大质量来自信息场冻结激发出的无穷大几何刚度，而非物质场投影。由公理3（$m = K \sin^3\theta_M$，0.1导出项2.2.2）及其等价形式质量映射统一式 $m = k \cdot S$（$k = m_e/S_e = 3.728940\ \text{keV}/c^2$，由电子基态零锚点定标，0.1；等价性证明见§4.2）：

$$M_{\text{BH}} c^2 = k \cdot S_{\text{BH}} \approx \frac{k}{\varepsilon^2}$$

反解信息场角（一致使用能量单位）：

$$\varepsilon = \sqrt{\frac{k}{M_{\text{BH}} c^2}}$$

取重子数 $N = 1.189 \times 10^{57}$（$N = M_\odot/m_p$，$M_\odot = 1.989 \times 10^{30}$ kg 为太阳质量观测值，$m_p = 1.673 \times 10^{-27}$ kg 为几何论导出质子质量，详见§4.2），由多体作用量可加性定理（第4章）得黑洞总作用量 $S_{\text{joint}} = N \cdot S_p = 2.992 \times 10^{62}$，质量：

$$M_{\text{BH}} = k \cdot S_{\text{joint}} = 3.728940 \times 10^{-6}\ \text{MeV} \times 2.992 \times 10^{62} = 1.989 \times 10^{30}\ \text{kg}$$

该值恢复太阳质量——不构成独立预言，而是理论自洽性检验（$N=M_\odot/m_p$ 代入后 $M_{\text{BH}} = k \cdot (M_\odot/m_p) \cdot S_p = M_\odot$）。反解信息场角：

$$\varepsilon = \sqrt{\frac{3.728940}{1.1157 \times 10^{63}}} = 5.781 \times 10^{-32}\ \text{rad} = 3.312 \times 10^{-30\circ}$$

该角度极小，在物理上可视为零，在数学上保持正数以避免奇异性。
