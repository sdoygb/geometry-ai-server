# 第一章：公理3与流形结构

## §1.1 公理3

**来源：0.0.6 公理3。** 在 CPS 类 M(a) = S³(a) × S³(a/√3) × S³(a/√6) 上，切丛存在直和分解：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

三个子丛的秩均为 3（在 0.0.6 的九维设定中），或均为 1（在本文的三维限制中——详见 §1.3）。

## §1.2 三维限制

在本文中，我们将 M(a) 限制为 Material 扇区的 S³ 因子 S³_ℳ = S³(a)。在此限制下，ℳ、𝒞、ℐ 均为 TS³_ℳ 上的秩-1 子丛。

$$\dim(\mathcal{M}) = \dim(\mathcal{C}) = \dim(\mathcal{I}) = 1$$

$$\mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I} = TS^3_\mathcal{M}$$

## §1.3 M(a) 的双重含义澄清（诚实标注）

**本文的 M(a) 与 0.0.6 的 M(a) 是不同的数学对象：**

| | 0.0.6 | 本文 |
|:---|:---|:---|

本文对公理3的使用是将其限制到 ℳ 扇区 S³ 的三维切丛分解。该限制在数学上是合法的（公理3的九维直和分解限制到 ℳ 扇区 S³ 的切丛给出三维直和分解），但全文分析在此三维设定中进行。与 0.0.6 的链接需通过九维嵌入（第四章路线三）。

**定理4.2（Euler类=±1）** S³_ℳ → S² 的 Euler 类为 ±1。

*证明（Frobenius路径）* S³上S¹-纤维化按Euler类分类：总空间为S³的S¹-纤维化的Euler类必为±1（e=0给出S²×S¹≠S³；|e|>1给出透镜空间L(|e|,1)≠S³）。S³_ℳ微分同胚于S³，故e=±1。□
