# 第一章：公理3与流形结构

**来源：0.0.6 公理3。** 在 CPS 类 M(a) = S³(a) × S³(a/√3) × S³(a/√6) 上，切丛存在直和分解：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

三个子丛的秩均为 3（在 0.0.6 的九维设定中），或均为 1（在本文的三维限制中——详见 §1.3）。

## §1.2 三维限制

在本文中，我们将 M(a) 限制为 Material 扇区的 S³ 因子 S³_ℳ = S³(a)。在此限制下，ℳ、𝒞、ℐ 均为 TS³_ℳ 上的秩-1 子丛。

$$\dim(\mathcal{M}) = \dim(\mathcal{C}) = \dim(\mathcal{I}) = 1$$

$$\mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I} = TS^3_\mathcal{M}$$

## §1.3 M(a) 的双重含义澄清（诚实标注）

# 第四章：Hopf 纤维化的严格推导

## §4.1 拓扑背景

**Hopf 纤维化：** S³ → S²，纤维 = S¹。Euler 类 e ∈ H²(S²,ℤ) ≅ ℤ，分类为 e = ±1 给出 S³。

## §4.2 θ_M 的适用范围（G5闭合）

**定理4.1（全θ_M范围有效性）** 第四章的拓扑结论不依赖 θ_M 的特定值。

**论证（G5已闭合）**

1. ℳ 作为公理3定义的向量丛直和分解中的子丛，其丛同构类不依赖 TS³_ℳ 上的内积选择（向量丛是微分拓扑对象，内积是附加结构）。
2. Frobenius可积性仅依赖Lie括号，而Lie括号是微分结构的内禀运算，不依赖度量——因此叶状结构 ℱ_ℳ 的拓扑类型在θ_M连续变化下不变。
3. 左不变性来自CPS类的SU(2)刚性：三个S³因子的SU(2)李群结构是CPS类定义的一部分（0.0.5定义3.1），不随θ_M变化。
4. 商映射 π: S³ → S² 是 S¹-纤维化——由Ehresmann定理，紧致流形间的淹没若纤维连通则为局部平凡纤维化。叶空间S²的紧致性（S³紧致→商空间紧致；二维淹没的商必为紧致曲面）和纤维S¹的连通性（左不变向量场的积分曲线微分同胚于S¹）保证此点。
5. S³上S¹-纤维化的分类：由拓扑刚性，仅Euler类 e=±1 的S¹-纤维化总空间微分同胚于S³（其他Euler类给出透镜空间L(n,1)，n>1）。
6. 因此Euler类=±1的结论在θ_M=57.93°处同样成立——此结论不依赖ℳ方向向量场是否闭合（那是用θ_M=30°的旧论证路径，已被Frobenius路径取代）。

**结论：** 拓扑Hopf纤维化 S³ → S² 对所有θ_M值成立。物理锁定点θ_M=57.93°有效。θ_M=30°仅是方便计算的工作点，不是逻辑前提。

**状态：定理。**

## §4.3 Euler 类 = ±1

**定理4.2（Euler类=±1）** S³_ℳ → S² 的 Euler 类为 ±1。

*证明（Frobenius路径）* S³上S¹-纤维化按Euler类分类：总空间为S³的S¹-纤维化的Euler类必为±1（e=0给出S²×S¹≠S³；|e|>1给出透镜空间L(|e|,1)≠S³）。S³_ℳ微分同胚于S³，故e=±1。□

**状态：定理（全θ_M范围）。**
