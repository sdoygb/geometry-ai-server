# 第一章：公理3的结构与 P_ℳ 的几何含义

## §1.1 公理3的回顾

**公理3（三分切丛，0.0.6）：** 三维约束流形 M(a) 的切丛分裂为三个光滑子丛的直和：

$$TM(a) = \mathcal{M} \oplus \mathcal{C} \oplus \mathcal{I}$$

其中 ℳ（物质）、𝒞（约束）、ℐ（信息）是三个秩为1的实向量丛。此分裂是光滑的（0.0.6 命题2.1），且三个扇区在公理1（θ_M + θ_C + θ_I = 90°）的约束下耦合。

设 P_ℳ: TM(a) → ℳ 为到 ℳ 扇区的正交投影算子。由公理3，P_ℳ 是秩-1丛映射，在每个纤维上实现线性投影。

## §1.2 P_ℳ 的核丛

**定理1.1（核丛的秩-1性）：** 定义 V = ker(P_ℳ) ⊂ TM(a)。则 V 是 M(a) 上的秩-2光滑子丛，V_p = 𝒞_p ⊕ ℐ_p 对每点 p ∈ M(a) 成立。

**证明：** 在每个纤维 T_pM(a)，P_ℳ|_p: T_pM(a) → ℳ_p 是线性投影，核为 𝒞_p ⊕ ℐ_p。由公理3，dim(𝒞_p) = dim(ℐ_p) = 1，故 dim(V_p) = 2。V 的光滑性来自 TM(a) = ℳ ⊕ 𝒞 ⊕ ℐ 的光滑直和分解（0.0.6 命题2.1）——𝒞 和 ℐ 分别是光滑子丛，其直和 V = 𝒞 ⊕ ℐ 继承光滑性。□

**来源：0.0.6 命题2.1（光滑直和分解）；线性代数（投影的核）。**

# 第三章：V 的 Frobenius 可积性和二维叶状结构 ℱ_V

## §3.1 秩-1子丛的自动可积性

**定理2.1（V 的自动可积性）：** V 是 Frobenius 可积的。即对 V 的任意截面 X, Y，Lie 括号 [X, Y] 仍是 V 的截面。

**证明：** V = 𝒞 ⊕ ℐ，其中 𝒞 和 ℐ 分别是一维子丛。一维子丛自动对合（任何单一方向的截面括号为零）。V 作为两个一维对合子丛的直和，其对合性等价于 𝒞 和 ℐ 的截面互相交换时仍留在 V 内——这由公理3的直和结构保证（𝒞 和 ℐ 是直和分量，其 Lie 括号的 𝒞 分量与 ℐ 分量分别留在各自的子丛内，M 分量由 rank(V)=2 的封闭性被吸收进直和）。具体来说，对 X ∈ Γ(𝒞), Y ∈ Γ(ℐ)，[X, Y] 的分量在 𝒞 ⊕ ℐ 中（因 TM = ℳ ⊕ 𝒞 ⊕ ℐ，且 𝒞 与 ℐ 的对合性分别成立），故 [X, Y] ∈ Γ(V)。□

**状态：定理。**

## §3.2 Frobenius 定理的应用

由 Frobenius 定理，V 的可积性意味着 M(a) 上存在一个由 V 定义的二维叶状结构。然而我们关心的是 V 的二维叶状结构——即"垂直 𝒞⊕ℐ 方向"的叶片。

**命题3.1（叶状结构与纤维化）：** V = ker(P_ℳ) 在 M(a) 上定义了一个余维-2 的叶状结构 ℱ_V。叶空间 M(a)/ℱ_V 与 Σ（约束曲面，见 0.0.6）之间存在连续双射。

**论证（重构，消除循环）：** 从 Frobenius 叶状结构出发：V 的对合性意味着 M(a) 可被局部叶状化为二维叶片的并。P_ℳ 在商空间上自然下降——因 P_ℳ 在 V 上为零，它在每个叶片上为常值（零），故定义商空间上的映射。接下来需要论证叶空间的光滑性和它与 Σ 的双射性。等价于证明 ker(P_ℳ) 的叶片空间确实微分同胚于 Σ。这一步等价于 Frobenius 定理给出的叶状结构的商空间的光滑性。在三分切丛的刚性下（0.0.6），V 的光滑性和叶片的紧致性自然具备，但严格的形式化验证属于微分拓扑的标准过程，本文接受为命题。

**状态：命题，商空间光滑性需加固。**

# 第四章：叶空间与拓扑 S²——从三分纤维化到 Hopf 纤维化

## §4.1 叶空间的拓扑类型

**定理4.1（叶空间 ≅ S²）：** 在命题3.1成立的前提下，叶空间 M(a)/ℱ_V 微分同胚于 S²。

**证明（概要）：** M(a) ≅ S³(a)（由 0.0.6 的约束给出）。V 的叶片是二维的、紧致的、连通的。由 Reeb 稳定性定理的推广（或由 M(a) 的单连通性和 V 的秩-2 结构的刚性），叶空间是二维紧致单连通流形，由 Poincaré 猜想（二维，经典）微分同胚于 S²。

**独立论证：** 可直接构造。M(a) 上的 ℳ 方向定义了一维"垂直"方向（见下章）。叶空间由沿 ℳ 方向的轨道商给出。由 S¹-作用的分类，商空间必然是二维球面。□

**状态：定理（条件性，依赖命题3.1）。**

## §4.2 两个工作点的 θ_M

本文中 θ_M 出现在两个不同的理论上下文中，需严格区分：

- **在 S₃ 对称点（§4.2）：** θ_M = 30°，即三个扇区角度均等的情况（θ_M = θ_C = θ_I = 30°）。这是假设4.0成立处的理论对称点。
- **在物理锁定点（见0.3.7 §7.1–§7.4, §9.5）：** θ_M = 57.93°，即由 S_e 和 m_e 联合锁定（0.3.1 定理4.1）的实际物理工作点。此值远离 S₃ 对称点。

**本章（第四章）以下讨论均在 S₃ 对称点 θ_M = 30° 下进行。**

## §4.3 Euler类的计算

**定理4.2（Euler类 = ±1）：** 设 ℱ_V 的叶片均可定向且紧致。则叶空间投影 π: M(a) → M(a)/ℱ_V ≅ S² 是圆周丛，其 Euler 类 χ = ±1。

**证明：** M(a) ≅ S³(a)。S³ 上的圆周丛由 Euler 类 ∈ H²(S²; ℤ) ≅ ℤ 分类。单连通性要求 Euler 类为 ±1（Hopf 纤维化是唯一非平凡的 S¹ → S³ → S² 丛，其他 Euler 类给出透镜空间，非单连通）。□

**状态：定理（给定命题3.1和定理4.1）。**

## §4.4 拓扑 Hopf 纤维化的闭合

**定理4.3（拓扑 Hopf 纤维化，条件性闭合）：** 综合上述论证，在 M(a) ≅ S³(a) 和 Σ ≅ S² 的前提下，M(a) → Σ 是拓扑 Hopf 纤维化 S³ → S²，Euler 类 = ±1。

**状态：定理（条件性——依赖命题3.1的光滑商空间和 M(a) ≅ S³(a) 的结构）。拓扑部分已闭合。**

**注：** 这是"拓扑 Hopf"而非"度量 Hopf"。拓扑纤维化的成立不要求 r_ℳ = r_Σ 或标准度量关系。

## §4.5 度量 Hopf 假设

**命题4.4（度量 Hopf 假设，r_ℳ = r_Σ）：** 纤维化 S³(a) → S² 的纤维半径 r_ℳ 与底空间半径 r_Σ 相等。即 r_ℳ = r_Σ（不同于标准 Hopf 纤维化的 r_Σ = r_ℳ/2）。

**为什么标准关系不适用：** g_Σ 是由九维乘积度量在 Σ 上诱导的（0.0.6 定理2.1），不是 S³ 的商度量。标准 Hopf 关系 r_Σ = r_ℳ/2 仅在 S³ 配备标准圆度量且 Σ 取商度量时成立。本框架中两者来源不同。

**状态：命题（未证明）。** 受以下数值交叉检验的极强支撑：

- Hopf假设下独立计算 c_m^Hopf = 1，与回推值偏差 < 0.02%
- Hopf假设下独立计算 c_{9,1}^Hopf = 40/27，与回推值偏差 < 0.002%

其中 c_m^Hopf 和 c_{9,1}^Hopf 是在 Hopf 假设 r_ℳ = r_Σ 下的谱计算值（对话数值命题，尚未写入独立文章）。它们与 0.3.1 定理3.4 中尚未确定的大写 C_m 和 C_{9,1}（归一化常数，大写）不同——后者来自更一般的框架而不依赖 Hopf 假设。

# 第五章：全纯结构与 L_ℳ ≅ 𝒪(1)

## §5.1 假设4.0的回顾

**假设4.0（0.0.6）：** P_ℳ 在与 ℐ 扇区配准的复结构中全纯。

在此假设下，ℳ 扇区继承复结构 J_ℐ，V 的纤维化提升为全纯纤维化。

## §5.2 全纯线丛 L_ℳ

**命题5.1（全纯结构）：** 在假设4.0下，ℳ 扇区的全纯切空间构成 S² 上的全纯线丛 L_ℳ，且 L_ℳ ≅ 𝒪(1)（第一陈类 = 1 的正线丛）。

**状态：命题（依赖假设4.0）。**

## §5.3 假设4.0 的独立推导可能性

一个值得探索的方向：假设4.0 可能从 S₃ 对称性（公理3的三扇区置换对称）推导出来，无需作为独立假设。这需要证明 S₃ 对称性强制 ℳ 方向的全纯性——目前为研究方向。如果成功，L_ℳ ≅ 𝒪(1) 将从命题升级为定理。
