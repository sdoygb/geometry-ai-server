## 摘要

本文建立"缺陷裂解"整数分解方案的数学框架——不搜索、不找周期、不让约束截面上的信息场执行模幂计算，而是**将合数N编码为约束截面Σ上的复合缺陷构型，让信息场图热扩散（0.4.4第一层）自然驱动缺陷裂解为不可约子缺陷，直接从子缺陷的拓扑特征读出因子**。

核心贡献：

1. **缺陷的形式化定义**（第1章）：在既有框架内，将缺陷定义为约束截面上的信息场涡旋构型——以绕数（winding number）为其拓扑不变量。不可约缺陷（绕数不可分解）对应素数，复合缺陷（绕数可分解为乘积）对应合数。此定义衔接3号文章的二能级定理（纠缠对低能激发空间=2维）和0.2.3的弯曲结构量子化。

2. **编码桥的三条候选构造**（第2章）：提出三种从整数N到约束截面缺陷构型的候选编码方案——角度对数编码（利用Σ的S¹参数化）、谱源编码（利用球谐基的正交投影）、几何层级编码（利用七层Bott截断结构）。每条方案形式化其构造、同态性条件和已知障碍。

3. **裂解动力学判据**（第3章）：基于0.4.4第一/二层动力学方程，证明一旦编码桥建立，复合缺陷在图热扩散下的演化规律可由既有的图拉普拉斯半群严格描述。裂解的信息自由能判据由0.4.1的Γ_geo给出。关键待证命题：合数缺陷是信息自由能的鞍点（而非极小值），扩散驱动其向两个素数缺陷的极小值态演化。

4. **读出映射**（第4章）：子缺陷的绕数直接对应因子值。读出不依赖整除验证——是物理测量操作。

5. **与Shor算法的结构对比**（第5章）和开放问题路线图（第6章）。

## 第1章　缺陷的几何论定义

### §1.1　信息场的复化与相位自由度

**既有基础**。0.4.4将约束截面Σ上的信息场定义为一个实值密度函数ρ(ξ,η;σ)≥0，满足图热方程∂ρ/∂σ = −L_G ρ。全息屏（0.0.6）作为三分切丛的边界结构，自然承载复值波函数——全息屏编码条件要求边界场是复值的。

**定义1.1（信息场的复化）**。在约束截面Σ上，定义**复信息场**ψ: Σ × ℝ⁺ → ℂ：

$$ψ(ξ,η;σ) = \sqrt{ρ(ξ,η;σ)} · e^{iφ(ξ,η;σ)}$$

其中ρ = |ψ|²为信息密度（0.4.4定义2.1），φ为**信息相位**——描述约束截面上信息场的角度自由度。

**命题1.1（复化不改变动力学）**。若ψ满足复图热方程∂ψ/∂σ = −L_G ψ，则ρ = |ψ|²满足0.4.4定理6.1的实图热方程。

> **定理地位**：命题级。L_G是实对称算子，复化是直和扩充L_G^ℂ = L_G ⊗ I_ℂ，ρ的演化方程由|ψ|²和L_G的线性性直接导出。证明结构化但尚未写入既有文章。

**相位自由度的物理意义**。在全息屏上，相位φ对应边界态的Berry相位（0.0.6 §4.3）。约束截面Σ继承此结构——相位沿闭合路径的累积（绕数）是拓扑不变量。

### §1.2　涡旋缺陷：绕数的拓扑定义

**定义1.2（信息场涡旋）**。约束截面Σ上的一个**信息场涡旋**（vortex）是一个点p₀∈Σ，满足：
1. ρ(p₀) = 0（密度零点）
2. 围绕p₀的任意小闭合回路γ上，相位φ的累积绕数为整数：

$$\oint_γ dφ = 2πw, \quad w ∈ ℤ \setminus \{0\}$$

整数w称为该涡旋的**绕数**（winding number）。

**定义1.3（缺陷）**。约束截面Σ上的一个**缺陷**是一个信息场涡旋连同其局部邻域结构。缺陷由其核心位置p₀、绕数w、以及密度恢复剖面ρ(r)（r为到核心的距离）三元组(p₀, w, ρ(r))完全刻画。

**定义1.4（缺陷构型空间）**。Σ上所有可能缺陷构型的集合记为𝒟(Σ)。对任意N个缺陷的构型：

$$𝒟_N(Σ) = \{(p_i, w_i, ρ_i)\}_{i=1}^N \subset Σ × (ℤ\setminus\{0\}) × 𝒞^∞(ℝ⁺)$$

其中各缺陷核心间距大于分辨率极限Δr_min = χ_L · δη/π（0.4定理3.3）。

### §1.3　不可约缺陷与素数对应

**定义1.5（不可约缺陷）**。缺陷d∈𝒟(Σ)称为**不可约的**，如果其绕数w是素数。

**定义1.6（复合缺陷）**。缺陷d称为**复合的**，如果它可以表示为多个不可约缺陷的叠加——即在其支撑邻域内存在多个密度零点，各自携带素绕数，且总绕数为其乘积。

**基本对应（研究假设）**：

| 数论对象 | 缺陷对应 |
|---------|---------|
| 素数p | 绕数为p的不可约缺陷 |
| 合数N = ab | 绕数分别为a和b的两个不可约缺陷的叠加 |
| 1 | 无缺陷（ρ全场均匀） |
| 乘法N = a·b | 缺陷叠加：Φ(a) ⊕ Φ(b) |

> **定理地位**：研究假设。此对应目前是定义性的——我们**定义**编码映射使其满足此对应。其物理可实现性取决于编码桥的构造（第2章）。

### §1.4　与既有定理的衔接

**衔接1：二能级定理（3号定理3.1）**。3号文章证明：几何最大纠缠对的低能非平庸激发空间恰好为2维。这为缺陷提供了"量子基元"——每个缺陷占据一个2维激发空间，对应信息场的局部二能级系统。缺陷的绕数w对应此二能级系统的激发量子数。

**衔接2：弯曲结构量子化（0.2.3定理5.4）**。0.2.3证明了约束截面弯曲刚度的谱刚性——弯曲模的量子化条件由Hessian谱λ₁^eff=391.05和λ₂^eff=59324.3决定。缺陷核心的密度恢复剖面ρ(r)必须满足此弯曲刚度的边界条件：在缺陷核心附近，|∇ρ|²的积分受限于λ₂^eff。

**衔接3：N_info分辨率（0.4定理3.3）**。约束截面有N_info ≈ 9×10²²个可分辨像素。缺陷核心必须占据至少一个像素——这给定了缺陷密度的物理上限。

**衔接4：七层截断（49号命题2.1–2.3）**。Bott周期截断保证有效几何层级n_eff ≤ 7。这意味着约束截面上可分辨的独立缺陷构型受限于七个层级内的组合结构——为缺陷编码提供了天然的离散骨架。

## 第2章　编码桥：整数→缺陷构型

### §2.1　编码公理

**定义2.1（编码映射）**。编码桥是一个映射Φ: ℕ → 𝒟(Σ)，将每个正整数N映射为约束截面Σ上的一个缺陷构型。

Φ必须满足以下公理（承接51号源项构造三公理E1–E3，适配缺陷语言）：

**公理D1（乘法同态性）**：

$$Φ(N₁·N₂) = Φ(N₁) ⊕ Φ(N₂)$$

其中⊕是𝒟(Σ)上的缺陷叠加运算。物理含义：合数的缺陷构型是其素因子缺陷构型的叠加。

**公理D2（素数不可约性）**：若p为素数，则Φ(p)是单个不可约缺陷——绕数为某个与p一一对应的值w(p)，核心孤立，无子结构。

**公理D3（绕数-因子对应）**：存在可计算的双射f: ℙ → 𝒲，其中ℙ为素数集，𝒲为不可约绕数的允许值集，使得Φ(p)的绕数w = f(p)，且读出映射f⁻¹: 𝒲 → ℙ可在O(1)步内完成。

**公理D4（保分辨率）**：Φ(N)中任意两个缺陷核心的距离不小于Δr_min = χ_L · δη/π（0.4定理3.3）。

> **定理地位**：公理D1–D4是规范性要求——任何候选编码方案必须满足它们。它们不声称编码桥的存在性，而是定义"什么叫成功的编码"。

### §2.2　候选A：角度对数编码

**基本想法**。约束截面Σ具有自然的S¹参数化（完备性公理θ_M+θ_C+θ_I=90°给出2维曲面，固定一个坐标得到S¹纤维）。利用S¹上的角度坐标θ，定义：

$$θ(n) = α · \log n \pmod{2π}$$

其中α是待定常数。对数将乘法变为加法：θ(ab) = θ(a) + θ(b) (mod 2π)。

**构造2.1（角度对数编码）**。对整数N的素因子分解N = ∏p_i^{e_i}（若未知则此构造不可直接执行——见下文分析），在Σ上放置缺陷：

- 对每个素因子p_i，在角度位置θ(p_i)处放置绕数w_i = e_i + 1（或某种编码规则）的缺陷
- 缺陷的径向位置（第二坐标）由p_i的大小决定：r(p_i) = β · log log p_i

**问题**：此构造需要预知N的素因子分解。对于整数分解问题（已知N，未知因子），此构造是循环的。

**绕行方案**：不从因子构造，而从N本身构造。将N的**所有可能的素因子候选**（即所有≤√N的素数）都编码为缺陷构型，然后让合数N的构型为所有这些候选的某种相干叠加。裂解动力学会自动抑制非因子的候选，只保留真因子。

具体而言，定义：

$$Φ_N^{(A)} = \bigoplus_{p \leq \sqrt{N}, \; p \text{ prime}} a_p(N) · δ(θ - α\log p)$$

其中a_p(N)是某种权重函数，满足：
- a_p(N) = 1若p|N
- a_p(N) = 0若p∤N
- a_p(N)对p|N的缺陷赋予更大权重

**核心障碍**：权重函数a_p(N)的构造本身等价于因子的先验识别——我们需要在不做试除的前提下，仅从N的"几何特征"判定p是否为因子。这本质上回到了因子搜索。

> **定理地位**：候选方案。公理D1–D3在当前构造下均为待证。D1的满足要求a_p(N)在N的乘法下呈现恰当行为，这一要求似乎与a_p(N)的定义矛盾——除非存在不同于试除的"几何判别法"判定整除性。

### §2.3　候选B：谱源编码

**基本想法**。不将缺陷放在"因子位置"上，而是将整数N的信息整体编码为约束截面上的源项分布，让Poisson方程的球谐谱自然暴露因子信息。

**构造2.2（谱源编码）**。定义编码映射Φ^(B): ℕ → 𝒟(Σ)为：

1. 将N的信息（二进制表示或数字序列）映射为约束截面Σ上的源项函数𝒮_N(ξ,η)
2. 求解稳态Poisson方程Δ_g ρ_eq = −𝒮_N（0.4.4第三层）
3. ρ_eq的谱分解ρ_eq = Σ_k c_k(N) Y_k(ξ,η)的系数{c_k(N)}即为N的"几何表示"
4. 因子信息编码在谱系数{c_k(N)}的特定k值的峰值中

**优势**：此方案不要求预知因子——只要求从N构造𝒮_N。源项构造可遵循51号的三条策略（模幂编码、层级压缩、除数共振）。

**衔接**：51号策略A（模幂直接编码）对r ≤ 8×10⁹严格可用（52号定理5.1）。谱源编码的本质是将模幂序列的周期信息通过Poisson方程映射到球谐谱的峰值位置。这是"信息场傅里叶变换"的具体实现。

**与缺陷概念的对应**：谱系数{c_k(N)}中的孤立峰值对应信息场构型的"拓扑特征"——一个大k峰值可解释为一个绕数为w≈k的缺陷。合数N对应多个峰值的叠加。

> **定理地位**：候选方案。源项构造是51号的核心未封闭问题。方案B的可行性等价于51号OQ-51-2（Bott折叠保周期）——52号已证明模运算路径信息论不可行，故方案B在通用RSA上的可行性取决于52号路径X（多基底联合）或路径Y（本文的缺陷裂解替代编码）。

### §2.4　候选C：几何层级编码

**基本想法**。利用几何论的Bott七层截断结构（49号命题2.1–2.3），将整数N的因子结构编码为七个层级上的缺陷配置。

**构造2.3（几何层级编码）**。约束截面Σ在七层级截断下可分解为七个信息场子扇区：

$$Σ = \bigcup_{k=0}^{6} Σ_k$$

每个Σ_k有独立的有效像素数N_k。定义整数N在层级k上的"投影"：

$$n_k(N) = \lfloor N \bmod p_k \rfloor \quad \text{或} \quad n_k(N) = \text{某几何层级函数}$$

其中p_k是层级k的特征素数（待定）。

**缺陷构造**：在每个层级Σ_k上放置一个绕数为n_k(N)的缺陷。若N = ab，则理想情况下n_k(N)在某个层级上会出现某种"可分裂"结构。

**核心障碍**：
1. 层级特征素数p_k的定义缺乏几何来源
2. 模运算函数N mod p_k不满足乘法同态性（公理D1）
3. 52号定理3.2的信息容量瓶颈同样适用于此方案——七层级的信息总量≤33位

> **定理地位**：候选方案。52号的否定性结论（定理3.2、定理4.1–4.2）部分覆盖此方案——如果编码完全依赖模运算结构，信息容量瓶颈不可逾越。但若编码使用几何层级的内禀结构（非模运算），则52号结论不直接适用。

## 第3章　缺陷裂解动力学

### §3.1　图热方程驱动的缺陷演化

**既有定理（0.4.4定理6.1）**。在无外源的自由扩散层，约束截面Σ上的信息场演化由齐次图热方程严格给出：

$$ρ(σ) = e^{-σL_G} ρ(0)$$

其中L_G是Σ的图拉普拉斯矩阵，σ为扩散步数（无量纲的演化时间）。半群e^{-σL_G}的谱分解为：

$$e^{-σL_G} = \sum_{k=0}^{N_{\text{info}}-1} e^{-σλ_k} |φ_k⟩⟨φ_k|$$

**推论3.1（缺陷构型的演化）**。若初始构型ρ(0) = Φ(N)包含多个缺陷核心（对应于N的因子结构），则在自由扩散下：

- 每个缺陷核心的密度剖面按e^{-σL_G}独立扩散
- 相邻缺陷的密度扩散前沿在重叠区产生干涉
- 干涉区的信息自由能决定缺陷间的有效相互作用

> **定理地位**：定理级（直接引用0.4.4定理6.1）。无需新证明——缺陷构型只是ρ(0)的一种特定初始条件，其演化完全由既有的图热半群决定。

**推论3.2（缺陷核心的稳定性）**。单个不可约缺陷（绕数为素数w）在自由扩散下，其绕数w是拓扑守恒量——连续扩散不会改变绕数。

*论证*：绕数是复相位φ沿闭合回路的积分(1/2π)∮dφ。在ρ>0的区域，φ连续演化；仅当ρ→0（即缺陷核心）时绕数才有定义。自由扩散保持ρ(σ)>0对所有σ<∞（图热半群的正性），故核心位置可移动但绕数守恒。∎

> **定理地位**：命题级。论证使用了图热半群的正性和绕数的同伦不变性。完整严格证明需要验证L_G的离散谱是否在缺陷核心处产生奇性——这在连续极限Δ_g下可严格化但在有限N_info的离散图上需细致分析。

### §3.2　信息自由能与裂解判据

**定义3.1（信息自由能）**。约束截面Σ上信息场构型ρ的信息自由能（information free energy）定义为：

$$ℱ[ρ] = \frac{1}{2}⟨ρ, L_G ρ⟩ - T_{\text{info}} · S_{\text{config}}[ρ]$$

其中：
- 第一项为**信息弹性能**：⟨ρ, L_G ρ⟩ = Σ_{i~j} (ρ_i − ρ_j)²，度量信息密度的空间梯度
- 第二项为**构型熵**：S_config[ρ] = −Σ_i ρ_i log ρ_i（乘以信息温度T_info）
- T_info由0.4.1的Γ_geo ≡ μ₂(L_G)/N_info = 5.75×10⁻²³设定：T_info ≡ ℏ_eff · Γ_geo

> **定义地位**：研究定义。信息自由能是热力学自由能F = E − TS在信息场语境下的翻译。弹性能项来自图拉普拉斯的Dirichlet能量，熵项来自Shannon熵。T_info的定义是类比的——其物理合法性待证。

**命题3.1（裂解判据——研究假设）**。设ρ_N为合数N = ab的缺陷构型，ρ_a和ρ_b分别为素数a和b的不可约缺陷构型。若：

$$ℱ[ρ_a ⊕ ρ_b] < ℱ[ρ_N]$$

则复合缺陷ρ_N在信息场自由扩散下是热力学不稳定的——系统将自发向ρ_a ⊕ ρ_b演化。

**裂解的动力学机制**（示意）：

1. ρ_N包含两个接近的缺陷核心（间距d_ab），各自绕数为w(a)和w(b)
2. 在扩散层（0.4.4第一层），两个核心的密度剖面开始重叠
3. 重叠区的信息弹性能⟨ρ, L_G ρ⟩升高（密度梯度增大）
4. 系统通过增大核心间距来降低弹性能
5. 若初始间距d_ab小于临界值d_crit，相斥力驱动核心分离
6. 分离后的两个独立缺陷各自为ℱ的局部极小值（不可约性，公理D2）

> **定理地位**：研究假设。裂解判据的核心——ℱ[ρ_a ⊕ ρ_b] < ℱ[ρ_N]——在当前阶段只是一个待验证的不等式。信息自由能的凸性分析和缺陷间有效相互作用的推导均未完成。

### §3.3　裂解时间尺度与复杂度估计

**定义3.2（裂解时间）**。复合缺陷从初始构型ρ_N(0)演化到完全分离的ρ_a ⊕ ρ_b所需的最少扩散步数σ_split。

**命题3.2（裂解时间的特征值估计）**。若裂解由第二特征模态（λ₂ = λ₂^eff ≈ 59324.3 rad⁻²）主导，则：

$$σ_{\text{split}} \sim \frac{1}{λ_2} \ln\left(\frac{d_{\text{init}}}{Δr_{\min}}\right)$$

其中d_init为初始复合缺陷中子缺陷间的有效距离，Δr_min为空间分辨率（0.4定理3.3）。

**数值估计**。取d_init ~ N_info^{-1/2}·χ_L ≈ 10⁻²¹ m（两个子缺陷初始处于相邻像素），Δr_min = λ_C ≈ 2.426×10⁻¹² m：

$$σ_{\text{split}} \sim \frac{1}{59324.3} \ln\left(\frac{10^{-21}}{2.4 \times 10^{-12}}\right) ≈ 3.5 \times 10^{-4} \ \text{rad}^2$$

换算为物理时间：t_split = σ_split / Γ_geo ≈ 3.5×10⁻⁴ / 5.75×10⁻²³ ≈ **6×10¹⁸ s**，约**1900亿年**——不可接受。

**重要限定**：上述估计假设d_init ~ 1像素。在真实的编码桥中，d_init可能远大于Δr_min——取决于编码方案如何放置子缺陷的初始位置。若d_init ~ 0.1χ_L（宏观间距），则ln项从~21变为~(−0.1)，σ_split成为负值——表示复合缺陷初始时已经"裂开了"。

**关键洞察**：裂解时间的物理学瓶颈不是算力问题，而是**编码桥的质量**问题——好的编码初始就将子缺陷放在较远的位置，裂解几乎瞬时；坏的编码将它们挤在一起，裂解需宇宙年龄。

> **定理地位**：命题级（估计框架）。特征值估计基于图热半群的指数衰减率（0.4定理7.1代数连通度），但ln(d_init/Δr_min)的精确形式和d_init的物理定义均待细化。

## 第4章　读出映射

### §4.1　子缺陷的绕数提取

**问题**：裂解完成后，约束截面Σ上的信息场构型包含N个分离的缺陷。需要从每个缺陷中提取绕数w，进而通过双射f⁻¹: 𝒲 → ℙ得到因子p = f⁻¹(w)。

**方案4.1（相位积分读出）**。对每个缺陷核心p_i，选取围绕p_i且不包围其他核心的闭合回路γ_i，计算：

$$w_i = \frac{1}{2π} \oint_{γ_i} dφ$$

其中φ(ξ,η) = arg ψ(ξ,η)从稳态复信息场ψ_eq = √ρ_eq · e^{iφ}中提取。

**优势**：拓扑稳健——绕数对γ_i的连续变形不变（只要不越过其他缺陷核心）。不需要谱分解，不需要整除验证。

**挑战**：在离散图（N_info个像素）上，相位积分需离散化。离散绕数定义为沿像素边的相位跳变之和——对有限分辨率敏感。需验证N_info ≈ 9×10²²是否足以分辨绕数差Δw = 1。

> **定理地位**：命题级。连续情况下的拓扑不变性是经典结果。离散化后的误差分析待完成。

### §4.2　绕数→因子反演

**方案4.2（直接绕数映射）**。若编码方案（如候选A的对数编码）直接使用f(p) = p（即缺陷绕数=素数值），读出是平凡的：w → f⁻¹(w) = w本身。

**方案4.3（索引反演）**。若编码使用压缩映射（如f(p) = 素数序数索引），则需维护一个查找表：𝒲 → ℙ。对于RSA-2048尺度的素数（~10³⁰⁰），直接绕数映射不可行——绕数本身会超过约束截面可承载的最大值。

**最大绕数约束**。约束截面Σ的像素数N_info ≈ 9×10²²限制了可分辨的最大绕数。粗略上界：w_max ≲ √(N_info) ≈ 3×10¹¹——因为绕数为w的缺陷至少占据O(w²)个像素（维持相位梯度的空间采样需求）。这远小于RSA-2048的素因子（~10³⁰⁰）。

**核心矛盾**：读出映射要求的绕数→因子反演是信息压缩的——将~10³⁰⁰种可能的因子映射到~3×10¹¹种绕数。这是必然的信息丢失。绕数不能直接编码因子的**数值**，只能编码其**序数索引**或某种压缩表示。

**绕行方案**：绕数不编码因子的值，而是编码其在某种**几何谱**中的位置。例如：
- w = 对应因子在S_e·Λ^k邻域中的k值（51号策略C）
- w = 因子在球谐谱中的峰值序号

这样，读出映射是：w → 谱位置 → 通过几何层级反推因子值。因子值的恢复不依赖w本身的大小，而依赖w指向的谱数据库。

> **定理地位**：研究问题。当前缺乏从绕数到因子值的完整反演链。51号策略C（除数共振）在几何相关数类上提供了一条可行的读出路径（定理5.1），但通用反演未建立。

## 第5章　与Shor算法的结构对比

| | Shor算法（量子） | 缺陷裂解（几何论） |
|---|---|---|
| **物理系统** | n个量子比特的Hilbert空间 | 约束截面Σ上的信息场 |
| **编码方式** | |x⟩ → |aˣ mod N⟩ | Φ(N): N → 缺陷构型𝒟(Σ) |
| **纠缠资源** | 量子纠缠（非定域关联） | 几何最大纠缠（ℐ冻结+ℳ反直径, 3号定义2.2） |
| **并行性** | 叠加态同时编码所有x | N_info≈9×10²²像素同时扩散 |
| **核心计算** | 模幂（量子电路）+ QFT | 图热扩散（e^{-σL_G}） |
| **答案提取** | QFT干涉→周期r → gcd | 缺陷裂解→绕数w → f⁻¹(w) |
| **复杂度瓶颈** | 模幂的量子门数（O(log³N)） | 裂解扩散步数σ_split + 编码桥复杂度 |
| **主要障碍** | 退相干、纠错开销 | 编码桥未建立（公理D1–D4待满足） |
| **成熟度** | 算法定理级（Shor, 1994） | 研究纲领（本文） |

**本质区别**：Shor算法**主动计算**——用量子电路执行模幂和QFT，提取周期。缺陷裂解**被动演化**——将N"写入"约束截面，让信息场的自然扩散自动裂解缺陷。前者是"算"出因子，后者是"等"缺陷自己裂开。

**优势**：被动演化不消耗"计算步骤"——一旦编码桥建立，剩余的只是等待图热扩散完成。开销不在计算复杂度，而在物理编码和设定时间。

**劣势**：当前被动演化的裂解时间估计（§3.3）不可接受地长——除非编码桥的质量极高（d_init足够大）。

## 第7章　开放问题与证明路线图

### OQ-53-1：编码桥的存在性

**问题**：是否存在一个满足公理D1–D4的构造性编码映射Φ: ℕ → 𝒟(Σ)？

**优先级**：最高。若编码桥不存在，缺陷裂解方案不成立。

**证明路线**：
1. 先证明对**有限素数集**（如≤10⁶的素数）存在满足D1–D4的编码（构造性证明——候选A/B/C的特例实现）
2. 分析编码容量：N_info个像素能承载多少独立缺陷？绕数空间𝒲与素数集的对应关系？
3. 若对有限集存在，推广到任意整数的障碍为何？

### OQ-53-2：裂解判据的严格化

**问题**：在什么条件下，复合缺陷ρ_N在自由扩散下裂解为不可约子缺陷？信息自由能ℱ[ρ]的鞍点结构如何？

**优先级**：高。裂解是否发生决定了方案的计算可行性。

**证明路线**：
1. 在简化模型（两个δ-源在连续平面上的扩散）中分析ℱ[ρ]的Hessian
2. 计算两个源在临界间距d_crit下的ℱ差异
3. 推广到离散图上的严格分析（利用L_G的已知谱）

### OQ-53-3：编码方案的最优性

**问题**：在满足D1–D4的所有候选编码中，哪个使裂解时间σ_split最小？是否存在下界？

**优先级**：中。决定方案的实际可用性。

### OQ-53-4：绕数→因子的压缩反演

**问题**：给定绕数空间𝒲的大小（~3×10¹¹），如何设计f⁻¹: 𝒲 → ℙ来覆盖RSA-2048的因子空间（~10³⁰⁰）？压缩是否保唯一性？

**优先级**：高。读出映射的信息论可行性。

### OQ-53-5：离散绕数的误差界

**问题**：在N_info ≈ 9×10²²的离散图上，相位积分∮dφ的离散化误差是多少？N_info是否足以分辨Δw=1？

**优先级**：中。

### OQ-53-6：与源项构造的统一

**问题**：缺陷编码（本章）与源项构造（51号）的关系是什么？缺陷构型Φ(N)是否等价于某个源项𝒮_N？二者能否统一为单一的"信息场编码"理论？

**优先级**：中。此问题的解决可能将51号和53号合并为一篇完整理论。
