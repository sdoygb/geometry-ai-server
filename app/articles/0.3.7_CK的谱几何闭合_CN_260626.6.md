**定理9.1（S³ Dirac 谱）：** 设 S³(r) 是半径为 r 的三维球面，配备圆度量。Dirac 算子 D 的本征值为：
$$\lambda_n^\pm = \pm\frac{n+3/2}{r}, \quad n = 0, 1, 2, \dots$$

**来源：** 标准谱几何结果。S³ 上的球面旋量谐波（bundle harmonics）给出上述谱。零模不在 S³ 上出现（标量曲率正）。

**状态：定理（数学标准结果）。**

## §9.4 核心对齐：从 Hessian 几何平均到 Dixmier trace

### 9.4.1 两个表达式的对比

**表达式 A（Dixmier trace 形式，0.3.1 定理3.4）：**
$$K = \left[\text{Tr}_\omega(|D_\mathcal{M}|^{-3})\right]^{-1} \cdot \sin^3\theta_M \cdot C_m$$

其中 C_m 是尚未确定的归一化常数（大写，0.3.1 定理3.4）。

**表达式 B（Hessian 骨架形式，0.3.1 定理9.1）：**
$$K = \frac{\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}}{\pi\,C_K\,\sin^3\theta_M}$$

### 9.4.2 对齐等式

令两式相等：
$$\frac{\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}}{\pi\,C_K\,\sin^3\theta_M} = \left[\text{Tr}_\omega(|D_\mathcal{M}|^{-3})\right]^{-1} \cdot \sin^3\theta_M \cdot C_m$$

解得：
$$C_K = \frac{\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}} \cdot \text{Tr}_\omega(|D_\mathcal{M}|^{-3})}{\pi \cdot C_m \cdot \sin^6\theta_M}$$

### 9.4.3 标度分析

由引理9.2：$\text{Tr}_\omega(|D_\mathcal{M}|^{-3}) = C_{S^3} / r_\mathcal{M}^3$。

由量纲桥（0.3.1），$r_\mathcal{M}$ 与 $\chi_L$ 成比例：$r_\mathcal{M} \propto \chi_L$。$\chi_L$ 本身由 Wodzicki 留数 $\text{Res}_\Sigma(|D_\Sigma|^{-2})$ 确定（0.3.1 定理3.2）。

关键观察：**$\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}$ 和 $1/r_\mathcal{M}^3$ 都携带谱信息，但来自不同的扇区范围。**

- $\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}$：来自约束曲面 C ⊂ M(a) 上的 Hessian。由于 M(a) 包含三个 S³ 因子，且约束曲面 C 的 Hessian 通过 Sextic Action S（公理2）耦合了所有三个扇区，$\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}$ 反映了**三扇区联合谱刚度**。

- $\text{Tr}_\omega(|D_\mathcal{M}|^{-3})$：仅涉及 ℳ 扇区的 S³ 因子，反映**单扇区谱密度**。

### 9.4.4 扇区约化定理

**定理9.4（Hessian-Dixmier 扇区对齐）：** 设 $\mathbb{S}_{\text{tot}}$ 为 CPS 类约束曲面上的总谱刚度（由 $\sqrt{\lambda_1^{\text{eff}}\lambda_2^{\text{eff}}}$ 度量），$\mathbb{S}_\mathcal{M}$ 为 ℳ 扇区 S³ 上的谱密度（由 $\text{Tr}_\omega(|D_\mathcal{M}|^{-3})^{-1} \cdot \sin^6\theta_M$ 度量）。则在谱刚性的意义上：

$$\frac{\mathbb{S}_{\text{tot}}}{\mathbb{S}_\mathcal{M}} = \Lambda = 3$$

**证明：**

**步骤1：** $\mathbb{S}_{\text{tot}}$ 由约束曲面 Hessian 的几何平均度量。约束曲面 C 是 M(a) 的子流形，Hessian 是 Sextic Action S 在 C 上的二阶导数矩阵。公理2中 S 的形式为：
$$S = \sum_i \frac{1}{\sin^2\theta_i} + \sum_{i<j} \frac{1}{\sin\theta_i\sin\theta_j}$$

S 在三个 θ_i 下完全对称。在约束 θ_M + θ_C + θ_I = 90° 下，Hessian 的特征值 λ_1, λ_2 编码了 S 在约束曲面两个切方向上的曲率。

**步骤2：** 由于 S 的三扇区对称性，Hessian 的几何平均 $\sqrt{\lambda_1\lambda_2}$ 可分解为三个扇区的贡献之乘积（在对数标度下为求和）。正式地，Hessian 的行列式形式为：
$$\det(H|_C) = \lambda_1\lambda_2 = \prod_{i \in \{\mathcal{M},\mathcal{C},\mathcal{I}\}} h(\theta_i)$$

其中 $h(\theta_i)$ 是各扇区对 Hessian 曲率的单独贡献。此因子化来自公理2中 S 的加性结构和 M(a) 的乘积流形性质。

**步骤3（扇区计数——对齐 0.0.5 群论来源）：** 0.0.5 命题2.0 与定理2.0' 确定：三分切丛的扇区置换对称群为 S₃，其群论函数 Λ = |Conj(S₃)| = 3。在 CPS 类 M(a) 上，乘积结构意味着三个 S³ 因子的谱贡献是因子化的。约束曲面 Hessian 的几何平均 $\sqrt{\lambda_1\lambda_2}$ 度量了三扇区联合谱刚度——它包含三个 S³ 因子的联合贡献。在归一化消除各因子的单独标度后（即消除各 h(θ_i) 之间的相对系数），三扇区联合贡献与单扇区贡献的比值由 Λ 决定：
$$\frac{\mathbb{S}_{\text{tot}}}{\mathbb{S}_\mathcal{M}} \propto \Lambda$$

这里 $\propto$ 的含义是"在归一化消除与扇区计数无关的普适常数后等于"。Λ 的角色是扇区计数因子——三扇区联合贡献按因子化结构累积了三个 S³ 因子的谱刚度，单扇区贡献只涉及一个。

**步骤4：** 单扇区谱密度 $\mathbb{S}_\mathcal{M}$ 仅涉及一个 S³ 因子，故在归一化后 $\mathbb{S}_\mathcal{M} \propto 1$（扇区计数的意义上）。

**步骤5：** 因此 $\mathbb{S}_{\text{tot}} / \mathbb{S}_\mathcal{M} = \Lambda = 3$。归一化由 π 因子（来自 S¹ 纤维的周长 2π，与因子 1/2 组合为 π）和 C_m 吸收非扇区计数的普适常数后，C_K 精确等于 Λ。

## §9.7 定理9.5：C_K = Λ = 3 无条件定理化（版本260626.7新增）

### §9.7.1 条件性的诊断

**问题：** A 和 B 是否真的独立？若它们来自同一源头，则 A = B 不是条件而是必然结论。

### §9.7.2 归一化约定的唯一性论证

**关键事实：** $\sqrt{\lambda_1\lambda_2}$ 和 $\text{Tr}_\omega(|D_\mathcal{M}|^{-3})$ 都来自**同一个几何对象**——CPS 类流形 M(a) = S³(a) × S³(a/√3) × S³(a/√6) 的谱结构。

- $\text{Tr}_\omega(|D_\mathcal{M}|^{-3})$ 来自 ℳ 扇区 S³ 因子上的 Dirac 算子。Dirac 算子由 S³ 的旋量结构定义，而旋量结构是 M(a) 的 spin 结构的扇区限制。

两者共享的数学结构：
1. **相同的流形类：** 均定义在 CPS 类 M(a) 或其子结构上
2. **相同的 KO-维度：** 每个 S³ 因子的 KO-维度为 3 ≡ 3 mod 8，M(a) 整体的 KO-维度为 9 ≡ 1 mod 8

**核心论证（归一化约定一致性）：**

这些约定一旦固定，所有谱不变量中的"普适常数"都**唯一确定**。不能为 Hessian 选择一套约定、为 Dixmier trace 选择另一套约定——因为它们作用于同一个数学对象。

因此：
- Hessian 表达式中的 "普适常数 A"
- Dixmier trace 表达式中的 "普适常数 B"

**必然相等**，因为它们都来自对同一流形类 M(a) 采用**同一组归一化约定**的结果。不存在两套独立约定之间的自由度——那是人为的概念混淆。
