## 摘　要

5号文章定理2.2（SU(3)结构）依赖于一个未证明的核心跃迁：**S_total=0 ⟹ det=1（幺模性）**。该命题已被5号文章诚实标注为"条件性假设"，并列为开放问题。本文旨在封闭这一缺口。

核心困难在于：S_total 定义在联合截面 Σ_joint^(N)（粒子态空间）上，而 det 是 ℂ³（单粒子扇区复化空间）上 U(3) 群元素的模。两者之间的桥梁从未被显式建立。

本文沿两条互补路径推进：

- **路径A（辛约化）**：将 S_total 识别为 U(1) 中心子群在联合截面上的动量映射。S_total=0 → 动量映射零水平集 → Marsden-Weinstein 辛约化 → 消去 U(1) 因子 → SU(3)。
- **路径B（代数约束）**：从 𝔲(3) = 𝔲(1) ⊕ 𝔰𝔲(3) 的直和分解出发，证明 S_total=0 冻结了 𝔲(1) 生成元的作用，从而使有效对称群从 U(3) 约化为 SU(3)。

两条路径共享同一个核心缺口：**S_total 与 U(1) 生成元（或动量映射）之间的精确正比关系尚未被显式计算。** 本文为该缺口的封闭提供了完整的数学框架，精确标注需补充的计算步骤，并将定理2.2从"条件性命题"升级为"构造性定理（待完成步骤3）"。

**关键词：** 幺模性；S_total；辛约化；动量映射；SU(3)；U(1)中心子群

## §3 路径A：辛约化（Hamiltonian约化）

### 3.1 策略概述

1. 将 S_total 识别为联合截面上 U(1) 中心子群作用的动量映射（或与之成正比）。
2. S_total=0 → 零水平集 μ⁻¹(0)。
3. 应用 Marsden-Weinstein 辛约化 → U(1) 因子被消去 → 剩余对称群为 SU(3)。

### 3.2 步骤A1：联合截面的辛结构

由0.0.6 §4.1，六项作用量 S(θ) 是角度构型空间 (D_θ, ω_θ) 上的 Hamilton 函数。具体地，存在辛形式 ω_θ 使得 S(θ) 的 Hamilton 向量场 X_S 满足：

$$i_{X_S} \omega_\theta = -dS$$

联合截面 Σ_joint^(N) 是 N 个单粒子截面 Σ_η 的乘积的子流形（5号文章定义1.2）。乘积辛流形上的 Hamilton 函数为：

$$S_{\text{total}} = \sum_{i=1}^N S^{(i)}(\theta^{(i)})$$

其中每个 S^(i) 是其对应因子辛流形上的 Hamilton 函数。因此 S_total 是乘积辛流形上的 Hamilton 函数。

**已确立**：S_total 是联合截面 Σ_joint^(N) 上的 Hamilton 函数。

### 3.3 步骤A2：ℂ³ 上的 U(1) 作用与动量映射

由5号文章定理2.1，每个扇区 V_X^ℂ ≅ ℂ。由定义2.2，三扇区复化联合构成 ℂ³。

U(3) 在 ℂ³ 上有标准作用（矩阵乘法）。U(1) 中心子群的作用为：

$$e^{i\phi} \cdot I_3 : \Psi = (\psi_{\mathcal{M}}, \psi_{\mathcal{C}}, \psi_{\mathcal{I}}) \mapsto e^{i\phi}(\psi_{\mathcal{M}}, \psi_{\mathcal{C}}, \psi_{\mathcal{I}})$$

此作用保持厄米特形式 h(Ψ,Ψ) = ‖Ψ‖² 不变。5号文章定理2.2的论证起点——"h(Ψ,Ψ) 守恒 → U(3)"——正对应于此。

U(1) 中心子群在 ℂ³ 上的动量映射（外部数学事实 MW-2）为：

$$\mu(\Psi) = \frac{i}{2} \|\Psi\|^2 = \frac{i}{2} h(\Psi,\Psi)$$

（常数因子 i/2 是约定；任何非零常数倍在辛约化中给出等价结果。）

### 3.4 步骤A3：桥接 S_total 与动量映射——核心缺口

这是整个封闭链条中尚未完成的步骤。

**需要证明的命题**：

> **命题A3（待证）。** 在联合截面 Σ_joint^(N) 的切空间上，S_total 的 Hamilton 向量场 X_{S_total} 与 U(1) 中心子群作用生成的向量场 ξ_U(1) 处处平行（即存在非零光滑函数 f 使得 X_{S_total} = f · ξ_U(1)）。

等价地：

> **命题A3'（待证）。** 动量映射 μ(Ψ) ∝ S_total（相差非零函数因子），即 μ 的零水平集与 S_total 的零水平集重合。

**为什么这在几何上是自然的（非证明的启发性论证）**：

1. S_total 是联合截面上唯一有量纲的标量 Hamilton 函数。U(1) 中心子群的作用（全局相位旋转）改变的是标度信息。
2. 在真空极限 p₀ 处，S_total=0 且 ‖Ψ‖=0（真空无激发）。两者的零点重合。
3. 联合截面的维度消去机制（5号文章定理1.1：S_total=0 消去一个全局标度自由度）在物理上恰好对应 U(1) 因子被冻结。

但这些是启发性论证，不是严格证明。尚需完成的具体计算包括：

- **(a)** 显式写出联合截面上的 Hamilton 向量场 X_{S_total}；
- **(b)** 显式写出 U(1) 中心子群在联合截面切空间上的作用；
- **(c)** 验证 (a) 和 (b) 的向量场处处平行。

**缺口等级**：技术性缺口——步骤清晰，需要的是显式计算而非新概念。

### 3.5 步骤A4：辛约化给出 SU(3)

假设命题A3成立。则：

1. S_total=0 ⟺ μ⁻¹(0)（动量映射零水平集）。
2. 由 Marsden-Weinstein 定理（MW-1）：约化空间 μ⁻¹(0)/U(1) 是辛流形，原 U(3) 作用中的 U(1) 中心子群因子被消去。
3. U(3)/U(1) ≅ SU(3)/ℤ₃（MW-3）。ℤ₃ 是离散子群，不影响李代数结构。
4. 因此：有效对称群为 SU(3)，det=1 自动成立。

**这一步在命题A3成立的条件下是严格的。** 辛约化的所有要素——辛流形、紧李群作用、动量映射——均已在框架内或外部数学事实中确立。

## §4 路径B：代数约束（标度假定）

### 4.1 策略概述

不依赖辛几何，直接从李代数层面论证：

1. 𝔲(3) = 𝔲(1) ⊕ 𝔰𝔲(3)，其中 𝔲(1) 生成元为 I₃。
2. 𝔲(1) 生成元的作用对应于系统的全局标度变换。
3. S_total=0 作为真空标记条件，固定了全局标度 → 𝔲(1) 生成元的作用被冻结。
4. 有效李代数 → 𝔰𝔲(3)，有效李群 → SU(3)。

### 4.2 步骤B1：𝔲(3)的直和分解

由5号文章定理2.3，𝔰𝔲(3) 的8个生成元已显式构造（Gell-Mann矩阵 λ_a/2）。𝔲(3) 的完整李代数为：

$$\mathfrak{u}(3) = \mathfrak{u}(1) \oplus \mathfrak{su}(3)$$

其中 𝔲(1) 生成元为 I₃（3×3 单位矩阵），其在 ℂ³ 上的指数映射为 e^{iφ}I₃——即 U(1) 中心子群。

### 4.3 步骤B2：U(1) 生成元的物理意义——核心缺口

**需要证明的命题**：

> **命题B2（待证）。** U(1) 中心子群的生成元 I₃ 在联合截面 Σ_joint^(N) 上的作用等价于改变 S_total 的值。具体地，存在李代数同态 φ: 𝔲(1) → ℝ 使得 exp(tI₃) 的作用将 S_total 平移 t·c（c ≠ 0 为常数）。

等价地：

> **命题B2'（待证）。** S_total 是 U(1) 中心子群作用的 Noether 荷（相差常数因子）。

如果命题B2成立，则 S_total=0 意味着 U(1) 生成元的作用不改变物理态——即 U(1) 对称性在真空标记条件下被平凡化。此时：

- 有效对称李代数 = 𝔲(3) / 𝔲(1) ≅ 𝔰𝔲(3)
- 有效对称李群 = SU(3)（mod ℤ₃ 离散商）
- det=1 自动成立

### 4.4 为什么这不等价于路径A

路径A（辛约化）和路径B（代数约束）在数学上是互补的：

- **路径A** 依赖辛几何结构（ω_θ, Hamilton 向量场），从动量映射零水平集的辛约化得到 SU(3)。这在数学上更严格，但要求显式计算向量场。
- **路径B** 依赖李代数表示论，从 𝔲(3) 的直和分解和 S_total 的 Noether 性质得到 SU(3)。这在物理上更直观，但要求建立 S_total 与 U(1) 生成元之间的代数关系。

两条路径最终可以在 Noether 定理的框架下统一：动量映射 μ 正是 Noether 荷的辛几何版本。但在当前框架内，两条路径提供了独立的验证手段——如果两者都走通，幺模性封闭将具有双重数学保障。

- 这与2号文章（260626.6）§8.3 的诚实声明完全一致：Cl(7)不能自然产生 𝔰𝔲(3)，但 SU(3) 可以通过其他几何构造（本文的三扇区复化辛约化）获得。

### 6.3 对"SU(3)在几何论中没有起源"指责的回应

当CG-1封闭后，以下辩护成立：

> SU(3) 在几何论中**有**起源——它来自三扇区复化 ℂ³ 的 U(3) 对称性经 S_total=0 真空标记条件的辛约化。它不在 Clifford 代数约化链中出现，是因为 𝔰𝔲(3)（8维，跨扇区）和 𝔰𝔲(2)（3维，扇区内）属于不同的几何层次。两个层次不冲突，如同流形的全局对称性与局部全息屏的对称性不冲突。
